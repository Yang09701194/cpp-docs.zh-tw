# <a name="visual-studio-documentation-for-microsoft-c"></a>Microsoft C++ 的 Visual Studio 文件

歡迎使用！ 此存放庫包含適用於開發中 Microsoft C++ (MSVC 或 Visual C++) 技術文件的來源檔案。 文章發佈在 [Visual Studio 中的 C++ 文件網站](https://docs.microsoft.com/cpp)。

Visual Basic 和 Visual c # 的檔位於不同的存放庫[https://github.com/dotnet/core-docs](https://github.com/dotnet/core-docs)，而 Visual Studio 檔則位於位於的存放庫中。 [https://github.com/Microsoft/visualstudio-docs](https://github.com/Microsoft/visualstudio-docs)

## <a name="contributing-to-the-documentation"></a>參與文件

我們歡迎您的貢獻，協助我們改進 MSVC 檔。如需參與 docs.microsoft.com 的完整指南，請參閱[Microsoft Docs 參與者指南總覽](https://docs.microsoft.com/contribute)。 如需如何參與 MSVC 文件的詳細資料，請參閱[指南](CONTRIBUTING.md)。

MSVC 之數個功能區域在此存放庫中都有自己的資料夾，例如 `standard-library` 有 C++ 標準程式庫的文章，`ide` 有 Visual Studio 互動式開發環境 (IDE) 的 C++ 特定文章等。 每個資料夾之 `/media` 子資料夾都包含文章的美工檔案。 [參與指南](CONTRIBUTING.md)有詳細資訊。
