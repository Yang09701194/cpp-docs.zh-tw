# <a name="even-and-align-directives"></a>EVEN 和 ALIGN 指示詞
## <a name="microsoft-specific"></a>Microsoft 特定的  
 雖然內嵌組合語言不支援大多數的 MASM 指示詞，不過它支援`EVEN`和**對齊**。 這些指示詞放**NOP** （不執行作業） 的組件程式碼，視需要為特定界限的對齊標籤中的指示。 這可讓某些處理器的指令擷取作業更有效率。  
  
 **結束 Microsoft 特定的**
