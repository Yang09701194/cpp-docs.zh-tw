## Microsoft 特定的  
 Visual C\+\+ 編譯器可藉由 Pentium 4 和 AMD Athlon 支援所有 opcode。  目標處理器支援的其他指令可以透過 [\_emit 虛擬指令](../../assembler/inline/emit-pseudoinstruction.md)建立。  
  
 **END Microsoft 特定的**
