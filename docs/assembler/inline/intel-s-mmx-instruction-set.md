# <a name="intel39s-mmx-instruction-set"></a>Intel&#39;s MMX 指令集

**Microsoft 專屬**

Microsoft C++編譯器可讓您在內嵌組譯工具中使用 INTEL 的 MMX （多媒體擴充功能）指令集。 偵錯工具的反組譯碼也支援 MMX 指令。 如果函式包含 MMX 指令，但不包含會清空多媒體狀態的 EMMS 指令，則編譯器會產生警告訊息。 如需詳細資訊，請參閱 Intel 網站。

**END Microsoft 特定的**
