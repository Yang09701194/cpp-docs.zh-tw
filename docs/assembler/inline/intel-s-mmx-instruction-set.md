# Intel 的 MMX 指令集
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

## Microsoft 特定的  
 Visual C\+\+ 編譯器可讓您在內嵌組合語言中使用 Intel 的 MMX \(多媒體延伸\) 指令集。  偵錯工具的反組譯碼也支援 MMX 指令。  如果函式包含 MMX 指令，但不包含會清空多媒體狀態的 EMMS 指令，則編譯器會產生警告訊息。  如需詳細資訊，請參閱 Intel 網站。  
  
 **END Microsoft 特定的**
