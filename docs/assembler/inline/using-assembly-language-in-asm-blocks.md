# 在 __asm 區塊中使用組合語言
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

## Microsoft 特定的  
 內嵌組譯碼與其他組合語言有許多相同之處。  例如，它可接受 MASM 中所有合法的運算式。  本節將描述 `__asm` 區塊中組合語言功能的用法。
