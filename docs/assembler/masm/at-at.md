# <a name=""></a>\@\@：

定義程式碼的標籤之間才可辨識*label1*並*label2*，其中*label1*程式碼的開頭或先前`@@:`標籤以及*label2*可能是程式碼結尾或下一個`@@:`標籤。 請參閱[ \@B](../../assembler/masm/at-b.md)並[ \@F](../../assembler/masm/at-f.md)。
