# <a name=""></a>\@\@：

定義只能在*label1*和*label2*之間辨識的程式碼標籤，其中*label1*是程式碼的開頭或前一個 `@@:` 標籤，而*label2*則是程式碼的結尾或下一個 `@@:` 的標籤。 請參閱[\@B](../../assembler/masm/at-b.md)和[\@F](../../assembler/masm/at-f.md)。

## <a name="syntax"></a>語法

> **\@\@：**
