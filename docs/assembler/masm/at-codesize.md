# <a name="codesize"></a>\@CodeSize

0 for **TINY**, **SMALL**, **COMPACT**, and **FLAT** models, and 1 for **MEDIUM**, **LARGE**, and **HUGE** models (numeric equate).

## <a name="syntax"></a>語法

> **\@CodeSize**
