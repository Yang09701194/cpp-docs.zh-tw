# <a name="codesize"></a>@CodeSize
0 代表**微小**，**小**， **COMPACT**，和**一般**模型，以及 1**媒體**， **大型**，和**超大型**（等同的數值） 的模型。  
  
## <a name="syntax"></a>語法  
  
```  
  
@CodeSize  
  
```
