# @CodeSize
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

0 表示**微小**， **小**， **壓縮**，和 **平面** 模型，1 表示 **中**， **大**，和 **超大型** \(等同於數字\) 的模型。  
  
## 語法  
  
```  
  
@CodeSize  
  
```
