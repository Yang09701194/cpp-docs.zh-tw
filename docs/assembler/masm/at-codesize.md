# <a name="codesize"></a>@CodeSize

0 代表**微小**，**小型**， **COMPACT**，以及**一般**模型，以及 1**媒體**， **大型**，並**超大型**（等同於數字） 的模型。

## <a name="syntax"></a>語法

> @CodeSize
