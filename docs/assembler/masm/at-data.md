# <a name="data"></a>\@資料

預設資料群組的名稱。 針對**一般以外的所有模型，評估**為 DGROUP。 **在一般記憶體模型**（文字宏）底下評估為**平面**。

## <a name="syntax"></a>語法

> **\@資料**
