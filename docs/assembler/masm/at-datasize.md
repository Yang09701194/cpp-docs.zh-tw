# @DataSize
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

0 表示**微小**， **小**， **中**，和 **平面** 模型，1 代表 **壓縮** 和 **大** 模型，而且 2 則表示 **超大型**模型 \(等同於數字\)。  
  
## 語法  
  
```  
  
@DataSize  
  
```
