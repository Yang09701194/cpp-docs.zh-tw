# <a name="datasize"></a>\@DataSize

0代表**小型**、**小型**、**中型**和**平面**模型，1代表**COMPACT**和**大型**模型，2用於**大型**模型（數值等於）。

## <a name="syntax"></a>語法

> **\@DataSize**
