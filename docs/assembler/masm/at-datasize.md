# <a name="datasize"></a>@DataSize
0 代表**微小**，**小**，**媒體**，和**一般**模型，1 代表**COMPACT**和**大型**模型，以及 2**超大型**模型 （等同的數字）。  
  
## <a name="syntax"></a>語法  
  
```  
  
@DataSize  
  
```
