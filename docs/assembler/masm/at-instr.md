# <a name="instr"></a>@InStr

尋找第一個出現的巨集函式*string2*中*string1*，開始位置在*位置*內*string1*。 如果*位置*未出現，在開頭開始搜尋*string1*。 傳回的位置的整數或 0，如果*string2*找不到。

## <a name="syntax"></a>語法

> @InStr（[[位置]]，string1，string2）
