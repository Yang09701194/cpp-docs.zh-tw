# <a name="instr"></a>\@InStr

從*string1*中的*位置*開始，在*string1*中尋找第一個出現的*string2*的宏函式。 如果*位置*沒有出現，搜尋就會從*string1*開頭開始。 傳回位置整數，如果找不到*string2* ，則傳回0。

## <a name="syntax"></a>語法

> **\@InStr （** ⟦*position*⟧ __，__ *string1* __，__ *string2* **）**
