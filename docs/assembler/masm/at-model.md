# <a name="model"></a>@Model

1 代表**微小**模型、 2**小型**模型、 3 **COMPACT**模型，就會有 4 個**媒體**模型，為 5**大型**模型，6**超大型**模型，以及 7**一般**模型 （等同於數字）。

## <a name="syntax"></a>語法
