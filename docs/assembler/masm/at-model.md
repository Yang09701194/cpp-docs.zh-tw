# @Model
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

1 代表**微小** 模型，2 是 **小** 模型，3 是 **壓縮** 模型，4 是 **媒體** 模型，5 是 **大** 模型，6 是 **超大型** 模型和 7 **平面**模型 \(等同於數字\)。  
  
## 語法  
  
```  
  
@Model  
  
```
