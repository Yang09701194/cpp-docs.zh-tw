# <a name="version"></a>\@版本

在命令列中，以單一數位（文字宏）報告之 MASM 的主要和次要版本。 例如，MASM 版本14.23.28107.0 會產生 "1423"。

## <a name="syntax"></a>語法

> **\@Version**
