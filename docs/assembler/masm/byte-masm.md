# <a name="byte"></a>BYTE

配置並選擇性地為每個初始化*運算式*初始化一個位元組的儲存區。

## <a name="syntax"></a>語法

> ⟦*名稱*⟧ **BYTE** *初始化運算式*⟦ __，__ *初始化運算式*.。。⟧

## <a name="remarks"></a>備註

也可以當做類型為合法的任何位置的類型規範使用。
