# <a name="byte-masm"></a>BYTE (MASM)

配置並選擇性地針對每個初始化一個位元組的儲存體`initializer`。

## <a name="syntax"></a>語法

> [[name]]BYTEinitializer [[，初始設定式]]...

## <a name="remarks"></a>備註

也可用來當做型別規範的類型是合法的任何位置。
