# <a name="byte-masm"></a>BYTE (MASM)
配置並選擇性地為每個初始化一個位元組的儲存體`initializer`。  
  
## <a name="syntax"></a>語法  
  
```  
[[name]] BYTEinitializer [[, initializer]] ...  
```  
  
## <a name="remarks"></a>備註  
 也可用以做為類型指定型別是合法的任何位置。
