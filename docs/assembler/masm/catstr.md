# <a name="catstr"></a>CATSTR
串連文字項目。  
  
## <a name="syntax"></a>語法  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## <a name="remarks"></a>備註  
 每個文字項目可以是常值的字串常數前面加上`%`，或巨集函式所傳回的字串。
