# <a name="catstr"></a>CATSTR

串連文字專案。

## <a name="syntax"></a>語法

> *name* **CATSTR** ⟦*textitem1* ⟦ __，__ *textitem2* .。。⟧⟧

## <a name="remarks"></a>備註

每個文字專案可以是常值字串、前面加上 **%** 的常數，或宏函數所傳回的字串。 **CATSTR**是[TEXTEQU](textequ.md)的同義字。
