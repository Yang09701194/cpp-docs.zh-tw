# CATSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

串連文字項目。  
  
## 語法  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## 備註  
 每個文字項目可以是常值字串常數，前面加上`%`，或巨集函式所傳回的字串。
