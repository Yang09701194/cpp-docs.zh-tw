# <a name="comment-masm"></a>COMMENT (MASM)
會將所有視為*文字*之間，或為註解分隔符號的同一行上。  
  
## <a name="syntax"></a>語法  
  
```  
  
      COMMENT delimiter [[text]]   
[[text]]   
[[text]] delimiter [[text]]  
```
