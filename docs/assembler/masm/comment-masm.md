# <a name="comment"></a>COMMENT

將或兩者之間的所有*文字*，視為批註的分隔符號。

## <a name="syntax"></a>語法

> **批註***分隔符號*⟦*文字*⟧ \
> ⟦*文字*⟧ \
> ⟦*文字*⟧*分隔符號*⟦*文字*⟧
