# <a name="comment-masm"></a>COMMENT (MASM)

Treats all *text* between or on the same line as the delimiters as a comment.

## <a name="syntax"></a>語法

> **COMMENT** *delimiter* ⟦*text*⟧\
> ⟦*text*⟧\
> ⟦*text*⟧ *delimiter* ⟦*text*⟧
