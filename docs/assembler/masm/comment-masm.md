# <a name="comment-masm"></a>COMMENT (MASM)

會將所有*文字*之間，或為註解分隔符號的同一行上。

## <a name="syntax"></a>語法

> 註解分隔符號 [[文字]]<br/>
> [[text]]<br/>
> [[文字]] 分隔符號 [[文字]]
