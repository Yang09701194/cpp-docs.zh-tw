# <a name="db"></a>DB

配置並選擇性地為每個初始化*運算式*初始化一個位元組的儲存區。 **DB**是[BYTE](byte-masm.md)的同義字。

## <a name="syntax"></a>語法

> ⟦*name*⟧ **DB** *初始化運算式*⟦ __，__ *初始化運算式*.。。⟧

## <a name="remarks"></a>備註

也可以當做類型為合法的任何位置的類型規範使用。
