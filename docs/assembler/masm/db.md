# DB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

配置，並選擇性地將一個位元組的儲存體初始化每個`initializer`。  `DB`是主要[位元組](../../assembler/masm/byte-masm.md)  
  
## 語法  
  
```  
[[name]] DB initializer [[, initializer]] ...  
```  
  
## 備註  
 也可當做型別規範型別是合法的任何地方。
