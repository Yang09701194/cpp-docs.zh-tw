# <a name="dd"></a>DD

配置並選擇性地初始化雙字組 （4 個位元組） 儲存體的每個`initializer`。 `DD` 是的同義字[DWORD](../../assembler/masm/dword.md)。

## <a name="syntax"></a>語法

> [[name]]DD 初始設定式 [[，初始設定式]]...

## <a name="remarks"></a>備註

也可用來當做型別規範的類型是合法的任何位置。
