# DD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

配置，並選擇性地儲存區的初始化雙字 \(4 個位元組\)，每個`initializer`。  `DD`是主要 [DWORD](../../assembler/masm/dword.md)。  
  
## 語法  
  
```  
[[name]] DD initializer [[, initializer]]...  
```  
  
## 備註  
 也可當做型別規範型別是合法的任何地方。
