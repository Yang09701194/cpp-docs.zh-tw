# <a name="dd"></a>DD

配置並選擇性地為每個初始化*運算式*初始化雙字（4個位元組）的儲存區。 **DD**是[DWORD](../../assembler/masm/dword.md)的同義字。

## <a name="syntax"></a>語法

> ⟦*名稱*⟧ **DD** *初始化運算式*⟦ __，__ *初始化運算式*.。。⟧

## <a name="remarks"></a>備註

也可以當做類型為合法的任何位置的類型規範使用。
