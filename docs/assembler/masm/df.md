# <a name="df"></a>DF
配置並選擇性地為每個初始化 6 個位元組的儲存體`initializer`。 `DF` 是的同義字[FWORD](../../assembler/masm/fword.md)。  
  
## <a name="syntax"></a>語法  
  
```  
[[name]] DF initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>備註  
 也可以用為類型規範的類型是合法的任何位置。
