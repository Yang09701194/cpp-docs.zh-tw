# DF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

配置，並選擇性地初始化 6 個位元組的儲存空間，每個`initializer`。  `DF`是主要 [FWORD](../../assembler/masm/fword.md)。  
  
## 語法  
  
```  
[[name]] DF initializer [[, initializer]]...  
```  
  
## 備註  
 也可以當做型別規範型別是合法的任何地方。
