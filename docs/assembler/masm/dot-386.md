# .386
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

可讓 80386 處理器 ； 無指令的組件 停用指令有較新的處理器導入的組的件。  
  
## 語法  
  
```  
  
.386  
  
```  
  
## 備註  
 也可以讓 80387 的指示進行。
