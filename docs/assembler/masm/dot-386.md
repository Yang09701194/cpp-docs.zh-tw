# <a name="386"></a>.386

可讓組件的特殊指示 80386 處理器;停用組件所引進的更新版本的處理器的指示。

## <a name="syntax"></a>語法

> .386

## <a name="remarks"></a>備註

也可讓 80387 的指示。
