# <a name="386-32-bit-masm"></a>386（32位 MASM）

啟用80386處理器的未具許可權指令元件;停用後續處理器引進的指令程式集。 （僅限 32-bit MASM）。

## <a name="syntax"></a>語法

> .386

## <a name="remarks"></a>備註

也會啟用80387指示。
