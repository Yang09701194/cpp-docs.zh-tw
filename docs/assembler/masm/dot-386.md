# <a name="386"></a>.386
可讓組件的未具權限的指示 80386 處理器;停用組件的指示所導入的更新版本的處理器。  
  
## <a name="syntax"></a>語法  
  
```  
  
.386  
  
```  
  
## <a name="remarks"></a>備註  
 也可讓 80387 指示。
