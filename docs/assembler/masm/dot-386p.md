# <a name="386p-32-bit-masm"></a>. .386P （32-bit MASM）

啟用80386處理器的所有指令（包括特殊許可權）的元件;停用後續處理器引進的指令程式集。 （僅限 32-bit MASM）。

## <a name="syntax"></a>語法

> **.386P**

## <a name="remarks"></a>備註

也會啟用80387指示。
