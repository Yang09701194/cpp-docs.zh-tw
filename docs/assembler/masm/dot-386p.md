# <a name="386p"></a>.386P
可讓組件的所有指示 （包括特殊權限） 80386 處理器;停用組件的指示所導入的更新版本的處理器。  
  
## <a name="syntax"></a>語法  
  
```  
  
.386P  
  
```  
  
## <a name="remarks"></a>備註  
 也可讓 80387 指示。
