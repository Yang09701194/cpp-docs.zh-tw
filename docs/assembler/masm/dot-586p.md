# <a name="586p-32-bit-masm"></a>. .586P （32-bit MASM）

啟用 Pentium 處理器的所有指令（包括特殊許可權）的元件。 （僅限 32-bit MASM）。

## <a name="syntax"></a>語法

> **.586P**
