# <a name="686p-32-bit-masm"></a>. .686P （32-bit MASM）

針對 Pentium Pro 處理器啟用所有指令的元件（包括特殊許可權）。 （僅限 32-bit MASM）。

## <a name="syntax"></a>語法

> **.686P**
