# <a name="break-32-bit-masm"></a>.BREAK （32位 MASM）

產生程式碼以終止[。WHILE](dot-while.md)或[。](dot-repeat.md)If*條件*為 TRUE 時，重複 block。 （僅限 32-bit MASM）。

## <a name="syntax"></a>語法

> **.中斷**⟦ **。IF** *條件*⟧
