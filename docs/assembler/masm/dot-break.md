# .BREAK
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

產生程式碼來終止 [。WHILE](../../assembler/masm/dot-while.md) or [.重複](../../assembler/masm/dot-repeat.md) 封鎖如果 *條件*為 true。  
  
## 語法  
  
```  
  
.BREAK [[.IF condition]]   
```
