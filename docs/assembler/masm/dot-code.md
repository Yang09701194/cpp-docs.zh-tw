# <a name="code"></a>.CODE

當搭配[。模型](../../assembler/masm/dot-model.md)，表示程式碼片段的開頭。

## <a name="syntax"></a>語法

> .CODE [[name]]

#### <a name="parameters"></a>參數

|參數|描述|
|---------------|-----------------|
|`name`|指定的程式碼片段名稱的選擇性參數。 預設名稱是很小，很小，壓縮和一般 _TEXT[模型](../../assembler/masm/dot-model.md)。 預設名稱是*modulename*_TEXT 其他模型。|
