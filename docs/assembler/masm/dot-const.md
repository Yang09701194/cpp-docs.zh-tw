# <a name="const"></a>.CONST

當搭配[。模型](../../assembler/masm/dot-model.md)，啟動 常數的資料區段 （區段的名稱，CONST）。

## <a name="syntax"></a>語法

> .CONST

## <a name="remarks"></a>備註

此區段會具有唯讀屬性。
