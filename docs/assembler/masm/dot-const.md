# <a name="const-32-bit-masm"></a>.CONST （32位 MASM）

與搭配使用時[。MODEL](../../assembler/masm/dot-model.md)，啟動常數資料區段（使用區段名稱**CONST**）。

## <a name="syntax"></a>語法

> **.CONST**

## <a name="remarks"></a>備註

此區段具有唯讀屬性。
