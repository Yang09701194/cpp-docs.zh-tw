# .CONST
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

搭配使用時 [。模型](../../assembler/masm/dot-model.md)，會啟動一個常數的資料區段 \(區段以命名常數\)。  
  
## 語法  
  
```  
  
.CONST  
  
```  
  
## 備註  
 此區段會有唯讀屬性。
