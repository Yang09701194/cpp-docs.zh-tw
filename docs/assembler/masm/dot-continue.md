# <a name="continue"></a>.CONTINUE
產生程式碼以跳至頂端[。雖然](../../assembler/masm/dot-while.md)或[。重複](../../assembler/masm/dot-repeat.md)封鎖`condition`為 true。  
  
## <a name="syntax"></a>語法  
  
```  
  
.CONTINUE [[.IF condition]]  
```
