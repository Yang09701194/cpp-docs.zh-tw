# <a name="continue-32-bit-masm"></a>.繼續（32位 MASM）

產生程式碼以跳至的頂端[。WHILE](dot-while.md)或[。](dot-repeat.md)If*條件*為 TRUE 時，重複 block。 （僅限 32-bit MASM）。

## <a name="syntax"></a>語法

> **.繼續**⟦ **。IF** *條件*⟧
