# .DATA?
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

搭配使用時 [。模型](../../assembler/masm/dot-model.md)，會開始接近的資料區段，為未初始化的資料 \(區段名稱 \_BSS\)。  
  
## 語法  
  
```  
  
.DATA?  
  
```
