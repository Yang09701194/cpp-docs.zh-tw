# <a name="endprolog"></a>.ENDPROLOG

發出序言宣告結尾的信號。

## <a name="syntax"></a>語法

> **.ENDPROLOG**

## <a name="remarks"></a>備註

在[進程](../../assembler/masm/proc.md)**框架**和之間使用區域以外的任何序言宣告是錯誤的 **。ENDPROLOG**。

如需詳細資訊，請參閱[MASM for x64 （ml64 .exe）](../../assembler/masm/masm-for-x64-ml64-exe.md)。
