# .ENDPROLOG
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

表示初構宣告的結束。  
  
## 語法  
  
```  
.ENDPROLOG  
```  
  
## 備註  
 就會發生錯誤，若要使用的初構宣告之外的區域之間[程序](../../assembler/masm/proc.md)框架和。ENDPROLOG。  
  
 如需詳細資訊，請參閱 [MASM for x64 \(ml64.exe\)](../../assembler/masm/masm-for-x64-ml64-exe.md)。
