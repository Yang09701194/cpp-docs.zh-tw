# <a name="errdif-errdifi"></a>.ERRDIF、.ERRDIFI

如果文字專案不同，則會產生錯誤。

## <a name="syntax"></a>語法

> **..Errdif** *textitem1* __，__ *textitem2* ⟦ __，__ *message*⟧ \
> **.ERRDIFI** *textitem1* __，__ *textitem2* ⟦ __，__ *message*⟧

## <a name="remarks"></a>備註

如果為 **。** 已指定 ERRDIFI，比較不區分大小寫。
