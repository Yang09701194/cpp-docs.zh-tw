# <a name="errdifi"></a>.ERRDIF[[I]]
如果文字項目不同，會產生錯誤。  
  
## <a name="syntax"></a>語法  
  
```  
  
.ERRDIF[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## <a name="remarks"></a>備註  
 如果**我**是假設，比較會區分大小寫。
