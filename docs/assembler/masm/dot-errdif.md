# <a name="perioderrdif-perioderrdifi"></a>&period;ERRDIF， &period;ERRDIFI

如果文字項目不同，會產生錯誤。

## <a name="syntax"></a>語法

> **.ERRDIF**\[**I**] *textitem1* __,__ *textitem2* \[ __,__ *message*]

## <a name="remarks"></a>備註

如果 **&period;ERRDIFI**是，比較不區分大小寫。
