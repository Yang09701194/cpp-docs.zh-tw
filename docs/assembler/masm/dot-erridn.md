# <a name="erridn"></a>.ERRIDN]

如果文字項目相同，則會產生錯誤。

## <a name="syntax"></a>語法

> .[[I]] ERRIDN *textitem1*， *textitem2* [[，*訊息*]]

## <a name="remarks"></a>備註

如果**我**是，比較不區分大小寫。
