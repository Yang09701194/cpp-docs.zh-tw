# <a name="exit-32-bit-masm"></a>.EXIT （32-位 MASM）

產生終止程式碼。 （僅限 32-bit MASM）。

## <a name="syntax"></a>語法

> **.EXIT** ⟦*expression*⟧

## <a name="remarks"></a>備註

傳回 shell 的選擇性*運算式*。
