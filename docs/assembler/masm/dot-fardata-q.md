# .FARDATA?
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

搭配使用時 [。模型](../../assembler/masm/dot-model.md)，開始為未初始化的資料遠程的資料區段 \(區段名稱 FAR\_BSS 或 *名稱*\)。  
  
## 語法  
  
```  
  
.FARDATA? [[name]]  
```
