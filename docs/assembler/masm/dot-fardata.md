# <a name="fardata"></a>.FARDATA

當搭配[。模型](../../assembler/masm/dot-model.md)，開始為初始化的資料遠的資料區段 (區段名稱 FAR_DATA 或*名稱*)。

## <a name="syntax"></a>語法

> .FARDATA [[name]]
