# .LISTALL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

正在列出所有的陳述式，就會啟動。  
  
## 語法  
  
```  
  
.LISTALL  
  
```  
  
## 備註  
 對等的組合 [。LIST](../../assembler/masm/dot-list.md), [.LISTIF](../../assembler/masm/dot-listif.md)，以及  [。LISTMACROALL](../../assembler/masm/dot-listmacroall.md)。
