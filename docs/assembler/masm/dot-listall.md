# <a name="listall"></a>.LISTALL
啟動所有的陳述式的清單。  
  
## <a name="syntax"></a>語法  
  
```  
  
.LISTALL  
  
```  
  
## <a name="remarks"></a>備註  
 對等的組合[。清單](../../assembler/masm/dot-list.md)， [。LISTIF](../../assembler/masm/dot-listif.md)，和[。LISTMACROALL](../../assembler/masm/dot-listmacroall.md)。
