# <a name="listif"></a>.LISTIF

會啟動，則為 false 的條件式區塊中的陳述式的清單。

## <a name="syntax"></a>語法

> .LISTIF

## <a name="remarks"></a>備註

與相同[。LFCOND](../../assembler/masm/dot-lfcond.md)。
