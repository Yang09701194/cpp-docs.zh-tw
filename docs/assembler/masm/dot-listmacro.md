# <a name="listmacro"></a>.LISTMACRO

Starts listing of macro expansion statements that generate code or data.

## <a name="syntax"></a>語法

> **.LISTMACRO**

## <a name="remarks"></a>備註

這是預設值。 Same as [.XALL](../../assembler/masm/dot-xall.md).
