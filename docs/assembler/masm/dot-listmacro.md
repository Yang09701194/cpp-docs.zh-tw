# <a name="listmacro"></a>.LISTMACRO

啟動 產生程式碼或資料的巨集展開陳述式的清單。

## <a name="syntax"></a>語法

> .LISTMACRO

## <a name="remarks"></a>備註

這是預設值。 與相同[。XALL](../../assembler/masm/dot-xall.md)。
