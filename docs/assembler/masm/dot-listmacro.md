# .LISTMACRO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

列出的巨集展開陳述式產生程式碼或資料，就會啟動。  
  
## 語法  
  
```  
  
.LISTMACRO  
  
```  
  
## 備註  
 這是預設值。  相同的 [。XALL](../../assembler/masm/dot-xall.md)。
