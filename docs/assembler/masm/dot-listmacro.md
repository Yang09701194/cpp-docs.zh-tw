# <a name="listmacro"></a>.LISTMACRO

開始列出產生程式碼或資料的宏展開語句。

## <a name="syntax"></a>語法

> **.LISTMACRO**

## <a name="remarks"></a>備註

這是預設設定。 與相同[.XALL](dot-xall.md)。
