# <a name="nocref"></a>.NOCREF

隱藏符號表和瀏覽器檔案中的符號清單。

## <a name="syntax"></a>語法

> **.Nocref** ⟦*name*⟦ __，__ *name* 。⟧⟧

## <a name="remarks"></a>備註

如果指定了名稱，則只會隱藏指定的名稱。 與相同[.XCREF](../../assembler/masm/dot-xcref.md)。
