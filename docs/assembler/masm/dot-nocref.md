# .NOCREF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

隱藏的符號，符號的資料表和瀏覽器檔案中的清單。  
  
## 語法  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## 備註  
 如果指定名稱，然後將被抑制只有指定的名稱。  相同的 [。XCREF](../../assembler/masm/dot-xcref.md)。
