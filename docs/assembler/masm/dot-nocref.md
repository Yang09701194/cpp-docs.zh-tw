# <a name="nocref"></a>.NOCREF

隱藏符號表和瀏覽器檔中的符號的清單。

## <a name="syntax"></a>語法

> .NOCREF [[名稱 [[，name]]...]]

## <a name="remarks"></a>備註

如果指定的名稱，會隱藏只包含指定的名稱。 與相同[。XCREF](../../assembler/masm/dot-xcref.md)。
