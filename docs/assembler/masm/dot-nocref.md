# <a name="nocref"></a>.NOCREF
隱藏符號資料表和瀏覽器檔案中的符號清單。  
  
## <a name="syntax"></a>語法  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## <a name="remarks"></a>備註  
 如果指定的名稱，然後指定名稱會隱藏。 與相同[。XCREF](../../assembler/masm/dot-xcref.md)。
