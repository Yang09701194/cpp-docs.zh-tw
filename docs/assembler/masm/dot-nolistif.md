# <a name="nolistif"></a>.NOLISTIF

抑制條件評估為 false （0）的條件式區塊清單。

## <a name="syntax"></a>語法

> **.NOLISTIF**

## <a name="remarks"></a>備註

這是預設設定。 與相同[.SFCOND](dot-sfcond.md)。
