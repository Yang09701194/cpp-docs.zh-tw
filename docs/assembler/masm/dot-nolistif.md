# <a name="nolistif"></a>.NOLISTIF
隱藏其條件評估為 false (0) 的條件式區塊的清單。  
  
## <a name="syntax"></a>語法  
  
```  
  
.NOLISTIF  
  
```  
  
## <a name="remarks"></a>備註  
 這是預設值。 與相同[。SFCOND](../../assembler/masm/dot-sfcond.md)。
