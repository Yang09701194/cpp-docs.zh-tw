# .NOLISTIF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

隱藏清單的條件評估為 false \(0\) 的條件式區塊。  
  
## 語法  
  
```  
  
.NOLISTIF  
  
```  
  
## 備註  
 這是預設值。  相同的 [。SFCOND](../../assembler/masm/dot-sfcond.md)。
