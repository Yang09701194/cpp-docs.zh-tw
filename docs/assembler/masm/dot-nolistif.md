# <a name="nolistif"></a>.NOLISTIF

Suppresses listing of conditional blocks whose condition evaluates to false (0).

## <a name="syntax"></a>語法

> **.NOLISTIF**

## <a name="remarks"></a>備註

這是預設值。 Same as [.SFCOND](../../assembler/masm/dot-sfcond.md).
