# <a name="repeat-32-bit-masm"></a>.重複（32位 MASM）

產生會重複執行*語句*區塊的程式碼，直到*條件*變成 true 為止。 [.UNTILCXZ](../../assembler/masm/dot-untilcxz.md)，如果 CX 為零，則可替代[。直到](../../assembler/masm/dot-until.md)。 *條件*是選擇性的 **.UNTILCXZ**。 （僅限 32-bit MASM）。

## <a name="syntax"></a>語法

> **.重複**\
> *語句*\
> **.UNTIL** *條件*
