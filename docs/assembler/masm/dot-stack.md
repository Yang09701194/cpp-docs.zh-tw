# <a name="stack"></a>.STACK

當搭配[。模型](../../assembler/masm/dot-model.md)，定義堆疊區段 （以區段名稱堆疊）。 選擇性`size`指定的堆疊 （預設值為 1,024） 的位元組數目。 `.STACK`指示詞會自動關閉堆疊陳述式。

## <a name="syntax"></a>語法

> .堆疊 [[大小]]
