# <a name="stack"></a>.STACK
當搭配[。模型](../../assembler/masm/dot-model.md)，定義堆疊區段 （使用堆疊的區段名稱）。 選擇性`size`指定堆疊 （預設值是 1,024） 的位元組數目。 `.STACK`指示詞就會自動關閉堆疊陳述式。  
  
## <a name="syntax"></a>語法  
  
```  
  
.STACK [[size]]  
```
