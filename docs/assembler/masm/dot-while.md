# <a name="while-32-bit-masm"></a>.WHILE （32-bit MASM）

產生執行*語句*區塊的程式碼，而*條件*則維持為 true。 （僅限 32-bit MASM）。

## <a name="syntax"></a>語法

> **.WHILE** *條件*\
> *語句*\
> **.ENDW**
