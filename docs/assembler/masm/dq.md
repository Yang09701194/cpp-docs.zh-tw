# <a name="dq"></a>DQ

配置並選擇性地為每個*初始化運算式*初始化8個位元組的儲存體。 也可以當做類型為合法的任何位置的類型規範使用。 **DQ**是[QWORD](../../assembler/masm/qword.md)的同義字。

## <a name="syntax"></a>語法

> ⟦*名稱*⟧ **DQ** *初始化運算式*⟦ __，__ *初始化運算式*.。。⟧
