# <a name="dq"></a>DQ
配置並選擇性地為每個初始化 8 個位元組的儲存體`initializer`。 也可以用為類型規範的類型是合法的任何位置。 `DQ`是的同義字[QWORD](../../assembler/masm/qword.md)。  
  
## <a name="syntax"></a>語法  
  
```  
[[name]] DQ initializer [[, initializer]]...  
```
