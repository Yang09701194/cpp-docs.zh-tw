# <a name="dt"></a>DT

配置並選擇性地針對每個初始化 10 個位元組的儲存體`initializer`。 也可用來當做型別規範的類型是合法的任何位置。 `DT` 是的同義字[TBYTE](../../assembler/masm/tbyte.md)。

## <a name="syntax"></a>語法

> [[name]]DT 初始設定式 [[，初始設定式]]...
