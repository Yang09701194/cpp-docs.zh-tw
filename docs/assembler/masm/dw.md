# <a name="dw"></a>DW
配置並選擇性地初始化儲存體的單字 （2 個位元組） 為每個`initializer`。 也可用以做為類型指定型別是合法的任何位置。 `DW` 是的同義字[WORD](../../assembler/masm/word.md)。  
  
## <a name="syntax"></a>語法  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
