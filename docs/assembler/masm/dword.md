# <a name="dword"></a>DWORD

配置並選擇性地初始化雙字組 （4 個位元組） 儲存體的每個`initializer`。

## <a name="syntax"></a>語法

> [[name]]DWORD 初始設定式 [[，初始設定式]]...

## <a name="remarks"></a>備註

也可用來當做型別規範的類型是合法的任何位置。
