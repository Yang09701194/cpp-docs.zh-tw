# <a name="dword"></a>DWORD
配置及選擇性的儲存體初始化雙字組 （4 個位元組），每個`initializer`。  
  
## <a name="syntax"></a>語法  
  
```  
[[name]] DWORD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>備註  
 也可用以做為類型指定型別是合法的任何位置。
