# <a name="echo"></a>ECHO

顯示*訊息*到標準輸出裝置 （依預設，螢幕）。

## <a name="syntax"></a>語法

> ECHO*訊息*

## <a name="remarks"></a>備註

與相同[%out](../../assembler/masm/percent-out.md)。
