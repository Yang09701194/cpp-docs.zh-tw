# <a name="echo"></a>ECHO

顯示*訊息*至標準輸出裝置（根據預設，畫面）。

## <a name="syntax"></a>語法

> **回應***消息*

## <a name="remarks"></a>備註

與[% OUT](percent-out.md)相同。
