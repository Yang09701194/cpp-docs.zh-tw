# <a name="elseif"></a>ELSEIF

將[ELSE](else-masm.md)和[IF](if-masm.md)結合成一個語句。

## <a name="syntax"></a>語法

> **ELSEIF** *constantExpression*\
> *語句*\
> ⟦**ELSE**\
> *else 語句*⟧ \
> **ENDIF**
