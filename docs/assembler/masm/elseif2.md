# <a name="elseif2"></a>ELSEIF2

如果**選項： SETIF2**為**TRUE**，則會在每個元件階段評估[ELSEIF](elseif-masm.md)區塊。

## <a name="syntax"></a>語法

> **ELSEIF2**

## <a name="remarks"></a>備註

如需 MASM 5.1 與 MASM 6.1 之2階段行為的詳細資訊，請參閱[IF1 和 IF2](if2.md)。
