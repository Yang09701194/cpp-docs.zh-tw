# <a name="end-masm"></a>END (MASM)

標示模組的結尾，並選擇性地將程式進入點設定為*address*。

## <a name="syntax"></a>語法

> **結束**⟦*位址*⟧

## <a name="remarks"></a>備註

*Address*引數只在32位 MASM 中有效。
