# <a name="end"></a>END

標記模組的結尾，並選擇性地將程式進入點設定為*procId*。

## <a name="syntax"></a>語法

> **結束**⟦*procId*⟧

## <a name="remarks"></a>備註

*ProcId*引數只在32位 MASM 中有效。
