# <a name="endm"></a>ENDM

結束巨集或重複的區塊。

## <a name="syntax"></a>語法

> ENDM

## <a name="remarks"></a>備註

請參閱[巨集](../../assembler/masm/macro.md)，[如](../../assembler/masm/for-masm.md)， [FORC](../../assembler/masm/forc.md)，[重複](../../assembler/masm/repeat.md)，或**雖然**。
