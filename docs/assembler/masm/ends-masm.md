# ENDS (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

標記結尾的區段、 結構或等位*名稱* 之前開始的 [區段](../../assembler/masm/segment.md)， [結構](../../assembler/masm/struct-masm.md)， [等位](../../assembler/masm/union.md)，或簡化的區段指示詞。  
  
## 語法  
  
```

```
