# <a name="equ"></a>EQU

第一個指示詞會將*expression*的數值指派給*name*。

## <a name="syntax"></a>語法

> *name* **等於***運算式*
>
> *名稱***等於** __\<__ *文字* __>__

## <a name="remarks"></a>備註

此*名稱*稍後無法重新定義。

第二個指示詞會指派指定的*文字*給*名稱*。 稍後可以為*名稱*指派不同的*文字*。 請參閱[TEXTEQU](../../assembler/masm/textequ.md)。
