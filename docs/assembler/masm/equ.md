# <a name="equ"></a>EQU

第一個指示詞會指派值的數值*運算式*要*名稱*。

## <a name="syntax"></a>語法

> *名稱*EQU*運算式*

> *名稱*EQU \<*文字*>

## <a name="remarks"></a>備註

*名稱*不能在稍後重新定義。

指定第二個指示詞指派*文字*要*名稱*。 *名稱*可以指派不同*文字*更新版本。 請參閱[TEXTEQU](../../assembler/masm/textequ.md)。
