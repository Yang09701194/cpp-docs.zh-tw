# <a name="exitm"></a>EXITM

終止目前重複或宏區塊的展開，並開始區塊外下一個語句的元件。

## <a name="syntax"></a>語法

> **EXITM** ⟦*textitem*⟧

## <a name="remarks"></a>備註

在宏函式中， *textitem*是傳回的值。
