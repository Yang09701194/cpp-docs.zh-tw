# EXITM
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

結束目前的重複或巨集區塊的擴充，開始下一個陳述式區塊外的組件。  
  
## 語法  
  
```  
  
EXITM [[textitem]]   
```  
  
## 備註  
 在 \[巨集函式中，  *textitem* 傳回的值。
