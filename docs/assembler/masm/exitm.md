# <a name="exitm"></a>EXITM
結束目前的重複或巨集區塊的擴充，並開始的下一個陳述式，在區塊內的組件。  
  
## <a name="syntax"></a>語法  
  
```  
  
EXITM [[textitem]]   
```  
  
## <a name="remarks"></a>備註  
 巨集函式中*textitem*傳回的值。
