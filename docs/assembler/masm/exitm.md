# <a name="exitm"></a>EXITM

終止目前的重複或巨集區塊的擴充，並開始下一個陳述式，在區塊外部組件。

## <a name="syntax"></a>語法

> EXITM [[textitem]]

## <a name="remarks"></a>備註

在巨集函式*textitem*傳回的值。
