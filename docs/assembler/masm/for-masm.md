# FOR (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

將標記的區塊，就會重複出現一次每個`argument`，與您目前`argument`取代`parameter`在每次重複。  
  
## 語法  
  
```  
  
   FOR   
   parameter [[:REQ | :=default]] , <argument [[, argument]]...>  
statements  
ENDM  
```
