# <a name="for-masm"></a>FOR (MASM)
標記將每一次重複的區塊`argument`，與目前`argument`取代`parameter`上每個重複。  
  
## <a name="syntax"></a>語法  
  
```  
  
   FOR   
   parameter [[:REQ | :=default]] , <argument [[, argument]]...>  
statements  
ENDM  
```  
  
## <a name="remarks"></a>備註  
 與相同[IRP](../../assembler/masm/irp.md)。
