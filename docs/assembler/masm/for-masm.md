# <a name="for"></a>FOR

標示每個*引數*會重複一次的區塊，並以目前的*引數*取代每個重複的*參數*。

## <a name="syntax"></a>語法

> **對於***參數*⟦ **：** 要求 |  __： =__ *預設*⟧ **，\<** _引數_⟦ __，__ *引數*.。。⟧ **>** \
> *語句*\
> **ENDM**

## <a name="remarks"></a>備註

與[IRP](irp.md)相同。
