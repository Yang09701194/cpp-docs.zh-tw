# <a name="forc"></a>FORC
將重複的每個字元的一次一個區塊會將標示`string`，與目前的字元取代`parameter`上每個重複。  
  
## <a name="syntax"></a>語法  
  
```  
  
   FORC  
parameter, <string> statements  
ENDM  
```  
  
## <a name="remarks"></a>備註  
 與相同[IRPC](../../assembler/masm/irpc.md)。
