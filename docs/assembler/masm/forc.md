# <a name="forc"></a>FORC

將重複的每個字元的一次一個區塊會將標示*字串*，使用目前的字元取代*參數*上每個重複作業。

## <a name="syntax"></a>語法

> FORC*參數*， \<*字串*><br/>
> *陳述式*<br/>
> ENDM

## <a name="remarks"></a>備註

與相同[IRPC](../../assembler/masm/irpc.md)。
