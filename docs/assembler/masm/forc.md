# <a name="forc"></a>FORC

針對*字串*中的每個字元，將重複一次的區塊標記為，並在每次重複時取代*參數*。

## <a name="syntax"></a>語法

> **FORC** *參數*， __\<__ *字串* __>__ \
> *語句*\
> **ENDM**

## <a name="remarks"></a>備註

與[IRPC](../../assembler/masm/irpc.md)相同。
