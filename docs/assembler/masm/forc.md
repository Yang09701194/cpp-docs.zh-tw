# FORC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

將標記的區塊，就會重複出現一次每個字元`string`，與目前的字元取代`parameter`在每次重複。  
  
## 語法  
  
```  
  
   FORC  
parameter, <string> statements  
ENDM  
```
