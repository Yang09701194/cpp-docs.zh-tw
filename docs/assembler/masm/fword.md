# <a name="fword"></a>FWORD

配置並選擇性地為每個*初始化運算式*初始化6個位元組的儲存區。

## <a name="syntax"></a>語法

> ⟦*名稱*⟧ **FWORD** *初始化運算式*⟦ __，__ *初始化運算式*.。。⟧

## <a name="remarks"></a>備註

也可以當做類型為合法的任何位置的類型規範使用。
