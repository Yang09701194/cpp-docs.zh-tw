# <a name="fword"></a>FWORD

配置並選擇性地針對每個初始化 6 個位元組的儲存體`initializer`。

## <a name="syntax"></a>語法

> [[name]]FWORD 初始設定式 [[，初始設定式]]...

## <a name="remarks"></a>備註

也可用來當做型別規範的類型是合法的任何位置。
