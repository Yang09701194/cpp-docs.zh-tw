# FWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

配置，並選擇性地初始化 6 個位元組的儲存空間，每個`initializer`。  
  
## 語法  
  
```  
[[name]] FWORD initializer [[, initializer]]...  
```  
  
## 備註  
 也可以當做型別規範型別是合法的任何地方。
