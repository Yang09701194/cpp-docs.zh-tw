# <a name="group-32-bit-masm"></a>GROUP （32位 MASM）

將指定的*區段*新增至名為*name*的群組。 （僅限 32-bit MASM）。

## <a name="syntax"></a>語法

> *名稱***群組***區段*⟦ __，__ *區段*.。。⟧
