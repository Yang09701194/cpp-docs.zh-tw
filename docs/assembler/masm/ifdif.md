# <a name="ifdif-ifdifi"></a>IFDIF、IFDIFI

如果文字專案不同，則授與元件。

## <a name="syntax"></a>語法

> **IFDIF** *textitem1* __、__ *textitem2*\
> **IFDIFI** *textitem1* __，__ *textitem2*

## <a name="remarks"></a>備註

如果指定**IFDIFI** ，則比較不區分大小寫。 如需完整語法[，請參閱](../../assembler/masm/if-masm.md)。
