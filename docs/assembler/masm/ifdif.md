# <a name="ifdif-ifdifi"></a>IFDIF IFDIFI

授與組件的文字項目是否不同。

## <a name="syntax"></a>語法

> **IFDIF**\[**I**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>備註

如果**IFDIFI**是，比較不區分大小寫。 請參閱[如果](../../assembler/masm/if-masm.md)如需完整語法。
