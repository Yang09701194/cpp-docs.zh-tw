# IFIDN]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

如果文字項目都一樣授與組件。  
  
## 語法  
  
```  
  
IFIDN[[I]] textitem1, textitem2  
```  
  
## 備註  
 如果**我**時，這個比較是不區分大小寫。  請參閱 [IF](../../assembler/masm/if-masm.md) 如需完整的語法。
