# <a name="ifidn-ifidni"></a>IFIDN IFIDNI

如果相同的文字項目授與組件。

## <a name="syntax"></a>語法

> **IFIDN**\[**I**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>備註

如果**IFIDNI**是，比較不區分大小寫。 請參閱[如果](../../assembler/masm/if-masm.md)如需完整語法。
