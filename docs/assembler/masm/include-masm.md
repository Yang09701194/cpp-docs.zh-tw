# <a name="include-masm"></a>INCLUDE (MASM)
插入來源所提供的來源檔案的程式碼*filename*到目前的原始程式檔，在組件。  
  
## <a name="syntax"></a>語法  
  
```  
  
INCLUDE filename  
```  
  
## <a name="remarks"></a>備註  
 *Filename*必須括在角括號中，如果它包含一個反斜線，以分號、 大-符號，小於-比符號、 單引號或雙引號。
