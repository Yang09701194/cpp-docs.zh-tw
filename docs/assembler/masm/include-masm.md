# INCLUDE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

插入來源所提供的原始程式檔中的程式碼*檔名*到目前的原始程式檔，在組件。  
  
## 語法  
  
```  
  
INCLUDE filename  
```  
  
## 備註  
 *檔名*必須用角括弧中如果包含反斜線，而分號，較大的符號，較少\-比符號、 單引號括住或雙引號。
