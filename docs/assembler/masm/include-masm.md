# <a name="include-masm"></a>INCLUDE (MASM)

從原始程式檔所指定的插入原始程式碼*檔名*至目前的原始程式檔，在組件。

## <a name="syntax"></a>語法

> INCLUDE 檔檔名

## <a name="remarks"></a>備註

*Filename*必須括在括弧中，如果它包含一個反斜線，分號，大於-符號，小於-比符號、 單引號或雙引號。
