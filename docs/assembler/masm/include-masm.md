# <a name="include"></a>INCLUDE

在元件期間，從*檔案名*指定的原始程式檔中，將原始程式碼插入目前的來源檔案。

## <a name="syntax"></a>語法

> **包含***檔案名*

## <a name="remarks"></a>備註

*檔案名*必須以角括弧括住，如果它包含反斜線、分號、大於符號、小於符號、單引號或雙引號。
