# <a name="includelib-masm"></a>INCLUDELIB (MASM)
通知連結器應該以連結目前模組*libraryname*。  
  
## <a name="syntax"></a>語法  
  
```  
  
INCLUDELIB libraryname  
```  
  
## <a name="remarks"></a>備註  
 *Libraryname*必須括在角括號中，如果它包含一個反斜線，以分號、 大-符號，小於-比符號、 單引號或雙引號。
