# <a name="includelib-masm"></a>INCLUDELIB (MASM)

通知目前的模組應該與連結，連結器*libraryname*。

## <a name="syntax"></a>語法

> INCLUDELIB libraryname

## <a name="remarks"></a>備註

*Libraryname*必須括在括弧中，如果它包含一個反斜線，分號，大於-符號，小於-不是符號、 單引號或雙引號。
