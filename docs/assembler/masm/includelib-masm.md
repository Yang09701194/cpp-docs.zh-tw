# <a name="includelib"></a>INCLUDELIB

通知連結器，目前模組應與*libraryname*連結。

## <a name="syntax"></a>語法

> **INCLUDELIB** *libraryname*

## <a name="remarks"></a>備註

如果*libraryname*包含反斜線、分號、大於符號、小於符號、單引號或雙引號，則必須以角括弧括住。
