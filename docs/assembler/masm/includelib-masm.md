# INCLUDELIB (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

告知目前的模組應該與連結，連結器*程式庫名稱*。  
  
## 語法  
  
```  
  
INCLUDELIB libraryname  
```  
  
## 備註  
 *程式庫名稱*必須用角括弧中如果包含反斜線，而分號，較大的符號，較少\-比符號、 單引號括住或雙引號。
