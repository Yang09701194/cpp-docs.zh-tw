# <a name="instr"></a>INSTR

尋找第一個出現*textitem2*中*textitem1*。

## <a name="syntax"></a>語法

> *名稱*INSTR [[*位置*、]] *textitem1*， *textitem2*

## <a name="remarks"></a>備註

起始*位置*是選擇性的。 每個文字項目可以是常值的字串常數前面加上`%`，或巨集函式所傳回的字串。
