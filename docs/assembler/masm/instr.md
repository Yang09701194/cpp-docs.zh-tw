# <a name="instr"></a>INSTR

在*textitem1*中尋找第一個出現的*textitem2* 。

## <a name="syntax"></a>語法

> *name* **INSTR** ⟦*position* __，__ ⟧ *textitem1* __，__ *textitem2*

## <a name="remarks"></a>備註

開始*位置*是選擇性的。 每個文字專案可以是常值字串、前面加上 **%** 的常數，或宏函數所傳回的字串。
