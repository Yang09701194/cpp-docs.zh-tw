# <a name="invoke"></a>INVOKE

呼叫程序，在所指定的地址*運算式*，在堆疊上，或根據語言類型之標準呼叫慣例的暫存器中傳遞的引數。

## <a name="syntax"></a>語法

> 叫用*運算式*[[，*引數*]]

## <a name="remarks"></a>備註

每個引數傳遞至程序可能是運算式、 一個暫存器配對或地址運算式 (運算式前面加上`ADDR`)。
