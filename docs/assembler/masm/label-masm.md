# <a name="label"></a>LABEL

將目前的位置計數器值和指定的*qualifiedType*指派給*name*，以建立新的標籤。

## <a name="syntax"></a>語法

> *名稱***標籤** *qualifiedType*
>
> *名稱***標籤**⟦**接近** | **遠** | **PROC**⟧ **PTR** ⟦*qualifiedType*⟧
