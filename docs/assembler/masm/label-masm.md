# <a name="label-masm"></a>LABEL (MASM)
藉由指定目前位置計數器值建立新的標籤和指定`type`至*名稱*。  
  
## <a name="syntax"></a>語法  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
