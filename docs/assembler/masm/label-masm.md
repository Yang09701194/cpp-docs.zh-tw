# <a name="label-masm"></a>LABEL (MASM)

藉由將目前的位置計數器值和指定的*類型*指派給*名稱*，建立新的標籤。

## <a name="syntax"></a>語法

> *名稱***標籤***類型*
>
> *名稱***標籤**⟦**接近** | **遠** | **PROC**⟧ **PTR** ⟦*類型*⟧
