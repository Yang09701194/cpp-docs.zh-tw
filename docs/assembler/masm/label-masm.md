# <a name="label-masm"></a>LABEL (MASM)

藉由指定目前位置計數器值建立新的標籤和給定*型別*要*名稱*。

## <a name="syntax"></a>語法

> *名稱*標籤*類型*
>
> *名稱*標籤\[NEAR |到目前為止 |處理序] PTR \[*型別*]
