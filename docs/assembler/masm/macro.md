# <a name="macro"></a>MACRO

標記名為*name*的宏區塊，並為呼叫宏時所傳遞的引數建立*參數*預留位置。

## <a name="syntax"></a>語法

> *name***宏**⟦*parameter* ⟦ **：需求**|： =*default* |  **： VARARG**⟧ .。。⟧\
> *語句*\
> **ENDM** ⟦*value*⟧

## <a name="remarks"></a>備註

宏函式會將*值*傳回給呼叫語句。
