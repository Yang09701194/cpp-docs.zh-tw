# <a name="macro"></a>MACRO

呼叫的巨集區塊會將標示*名稱*，並建立*參數*巨集呼叫時，傳遞引數的預留位置。

## <a name="syntax"></a>語法

> *name* MACRO [[*parameter* [[:REQ | :=*default* | :VARARG]] ]] ...<br/>
> *statements*<br/>
> ENDM [[*值*]]

## <a name="remarks"></a>備註

巨集函式會傳回*值*呼叫的陳述式。
