# <a name="microsoft-macro-assembler-bnf-grammar"></a>微軟巨集彙編器 BNF 語法

此頁包含 MASM 語法的 BNF 說明。 它作為參考的補充提供,不保證完整。 有關關鍵字、參數、操作等的完整資訊,請參閱參考。

為了說明 BNF 的使用,下圖顯示了 TYPEDEF 指令的定義*`typedefDir`*,從非終端 開始。

![MASM BNF 範例](media/bnf.png)

每個水準**`NEAR16`** 大括弧下的項目都是終端,如、**`NEAR32`****`FAR16`****`FAR32`** 與。 或者,*`qualifier`* 它們是非終端,如*`qualifiedType`**`distance`**`protoSpec`*, 可以進一步定義。 *`typedefDir`* 定義中的每個斜體非終端也是BNF中的條目。 三個垂直點指示非終端的分支定義,為了簡單起見,此圖無法說明。

BNF 語法允許遞歸定義。 例如,語法使用限定類型作為限定類型的可能定義,該類型也是限定符定義的一個元件。 "*"符號指定備用表示式之間的選擇,例如*`endOfLine`* | *`comment`*。 雙大括弧指定可選參數,例如*`macroParmList`*。 括弧實際上並不顯示在原始碼中。
