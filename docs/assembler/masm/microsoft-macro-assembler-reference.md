# Microsoft Macro Assembler Reference
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Microsoft 巨集組合語言 \(MASM\) 提供內嵌組譯碼的幾個優點。  MASM 包含巨集語言具有這樣的迴圈、 算術、 和文字的字串處理功能。  能 MASM 也提供硬體較大的控制權，因為它支援 386、 486 及 Pentium 處理器效能的指令集。  藉由使用 MASM，您也可以減少時間及記憶體額外負荷。
