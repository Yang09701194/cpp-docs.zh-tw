# <a name="microsoft-macro-assembler-reference"></a>Microsoft 巨集組譯參考

Microsoft 巨集組合程式 (MASM) 提供內嵌組譯碼的幾項優點。 MASM 包含巨集語言的功能，例如迴圈、算術和文字字串處理。 MASM 也可讓您更充分地掌控硬體，因為它支援 386、486 和 Pentium 處理器的指令集。 您也可以使用 MASM 來減少時間和記憶體的額外負荷。
