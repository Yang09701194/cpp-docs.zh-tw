# <a name="ml-fatal-error-a1000"></a>ML 嚴重錯誤 A1000

**無法開啟檔案：檔案名**

組合器無法開啟來源、包含或輸出檔案。

下列其中一項可能是原因：

- 檔案不存在。

- 檔案正由另一個進程使用中。

- 檔案名無效。

- 具有輸出檔案名的唯讀檔案已經存在。

- 目前的磁片磁碟機已滿。

- 目前的目錄是根節點，已滿。

- 無法寫入裝置。

- 磁片磁碟機未就緒。
