# ML Fatal Error A1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**無法開啟檔案: 檔案名稱**  
  
 組譯工具無法開啟來源、 包含或輸出檔。  
  
 下列其中一項可能的原因：  
  
-   檔案不存在。  
  
-   檔案是由其他處理程序使用中。  
  
-   檔名不正確。  
  
-   唯讀的檔案，以輸出檔名已經存在。  
  
-   目前的磁碟機已滿。  
  
-   目前的目錄是根目錄，並且已滿。  
  
-   無法寫入該裝置。  
  
-   磁碟機尚未就緒。
