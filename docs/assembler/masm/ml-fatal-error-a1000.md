# <a name="ml-fatal-error-a1000"></a>ML 嚴重錯誤 A1000
**無法開啟檔案： filename**  
  
 組譯工具無法開啟來源、 包含或輸出檔。  
  
 下列其中一種可能的原因：  
  
-   檔案不存在。  
  
-   檔案是由另一個處理序使用中。  
  
-   檔案名稱無效。  
  
-   輸出檔案名稱的唯讀檔案已經存在。  
  
-   目前的磁碟機已滿。  
  
-   目前的目錄是根目錄，並已滿。  
  
-   無法寫入該裝置。  
  
-   磁碟機未就緒。
