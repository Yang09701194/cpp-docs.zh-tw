# <a name="ml-fatal-error-a1005"></a>ML 嚴重錯誤 A1005
**組譯工具限制： 巨集完整的參數名稱資料表**  
  
 太多參數、 區域變數或巨集標籤定義的巨集。 在巨集名稱表格中時發生沒有更多空間。  
  
 定義短或更少的名稱或移除不必要的巨集。
