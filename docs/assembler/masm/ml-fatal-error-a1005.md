# <a name="ml-fatal-error-a1005"></a>ML 嚴重錯誤 A1005

**組合器限制：巨集引數名稱資料表已滿**

為巨集定義了太多參數、區域變數或宏標籤。 宏名稱表中已沒有足夠的空間。

請定義較短或較少的名稱，或移除不必要的宏。
