# ML Fatal Error A1005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**組合器限制: 巨集完整的參數名稱表格**  
  
 太多的參數、 區域變數或巨集的標籤所定義的巨集。  巨集名稱表格中沒有更多空間。  
  
 定義短或較少的名稱或移除不必要的巨集。
