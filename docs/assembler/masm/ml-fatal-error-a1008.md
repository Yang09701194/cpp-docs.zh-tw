# <a name="ml-fatal-error-a1008"></a>ML 嚴重錯誤 A1008
**不相符的巨集巢狀結構**  
  
 巨集未結束的檔案或終止指示詞之前終止[ENDM](../../assembler/masm/endm.md)找不到巨集區塊之外。  
  
 此錯誤的其中一個原因是之前的點省略[。重複](../../assembler/masm/dot-repeat.md)或[。雖然](../../assembler/masm/dot-while.md)。
