# ML Fatal Error A1008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**不相符的巨集巢狀結構**  
  
 不論是哪一類型的巨集未結束的檔案或結束的指示詞結尾之前，  [ENDM](../../assembler/masm/endm.md) 找不到巨集區塊之外。  
  
 這類錯誤的其中一個原因是略過前面點 [。重複](../../assembler/masm/dot-repeat.md) 或  [。雖然](../../assembler/masm/dot-while.md)。
