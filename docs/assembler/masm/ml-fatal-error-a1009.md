# <a name="ml-fatal-error-a1009"></a>ML 嚴重錯誤 A1009
**行太長**  
  
 原始程式檔中的資料行超過 512 個字元的限制。  
  
 如果使用行接續字元 (\) 串連多個實體的線條，產生邏輯程式敘述行則仍會限制為 512 個字元。
