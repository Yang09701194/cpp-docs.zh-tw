# <a name="ml-fatal-error-a1009"></a>ML 嚴重錯誤 A1009

**行太長**

原始程式檔中的一行超過512個字元的限制。

如果多個實體線與行接續字元（\）串連，則產生的邏輯行仍然限制為512個字元。
