# <a name="ml-fatal-error-a1016"></a>ML 嚴重錯誤 A1016

**內部組譯工具錯誤**

MASM 的驅動程式，稱為 ML.exe，產生的系統錯誤。

請記錄發生錯誤的情況，並通知 Microsoft Corporation。 產品支援服務將會位於[ http://support.microsoft.com/ ](http://support.microsoft.com/)。
