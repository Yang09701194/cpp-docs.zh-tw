# <a name="ml-fatal-error-a1016"></a>ML 嚴重錯誤 A1016

**內部組譯工具錯誤**

名為 ML.exe 的 MASM 驅動程式產生了系統錯誤。

請記錄發生錯誤的情況，並通知 Microsoft Corporation。 產品支援服務位於 [https://support.microsoft.com/](https://support.microsoft.com/)。
