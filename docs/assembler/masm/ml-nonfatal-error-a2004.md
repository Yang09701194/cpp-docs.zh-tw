# ML Nonfatal Error A2004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**符號型別衝突: 識別項**  
  
 [EXTERNDEF](../../assembler/masm/externdef.md) 或[標籤](../../assembler/masm/label-masm.md)指示詞用於變數、 符號、 資料結構或相同的模組，但不同的型別所定義的標籤。
