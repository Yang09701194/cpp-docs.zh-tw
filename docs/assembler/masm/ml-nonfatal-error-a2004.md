# <a name="ml-nonfatal-error-a2004"></a>ML 非嚴重錯誤 A2004

**符號類型衝突：識別碼**

[EXTERNDEF](../../assembler/masm/externdef.md)或[label](../../assembler/masm/label-masm.md)指示詞用於在相同模組中定義但具有不同類型的變數、符號、資料結構或標籤上。
