# <a name="ml-nonfatal-error-a2006"></a>ML 非嚴重錯誤 A2006
**未定義的符號： 識別項**  
  
 您嘗試使用未定義的符號。  
  
 可能發生下列其中一項：  
  
-   未定義符號。  
  
-   欄位不是指定之結構的成員。  
  
-   符號已定義未包含的 include 檔中。  
  
-   外部符號時未搭配[EXTERN](../../assembler/masm/extern-masm.md)或[EXTERNDEF](../../assembler/masm/externdef.md)指示詞。  
  
-   符號名稱的拼字錯誤。  
  
-   本機的程式碼標籤為參考其範圍之外。
