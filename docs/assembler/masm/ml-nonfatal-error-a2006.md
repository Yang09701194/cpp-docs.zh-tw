# ML Nonfatal Error A2006
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**未定義的符號: 識別項**  
  
 您嘗試使用未定義的符號。  
  
 可能發生下列其中一項：  
  
-   未定義的符號。  
  
-   欄位不屬於指定的結構。  
  
-   未包含的 include 檔中定義的符號。  
  
-   使用外部的符號 [EXTERN](../../assembler/masm/extern-masm.md) 或 [EXTERNDEF](../../assembler/masm/externdef.md) 指示詞。  
  
-   符號名稱的拼字錯誤。  
  
-   本機程式碼標籤被參考的領域之外。
