# <a name="ml-nonfatal-error-a2008"></a>ML 非嚴重錯誤 A2008

**語法錯誤：**

目前位置的 token 導致語法錯誤。

可能發生下列其中一種情況：

- 已在指示詞中加入或省略點前置詞。

- 保留字（例如**C**或**大小**）已當做識別碼使用。

- 目前的處理器或副處理器選取專案未提供使用的指示。

- 比較執行時間運算子（例如 `==`）用於條件式元件語句，而不是關聯式運算子（例如[EQ](../../assembler/masm/operator-eq.md)）。

- 指定了太少運算元的指令或指示詞。

- 使用了過時的指示詞。
