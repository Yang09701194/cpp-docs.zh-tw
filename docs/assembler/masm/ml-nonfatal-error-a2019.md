# ML Nonfatal Error A2019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**運算元必須是記錄類型或欄位**  
  
 運算元以下[寬度](../../assembler/masm/operator-width.md) 或 [遮罩](../../assembler/masm/operator-mask.md)運算子不正確。  
  
 [寬度](../../assembler/masm/operator-width.md)運算子會產生運算元為欄位或記錄的名稱。  [遮罩](../../assembler/masm/operator-mask.md)運算子會產生，且該欄位或記錄類型的運算元。
