# <a name="ml-nonfatal-error-a2019"></a>ML 非嚴重錯誤 A2019

**運算元必須是記錄類型或欄位**

[寬度](operator-width.md)或[遮罩](operator-mask.md)運算子之後的運算元無效。

[WIDTH](operator-width.md)運算子採用的運算元是欄位或記錄的名稱。 [MASK](operator-mask.md)運算子採用的運算元是欄位或記錄類型的名稱。
