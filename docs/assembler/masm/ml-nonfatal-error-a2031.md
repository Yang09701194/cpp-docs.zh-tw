# <a name="ml-nonfatal-error-a2031"></a>ML 非嚴重錯誤 A2031
**必須是基底或索引暫存器**  
  
 您嘗試使用不是基底或索引暫存器記憶體運算式中的暫存器。  
  
 例如，下列運算式會造成這個錯誤：  
  
```  
[ax]  
[bl]  
```
