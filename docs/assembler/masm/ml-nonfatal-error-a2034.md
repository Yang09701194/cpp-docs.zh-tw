# <a name="ml-nonfatal-error-a2034"></a>ML 非嚴重錯誤 A2034

**必須是區段區塊中**

下列其中一項位於區段區塊：

- 指示

- 標籤定義

- A [THIS](../../assembler/masm/operator-this.md)運算子

- A [ $ ](../../assembler/masm/dollar.md)運算子

- 程序定義

- [對齊](../../assembler/masm/align-masm.md)指示詞

- [ORG](../../assembler/masm/org.md)指示詞
