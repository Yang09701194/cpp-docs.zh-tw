# <a name="ml-nonfatal-error-a2034"></a>ML 非嚴重錯誤 A2034

**必須在區段區塊中**

在區段區塊外找到下列其中一項：

- 指示

- 標籤定義

- [這個](../../assembler/masm/operator-this.md)運算子

- [$](../../assembler/masm/dollar.md)運算子

- 程序定義

- [ALIGN](../../assembler/masm/align-masm.md)指示詞

- [組織](../../assembler/masm/org.md)指示詞
