# ML Nonfatal Error A2037
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**不允許在結構定義陳述式**  
  
 結構的定義包含無效的陳述式。  
  
 結構無法包含指示、 標籤、 程序、 流程控制指示詞，  [。啟動](../../assembler/masm/dot-startup.md)，或  [。結束](../../assembler/masm/dot-exit.md)。
