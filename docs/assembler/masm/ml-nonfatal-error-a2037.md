# <a name="ml-nonfatal-error-a2037"></a>ML 非嚴重錯誤 A2037

**結構定義內不允許有語句**

結構定義包含不正確語句。

結構不能包含指令、標籤、程式、控制流程指示詞[。啟動](../../assembler/masm/dot-startup.md)或[。](../../assembler/masm/dot-exit.md)結束。
