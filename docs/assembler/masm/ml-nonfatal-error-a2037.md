# <a name="ml-nonfatal-error-a2037"></a>ML 非嚴重錯誤 A2037
**不允許結構定義內的陳述式**  
  
 結構定義包含無效的陳述式。  
  
 結構無法包含指示、 標籤、 程序、 控制流程的指示詞， [。啟動](../../assembler/masm/dot-startup.md)，或[。結束](../../assembler/masm/dot-exit.md)。
