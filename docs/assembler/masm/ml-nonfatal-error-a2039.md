# <a name="ml-nonfatal-error-a2039"></a>ML 非嚴重錯誤 A2039

**行太長**

原始程式檔行超過512個字元的限制。

如果多個實體線與行接續字元（\）串連，則產生的邏輯行仍然限制為512個字元。
