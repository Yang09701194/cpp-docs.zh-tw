# <a name="ml-nonfatal-error-a2039"></a>ML 非嚴重錯誤 A2039

**行太長**

來源檔案行超過 512 個字元的限制。

如果使用行接續字元 (\)，會串連多個實際的程式碼行，產生邏輯程式敘述行則仍會限制為 512 個字元。
