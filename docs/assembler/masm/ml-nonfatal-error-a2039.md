# ML Nonfatal Error A2039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**列太長**  
  
 原始程式檔行超過 512 個字元的限制。  
  
 多個實體行都串連利用行接續字元 \(\\\)，然後產生邏輯程式敘述行仍然限制在 512 個字元。
