# ML Nonfatal Error A2044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**在檔案中有無效的字元**  
  
 原始程式檔所包含的字元以外的註解的字串，或常值，無法辨識為操作員或其他合法的字元。
