# ML Nonfatal Error A2047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**空 \(null\) 的字串**  
  
 字串是由分隔對引號和內任何字元所組成。  
  
 為有效的字串，它必須包含 1–255 個字元。
