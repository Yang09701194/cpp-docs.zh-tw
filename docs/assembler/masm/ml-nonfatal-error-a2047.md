# <a name="ml-nonfatal-error-a2047"></a>ML 非嚴重錯誤 A2047
**空 (null) 的字串**  
  
 字串的引號內的任何字元分隔組所組成。  
  
 針對有效的字串，它必須包含 1-255 個字元。
