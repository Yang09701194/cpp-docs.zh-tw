# <a name="ml-nonfatal-error-a2059"></a>ML 非嚴重錯誤 A2059

**強制錯誤： 字串相等**

條件式錯誤指示詞[。ERRIDN](../../assembler/masm/dot-erridn.md)或 **。ERRIDNI**用來產生這個錯誤。
