# ML Nonfatal Error A2060
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

 條件錯誤指示詞 [。ERRDIF](../../assembler/masm/dot-errdif.md) 或 **。ERRDIFI** 用來產生這個錯誤。
