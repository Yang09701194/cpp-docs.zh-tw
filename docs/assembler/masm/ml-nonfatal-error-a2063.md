# <a name="ml-nonfatal-error-a2063"></a>ML 非嚴重錯誤 A2063

**只能對齊2： expression 的乘冪**

使用[ALIGN](../../assembler/masm/align-masm.md)指示詞指定的運算式無效。

**ALIGN**運算式必須是2到256之間的2乘冪，而且必須小於或等於目前區段、結構或等位的對齊方式。
