# <a name="ml-nonfatal-error-a2063"></a>ML 非嚴重錯誤 A2063
**可以對齊只有 2 的乘冪： 運算式**  
  
 使用指定的運算式[對齊](../../assembler/masm/align-masm.md)指示詞無效。  
  
 **對齊**運算式必須是介於 2 到 256 之間的 2 的乘冪，而且必須小於或等於目前區段、 結構或等位的對齊方式。
