# ML Nonfatal Error A2063
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**可以只為 2 的乘冪的對齊: 運算式**  
  
 使用指定的運算式[對齊](../../assembler/masm/align-masm.md)指示詞不正確。  
  
 **對齊**運算式必須是介於 2 到 256 之間，2 的乘冪，而且必須小於或等於目前的區段、 結構或等位的對齊方式。
