# <a name="ml-nonfatal-error-a2066"></a>ML 非嚴重錯誤 A2066

**不相容的 CPU 模式和區段大小**

嘗試使用與指定的 CPU 不相容的**USE16**、 **USE32**或一般屬性來開啟區段 **，或在**32 位區段中變更為16位 CPU。

**USE32**和**單層**屬性的前面必須加上386或更高的處理器指示詞。
