# <a name="ml-nonfatal-error-a2066"></a>ML 非嚴重錯誤 A2066

**不相容的 CPU 模式和區段大小**

嘗試開啟區段**USE16**， **USE32**，或**一般**屬性不相容，而指定的 CPU，或變更為在 32 位元中的 16 位元 cpu。區段。

**USE32**並**一般**屬性前面必須有.386 或更高的處理器指示詞。
