# ML Nonfatal Error A2066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**不相容的 CPU 模式與區段大小**  
  
 嘗試開啟具有區段 **USE16**，  **USE32**，或 **平面**不相容，以指定的 CPU，或將變更為 16 位元 CPU 32 位元區段中的屬性。  
  
 **USE32** 和**平面**之前的屬性必須搭配.386 或更大的處理器指示詞。
