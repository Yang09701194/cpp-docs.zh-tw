# <a name="ml-nonfatal-error-a2066"></a>ML 非嚴重錯誤 A2066
**不相容的 CPU 模式和區段大小**  
  
 嘗試開啟具有區段**USE16**， **USE32**，或**一般**不相容，使用指定的 CPU，或變更為 16 位元 CPU 在 32 位元中的屬性區段。  
  
 **USE32**和**一般**屬性前面必須有.386 或更高的處理器指示詞。
