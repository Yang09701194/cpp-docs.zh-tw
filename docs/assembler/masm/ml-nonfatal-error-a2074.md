# ML Nonfatal Error A2074
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**無法存取透過區段登錄器的標籤**  
  
 嘗試透過來存取一個標籤不會被視為其區段或群組的區段暫存器。
