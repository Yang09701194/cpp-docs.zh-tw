# ML Nonfatal Error A2078
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**指示不允許遠間接定址**  
  
 條件式的躍進或迴圈不能使用記憶體的運算元。  必須為它指定相對的地址或標籤。
