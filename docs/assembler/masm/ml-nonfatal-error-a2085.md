# <a name="ml-nonfatal-error-a2085"></a>ML 非嚴重錯誤 A2085

**目前的 CPU 模式不接受指示或註冊**

嘗試使用對目前處理器模式不正確指令、註冊或關鍵字。

例如，32位暫存器需要[386](dot-386.md)或更新版本。 控制暫存器（例如 CR0 暫存器）需要特殊許可權模式[。 .386p](dot-386p.md)或更新版本。 也會針對需要的**NEAR32**、 **FAR32**和**一般關鍵字產生**此錯誤。**386**或更新版本。
