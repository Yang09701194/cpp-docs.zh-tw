# <a name="ml-nonfatal-error-a2096"></a>ML 非嚴重錯誤 A2096

**需要區段、群組或區段註冊**

需要區段或群組，但找不到。

發生下列其中一種情況：

- 以區段覆寫運算子（ **：** ）指定的左運算元不是區段暫存器（CS、DS、SS、ES、FS 或 GS）、組名、區段名稱或區段運算式。

- [假設](assume.md)指示詞在沒有有效區段位址、區段暫存器、群組**或特殊一般**群組的情況下，已獲得區段暫存器。
