# ML Nonfatal Error A2107
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**不能有隱含的遠程跳躍或即將接近標籤的呼叫**  
  
 您嘗試進行隱含遠跳或致電用於接近的標籤，在另一個區段。
