# <a name="ml-nonfatal-error-a2107"></a>ML 非嚴重錯誤 A2107

**無法對接近的標籤進行隱含的跳躍或呼叫**

嘗試在另一個區段中，對接近的標籤進行隱含的跳躍或呼叫。
