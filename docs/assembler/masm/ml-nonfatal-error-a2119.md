# <a name="ml-nonfatal-error-a2119"></a>ML 非嚴重錯誤 A2119

**必須指定語言類型**

程序定義或原型未指定語言型別。

如果未指定預設語言類型，必須每個程序定義或原型中宣告語言型別。 使用設定為預設語言型別的[。模型](../../assembler/masm/dot-model.md)指示詞**選項 LANG**，或 ML 命令列選項 **/Gc**或是 **/Gd**。
