# ML Nonfatal Error A2119
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**必須指定語言類型**  
  
 程序定義或原型未指定語言類型。  
  
 語言型別必須宣告每個程序定義或原型中，如果沒有指定預設的語言類型。  使用其中一個設定為預設語言型別的 [。模型](../../assembler/masm/dot-model.md) 指示詞， **選項蘭城**，或毫升命令列選項  **\/Gc** 或 **\/Gd**。
