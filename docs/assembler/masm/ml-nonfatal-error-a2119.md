# <a name="ml-nonfatal-error-a2119"></a>ML 非嚴重錯誤 A2119

**必須指定語言類型**

未指定程式定義或原型的語言類型。

如果未指定預設語言類型，則必須在每個程序定義或原型中宣告語言類型。 預設語言類型是使用來設定[。MODEL](../../assembler/masm/dot-model.md)指示詞、 **OPTION LANG**或 ML 命令列選項 **/Gc**或 **/gd**。
