# <a name="ml-nonfatal-error-a2133"></a>ML 非嚴重錯誤 A2133

**INVOKE 所覆寫的 register 值**

暫存器已當做引數傳遞至程式，但由[INVOKE](invoke.md)產生的程式碼傳遞其他引數，以終結暫存器的內容。

組合語言可能會使用 AX、AL、AH、EAX、DX、DL、DH 和 EDX 暫存器來執行資料轉換。

使用不同的註冊。
