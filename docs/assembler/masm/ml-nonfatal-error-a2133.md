# <a name="ml-nonfatal-error-a2133"></a>ML 非嚴重錯誤 A2133

**註冊叫用覆寫的值**

暫存器做為引數傳遞至程序，但產生的程式碼[INVOKE](../../assembler/masm/invoke.md)傳遞其他引數被終結的暫存器的內容。

AX、 AL、 AH、 EAX、 DX、 DL、 DH 及 EDX 暫存器可能使用組合器，來執行資料轉換。

使用不同的註冊。
