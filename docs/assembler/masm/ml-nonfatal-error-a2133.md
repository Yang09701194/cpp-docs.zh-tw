# <a name="ml-nonfatal-error-a2133"></a>ML 非嚴重錯誤 A2133
**登錄值叫用由覆寫**  
  
 暫存器做為引數傳遞至程序，但產生的程式碼[INVOKE](../../assembler/masm/invoke.md)傳遞其他引數終結的暫存器的內容。  
  
 若要執行資料轉換組合器會使用 AX、 AL、 AH、 EAX、 DX、 DL、 DH 和 EDX 暫存器。  
  
 使用不同的暫存器。
