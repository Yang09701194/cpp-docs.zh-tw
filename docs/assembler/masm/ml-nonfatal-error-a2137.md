# <a name="ml-nonfatal-error-a2137"></a>ML 非嚴重錯誤 A2137
**若要叫用的引數太少**  
  
 使用傳遞的引數數目[INVOKE](../../assembler/masm/invoke.md)指示詞已在程序所叫用原型中指定的必要參數數目較少。
