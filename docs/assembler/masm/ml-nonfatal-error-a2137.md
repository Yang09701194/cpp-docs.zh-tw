# <a name="ml-nonfatal-error-a2137"></a>ML 非嚴重錯誤 A2137

使用傳遞的引數[INVOKE](../../assembler/masm/invoke.md)指示詞已叫用的程序的原型中指定的必要參數的數目少於。
