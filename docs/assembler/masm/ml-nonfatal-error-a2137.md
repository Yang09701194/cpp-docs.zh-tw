# <a name="ml-nonfatal-error-a2137"></a>ML 非嚴重錯誤 A2137

**要叫用的引數太少**

使用[INVOKE](../../assembler/masm/invoke.md)指示詞傳遞的引數數目，少於所叫用程式在原型中指定的必要參數數目。
