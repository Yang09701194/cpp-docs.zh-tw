# ML Nonfatal Error A2137
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

 使用傳遞的引數個數[叫用](../../assembler/masm/invoke.md)指示詞不少於一個的必要程序正在叫用原型中所指定的參數數目。
