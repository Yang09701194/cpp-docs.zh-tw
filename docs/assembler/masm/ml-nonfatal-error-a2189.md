# ML Nonfatal Error A2189
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**使用區段的對齊方式的組合不正確**  
  
 所指定的對齊方式**對齊**或`EVEN`指示詞是大於目前的區段對齊方式，所指定的**區段**指示詞。
