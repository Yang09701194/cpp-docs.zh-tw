# <a name="ml-nonfatal-error-a2189"></a>ML 非嚴重錯誤 A2189

**區段對齊的組合無效**

所指定的對齊**對齊**或是`EVEN`指示詞已大於所指定的目前區段對齊**區段**指示詞。
