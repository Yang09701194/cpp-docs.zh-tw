# ML Nonfatal Error A2206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**在運算式中少了運算元**  
  
 無法評估運算式，因為缺少一個運算子。  這個錯誤訊息也可能是先前的程式錯誤的副作用。  
  
 下面這一行會產生這個錯誤：  
  
```  
value1 = ( 1 + 2 ) 3  
  
```
