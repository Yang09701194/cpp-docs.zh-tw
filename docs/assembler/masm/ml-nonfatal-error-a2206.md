# <a name="ml-nonfatal-error-a2206"></a>ML 非嚴重錯誤 A2206

**運算式中遺漏運算子**

無法評估運算式，因為它遺漏運算子。 此錯誤訊息也可能是先前程式錯誤的副作用。

下面這一行會產生此錯誤：

```asm
value1 = ( 1 + 2 ) 3
```
