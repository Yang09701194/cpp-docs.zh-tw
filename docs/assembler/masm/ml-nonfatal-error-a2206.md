# <a name="ml-nonfatal-error-a2206"></a>ML 非嚴重錯誤 A2206
**在運算式中遺漏運算子**  
  
 無法評估運算式，因為它遺漏運算子。 這則錯誤訊息也可能造成副作用前述的程式錯誤。  
  
 下列程式會產生此錯誤：  
  
```  
value1 = ( 1 + 2 ) 3  
  
```
