# ML Nonfatal Error A2219
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**不正確的對應中的位移回溯程式碼**  
  
 運算元為[.ALLOCSTACK](../../assembler/masm/dot-allocstack.md)和[.SAVEREG](../../assembler/masm/dot-savereg.md)必須是 8 的倍數。  運算元為[.SAVEXMM128](../../assembler/masm/dot-savexmm128.md)和[.SETFRAME](../../assembler/masm/dot-setframe.md)必須是 16 的倍數。
