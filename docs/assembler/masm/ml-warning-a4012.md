# ML Warning A4012
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**無類別 'CODE' 區段的行號資訊**  
  
 沒有指示區段並沒有類別名稱結尾為 「 代碼 」。 組譯工具不會產生可檢視程式碼的資訊，這些指示。  
  
 可檢視程式碼無法處理使用的類別名稱中不會結束與 「 程式碼 」 區段中的程式碼的模組
