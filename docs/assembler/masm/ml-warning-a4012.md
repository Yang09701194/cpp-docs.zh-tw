# <a name="ml-warning-a4012"></a>ML 警告 A4012
**沒有類別 'CODE' 區段的行號資訊**  
  
 沒有指示在區段中沒有類別名稱結尾為"CODE"。 組譯工具不會產生這些指示的 CodeView 資訊。  
  
 CodeView 無法處理具有類別名稱不是以 「 程式碼。"結尾的區段中的程式碼模組
