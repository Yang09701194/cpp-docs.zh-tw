# <a name="ml-warning-a4012"></a>ML 警告 A4012

**沒有類別 'CODE' 區段的行號資訊**

沒有指示在區段中沒有類別名稱的結尾是"CODE"。 組譯工具不會產生這些指示的 CodeView 資訊。

CodeView 無法處理具有不以"CODE"。 結束的類別名稱的區段中的程式碼模組
