# <a name="ml-warning-a4012"></a>ML 警告 A4012

**沒有類別 ' CODE ' 的區段的行號資訊**

區段中的指示沒有以 "CODE" 結尾的類別名稱。 組合語言不會產生這些指示的 CodeView 資訊。

CodeView 無法使用不是以 "CODE" 結尾之類別名稱的區段中的程式碼來處理模組。
