# <a name="ml-warning-a4014"></a>ML 警告 A4014
指示和初始化的 BSS 區段中不支援的資料  
  
 嘗試定義初始化的 BSS 區段內的資料。  BSS 區段會定義為其名稱是 BSS 類別。  這包括簡化的區段`.data?`。
