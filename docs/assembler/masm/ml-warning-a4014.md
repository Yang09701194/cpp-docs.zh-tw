# ML Warning A4014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

指示及初始化的 BSS 區段不支援的資料  
  
 嘗試定義初始化的 BSS 區段內的資料。  BSS 區段被定義為類別，其名稱是 BSS。  這包括簡化的區段`.data?`。
