# <a name="ml-warning-a4014"></a>ML 警告 A4014

BSS 區段中不支援的指示和初始化資料

嘗試在 BSS 區段中定義初始化資料。  BSS 區段會定義為其名稱為 BSS 的類別。  這包括簡化的區段 `.data?`。
