# <a name="operator-"></a>運算子 +
第一個運算子會傳回*expression1*加上*expression2*。 第二個運算子變換正負號的*運算式*。  
  
## <a name="syntax"></a>語法  
  
```  
  
      expression1 + expression2  
-expression  
```
