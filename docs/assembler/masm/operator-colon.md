# <a name="operator-"></a>運算子 :

以*區段*覆寫*運算式*的預設區段。 *區段*可以是區段暫存器、組名、區段名稱或區段運算式。 *運算式*必須是常數。

## <a name="syntax"></a>語法

> *區段* **：** *運算式*
