# <a name="operator-"></a>運算子 :

Overrides the default segment of *expression* with *segment*. The *segment* can be a segment register, group name, segment name, or segment expression. The *expression* must be a constant.

## <a name="syntax"></a>語法

> *segment* **:** *expression*
