# <a name="operator-"></a>運算子 :
覆寫的預設區段*運算式*與`segment`。 `segment`可以區段暫存器、 群組名稱、 區段名稱或區段運算式。 *運算式*必須是常數。  
  
## <a name="syntax"></a>語法  
  
```  
  
segment  
: expression  
```
