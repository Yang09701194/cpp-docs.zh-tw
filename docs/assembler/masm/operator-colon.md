# operator :
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

會覆寫的預設區段*運算式*與`segment`。  `segment`可以是區段暫存器、 群組名稱、 區段名稱或區段的運算式。  *運算式*必須是常數。  
  
## 語法  
  
```  
  
segment  
: expression  
```
