# <a name="operator-"></a>運算子 .

第一個運算子會傳回*運算式*，加上*欄位*在其結構或等位內的位移。 第二個運算子會在其結構或等位內，于*register*加上*欄位*位移的位置傳回值。

## <a name="syntax"></a>語法

> *expression* __。__ *欄位*⟦ __。__ *欄位*.。。⟧
>
> **\[** _register_ **]** __。__ *欄位*⟦ __。__ *欄位*.。。⟧
