# <a name="operator-"></a>運算子 .

第一個運算子會傳回*運算式*再加上位移*欄位*其結構或等位內。 第二個運算子傳回值所指向的位置*註冊*再加上位移*欄位*其結構或等位內。

## <a name="syntax"></a>語法

> *運算式* __。__ *欄位* \[ __。__ *field*] ...
>
> __\[__*register*__]__ __.__ *欄位* \[ __。__ *field*] ...
