# <a name="operator-"></a>運算子 .
第一個運算子會傳回*運算式*再加上位移*欄位*其結構或等位內。 第二個運算子所傳回的值所指向的位置*註冊*再加上位移*欄位*其結構或等位內。  
  
## <a name="syntax"></a>語法  
  
```  
  
      expression  
      . field [[. field]]...  
[register]. field [[. field]]...  
```
