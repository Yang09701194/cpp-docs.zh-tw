# operator .
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

第一個運算子會傳回*運算式* 再加上位移的 *欄位*在其結構或等位。  第二個運算子會傳回所指位置的值*註冊* 再加上位移的 *欄位*在其結構或等位。  
  
## 語法  
  
```  
  
      expression  
      . field [[. field]]...  
[register]. field [[. field]]...  
```
