# <a name="operator-eq"></a>運算子 EQ
如果為 true (-1) *expression1*等於*expression2*，或如果沒有，則傳回 false (0)。  
  
## <a name="syntax"></a>語法  
  
```  
  
expression1 EQ expression2  
```
