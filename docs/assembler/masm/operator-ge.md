# <a name="operator-ge"></a>運算子 GE
如果為 true (-1) *expression1*大於或等於*expression2*，或如果不是傳回 false (0)。  
  
## <a name="syntax"></a>語法  
  
```  
  
expression1 GE expression2  
```
