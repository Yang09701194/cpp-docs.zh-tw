# <a name="operator-ge"></a>運算子 GE

如果*運算式*1 大於或等於*運算式*2，則傳回 true （-1），如果不是，則傳回 false （0）。

## <a name="syntax"></a>語法

> *運算式*2- **GE** *運算式*
