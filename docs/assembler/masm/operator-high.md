# operator HIGH
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

傳回高度的 8 位元的低的 16 位元的*運算式*。  MASM 運算式是 64 位元值。  
  
## 語法  
  
```  
  
HIGH expression  
```
