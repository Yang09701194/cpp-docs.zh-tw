# operator HIGH32
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

傳回高 32 位元的*運算式*。  MASM 運算式是 64 位元值。  
  
## 語法  
  
```  
  
HIGH32 expression  
```
