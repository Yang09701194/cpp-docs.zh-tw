# <a name="operator-highword"></a>運算子 HIGHWORD
傳回的低 32 位元的高 16 個位元*運算式*。 MASM 運算式是 64 位元值。  
  
## <a name="syntax"></a>語法  
  
```  
  
HIGHWORD expression  
```
