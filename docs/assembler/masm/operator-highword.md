# operator HIGHWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

傳回的低 32 位元高的 16 位元*運算式*。  MASM 運算式是 64 位元值。  
  
## 語法  
  
```  
  
HIGHWORD expression  
```
