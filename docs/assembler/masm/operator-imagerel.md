# <a name="operator-imagerel"></a>運算子 IMAGEREL
傳回的映像相對位移`expression`。  
  
## <a name="syntax"></a>語法  
  
```  
IMAGEREL expression  
```  
  
## <a name="remarks"></a>備註  
 產生的值通常稱為 RVA 或相對虛擬位址。  
  
 IMAGEREL 是只能搭配 COFF 物件發出。
