# <a name="operator-imagerel"></a>運算子 IMAGEREL

傳回*運算式*的影像相對位移。

## <a name="syntax"></a>語法

> **IMAGEREL** *運算式*

## <a name="remarks"></a>備註

產生的值通常稱為「RVA」或「相對虛擬位址」。

IMAGEREL 僅適用于發出的 COFF 物件。
