# <a name="operator-imagerel"></a>運算子 IMAGEREL

傳回映像的相對位移`expression`。

## <a name="syntax"></a>語法

> IMAGEREL 運算式

## <a name="remarks"></a>備註

產生的值通常稱為 RVA 或相對虛擬位址。

可以只使用 COFF 物件發出 IMAGEREL。
