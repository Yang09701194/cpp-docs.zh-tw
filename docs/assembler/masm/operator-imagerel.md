# operator IMAGEREL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

傳回影像的相對位移的`expression`。  
  
## 語法  
  
```  
IMAGEREL expression  
```  
  
## 備註  
 產生的值通常稱為 RVA 或相對虛擬位址。  
  
 IMAGEREL 是只適用於 COFF 物件發出的。
