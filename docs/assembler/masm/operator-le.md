# <a name="operator-le"></a>運算子 LE

如果*運算式*1 小於或等於*運算式*2，則傳回 true （-1），如果不是，則傳回 false （0）。

## <a name="syntax"></a>語法

> *運算式*2 **LE** *運算式*2
