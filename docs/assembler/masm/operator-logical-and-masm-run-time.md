# operator &amp;&amp; (MASM Run Time)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

邏輯 **AND**。  只有在使用 [。IF](../../assembler/masm/dot-if.md), [.WHILE](../../assembler/masm/dot-while.md), or [.重複](../../assembler/masm/dot-repeat.md)會封鎖，並在 run time 時，不是在組件的時間評估。  
  
## 語法  
  
```  
  
expression1 && expression2  
```
