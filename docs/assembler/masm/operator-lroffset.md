# operator LROFFSET
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

傳回的位移*運算式*。  相同的**的位移**，不過會產生載入器解析的位移，可讓 Windows 重新定位程式碼區段。  
  
## 語法  
  
```  
  
LROFFSET expression  
```
