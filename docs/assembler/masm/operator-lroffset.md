# <a name="operator-lroffset"></a>運算子 LROFFSET

傳回的位移*運算式*。 與相同**位移**，但它會產生載入器解析的位移，可讓 Windows 即可重新定位程式碼區段。

## <a name="syntax"></a>語法

> LROFFSET 運算式
