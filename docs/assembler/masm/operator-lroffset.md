# <a name="operator-lroffset"></a>運算子 LROFFSET

傳回*運算式*的位移。 與**offset**相同，但它會產生載入器已解決的位移，讓 Windows 能夠重新放置程式碼區段。

## <a name="syntax"></a>語法

> **LROFFSET** *運算式*
