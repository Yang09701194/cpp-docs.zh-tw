# operator LT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

如果傳回 true \(– 1\)  *expression1* 是小於*表示*，或如果它不會傳回 false \(0\)。  
  
## 語法  
  
```  
  
expression1 LT expression2  
```
