# <a name="operator-mask"></a>運算子 MASK

傳回位元遮罩中的位元*recordfieldname*或是*記錄*設定且已清除其他所有的位元。

## <a name="syntax"></a>語法

> 遮罩 {recordfieldname | 記錄}
