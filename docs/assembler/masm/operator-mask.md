# <a name="operator-mask"></a>運算子 MASK
傳回一個位元遮罩中的位元*recordfieldname*或*記錄*設定，並已清除其他所有的位元。  
  
## <a name="syntax"></a>語法  
  
```  
  
MASK {recordfieldname | record}  
```
