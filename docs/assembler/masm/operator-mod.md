# operator MOD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

傳回除法運算時的餘數 \(modulo\) 的整數值 *expression1* 的*表示*。  
  
## 語法  
  
```  
  
expression1 MOD expression2  
```
