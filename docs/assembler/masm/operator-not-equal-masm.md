# <a name="operator--masm"></a>運算子 != (MASM)

不等於。 只能在中使用[。如果](../../assembler/masm/dot-if.md)為，則為[。WHILE](../../assembler/masm/dot-while.md)、或[。重複](../../assembler/masm/dot-repeat.md)區塊，並在執行時間進行評估，而不是在元件時間。

## <a name="syntax"></a>語法

> *運算式*2 **！ =** *運算式*2
