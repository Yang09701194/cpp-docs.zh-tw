# <a name="operator--masm"></a>運算子 != (MASM)

不等於。 只有在使用[。如果](../../assembler/masm/dot-if.md)， [。雖然](../../assembler/masm/dot-while.md)，或[。重複](../../assembler/masm/dot-repeat.md)封鎖，並在執行階段，不是在組件階段評估。

## <a name="syntax"></a>語法

> expression1 != expression2
