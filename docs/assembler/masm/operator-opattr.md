# operator OPATTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

傳回定義的模式和範圍的字*運算式*。  低位元組是等於所傳回的位元組 [。型別](../../assembler/masm/operator-dot-type.md)。  高位元組包含其他資訊。  
  
## 語法  
  
```  
  
OPATTR expression  
```
