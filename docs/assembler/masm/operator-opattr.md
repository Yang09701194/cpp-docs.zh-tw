# <a name="operator-opattr"></a>運算子 OPATTR

傳回定義*運算式*模式和範圍的單字。 低位元組與所傳回的位元組相同[。輸入](../../assembler/masm/operator-dot-type.md)。 較高的位元組包含額外的資訊。

## <a name="syntax"></a>語法

> **OPATTR** *運算式*
