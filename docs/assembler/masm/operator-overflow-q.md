# <a name="operator-overflow"></a>運算子 OVERFLOW?
溢位旗標的狀態。 只有在使用[。如果](../../assembler/masm/dot-if.md)， [。雖然](../../assembler/masm/dot-while.md)，或[。重複](../../assembler/masm/dot-repeat.md)封鎖，且在執行階段，不是在組件時進行評估。  
  
## <a name="syntax"></a>語法  
  
```  
  
OVERFLOW?  
  
```
