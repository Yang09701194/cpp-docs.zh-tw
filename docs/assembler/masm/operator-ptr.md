# <a name="operator-ptr"></a>運算子 PTR

第一個運算子會強制*運算式*被視為具有指定*型別*。 第二個運算子指定的指標*型別*。

## <a name="syntax"></a>語法

> *型別*PTR*運算式*

> [[*距離*]] PTR*類型*
