# operator PTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

第一個運算子強制*運算式*被視為具有指定`type`。  第二個運算子指定變數的指標， `type`。  
  
## 語法  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
