# <a name="operator-sectionrel"></a>運算子 SECTIONREL
傳回運算式相對於包含最終可執行檔中的目標區段的區段相對位移。  
  
## <a name="syntax"></a>語法  
  
```  
SECTIONREL expression  
```  
  
## <a name="remarks"></a>備註  
 SECTIONREL 是只能搭配 COFF 物件發出。
