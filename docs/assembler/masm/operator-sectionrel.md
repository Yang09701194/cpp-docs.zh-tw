# <a name="operator-sectionrel"></a>運算子 SECTIONREL

傳回相對於最後一個可執行檔中包含目標之區段的運算式相對位移。

## <a name="syntax"></a>語法

> **SECTIONREL** *運算式*

## <a name="remarks"></a>備註

**SECTIONREL**僅適用于發出的 COFF 物件。
