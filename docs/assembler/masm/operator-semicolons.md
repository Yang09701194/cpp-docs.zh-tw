# <a name="operator-"></a>運算子 ;;
會將*文字*為巨集，只會出現在巨集定義中的註解。 清單不會顯示*文字*展開的巨集的位置。  
  
## <a name="syntax"></a>語法  
  
```  
  
;;  
text  
  
```
