# <a name="operator-short"></a>運算子 SHORT

將*標籤*的類型設定為 short。 所有跳到*標籤*都必須是簡短的（範圍介於-128 到 + 127 位元組之間，從跳躍指示到*標籤*）。

## <a name="syntax"></a>語法

> **簡短***標籤*
