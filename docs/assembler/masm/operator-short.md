# operator SHORT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

設定的型別*標籤*為短。  所有跳到*標籤* 必須很短 \(在範圍\-128 到 \+127 個位元組的跳躍點指令從 *標籤*\)。  
  
## 語法  
  
```  
  
SHORT label  
```
