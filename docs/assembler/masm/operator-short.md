# <a name="operator-short"></a>運算子 SHORT

設定的型別*標籤*簡短。 所有會跳至*標籤*必須是簡短 (範圍從跳躍指令的-128 到 + 127 個位元組來*標籤*)。

## <a name="syntax"></a>語法

> 簡短標籤
