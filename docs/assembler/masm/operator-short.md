# <a name="operator-short"></a>運算子 SHORT
設定的型別*標籤*簡短。 所有跳到*標籤*必須簡短 (若要從跳躍指令-128 到 + 127 個位元組範圍內*標籤*)。  
  
## <a name="syntax"></a>語法  
  
```  
  
SHORT label  
```
