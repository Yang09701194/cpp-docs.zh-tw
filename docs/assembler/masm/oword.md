# <a name="oword"></a>OWORD

當需要16個位元組的資料類型時，會當做類型規範使用。

## <a name="syntax"></a>語法

> ⟦*名稱*⟧ **OWORD**  *初始化運算式*⟦ __，__ *初始化運算式*.。。⟧
