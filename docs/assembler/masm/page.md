# <a name="page"></a>PAGE
第一個指示詞設定行*長度*和字元*寬度*的程式清單。 如果不指定任何引數，就會產生分頁符號。 第二個指示詞遞增區段數目，並將頁碼重設為 1。  
  
## <a name="syntax"></a>語法  
  
```  
  
      PAGE [[[[length]], width]]  
PAGE +  
```
