# <a name="page"></a>PAGE

第一個指示詞會設定程式清單的行*長度*和字元*寬度*。 如果未指定任何引數，則會產生分頁符號。 第二個指示詞會遞增區段編號，並將頁碼重設為1。

## <a name="syntax"></a>語法

> **頁面**⟦*長度*⟧⟦ __，__ *寬度*⟧

> **頁面 +**
