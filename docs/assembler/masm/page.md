# PAGE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

第一個指示詞設定列*長度* 和字元 *寬度*的程式清單。  如果沒有指定引數會指定的話，會產生一個分頁符號。  第二個指示詞會遞增的節編號，並且重設為 1 的頁面數量。  
  
## 語法  
  
```  
  
      PAGE [[[[length]], width]]  
PAGE +  
```
