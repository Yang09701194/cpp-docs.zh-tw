# <a name="popcontext"></a>POPCONTEXT

還原部分或全部的目前*內容*（由[PUSHCONTEXT](pushcontext.md)指示詞儲存）。 *內容*可以**假設**為（僅限32位 MASSM）、**基數**、**清單**、 **CPU** （僅限32位 MASSM）或**ALL**。

## <a name="syntax"></a>語法

> **POPCONTEXT** *內容*
