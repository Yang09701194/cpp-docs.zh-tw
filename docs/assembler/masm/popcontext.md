# <a name="popcontext"></a>POPCONTEXT
還原目前的部分或全部`context`(儲存[PUSHCONTEXT](../../assembler/masm/pushcontext.md)指示詞)。 `context`可以**採用**， `RADIX`，**列出**， **CPU**，或**所有**。  
  
## <a name="syntax"></a>語法  
  
```  
  
POPCONTEXT context  
```
