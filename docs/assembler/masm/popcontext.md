# <a name="popcontext"></a>POPCONTEXT

還原部分或全部的目前*內容*（由[PUSHCONTEXT](../../assembler/masm/pushcontext.md)指示詞儲存）。 *內容*可以是「**假設**」、「**基數**」、「**清單**」、「 **CPU**」或「**全部**」。

## <a name="syntax"></a>語法

> **POPCONTEXT** *內容*
