# <a name="public"></a>PUBLIC

讓程式中的所有其他模組都可以使用指定為*名稱*的每個變數、標籤或絕對符號。

## <a name="syntax"></a>語法

> **PUBLIC** ⟦*language-type*⟧ *name* ⟦ __，__ ⟦*language-type*⟧ *name* .。。⟧

## <a name="remarks"></a>備註

*Language 類型*引數只在32位 MASM 中有效。
