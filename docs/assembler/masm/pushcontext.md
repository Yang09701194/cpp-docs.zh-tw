# <a name="pushcontext"></a>PUSHCONTEXT
儲存目前的部分或全部`context`： 區段登錄假設基數值、 清單和 cref 旗標或處理器/處理器值。 `context`可以**採用**， `RADIX`，**列出**， **CPU**，或**所有**。  
  
## <a name="syntax"></a>語法  
  
```  
  
PUSHCONTEXT context  
```
