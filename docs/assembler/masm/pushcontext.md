# <a name="pushcontext"></a>PUSHCONTEXT

儲存部分或全部的目前*內容*：區段暫存器假設、基數值、清單和 cref 旗標，或處理器/副處理器值。 *內容*可以是「**假設**」、「**基數**」、「**清單**」、「 **CPU**」或「**全部**」。

## <a name="syntax"></a>語法

> **PUSHCONTEXT** *內容*
