# QWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

配置，並選擇性地將 8 位元組的儲存空間初始化每個`initializer`。  也可以當做型別規範型別是合法的任何地方。  
  
## 語法  
  
```  
[[name]] QWORD initializer [[, initializer]]...  
```
