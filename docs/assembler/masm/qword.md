# <a name="qword"></a>QWORD
配置並選擇性地為每個初始化 8 個位元組的儲存體`initializer`。 也可以用為類型規範的類型是合法的任何位置。  
  
## <a name="syntax"></a>語法  
  
```  
[[name]] QWORD initializer [[, initializer]]...  
```
