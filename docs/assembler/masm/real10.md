# <a name="real10"></a>REAL10

配置並選擇性地為每個*初始化運算式*初始化10位元組的浮點數。

## <a name="syntax"></a>語法

> *name* **REAL10** *初始化運算式*⟦ __，__ *初始化運算式*.。。⟧
