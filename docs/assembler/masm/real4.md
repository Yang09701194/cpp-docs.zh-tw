# <a name="real4"></a>REAL4

配置並選擇性地初始化每個*初始化運算式*的單精確度（4位元組）浮點數。

## <a name="syntax"></a>語法

> *name*  **REAL4** *初始化運算式*⟦ __，__ *初始化運算式*.。。⟧
