# <a name="real8"></a>REAL8

配置並選擇性地為每個*初始化運算式*初始化雙精確度（8位元組）浮點數。

## <a name="syntax"></a>語法

> *name* **REAL8** *初始化運算式*⟦ __，__ *初始化運算式*.。。⟧
