# <a name="record-masm"></a>RECORD (MASM)

宣告記錄類型，其中包含指定的欄位。 *fieldname*名稱 欄位中，*寬度*指定位元為單位的數目並*運算式*提供其初始值。

## <a name="syntax"></a>語法

> *recordname*記錄*fieldname*:*寬度*[[=*運算式*]] [[， *fieldname*:*寬度*[[=*運算式*]]]]...
