# <a name="record"></a>RECORD

宣告包含指定欄位的記錄類型。 *fieldname*會將欄位命名為， *width*會指定位的數目，而*expression*會提供其初始值。

## <a name="syntax"></a>語法

> *recordname* **記錄** *fieldname* __：__ *width* ⟦ __=__ *expression*⟧⟦ __，__ *fieldname* __：__ *width* ⟦ __=__ *expression*⟧ .。。⟧
