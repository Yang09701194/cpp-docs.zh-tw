# RECORD (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

宣告指定的欄位所組成的資料錄型別。  *欄位名稱* 名稱\] 欄位中， *寬度* 指定的位元，以及 *運算式*提供其初始數值。  
  
## 語法  
  
```  
  
   recordname RECORD fieldname:width [[= expression]]   
[[, fieldname:width [[= expression]]]]...  
```
