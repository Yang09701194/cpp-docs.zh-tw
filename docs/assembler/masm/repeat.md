# <a name="repeat"></a>REPEAT
將標示為重複的區塊*運算式*時間。 與相同[REPT](../../assembler/masm/rept.md)。  
  
## <a name="syntax"></a>語法  
  
```  
  
   REPEAT expression  
statements  
ENDM  
```
