# <a name="sbyte-masm"></a>SBYTE (MASM)

配置並選擇性地針對每個初始化的儲存體的帶正負號的位元組`initializer`。 也可用來當做型別規範的類型是合法的任何位置。

## <a name="syntax"></a>語法

> 名稱 SBYTEinitializer [[，初始設定式]]...
