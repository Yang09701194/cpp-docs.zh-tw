# <a name="sbyte-masm"></a>SBYTE (MASM)
配置並選擇性地為每個初始化儲存體的帶正負號的位元組`initializer`。 也可用以做為類型指定型別是合法的任何位置。  
  
## <a name="syntax"></a>語法  
  
```  
  
name SBYTEinitializer [[, initializer]]...  
```
