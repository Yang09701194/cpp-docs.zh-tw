# SBYTE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

配置，並選擇性地將帶正負號的位元組的儲存空間初始化每個`initializer`。  也可當做型別規範型別是合法的任何地方。  
  
## 語法  
  
```  
  
name SBYTE initializer [[, initializer]]...  
```
