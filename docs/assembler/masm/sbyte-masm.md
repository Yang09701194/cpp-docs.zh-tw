# <a name="sbyte-masm"></a>SBYTE (MASM)

配置並選擇性地為每個*初始化運算式*初始化帶正負號的儲存區。 也可以當做類型為合法的任何位置的類型規範使用。

## <a name="syntax"></a>語法

> *命名*  **SBYTE** *初始化運算式*⟦ __，__ *初始化運算式*.。。⟧
