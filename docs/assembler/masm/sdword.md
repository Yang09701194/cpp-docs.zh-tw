# <a name="sdword"></a>SDWORD

配置並選擇性地為每個初始化*運算式*初始化儲存區的帶正負號雙字（4個位元組）。 也可以當做類型為合法的任何位置的類型規範使用。

## <a name="syntax"></a>語法

> *name* **SDWORD** *初始化運算式*⟦ __，__ *初始化運算式*.。。⟧
