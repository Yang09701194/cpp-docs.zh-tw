# SDWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

配置，並選擇性地將帶正負號的雙字 \(4 個位元組\) 的儲存體初始化每個`initializer`。  也可以當做型別規範型別是合法的任何地方。  
  
## 語法  
  
```  
  
name SDWORD initializer [[, initializer]]...  
```
