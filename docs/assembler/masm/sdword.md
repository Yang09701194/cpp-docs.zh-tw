# <a name="sdword"></a>SDWORD

配置並選擇性地針對每個初始化帶正負號的雙字組 （4 個位元組） 的儲存體`initializer`。 也可用來當做型別規範的類型是合法的任何位置。

## <a name="syntax"></a>語法

> 名稱 SDWORD 初始設定式 [[，初始設定式]]...
