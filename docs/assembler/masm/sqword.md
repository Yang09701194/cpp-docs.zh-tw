# <a name="sqword"></a>SQWORD

配置並選擇性地針對每個初始化帶正負號的 8 個位元組儲存體`initializer`。 也可用來當做型別規範的類型是合法的任何位置。

## <a name="syntax"></a>語法

> 名稱 SQWORD 初始設定式 [[，初始設定式]]...
