# SQWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

配置，並選擇性地將帶正負號的 8 個位元組儲存體初始化每個`initializer`。  也可以當做型別規範型別是合法的任何地方。  
  
## 語法  
  
```  
  
name SQWORD initializer [[, initializer]]...  
```
