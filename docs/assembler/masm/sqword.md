# <a name="sqword"></a>SQWORD
配置並選擇性地初始化帶正負號的 8 個位元組儲存每個`initializer`。 也可以用為類型規範的類型是合法的任何位置。  
  
## <a name="syntax"></a>語法  
  
```  
  
name SQWORD initializer [[, initializer]]...  
```
