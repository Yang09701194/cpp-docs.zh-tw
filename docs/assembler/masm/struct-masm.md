# STRUCT (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

宣告為結構型別，需要指定 *fielddeclarations*。  每個欄位必須是有效的資料定義。  相同的 [STRUC](../../assembler/masm/struc.md)。  
  
## 語法  
  
```  
  
   name STRUCT [[alignment]] [[, NONUNIQUE]]  
fielddeclarations  
name ENDS  
```
