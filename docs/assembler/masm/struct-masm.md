# <a name="struct-masm"></a>STRUCT (MASM)
宣告結構類型具有指定*fielddeclarations*。 每個欄位必須是有效的資料定義。 與相同[STRUC](../../assembler/masm/struc.md)。  
  
## <a name="syntax"></a>語法  
  
```  
  
   name STRUCT [[alignment]] [[, NONUNIQUE]]  
fielddeclarations  
name ENDS  
```
