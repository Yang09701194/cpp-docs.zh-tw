# <a name="struct"></a>STRUCT

宣告具有指定*欄位*宣告的結構類型。 每個欄位都必須是有效的資料定義。 與[STRUC](struc.md)相同。

## <a name="syntax"></a>語法

> *name* **結構**⟦*對齊*⟧⟦__，__ 非**唯一**⟧ \
> *欄位宣告*\
> *名稱***結束**

## <a name="remarks"></a>備註

開頭和結尾語句中的*name*引數必須相同。
