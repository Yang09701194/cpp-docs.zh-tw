# <a name="struct-masm"></a>STRUCT (MASM)

宣告具有指定的結構類型*欄位宣告*。 每個欄位必須是有效的資料定義。 與相同[STRUC](../../assembler/masm/struc.md)。

## <a name="syntax"></a>語法

> *名稱*結構 [[*對齊*]] [[，非唯一]]<br/>
> *欄位宣告*<br/>
> *名稱*結束
