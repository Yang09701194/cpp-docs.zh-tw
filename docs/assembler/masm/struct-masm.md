# <a name="struct-masm"></a>STRUCT (MASM)

宣告具有指定*欄位*宣告的結構類型。 每個欄位都必須是有效的資料定義。 與[STRUC](../../assembler/masm/struc.md)相同。

## <a name="syntax"></a>語法

> *name* **結構**⟦*對齊*⟧⟦ __，__ 非**唯一**的⟧*欄位*宣告\
> *名稱***結束**
