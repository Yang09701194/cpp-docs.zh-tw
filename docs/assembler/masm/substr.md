# <a name="substr"></a>SUBSTR

傳回字串的子*textitem*，從開始*位置*。 *Textitem*可以是常值的字串常數前面加上`%`，或巨集函式所傳回的字串。

## <a name="syntax"></a>語法

> 命名 SUBSTR textitem、 位置 [[，長度]]
