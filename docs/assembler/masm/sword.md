# <a name="sword"></a>SWORD

配置並選擇性地為每個初始化*運算式*初始化儲存區的已簽署字組（2個位元組）。 也可以當做類型為合法的任何位置的類型規範使用。

## <a name="syntax"></a>語法

> *name* **寶劍***初始化運算式*⟦ __，__ *初始化運算式*.。。⟧
