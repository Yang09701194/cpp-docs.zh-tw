# SWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

配置，並選擇性地將帶正負號的字 \(2 位元組\) 的儲存體初始化每個`initializer`。  也可當做型別規範型別是合法的任何地方。  
  
## 語法  
  
```  
  
name SWORD initializer [[, initializer]]...  
```
