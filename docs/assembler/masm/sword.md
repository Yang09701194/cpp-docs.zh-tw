# <a name="sword"></a>SWORD
配置並選擇性地初始化儲存體的帶正負號的單字 （2 個位元組） 為每個`initializer`。 也可用以做為類型指定型別是合法的任何位置。  
  
## <a name="syntax"></a>語法  
  
```  
  
name SWORD initializer [[, initializer]]...  
```
