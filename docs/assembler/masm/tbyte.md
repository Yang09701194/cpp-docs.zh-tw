# <a name="tbyte"></a>TBYTE

配置並選擇性地為每個初始化*運算式*初始化10個位元組的儲存區。 也可以當做類型為合法的任何位置的類型規範使用。

## <a name="syntax"></a>語法

> ⟦*名稱*⟧ **TBYTE** *初始化運算式*⟦ __，__ *初始化運算式*.。。⟧
