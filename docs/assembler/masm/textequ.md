# TEXTEQU
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

將指派給 *textitem* 到*名稱*。   *Textitem* 可以是常值字串常數，前面加上`%`，或巨集函式所傳回的字串。  
  
## 語法  
  
```  
  
name TEXTEQU [[textitem]]  
```
