# <a name="textequ"></a>TEXTEQU
指派*textitem*至*名稱*。 *Textitem*可以是常值的字串常數前面加上`%`，或巨集函式所傳回的字串。  
  
## <a name="syntax"></a>語法  
  
```  
  
name TEXTEQU [[textitem]]  
```
