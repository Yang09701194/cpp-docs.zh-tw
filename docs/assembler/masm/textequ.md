# <a name="textequ"></a>TEXTEQU

將*textitem*指派給*名稱*。 *Textitem*可以是常值字串、前面加上 `%`的常數，或宏函數所傳回的字串。

## <a name="syntax"></a>語法

> *name* **TEXTEQU** ⟦*textitem*⟧
