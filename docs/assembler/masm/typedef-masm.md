# <a name="typedef"></a>TYPEDEF

定義名為*name*的新類型或**PROTO** ，這相當於*type*或*protoDefinition*。

## <a name="syntax"></a>語法

> *名稱* **TYPEDEF** *類型* | **PROTO** *protoDefinition*
