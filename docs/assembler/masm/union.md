# <a name="union"></a>UNION
宣告一或多個資料類型的聯集。 *Fielddeclarations*必須是有效的資料定義。 省略[結束](../../assembler/masm/ends-masm.md)*名稱*索引標籤上的巢狀**UNION**定義。  
  
## <a name="syntax"></a>語法  
  
```  
  
      name   
      UNION [[alignment]] [[, NONUNIQUE]]  
   fielddeclarations  
[[name]] ENDS  
```
