# <a name="union"></a>UNION

宣告一或多個資料類型的聯集。 *欄位宣告*必須是有效的資料定義。 省略[結束](../../assembler/masm/ends-masm.md)*名稱*標籤上的巢狀**UNION**定義。

## <a name="syntax"></a>語法

> *名稱*聯集 [[*對齊*]] [[，非唯一]]<br/>
> *欄位宣告*<br/>
> [[*名稱*]] 結束
