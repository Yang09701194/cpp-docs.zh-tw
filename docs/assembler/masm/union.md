# <a name="union"></a>UNION

宣告一或多個資料類型的聯集。 *欄位*宣告必須是有效的資料定義。 省略嵌套聯**集**定義上的[結束](../../assembler/masm/ends-masm.md)*名稱*標籤。

## <a name="syntax"></a>語法

> *名稱聯***集**⟦*對齊*⟧⟦ __，__ 非**唯一**⟧ \
> *欄位*宣告\
> ⟦*名稱*⟧**結束**
