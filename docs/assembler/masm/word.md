# <a name="word"></a>WORD

Allocates and optionally initializes a word (2 bytes) of storage for each *initializer*. Can also be used as a type specifier anywhere a type is legal.

## <a name="syntax"></a>語法

> ⟦*name*⟧ **WORD** *initializer* ⟦ __,__ *initializer* ...⟧
