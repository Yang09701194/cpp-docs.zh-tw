# <a name="ymmword"></a>YMMWORD

用於具有 Intel Advanced Vector Extensions （AVX）指示的256位多媒體運算元。

## <a name="syntax"></a>語法

> **YMMWORD**

## <a name="remarks"></a>備註

**YMMWORD**的目的是要代表與 AVX 內建函式的 `__m256` 相同的類型。
