# <a name="ymmword"></a>YMMWORD
用於搭配 Intel Advanced Vector 延伸模組 (AVX) 指令的 256 位元多媒體運算元。  
  
## <a name="syntax"></a>語法  
  
```  
YMMWORD  
```  
  
## <a name="remarks"></a>備註  
 `YMMWORD` 用來代表相同的型別`__m256`AVX 內建函式。
