# <a name="ymmword"></a>YMMWORD

使用 256 位元多媒體運算元包含 Intel Advanced Vector Extensions (AVX) 指示。

## <a name="syntax"></a>語法

> YMMWORD

## <a name="remarks"></a>備註

`YMMWORD` 用來表示相同的型別`__m256`的 AVX 內建函式。
