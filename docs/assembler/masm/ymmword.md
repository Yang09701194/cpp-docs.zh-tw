# YMMWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

用於 256 位元的 Intel 進階向量擴充功能 \(AVX\) 指示多媒體的運算元。  
  
## 語法  
  
```  
YMMWORD  
```  
  
## 備註  
 `YMMWORD`用來表示相同的型別`__m256` AVX 的內建值。
