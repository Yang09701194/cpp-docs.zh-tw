# Date and Time: Database Support
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

自 4.0 版， MFC 資料庫程式撰寫使用 [COleDateTime](../atl-mfc-shared/reference/coledatetime-class.md) 和 [COleDateTimeSpan](../atl-mfc-shared/reference/coledatetimespan-class.md) 類別表示日期和時間資料。  這些類別，也可用來自動化，請從類別衍生 [COleVariant](../mfc/reference/colevariant-class.md)。  它們比 [CTime](../atl-mfc-shared/reference/ctime-class.md) 和 [CTimeSpan](../atl-mfc-shared/reference/ctimespan-class.md)提供更好的支援來管理的日期和時間資料。
