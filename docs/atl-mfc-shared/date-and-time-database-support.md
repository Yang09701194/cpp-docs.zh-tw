# <a name="date-and-time-database-support"></a>日期和時間：資料庫支援

從 4.0 版開始，MFC 資料庫程式設計的用法[COleDateTime](../atl-mfc-shared/reference/coledatetime-class.md)並[COleDateTimeSpan](../atl-mfc-shared/reference/coledatetimespan-class.md)類別來代表日期和時間資料。 這些類別，也用在自動化中，衍生自類別[COleVariant](../mfc/reference/colevariant-class.md)。 它們提供更佳的支援，讓您管理的日期和時間資料，相較於[CTime](../atl-mfc-shared/reference/ctime-class.md)並[CTimeSpan](../atl-mfc-shared/reference/ctimespan-class.md)。
