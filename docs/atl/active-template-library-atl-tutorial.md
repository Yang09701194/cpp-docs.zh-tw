# <a name="active-template-library-atl-tutorial"></a>Active Template Library (ATL) 教學課程
ATL 被為了簡化的程序建立有效率、 有彈性的輕量型控制項。 本教學課程將引導您完成建立 ActiveX 控制項，來示範許多 ATL 和 COM 的基本概念。  
  
 遵循此教學課程中，您將學習如何將控制項加入繪製圓形和填滿的多邊形的 ATL 專案。 然後，您會加入屬性以表示幾個邊的多邊形會有並建立在屬性變更時，更新控制項的繪圖程式碼。 然後會在網頁上，以回應事件的容易使用某些 VBScript 顯示控制項。  
  
 本教學課程分成七個步驟。 接下來的步驟取決於先前已完成的工作順序執行每個步驟。 在開始之前，您應該確認您有註冊您的特定電腦上的 ActiveX 元件所需的權限。 如果您正在執行 Visual Studio.NET 透過終端機服務連線，這是通常只有一項考量。  
  
-   [步驟 1： 建立專案](../atl/creating-the-project-atl-tutorial-part-1.md)  
  
-   [步驟 2： 將控制項加入至您的專案](../atl/adding-a-control-atl-tutorial-part-2.md)  
  
-   [步驟 3。將屬性加入至您的控制項](../atl/adding-a-property-to-the-control-atl-tutorial-part-3.md)  
  
-   [步驟 4： 變更控制項的繪圖程式碼](../atl/changing-the-drawing-code-atl-tutorial-part-4.md)  
  
-   [步驟 5： 加入事件](../atl/adding-an-event-atl-tutorial-part-5.md)  
  
-   [步驟 6： 加入屬性頁](../atl/adding-a-property-page-atl-tutorial-part-6.md)  
  
-   [步驟 7： 讓您在網頁上的控制項](../atl/putting-the-control-on-a-web-page-atl-tutorial-part-7.md)
