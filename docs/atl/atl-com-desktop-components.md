# <a name="atl-com-desktop-components"></a>ATL COM 桌面元件

ATL 參考記載了 Active Template Library (ATL), 這是一組以範本為C++基礎的類別, 可簡化元件物件模型 (COM) 物件的程式設計。 COM 是在 Windows 上建立和使用軟體元件的二進位規格。 為了充分利用 ATL, 強烈建議使用 COM 的熟悉程度。 如需 COM 的詳細資訊, 請參閱[元件物件模型 (COM)](/windows/win32/com/component-object-model--com--portal)。
