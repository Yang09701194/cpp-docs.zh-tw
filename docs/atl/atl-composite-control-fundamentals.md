# ATL 複合控制項基本概念
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

複合控制項是可以包含 ActiveX 控制項的型別 \(類似對話方塊\) 其他 ActiveX 控制項或視窗的控制項。  一旦複合控制項時，可以將它插入任何位置 ActiveX 控制項可以裝載。  
  
 ATL 專案精靈和 **加入類別** 對話方塊自動建立和實作複合控制項的程序，類似於執行應用程式精靈的結果建立 MFC 應用程式架構。  開發程序包含五個步驟:  
  
-   [建立 ATL 專案](../atl/reference/creating-an-atl-project.md)  
  
-   [插入複合控制項](../atl/inserting-a-composite-control.md)  
  
-   [修改 ATL 專案](../atl/modifying-the-atl-project.md)  
  
-   [將功能加入至複合控制項](../atl/adding-functionality-to-the-composite-control.md)  
  
-   [建置和測試 ATL 專案](../atl/building-and-testing-the-atl-project.md)
