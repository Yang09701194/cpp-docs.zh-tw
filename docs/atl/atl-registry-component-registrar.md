# <a name="atl-registry-component-registrar"></a>ATL 登錄元件 (登錄器)

ATL 註冊器通過自定義介面提供對系統註冊表的優化訪問。 註冊器是自由線程的,允許C++用戶端的靜態代碼連結。

> [!NOTE]
> ATL 註冊器的原始碼可在 atlmfc_include_atliface.h 中找到。
