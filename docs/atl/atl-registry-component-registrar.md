# <a name="atl-registry-component-registrar"></a>ATL 登錄元件 (登錄器)
ATL 登錄器最佳化存取系統登錄的自訂介面。 在註冊機構是 無限制執行緒，可讓靜態連結的程式碼針對 c + + 用戶端。  
  
> [!NOTE]
>  ATL 登錄器的原始程式碼位於 atlmfc\include\atliface.h。
