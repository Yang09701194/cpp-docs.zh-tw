# <a name="atl-window-classes"></a>ATL 視窗類別
ATL 包含數個類別，可讓您使用及實作 windows。 這些類別，其他 ATL 類別一樣，提供有效率的實作，不會造成您的程式碼的負荷。  
  
 本章節描述 ATL 視窗類別，並說明如何使用它們。

 [使用視窗](../atl/using-a-window.md)  
 討論如何使用`CWindow`管理視窗。

 討論訊息處理常式、 訊息對應，以及使用`CWindowImpl`。 包含 superclassing 和子類別化的詳細資訊。  
  
 [實作對話方塊](../atl/implementing-a-dialog-box.md)  
 討論加入對話方塊類別的兩個方法，並顯示程式碼範例。  
  
 [使用自主視窗](../atl/using-contained-windows.md)  
 討論包含的 windows 在 ATL，也就是其訊息委派至容器物件，而不是處理它們自己的類別中的 windows。

 討論 ATL 中的視窗 traits 類別 這些類別提供標準化的樣式，用於建立視窗物件的簡單方法。
