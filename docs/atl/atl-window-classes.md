# <a name="atl-window-classes"></a>ATL 視窗類別

ATL 包含可讓您使用，並實作視窗的數個類別。 這些類別，例如其他 ATL 類別，提供不會造成額外負荷，在您的程式碼的有效實作。

本章節描述 ATL 視窗類別，並說明如何使用它們。

[ATL 視窗類別簡介](../atl/introduction-to-atl-window-classes.md)<br/>

[使用視窗](../atl/using-a-window.md)<br/>
討論如何使用`CWindow`管理視窗。

[實作視窗](../atl/implementing-a-window.md)<br/>
討論訊息處理常式、 訊息對應，以及使用`CWindowImpl`。 包含有關 superclassing 及子類別化。

[實作對話方塊](../atl/implementing-a-dialog-box.md)<br/>
討論兩種方法，加入對話方塊類別，並顯示的程式碼範例。

[使用自主視窗](../atl/using-contained-windows.md)<br/>
討論包含的 windows 在 ATL，也就是其訊息委派至容器物件，而不是處理它們自己的類別中的 windows。

[了解視窗特性](../atl/understanding-window-traits.md)<br/>
討論在 ATL 中的視窗 traits 類別 這些類別提供簡單的方法標準化用於建立視窗物件的樣式。
