# <a name="building-and-testing-the-atl-project"></a>建置和測試 ATL 專案

中所述[插入複合控制項](../atl/inserting-a-composite-control.md)，其中一個專案的初始元件是裝載您新的複合控制項的預設 HTML 網頁。 您完成修改複合控制項之後，請按一下**建置方案**或是**重建方案**從**建置**功能表。 專案建置成功後，載入的 HTML 網頁，位於您的複合控制項專案的根目錄到 Internet Explorer 或另一個瀏覽器，並測試您的控制項的功能。

您也可以測試您使用測試容器工具或任何其他應用程式可裝載 ActiveX 控制項的複合控制項。 如需測試容器存取方法的詳細資訊，請參閱 [以測試容器測試屬性和事件](../mfc/testing-properties-and-events-with-test-container.md) 。
