# Building and Testing the ATL Project
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

依照 [插入複合控制項](../atl/inserting-a-composite-control.md)中所討論，其中一個專案的初始元件是裝載您的新複合控制項的預設 HTML 網頁。  在您完成修改複合控制項後，請按一下 \[**建置方案**\] 或 **重建方案** 從 \[**建置**\] 功能表。  在成功建置專案，裝載 HTML 網頁，位於您的複合控制項專案根目錄，到 Internet Explorer 或其他瀏覽器並測試控制項的功能。  
  
 使用測試容器工具，您也可以測試您的複合控制項，或是可以裝載 ActiveX 控制項的其他應用程式。  如需存取測試容器的詳細資訊，請參閱[用測試容器測試屬性和事件](../mfc/testing-properties-and-events-with-test-container.md)。
