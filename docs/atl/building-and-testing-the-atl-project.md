# <a name="building-and-testing-the-atl-project"></a>建置和測試 ATL 專案
中所述[插入複合控制項](../atl/inserting-a-composite-control.md)，其中一個專案的初始元件會裝載新的複合控制項的預設 HTML 網頁。 完成修改複合控制項之後，請按一下**建置方案**或**重建方案**從**建置**功能表。 一旦成功建置專案，載入 HTML 網頁，位於複合控制項專案的根目錄中到 Internet Explorer 或其他瀏覽器，並測試您的控制項的功能。  
  
 您也可以測試使用測試容器工具或其他任何可以裝載 ActiveX 控制項的應用程式的複合控制項。 如需測試容器存取方法的詳細資訊，請參閱 [以測試容器測試屬性和事件](../mfc/testing-properties-and-events-with-test-container.md) 。
