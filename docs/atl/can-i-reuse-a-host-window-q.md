# Can I Reuse a Host Window?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

建議您不要重複使用主視窗。  若要確保程式碼的穩定性，您應該繫結的存留期 \(Lifetime\) 主視窗至單一控制項的存留期。
