# Class Factories Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下列類別會實作或支援 Class Factory:  
  
-   [CComClassFactory](../atl/reference/ccomclassfactory-class.md) 為物件建立提供預設的 Class Factory。  
  
-   [CComClassFactory2](../atl/reference/ccomclassfactory2-class.md) 控制項透過授權物件建立。  
  
-   [CComClassFactoryAutoThread](../atl/reference/ccomclassfactoryautothread-class.md) 在多個執行緒集區的 Apartment 允許建立物件。  
  
-   [CComClassFactorySingleton](../atl/reference/ccomclassfactorysingleton-class.md) 建立單一物件。  
  
-   [CComCoClass](../atl/reference/ccomcoclass-class.md) 定義物件的 Class Factory。
