# Creating an ATL DHTML Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ATL 控制項精靈 Automation 建立 DHTML 控制項的程序。  它會產生必要的資源檔，包括含有範例程式碼的 HTML 檔。  
  
#### 建立 ATL DHTML 控制項  
  
1.  遵循 [建立 ATL 專案](../atl/reference/creating-an-atl-project.md)的步驟。  
  
2.  在 \[類別檢視\] 中，以滑鼠右鍵按一下專案節點，指向  **新增**，然後按一下捷徑功能表上的 **加入類別** 。  在 **加入類別** 對話方塊中，按兩下 [ATL 控制項精靈](../atl/reference/atl-control-wizard.md)。  在 ATL 控制項精靈中，按一下 \[**選項**\] 索引標籤並選取 **DHTML control**。  
  
 您現在可以 [測試預設控制項。](../atl/testing-the-atl-dhtml-control.md)。
