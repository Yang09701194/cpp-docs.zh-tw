# Data Transfer Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下列類別可支援傳送資料的各種型別:  
  
-   [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) 使用擷取和設定資料的標準格式的支援制式資料傳輸。  控制代碼資料變更告知會嘗試連接的通知接收。  
  
-   [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md) 允許非同步 Moniker 與您的物件傳送和收到非同步傳送資料的資訊。
