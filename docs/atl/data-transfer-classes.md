# <a name="data-transfer-classes"></a>資料傳輸類別
下列類別會支援各種類型的資料傳輸：  
  
-   [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md)支援制式資料傳輸使用標準格式來擷取及設定資料。 處理資料變更通知管理通知接收的連線。  
  
-   [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md)可讓非同步 moniker 來傳送和接收非同步資料傳輸與您物件的相關資訊。
