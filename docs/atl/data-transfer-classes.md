# <a name="data-transfer-classes"></a>資料傳輸類別

下列類別會支援各種類型的資料傳輸：

- [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md)支援制式資料傳輸所使用的標準格式擷取及設定資料。 控制代碼的資料變更通知管理通知接收的連接。

- [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md)允許非同步 moniker 來傳送和接收非同步資料傳送進出您的物件相關的資訊。
