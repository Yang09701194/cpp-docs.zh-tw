# <a name="enumerators-and-collections-classes"></a>列舉值和集合類別
下列類別會提供支援 COM 集合和列舉型別：  
  
-   [CComEnum](../atl/reference/ccomenum-class.md)定義 COM 列舉值物件的陣列為基礎。  
  
-   [CComEnumImpl](../atl/reference/ccomenumimpl-class.md)提供列舉的項目儲存在陣列中的 COM 列舉程式介面的實作。  
  
-   [CComEnumOnSTL](../atl/reference/ccomenumonstl-class.md)定義 COM 列舉程式物件會根據 c + + 標準程式庫集合。  
  
-   [IEnumOnSTLImpl](../atl/reference/ienumonstlimpl-class.md)提供列舉的項目中的 c + + 標準程式庫相容容器的儲存位置的 COM 列舉程式介面的實作。  
  
-   [ICollectionOnSTLImpl](../atl/reference/icollectiononstlimpl-class.md)提供的實作**計數**，**項目**，和`_NewEnum`集合介面的屬性。
