下列類別指出如何處理錯誤資訊:

- [Isupporterrorinfoimpl 當成基類](../atl/reference/isupporterrorinfoimpl-class.md)判斷物件是否支援[IErrorInfo](/windows/win32/api/oaidl/nn-oaidl-ierrorinfo)介面。 `IErrorInfo`允許將錯誤資訊傳播回用戶端。
