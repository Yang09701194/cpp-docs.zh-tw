# <a name="error-information-classes"></a>錯誤資訊類別
下列類別會指出如何處理資訊時發生錯誤：  
  
-   [ISupportErrorInfoImpl](../atl/reference/isupporterrorinfoimpl-class.md)判斷物件是否支援[IErrorInfo](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-ierrorinfo)介面。 `IErrorInfo` 允許傳播回用戶端的資訊時發生錯誤。
