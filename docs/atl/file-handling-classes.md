# File Handling Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

這些類別會提供管理檔案、暫存檔及記憶體對應檔案的方法。  
  
-   [CAtlFile](../atl/reference/catlfile-class.md) 這個類別會在檔案處理 API 提供的視窗周圍的小型包裝函式。  
  
-   [CAtlFileMapping](../atl/reference/catlfilemapping-class.md) 這個類別表示記憶體對應檔案，將轉型運算子套用至 [CAtlFileMappingBase](../atl/reference/catlfilemappingbase-class.md)方法。  
  
-   [CAtlFileMappingBase](../atl/reference/catlfilemappingbase-class.md) 這個類別表示記憶體對應檔案。  
  
-   [CAtlTemporaryFile](../atl/reference/catltemporaryfile-class.md) 這個類別提供方法來為暫存檔的建立和使用。
