# <a name="interface-pointers-classes"></a>介面指標類別
下列類別會管理指定的介面指標：  
  
-   [CComPtr](../atl/reference/ccomptr-class.md)執行自動參考計數。  
  
-   [CComQIPtr](../atl/reference/ccomqiptr-class.md)類似`CComPtr`，但也會執行自動查詢的介面。  
  
-   [CInterfaceArray](../atl/reference/cinterfacearray-class.md)提供方法建構的 COM 陣列時非常實用的介面指標。  
  
-   [CInterfaceList](../atl/reference/cinterfacelist-class.md)有用建構 COM 的清單時提供方法的介面指標。  
  
-   [CComGITPtr](../atl/reference/ccomgitptr-class.md)提供介面指標，以及全域介面表 (GIT) 的處理方法。
