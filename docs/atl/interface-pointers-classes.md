# Interface Pointers Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下列類別會管理特定介面指標:  
  
-   [CComPtr](../atl/reference/ccomptr-class.md) 執行自動參考計數。  
  
-   [CComQIPtr](../atl/reference/ccomqiptr-class.md) 類似， `CComPtr`，還會執行自動查詢介面。  
  
-   以建構一個陣列 COM 介面指標時，[CInterfaceArray](../atl/reference/cinterfacearray-class.md) 提供有用的方法。  
  
-   以建構清單 COM 介面指標時，[CInterfaceList](../atl/reference/cinterfacelist-class.md) 提供有用的方法。  
  
-   [CComGITPtr](../atl/reference/ccomgitptr-class.md) 用於處理方法提供介面指標和全域介面表 \(GIT\)。
