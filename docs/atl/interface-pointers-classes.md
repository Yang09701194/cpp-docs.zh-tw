# <a name="interface-pointers-classes"></a>介面指標類別

下列類別來管理指定的介面指標：

- [CComPtr](../atl/reference/ccomptr-class.md)執行自動參考計數。

- [CComQIPtr](../atl/reference/ccomqiptr-class.md)類似於`CComPtr`，但也會執行自動查詢的介面。

- [CInterfaceArray](../atl/reference/cinterfacearray-class.md)提供方法建構的 COM 陣列時非常實用的介面指標。

- [CInterfaceList](../atl/reference/cinterfacelist-class.md)提供方法有用時建構一份 COM 介面指標。

- [CComGITPtr](../atl/reference/ccomgitptr-class.md)提供介面指標，以及全域介面表 (GIT) 的方法來處理。
