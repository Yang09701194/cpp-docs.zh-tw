# <a name="introduction-to-atl"></a>ATL 簡介

ATL 是 Active Template Library, 這是一組樣板型C++類別, 可讓您輕鬆地建立小型、快速的元件物件模型 (COM) 物件。 它具有重要 COM 功能的特殊支援, 包括: [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown)、 [IClassFactory](/windows/win32/api/unknwnbase/nn-unknwnbase-iclassfactory)、 [IClassFactory2](/windows/win32/api/ocidl/nn-ocidl-iclassfactory2)和`IDispatch`、雙重介面、標準 COM 列舉值介面、連接點、卸載介面和 ActiveX 控制項。

ATL 程式碼可以用來建立單一執行緒物件、單元模型物件、無限制執行緒的模型物件, 或無限制執行緒和單元模型物件。

本節涵蓋的主題包括:

- 範本連結[庫](../atl/using-a-template-library.md)與標準程式庫的不同之處。

- 您[可以和無法使用 ATL 進行的](../atl/scope-of-atl.md)動作。

- [在 ATL 和 MFC 之間選擇的建議](../atl/recommendations-for-choosing-between-atl-and-mfc.md)。
