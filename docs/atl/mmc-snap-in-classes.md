# MMC Snap-In Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下列類別會提供開發 Microsoft Management Console \(MMC\) 嵌入式管理單元的元件支援:  
  
-   [CSnapInItemImpl](../atl/reference/csnapinitemimpl-class.md) 實作嵌入式管理單元的物件 \(例如，將功能表項目和工具列以及將變更這個嵌入式管理單元的節點順序為適當的處理函式。  
  
-   [CSnapInPropertyPageImpl](../atl/reference/csnapinpropertypageimpl-class.md) 實作嵌入式管理單元屬性頁面物件。
