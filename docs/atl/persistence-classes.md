# Persistence Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下列類別會實作物件保存:  
  
-   [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) 允許用戶端載入和儲存這個物件的屬性包。  
  
-   [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md) 允許用戶端載入和儲存物件的持續性資料至資料流。  
  
-   [IPersistStorageImpl](../atl/reference/ipersiststorageimpl-class.md) 允許用戶端載入和儲存物件的持續性資料儲存體中。
