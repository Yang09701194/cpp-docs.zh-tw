# <a name="persistence-classes"></a>永續性類別
下列類別會實作物件的持續性：  
  
-   [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md)可讓用戶端，以載入和儲存物件的屬性至屬性包。  
  
-   [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md)可讓用戶端，以載入和儲存的資料流物件的持續性資料。  
  
-   [IPersistStorageImpl](../atl/reference/ipersiststorageimpl-class.md)允許載入及物件的持續性資料儲存到儲存體用戶端。
