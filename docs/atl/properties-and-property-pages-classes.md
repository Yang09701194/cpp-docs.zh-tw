# Properties and Property Pages Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下列類別可支援屬性和屬性頁:  
  
-   [CComDispatchDriver](../Topic/CComDispatchDriver.md) 傳遞 `IDispatch` 指標擷取或設定物件的屬性。  
  
-   [CStockPropImpl](../atl/reference/cstockpropimpl-class.md) 實作 ATL 支援的內建屬性。  
  
-   [IPerPropertyBrowsingImpl](../atl/reference/iperpropertybrowsingimpl-class.md) 存取物件之屬性頁的詳細資訊。  
  
-   [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) 處於用戶端所提供的屬性包中儲存的物件屬性。  
  
-   [IPropertyPageImpl](../atl/reference/ipropertypageimpl-class.md) 處理屬性工作表內的特定屬性頁。  
  
-   [IPropertyPage2Impl](../atl/reference/ipropertypage2impl-class.md) 類似， `IPropertyPageImpl`，但也可以允許用戶端選取  屬性頁上的特定屬性。  
  
-   [ISpecifyPropertyPagesImpl](../atl/reference/ispecifypropertypagesimpl-class.md) 衍生物件支援的屬性頁的 CLSID。
