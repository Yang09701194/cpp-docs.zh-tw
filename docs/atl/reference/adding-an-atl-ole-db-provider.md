# 加入 ATL OLE DB 提供者
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

使用這個精靈來將 ATL OLE DB 提供者 \(Provider\) 加入至專案。  ATL OLE DB 提供者是由資料來源、工作階段 \(Session\)、命令及資料列集 \(Rowset\) 類別 \(Class\) 組成。  專案必須建立為 ATL COM 應用程式。  
  
### 若要將 ATL OLE DB 提供者加入至您的專案  
  
1.  在 \[類別檢視\] 中以滑鼠右鍵按一下專案。  按一下捷徑功能表上的 \[**加入**\]，然後按一下 \[**加入類別**\]。  
  
2.  在 Visual C\+\+ 資料夾中，按兩下 \[**ATL OLE DB 提供者**\] 圖示或選取它，然後按一下 \[**開啟**\]。  
  
     接著就會開啟 ATL OLE DB 提供者精靈。  
  
3.  定義如 [ATL OLE DB 提供者精靈](../../atl/reference/atl-ole-db-provider-wizard.md)所描述的設定。  
  
4.  按一下 \[**完成**\] 關閉精靈，這會將剛建立的 OLE DB 提供者程式碼插入至您的專案中。
