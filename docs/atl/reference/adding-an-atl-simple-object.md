# <a name="adding-an-atl-simple-object"></a>新增 ATL 簡單物件

若要將 ATL (Active Template Library) 物件新增至您的專案，您的專案必須已建立以 ATL 應用程式，或包含 ATL 支援的 MFC 應用程式。 您可以使用 [ATL 專案精靈](../../atl/reference/atl-project-wizard.md)建立 ATL 應用程式，或[將 ALT 物件新增至 MFC 應用程式](../../mfc/reference/adding-atl-support-to-your-mfc-project.md)，以實作 MFC 應用程式的 ATL 支援。

當您第一次建立，或稍後新增使用時，您可以為新的 ATL 物件定義 COM 介面[實作介面](../../ide/implement-interface-wizard.md)命令**類別檢視**快顯功能表。

## <a name="to-add-an-atl-simple-object-to-your-atl-com-project"></a>加入 ATL COM 專案新增 ATL 簡單物件

1. 採用**方案總管**或[類別檢視](/visualstudio/ide/viewing-the-structure-of-code)，以滑鼠右鍵按一下您要新增 ATL 簡單物件專案的名稱。

1. 從捷徑功能表按一下 [新增]，然後按一下 [新增類別]。

1. 中[加入類別](../../ide/add-class-dialog-box.md)對話方塊中，於**範本**窗格中，按一下  **ATL 簡單物件**，然後按一下**開啟**顯示[ATL 簡單物件精靈](../../atl/reference/atl-simple-object-wizard.md)。

1. 設定其他選項，為您的專案[選項](../../atl/reference/options-atl-simple-object-wizard.md)頁**ATL 簡單物件**精靈。

1. 按一下 **完成**將物件加入至您的專案。
