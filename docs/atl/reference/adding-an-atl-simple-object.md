# <a name="adding-an-atl-simple-object"></a>加入 ATL 簡單物件
將 ATL (Active Template Library) 物件新增至您的專案，您的專案必須已建立為 ATL 應用程式，或包含 ATL 支援的 MFC 應用程式。 您可以使用[ATL 專案精靈](../../atl/reference/atl-project-wizard.md)建立 ATL 應用程式，或[MFC 應用程式中加入 ATL 物件](../../mfc/reference/adding-atl-support-to-your-mfc-project.md)實作 ATL 支援的 MFC 應用程式。  
  
 當您第一次建立，或稍後再新增使用，您可以為新的 ATL 物件定義 COM 介面[實作介面](../../ide/implement-interface-wizard.md)從類別檢視捷徑功能表命令。  
  
### <a name="to-add-an-atl-simple-object-to-your-atl-com-project"></a>將 ATL 簡單物件加入至您的 ATL COM 專案  
  
1.  在**方案總管 中**或[類別檢視](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925)，以滑鼠右鍵按一下您要將 ATL 簡單物件加入的專案的名稱。  
  
2.  從捷徑功能表，按一下 **新增**，然後按一下 **加入類別**。  
  
3.  在[加入類別](../../ide/add-class-dialog-box.md)對話方塊，在 [範本] 窗格中，按一下**ATL 簡單物件**，然後按一下 **開啟**顯示[ATL 簡單物件精靈](../../atl/reference/atl-simple-object-wizard.md)。  
  
4.  設定專案的其他選項[選項](../../atl/reference/options-atl-simple-object-wizard.md)ATL 簡單物件精靈 頁面。  
  
5.  按一下 **完成**將物件加入至您的專案。
