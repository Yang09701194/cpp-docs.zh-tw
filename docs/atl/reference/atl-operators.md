# ATL Operators Alphabetical Reference
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

本節包含 ATL 全域運算子的參考主題。  
  
|運算子|描述|  
|---------|--------|  
|[運算子\=\=](../Topic/operator%20==%20\(ATL\).md)|比較兩個物件 `CSid` 或 `SID` 結構是否相等。|  
|[運算子\! \=](../Topic/operator%20!=%20\(ATL\).md)|比較兩個物件 `CSid` 或 `SID` 結構是否不相等。|  
|[運算子\<](../Topic/operator%20%3C%20\(ATL\).md)|測試，如果 `CSid` 物件或 `SID` 結構左側的比 `CSid` 物件或 `SID` 結構小於右邊 \(適用於 STL 相容\)。|  
|[運算子\>](../Topic/operator%20%3E%20\(ATL\).md)|測試，如果 `CSid` 物件或 `SID` 結構左側的比 `CSid` 物件或 `SID` 結構大於右邊 \(適用於 STL 相容\)。|  
|[運算子\<\=](../Topic/operator%20%3C=%20\(ATL\).md)|測試，如果 `CSid` 物件或 `SID` 結構左側的小於或等於 `CSid` 物件或 `SID` 結構右邊 \(適用於 STL 相容\)。|  
|[運算子\>\=](../Topic/operator%20%3E=%20\(ATL\).md)|測試，如果 `CSid` 物件或 `SID` 結構在運算子左方的執行個體是否大於或等於 `CSid` 物件或 `SID` 結構右邊 \(適用於 STL 相容\)。|  
  
 這些運算子都在檔案 atlsecurity.h 中定義。
