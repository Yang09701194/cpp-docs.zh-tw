Active Template Library 包含下列 Typedef。  
  
|||  
|-|-|  
|[\_ATL\_BASE\_MODULE](../Topic/_ATL_BASE_MODULE.md)|定義為根據的 [\_ATL\_BASE\_MODULE70](../../atl/reference/atl-base-module70-structure.md)typedef。|  
|[\_ATL\_COM\_MODULE](../Topic/_ATL_COM_MODULE.md)|定義為根據的 [\_ATL\_COM\_MODULE70](../../atl/reference/atl-com-module70-structure.md)typedef。|

|[CComDispatchDriver](../Topic/CComDispatchDriver.md)|這個類別會處理 COM 介面指標。|  
|[CComGlobalsThreadModel](../Topic/CComGlobalsThreadModel.md)|呼叫適當的執行緒模型，不論使用的執行緒模型。|  
|[CComObjectThreadModel](../Topic/CComObjectThreadModel.md)|呼叫適當的執行緒模型，不論使用的執行緒模型。|

|[CPathA](../Topic/CPathA.md)|[CPathT](../../atl/reference/cpatht-class.md) 的特製化使用 `CStringA`的。|  
|[CPathW](../Topic/CPathW.md)|[CPathT](../../atl/reference/cpatht-class.md) 的特製化使用 `CStringW`的。|  
|[CSimpleValArray](../Topic/CSimpleValArray.md)|表示儲存的簡單型別陣列。|  
|[DefaultThreadTraits](../../atl/reference/atl-typedefs.md)|預設執行緒的類別。|  
|[LPCURL](../Topic/LPCURL.md)|為常數 [卷毛](../../atl/reference/curl-class.md) 物件的指標。|  
|[LPURL](../Topic/LPURL.md)|為 [卷毛](../../atl/reference/curl-class.md) 物件的指標。|
