# <a name="catldebuginterfacesmodule-class"></a>CAtlDebugInterfacesModule 類別

這個類別提供支援偵錯介面。

## <a name="syntax"></a>語法

```
class CAtlDebugInterfacesModule
```

## <a name="remarks"></a>備註

`CAtlDebugInterfacesModule` 提供偵錯介面所需的支援。 它包含在定義符號的任何專案[_ATL_DEBUG_QI](debugging-and-error-reporting-macros.md#_atl_debug_qi)。

## <a name="requirements"></a>需求

**標頭：** atlbase.h
