# CAtlDebugInterfacesModule Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這個類別會提供偵錯介面的支援。  
  
## 語法  
  
```  
  
class CAtlDebugInterfacesModule  
  
```  
  
## 備註  
 `CAtlDebugInterfacesModule` 提供對於偵錯介面所需要的支援。  它會定義符號 [\_ATL\_DEBUG\_QI](../Topic/_ATL_DEBUG_QI.md)的所有專案中。

 **Header:** atlbase.h
