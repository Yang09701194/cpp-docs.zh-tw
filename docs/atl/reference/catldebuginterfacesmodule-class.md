# <a name="catldebuginterfacesmodule-class"></a>CAtlDebugInterfacesModule 類別
這個類別提供支援偵錯介面。  
  
## <a name="syntax"></a>語法  
  
```
class CAtlDebugInterfacesModule
```  
  
## <a name="remarks"></a>備註  
 `CAtlDebugInterfacesModule`提供偵錯介面所需的支援。 包含在任何專案中定義的符號[_ATL_DEBUG_QI](http://msdn.microsoft.com/library/4f1feddf-1469-42ad-a2f7-5a816d1c0bc5)。
