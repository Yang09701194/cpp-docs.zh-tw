# <a name="catldebuginterfacesmodule-class"></a>CAtlDebugInterfacesModule 類別

這個類別會提供對偵測介面的支援。

## <a name="syntax"></a>語法

```cpp
class CAtlDebugInterfacesModule
```

## <a name="remarks"></a>備註

`CAtlDebugInterfacesModule`提供偵錯工具所需的支援。 它包含在定義符號[_ATL_DEBUG_QI](debugging-and-error-reporting-macros.md#_atl_debug_qi)的任何專案中。

## <a name="requirements"></a>需求

**標頭：** atlbase.h。h
