# Composite Control Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這些巨集定義事件接收對應和項目。  
  
|||  
|-|-|  
|[BEGIN\_SINK\_MAP](../Topic/BEGIN_SINK_MAP.md)|標記事件接收對應的開頭複合控制項的。|  
|[END\_SINK\_MAP](../Topic/END_SINK_MAP.md)|標記事件接收對應的結束複合控制項的。|  
|[SINK\_ENTRY](../Topic/SINK_ENTRY.md)|為事件接收對應的項目。|  
|[SINK\_ENTRY\_EX](../Topic/SINK_ENTRY_EX.md)|為事件接收對應的項目與其他參數。|  
|[SINK\_ENTRY\_INFO](../Topic/SINK_ENTRY_INFO.md)|為事件接收對應以手動方式提供的型別資訊 [IDispEventSimpleImpl](../../atl/reference/idispeventsimpleimpl-class.md)與搭配使用。|
