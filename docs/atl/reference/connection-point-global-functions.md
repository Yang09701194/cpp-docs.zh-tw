# Connection Point Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這些函式用於連接點的支援並接收對應。  
  
> [!IMPORTANT]
>  下表中列出的函式不能用於 [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)]執行的應用程式。  
  
|||  
|-|-|  
|[AtlAdvise](../Topic/AtlAdvise.md)|建立物件的連接點和用戶端的接收之間的連接。|  
|[AtlUnadvise](../Topic/AtlUnadvise.md)|結束藉由 `AtlAdvise`已建立的連接。|  
|[AtlAdviseSinkMap](../Topic/AtlAdviseSinkMap.md)|建議您在或事件接收對應的 unadvises 輸入。|
