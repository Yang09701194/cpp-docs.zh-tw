# <a name="creating-an-atl-project"></a>建立的 ATL 專案
若要建立的 ATL 專案最簡單的方式是使用 ATL 專案精靈中的 Win32 專案資料夾**新增專案 對話方塊**。  
  
### <a name="to-create-an-atl-project-using-the-atl-project-wizard"></a>若要建立使用 ATL 專案精靈的 ATL 專案  
  
1.  請依照下列主題中的指示[使用 Visual c + + 應用程式精靈建立專案](../../ide/creating-desktop-projects-by-using-application-wizards.md)。  
  
2.  選取**ATL 專案**圖示以開啟 ATL 專案精靈的 範本 窗格中。  
  
3.  定義使用的應用程式設定[應用程式設定](../../atl/reference/application-settings-atl-project-wizard.md)頁面`ATL Project Wizard`。  
  
    > [!NOTE]
    >  若要保留精靈的預設值，請略過此步驟。  
  
4.  按一下**完成**来關閉精靈並在開發環境中開啟新專案。  
  
 您的專案建立之後，您可以檢視中建立的檔案**方案總管 中**。 如需精靈建立之專案檔案的詳細資訊，請參閱專案所產生的 ReadMe.txt 檔案。 如需檔案類型的詳細資訊，請參閱[Visual c + + 專案建立的檔案類型](../../ide/file-types-created-for-visual-cpp-projects.md)。 如需新的 ATL 專案，以及如何變更這些設定的詳細資訊，請參閱[預設 ATL 專案組態](../../atl/reference/default-atl-project-configurations.md)。
