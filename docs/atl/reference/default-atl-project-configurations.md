# <a name="default-atl-project-configurations"></a>預設的 ATL 專案組態
ATL 專案精靈會根據預設，建立兩個專案組態：  
  
|組態|字元集|ATL 用途|  
|-------------------|-------------------|----------------|  
|發行|MBCS|DLL|  
|偵錯|MBCS|DLL|  
  
 **字元集**， **ATL 用法**，都可以變更在**專案設定**對話方塊底下**一般** 索引標籤。您也可以加入自己使用 Configuration Manager 的組態。 如需詳細資訊，請參閱[組建組態](/visualstudio/ide/understanding-build-configurations)。
