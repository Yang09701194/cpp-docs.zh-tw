# IAtlAutoThreadModule Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這個類別表示介面的 `CreateInstance` 方法。  
  
> [!IMPORTANT]
>  這個類別和其成員不能用於 [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)]執行的應用程式。  
  
## 語法  
  
```  
  
__interface IAtlAutoThreadModule  
  
```  
  
## 備註  
 類別 [CAtlAutoThreadModuleT](../../atl/reference/catlautothreadmodulet-class.md) 從 `IAtlAutoThreadModule`衍生自類別，並使用它來建立物件和擷取介面指標的程式碼。

 **Header:** atlbase.h
