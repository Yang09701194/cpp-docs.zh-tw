# <a name="iatlautothreadmodule-class"></a>IAtlAutoThreadModule 類別
此類別代表介面`CreateInstance`方法。  
  
> [!IMPORTANT]
>  這個類別及其成員不能在 Windows 執行階段中執行的應用程式。  
  
## <a name="syntax"></a>語法  
  
```
__interface IAtlAutoThreadModule
```  
  
## <a name="remarks"></a>備註  
 類別[CAtlAutoThreadModuleT](../../atl/reference/catlautothreadmodulet-class.md)衍生自`IAtlAutoThreadModule`，使用它來提供程式碼建立的物件，擷取介面指標。  
  
## <a name="requirements"></a>需求  
 **標頭：** atlbase.h
