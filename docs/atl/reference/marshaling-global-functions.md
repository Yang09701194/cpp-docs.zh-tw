# Marshaling Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這些函式進行封送處理和轉換封送處理資料的支援型別的介面指標。  
  
> [!IMPORTANT]
>  下表中列出的函式不能用於 [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)]執行的應用程式。  
  
|||  
|-|-|  
|[AtlFreeMarshalStream](../Topic/AtlFreeMarshalStream.md)|釋放封送處理資料和 `IStream` 指標。|  
|[AtlMarshalPtrInProc](../Topic/AtlMarshalPtrInProc.md)|建立新的資料流物件並將其封送處理指定的介面指標。|  
|[AtlUnmarshalPtr](../Topic/AtlUnmarshalPtr.md)|轉換資料流的封送處理資料到介面指標。|
