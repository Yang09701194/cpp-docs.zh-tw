# Object Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這些巨集定義物件對應和項目。  
  
|||  
|-|-|  
|[DECLARE\_OBJECT\_DESCRIPTION](../Topic/DECLARE_OBJECT_DESCRIPTION.md)|可讓您指定類別物件的文字描述，都會輸入物件對應。|  
|[OBJECT\_ENTRY\_AUTO](../Topic/OBJECT_ENTRY_AUTO.md)|輸入 ATL 物件至物件的對應，更新登錄，並建立物件的執行個體。|  
|[OBJECT\_ENTRY\_NON\_CREATEABLE\_EX\_AUTO](../Topic/OBJECT_ENTRY_NON_CREATEABLE_EX_AUTO.md)|可讓您指定應該註冊和初始化物件，不過，它不應該是外部可透過 `CoCreateInstance`。|
