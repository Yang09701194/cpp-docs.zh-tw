# <a name="options-atl-property-page-wizard"></a>選項, ATL 屬性頁精靈
使用精靈的這個頁面來定義您要建立的屬性頁的執行緒模型和彙總層級。  
  
 **執行緒模型**  
 指定使用屬性頁的執行緒模型。  
  
 請參閱[指定專案的執行緒模型](../../atl/specifying-the-threading-model-for-a-project-atl.md)如需詳細資訊。  
  
|選項|描述|  
|------------|-----------------|  
|`Single`|只有在主要 COM 執行緒中執行的屬性頁。|  
|**Apartment**|可以在任何單一執行緒 apartment 中建立屬性頁。 預設值。|  
  
 **彙總**  
 加入您所建立的屬性頁的彙總支援。 請參閱[彙總](../../atl/aggregation.md)如需詳細資訊。  
  
|選項|描述|  
|------------|-----------------|  
|**是**|建立可以彙總的屬性頁。|  
|**No**|建立不能彙總的屬性頁。|  
|**只有**|建立只具現化透過彙總的屬性頁。|
