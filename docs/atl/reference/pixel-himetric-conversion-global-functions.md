# Pixel/HIMETRIC Conversion Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這些函式提供支援來回像素和 HIMETRIC 單位。  
  
> [!IMPORTANT]
>  下表中列出的函式不能用於 [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)]執行的應用程式。  
  
|||  
|-|-|  
|[AtlHiMetricToPixel](../Topic/AtlHiMetricToPixel.md)|轉換 HIMETRIC 單位 \(每單位為 0.01 公釐為單位\)。|  
|[AtlPixelToHiMetric](../Topic/AtlPixelToHiMetric.md)|為 HIMETRIC 單位轉換為像素 \(每單位為 0.01 公釐\)。|
