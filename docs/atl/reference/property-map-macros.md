# Property Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這些巨集定義屬性對應和項目。  
  
|||  
|-|-|  
|[BEGIN\_PROP\_MAP](../Topic/BEGIN_PROP_MAP.md)|標記 ATL 屬性對應的開頭。|  
|[PROP\_DATA\_ENTRY](../Topic/PROP_DATA_ENTRY.md)|表示範圍或維度， ActiveX 控制項。|  
|[PROP\_ENTRY\_TYPE](../Topic/PROP_ENTRY_TYPE.md)|輸入屬性描述、屬性和 DISPID 屬性頁 CLSID 的屬性對應。|  
|[PROP\_ENTRY\_TYPE\_EX](../Topic/PROP_ENTRY_TYPE_EX.md)|輸入屬性描述、屬性 DISPID，屬性頁 CLSID 和 `IDispatch` IID 的屬性對應。|  
|[PROP\_PAGE](../Topic/PROP_PAGE.md)|輸入屬性頁 CLSID 的屬性對應。|  
|[END\_PROP\_MAP](../Topic/END_PROP_MAP.md)|標記 ATL 屬性對應的。|
