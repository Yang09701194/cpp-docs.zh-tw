# Registry Data Exchange Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這些巨集執行註冊資料交換作業。  
  
|||  
|-|-|  
|[BEGIN\_RDX\_MAP](../Topic/BEGIN_RDX_MAP.md)|這個標記會註冊資料交換對應的開頭。|  
|[END\_RDX\_MAP](../Topic/END_RDX_MAP.md)|這個標記會註冊資料交換對應的結束。|  
|[RDX\_BINARY](../Topic/RDX_BINARY.md)|會將指定的登錄項目具有型別的位元組指定的成員變數。|  
|[RDX\_CSTRING\_TEXT](../Topic/RDX_CSTRING_TEXT.md)|會將指定的登錄項目與型別 CString 中指定的成員變數。|  
|[RDX\_DWORD](../Topic/RDX_DWORD.md)|會將指定的登錄項目與型別 DWORD 中指定的成員變數。|  
|[RDX\_TEXT](../Topic/RDX_TEXT.md)|會將指定的登錄項目以 TCHAR 型別的指定成員變數。|
