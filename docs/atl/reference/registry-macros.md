# Registry Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這些巨集定義有用型別程式庫和註冊安裝。  
  
|||  
|-|-|  
|[\_ATL\_STATIC\_REGISTRY](../Topic/_ATL_STATIC_REGISTRY.md)|指示您希望物件的註冊程式碼可以避免在的物件在 ATL.DLL 的相依性。|  
|[DECLARE\_LIBID](../Topic/DECLARE_LIBID.md)|提供 ATL 提供衍生這個型別程式庫的 *libid* 。|  
|[DECLARE\_NO\_REGISTRY](../Topic/DECLARE_NO_REGISTRY.md)|避免預設 ATL 註冊。|  
|[DECLARE\_REGISTRY](../Topic/DECLARE_REGISTRY.md)|輸入或移除系統中登錄的主要物件的輸入。|  
|[DECLARE\_REGISTRY\_APPID\_RESOURCEID](../Topic/DECLARE_REGISTRY_APPID_RESOURCEID.md)|指定要求的資訊會自動註冊 *appid*。|  
|[DECLARE\_REGISTRY\_RESOURCE](../Topic/DECLARE_REGISTRY_RESOURCE.md)|在其中尋找具名資源並執行註冊指令碼。|  
|[DECLARE\_REGISTRY\_RESOURCEID](../Topic/DECLARE_REGISTRY_RESOURCEID.md)|在其中尋找資源所識別 ID 編號和執行註冊指令碼。|
