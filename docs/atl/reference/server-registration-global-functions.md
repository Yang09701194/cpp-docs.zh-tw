# Server Registration Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這些函式來登錄的支援，所以物件的註冊物件的對應。  
  
> [!IMPORTANT]
>  下表中列出的函式不能用於 [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)]執行的應用程式。  
  
|||  
|-|-|  
|[AtlComModuleRegisterServer](../Topic/AtlComModuleRegisterServer.md)|這個函式呼叫登錄物件中對應的每個物件。|  
|[AtlComModuleUnregisterServer](../Topic/AtlComModuleUnregisterServer.md)|這個函式呼叫移除註冊物件中對應的每個物件。|  
|[AtlComModuleRegisterClassObjects](../Topic/AtlComModuleRegisterClassObjects.md)|這個函式呼叫已註冊之類別的物件。|  
|[AtlComModuleRevokeClassObjects](../Topic/AtlComModuleRevokeClassObjects.md)|呼叫這個函式會從 COM 模組的類別物件。|  
|[AtlComModuleGetClassObject](../Topic/AtlComModuleGetClassObject.md)|這個函式會取得類別物件。|
