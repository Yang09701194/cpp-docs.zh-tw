# Service Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這些巨集定義服務對應和項目。  
  
|||  
|-|-|  
|[BEGIN\_SERVICE\_MAP](../Topic/BEGIN_SERVICE_MAP.md)|標記 ATL 服務對應的開頭。|  
|[END\_SERVICE\_MAP](../Topic/END_SERVICE_MAP.md)|標記導覽 ATL 服務的結束。|  
|[SERVICE\_ENTRY](../Topic/SERVICE_ENTRY.md)|指示物件支援特定的服務 ID。|  
|[SERVICE\_ENTRY\_CHAIN](../Topic/SERVICE_ENTRY_CHAIN.md)|指示 [IServiceProviderImpl::QueryService](../Topic/IServiceProviderImpl::QueryService.md) 繫結至指定的物件。|
