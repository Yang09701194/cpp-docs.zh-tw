# <a name="stock-properties-atl-control-wizard"></a>內建屬性, ATL 控制項精靈

在精靈的此頁面會識別內建控制項支援的屬性。 根據預設，會識別沒有屬性。

**不支援**  
指出目前控制項不支援的可用屬性。

**支援**  
指出控制項目前支援的屬性。

|傳輸按鈕|描述|
|---------------------|-----------------|
|**>**|將加入至**Supported**清單中目前選取的屬性名稱**不支援**清單。|
|**>>**|將加入至**Supported**清單中可用的所有屬性名稱**不支援**都清單。|
|**<**|移除中目前選取的屬性名稱**支援**清單。|
|**<<**|移除目前所列的所有屬性名稱**支援**清單。|
