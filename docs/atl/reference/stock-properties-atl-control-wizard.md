# 內建屬性, ATL 控制項精靈
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

此精靈頁面可識別控制項支援的內建屬性 \(Stock Property\)。  依照預設不會識別任何屬性。  
  
 **不支援**  
 表示目前控制項不支援的可用屬性。  
  
 **支援項目**  
 表示目前控制項支援的屬性。  
  
|傳送按鈕|描述|  
|----------|--------|  
|**\>**|將目前在 \[不支援\] 清單中選取的屬性名稱加入至 \[支援\] 清單。|  
|**\>\>**|將 \[不支援\] 清單中的所有可用屬性加入至 \[支援\] 清單。|  
|**\<**|移除目前在 \[支援\] 清單中選取的屬性名稱。|  
|**\<\<**|移除列在 \[支援\] 清單中的所有屬性名稱。|
