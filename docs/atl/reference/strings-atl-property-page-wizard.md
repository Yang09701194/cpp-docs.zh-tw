# 字串, ATL 屬性頁精靈
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

提供與屬性頁相關的文字。  
  
 **標題**  
 設定出現在屬性頁索引標籤上的文字。  
  
 **Doc 字串**  
 設定描述頁面的文字字串。  這個字串可以顯示在屬性工作表對話方塊中。  屬性框 \(Property Frame\) 可在狀態列或工具提示中使用此描述。  標準屬性框目前並未使用這個字串。  
  
 **說明檔**  
 設定描述如何使用屬性頁的說明檔檔名。  這個名稱不應包含路徑。  當使用者按下 \[說明\] 時，框架 \(Frame\) 會開啟說明檔，位於 CLSID 下、屬性頁登錄項目中之 HelpDir 機碼 \(Key\) 值所指定的目錄中。
