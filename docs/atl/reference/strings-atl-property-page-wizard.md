# <a name="strings-atl-property-page-wizard"></a>字串, ATL 屬性頁精靈

::: moniker range="vs-2019"

Visual Studio 2019 及更新版本中未提供 ATL 屬性頁精靈。

::: moniker-end

::: moniker range="vs-2017"

提供與屬性頁相關聯的文字。

- **標題**

   設定要顯示於屬性頁索引標籤上的文字。

- **文件字串**

   設定描述該頁面的文字字串。 此字串可以顯示於屬性工作表對話方塊中。 屬性框架可以在狀態列或工具提示中使用描述。 標準屬性框架目前不會使用此字串。

- **說明檔**

   設定描述如何使用屬性頁的說明檔名稱。 此名稱不應包含路徑。 當使用者按下 [說明] 時，框架即會開啟目錄中的說明檔，此目錄的名稱為其 CLSID 下方屬性頁登錄項目中的 HelpDir 機碼值。

::: moniker-end
