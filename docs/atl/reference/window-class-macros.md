# Window Class Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

|||  
|-|-|  
|[DECLARE\_WND\_CLASS](../Topic/DECLARE_WND_CLASS.md)|可讓您指定新的視窗類別的名稱。|  
|[DECLARE\_WND\_SUPERCLASS](../Topic/DECLARE_WND_SUPERCLASS.md)|可讓您指定新的視窗類別現有視窗類別的名稱。|  
|[DECLARE\_WND\_CLASS\_EX](../Topic/DECLARE_WND_CLASS_EX.md)|可讓您指定類別的參數。|
