# Scope of ATL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ATL 可讓您輕鬆地建立 COM 物件、Automation 伺服程式和 ActiveX 控制項。  ATL 提供許多基本 COM 介面提供內建支援。  
  
 ATL 傳輸為您在應用程式中加入的原始程式碼。  ATL DLL 也允許可用 \(atl90.dll\)，其中包含程式碼可跨共用元件。  不過，這個 DLL 不是必要的。
