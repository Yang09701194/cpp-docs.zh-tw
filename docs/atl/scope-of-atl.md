# <a name="scope-of-atl"></a>ATL 的範圍
ATL 可讓您輕鬆建立 COM 物件、 automation 伺服程式和 ActiveX 控制項。 ATL 提供基本的 COM 介面的許多的內建支援。  
  
 ATL 隨附做為包含在您的應用程式中的原始程式碼。 ATL 也可讓 DLL 可用 (atl90.dll)，其中包含可以橫跨元件的程式碼。 不過，不需要這個 DLL。
