# <a name="scope-of-atl"></a>ATL 的範圍

ATL 可讓您輕鬆地建立 COM 物件、 Automation 伺服程式和 ActiveX 控制項。 ATL 提供基本的 COM 介面的許多的內建支援。

ATL 會隨附原始程式碼包含在您的應用程式。 ATL 也可讓 DLL 可用 (atl90.dll)，其中包含可以跨元件共用的程式碼。 不過，不需要這個 DLL。
