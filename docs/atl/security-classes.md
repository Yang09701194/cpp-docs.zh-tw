# Security Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

這些類別是常見的 Win32 安全性類別和物件的包裝函式。  
  
-   [CAccessToken](../atl/reference/caccesstoken-class.md) 這個類別是存取語彙基元的包裝函式。  
  
-   [CAcl](../atl/reference/cacl-class.md) 這個類別是 **ACL** \(存取控制清單 \(SACL\)\) 結構的包裝函式。  
  
-   [CDacl](../atl/reference/cdacl-class.md) 這個類別是 **DACL** \(Discretionary 存取控制清單 \(DACL\)\) 結構的包裝函式。  
  
-   [CPrivateObjectSecurityDesc](../atl/reference/cprivateobjectsecuritydesc-class.md) 這個類別表示私用物件安全性描述元物件。  
  
-   [CSacl](../atl/reference/csacl-class.md) 這個類別是 **SACL** \(系統存取控制清單 \(SACL\)\) 結構的包裝函式。  
  
-   [CSecurityAttributes](../atl/reference/csecurityattributes-class.md) 這個類別是 **SECURITY\_ATTRIBUTES** 結構的小型包裝函式。  
  
-   [CSecurityDesc](../atl/reference/csecuritydesc-class.md) 這個類別是 **SECURITY\_DESCRIPTOR** 結構的包裝函式。  
  
-   [CSid](../atl/reference/csid-class.md) 這個類別是 `SID` \(Security Identifier\) 結構的包裝函式。  
  
-   [CTokenGroups](../atl/reference/ctokengroups-class.md) 這個類別是 **TOKEN\_GROUPS** 結構的包裝函式。  
  
-   [CTokenPrivileges](../atl/reference/ctokenprivileges-class.md) 這個類別是 **TOKEN\_PRIVILEGES** 結構的包裝函式。
