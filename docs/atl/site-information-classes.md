# <a name="site-information-classes"></a>網站資訊類別

下列類別允許物件與其站台與通訊：

- [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md)擷取和設定物件的站台的指標。 用於不是控制項的物件。

- [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md)擷取和設定物件的站台的指標。 用於控制項。
