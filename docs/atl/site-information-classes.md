# Site Information Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下列類別可讓物件與其網站通訊:  
  
-   [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) 擷取和設定指向物件的網站。  使用為非控制項的物件。  
  
-   [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) 擷取和設定指向物件的網站。  使用控制項。
