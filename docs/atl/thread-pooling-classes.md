# Thread Pooling Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下列類別可支援執行緒共用:  
  
-   [CComAutoThreadModule](../atl/reference/ccomautothreadmodule-class.md) 實作 EXE 模組，可支援多個執行緒集區 Apartment。  
  
-   [CComApartment](../atl/reference/ccomapartment-class.md) 處理在執行緒集區的 EXE 模組的單一執行緒 Apartment。  
  
-   [CComSimpleThreadAllocator](../atl/reference/ccomsimplethreadallocator-class.md) 處理 EXE 模組的執行緒選取範圍。
