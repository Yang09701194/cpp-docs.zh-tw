# Using a Template Library
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

範本有點類似巨集。  使用巨集，叫用樣板會展開 \(使用適當的參數取代\) 程式碼撰寫。  不過，此範本進一步遠比這個移至允許根據您傳遞做為參數型別的新類別的建立。  執行會在範本程式碼表示的操作這些新的類別所實作的型別安全方式。  
  
 樣板程式庫 \(例如 ATL 以傳統 C\+\+ 類別庫不同它們通常只會提供做為原始程式碼 \(或用一點的原始程式碼，以支援執行階段\) 而不是原本或需要階層本質上。  而不是從類別取得您的功能時，您會從範本的類別。
