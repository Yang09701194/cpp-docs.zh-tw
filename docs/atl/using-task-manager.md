# <a name="using-task-manager"></a>使用工作管理員
其中一種偵錯服務最簡單的方式是透過使用工作管理員。 執行服務時，啟動 [工作管理員，然後按一下**處理程序**] 索引標籤。可執行檔名稱上按一下滑鼠右鍵，然後按一下**偵錯**。 這會啟動附加至該執行的處理序的 Visual c + +。 現在，按一下 **中斷**上**偵錯**功能表可讓您在程式碼中設定中斷點。 按一下**執行**執行到您選取的中斷點。
