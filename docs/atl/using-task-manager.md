# Using Task Manager
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

其中一個簡易的方式偵錯服務會使用 Windows 4.0 或 Windows NT 2000 的工作管理員。  當服務正在執行時，啟動工作管理員並按一下 **其他處理序** 索引標籤。  以滑鼠右鍵按一下可執行檔的名稱並按一下 **偵錯**。  這樣就會啟動 Visual C\+\+ 附加至執行中的處理序。  現在，可讓您的 **偵錯** 功能表中按一下 **中斷** 設定中斷點的程式碼。  按一下 **執行** 執行至選取的中斷點。
