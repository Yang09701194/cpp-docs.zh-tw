# <a name="using-task-manager"></a>使用 工作管理員

偵錯服務最簡單的方法之一是透過使用工作管理員。 當服務執行時，啟動 [工作管理員]，然後按一下**處理序**] 索引標籤。以滑鼠右鍵按一下 [EXE 的名稱，然後按一下**偵錯**。 這會啟動 VisualC++附加至該執行的處理序。 現在，按一下 **中斷**上**偵錯**功能表可讓您在您的程式碼中設定中斷點。 按一下 **執行**執行到您選取的中斷點。
