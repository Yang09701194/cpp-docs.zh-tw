# <a name="utility-classes"></a>公用程式類別
提供下列的獨立於 MFC 的公用程式類別：  
  
-   [CImage](../atl-mfc-shared/reference/cimage-class.md)提供增強的點陣圖支援，包括能夠載入，並將影像儲存在資料 JPEG、 BMP、 GIF 和可攜式網路圖形 (PNG) 格式。  
  
-   [CPoint](../atl-mfc-shared/reference/cpoint-class.md)提供將儲存 （x，y） 座標組的實作。  
  
-   [CRect](../atl-mfc-shared/reference/crect-class.md)提供將儲存的矩形區域座標的實作。  
  
-   [CSize](../atl-mfc-shared/reference/csize-class.md)提供距離、 相對位置或配對的值將儲存的實作。  
  
-   [CString](../atl-mfc-shared/reference/cstringt-class.md)提供將儲存字元字串的實作。  
  
-   [CAdapt](../atl/reference/cadapt-class.md)簡單的範本，用來包裝重新定義傳址運算子的類別。  
  
-   [_U_RECT](../atl/reference/u-rect-class.md)引數配接器類別，可讓`RECT`指標或參考傳遞至函式實作方面的指標。
