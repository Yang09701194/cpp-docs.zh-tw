# Utility Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

提供下列 MFC 無關的公用程式類別:  
  
-   [CImage](../atl-mfc-shared/reference/cimage-class.md) 提供增強的點陣圖支援，包括載入和儲存影像在 JPEG、GIF、BMP 和可攜式網路圖形 \(PNG\) 格式。  
  
-   [CPoint](../atl-mfc-shared/reference/cpoint-class.md) 座標儲存為提供實作 \(x， y\)。  
  
-   [CRect](../atl-mfc-shared/reference/crect-class.md) 用於儲存矩形區域座標的實作。  
  
-   [CSize](../atl-mfc-shared/reference/csize-class.md) 用於儲存距離、相對位置或配對的值提供實作。  
  
-   [CString](../atl-mfc-shared/reference/cstringt-class.md) 用來儲存字串提供實作。  
  
-   用來[CAdapt](../atl/reference/cadapt-class.md) A 簡單範本包裝重新定義傳址運算子的類別。  
  
-   [\_U\_RECT](../atl/reference/u-rect-class.md) 允許 `RECT` 指標或參考傳遞至函式實作以指標的引數配置器類別。
