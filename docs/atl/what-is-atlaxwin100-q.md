# <a name="what-is-atlaxwin100"></a>什麼是 AtlAxWin100？
`AtlAxWin100`是可協助提供 ATL 的控制項裝載功能的視窗類別名稱。 當您建立這個類別的執行個體時，視窗程序會自動使用的控制項裝載 API 來建立與視窗相關聯的主機物件，並將其載入與您指定為視窗的標題控制項。
