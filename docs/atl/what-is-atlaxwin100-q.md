# What Is AtlAxWin100?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

`AtlAxWin100` 是協助提供的 ATL 控制項載入功能視窗類別的名稱。  當您建立此類別的執行個體，視窗程序會自動使用控制項裝載 API 建立主應用程式物件與視窗和在您指定為視窗標題的控制項將它載入。
