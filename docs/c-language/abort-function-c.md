# abort 函式 (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.10.4.1** 與已開啟檔案和暫存檔案相關的 **abort** 函式行為  
  
 **abort** 函式不會關閉已開啟的檔案或暫存檔案。  它不會清除資料流緩衝區。
