# ANSI 一致性
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Microsoft C 符合 ANSI C 標準 9899:1990 版所設定的 C 語言標準。  
  
 符合 ANSI C 標準的 Microsoft 擴充功能記錄在本書及線上參考的文字和語法中。  擴充功能不是 ANSI C 標準的一部分，因此使用擴充功能可能會限制程式在系統之間的可攜性。  根據預設，會啟用 Microsoft 擴充功能。  若要停用擴充功能，請指定 \/Za 編譯器選項。  指定 \/Za 時，所有非 ANSI 程式碼都會產生錯誤或警告。
