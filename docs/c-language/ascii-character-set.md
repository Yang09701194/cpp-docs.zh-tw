# <a name="ascii-character-set"></a>ASCII 字元集

**ANSI 2.2.1** 來源和執行字元集的成員

來源字元集是一組合法的字元，可以出現在原始程式檔中。 對於 Microsoft C 來說，來源字元集是指標準 ASCII 字元集。

> [!NOTE]
> **警告** 由於鍵盤和主控台驅動程式可能會重新對應字元集，因此主要供國際發行的程式應該檢查國碼/區碼。
