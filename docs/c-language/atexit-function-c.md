# atexit 函式 (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.10.4.3** 引數的值不是零、EXIT\_SUCCESS 或 EXIT\_FAILURE 時，`atexit` 函式所傳回的狀態  
  
 如果成功，`atexit` 函式會傳回零，如果失敗，則傳回非零值。
