# <a name="atexit-function-c"></a>atexit 函式 (C)

**ANSI 4.10.4.3** 引數的值不是零、EXIT_SUCCESS 或 EXIT_FAILURE 時，`atexit` 函式所傳回的狀態

如果成功，`atexit` 函式會傳回零，如果失敗，則傳回非零值。
