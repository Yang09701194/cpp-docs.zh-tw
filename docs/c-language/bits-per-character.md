# <a name="bits-per-character"></a>每個字元位元數

**ANSI 2.2.4.2.1** 字元中的位元數

字元中的位元數是以資訊清單常數 **CHAR_BIT** 表示。 LIMITS.H 檔案將 **CHAR_BIT** 定義為 8。
