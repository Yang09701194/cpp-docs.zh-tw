# 每個字元位元數
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 2.2.4.2.1** 字元中的位元數  
  
 字元中的位元數是以資訊清單常數 **CHAR\_BIT** 表示。  LIMITS.H 檔案將 **CHAR\_BIT** 定義為 8。
