# <a name="c-language-syntax-summary"></a>C 語言語法摘要
本節提供 C 語言和 Microsoft 專有 C 語言功能的完整描述。 您可以使用本節中的語法標記法，來判斷所有語言元件的實際語法。 語法的說明會出現在本手冊討論主題的章節中。  
  
> [!NOTE]
>  這個語法摘要不是 ANSI C 標準的一部分，包括在內僅供參考之用。 Microsoft 專有語法會在語法之後註解中註明。
