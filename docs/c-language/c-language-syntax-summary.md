# C 語言語法摘要
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

本節提供 C 語言和 Microsoft 特定 C 語言功能的完整描述。  您在這個區段可以使用語法符號判斷所有語言元件的實際語法。  語法的說明會出現在主題中所討論這個方針的部分。  
  
> [!NOTE]
>  這個語法摘要不是 ANSI C 標準的一部分，只是包括其僅供參考之用。  Microsoft 特定語法會在語法之後註解中註明。
