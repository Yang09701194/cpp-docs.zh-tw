# <a name="c-primary-expressions"></a>C 主要運算式

運算式中的運算元稱為「主要運算式」。

## <a name="syntax"></a>語法

*主要-運算式*：<br/>
&nbsp;&nbsp;&nbsp;&nbsp;*標識*<br/>
&nbsp;&nbsp;&nbsp;&nbsp;*常數*<br/>
&nbsp;&nbsp;&nbsp;&nbsp;*字串常值*<br/>
&nbsp;&nbsp;&nbsp;&nbsp;**（** *運算式* **）**

*expression*：<br/>
&nbsp;&nbsp;&nbsp;&nbsp;*指派-運算式*<br/>
&nbsp;&nbsp;&nbsp;&nbsp;*運算式* **，** *指派運算式*
