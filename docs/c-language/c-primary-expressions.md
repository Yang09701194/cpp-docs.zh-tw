# <a name="c-primary-expressions"></a>C 主要運算式
運算式中的運算元稱為「主要運算式」。  
  
## <a name="syntax"></a>語法  
 *primary-expression*：  
 *identifier*  
  
 *constant*  
  
 *string-literal*  
  
 **(**  *expression*  **)**  
  
 *expression*：  
 *assignment-expression*  
  
 *expression*  **,**  *assignment-expression*
