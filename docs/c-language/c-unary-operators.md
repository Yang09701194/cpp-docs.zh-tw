# C 一元運算子
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

一元運算子會出現在其運算元之前，以及從右邊連結到左邊。  
  
## 語法  
 *unary\-expression*：  
 *postfix\-expression*  
  
 **\+\+**  *unary\-expression*  
  
 `--`  *unary\-expression*  
  
 *unary\-operator cast\-expression*  
  
 **sizeof**  *unary\-expression*  
  
 **sizeof \(**  *type\-name*  **\)**  
  
 *unary\-operator*：下列其中一項  
 **&**  
 **\* \+ –** `~` **\!**
