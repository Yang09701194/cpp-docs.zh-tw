# <a name="c-unary-operators"></a>C 一元運算子
一元運算子會出現在其運算元之前，以及從右邊連結到左邊。  
  
## <a name="syntax"></a>語法  
 *unary-expression*：  
 *postfix-expression*  
  
 **++**  *unary-expression*  
  
 `--`  *unary-expression*  
  
 *unary-operator cast-expression*  
  
 **sizeof**  *unary-expression*  
  
 **sizeof (**  *type-name*  **)**  
  
 *unary-operator*：下列其中一個  
 **& \* + -** `~` **!**
