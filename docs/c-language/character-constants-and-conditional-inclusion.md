# <a name="character-constants-and-conditional-inclusion"></a>字元常數和條件式包含
**ANSI 3.8.1**：單一字元的字元常數值在控制條件式包含的常數運算式中，是否符合執行字元集中的相同字元常數的值。 這類字元常數是否可以有負數值  
  
 前置處理器陳述式中使用的字元集與執行字元集相同。 前置處理器會辨識負數字元值。
