# 字元常數和條件式包含
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.8.1**：單一字元的字元常數值在控制條件的常數運算式中是否包含執行字元集中的相同字元常數值。  這類字元常數是否可以有負數值  
  
 前置處理器陳述式中使用的字元集與執行字元集相同。  前置處理器會辨識負數字元值。
