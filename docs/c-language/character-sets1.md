# 字元集
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.1.3.4** 對應原始程式碼字元集的成員  
  
 原始程式碼字元集和執行字元集包含下表所列的 ASCII 字元。  表中也列出逸出序列。  
  
### 逸出序列  
  
|逸出序列|字元|ASCII 值|  
|----------|--------|-------------|  
|\\a|警示\/鈴響|7|  
|\\b|退格鍵|8|  
|\\f|Formfeed|12|  
|\\n|新行字元|10|  
|\\r|歸位字元|13|  
|\\t|水平 Tab|9|  
|\\v|垂直 Tab|11|  
|\\"|雙引號|34|  
|\\'|單引號|39|  
|\\\\|反斜線|92|
