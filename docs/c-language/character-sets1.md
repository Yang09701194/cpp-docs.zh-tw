# <a name="character-sets"></a>字元集

**ANSI 3.1.3.4**：來源字元集的對應成員

原始程式碼字元集和執行字元集包含下表所列的 ASCII 字元。 表中也列出逸出序列。

## <a name="escape-sequences"></a>逸出序列

|逸出序列|字元|ASCII 值|
|---------------------|---------------|-----------------|
|&#92;a|警示/鈴響|7|
|&#92;b|退格鍵|8|
|&#92;f|Formfeed|12|
|&#92;n|新行字元|10|
|&#92;r|歸位字元|13|
|&#92;t|水平 Tab|9|
|&#92;v|垂直 Tab|11|
|&#92;"|雙引號|34|
|&#92;'|單引號|39|
|&#92;&#92;|反斜線|92|
