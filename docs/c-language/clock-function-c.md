# clock 函式 (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.12.2.1**：`clock` 函式的紀元  
  
 在 C 程式開始執行時，會開始執行 `clock` 函式的紀元 \(其值為 0\)。  它會傳回以 1\/**CLOCKS\_PER\_SEC** 測量的時間 \(等於 Microsoft C 的 1\/1000\)。
