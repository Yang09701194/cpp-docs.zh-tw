# <a name="constants-in-primary-expressions"></a>主要運算式中的常數
常數運算元具有其所代表之常數值的值和類型。 字元常數具有 `int` 類型。 視整數的大小與指定值的方式而定，整數常數具有 `int`、**long**、`unsigned long` 或 `unsigned int` 類型。 如需詳細資訊，請參閱[常數](../c-language/c-constants.md)。
