# 主要運算式中的常數
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

常數運算元具有其所代表之常數值的值和類型。  字元常數具有 `int` 類型。  視整數的大小與指定值的方式而定，整數常數具有`int`、**long**、`unsigned int` 或 `unsigned long` 類型。  如需詳細資訊，請參閱[常數](../c-language/c-constants.md)。
