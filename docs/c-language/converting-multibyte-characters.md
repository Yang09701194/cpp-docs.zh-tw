# <a name="converting-multibyte-characters"></a>轉換多位元組字元

**ANSI 3.1.3.4**：目前的地區設定，用來將多位元組字元針對寬字元常數轉換成對應的寬字元 (程式碼)

預設的目前地區設定為 "C" 地區設定。 您可以使用 [#pragma setlocale](../preprocessor/setlocale.md) 來變更它。
