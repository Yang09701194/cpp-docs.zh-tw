# 轉換多位元組字元
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.1.3.4**：目前的地區設定，用來將多位元組字元轉換成寬字元常數的對應寬字元 \(程式碼\)  
  
 預設的目前地區設定為 "C" 地區設定。  您可以使用 [\#pragma setlocale](../preprocessor/setlocale.md) 將它變更。
