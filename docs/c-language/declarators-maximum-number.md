# 宣告子：最大數目
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.5.4** 可以修改算術、結構或等位類型的宣告子數目上限  
  
 Microsoft C 不會限制宣告子的數目。  這些數目只會受到可用記憶體的限制。
