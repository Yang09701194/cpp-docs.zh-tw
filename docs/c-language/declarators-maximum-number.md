# <a name="declarators-maximum-number"></a>宣告子：最大數目

**ANSI 3.5.4**：可以修改算術、結構或等位型別的宣告子數目上限

Microsoft C 不會限制宣告子的數目。 這些數目只會受到可用記憶體的限制。
