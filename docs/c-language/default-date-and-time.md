# <a name="default-date-and-time"></a>預設日期和時間

**ANSI 3.8.8**：日期和時間各自的轉譯無法使用時，`__DATE__` 和 `__TIME__` 的定義

當作業系統未提供轉譯的日期和時間時，`__DATE__` 和 `__TIME__` 的預設值會是 `May 03 1957` 和 `17:00:00`。
