# 預設日期和時間
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.8.8**：日期和時間各自的轉譯無法使用時，\_DATE\_ 和 \_TIME\_ 的定義。  
  
 當作業系統未提供轉譯的日期和時間時，\_DATE\_ 和 \_TIME\_ 的預設值會是 `May 03 1957` 和 `17:00:00"`。
