# 預設訊號
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.7.1.1** 如果沒有在呼叫信號處理常式之前執行 **signal \(***sig***, SIG\_DFL\)** 的同等項，會封鎖所執行的訊號  
  
 在程式開始執行時，會將訊號設為其預設狀態。
