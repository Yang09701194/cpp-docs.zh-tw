# 刪除開啟檔案
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.4.1**：在開啟的檔案上移除函式造成的影響  
  
 移除函式會刪除檔案。  如果檔案已開啟，則此函式會失敗且傳回 \-1。
