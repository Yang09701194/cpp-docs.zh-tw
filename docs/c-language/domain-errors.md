**ANSI 4.5.1**：由定義域上的數學函式傳回的值發生錯誤  
  
 ERRNO.H 檔案中將定義域錯誤常數 `EDOM` 定義為 33。如需傳回值的詳細資訊，請參閱說明主題中導致錯誤的特定函式主題。
