# <a name="domain-errors"></a>定義域錯誤

**ANSI 4.5.1**：由數學函式針對定義域錯誤所傳回的值

ERRNO.H 檔案中將定義域錯誤常數 `EDOM` 定義為 33。  如需傳回值的詳細資訊，請參閱有關導致錯誤之特定函式的說明主題。
