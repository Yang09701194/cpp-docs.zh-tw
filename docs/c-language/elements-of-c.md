# C 的元素
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

本節說明 C 程式設計語言的項目，包括用來建構 C 程式的識別、數字和字元 。  ANSI C 語言標記了這些元件語彙基元。  
  
 本節將說明如何定義語彙基元，以及編譯器如何評估它們。  
  
 將會討論下列主題：  
  
-   [語彙基元](../c-language/c-tokens.md)  
  
-   [註解](../c-language/c-comments.md)  
  
-   [Keywords](../c-language/c-keywords.md)  
  
-   [識別項](../c-language/c-identifiers.md)  
  
-   [常數](../c-language/c-constants.md)  
  
-   [字串常值](../c-language/c-string-literals.md)  
  
-   [標點符號和特殊字元](../c-language/punctuation-and-special-characters.md)  
  
 本節也包含 [三併詞](../c-language/trigraphs.md)、 [浮點常數的限制](../c-language/limits-on-floating-point-constants.md)、 [C\+\+ 整數限制](../c-language/cpp-integer-limits.md)和 [逸出序列](../c-language/escape-sequences.md)的參考資料表。  
  
 運算子是指定如何操作值的符號 \(單一字元和字元組合\) 。  每個符號會解譯為單一單位，叫作語彙基元。  如需詳細資訊，請參閱[運算子](../c-language/c-operators.md)。
