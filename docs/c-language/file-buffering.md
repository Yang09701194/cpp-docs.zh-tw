# <a name="file-buffering"></a>檔案緩衝

**ANSI 4.9.3**：檔案緩衝的特性

透過標準 I/O 函式存取的磁碟檔案會進行完整的緩衝。 根據預設，緩衝區會保存 512 個位元組。
