# <a name="file-position-in-append-mode"></a>附加模式的檔案位置

**ANSI 4.9.3**：附加模式資料流的檔案位置指示器起初是位於檔案的開頭或結尾

在附加模式中開啟檔案時，檔案位置指示器起初會指向檔案的結尾。
