# 附加模式的檔案位置
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** 付加模式資料流的檔案位置指示器起初是位於檔案的開頭或結尾  
  
 在附加模式中開啟檔案時，檔案位置指示器起初會指向檔案的結尾。
