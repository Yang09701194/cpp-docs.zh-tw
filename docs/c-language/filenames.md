# <a name="filenames"></a>檔案名稱
**ANSI 4.9.3**：組成有效檔案名稱的規則  
  
 檔案規格可以包含選擇性的磁碟機代號 (後面一律需加冒號)、一連串的選擇性目錄名稱 (以反斜線分隔)，以及檔案名稱。  
  
 如需詳細資訊，請參閱[命名檔案](http://msdn.microsoft.com/library/windows/desktop/aa365247)。
