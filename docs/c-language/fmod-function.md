# fmod 函式
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.5.6.4** 在 `fmod` 函式的第二個引數為零時是否發生網域錯誤或者傳回零  
  
 當 `fmod` 函式的第二個引數為零時，函式會傳回零。
