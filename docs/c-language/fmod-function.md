# <a name="fmod-function"></a>fmod 函式

**ANSI 4.5.6.4** 當 `fmod` 函式的第二個引數為零時，是否發生定義域錯誤或者傳回零

當 `fmod` 函式的第二個引數為零時，函式會傳回零。
