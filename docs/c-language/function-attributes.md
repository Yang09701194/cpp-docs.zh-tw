# <a name="function-attributes"></a>函式屬性
**Microsoft 特定的**  
  
 選擇性的 *attribute-seq* 非終結端可讓您依據函式來選取呼叫慣例。 您也可以將函式指定為 `__fastcall` 或 `__inline`。
