# <a name="implementation-defined-behavior"></a>實作定義的行為

ANSI X3.159-1989,*美國資訊系統* - *程式設計語言* - *C*國家標準,包含一個叫做"可移植問題"的部分。 ANSI 章節列出 C 語言當中 ANSI 對每種特定實作開放的區域。 本節說明 Microsoft C 如何處理 C 語言中這些由實作定義的區域。

本節的順序比照 ANSI 章節的順序。 其中討論的每個項目均包含說明實作定義行為的 ANSI 參考。

> [!NOTE]
> 本節只說明美國英文版的 C 編譯器。 其他語言的 Microsoft C 實作可能稍有不同。
