# <a name="largest-array-size"></a>陣列大小上限

**ANSI 3.3.3.4、4.1.1**：保留陣列大小上限所需的整數型別，即 **size_t** 的大小

`size_t` typedef 在 32 位元 x86 平台上是 `unsigned int`。 在 64 位元平台上，`size_t` typedef 為 `unsigned __int64`。
