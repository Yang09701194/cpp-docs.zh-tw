# <a name="multibyte-characters"></a>多位元組字元

**ANSI 2.2.1.2**：多位元組字元的移位狀態

某些實作 (包括 Microsoft C) 會使用多位元組字元來表示不存在於在基底字元集中的外部語言字元。 不過，Microsoft C 不支援任何與狀態相關的編碼。 因此，其中不會內含移位狀態。 如需詳細資訊，請參閱[多位元組字元和寬字元](../c-language/multibyte-and-wide-characters.md)。
