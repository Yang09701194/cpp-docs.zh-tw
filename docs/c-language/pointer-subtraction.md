# <a name="pointer-subtraction"></a>指標減法
**ANSI 3.3.6、4.1.1**：用來保存相同陣列兩個元素指標之間差異的整數類型，即 **ptrdiff_t**  
  
 **ptrdiff_t** 的類型是 **signed int**。
