# <a name="punctuation-and-special-characters"></a>標點符號和特殊字元

C 字元集中的標點符號和特殊字元具有各種用途，從組織程式文字到定義編譯器或編譯器所執行的工作。它們不會指定要執行的作業。 部分標點符號也是運算子 (請參閱 [運算子](../c-language/c-operators.md))。 編譯器會透過內容判斷其用法。

## <a name="syntax"></a>語法

`punctuator`：其中一個 **（） [] {} \* ，： =;  ... #**

這些字元在 C 中有特殊的意義。本書詳細說明其用法。 井字型大小（**#**）只能出現在前置處理[指令](../preprocessor/preprocessor-directives.md)詞中。
