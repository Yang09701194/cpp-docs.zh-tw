# <a name="reading-pointer-values"></a>讀取指標值

**ANSI 4.9.6.2**：`fscanf` 函式中 **%p** 轉換的輸入內容

指定 **%p** 格式字元時，`fscanf` 函式會將指標從十六進位 ASCII 值轉換成正確的位址。
