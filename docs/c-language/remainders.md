# <a name="remainders"></a>餘數
**ANSI 3.3.5**：整數除法上餘數的正負號  
  
 餘數的正負號與被除數的正負號相同。 例如，套用至物件的  
  
```  
 50 / -6 == -8  
 50 % -6 ==  2  
-50 /  6 == -8  
-50 %  6 == -2  
```
