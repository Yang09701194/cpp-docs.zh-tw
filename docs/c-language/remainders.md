# 餘數
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.3.5**：整數除法上餘數的正負號  
  
 餘數的正負號與被除數的正負號相同。  例如：  
  
```  
 50 / -6 == -8  
 50 % -6 ==  2  
-50 /  6 == -8  
-50 %  6 == -2  
```
