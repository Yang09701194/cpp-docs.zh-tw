# <a name="renaming-with-a-name-that-exists"></a>以現有名稱重新命名
**ANSI 4.9.4.2**：具有新名稱的檔案在呼叫 **rename** 函式之前即已存在時的作用  
  
 如果您嘗試使用已存在的名稱為檔案重新命名，**rename** 函式會失敗並傳回錯誤碼。
