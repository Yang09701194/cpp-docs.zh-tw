# <a name="scope-of-this-manual"></a>本手冊的範圍
C 是一種具彈性的語言，可讓您自行決定程式設計的許多層面。 基於這個原理，C 對於像是類型轉換等方面的作業會有少數限制。 雖然語言的這種特性可讓程式設計工作更輕鬆，但是您必須熟知語言，才能了解程式的行為。 本書提供 C 語言元件和 Microsoft 實作功能的相關資訊。 C 語言的語法來自 ANSI X3.159-1989，美國國家資訊系統標準 - 程式語言 - C (以下稱為 ANSI C 標準)，不過它不屬於 ANSI C 標準的一部分。 [C 語言語法摘要](../c-language/c-language-syntax-summary.md)提供語法以及如何讀取和使用語法定義的描述。  
  
 本書不會討論使用 C++ 的設計程式。 如需 C++ 語言的資訊，請參閱 [C++ 語言參考](../cpp/cpp-language-reference.md)。
