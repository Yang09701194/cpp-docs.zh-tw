# <a name="sign-of-bit-fields"></a>位元欄位的正負號
**ANSI 3.5.2.1**：「純文字」`int` 欄位會視為 **signed int** 位元欄位或是 unsigned int 位元欄位  
  
 位元欄位可以帶正負號或不帶正負號。 純文字位元欄位會視為帶正負號。
