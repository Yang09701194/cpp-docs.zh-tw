# <a name="signal-function-c"></a>signal 函式 (C)

**ANSI 4.7.1.1** **signal** 函式的訊號組

傳遞至 **signal** 的第一個引數必須是《執行階段程式庫參考》中針對 **signal** 函式描述的其中一項符號常數。 《執行階段程式庫參考》中的資訊也會列出各訊號的作業模式支援。 SIGNAL.H 中也會定義常數。
