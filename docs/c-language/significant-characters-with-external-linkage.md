# <a name="significant-characters-with-external-linkage"></a>有外部連結的有效字元數
**ANSI 3.1.2**：有外部連結的有效字元數  
  
 在使用 Microsoft C 編譯的程式中宣告的識別項 `extern`，其有效字元為 247 個字元。 您可以使用 /H (限制外部名稱的長度) 選項將此預設值修改為較小的數字。
