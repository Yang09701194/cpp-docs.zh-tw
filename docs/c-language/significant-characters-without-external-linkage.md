# <a name="significant-characters-without-external-linkage"></a>沒有外部連結的有效字元數
**ANSI 3.1.2**：沒有外部連結的有效字元數  
  
 識別項的有效字元數上限為 247 個。 編譯器不會限制您在識別項中使用的字元數，而是會直接忽略超出限制的所有字元。
