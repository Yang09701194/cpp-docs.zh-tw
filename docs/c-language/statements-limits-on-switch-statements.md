# 陳述式：Switch 陳述式的限制
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.6.4.2** `switch` 陳述式中，**case** 值的最大數  
  
 Microsoft C 不會限制 **switch** 陳述式中 **case** 值的數目。  這些數目只會受到可用記憶體的限制。
