# <a name="storage-class-specifiers-for-internal-level-declarations"></a>內部層級宣告的儲存類別指定名稱

您只能在內部層級使用四個 *storage-class-specifier* 終端項當中的任何一個，以進行變數宣告。 當您從這類宣告中省略*儲存類別規範*時，預設的儲存類別會是**auto**。因此，C 程式很少會出現**auto**關鍵字。
