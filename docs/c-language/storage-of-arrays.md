# 陣列的儲存
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

與陣列類型相關的存放區是其所有項目所需要的存放區。  陣列的項目是儲存在連續且不斷增加的記憶體位置，從第一個項目到最後一個項目。
