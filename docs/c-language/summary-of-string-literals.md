# 字串常值的摘要
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

*string\-literal*：  
 **'** *s\-char\-sequence*  opt **'**  
  
 **L'** *s\-char\-sequence*  opt             **'**  
  
 *s\-char\-sequence*：  
 *s\-char*  
  
 *s\-char\-sequence s\-char*  
  
 *s\-char*：  
 原始程式碼字元集的任何成員，但雙引號 \("\)、反斜線 \(\\\) 或新行字元 *escape\-sequence* 除外
