# <a name="summary-of-string-literals"></a>字串常值的摘要
*string-literal*:  
 **'** *s-char-sequence* opt**'**  
  
 **L'** *s-char-sequence* opt**'**  
  
 *s-char-sequence*:  
 *s-char*  
  
 *s-char-sequence s-char*  
  
 *s-char*:  
 原始程式碼字元集的任何成員，但雙引號 (")、反斜線 (\\) 或新行字元 *escape-sequence* 除外
