# <a name="truncation-of-floating-point-values"></a>浮點值截斷
**ANSI 3.2.1.4** 將浮點數轉換為較小的浮點數時，截斷或進位的方向  
  
 發生反向溢位時，會將浮點變數的值向下捨入為零。 溢位可能會造成執行階段錯誤，或可能會因為所指定的最佳化而產生無法預期的值。
