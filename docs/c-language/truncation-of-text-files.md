# <a name="truncation-of-text-files"></a>文字檔截斷
**ANSI 4.9.3** 文字資料流上的寫入作業是否會導致相關聯的檔案於該點之後被截斷  
  
 寫入文字資料流不會在該點之後截斷檔案。
