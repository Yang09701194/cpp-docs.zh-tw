# 文字檔截斷
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3**：文字資料流上的寫入作業是否會導致相關聯的檔案於該點之後遭到截斷  
  
 寫入文字資料流不會在該點之後截斷檔案。
