# <a name="type-for-string-literals"></a>字串常值的類型

字串常值包含 `char` 的類型陣列 (亦即 **char[ ]**) （寬字元字串具有的`wchar_t`類型陣列（也就是**wchar_t []**））。這表示字串是包含類型`char`之元素的陣列。 陣列中的元素數目等於字串中的字元數目加一 (代表終止 Null 字元)。
