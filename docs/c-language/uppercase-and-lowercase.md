# 大寫和小寫
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.1.2** 是否區分大小寫  
  
 Microsoft C 會將編譯單位中的識別項視為區分大小寫。  
  
 Microsoft 連結器會區分大小寫。  您必須根據大小寫一致地指定所有的識別項。
