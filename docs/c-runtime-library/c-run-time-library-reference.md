# C 執行階段程式庫參考
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Microsoft 執行階段程式庫可以提供程式設計提供常式作業系統的 Windows 架構應用程式。  這些常式會自動執行 C 和 C\+\+ 語言所未提供的許多常見的程式設計工作。  
  
 範例程式中的參考主題包含大部分常式的程式庫中。  
  
## 在本節中  
 [C 執行階段程式庫](../c-runtime-library/crt-library-features.md)  
 討論包含 C 執行階段程式庫的 .lib 檔。  
  
 [依分類區分的執行階段常式](../c-runtime-library/run-time-routines-by-category.md)  
 提供連結至這個執行階段程式庫的類別。  
  
 [全域變數和標準類型](../c-runtime-library/global-variables-and-standard-types.md)  
 提供連結至這個執行階段程式庫和標準型別提供的全域變數。  
  
 [全域常數](../c-runtime-library/global-constants.md)  
 提供連結至這個執行階段程式庫中定義的全域常數。  
  
 [依字母順序排列的函式參考](../c-runtime-library/reference/crt-alphabetical-function-reference.md)  
 提供目錄的進入點 \(任何 C 執行階段程式庫函式以依字母順序列出。  
  
 [泛型文字對應](../c-runtime-library/generic-text-mappings.md)  
 提供連結至定義於 Tchar.h 的泛用文字對應。
