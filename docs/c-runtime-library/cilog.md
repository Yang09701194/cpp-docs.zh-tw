# <a name="cilog"></a>_CIlog

計算堆疊頂端值的自然對數。

## <a name="syntax"></a>語法

```
void __cdecl _CIlog();
```

## <a name="remarks"></a>備註

這個版本的 `log` 函式具有編譯器了解的特定呼叫慣例。 因為它可以防止產生複本並協助暫存器配置，所以會加速執行。

產生的值會推入至堆疊的頂端。

## <a name="requirements"></a>需求

**平台：** x86
