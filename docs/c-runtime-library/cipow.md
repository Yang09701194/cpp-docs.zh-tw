# <a name="cipow"></a>_CIpow

根據堆疊頂端值計算 *x* 的 *y* 次方。

## <a name="syntax"></a>語法

```
void __cdecl _CIpow();
```

## <a name="remarks"></a>備註

這個版本的 `pow` 函式具有編譯器了解的特定呼叫慣例。 因為它可以防止產生複本並協助暫存器配置，所以會加速執行。

產生的值會推入至堆疊的頂端。

## <a name="requirements"></a>需求
 **平台：** x86
