# <a name="_cipow"></a>_CIpow

根據堆疊頂端值計算 *x* 的 *y* 次方。

## <a name="syntax"></a>語法

```cpp
void __cdecl _CIpow();
```

## <a name="remarks"></a>備註

這個版本的 `pow` 函式具有編譯器了解的特定呼叫慣例。 因為它可以防止產生複本並協助暫存器配置，所以會加速執行。

產生的值會推入至堆疊的頂端。

根據預設，此函式的全域狀態範圍設定為應用程式。 若要變更此項，請參閱[CRT 中的全域狀態](global-state.md)。

## <a name="requirements"></a>需求

**平臺：** x86
