# <a name="debug"></a>_DEBUG
當您指定 /MTd 或 /MDd 選項時，編譯器會定義 `_DEBUG`。 這些選項指定 C 執行階段程式庫的偵錯版本。  
  
 如需詳細資訊，請參閱 [CRT 偵錯技術](/visualstudio/debugger/crt-debugging-techniques)。
