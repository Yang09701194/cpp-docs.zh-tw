# _DEBUG
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

當您指定 \/MTd 或 \/MDd 選項時，編譯器會定義 `_DEBUG` 。  這些選項會指定偵錯 C 執行階段程式庫的版本。  
  
 如需詳細資訊，請參閱 [CRT 偵錯技術](../Topic/CRT%20Debugging%20Techniques.md) 。
