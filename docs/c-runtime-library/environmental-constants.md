# <a name="environmental-constants"></a>環境常數

## <a name="syntax"></a>語法

```
#include <stdlib.h>
```

## <a name="remarks"></a>備註

此常數能定義字串的環境長度。

|常數|意義|
|--------------|-------------|
|`_MAX_ENV`|環境字串的最大字串大小。|
