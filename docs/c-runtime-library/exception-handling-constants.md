# 例外狀況處理常數
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

當例外狀況發生於執行 **try\-except** 陳述式的保護區段中時，傳回 `EXCEPTION_CONTINUE_SEARCH`、 `EXCEPTION_CONTINUE_EXECUTION`或 `EXCEPTION_EXECUTE_HANDLER` 常數。  傳回值決定例外狀況的處理方式。  如需詳細資訊，請參閱 *C\+\+ 語言參考* 中的[try\-except Statement](../cpp/try-except-statement.md)。
