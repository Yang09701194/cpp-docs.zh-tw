# <a name="exitsuccess-exitfailure"></a>EXIT_SUCCESS、EXIT_FAILURE

## <a name="required-header"></a>必要的標頭

```c
#include <stdlib.h>
```

## <a name="remarks"></a>備註

這些是 [exit](reference/exit-exit-exit.md) 和 [_exit](reference/exit-exit-exit.md) 函式的引數，以及 [atexit](reference/atexit.md) 和 [_onexit](reference/onexit-onexit-m.md) 函式的傳回值。

|常數|定義的值|
|-|-|
|EXIT_SUCCESS|0|
|EXIT_FAILURE|1|
