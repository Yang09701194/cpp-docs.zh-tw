# <a name="fopenmax-sysopen"></a>FOPEN_MAX, _SYS_OPEN

## <a name="syntax"></a>語法

```
#include <stdio.h>
```

## <a name="remarks"></a>備註

這是可以同時開啟的檔案數目上限。 `FOPEN_MAX` 是 ANSI 相容名稱。 `_SYS_OPEN` 是為了與現有程式碼相容而提供。
