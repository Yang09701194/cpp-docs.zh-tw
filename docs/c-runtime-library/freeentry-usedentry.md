# <a name="freeentry-usedentry"></a>_FREEENTRY, _USEDENTRY

## <a name="syntax"></a>語法

```
#include <malloc.h>
```

## <a name="remarks"></a>備註

這些常數代表由 `_heapwalk` 常式指派給 **_HEAPINFO** 結構 **_useflag** 元素的值。 這些代表堆積項目的狀態。
