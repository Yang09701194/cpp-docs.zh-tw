# 泛型文字對應
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

為了簡化因應國際化市場撰寫程式碼，泛用文字對應在 TCHAR.H 被定義為:  
  
-   [資料型別](../c-runtime-library/data-type-mappings.md)  
  
-   [常數和全域變數](../c-runtime-library/constant-and-global-variable-mappings.md)  
  
-   [常式對應](../c-runtime-library/routine-mappings.md)  
  
 如需詳細資訊，請參閱 [使用泛用文字的對應](../c-runtime-library/using-generic-text-mappings.md)。  泛用文字對應是與 ANSI 不相容的 Microsoft 擴充。
