# 國際化
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Microsoft 執行階段程式庫會提供許多可用於針對國際市場建立不同程式版本的實用常式。  這包括[地區設定相關常式](../c-runtime-library/locale.md)、寬字元常式、多位元組字元常式和泛用文字常式。  為了方便起見，大部分地區設定相關常式也會根據所執行的作業依此參考進行分類。  在本章節與字母順序參考中，多位元組字元常式和寬字元常式會使用其所在的單一位元組字元對應來描述。  
  
 其中也包含 ISO646 運算子替代項目。  
  
 如需關於 .NET Framework 對應項的詳細資訊，請參閱 [System::Globalization::CultureInfo 類別](https://msdn.microsoft.com/en-us/library/system.globalization.cultureinfo.aspx)。
