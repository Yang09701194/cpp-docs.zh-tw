# <a name="iob"></a>_iob

stdio 控制結構的陣列。

## <a name="syntax"></a>語法

```
FILE _iob[_IOB_ENTRIES];
```

## <a name="remarks"></a>備註

`IOB_ENTRIES` 在 stdio.h 中定義為 20。
