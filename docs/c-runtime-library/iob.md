# _iob
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

標準輸入輸出結構的陣列。  
  
## 語法  
  
```  
FILE _iob[_IOB_ENTRIES];  
```  
  
## 備註  
 `IOB_ENTRIES` 在 stdio.h 中被定義為 20 。
