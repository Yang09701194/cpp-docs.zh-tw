# ISO646 運算子
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

提供某些特定運算子或標點符號的可讀取替代項目。  標準標頭 \<iso646.h\> 即使在獨立式實作也依然可用。

|[not\_eq](../c-runtime-library/reference/not-eq.md)|\!\= 運算子的替代項目。|
