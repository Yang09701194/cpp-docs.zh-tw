# <a name="mbcurmax"></a>MB_CUR_MAX

表示目前地區設定中多位元組字元之最大位元組數目的巨集。

## <a name="syntax"></a>語法

`#include <stdlib.h>`

## <a name="remarks"></a>備註

內容︰ANSI 多位元組及寬字元轉換函式

`MB_CUR_MAX` 值是目前地區設定的多位元組字元的最大位元組數目。
