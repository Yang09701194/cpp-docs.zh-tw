# NULL (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**NULL** 為用於許多指標作業和函式的 null 指標值。  它就相當於 0。  **NULL** 定義於以下標頭檔:CRTDBG.H、LOCALE.H、STDDEF.H、STDIO.H、STDLIB.H、STRING.H、TCHAR.H、TIME.H 和 WCHAR.H。
