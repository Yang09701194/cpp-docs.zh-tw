# <a name="null-crt"></a>NULL (CRT)

**NULL** 是搭配許多指標作業和函式使用的 null 指標值。 它相當於 0。 **NULL** 已在下列標頭檔中定義：CRTDBG.H、LOCALE.H、STDDEF.H、STDIO.H、STDLIB.H、STRING.H、TCHAR.H、TIME.H 及 WCHAR.H。
