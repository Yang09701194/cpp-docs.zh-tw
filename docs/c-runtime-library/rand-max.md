# RAND_MAX
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## 語法  
  
```  
  
#include <stdlib.h>  
  
```  
  
## 備註  
 `RAND_MAX` 常數是可由 `rand` 函式傳回的最大值。  `RAND_MAX` 定義為值 0x7fff。
