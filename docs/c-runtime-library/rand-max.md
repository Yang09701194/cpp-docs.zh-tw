# <a name="randmax"></a>RAND_MAX

## <a name="syntax"></a>語法

```

#include <stdlib.h>

```

## <a name="remarks"></a>備註

常數 `RAND_MAX` 是 `rand` 函式可傳回的最大值。 `RAND_MAX` 是以如 0x7fff 的值來定義。
