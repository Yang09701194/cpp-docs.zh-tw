# <a name="asinh-asinhf-asinhl"></a>asinh、asinhf、asinhl
計算反雙曲正弦。  
  
## <a name="syntax"></a>語法  
  
```  
double asinh(  
   double x   
);  
float asinh(  
   float x   
);  // C++ only  
long double asinh(  
   long double x  
);  // C++ only  
float asinhf(  
   float x   
);  
long double asinhl(  
   long double x  
);  
```  
  
#### <a name="parameters"></a>參數  
 `x`  
 浮點值。  
  
## <a name="return-value"></a>傳回值  
 `asinh` 函式會傳回 `x` 的反雙曲正弦 (Arc 雙曲正弦)。 此函式在浮點網域中有效。 若 `x` 為無訊息 NaN、不確定或無限大，則會傳回相同的值。  
  
|輸入|SEH 例外狀況|`_matherr` 例外狀況|  
|-----------|-------------------|--------------------------|  
|± QNAN、IND、INF|無|無|  
  
## <a name="remarks"></a>備註  
 當您使用 C++ 時，可以呼叫採用並傳回 `asinh` 或 `float` 值的 `long double` 的多載。 在 C 程式中，`asinh` 會一律採用並傳回 `double`。  
  
## <a name="requirements"></a>需求  
  
|函式|C 標頭|C++ 標頭|  
|--------------|--------------|------------------|  
|`asinh`, `asinhf`, `asinhl`|\<math.h>|\<cmath>|  
  
 如需相容性的詳細資訊，請參閱[相容性](../../c-runtime-library/compatibility.md)。  
  
## <a name="example"></a>範例  
  
```C  
// crt_asinh.c  
// Compile by using: cl /W4 crt_asinh.c  
// This program displays the hyperbolic sine of pi / 4  
// and the arc hyperbolic sine of the result.  
  
#include <math.h>  
#include <stdio.h>  
  
int main( void )  
{  
   double pi = 3.1415926535;  
   double x, y;  
  
   x = sinh( pi / 4 );  
   y = asinh( x );  
   printf( "sinh( %f ) = %f\n", pi/4, x );  
   printf( "asinh( %f ) = %f\n", x, y );  
}  
```  
  
```Output  
sinh( 0.785398 ) = 0.868671  
asinh( 0.868671 ) = 0.785398  
```
