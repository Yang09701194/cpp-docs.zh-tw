# <a name="bitand"></a>bitand

& 運算子的替代項目。

## <a name="syntax"></a>語法

```C

#define bitand &

```

## <a name="remarks"></a>備註

巨集會產生運算子

## <a name="example"></a>範例

```cpp
// iso646_bitand.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 1, b = 2, result;

   result = a & b;
   cout << result << endl;

   result= a bitand b;
   cout << result << endl;
}
```

```Output
0
0
```
