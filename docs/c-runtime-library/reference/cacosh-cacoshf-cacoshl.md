# <a name="cacosh-cacoshf-cacoshl"></a>cacosh、cacoshf、cacoshl

擷取複數的反雙曲餘弦值，而且實際軸中有值小於 1 的分支部分。 。

## <a name="syntax"></a>語法

```C
_Dcomplex cacosh(
   _Dcomplex z
);
_Fcomplex cacosh(
   _Fcomplex z
);  // C++ only
_Lcomplex cacosh(
   _Lcomplex z
);  // C++ only
_Fcomplex cacoshf(
   _Fcomplex z
);
_Lcomplex cacoshl(
   _Lcomplex z
);
```

### <a name="parameters"></a>參數

*z*<br/>
代表角度的複數 (弧度)。

## <a name="return-value"></a>傳回值

反雙曲線餘弦*z*，以弧度為單位。 結果是未繫結和非負真實座標軸和間隔 [-iπ，+ iπ] 軸虛數。

## <a name="remarks"></a>備註

因為 c + + 允許多載，所以您可以呼叫的多載**cacosh**採用並傳回 **_Fcomplex**和 **_Lcomplex**值。 在 C 程式中， **cacosh**一律採用並傳回 **_Dcomplex**值。
