# <a name="casin-casinf-casinl"></a>casin、casinf、casinl

擷取與實際的軸間隔 [-1，+ 1] 以外的分支剪下複數的反正弦值。

## <a name="syntax"></a>語法

```C
_Dcomplex casin(
   _Dcomplex z
);
_Fcomplex casin(
   _Fcomplex z
);  // C++ only
_Lcomplex casin(
   _Lcomplex z
);  // C++ only
_Fcomplex casinf(
   _Fcomplex z
);
_Lcomplex casinl(
   _Lcomplex z
);
```

### <a name="parameters"></a>參數

*z*<br/>
代表角度的複數 (弧度)。

## <a name="return-value"></a>傳回值

反正弦值*z*，以弧度為單位。 結果是虛數座標軸和間隔中未繫結 [-π/2 + π/2] 真實的軸。

## <a name="remarks"></a>備註

因為 c + + 允許多載，所以您可以呼叫的多載**casin**採用並傳回 **_Fcomplex**和 **_Lcomplex**值。 在 C 程式中， **casin**一律採用並傳回 **_Dcomplex**值。
