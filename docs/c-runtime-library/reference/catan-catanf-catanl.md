# <a name="catan-catanf-catanl"></a>catan、catanf、catanl
擷取複數的虛數座標軸的間隔 [-1; + 1] 以外的分支剪下的反正切值。  
  
## <a name="syntax"></a>語法  
  
```  
_Dcomplex catan(   
   _Dcomplex z   
);  
_Fcomplex catan(   
   _Fcomplex z   
);  // C++ only  
_Lcomplex catan(   
  _Lcomplex z   
);  // C++ only  
_Fcomplex catanf(   
   _Fcomplex z   
);  
_Lcomplex catanl(   
   _Lcomplex z   
);  
```  
  
#### <a name="parameters"></a>參數  
 `z`  
 代表角度的複數 (弧度)。  
  
## <a name="return-value"></a>傳回值  
 `z` 的反正切值 (弧度)。 結果是虛數座標軸和間隔中未繫結 [-π/2; + π/2] 真實的軸。  
  
## <a name="remarks"></a>備註  
 因為 C++ 允許多載，所以您可以呼叫採用並傳回 `catan` 和 `_Fcomplex` 值的 `_Lcomplex` 的多載。 在 C 程式中， `catan` 會一律採用及傳回 `_Dcomplex` 。  
  
## <a name="requirements"></a>需求  
  
|常式傳回的值|C 標頭|C++ 標頭|  
|-------------|--------------|------------------|  
|`catan`,               `catanf`, `catanl`|\<complex.h>|\<ccomplex>|
