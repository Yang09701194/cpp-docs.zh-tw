# <a name="catan-catanf-catanl"></a>catan、catanf、catanl

擷取複數的虛數座標軸的間隔 [-1; + 1] 以外的分支剪下的反正切值。

## <a name="syntax"></a>語法

```C
_Dcomplex catan( _Dcomplex z );
_Fcomplex catanf( _Fcomplex z );
_Lcomplex catanl( _Lcomplex z );
```

```cpp
_Fcomplex catan( _Fcomplex z );  // C++ only
_Lcomplex catan( _Lcomplex z );  // C++ only
```

### <a name="parameters"></a>參數

*z*<br/>
代表角度的複數 (弧度)。

## <a name="return-value"></a>傳回值

反正切*z*，以弧度為單位。 結果是虛數座標軸和間隔中未繫結 [-π/2; + π/2] 真實的軸。

## <a name="remarks"></a>備註

因為 c + + 允許多載，所以您可以呼叫的多載**catan**採用並傳回 **_Fcomplex**和 **_Lcomplex**值。 在 C 程式中， **catan**一律採用並傳回 **_Dcomplex**值。
