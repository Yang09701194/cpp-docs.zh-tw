# <a name="ccosh-ccoshf-ccoshl"></a>ccosh、ccoshf、ccoshl

擷取複數的雙曲餘弦值。

## <a name="syntax"></a>語法

```C
_Dcomplex ccosh(
   _Dcomplex z
);
_Fcomplex ccosh(
   _Fcomplex z
);  // C++ only
_Lcomplex ccosh(
   _Lcomplex z
);  // C++ only
_Fcomplex ccoshf(
   _Fcomplex z
);
_Lcomplex ccoshl(
   _Lcomplex z
);
```

### <a name="parameters"></a>參數

*z*<br/>
代表角度的複數 (弧度)。

## <a name="return-value"></a>傳回值

雙曲線餘弦*z*，以弧度為單位。

## <a name="remarks"></a>備註

因為 c + + 允許多載，所以您可以呼叫的多載**ccosh**採用並傳回 **_Fcomplex**和 **_Lcomplex**值。 在 C 程式中， **ccosh**一律採用並傳回 **_Dcomplex**值。
