# <a name="cimag-cimagf-cimagl"></a>cimag、cimagf、cimagl
擷取複數的虛數部分。  
  
## <a name="syntax"></a>語法  
  
```  
double cimag(   
   _Dcomplex z   
);  
float cimag(   
   _Fcomplex z   
);  // C++  
long double cimag(   
  _Lcomplex z   
);  // C++  
float cimagf(   
   _Fcomplex z   
);  
long double cimagl(   
   _Lcomplex z   
);  
```  
  
#### <a name="parameters"></a>參數  
 `z`  
 複數。  
  
## <a name="return-value"></a>傳回值  
 `z` 的虛數部分。  
  
## <a name="remarks"></a>備註  
 因為 C++ 允許多載，所以您可以呼叫採用 `_Fcomplex` 或 `_Lcomplex` 值並傳回 `float` 或 `long double` 值之 `cimag` 的多載。 在 C 程式中，`cimag` 會一律採用 `_Dcomplex` 值並傳回 `double` 值。  
  
## <a name="requirements"></a>需求

|-------------|--------------|------------------|
