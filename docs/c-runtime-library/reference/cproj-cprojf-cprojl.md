# <a name="cproj-cprojf-cprojl"></a>cproj、cprojf、cprojl
擷取黎曼球面上複數的投影。  
  
## <a name="syntax"></a>語法  
  
```  
_Dcomplex cproj(   
   _Dcomplex z   
);  
_Fcomplex cproj(   
   _Fcomplex z   
);  // C++ only  
_Lcomplex cproj(   
   _Lcomplex z   
);  // C++ only  
_Fcomplex cprojf(   
   _Fcomplex z   
);  
_Lcomplex cprojl(   
   _Lcomplex z   
);  
```  
  
#### <a name="parameters"></a>參數  
 `z`  
 複數。  
  
## <a name="return-value"></a>傳回值  
 黎曼球面上 `z` 的投影。  
  
## <a name="remarks"></a>備註  
 因為 C++ 允許多載，所以您可以呼叫採用並傳回 `cproj` 和 `_Fcomplex` 值的 `_Lcomplex` 的多載。 在 C 程式中， `cproj` 會一律採用及傳回 `_Dcomplex` 。  
  
## <a name="requirements"></a>需求  
  
|常式傳回的值|C 標頭|C++ 標頭|  
|-------------|--------------|------------------|  
|`cproj`,               `cprojf`, `cprojl`|\<complex.h>|\<ccomplex>|  
  
 如需相容性詳細資訊，請參閱簡介中的 [相容性](../../c-runtime-library/compatibility.md) 。
