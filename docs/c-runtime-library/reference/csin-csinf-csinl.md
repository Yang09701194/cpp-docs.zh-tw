# <a name="csin-csinf-csinl"></a>csin、csinf、csinl

擷取複數的正弦值。

## <a name="syntax"></a>語法

```C
_Dcomplex csin(
   _Dcomplex z
);
_Fcomplex csin(
   _Fcomplex z
);  // C++ only
_Lcomplex csin(
   _Lcomplex z
);  // C++ only
_Fcomplex csinf(
   _Fcomplex z
);
_Lcomplex csinl(
   _Lcomplex z
);
```

### <a name="parameters"></a>參數

*z*<br/>
代表角度的複數 (弧度)。

## <a name="return-value"></a>傳回值

正弦*z*，以弧度為單位。

## <a name="remarks"></a>備註

因為C++允許多載，您可以呼叫多載**csin**採用並傳回 **_Fcomplex**並 **_Lcomplex**的值。 在 C 程式中， **csin**一律採用並傳回 **_Dcomplex**值。

## <a name="requirements"></a>需求

|常式傳回的值|C 標頭|C++ 標頭|
|-------------|--------------|------------------|
|**csin**,               **csinf**, **csinl**|\<complex.h>|\<ccomplex>|

如需相容性的詳細資訊，請參閱 [相容性](../../c-runtime-library/compatibility.md)。
