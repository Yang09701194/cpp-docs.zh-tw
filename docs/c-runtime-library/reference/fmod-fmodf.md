# <a name="fmod-fmodf-fmodl"></a>fmod、fmodf、fmodl

計算浮點餘數。

## <a name="syntax"></a>語法

```C
double fmod(
   double x,
   double y
);
float fmod(
   float x,
   float y
);  // C++ only
long double fmod(
   long double x,
   long double y
);  // C++ only
float fmodf(
   float x,
   float y
);
long double fmodl(
   long double x,
   long double y
);
```

### <a name="parameters"></a>參數

*x*, *y*<br/>
浮點值。

## <a name="return-value"></a>傳回值

**fmod**傳回*x* / *y*的浮點餘數。 如果*y*的值為 0.0,fmod 將返回一個安靜**fmod**的 NaN。 有關**printf**家族表示安靜 NaN 的資訊,請參閱[printf](printf-printf-l-wprintf-wprintf-l.md)。

## <a name="remarks"></a>備註

**fmod**函數計算*x* / *y*的浮點餘數*f,* 以便*x* = *i* \* *y* + *f*(其中*i*是整數 *),f*與*x*具有相同的符號,並且*f*的絕對值小於*y*的絕對值。

C++允許重載,因此您可以呼叫帶和返回**浮點**和**長****雙**值的**fmod**重載。 在 C 程式中 **,fmod**始終採用兩**個雙**參數並傳回**一個雙**參數 。

默認情況下,此函數的全域狀態範圍為應用程式。 要改變此情況,請參閱[CRT 中的全域狀態](../global-state.md)。

## <a name="example"></a>範例

```C
// crt_fmod.c
// This program displays a floating-point remainder.

#include <math.h>
#include <stdio.h>

int main( void )
{
   double w = -10.0, x = 3.0, z;

   z = fmod( w, x );
   printf( "The remainder of %.2f / %.2f is %f\n", w, x, z );
}
```

```Output
The remainder of -10.00 / 3.00 is -1.000000
```
