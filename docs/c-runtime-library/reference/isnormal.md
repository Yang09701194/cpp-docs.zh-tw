# <a name="isnormal"></a>isnormal

判斷浮點值是否為無限大的值。

## <a name="syntax"></a>語法

```C
int isnormal(
   /* floating-point */ x
); /* C-only macro */

template <class FloatingType>
inline bool isnormal(
   FloatingType x
) throw(); /* C++-only template function */
```

### <a name="parameters"></a>參數

*x*<br/>
要測試的浮點值。

## <a name="return-value"></a>傳回值

**isnormal**傳回非零值 (**，則為 true** c + + 程式碼中) 如果引數*x*是有限且未 subnormal。 **isnormal**會傳回 0 (**false** c + + 程式碼中) 如果引數為 subnormal、 無限大或 NAN。

## <a name="remarks"></a>備註

**isnormal**當編譯為 C 和內嵌範本函式編譯為 c + + 時，使用巨集。
