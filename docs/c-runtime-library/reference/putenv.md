# putenv
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這個 POSIX 函式已不建議使用。  使用 ISO C\+\+ 符合 [\_putenv](../../c-runtime-library/reference/putenv-wputenv.md) 或安全性增強的 [\_putenv\_s](../../c-runtime-library/reference/putenv-s-wputenv-s.md) 。  
  
> [!IMPORTANT]
>  這個應用程式開發介面不能用於 Windows 執行階段執行的應用程式。  如需詳細資訊，請參閱 [\/ZW 不支援 CRT 函式](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx)。
