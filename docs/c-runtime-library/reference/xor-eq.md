# <a name="xoreq"></a>xor_eq
^= 運算子的替代項目。  
  
## <a name="syntax"></a>語法  
  
```  
  
#define xor_eq ^=  
  
```  
  
## <a name="remarks"></a>備註  
 巨集會產生 ^= 運算子。  
  
## <a name="example"></a>範例  
  
```  
// iso646_xor_eq.cpp  
// compile with: /EHsc  
#include <iostream>  
#include <iso646.h>  
  
int main( )  
{  
   using namespace std;  
   int a = 3, b = 2, result;  
  
   result= a ^= b;  
   cout << result << endl;  
  
   a = 3;  
   b = 2;  
  
   result= a xor_eq b;  
   cout << result << endl;  
}  
```  
  
```Output  
1  
1  
```
