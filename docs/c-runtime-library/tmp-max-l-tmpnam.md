# <a name="tmpmax-ltmpnam"></a>TMP_MAX、L_tmpnam

## <a name="syntax"></a>語法

```
#include <stdio.h>
```

## <a name="remarks"></a>備註

`TMP_MAX` 是 `tmpnam` 函式可以產生的唯一檔案名稱數目。 `L_tmpnam` 是 `tmpnam` 產生的暫存檔案名稱長度。
