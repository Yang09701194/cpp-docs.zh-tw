# Windows 平台 (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

[!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] 的 C 執行階段程式庫支援 [!INCLUDE[win8](../build/includes/win8_md.md)]， [!INCLUDE[winserver8](../build/includes/winserver8_md.md)]， [!INCLUDE[win7](../build/includes/win7_md.md)]， [!INCLUDE[winsvr08](../build/includes/winsvr08_md.md)]， Windows Vista， x86 的 [!INCLUDE[winxp](../build/includes/winxp_md.md)] Service Pack 2 \(SP2\) ， x64 的 [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] Service Pack 2 \(SP2\) ，x86 和 x64 的 [!INCLUDE[winxp](../build/includes/winxp_md.md)] Service Pack 3 \(SP3\) 。  所有這些作業系統支援 Windows 應用程式開發介面 \(API\) 並提供 Unicode 支援。  此外，任何 Win32 應用程式可以使用多位元組字元集 \(MBCS\) 。  
  
> [!NOTE]
>  [!INCLUDE[vs_dev11_long](../build/includes/vs_dev11_long_md.md)] 不支援 [!INCLUDE[winxp](../build/includes/winxp_md.md)] 和 [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] 開發。  您必須安裝 [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] 加入 [!INCLUDE[winxp](../build/includes/winxp_md.md)] 和 [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] 的支援。
