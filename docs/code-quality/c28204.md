# <a name="c28204"></a>C28204

> 警告 C28204： \<函式 >：只有這個多載的其中一個，以及 \<filename > （\<行 >）的其中一個會針對 \<paramname > 加上批註：兩個或兩者都不需要加上批註。

當注釋中有錯誤時，就會回報此警告。
