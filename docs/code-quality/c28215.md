# <a name="c28215"></a>C28215

> 警告 C28215： typefix 不能套用至已有 typefix 的參數

將 `__typefix` 注釋套用至已經有該注釋的參數會是錯誤。 只有幾個特殊情況下才會使用 `__typefix` 注釋，而且一般使用中不應顯示此警告。
