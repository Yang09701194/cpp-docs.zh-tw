# <a name="c28219"></a>C28219

> 警告 C28219：注釋中的參數需要列舉的成員

注釋的參數必須是已命名 `enum` 類型的成員，而且遇到一些其他符號;使用該 `enum` 類型的成員。 這通常表示批註宏的編碼不正確。
