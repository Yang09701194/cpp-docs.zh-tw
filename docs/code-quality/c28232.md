# <a name="c28232"></a>C28232

> 警告 C28232： \_前置\_、\_Post\_或 \_Deref\_ 未套用至任何注釋

此警告表示 `_Pre_`、`_Post_`或 `_Deref_` 運算子出現在注釋運算式中，但沒有後續的功能批註。已忽略修飾詞，但這表示批註的編碼不正確。
