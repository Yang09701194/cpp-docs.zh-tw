# <a name="c28243"></a>C28243

> 警告 C28243：參數上的函式注釋所需的取值比實際類型批註允許的更多

註釋上 `__deref` 運算子的數目大於參數類型定義的指標層級數目。 請變更註釋中的數目或所參考參數的指標層級，以修正這個警告。
