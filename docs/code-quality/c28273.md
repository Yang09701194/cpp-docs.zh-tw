# <a name="c28273"></a>C28273

> 警告 C28273：針對函式，線索與函式宣告不一致

此警告表示注釋中的錯誤，而不是在要分析的程式碼中。 出現在函式定義上的注釋與宣告上出現的批註不一致。 這兩個批註應解析為符合。
