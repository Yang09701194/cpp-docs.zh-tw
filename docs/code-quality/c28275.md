# <a name="c28275"></a>C28275

> 警告 C28275： \_宏\_值\_ 的參數為 null

此警告表示模型檔案中發生內部錯誤，而不是在要分析的程式碼中。 在沒有參數的情況下呼叫*macroValue*函式。
