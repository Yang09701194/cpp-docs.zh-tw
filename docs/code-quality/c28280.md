# <a name="c28280"></a>C28280

> 警告 C28280：若為符號，找到 ' end ' 但沒有相符的 ' begin '

注釋語言支援 begin 和 end （`{` 和 `}` C）結構，而配對已取得不平衡。 如果使用宏，就可以避免這種情況。
