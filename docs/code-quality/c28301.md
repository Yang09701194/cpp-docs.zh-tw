# <a name="c28301"></a>C28301

> 警告 C28301： \<函數 > 的第一個宣告沒有注釋。\<note1 > 參閱 \<filename > （\<行 >）。 \<note2 >

當指定函式的第一個宣告中找不到注釋時，就會回報此警告。
