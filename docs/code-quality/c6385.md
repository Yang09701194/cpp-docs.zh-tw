# <a name="c6385"></a>C6385

> 警告 C6385：不正確資料：存取 \<的緩衝區名稱 >，可讀取的大小 \<size1 > 個位元組，但可能會讀取 \<size2 > 位元組：行： x，y

此警告表示指定緩衝區的可讀取範圍可能小於用來讀取它的索引。 嘗試讀取有效範圍以外的資料會導致緩衝區溢位。

## <a name="example"></a>範例

下列程式碼會產生這個警告：

```cpp
void f(int i)
{
   char a[20];
   char j;
   if (i <= 20)
   {
      j = a[i];
   }
}
```

若要修正這個警告，請使用下列程式碼：

```cpp
void f(int i)
{
   char a[20];
   char j;
   if (i < 20)
   {
      j = a[i];
   }
}
```
