# <a name="c6530"></a>C6530

> 警告 C6530：無法辨識的格式字串樣式 \<名稱 >

> [!NOTE]
> 只有使用舊版程式碼注釋語言（SAL）的程式碼時，才會出現這個警告。 我們建議您移植程式碼，以使用最新版本的 SAL。 如需詳細資訊，請參閱[使用 SAL 注釋減少 CC++ /程式碼](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)缺失。

此警告表示 [格式字串] 屬性使用 scanf 或 printf 以外的值。 若要修正這個警告，請檢查您的程式碼，並使用 Style 屬性的有效值。

## <a name="example"></a>範例

下列程式碼會產生此警告，因為 `Style` 屬性中出現錯誤：

```cpp
// C
#include <CodeAnalysis\SourceAnnotations.h>
void f([SA_FormatString(Style="printfd")] char *px);

// C++
#include <CodeAnalysis\SourceAnnotations.h>
using namespace vc_attributes;
void f([FormatString(Style="printfd")] char *px);
```

若要更正這個警告，請使用有效的樣式，如下列程式碼所示：

```cpp
// C
#include <CodeAnalysis\SourceAnnotations.h>
void f([SA_FormatString(Style="printf")] char *px);

// C++
#include <CodeAnalysis\SourceAnnotations.h>
using namespace vc_attributes;
void f([FormatString(Style="printf")] char *px);
```
