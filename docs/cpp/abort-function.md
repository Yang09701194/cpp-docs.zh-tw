# <a name="abort-function"></a>abort 函式
**中止**函式，也在標準 include 檔 STDLIB 宣告。H，會終止 c + + 程式。 之間的差異**結束**和**中止**在於**結束**允許執行 c + + 執行階段終止處理 （全域物件解構函式會呼叫），而**中止**會立即終止程式。 如需詳細資訊，請參閱[中止](../c-runtime-library/reference/abort.md)中*執行階段程式庫參考*。
