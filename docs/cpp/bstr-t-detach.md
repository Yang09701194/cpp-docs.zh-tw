# <a name="bstrtdetach"></a>_bstr_t::Detach
**Microsoft 特定的**  
  
 傳回 `BSTR` 所包裝的 `_bstr_t`，並將 `BSTR` 與 `_bstr_t` 中斷連結。  
  
## <a name="syntax"></a>語法  
  
```  
  
BSTR Detach( ) throw;  
  
```  
  
## <a name="return-value"></a>傳回值  
 `BSTR` 包裝 `_bstr_t`。  
  
## <a name="example"></a>範例  
 請參閱[_bstr_t:: assign](../cpp/bstr-t-assign.md)範例使用**卸離**。
