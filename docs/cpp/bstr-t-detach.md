# <a name="_bstr_tdetach"></a>_bstr_t::Detach

**Microsoft 專屬**

傳回 `BSTR` 所包裝的 `_bstr_t`，並將 `BSTR` 與 `_bstr_t` 中斷連結。

## <a name="syntax"></a>語法

```
BSTR Detach( ) throw;
```

## <a name="return-value"></a>傳回值

`BSTR` 包裝 `_bstr_t`。

## <a name="example"></a>範例

如需使用卸**離**的範例，請參閱[_Bstr_t：： Assign](../cpp/bstr-t-assign.md) 。
