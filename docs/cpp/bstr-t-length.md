# <a name="bstrtlength"></a>_bstr_t::length
**Microsoft 特定的**  
  
 在 `_bstr_t` 中傳回封裝 `BSTR` 的字元數目，不計結束的 null 字元。  
  
## <a name="syntax"></a>語法  
  
```  
  
unsigned int length ( ) const throw( );  
  
```
