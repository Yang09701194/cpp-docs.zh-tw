# <a name="bstrtoperator-"></a>_bstr_t::operator !
**Microsoft 特定的**  
  
 如果會檢查封裝`BSTR`是**NULL**字串。  
  
## <a name="syntax"></a>語法  
  
```  
  
bool operator!( ) const throw( );  
  
```  
  
## <a name="return-value"></a>傳回值  
 它會傳回**true**如果是， **false**如果不是。  
  
 **END Microsoft 特定的**
