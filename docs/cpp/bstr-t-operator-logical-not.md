# <a name="bstrtoperator-"></a>_bstr_t::operator !

**Microsoft 專屬**

檢查封裝`BSTR`是 NULL 字串。

## <a name="syntax"></a>語法

```
bool operator!( ) const throw( );
```

## <a name="return-value"></a>傳回值

如果是，則會傳回 TRUE，如果沒有，則為 FALSE。
