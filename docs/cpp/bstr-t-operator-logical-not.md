# _bstr_t::operator !
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Microsoft 特定的**  
  
 檢查封裝的 `BSTR` 是否為 **NULL** 字串。  
  
## 語法  
  
```  
  
bool operator!( ) const throw( );  
  
```  
  
## 傳回值  
 如果是則傳回 **true**，否則便傳回 **false**。  
  
 **END Microsoft 特定的**
