# <a name="comerroroperator-"></a>_com_error::operator =

**Microsoft 專屬**

將現有的 `_com_error` 物件指派給另一個物件。

## <a name="syntax"></a>語法

```
_com_error& operator = (
   const _com_error& that
) throw ( );
```

#### <a name="parameters"></a>參數

*that*<br/>
`_com_error` 物件。

**結束 Microsoft 專屬**
