# <a name="comptrt-relational-operators"></a>_com_ptr_t 關係運算子

**Microsoft 專屬**

## <a name="remarks"></a>備註

比較的智慧型指標物件到另一個智慧型指標、 一般介面指標，或為 NULL。 除了 NULL 指標的測試，這些運算子會先查詢兩個指標`IUnknown`，並比較結果。
