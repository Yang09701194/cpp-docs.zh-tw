# <a name="indirection-on-array-types"></a>陣列類型的間接取值

使用間接取值運算子 (<strong>\*</strong>) 上*n*-維度陣列型別會產生*n*-1 維陣列。 如果*n*為 1，產生純量 （或陣列元素）。
