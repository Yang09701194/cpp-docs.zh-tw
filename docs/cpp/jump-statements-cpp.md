# <a name="jump-statements-c"></a>跳躍陳述式 (C++)
C ++ 跳躍陳述式會立即轉移區域的控制權。  
  
## <a name="syntax"></a>語法  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
