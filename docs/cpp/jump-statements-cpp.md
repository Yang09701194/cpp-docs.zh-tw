# 跳躍陳述式 (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C \+\+ 跳躍陳述式會立即轉移區域的控制權。  
  
## 語法  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
