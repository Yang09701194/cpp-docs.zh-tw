# <a name="m128d"></a>__m128d
## <a name="microsoft-specific"></a>Microsoft 特定的  
 `__m128d` Emmintrin.h 中定義資料類型，用於 Streaming SIMD Extensions 2 指令內建函式。  
  
```  
// data_types__m128d.cpp  
#include <emmintrin.h>  
int main() {  
   __m128d x;  
}  
```  
  
## <a name="remarks"></a>備註  
 您不可以直接存取 `__m128d` 欄位， 不過，可以在偵錯工具中看到這些類型。 類型 `__m128` 的變數對應到 XMM[0-7] 暫存器。  
  
 類型的變數**_m128d**會自動在 16 位元組界限上對齊。  
  
 ARM 處理器不支援 `__m128d` 資料類型。  
  
**結束 Microsoft 特定的**
