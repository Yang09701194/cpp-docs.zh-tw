# <a name="m128i"></a>__m128i
## <a name="microsoft-specific"></a>Microsoft 特定的  
 `__m128i`中定義資料類型，用於 Streaming SIMD Extensions 2 (SSE2) 指示內建函式， \<emmintrin.h >。  
  
```  
// data_types__m128i.cpp  
#include <emmintrin.h>  
int main() {  
   __m128i x;  
}  
```  
  
## <a name="remarks"></a>備註  
 您不應該存取**__m128i**直接欄位。 不過，可以在偵錯工具中看到這些類型。 類型 `__m128i` 的變數對應到 XMM[0-7] 暫存器。  
  
 類型的變數**_m128i**會自動在 16 位元組界限上對齊。  
  
> [!NOTE]
>  使用型別的變數`__m128i`會導致編譯器產生 SSE2 **movdqa**指令。 此指示 Pentium III 處理器不會造成錯誤，但將會導致無訊息的失敗，產生的任何指示可能的副作用與**movdqa** Pentium III 處理器上轉譯。  
  
 ARM 處理器不支援 `__m128i` 資料類型。  
  
**結束 Microsoft 特定的**
