# <a name="m64"></a>__m64
## <a name="microsoft-specific"></a>Microsoft 特定的  
 `__m64` 資料類型用於 MMX 和 3DNow!  內建函式，而定義在\<xmmintrin.h >。  
  
```  
// data_types__m64.cpp  
#include <xmmintrin.h>  
int main()  
{  
   __m64 x;  
}  
```  
  
## <a name="remarks"></a>備註  
 您不可以直接存取 `__m64` 欄位， 不過，可以在偵錯工具中看到這些類型。 類型 `__m64` 的變數對應到 MM[0-7] 暫存器。  
  
 類型的變數 **_m64**會自動對齊 8 位元組界限上。  
  
 `__m64` 處理器不支援 [!INCLUDE[vcprx64](../assembler/inline/includes/vcprx64_md.md)] 資料類型。 將 __m64 做為 MMX 內建一部分使用的應用程式，必須予以重新撰寫，才能使用對等的 SSE 和 SSE2 內建。  
  
**結束 Microsoft 特定的**
