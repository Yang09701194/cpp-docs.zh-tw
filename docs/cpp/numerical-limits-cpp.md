# 數值限制 (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

兩個標準的 Include 檔 \(LIMITS.H 和 FLOAT.H\) 定義了數字指定類型的變數可以保留的限制或最小和最大值。  這些最小值和最大值保證可以移植到與 ANSI C 使用相同資料表示的所有 C\+\+ 編譯器。  LIMITS.H Include 檔定義了[整數類資料類型的數值限制](../cpp/integer-limits.md)，而 FLOAT.H 則定義了[浮動類型的數值限制](../cpp/floating-limits.md)。
