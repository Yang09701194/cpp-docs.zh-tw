# <a name="numerical-limits-c"></a>數值限制 (C++)
兩個標準的 Include 檔 (LIMITS.H 和 FLOAT.H) 定義了數字指定類型的變數可以保留的限制或最小和最大值。 這些最小值和最大值保證可以移植到與 ANSI C 使用相同的資料表示任何 c + + 編譯器限制。H include 檔定義[整數類資料類型的數值限制](../cpp/integer-limits.md)，和 FLOAT。H 定義[浮點類型的數值限制](../cpp/floating-limits.md)。
