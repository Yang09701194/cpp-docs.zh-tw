# <a name="numerical-limits-c"></a>數值限制 (C++)

兩個標準 include 檔案， \<limits.h > 和\<float.h >，定義數值的限制或指定型別的變數可以保留的最小和最大值。 這些最小值和最大值保證可以移植到任何與 ANSI C 使用相同的資料表示法的 c + + 編譯器\<Limits.h > 包括檔案會定義[整數類資料類型的數值限制](../cpp/integer-limits.md)，以及\<float.h > 定義[浮動類型的數值限制](../cpp/floating-limits.md)。
