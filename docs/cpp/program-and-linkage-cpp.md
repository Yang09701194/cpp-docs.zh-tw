# 程式和連結 (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

程式是由一個或多個連結在一起的轉譯單位所構成。  執行 \(概念上\) 是從包含 **main** 函式的轉譯單位開始   \(如需轉譯單位的詳細資訊，請參閱《前置處理器參考》[](../preprocessor/phases-of-translation.md "Phases of Translation")中的*轉譯階段*。 如需 **main** 函式的詳細資訊，請參閱[程式啟動：main 函式](../cpp/main-program-startup.md)\)。
