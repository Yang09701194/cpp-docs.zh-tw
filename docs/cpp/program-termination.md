# 程式終止
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C\+\+ 提供數種結束程式的方法：  
  
-   呼叫 [exit](../cpp/exit-function.md) 函式。  
  
-   呼叫 [abort](../cpp/abort-function.md) 函式。  
  
-   從 **main** 執行 [return](../cpp/return-statement-in-program-termination-cpp.md) 陳述式。
