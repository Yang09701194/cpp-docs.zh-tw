# <a name="punctuators-c"></a>標點符號 （c + +）
在 C++ 中的標點符號對於編譯器具有語法和語意上的含意，但本身並不指定產生值的作業。 不論是單獨或組合，有些標點符號也可以是 C++ 運算子，或對前置處理器具有重大意義。  

 下列任何字元皆視為標點符號：  

```  
! % ^ & * ( ) - + = { } | ~  
[ ] \ ; ' : " < > ? , . / #  
```  

 標點符號 **[]**， **（)**，和 **{}** 必須出現在之後組[轉譯階段](../preprocessor/phases-of-translation.md)4。
