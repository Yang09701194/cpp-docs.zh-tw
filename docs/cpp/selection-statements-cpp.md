# <a name="selection-statements-c"></a>選擇陳述式 (C++)

C + + 選擇陳述式[如果](../cpp/if-else-statement-cpp.md)並[切換](../cpp/switch-statement-cpp.md)，提供依條件執行的程式碼區段的方法。

[__If_exists](../cpp/if-exists-statement.md)並[__if_not_exists](../cpp/if-not-exists-statement.md)陳述式可讓您有條件地包含根據符號是否存在的程式碼。

請參閱個別主題，了解每個陳述式的語法。
