# <a name="selection-statements-c"></a>選擇陳述式 (C++)

 [__If_exists](../cpp/if-exists-statement.md)和[__if_not_exists](../cpp/if-not-exists-statement.md)陳述式可讓您有條件地包含程式碼，根據符號是否存在。
