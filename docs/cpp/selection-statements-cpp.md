# <a name="selection-statements-c"></a>選擇陳述式 (C++)

C++選取範圍語句（[如果](../cpp/if-else-statement-cpp.md)和[參數](../cpp/switch-statement-cpp.md)）提供了有條件地執行程式碼區段的方式。

[__If_exists](../cpp/if-exists-statement.md)和[__if_not_exists](../cpp/if-not-exists-statement.md)語句可讓您根據符號是否存在，有條件地包含程式碼。

請參閱個別主題，了解每個陳述式的語法。
