# <a name="using-atexit"></a>使用 atexit

具有[atexit](../c-runtime-library/reference/atexit.md)函式，您可以指定在程式結束之前執行的結束處理函式。 任何呼叫之前初始化的全域靜態物件**atexit**會終結之前執行結束處理函式。
