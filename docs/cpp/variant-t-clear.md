# <a name="varianttclear"></a>_variant_t::Clear

**Microsoft 專屬**

清除封裝`VARIANT`物件。

## <a name="syntax"></a>語法

```
void Clear( );
```

## <a name="remarks"></a>備註

呼叫`VariantClear`上的已封裝`VARIANT`物件。
