# <a name="_variant_tclear"></a>_variant_t::Clear

**Microsoft 專屬**

清除封裝的 `VARIANT` 物件。

## <a name="syntax"></a>語法

```
void Clear( );
```

## <a name="remarks"></a>備註

在封裝的 `VARIANT` 物件上呼叫 `VariantClear`。
