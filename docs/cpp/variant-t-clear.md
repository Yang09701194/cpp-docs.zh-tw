# _variant_t::Clear
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Microsoft 特定的**  
  
 清除封裝的 **VARIANT** 物件。  
  
## 語法  
  
```  
  
void Clear( );  
  
```  
  
## 備註  
 在封裝的 **VARIANT** 物件上呼叫 **VariantClear**。
