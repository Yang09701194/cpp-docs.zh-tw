# <a name="virtual-specifier"></a>virtual 規範

[Virtual](../cpp/virtual-cpp.md)關鍵字只能套用至非靜態類別成員函式。 它表示將函式呼叫的繫結延後到執行階段。 如需詳細資訊，請參閱[虛擬](../cpp/virtual-functions.md)函式。
