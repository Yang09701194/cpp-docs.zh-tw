# Agile::Release 方法
捨棄目前 Agile 物件的基礎物件和內容。  
  
## 語法  
  
```cpp  
  
void Release() throw();  
  
```  
  
## 備註  
 捨棄目前 Agile 物件的基礎物件和內容 \(如果存在的話\)，然後將 Agile 物件的值設為 null。
