# COMException::COMException 建構函式
初始化 COMException 類別的新執行個體。  
  
## 語法  
  
```cpp  
COMException( int hresult )  
```  
  
#### 參數  
 hresult  
 由例外狀況表示的錯誤 HRESULT。
