# default::(type_name)::GetType 方法
傳回 Platform::Type^，表示目前的類型。  
  
## 語法  
  
```cpp  
  
Platform::Type^ GetType()  
```  
  
## 傳回值  
 表示目前物件的 [Platform::Type](../cppcx/platform-type-class.md)^ 物件。
