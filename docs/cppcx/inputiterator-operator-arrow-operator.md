# InputIterator::operator-&gt; 運算子
擷取目前 InputIterator 指定的項目位址。  
  
## 語法  
  
```  
pointer operator->() const;  
```  
  
## 傳回值  
 目前 InputIterator 指定的項目位址。  
  
## 需求  
 **標頭：**collection.h  
  
 **命名空間：**Platform::Collections
