# 命名空間參考 (C++/CX)
本文件的這一節說明支援 [!INCLUDE[cppwrt](../cppcx/includes/cppwrt-md.md)] \([!INCLUDE[cppwrt_short](../cppcx/includes/cppwrt-short-md.md)]\) 編譯器的命名空間。  
  
## 編譯器提供的命名空間  
 為了讓以 [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)]為目標的程式更易於編碼，[!INCLUDE[cppwrt_short](../cppcx/includes/cppwrt-short-md.md)] 編譯器及其支援的標頭檔提供了可定義多種型別的命名空間。 這些命名空間可定義內建的數值型別，字串、陣列與集合，表示 [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)]錯誤的 Visual C\+\+ 例外狀況，適用於標準 [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)]型别的語言強化功能。
