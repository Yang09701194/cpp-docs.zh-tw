# <a name="namespaces-reference-ccx"></a>命名空間參考 (C++/CX)

在本節中的文章，文件說明命名空間，可以支援編譯器的 C + + /CX。

## <a name="compiler-supplied-namespaces"></a>編譯器提供的命名空間

若要簡化程式碼撰寫的程式為目標的 Windows 執行階段 C + + /CX 編譯器和其支援的標頭檔提供可定義多種型別的命名空間。 命名空間定義內建的數值型別，字串、 陣列和集合，Visual c + + 例外狀況表示 Windows 執行階段錯誤。與標準的 Windows 執行階段類型的特定語言的增強功能。
