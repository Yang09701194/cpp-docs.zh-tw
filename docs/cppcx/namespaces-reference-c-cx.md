# <a name="namespaces-reference-ccx"></a>命名空間參考 (C++/CX)
本節中的文件說明命名空間，可以支援編譯器的 C + + /CX。  
  
## <a name="compiler-supplied-namespaces"></a>編譯器提供的命名空間  
 若要簡化程式更易於編碼的目標 Windows Runtime C + + /CX 編譯器及其支援的標頭檔提供了可定義多種型別的命名空間。 命名空間可定義內建的數值型別，字串、 陣列和集合。Visual c + + 例外狀況代表 Windows 執行階段錯誤。與標準的 Windows 執行階段類型的特定語言增強功能。
