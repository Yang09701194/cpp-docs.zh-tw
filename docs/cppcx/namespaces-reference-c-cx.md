## <a name="compiler-supplied-namespaces"></a>編譯器提供的命名空間

為了簡化以 Windows 執行階段為目標之程式的編碼， C++/cx 編譯器及其支援的標頭檔會提供定義各種類型的命名空間。 命名空間會定義內建的數位類型。字串、陣列和集合;C++表示 Windows 執行階段錯誤的例外狀況;標準 Windows 執行階段類型的語言特定增強功能。
