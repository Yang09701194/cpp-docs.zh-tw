# Platform::Boolean 實值類別
表示布林值。 相當於 `bool`。  
  
## 語法  
  
```cpp  
public value struct Boolean  
```  
  
## 成員  
 布林值具有衍生自 [Platform::Object 類別](../cppcx/platform-object-class.md) 的 Equals\(\)、GetHashCode\(\) 和 ToString\(\) 方法，以及衍生自 [Platform::Type 類別](../cppcx/platform-type-class.md) 的 GetTypeCode\(\) 方法。
