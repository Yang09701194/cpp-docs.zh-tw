# <a name="platformclassnotregisteredexception-class"></a>Platform::ClassNotRegisteredException 類別
在 COM 類別未登錄時擲回。  
  
## <a name="syntax"></a>語法  
  
```cpp  
public ref class ClassNotRegisteredException : COMException,    IException,    IPrintable,    IEquatable  
```  
  
### <a name="remarks"></a>備註  
 如需詳細資訊，請參閱 [COMException](../cppcx/platform-comexception-class.md) 類別。  
  
### <a name="requirements"></a>需求  
 **最低支援用戶端：** Windows 8  
  
 **最低支援伺服器：** Windows Server 2012  
  
 **命名空間：** Platform  
  
 **中繼資料：** platform.winmd
