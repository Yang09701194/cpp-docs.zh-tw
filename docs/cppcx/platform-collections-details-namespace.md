# Platform::Collections::Details 命名空間
此命名空間支援 `Platform` 基礎結構，而且不可直接從程式碼使用。  
  
## 語法  
  
```cpp  
  
namespace Platform {  namespace Collections {    namespace Details {}}}  
```  
  
## Members  
 此命名空間的成員定義在 collection.h 中，而且不會顯示在物件瀏覽器中。

## 需求  
 **標頭：**Collection.h  
  
 **命名空間：**Platform::Collection::Details
