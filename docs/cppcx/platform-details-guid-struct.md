# <a name="platformdetailsguid-struct"></a>Platform::Details::__GUID 結構
這個結構僅供內部使用，不適用於開發用途。  
  
## <a name="syntax"></a>語法  
  
```cpp

```  
  
### <a name="remarks"></a>備註
