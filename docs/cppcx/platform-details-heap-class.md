# <a name="platformdetailsheap-class"></a>Platform::Details::Heap 類別
這個類別僅供內部使用，不適用於開發用途。  
  
## <a name="syntax"></a>語法  
  
```cpp  
  
ref class Heap sealed;  
```  
  
### <a name="remarks"></a>備註
