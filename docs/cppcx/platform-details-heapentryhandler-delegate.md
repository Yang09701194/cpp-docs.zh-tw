# <a name="platformdetailsheapentryhandler-delegate"></a>Platform::Details::HeapEntryHandler 委派
這個委派僅供內部使用，不適用於開發用途。  
  
## <a name="syntax"></a>語法  
  
```cpp  
  
delegate HeapEntryHandler;  
```  
  
### <a name="remarks"></a>備註  
 這個類別只為完整性而提供，因為它可用瀏覽器進行檢查。
