# Platform::Details 命名空間
這個命名空間僅供內部使用，不適用於開發用途。  
  
## 語法  
  
```cpp  
  
namespace Platform {  
   namespace Details {  
}}  
```

 雖然此命名空間僅適合內部使用，但瀏覽器仍會顯示此命名空間的下列成員。  
  
|名稱|備註|  
|--------|--------|  
|主控台|類別。 在單元測試中顯示輸出。|  
|\_GUID|結構|  
|Heap|類別|  
|HeapAllocationTrackingLevel|列舉|  
|HeapEntryHandler|委派|

|IWeakReferenceSource|介面|
