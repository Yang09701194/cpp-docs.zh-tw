# <a name="platformdetails-namespace"></a>Platform::Details 命名空間
這個命名空間僅供內部使用，不適用於開發用途。  
  
## <a name="syntax"></a>語法  
  
```cpp  
  
namespace Platform {  
   namespace Details {  
}}  
```
