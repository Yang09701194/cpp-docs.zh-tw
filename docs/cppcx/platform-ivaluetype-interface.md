# <a name="platformivaluetype-interface"></a>Platform::IValueType 介面

`Platform::IValueType` 是實值類別和實值結構所實作的基礎結構介面。 無法在程式碼中明確使用。

## <a name="syntax"></a>語法

```cpp
interface class IValueType
```
