# Platform::NullReferenceException 類別
在嘗試解除 Null 物件的參考時擲回。  
  
## 語法  
  
```cpp  
public ref class NullReferenceException : COMException,    IException,    IPrintable,    IEquatable  
```  
  
## 備註  
 如需詳細資訊，請參閱 [COMException](../cppcx/platform-comexception-class.md) 類別。
