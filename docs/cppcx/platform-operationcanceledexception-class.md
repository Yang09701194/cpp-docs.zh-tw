# <a name="platformoperationcanceledexception-class"></a>Platform::OperationCanceledException 類別

## <a name="syntax"></a>語法  
  
```cpp  
public ref class OperationCanceledException : COMException,    IException,    IPrintable,    IEquatable  
```
