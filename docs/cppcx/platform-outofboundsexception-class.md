# Platform::OutOfBoundsException 類別
在作業嘗試存取有效範圍以外的資料時擲回。  
  
## 語法  
  
```cpp  
public ref class OutOfBoundsException : COMException,    IException,    IPrintable,    IEquatable  
```  
  
## 備註  
 如需詳細資訊，請參閱 [COMException](../cppcx/platform-comexception-class.md) 類別。
