# <a name="platformruntimeinteropservices-namespace"></a>Platform::Runtime::InteropServices 命名空間
這個命名空間僅供內部使用，不適用於開發用途。  
  
## <a name="syntax"></a>語法  
  
```cpp  
  
namespace Platform {  
   namespace InteropServices {  
}}  
```

 雖然此命名空間僅適合內部使用，但瀏覽器仍會顯示此命名空間的下列成員。
