# SizeT::SizeT 建構函式
使用指定的值來初始化 SizeT 的新執行個體。  
  
## 語法  
  
```cpp  
SizeT( uint32 value1 );   SizeT( void* value2 );  
```  
  
## 參數  
 value1  
 32 位元不帶正負號的值。  
  
 value2  
 32 位元不帶正負號之值的指標。
