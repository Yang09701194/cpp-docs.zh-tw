# StringReference::Data 方法
傳回這個 `StringReference` 的內容作為 char16 值的陣列。  
  
## 語法  
  
```cpp  
const ::default::char16 * Data() const  
```  
  
## 傳回值  
 char16 UNICODE 文字字元陣列。
