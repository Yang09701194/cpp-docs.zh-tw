# StringReference::GetString 方法
傳回 `Platform::String^` 形式的字串內容。  
  
## 語法  
  
```scr  
__declspec(no_release_return) __declspec(no_refcount)  
    ::Platform::String^ GetString() const  
```  
  
## 傳回值  
 包含字串資料的 `Platform::String^`。
