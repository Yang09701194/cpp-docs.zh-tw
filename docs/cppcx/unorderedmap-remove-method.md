# UnorderedMap::Remove 方法
從目前 UnorderedMap 物件中刪除指定的機碼值組。  
  
## 語法  
  
```cpp  
  
virtual void Remove(  
   K key);  
```  
  
#### 參數  
 `key`  
 機碼值組的機碼部分。`key` 的類型為 typename *K*。  
  
## 需求  
 **標頭：**collection.h  
  
 **命名空間：**Platform::Collections
