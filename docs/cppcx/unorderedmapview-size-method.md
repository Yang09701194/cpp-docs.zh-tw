# UnorderedMapView::Size 方法
傳回 UnorderedMapView 中的 [Windows::Foundation::Collections::IKeyValuePair\<K,V\>](http://msdn.microsoft.com/library/windows/apps/br226031.aspx) 項目數。  
  
## 語法  
  
```cpp  
  
virtual property unsigned int Size;  
```  
  
## 傳回值  
 UnorderedMapView 中的元素數目。  
  
## 需求  
 **標頭：**collection.h  
  
 **命名空間：**Platform::Collections
