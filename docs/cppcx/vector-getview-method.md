# Vector::GetView 方法
傳回 Vector 的唯讀檢視，也就是 IVectorView。  
  
## 語法  
  
```  
  
Windows::Foundation::Collections::IVectorView<T>^   
   GetView();  
```  
  
## 傳回值  
 IVectorView 物件。  
  
## 需求  
 **標頭：**collection.h  
  
 **命名空間：**Platform::Collections
