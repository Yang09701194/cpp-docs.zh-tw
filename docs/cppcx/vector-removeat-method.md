# Vector::RemoveAt 方法
從目前向量中刪除指定索引所識別的元素。  
  
## 語法  
  
```  
  
virtual void RemoveAt(  
   unsigned int index  
);  
```  
  
#### 參數  
 `index`  
 以零起始、不帶正負號的整數，在 Vector 物件中指定特別項目。  
  
## 需求  
 **標頭：**collection.h  
  
 **命名空間：**Platform::Collections
