# Vector::Size 方法
傳回目前 Vector 物件中的項目數。  
  
## 語法  
  
```  
  
virtual property unsigned int Size;  
```  
  
## 傳回值  
 目前 Vector 中的項目數。

 **標頭：**collection.h  
  
 **命名空間：**Platform::Collections
