# VectorIterator::operator* 運算子
擷取目前 VectorIterator 指定的項目位址。  
  
## 語法  
  
```  
reference operator*() const;  
```  
  
## 傳回值  
 目前 VectorIterator 指定的項目。  
  
## 需求  
 **標頭：**collection.h  
  
 **命名空間：**Platform::Collections
