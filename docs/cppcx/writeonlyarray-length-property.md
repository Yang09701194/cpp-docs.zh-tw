# WriteOnlyArray::Length 屬性
傳回呼叫端配置之陣列中元素數目。  
  
## 語法  
  
```cpp  
property unsigned int Length{  
   unsigned int get() const;  
}  
```  
  
## 傳回值  
 陣列中的元素數目。  
  
## 需求  
 **標頭：**vccorlib.h  
  
 **命名空間：**Platform
