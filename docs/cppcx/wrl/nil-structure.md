# <a name="nil-structure"></a>Nil 結構

支援 WRL 基礎結構，但不適合直接從您的程式碼使用。

## <a name="syntax"></a>語法

```cpp
struct Nil;
```

## <a name="remarks"></a>備註

用來表示未指定的選擇性範本參數。

**Nil**是空的結構。

## <a name="inheritance-hierarchy"></a>繼承階層

`Nil`
