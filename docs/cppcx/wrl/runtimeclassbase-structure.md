# <a name="runtimeclassbase-structure"></a>RuntimeClassBase 結構

支援 WRL 基礎結構，但不適合直接從您的程式碼使用。

## <a name="syntax"></a>語法

```cpp
struct RuntimeClassBase;
```

## <a name="remarks"></a>備註

用來偵測[Make](make-function.md)函數中的 `RuntimeClass`。

**RuntimeClassBase**是空的結構。
