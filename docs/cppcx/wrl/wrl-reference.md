# <a name="wrl-reference"></a>WRL 參考

本節包含 Windows 執行階段C++範本庫（WRL）的參考資訊。

> [!NOTE]
> Windows 執行階段C++範本程式庫會定義支援 Windows 執行階段C++範本庫基礎結構的功能，但不能直接從程式碼使用。 這類功能會在本檔中注明。

## <a name="in-this-section"></a>本節內容

[Microsoft::WRL 命名空間](microsoft-wrl-namespace.md)<br/>
定義組成 Windows 執行階段C++範本庫的基本類型。

[Microsoft::WRL::Wrappers 命名空間](microsoft-wrl-wrappers-namespace.md)<br/>
定義資源取得是初始化（RAII）包裝函式類型，可簡化物件、字串和控制碼的存留期管理。

[Microsoft::WRL::Wrappers::HandleTraits 命名空間](microsoft-wrl-wrappers-handletraits-namespace.md)<br/>
描述一般以控制碼為基礎之資源類型的特性。

[Windows::Foundation 命名空間](windows-foundation-namespace.md)<br/>
啟用基本 Windows 執行階段功能，例如物件和 factory 建立。

引進 Windows 執行階段C++範本庫，這是一個以 COM 為基礎的範本程式庫，提供了用來撰寫和使用 Windows 執行階段元件的低層級方式。
