# 將控制項插入 Visual C++ 應用程式
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

您可以經由工具列或使用[插入 ActiveX 控制項](../../mfc/insert-activex-control-dialog-box.md)對話方塊，將 ActiveX 控制項插入對話方塊中。  
  
### 若要從工具箱插入 ActiveX 控制項  
  
1.  以滑鼠右鍵按一下工具箱的空白區域。  
  
2.  在捷徑功能表內按一下 \[自訂工具箱\]，然後選取想要的控制項。  
  
3.  將控制項拖曳到對話方塊編輯器的對話方塊內。  
  
### 若要從對話方塊編輯器內插入 ActiveX 控制項  
  
1.  以滑鼠右鍵按一下對話方塊。  
  
2.  在捷徑功能表上，按一下[插入 ActiveX 控制項](../../mfc/insert-activex-control-dialog-box.md)。  
  
    > [!NOTE]
    >  當您從 \[**插入 ActiveX 控制項**\] 中將 ActiveX 控制項插入至專案時，並不會將[包裝函式類別](../../data/ado-rdo/wrapper-classes.md)包含在專案內。  您必須負責建立包裝函式類別來自訂控制項的功能。
