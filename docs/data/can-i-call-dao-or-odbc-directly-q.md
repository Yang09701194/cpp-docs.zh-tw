# 我可以直接呼叫 DAO 或 ODBC 嗎？
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

照例在 MFC 中，如果您需要微調控制項，除了透過類別來存取它們外，還可直接呼叫 DAO 或 ODBC。  MFC 嘗試簡化 Windows 的程式設計，不過如果您需要存取以下的 API，也會比較困難。
