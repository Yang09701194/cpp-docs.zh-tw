# 安裝 ATL 資料庫支援
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

安裝程式安裝了所有支援 OLE DB 程式設計的必要元件做為基礎安裝的一部分。  OLE DB 介面是 MDAC SDK 安裝的一部分。  OLE DB 樣板是 ATL 程式庫安裝的一部分，此樣板支援在 Visual C\+\+ .NET 中進行 OLE DB 程式設計。  Visual C\+\+ .NET 安裝程式會同時安裝 MDAC SDK \(2.8 版\) 和 ATL 程式庫。
