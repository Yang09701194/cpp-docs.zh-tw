# ODBC：設定 ODBC 資料來源
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

若要在已完成開發的應用程式上使用[資料來源](../../data/odbc/data-source-odbc.md)，就必須使用 ODBC 管理員進行設定。  ODBC 管理員會在 Windows 登錄中儲存可用的資料來源和其連接資訊。  使用 ODBC 管理員來加入、修改和刪除在 \[資料來源\] 對話方塊中的資料來源，並加入和刪除 ODBC 驅動程式。  
  
> [!NOTE]
>  此資訊適用於為 ODBC 存取使用 MFC 資料存取物件 \(DAO\) 類別以及使用 MFC ODBC 類別的時候。  
  
 ODBC 管理員會隨 Microsoft Foundation Class \(MFC\) 程式庫資料庫支援自動完成安裝。  如需 ODBC 管理員程式的詳細資訊，請參閱 [ODBC 管理員](../../data/odbc/odbc-administrator.md)和 ODBC API 的線上參考說明系統。  
  
 如需撰寫 MFC 資料庫應用程式的 ODBC 安裝和管理員程式的詳細資訊，請參閱[技術提示 48](../../mfc/tn048-writing-odbc-setup-and-administration-programs.md)。
