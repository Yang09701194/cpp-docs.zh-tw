# <a name="odbc-driver-list"></a>ODBC 驅動程式清單

Visual c + + 會提供下列資料庫中的 ODBC 驅動程式：

- SQL Server

- Microsoft Access

- Microsoft Excel

- dBASE

- Paradox

- Microsoft Oracle ODBC

- 文字檔

如需有關 ODBC 驅動程式可從 Microsoft 和其他公司，包括 ODBC 驅動程式套件資訊，請連絡 Microsoft 產品支援服務。
