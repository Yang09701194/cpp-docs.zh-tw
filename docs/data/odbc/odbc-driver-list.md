# <a name="odbc-driver-list"></a>ODBC 驅動程式清單

Visual C++提供下列資料庫的 ODBC 驅動程式：

- SQL Server

- Microsoft Access

- Microsoft Excel

- dBASE

- Paradox

- Microsoft Oracle ODBC

- 文字檔

如需 Microsoft 和其他公司提供之 ODBC 驅動程式的相關資訊，包括 ODBC 驅動程式套件，請聯絡 Microsoft 產品支援服務。
