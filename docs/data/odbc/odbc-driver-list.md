[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Visual C\+\+ 提供下列資料庫的 ODBC 驅動程式：  
  
-   SQL Server  
  
-   Microsoft Access  
  
-   Microsoft Excel  
  
-   dBASE  
  
-   Paradox  
  
-   Microsoft Oracle ODBC  
  
-   文字檔  
  
 如需 Microsoft 和其他公司的 ODBC 驅動程式詳細資訊 \(包括 ODBC Driver Pack\)，請聯絡 Microsoft 產品支援服務。
