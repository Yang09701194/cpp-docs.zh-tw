# <a name="recordset-declaring-a-class-for-a-table-odbc"></a>資料錄集：宣告資料表的類別 (ODBC)
本主題適用於 MFC ODBC 類別。  
  
 最常見的資料錄集類別會開啟單一資料表。 若要將單一資料表的資料錄集類別的宣告，使用[MFC ODBC 消費者精靈](../../mfc/reference/adding-an-mfc-odbc-consumer.md)從**加入類別**，然後選擇您想要命名對應的資料錄集欄位資料成員的每個資料行。  
  
 資料錄集的其他用途包括：  
  
-   聯結兩個或多個資料表。  
  
-   包含預先定義查詢的結果。
