# 資料錄集：宣告資料表的類別 (ODBC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

本文件適用於 MFC ODBC 類別。  
  
 最常見的資料錄集類別會開啟一個單一資料表。  若要宣告單一資料表的資料錄集類別，請使用 \[加入類別\] 中的 [MFC ODBC 消費者精靈](../../mfc/reference/adding-an-mfc-odbc-consumer.md)，並藉由命名對應的資料錄集欄位資料成員來選擇想要的每個資料行。  
  
 資料錄集的其他用途包括：  
  
-   聯結兩個或更多個資料表  
  
-   包含預先定義查詢的結果
