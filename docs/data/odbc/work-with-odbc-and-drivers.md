# <a name="work-with-odbc-and-drivers"></a>使用 ODBC 和驅動程式

ODBC 可讓您撰寫不只相依于一個資料庫管理系統（DBMS）的應用程式。 您可以撰寫一個可在多個 Dbms 上操作的應用程式，讓您的使用者擁有正確的 DBMS 特定 ODBC 驅動程式。
