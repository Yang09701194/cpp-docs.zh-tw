# <a name="working-with-documents-and-views"></a>使用文件和檢視

Microsoft Foundation class （MFC）程式庫依賴檔/視圖架構來進行許多功能。 一般來說，檔會儲存您的資料，而 view 會在框架視窗的工作區中顯示它，並管理使用者與資料的互動。 此視圖會與檔通訊，以取得及更新資料。 您可以使用資料庫類別搭配架構或不搭配。

如需在架構中使用資料庫類別的詳細資訊，請參閱[MFC：使用具有檔和視圖的資料庫類別](../../data/mfc-using-database-classes-with-documents-and-views.md)。

根據預設，MFC 應用程式精靈會建立無資料庫支援的基本架構應用程式。 不過，您可以選取選項來包含最小的資料庫支援或更完整的表單型支援。 如需有關 [應用程式精靈] 選項的詳細資訊，請參閱[MFC 應用程式精靈的資料庫支援](../../mfc/reference/database-support-mfc-application-wizard.md)。

您也可以使用資料庫類別，而不需要使用完整的檔/視圖架構。 如需詳細資訊，請參閱[MFC：使用沒有檔和視圖的資料庫類別](../../data/mfc-using-database-classes-without-documents-and-views.md)。
