# BEGIN_PARAM_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

標記參數對應項目的開頭。  
  
## 語法  
  
```  
  
BEGIN_PARAM_MAP(  
x  
 )  
  
```  
  
#### 參數  
 *x*  
 \[in\] 使用者資料記錄類別的名稱。  
  
## 備註  
 [命令](https://msdn.microsoft.com/en-us/library/ms724608.aspx) 會使用的參數。  
  
## 範例  
 請參閱 [BEGIN\_COLUMN\_MAP](../../data/oledb/begin-column-map.md) 巨集的範例。
