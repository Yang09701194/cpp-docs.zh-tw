# <a name="beginparammap"></a>BEGIN_PARAM_MAP
標記參數對應項目的開頭。  
  
## <a name="syntax"></a>語法  
  
```cpp
BEGIN_PARAM_MAP(x)  
  
```  
  
#### <a name="parameters"></a>參數  
 *x*  
 [in] 使用者資料錄類別的名稱。
