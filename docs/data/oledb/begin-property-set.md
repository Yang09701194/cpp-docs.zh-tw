# <a name="beginpropertyset"></a>BEGIN_PROPERTY_SET
在屬性中設定屬性的開頭的標記集合對應。  
  
## <a name="syntax"></a>語法  
  
```cpp
BEGIN_PROPERTY_SET(guid)  
  
```  
  
#### <a name="parameters"></a>參數  
 `guid`  
 [in]GUID 屬性。  
  
## <a name="example"></a>範例  
 請參閱 [BEGIN_PROPSET_MAP](../../data/oledb/begin-propset-map.md)。

 **Header:** atldb.h
