# BEGIN_PROPERTY_SET
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

標記在屬性集對應設定之屬性的開頭。  
  
## 語法  
  
```  
  
BEGIN_PROPERTY_SET(  
guid   
)  
  
```  
  
#### 參數  
 `guid`  
 \[in\] 屬性 GUID。  
  
## 範例  
 請參閱 [BEGIN\_PROPSET\_MAP](../../data/oledb/begin-propset-map.md)。  
  
## 需求  
 **標頭：** atldb.h
