# <a name="caccessorbasegethaccessor"></a>CAccessorBase::GetHAccessor
擷取指定之存取子的存取子控制代碼。  
  
## <a name="syntax"></a>語法  
  
```cpp
      HACCESSOR GetHAccessor(ULONG nAccessor) const;  
```  
  
#### <a name="parameters"></a>參數  
 `nAccessor`  
 [in] 存取子的零位移數字。  
  
## <a name="return-value"></a>傳回值  
 存取子控制代碼。  
  
## <a name="requirements"></a>需求  
 **標題:** atldbcli.h
