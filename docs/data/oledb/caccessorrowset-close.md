# <a name="caccessorrowsetclose"></a>CAccessorRowset::Close

## <a name="syntax"></a>語法  
  
```  
  
void Close( );  
  
```  
  
## <a name="remarks"></a>備註  
 釋放任何相關聯的記憶體。
