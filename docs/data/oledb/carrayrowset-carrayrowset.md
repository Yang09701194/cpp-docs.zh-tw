# <a name="carrayrowsetcarrayrowset"></a>CArrayRowset::CArrayRowset
建立新的 `CArrayRowset` 物件。  
  
## <a name="syntax"></a>語法  
  
```cpp
      CArrayRowset(int nMax = 100000);  
```  
  
#### <a name="parameters"></a>參數  
 `nMax`  
 [in] 資料列集的最大資料列數。

 **標題:** atldbcli.h
