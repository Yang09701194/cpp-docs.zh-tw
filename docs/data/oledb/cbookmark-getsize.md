# CBookmark::GetSize
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

擷取書籤緩衝區的大小。  
  
## 語法  
  
```  
  
virtual DBLENGTH GetSize( ) const throw( );  
  
```  
  
## 傳回值  
 緩衝區的大小 \(以位元組為單位\)。  
  
## 需求  
 **標題:** atldbcli.h
