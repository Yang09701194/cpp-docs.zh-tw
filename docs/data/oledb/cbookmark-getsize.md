# <a name="cbookmarkgetsize"></a>CBookmark::GetSize
擷取書籤緩衝區的大小。  
  
## <a name="syntax"></a>語法  
  
```  
  
virtual DBLENGTH GetSize( ) const throw( );  
  
```  
  
## <a name="return-value"></a>傳回值  
 以位元組為單位的緩衝區大小。  
  
## <a name="requirements"></a>需求  
 **標題:** atldbcli.h
