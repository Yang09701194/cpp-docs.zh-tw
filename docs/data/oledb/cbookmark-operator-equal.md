# <a name="cbookmarkoperator-"></a>CBookmark::operator =
指派 `CBookmark` 物件給另一個。  
  
## <a name="syntax"></a>語法  
  
```cpp
      CBookmark& operator =(const CBookmark& bookmark) throw();  
```  
  
## <a name="remarks"></a>備註  
 這個運算子只有在需要**CBookmark\<0 >**。

 **標題:** atldbcli.h
