# <a name="cbookmarksetbookmark"></a>CBookmark::SetBookmark
複製所參考的書籤值`pBuffer`至`CBookmark`緩衝和緩衝區大小設定為`nSize`。  
  
## <a name="syntax"></a>語法  
  
```  
  
      HRESULT SetBookmark(  
   DBLENGTH nSize,  
   BYTE* pBuffer   
) throw( );  
```  
  
#### <a name="parameters"></a>參數  
 *nSize*  
 [in]書籤緩衝區的大小。  
  
 `pBuffer`  
 [in]包含書籤值的位元組陣列指標。  
  
## <a name="return-value"></a>傳回值  
 標準 `HRESULT`。  
  
## <a name="remarks"></a>備註  
 此函式僅供以**CBookmark\<0 >**。  
  
## <a name="requirements"></a>需求  
 **標題:** atldbcli.h
