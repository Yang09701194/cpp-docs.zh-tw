# CBulkRowset::AddRefRows
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

呼叫[IRowset::AddRefRows](https://msdn.microsoft.com/en-us/library/ms719619.aspx) 會增加目前擷取自大量資料列集的所有資料行的參考計數。  
  
## 語法  
  
```  
  
HRESULT AddRefRows( ) throw( );  
  
```  
  
## 傳回值  
 標準版 `HRESULT`  
  
## 需求  
 **標題:** atldbcli.h
