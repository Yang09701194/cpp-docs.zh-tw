# <a name="cdataconnectioncopy"></a>CDataConnection::Copy
建立現有的資料連接的複本。  
  
## <a name="syntax"></a>語法  
  
```  
  
      CDataConnection& Copy(   
   const CDataConnection & ds    
) throw( );  
```  
  
#### <a name="parameters"></a>參數  
 `ds`  
 [in]若要複製現有的資料連接的參考。  
  
## <a name="requirements"></a>需求  
 **標題:** atldbcli.h
