# <a name="cdataconnectionoperator-bool"></a>CDataConnection::operator BOOL
判斷目前的工作階段是否為開啟。  
  
## <a name="syntax"></a>語法  
  
```  
  
operator BOOL( ) throw( );  
  
```  
  
## <a name="remarks"></a>備註  
 傳回**BOOL** (MFC typedef) 值。 **TRUE**表示目前的工作階段已開啟。**FALSE**表示目前的工作階段已關閉。  
  
## <a name="requirements"></a>需求  
 **標題:** atldbcli.h
