# <a name="cdbpropidsetaddpropertyid"></a>CDBPropIDSet::AddPropertyID
將屬性 ID 加入至屬性 ID 集。  
  
## <a name="syntax"></a>語法  
  
```cpp
      bool AddPropertyID(DBPROPID propid) throw();  
```  
  
#### <a name="parameters"></a>參數  
 `propid`  
 [in] 要加入至屬性 ID 集的屬性 ID。  
  
## <a name="requirements"></a>需求  
 **標題:** atldbcli.h
