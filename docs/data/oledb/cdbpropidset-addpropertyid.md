# CDBPropIDSet::AddPropertyID
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

將屬性 ID 加入至屬性 ID 集。  
  
## 語法  
  
```  
  
      bool AddPropertyID(   
   DBPROPID propid    
) throw( );  
```  
  
#### 參數  
 `propid`  
 \[in\] 要加入的屬性 ID 集的屬性 ID。  
  
## 需求  
 **標題:** atldbcli.h
