# <a name="cdbpropidsetoperator-"></a>CDBPropIDSet::operator =
將一個屬性 ID 集的內容指派至另一個 ID 屬性集。  
  
## <a name="syntax"></a>語法  
  
```cpp
      CDBPropIDSet& operator =(CDBPropIDSet& propset) throw();  
```
