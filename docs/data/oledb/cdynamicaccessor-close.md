# <a name="cdynamicaccessorclose"></a>CDynamicAccessor::Close
所有資料行解除繫結，釋放配置的記憶體，並釋放[IAccessor](https://msdn.microsoft.com/en-us/library/ms719672.aspx)類別中的介面指標。  
  
## <a name="syntax"></a>語法  
  
```cpp
void Close() throw();  
  
```

 **標題:** atldbcli.h
