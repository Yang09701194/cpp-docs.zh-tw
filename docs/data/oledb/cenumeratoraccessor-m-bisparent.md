# CEnumeratorAccessor::m_bIsParent
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

表示列舉值是否是父列舉程式的變數，如果資料行為列舉值。  
  
## 語法  
  
```  
  
VARIANT_BOOL m_bIsParent;  
  
```
