# CMultipleResults 類別
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

如果您想要命令處理多個結果集，請使用 `CMultipleResults` 為 `CCommand` 樣板引數 *TMultiple* 。  
  
## 語法  
  
```  
class CMultipleResults  
```  
  
## 備註  
 若要處理多個結果集，必須從 [CCommand](../../data/oledb/ccommand-class.md) 這個類別繼承。

 **Header:** atldbcli
