# <a name="creating-a-project-for-the-provider"></a>為提供者建立專案

## <a name="to-create-a-project-in-which-the-ole-db-provider-will-reside"></a>建立 OLE DB 提供者將位於其中的專案

1. 從 [檔案] 功能表中，按一下 [新增]，再按 [專案]。

   [新增專案] 對話方塊隨即出現。

1. 在 [**專案類型**] 窗格中，按一下 [**已安裝**的 > **Visual C++**  > **MFC/ATL**資料夾]。 在 [**範本**] 窗格中，按一下 [ **ATL 專案**]。

    > [!NOTE]
    > 在舊版的 Visual Studio 中，尋找 [**已安裝**的 > **範本**] 底下的專案類型 > [ **Visual C++**  > **ATL**]。

1. 在 [**名稱**] 方塊中，輸入專案的名稱，然後按一下 **[確定]** 。

   [ **ATL 專案嚮導]** 隨即出現。

1. 在**ATL 專案嚮導**中，針對 [**應用程式類型**] 選擇 [**動態連結程式庫（DLL）** ]。

1. 按一下 [完成]。
