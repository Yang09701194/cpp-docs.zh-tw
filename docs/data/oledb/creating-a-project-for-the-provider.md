# 為提供者建立專案
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

#### 若要建立位於 OLE DB 提供者的專案  
  
1.  在 \[檔案\] 功能表上按一下 `New`，再按一下 \[專案\]。  
  
     \[**新增專案**\] 對話方塊隨即出現。  
  
2.  在 \[專案類型\] 窗格上，按一下 \[Visual C\+\+ 專案\] 資料夾。  在 \[樣板\] 窗格中，按一下 \[**ATL 專案**\]。  
  
3.  在 \[名稱\] 對話方塊中輸入專案的名稱，然後按一下 \[**確定**\]。  
  
     \[ATL 專案精靈\] 便會出現。  
  
4.  在 \[ATL 專案精靈\] 中選擇 \[動態連結程式庫 \(DLL\)\] 做為 \[伺服器類型\]。  
  
5.  按一下 \[**完成**\]。
