# <a name="csessiongettransactioninfo"></a>CSession::GetTransactionInfo
傳回有關交易的資訊。  
  
## <a name="syntax"></a>語法  
  
```cpp
HRESULT GetTransactionInfo(XACTTRANSINFO* pInfo) const throw();  
```  
  
#### <a name="parameters"></a>參數  
 請參閱[ITransaction::GetTransactionInfo](https://msdn.microsoft.com/en-us/library/ms714975.aspx)中*OLE DB 程式設計人員參考*。  
  
## <a name="return-value"></a>傳回值  
 標準 `HRESULT`。  
  
## <a name="remarks"></a>備註  
 如需詳細資訊，請參閱[ITransaction::GetTransactionInfo](https://msdn.microsoft.com/en-us/library/ms714975.aspx)中*OLE DB 程式設計人員參考*。
