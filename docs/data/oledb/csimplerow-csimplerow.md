# <a name="csimplerowcsimplerow"></a>CSimpleRow::CSimpleRow
建構函式。  
  
## <a name="syntax"></a>語法  
  
```  
  
      CSimpleRow(  
   DBCOUNTITEM iRowsetCur   
);  
```  
  
#### <a name="parameters"></a>參數  
 `iRowsetCur`  
 [in]目前的資料列集的索引。  
  
## <a name="remarks"></a>備註  
 設定[m_iRowset](../../data/oledb/csimplerow-m-irowset.md)至`iRowsetCur`。
