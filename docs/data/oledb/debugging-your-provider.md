# <a name="debugging-your-provider"></a>偵錯提供者

有兩種方式可以偵錯您的提供者：

- 因為提供者會建立程序中，您可以建立一些通常使用的 OLE DB 消費者範本和逐步執行提供者的取用者程式碼。

- 您可以使用隨附於 Visual c + + 的各種公用程式。

## <a name="to-use-debugging"></a>若要使用偵錯

1. 開啟提供者的專案。

1. 在 **專案**功能表上，按一下**屬性**。

1. 在 **屬性頁** 對話方塊中，按一下**偵錯** 索引標籤。

1. 選取的選項為必要項，按一下**確定**。

1. 設定中斷點，，然後如常進行偵錯。
