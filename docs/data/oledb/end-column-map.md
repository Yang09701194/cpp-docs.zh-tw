# END_COLUMN_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

資料行對應項目的結尾。  
  
## 語法  
  
```  
  
END_COLUMN_MAP( )  
  
```  
  
## 備註  
 它使用與資料列集的單一存取子。  `BEGIN_COLUMN_MAP` 巨集完成與 `END_COLUMN_MAP` 巨集。
