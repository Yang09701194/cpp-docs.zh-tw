# <a name="endcolumnmap"></a>END_COLUMN_MAP
資料行對應項目結束標記。  
  
## <a name="syntax"></a>語法  
  
```cpp
END_COLUMN_MAP()  
  
```  
  
## <a name="remarks"></a>備註  
 它搭配單一存取子資料列集。 `BEGIN_COLUMN_MAP` 巨集會以 `END_COLUMN_MAP` 巨集完成。

 請參閱[BEGIN_COLUMN_MAP](../../data/oledb/begin-column-map.md)。
