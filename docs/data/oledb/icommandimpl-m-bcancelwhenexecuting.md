# <a name="icommandimplmbcancelwhenexecuting"></a>ICommandImpl::m_bCancelWhenExecuting
指出執行時，是否可以取消命令。  
  
## <a name="syntax"></a>語法  
  
```  
  
unsigned m_bCancelWhenExecuting:1;  
  
```  
  
## <a name="remarks"></a>備註  
 預設為**true** （您可以取消）。  
  
## <a name="requirements"></a>需求  
 **Header:** atldb.h
