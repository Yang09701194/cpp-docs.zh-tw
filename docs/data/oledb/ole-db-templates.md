# OLE DB 樣板
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

OLE DB 樣板使 OLE DB 資料存取更容易。  Visual C\+\+ 提供了實作許多常用 OLE DB 介面的樣板類別。  您可以使用 Visual C\+\+ 消費者樣板撰寫 OLE DB 用戶端 \(消費者\) 應用程式和提供者樣板撰寫伺服器 \(提供者\) 應用程式。
