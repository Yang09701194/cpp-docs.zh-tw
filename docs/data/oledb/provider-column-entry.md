# <a name="providercolumnentry"></a>PROVIDER_COLUMN_ENTRY
表示提供者支援的特定資料行。  
  
## <a name="syntax"></a>語法  
  
```cpp
PROVIDER_COLUMN_ENTRY (name  
, ordinal, member )  
```  
  
#### <a name="parameters"></a>參數  
 *name*  
 [in]資料行名稱。  
  
 `ordinal`  
 [in] 資料行編號。 除非資料行是書籤資料行，資料行數目必須為 0。  
  
 `member`  
 [in]中的成員變數`dataClass`對應此資料行。  
  
## <a name="requirements"></a>需求  
 **Header:** atldb.h
