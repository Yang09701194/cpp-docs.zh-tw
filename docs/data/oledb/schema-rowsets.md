# 結構描述資料列集
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

結構描述 \(或中繼資料\) 是任何資料庫結構或組織的相關資訊，例如提供者、資料列集、資料表、資料行的相關資訊，或是其他實質資料庫內容的周邊資訊。  OLE DB 讓您使用結構描述資料列集 \(不需要開啟資料列集\)，即可取得這些資訊。  
  
 如需如何使用結構描述資料列集的詳細資訊，請參閱[使用結構描述資料列集取得中繼資料](../../data/oledb/obtaining-metadata-with-schema-rowsets.md)。
