# <a name="record-mfc-data-access"></a>記錄 (MFC 資料存取)

記錄是單一實體的資料收集，例如一個帳戶或客戶儲存在資料表中 (資料表的資料列)。 記錄由包含各類資料的連續資料行 (有時候稱為欄位) 群組所組成。 一組從資料來源選取的記錄 (在資料庫詞彙中通常稱為結果集)，在 MFC 中稱為記錄集。 如需詳細資訊，請參閱[記錄集（ODBC）](../data/odbc/recordset-odbc.md)。
