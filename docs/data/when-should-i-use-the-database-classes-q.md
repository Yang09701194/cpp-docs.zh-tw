# 我何時應該使用資料庫類別？
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

許多應用程式都需要在資料庫內有一個資料存放區，而許多其他的應用程式則可透過使用資料庫而受惠。  許多狀況下，資料庫可由多重使用者和多重應用程式所存取，並提供您彈性資料儲存機制。  資料庫可儲存大量資料，並針對查詢與更新提供快速的資料存取。
