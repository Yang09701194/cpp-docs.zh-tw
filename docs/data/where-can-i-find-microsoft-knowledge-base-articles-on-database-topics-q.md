# 我可在何處尋找有關資料庫主題的 Microsoft 知識庫文件？
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

若要尋找 MFC ODBC 主題的知識庫文件，請在知識庫中使用關鍵字 MFCDATABASE 搜尋。  
  
 若要尋找 MFC DAO 主題的知識庫文件，請在知識庫中使用關鍵字 MFCDAO 搜尋。
