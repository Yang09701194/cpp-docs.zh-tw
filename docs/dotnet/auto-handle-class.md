# <a name="autohandle-class"></a>auto_handle 類別
自動資源管理可將 managed 類型中嵌入虛擬控制代碼。  
  
## <a name="syntax"></a>語法  
  
```  
template<typename _element_type>  
ref class auto_handle;  
```  
  
#### <a name="parameters"></a>參數  
*_element_type*<br/>
要內嵌的 managed 的類型。  
  
## <a name="requirements"></a>需求  
 **標頭檔** \<msclr\auto_handle.h >  
  
 **命名空間**msclr
