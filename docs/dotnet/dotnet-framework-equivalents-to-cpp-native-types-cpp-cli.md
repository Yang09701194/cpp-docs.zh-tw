# C++ 原生類型的 .NET Framework 對應項 (C++/CLI)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下表顯示內建 Visual C\+\+ 型別的關鍵字，這些是 **System** 命名空間裡預先定義型別的別名。  
  
|Visual C\+\+ 型別|.NET Framework 型別|  
|---------------------|-----------------------|  
|**bool**|**System.Boolean**|  
|**signed char** \(如需詳細資訊，請參閱 [\/J](../build/reference/j-default-char-type-is-unsigned.md)\)|**System.SByte**|  
|**unsigned char**|**System.Byte**|  
|**wchar\_t**|**System.Char**|  
|**double** 和 **long double**|**System.Double**|  
|**float**|**System.Single**|  
|**int**、**signed int**、**long** 和 **signed long**|**System.Int32**|  
|**unsigned int** 和 **unsigned long**|**System.UInt32**|  
|**\_\_int64** 和 **signed \_\_int64**|**System.Int64**|  
|**unsigned \_\_int64**|**System.UInt64**|  
|**short** 和 **signed short**|**System.Int16**|  
|**unsigned short**|**System.UInt16**|  
|**void**|**System.Void**|
