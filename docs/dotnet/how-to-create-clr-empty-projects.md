# 如何：建立 CLR 空專案
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

若要建立 CLR 空專案，請使用 \[**CLR 空專案**\] 範本，請從 \[**新的專案**\] 對話方塊中取得。  
  
> [!NOTE]
>  IDE 中的功能外觀會依您所使用的設定或版本而定，而且可能與 \[說明\] 中所描述的情形不同。  若要變更設定，請從 \[**工具**\] 功能表中選擇 \[**匯入和匯出設定**\]。  如需詳細資訊，請參閱 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/zh-tw/22c4debb-4e31-47a8-8f19-16f328d7dcd3)。  
  
### 若要建立 CLR 空專案  
  
1.  在 \[**檔案**\] 功能表上，按一下 \[**新增**\]，然後按一下 \[**專案**\]。  
  
     \[**新增專案**\] 對話方塊隨即出現。  
  
2.  在 \[**已安裝的範本**\] 下，按一下 \[**Visual C\+\+**\] 節點，然後按一下 \[**CLR**\] 節點。  選擇 **CLR 空專案**圖示。  
  
3.  請在 \[**名稱**\] 方塊中，輸入應用程式的唯一名稱。  
  
    > [!NOTE]
    >  您也可以從 \[**新增專案**\] 對話方塊中，指定其他專案和方案的設定，但是這些設定並非必要項。  
  
4.  按一下 \[**確定**\]。
