# <a name="native-and-net-interoperability"></a>原生和 .NET 互通性

Visual c + + 支援互通性功能，可讓 managed 和 unmanaged 結構，以同時存在，而且交互操作內相同的組件，並甚至在相同的檔案。 其他.NET 語言，也支援一小部分的這項功能，例如 P/Invoke，但大部分的 Visual c + + 所提供的互通性支援不適用於其他語言。
