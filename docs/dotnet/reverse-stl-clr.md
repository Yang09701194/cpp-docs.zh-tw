# <a name="reverse-stlclr"></a>reverse (STL/CLR)
反轉範圍內項目的順序。  
  
## <a name="syntax"></a>語法  
  
```  
template<class _BidIt> inline  
    void reverse(_BidIt _First, _BidIt _Last);  
```  
  
## <a name="remarks"></a>備註  
 此函式的行為與 c + + 標準程式庫函式相同`reverse`。 如需詳細資訊，請參閱[反向](../standard-library/algorithm-functions.md#reverse)。  
  
## <a name="requirements"></a>需求  
 **標頭：** \<演算法 cliext/>  
  
 **命名空間：** cliext
