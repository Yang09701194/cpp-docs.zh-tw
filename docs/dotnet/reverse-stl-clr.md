# reverse (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

反轉項目的順序在範圍內。  
  
## 語法  
  
```  
template<class _BidIt> inline  
    void reverse(_BidIt _First, _BidIt _Last);  
```  
  
## 備註  
 這個函式產生相同的行為做為 STL 函式的 `reverse`。  如需詳細資訊，請參閱[reverse](../Topic/reverse.md)。

 **命名空間:** cliext
