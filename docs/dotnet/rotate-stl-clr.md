# rotate (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

交換兩個相鄰範圍的項目。  
  
## 語法  
  
```  
template<class _FwdIt> inline  
    void rotate(_FwdIt _First, _FwdIt _Mid, _FwdIt _Last);  
```  
  
## 備註  
 這個函式的行為與 STL 函式 `rotate` 相同。  如需詳細資訊，請參閱[rotate](../Topic/rotate.md)。  
  
## 需求  
 **標頭：** \<cliext\/algorithm\>  
  
 **命名空間：** cliext
