# swap (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

切換項目的值在物件之間有兩種類型的，將第一個物件加入至物件內容和內容對話方塊的第一個。  
  
## 語法  
  
```  
<class _Ty> inline  
    void swap(_Ty% _Left, _Ty% _Right);  
```  
  
## 備註  
 這個函式產生相同的行為做為 STL 函式的 `swap`。  如需詳細資訊，請參閱[swap](../Topic/swap.md)。

 **命名空間:** cliext
