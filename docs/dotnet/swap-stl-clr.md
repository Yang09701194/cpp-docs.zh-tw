# <a name="swap-stlclr"></a>swap (STL/CLR)
在兩種類型的物件之間交換項目的值，將第一個物件的內容指派給第二個物件，並將第一個物件的內容指派給第一個物件。  
  
## <a name="syntax"></a>語法  
  
```  
<class _Ty> inline  
    void swap(_Ty% _Left, _Ty% _Right);  
```  
  
## <a name="remarks"></a>備註  
 此函式的行為與 c + + 標準程式庫函式相同`swap`。 如需詳細資訊，請參閱[交換](http://msdn.microsoft.com/Library/b471a2de-035e-4aff-b1c7-345d85d93972)。  
  
## <a name="requirements"></a>需求  
 **標頭：** \<演算法 cliext/>  
  
 **命名空間：** cliext
