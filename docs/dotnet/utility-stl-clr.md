# <a name="utility-stlclr"></a>utility (STL/CLR)
包括 STL/CLR 標頭`<cliext/utility>`來定義範本類別`pair`和數個支援的樣板函式。  
  
## <a name="syntax"></a>語法  
  
```  
#include <utility>  
```
