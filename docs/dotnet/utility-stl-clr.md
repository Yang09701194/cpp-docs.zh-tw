# utility (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

包含 STL\/CLR 標頭 `<cliext/utility>` 定義樣板類別 `pair` 和數個支援的樣板函式。  
  
## 語法  
  
```  
#include <utility>  
```  
  
## 宣告  
  
|類別|說明|  
|--------|--------|  
|[pair](../dotnet/pair-stl-clr.md)|封裝一組項目。|  
  
|運算子|說明|  
|---------|--------|  
|[operator\=\= \(pair\)](../dotnet/operator-equality-pair-stl-clr.md)|對相等比較。|  
|[operator\!\= \(pair\)](../dotnet/operator-inequality-pair-stl-clr.md)|配對不相等比較。|  
|[operator\< \(pair\)](../dotnet/operator-less-than-pair-stl-clr.md)|為小於比較。|  
|[operator\<\= \(pair\)](../dotnet/operator-less-or-equal-pair-stl-clr.md)|為小於或等於比較。|  
|[operator\> \(pair\)](../dotnet/operator-greater-than-pair-stl-clr.md)|對大於比較。|  
|[operator\>\= \(pair\)](../dotnet/operator-greater-or-equal-pair-stl-clr.md)|將設為大於或等於比較。|  
  
|功能|說明|  
|--------|--------|  
|[make\_pair](../dotnet/make-pair-stl-clr.md)|由一個對值進行配對。|
