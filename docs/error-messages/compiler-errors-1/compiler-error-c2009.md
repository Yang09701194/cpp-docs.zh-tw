# <a name="compiler-error-c2009"></a>編譯器錯誤 C2009

巨集型式 'identifier' 的重複使用

巨集定義的正式參數清單多次使用識別碼。 巨集引數清單中的識別碼必須是唯一的。

## <a name="example"></a>範例

下列範例會產生 C2009：

```cpp
// C2009.cpp
#include <stdio.h>

#define macro1(a,a) (a*a)   // C2009

int main()
{
    printf_s("%d\n", macro1(2));
}
```

## <a name="example"></a>範例

可能的解決方案：

```cpp
// C2009b.cpp
#include <stdio.h>

#define macro2(a)   (a*a)
#define macro3(a,b) (a*b)

int main()
{
    printf_s("%d\n", macro2(2));
    printf_s("%d\n", macro3(2,4));
}
```
