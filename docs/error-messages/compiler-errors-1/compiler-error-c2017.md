# <a name="compiler-error-c2017"></a>編譯器錯誤 C2017
不合法的逸出序列  
  
 逸出序列，例如 \t，出現在外的字元或字串常數。  
  
 下列範例會產生 C2017:  
  
```  
// C2017.cpp  
int main() {  
   char test1='a'\n;   // C2017  
   char test2='a\n';   // ok  
}  
```  
  
 字串化運算子搭配包含逸出序列的字串時，可能會發生 C2017。  
  
 下列範例會產生 C2017:  
  
```  
// C2017b.cpp  
#define TestDfn(x) AfxMessageBox(#x)  
TestDfn(CString("\\") + CString(".h\"\n\n"));   // C2017  
```
