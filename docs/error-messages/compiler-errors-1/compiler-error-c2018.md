# 編譯器錯誤 C2018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未知的字元 'hexnumber'  
  
 原始程式檔 \(Source File\) 包含了未預期的 ASCII 字元，該字元是由其十六進位數字來辨認。  若要解決這種錯誤，請移除該字元。
