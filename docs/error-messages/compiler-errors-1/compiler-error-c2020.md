# <a name="compiler-error-c2020"></a>編譯器錯誤 C2020
'member': 'class' 成員重複定義  
  
 已重新定義繼承自基底類別或結構的成員。 無法重新定義繼承的成員，除非宣告為`virtual`基底類別中。
