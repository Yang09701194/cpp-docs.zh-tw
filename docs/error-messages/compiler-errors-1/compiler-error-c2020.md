# <a name="compiler-error-c2020"></a>編譯器錯誤 C2020

' member '： ' 類別 ' 成員重新定義

繼承自基類或結構的成員會重新定義。 除非在基類中宣告為 `virtual`，否則無法重新定義繼承的成員。
