# 編譯器錯誤 C2020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'member' : 'class' 成員重複定義  
  
 自基底類別 \(Base Class\) 或結構繼承的成員被重複定義。  繼承的成員不可重複定義，除非它在基底類別內宣告為 `virtual`。
