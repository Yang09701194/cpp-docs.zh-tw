# <a name="compiler-error-c2048"></a>編譯器錯誤 C2048
有一個以上的 default  
  
 `switch` 陳述式包含多個 `default` 標籤。 刪除其中一個 `default` 標籤，來解決這個錯誤。  
  
 下列範例會產生 C2048：  
  
```  
// C2048.cpp  
int main() {  
   int a = 1;  
   switch (a) {  
      case 1:  
         a = 0;  
      default:  
         a = 2;  
      default:   // C2048  
         a = 3;  
   }  
}  
```  
  
 可能的解決方式：  
  
```  
// C2048b.cpp  
int main() {  
   int a = 1;  
   switch (a) {  
      case 1:  
         a = 0;  
      default:  
         a = 2;  
   }  
}  
```
