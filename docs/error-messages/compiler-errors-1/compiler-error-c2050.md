# 編譯器錯誤 C2050
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

switch 運算式不是整數  
  
 `switch` 運算式評估出非整數值。  在 switch 陳述式 內只使用整數值，便可解決這種錯誤。  
  
 下列範例會產生 C2050：  
  
```  
// C2050.cpp  
int main() {  
   int a = 1;  
   switch ("a") {   // C2050  
      case 1:  
         a = 0;  
      default:  
         a = 2;  
   }  
}  
```  
  
 可能的解決方案：  
  
```  
// C2050b.cpp  
int main() {  
   int a = 1;  
   switch (a) {  
      case 1:  
         a = 0;  
      default:  
         a = 2;  
   }  
}  
```
