# 編譯器錯誤 C2074
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 'class\-key' 初始設定需要使用大括弧  
  
 指定的類別、結構或等位初始設定式未以大括弧括住。
