# <a name="compiler-error-c2077"></a>編譯器錯誤 C2077

非純量欄位初始設定式 'identifier'

您嘗試初始化非純量值 （結構、 等位、 陣列或類別） 的位元欄位。 使用整數或浮點數的值。
