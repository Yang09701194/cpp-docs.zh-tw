# 編譯器錯誤 C2077
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

非純量初始設定式 'identifier'  
  
 您嘗試以非純量值 \(結構、等位、陣列或類別\) 來初始化位元欄位。  請使用整數或浮點數值。
