# 編譯器錯誤 C2083
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不合法的結構\/等位比較  
  
 結構或等位會直接與其他使用者定義的類型進行比較。 除非已定義比較運算子或存在純量類型的轉換，否則不允許這項作業。
