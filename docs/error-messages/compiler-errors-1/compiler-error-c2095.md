# <a name="compiler-error-c2095"></a>編譯器錯誤 C2095
'function': 實質參數具有類型 'void': 參數 'number'  
  
 傳遞至函式的參數是型別`void`，這不允許。 使用 void 的指標 ( `void *`) 改為。  
  
 `number`指出哪一個參數為`void`。
