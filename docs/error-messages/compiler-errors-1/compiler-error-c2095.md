# 編譯器錯誤 C2095
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 實質參數具有型別 'void' : 參數 'number'  
  
 傳至函式的參數為型別 `void`，這是不合法的。  請改用 void 指標 \( `void *`\)。  
  
 `number` 指出哪個參數為 `void`。
