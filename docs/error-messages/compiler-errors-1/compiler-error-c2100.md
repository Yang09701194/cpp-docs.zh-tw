# <a name="compiler-error-c2100"></a>編譯器錯誤 C2100
不合法的間接取值  
  
 間接取值運算子 ( `*` ) 套用至指標值。  
  
 下列範例會產生 C2100:  
  
```  
// C2100.cpp  
int main() {  
   int r = 0, *s = 0;  
   s = &r;  
   *r = 200;   // C2100  
   *s = 200;   // OK  
}  
```
