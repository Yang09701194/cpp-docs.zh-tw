# <a name="compiler-error-c2101"></a>編譯器錯誤 C2101
不可在常數上使用 '&'  
  
 傳址運算子 ( `&` ) 必須將左值 (l-value) 作為運算元。  
  
 下列範例會產生 C2101：  
  
```  
// C2101.cpp  
int main() {  
   char test;  
   test = &'a';   // C2101  
   test = 'a';   // OK  
}  
```
