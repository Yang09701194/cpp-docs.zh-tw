# <a name="compiler-error-c2104"></a>編譯器錯誤 C2104
在位元欄位上使用 '&' 將被忽略  
  
 您無法取得位元欄位的位址。  
  
 下列範例會產生 C2104:  
  
```  
// C2104.cpp  
struct X {  
   int sb : 1;  
};  
  
int main() {  
   X x;  
   &x.sb;   // C2104   
   x.sb;   // OK  
}  
```
