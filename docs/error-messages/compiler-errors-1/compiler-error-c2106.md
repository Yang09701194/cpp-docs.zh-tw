# <a name="compiler-error-c2106"></a>編譯器錯誤 C2106
'operator': 左的運算元必須是左值  
  
 運算子必須為左運算元是左值。  
  
 下列範例會產生 C2106:  
  
```  
// C2106.cpp  
int main() {  
   int a;  
   1 = a;   // C2106  
   a = 1;   // OK  
}  
```
