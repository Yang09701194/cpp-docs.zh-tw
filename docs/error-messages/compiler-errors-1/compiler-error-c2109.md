# <a name="compiler-error-c2109"></a>編譯器錯誤 C2109
註標需要陣列或指標類型  
  
 註標不是陣列的變數上使用。  
  
 下列範例會產生 C2109:  
  
```  
// C2109.cpp  
int main() {  
   int a, b[10] = {0};  
   a[0] = 1;   // C2109  
   b[0] = 1;   // OK  
}  
```
