# 編譯器錯誤 C2114
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator': 指標在左邊; 右邊需要整數值  
  
 `operator` 的左運算元是指標，因此右運算元必須是整數值。
