# <a name="compiler-error-c2121"></a>編譯器錯誤 C2121
'#': 無效的字元： 可能是巨集展開的結果  
  
 無效的 # 字元可能有不正確的巨集，而不是字串化運算子會使用語彙基元帶入的運算子 （#） 所插入 （#）。
