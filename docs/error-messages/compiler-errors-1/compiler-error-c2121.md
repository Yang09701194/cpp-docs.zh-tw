# 編譯器錯誤 C2121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\#' : 無效的字元：可能是巨集展開的結果  
  
 無效的 \# 字元可能是由不正確的巨集所插入，該巨集使用語彙基元 \(Token\) 結合運算子 \(\#\#\)，而非字串化運算子 \(\#\)。
