# <a name="compiler-error-c2121"></a>編譯器錯誤 C2121

' # '：不正確字元：可能是宏展開的結果

使用標記貼上運算子（# #）而不是字串化運算子（#）的錯誤宏，可能插入了不正確 # 字元。
