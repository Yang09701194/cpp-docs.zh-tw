# 編譯器錯誤 C2122
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 名稱清單中的原型參數不合法  
  
 參數不是合法的型別。  ANSI C 不支援使用者定義的型別。
