# <a name="compiler-error-c2129"></a>編譯器錯誤 C2129

靜態函式 ' function ' 已宣告但未定義

對從未定義的 `static` 函式進行正向參考。

必須在檔案範圍內定義 `static` 函數。 如果函式是在另一個檔案中定義，則必須將它宣告 `extern`。
