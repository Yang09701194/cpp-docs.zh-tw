# 編譯器錯誤 C2138
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

沒有使用任何成員來定義列舉型別是不合法的  
  
 當選取 \/Za \(停用 Microsoft 擴充功能\) 時，列舉型別至少要擁有一個成員。
