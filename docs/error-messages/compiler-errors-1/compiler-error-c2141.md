# <a name="compiler-error-c2141"></a>編譯器錯誤 C2141

陣列大小溢位

陣列超過 2 GB 的限制。 減少陣列大小。

## <a name="example"></a>範例

下列範例會產生 C2141。

```
// C2141.cpp
// processor: IPF
class A {
   short m_n;
};

int main()
{
   A* pA = (A*)(-1);
   pA = new A[0x8000000000000001];   // C2141

   A* pA2 = (A*)(-1);
   pA2 = new A[0x80000000000000F];   // OK
}
```
