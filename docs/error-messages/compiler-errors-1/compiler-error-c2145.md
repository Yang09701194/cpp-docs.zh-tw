# 編譯器錯誤 C2145
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

語法錯誤：在識別項之前遺漏 'token'  
  
 編譯器預期 `token` ，但找到識別項。  
  
 此錯誤可能因為區塊中最後一個宣告之後遺漏分號所致。
