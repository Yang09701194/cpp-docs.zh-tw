# <a name="compiler-error-c2147"></a>編譯器錯誤 C2147

語法錯誤: 'identifier' 是新的關鍵字

使用現在是保留的關鍵字的語言識別項。

下列範例會產生 C2147:

```
// C2147.cpp
// compile with: /clr
int main() {
   int gcnew = 0;   // C2147
   int i = 0;   // OK
}
```
