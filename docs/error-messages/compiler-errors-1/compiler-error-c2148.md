# <a name="compiler-error-c2148"></a>編譯器錯誤 C2148
陣列的總大小不可以超過 0x7fffffff 位元組  
  
 陣列超過限制。 減少陣列的大小。  
  
## <a name="example"></a>範例  
 下列範例會產生 C2148:  
  
```  
// C2148.cpp  
#include <stdio.h>  
#include <stdlib.h>  
  
int main( ) {  
   char MyArray[0x7ffffffff];   // C2148  
   char * MyArray2 = (char *)malloc(0x7fffffff);  
  
   if (MyArray2)  
      printf_s("It worked!");  
   else  
      printf_s("It didn't work.");  
}  
```
