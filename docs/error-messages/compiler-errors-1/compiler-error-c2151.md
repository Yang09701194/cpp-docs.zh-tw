# <a name="compiler-error-c2151"></a>編譯器錯誤 C2151
一個以上的語言屬性  
  
 函式具有一個以上的關鍵字 ( `__cdecl`， `__stdcall`，或`__fastcall`) 指定呼叫慣例。
