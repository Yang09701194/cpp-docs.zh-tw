# 編譯器錯誤 C2151
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

超過一個語言屬性  
  
 函式具有多個指定呼叫慣例 \(Calling Convention\) 的關鍵字 \( `__cdecl`、`__stdcall` 或 `__fastcall`\)。
