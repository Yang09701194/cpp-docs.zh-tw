# <a name="compiler-error-c2152"></a>編譯器錯誤 C2152

' identifier '：具有不同屬性之函式的指標

具有一個呼叫慣例（`__cdecl`、`__stdcall`或 `__fastcall`）之函式的指標，會指派給具有另一個呼叫慣例的函式指標。
