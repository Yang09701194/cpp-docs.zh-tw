# 編譯器錯誤 C2152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 具有不同屬性的函式指標  
  
 具有某個呼叫慣例 \(`__cdecl`、`__stdcall` 或 `__fastcall`\) 的函式的指標被指定給具有另一個呼叫慣例的函式的指標。
