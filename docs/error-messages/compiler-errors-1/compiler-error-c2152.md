# <a name="compiler-error-c2152"></a>編譯器錯誤 C2152

'identifier': 具有不同屬性的函式的指標

具有一個呼叫慣例的函式的指標 (`__cdecl`， `__stdcall`，或`__fastcall`) 指派給另一個呼叫慣例的函式的指標。
