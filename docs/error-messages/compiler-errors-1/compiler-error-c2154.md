# <a name="compiler-error-c2154"></a>編譯器錯誤 C2154
'type': 僅允許列舉類型做為編譯器內建類型特性 '__underlying_type' 的引數  
  
 您只能取得列舉型別的基礎型別。  
  
 如需詳細資訊，請參閱[類型特性的編譯器支援](../../windows/compiler-support-for-type-traits-cpp-component-extensions.md)。
