# <a name="compiler-error-c2155"></a>編譯器錯誤 C2155
'？ ': 無效的左運算元，必須是算術或指標類型  
  
 在左邊的運算式`?`無法比較為零。 您必須使用可以比較為零的算術或指標運算式。
