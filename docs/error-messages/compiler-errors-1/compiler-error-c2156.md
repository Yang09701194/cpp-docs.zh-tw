# <a name="compiler-error-c2156"></a>編譯器錯誤 C2156
Pragma 必須定義在函式之外  
  
 必須在全域層級 (函式主體外部) 指定的 pragma 出現在函式內。  
  
 下列範例會產生 C2156：  
  
```  
// C2156.cpp  
#pragma optimize( "l", on )   // OK  
int main() {  
   #pragma optimize( "l", on )   // C2156  
}  
```
