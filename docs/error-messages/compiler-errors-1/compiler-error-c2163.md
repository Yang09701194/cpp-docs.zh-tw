# <a name="compiler-error-c2163"></a>編譯器錯誤 C2163

' function '：不能當做內建函式使用

`intrinsic` 或 `function` pragma 會列出內部格式無法使用的函數。 例如，編譯使用/clr 程式設計的程式時，無法使用某些內建函式。
