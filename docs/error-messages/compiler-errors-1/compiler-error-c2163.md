# 編譯器錯誤 C2163
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 無法當做內建函式使用  
  
 `intrinsic` 或 `function` Pragma 列出一個不能用於內建形式的函式。  例如，在編譯使用 \/clr 程式設計的程式時，某些內建函式並不能使用。
