# <a name="compiler-error-c2163"></a>編譯器錯誤 C2163
'function': 無法當做內建函式的函式  
  
 `intrinsic`或`function`pragma 列出無法使用內建形式的函式。 例如，某些內建函式無法使用。 當編譯使用 /clr 程式設計的程式
