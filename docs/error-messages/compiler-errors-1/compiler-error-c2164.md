# <a name="compiler-error-c2164"></a>編譯器錯誤 C2164
'function': 未宣告的內建函式  
  
 `intrinsic` Pragma 會使用未宣告的函式 (只會發生**/Oi**)。 或者，不包括其標頭檔已使用的編譯器內建函式的其中一種。  
  
 下列範例會產生 C2164:  
  
```  
// C2164.c  
// compile with: /c  
// processor: x86  
// Uncomment the following line to resolve.  
// #include "xmmintrin.h"  
void b(float *p) {  
   _mm_load_ss(p);   // C2164  
}  
```
