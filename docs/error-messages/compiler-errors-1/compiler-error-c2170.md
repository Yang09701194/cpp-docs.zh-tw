# <a name="compiler-error-c2170"></a>編譯器錯誤 C2170
'identifier': 未宣告為函式，無法當做內建  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正  
  
1.  Pragma`intrinsic`用於非函式項目。  
  
2.  Pragma`intrinsic`用於沒有內建形式的函式。
