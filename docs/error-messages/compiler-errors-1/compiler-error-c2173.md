# 編譯器錯誤 C2173
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 實質參數不是指標：參數 number1，參數清單 number2  
  
 傳遞至參數清單 `number2` 的參數 `number1` 不是指標。 函式需要指標。
