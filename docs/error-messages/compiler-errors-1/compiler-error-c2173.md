# <a name="compiler-error-c2173"></a>編譯器錯誤 C2173
'function' : 實質參數不是指標：參數 number1，參數清單 number2  
  
 傳遞至參數清單 `number1` 的參數 `number2` 不是指標。 函式需要指標。
