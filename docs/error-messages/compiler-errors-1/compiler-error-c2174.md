# 編譯器錯誤 C2174
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 實際參數具有型別 'void' : 參數 number1，參數清單 number2  
  
 傳遞給參數清單 `number2` 的參數 `number1` 為 `void` 參數。  參數不能具有型別 `void`。  請改用 `void*`。
