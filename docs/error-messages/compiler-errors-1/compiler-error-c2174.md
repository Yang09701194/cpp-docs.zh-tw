# <a name="compiler-error-c2174"></a>編譯器錯誤 C2174

' function '：實際參數具有類型 ' void '：參數數位1，參數清單數位2

傳遞至參數清單 `number2` 的參數 `number1` 是 `void` 參數。 參數的類型不能 `void`。 請改用 `void*`。
