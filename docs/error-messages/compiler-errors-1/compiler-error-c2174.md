# <a name="compiler-error-c2174"></a>編譯器錯誤 C2174

'function': 實質參數具有類型 'void': 參數 number1，參數清單 number2

參數`number1`傳遞至參數清單`number2`是`void`參數。 參數不能有類型`void`。 請改用 `void*`。
