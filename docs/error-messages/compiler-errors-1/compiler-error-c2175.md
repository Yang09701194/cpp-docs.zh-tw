# 編譯器錯誤 C2175
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'locale' : 無效的地區設定  
  
 指定的地區設定 \(Locale\) 無效。  如需可支援的地區設定資訊，請參閱《*Run\-Time Library Reference*》的 [Language and Country\/Region Strings](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)。
