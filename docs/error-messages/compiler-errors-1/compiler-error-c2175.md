# <a name="compiler-error-c2175"></a>編譯器錯誤 C2175

'地區設定': 無效的地區設定

指定的地區設定不正確。 請參閱[語言和國家/地區字串](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)中*執行階段程式庫參考*支援地區設定。
