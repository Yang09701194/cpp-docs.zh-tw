# <a name="compiler-error-c2175"></a>編譯器錯誤 C2175

[地區設定]：不正確地區設定

指定的地區設定無效。 如需支援的地區設定，請參閱《*執行時間程式庫參考*》中的[語言和國家/地區字串](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)。
