# <a name="compiler-error-c2177"></a>編譯器錯誤 C2177
常數太大  
  
 常數值對所指派的變數類型而言太大。  
  
 下列範例會產生 C2177：  
  
```  
// C2177.cpp  
int main() {  
   int a=18446744073709551616;   // C2177  
   int b=18446744073709551615;   // OK  
}  
```
