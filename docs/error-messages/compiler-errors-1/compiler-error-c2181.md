# <a name="compiler-error-c2181"></a>編譯器錯誤 C2181
不合法的 else (沒有相符的 if)  
  
 每個 `else` 必須具有相符的 `if`。  
  
 下列範例會產生 C2181：  
  
```  
// C2181.cpp  
int main() {  
   int i = 0;  
   else   // C2181  
      i = 1;  
}  
```  
  
 可能的解決方式：  
  
```  
// C2181b.cpp  
int main() {  
   int i = 0;  
   if(i)  
      i = 0;  
   else  
      i = 1;  
}  
```
