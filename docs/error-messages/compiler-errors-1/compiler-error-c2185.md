# 編譯器錯誤 C2185
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 不合法的 \_\_based 配置  
  
 暫存器變數 \(Register Variable\) 或自動 \(區域\) 變數被宣告成 `__based`。  只有全域變數可被宣告成 `__based`。
