# <a name="compiler-error-c2186"></a>編譯器錯誤 C2186

'operator': 類型 'void' 的運算元不合法

運算子具有 `void` 運算元。

下列範例會產生 C2186：

```
// C2186.cpp
// compile with: /c
void func1( void );
int  func2( void );
int i = 2 + func1();   // C2186 func1() is type void
int j = 2 + func2();   // OK both operands are type int
```
