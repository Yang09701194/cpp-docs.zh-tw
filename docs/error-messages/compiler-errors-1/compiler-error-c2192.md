# <a name="compiler-error-c2192"></a>編譯器錯誤 C2192

參數 'number' 宣告不同

C 函式宣告第二次使用不同的參數清單。 C 不支援多載函式。

下列範例會產生 C2192:

```
// C2192.c
// compile with: /Za /c
void func( float, int );
void func( int, float );   // C2192, different parameter list
void func2( int, float );   // OK
```
