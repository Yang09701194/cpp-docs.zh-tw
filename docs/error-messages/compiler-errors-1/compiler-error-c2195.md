# 編譯器錯誤 C2195
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 是資料區段  
  
 `code_seg` Pragma 使用了已用於 `data_seg` Pragma 的區段名稱。  
  
 下列範例會產生 C2195：  
  
```  
// C2195.cpp  
#pragma data_seg("MYDATA")  
#pragma code_seg("MYDATA")   // C2195  
#pragma code_seg("MYDATA2")   // OK  
```
