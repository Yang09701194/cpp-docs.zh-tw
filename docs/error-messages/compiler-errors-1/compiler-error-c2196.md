# <a name="compiler-error-c2196"></a>編譯器錯誤 C2196
已經使用 case 值 'value'。  
  
 switch 陳述式重複使用相同的 case 值。  
  
 下列範例會產生 C2196：  
  
```  
// C2196.cpp  
int main() {  
   int i = 0;  
   switch( i ) {  
   case 0:  
      break;  
   case 0:   // C2196  
   // try the following line instead  
   // case 1:  
      break;  
   }  
}  
```
