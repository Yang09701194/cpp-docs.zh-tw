# <a name="compiler-error-c2197"></a>編譯器錯誤 C2197
'function': 呼叫的引數太多  
  
 編譯器偵測到函式呼叫的參數太多，或函式宣告有誤。  
  
 下列範例會產生 C2197：  
  
```  
// C2197.c  
// compile with: /Za /c  
void func( int );  
int main() {  
   func( 1, 2 );   // C2197 two actual parameters  
   func( 2 );   // OK  
}  
```
