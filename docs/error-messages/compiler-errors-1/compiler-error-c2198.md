# <a name="compiler-error-c2198"></a>編譯器錯誤 C2198
'function'：呼叫的引數太少  
  
 編譯器發現函式呼叫的參數太少，或函式宣告有誤。  
  
 下列範例會產生 C2198：  
  
```  
// C2198.c  
// compile with: /c  
void func( int, int );  
int main() {  
   func( 1 );   // C2198 only one actual parameter  
   func( 1, 1 );   // OK  
}  
```
