# <a name="compiler-error-c2206"></a>編譯器錯誤 C2206

'function': 函式定義不可以使用 typedef

`typedef` 已用來定義函式類型。

下列範例會產生 C2206：

```
// C2206.cpp
typedef int functyp();
typedef int MyInt;
functyp func1 {};   // C2206
int main() {
   MyInt i = 0;   // OK
}
```
