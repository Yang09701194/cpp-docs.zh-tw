# 編譯器錯誤 C2207
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'member'：類別範本的成員無法取得函式類型  
  
 類別範本的 `member` 先前剖析為非靜態資料成員。  它無法重新定義為成員函式。
