# <a name="compiler-error-c2212"></a>編譯器錯誤 C2212

' identifier '：函式的指標無法使用 __based

無法 `__based`宣告函式的指標。 如果您需要以程式碼為基礎的資料，請使用 `__declspec` 關鍵字或 `data_seg` pragma。
