# <a name="compiler-error-c2212"></a>編譯器錯誤 C2212
'identifier': __based 不提供函式指標  
  
 函式的指標不可以宣告為`__based`。 如果您需要的程式碼為基礎的資料，請使用`__declspec`關鍵字或`data_seg`pragma。
