# <a name="compiler-error-c2213"></a>編譯器錯誤 C2213

' 修飾詞 '： __based 的引數不合法

修改 `__based` 的引數無效。

下列範例會產生 C2213：

```cpp
// C2213.cpp
// compile with: /c
int i;
int *j;
char __based(i) *p;   // C2213
char __based(j) *p2;   // OK
```
