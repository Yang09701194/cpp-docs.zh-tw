# <a name="compiler-error-c2213"></a>編譯器錯誤 C2213

'modifier': __based 的引數不合法

引數的修改`__based`無效。

下列範例會產生 C2213:

```
// C2213.cpp
// compile with: /c
int i;
int *j;
char __based(i) *p;   // C2213
char __based(j) *p2;   // OK
```
