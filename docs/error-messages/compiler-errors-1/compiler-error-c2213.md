# 編譯器錯誤 C2213
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'modifier' :  \_\_based 的引數不合法  
  
 修飾 `__based` 的引數無效。  
  
 下列範例會產生 C2213：  
  
```  
// C2213.cpp  
// compile with: /c  
int i;  
int *j;  
char __based(i) *p;   // C2213  
char __based(j) *p2;   // OK  
```
