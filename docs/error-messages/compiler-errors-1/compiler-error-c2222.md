# Compiler Error C2222
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未預期的類型 'type'：必須是基底類別或成員  
  
 初始設定式清單只可以初始化基底類別或類型的成員。  若要修正這個錯誤，請確認在初始設定式清單中僅初始化基底類別或類型的成員。
