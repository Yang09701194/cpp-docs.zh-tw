# <a name="compiler-error-c2223"></a>編譯器錯誤 C2223
'->identifier' 的左邊必須指向結構/等位  
  
 `->` 左邊的運算元不是類別、結構或等位的指標。  
  
 這個錯誤可能因的左運算元是未定義的變數 (因此輸入`int`)。
