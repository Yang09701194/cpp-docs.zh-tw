# 編譯器錯誤 C2223
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\-\>identifier' 的左邊必須指向結構\/等位  
  
 `->` 左邊的運算元並非類別、結構或等位的指標。  
  
 這項錯誤可能是因左邊運算元為未定義的變數 \(因此是型別 `int`\) 而造成。
