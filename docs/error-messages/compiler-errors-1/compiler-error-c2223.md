# <a name="compiler-error-c2223"></a>編譯器錯誤 C2223

'-> 識別項' 的左邊必須指向結構/等位

`->` 左邊的運算元不是類別、結構或等位的指標。

此錯誤可能因是未定義的變數的左運算元 (因此輸入`int`)。
