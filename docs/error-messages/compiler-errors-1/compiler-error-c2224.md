# <a name="compiler-error-c2224"></a>編譯器錯誤 C2224

'. identifier ' 的左邊必須有 struct/union 類型

句點（.）左邊的運算元不是類別、結構或等位。

這個錯誤可能是因為沒有定義的變數而造成的左運算元（因此，類型 `int`）。
