# <a name="compiler-error-c2224"></a>編譯器錯誤 C2224

'.identifier '的左邊必須有結構/等位型別

句號 （.） 左邊的運算元不是類別、 結構或等位。

此錯誤可能因是未定義的變數的左運算元 (因此輸入`int`)。
