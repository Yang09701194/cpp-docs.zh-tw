# <a name="compiler-error-c2231"></a>編譯器錯誤 C2231
'。 ': 左運算元指向 'class-key'，請使用 '->'  
  
 成員選取運算 （.） 左邊的運算元是指標，而不是類別、 結構或等位。  
  
 下列範例會產生 C2231：  
  
```  
// C2231.c  
struct S {  
   int member;  
} s, *ps = &s;  
int main() {  
   ps.member = 0;   // C2231  
  
   // OK  
   ps->member = 0;   // crash  
   s.member = 0;  
}  
```
