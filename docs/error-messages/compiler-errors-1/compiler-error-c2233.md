# <a name="compiler-error-c2233"></a>編譯器錯誤 C2233

'identifier': 包含大小為零的陣列物件的陣列不合法

陣列中的每個物件必須包含至少一個項目。

下列範例會產生 C2233:

```
// C2233.cpp
// compile with: /c
class A {
   char somearray[1];
};

class B {
   char zeroarray[];
};

A array[100];   // OK
B array2[100];   // C2233
```
