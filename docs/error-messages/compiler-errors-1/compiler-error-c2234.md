# <a name="compiler-error-c2234"></a>編譯器錯誤 C2234
'name': 參考的陣列不合法  
  
 不允許參考的指標，因為不可能參考的陣列。  
  
 下列範例會產生 C2234:  
  
```  
// C2234.cpp  
int main() {  
   int i = 0, j = 0, k = 0, l = 0;  
   int &array[4] = {i,j,k,l};   // C2234  
   int array2[4] = {i,j,k,l};   // OK  
}  
```
