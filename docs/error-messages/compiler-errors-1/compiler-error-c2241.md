# <a name="compiler-error-c2241"></a>編譯器錯誤 C2241
'identifier': 成員存取受限制  
  
 程式碼嘗試存取私用或受保護的成員。  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>使用下列可能的解決方式來進行修正  
  
1.  變更成員的存取層級。  
  
2.  宣告成員為存取函式的 `friend` 。
