# <a name="compiler-error-c2251"></a>編譯器錯誤 C2251
命名空間 'namespace' 沒有成員 'member' - 您是指 'member' 嗎?  
  
 編譯器在指定的命名空間中找不到識別項。  
  
 下列範例會產生 C2251：  
  
```  
// C2251.cpp  
// compile with: /c  
namespace A {  
   namespace B {  
      void f1();  
   }  
  
   using namespace B;  
}  
  
void A::f1() {}   // C2251  
void A::B::f1() {}   // OK  
```
