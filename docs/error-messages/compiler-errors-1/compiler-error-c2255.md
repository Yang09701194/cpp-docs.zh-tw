# <a name="compiler-error-c2255"></a>編譯器錯誤 C2255
'element'：不能在類別定義之外  
  
 例如，宣告為 `friend`的非成員函式。  
  
 下列範例會產生 C2255：  
  
```  
// C2255.cpp  
// compile with: /c  
class A {  
private:  
   void func1();  
   friend void func2();  
};  
  
friend void func1() {}   // C2255  
void func2(){}  
```
