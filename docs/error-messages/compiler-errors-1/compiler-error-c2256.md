# <a name="compiler-error-c2256"></a>編譯器錯誤 C2256
在 'function' 上的 friend 規範的使用不正確  
  
 解構函式或建構函式不能指定為[friend](../../cpp/friend-cpp.md)。  
  
 下列範例會產生 C2256:  
  
```  
// C2256.cpp  
// compile with: /c  
class C {  
public:  
   friend ~C();   // C2256  
   ~C();   // OK  
};  
```
