# <a name="compiler-error-c2258"></a>編譯器錯誤 C2258
不合法的純虛擬函式語法，必須是 '= 0'  
  
 所宣告純虛擬函式的語法不正確。  
  
 下列範例會產生 C2258：  
  
```  
// C2258.cpp  
// compile with: /c  
class A {  
public:  
   void virtual func1() = 1; // C2258  
   void virtual func2() = 0;   // OK  
};  
```
