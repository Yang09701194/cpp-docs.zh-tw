# <a name="compiler-error-c2274"></a>編譯器錯誤 C2274
'type': 當做的右邊不合法 '。 ' 運算子  
  
 類型會顯示為成員存取 （.） 運算子的右運算元。  
  
 這個錯誤可能被因嘗試存取使用者定義型別轉換。 使用關鍵字`operator`之間的週期和`type`。  
  
 下列範例會產生 C2286：  
  
```  
// C2274.cpp  
struct MyClass {  
   operator int() {  
      return 0;  
   }  
};  
  
int main() {  
   MyClass ClassName;  
   int i = ClassName.int();   // C2274  
   int j = ClassName.operator int();   // OK  
}  
```
