# <a name="compiler-error-c2276"></a>編譯器錯誤 C2276

' operator '：系結成員函式運算式上不合法的運算

編譯器發現建立成員指標的語法有問題。

下列範例會產生 C2276：

```cpp
// C2276.cpp
class A {
public:
   int func(){return 0;}
} a;

int (*pf)() = &a.func;   // C2276
// try the following line instead
// int (A::*pf3)() = &A::func;

class B {
public:
   void mf() {
      &this -> mf;   // C2276
      // try the following line instead
      // &B::mf;
   }
};

int main() {
   A a;
   &a.func;   // C2276
   // try the following line instead
   // &A::func;
}
```
