# <a name="compiler-error-c2277"></a>編譯器錯誤 C2277
'identifier': 無法取得此成員函式的位址  
  
 您無法取得成員函式的位址。  
  
 下列範例會產生 C2277:  
  
```  
// C2277.cpp  
class A {  
public:  
   A();  
};  
(*pctor)() = &A::A;   // C2277   
```
