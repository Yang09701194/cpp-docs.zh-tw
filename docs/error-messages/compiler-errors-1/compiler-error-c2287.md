# <a name="compiler-error-c2287"></a>編譯器錯誤 C2287
'class': 繼承表示: 'representation1' 是必要的 'representation2' %$ m  
  
 使用比要求的簡單表示法宣告的類別。  
  
 下列範例會產生 C2287:  
  
```  
// C2287.cpp  
// compile with: /vmg /c  
class __single_inheritance X;  
class __single_inheritance Y;  
  
struct A { };  
struct B { };  
struct X : A, B { };  // C2287  X uses multiple inheritance  
struct Y : A { };  // OK  
```
