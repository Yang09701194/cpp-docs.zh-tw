# 編譯器錯誤 C2307
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

如果啟用累加編譯，必須將 Pragma 'pragma' 移出函式  
  
 如果您使用累加編譯，必須將 `data_seg` Pragma 放在函式之間。
