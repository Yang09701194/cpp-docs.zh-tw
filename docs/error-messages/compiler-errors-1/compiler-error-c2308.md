# 編譯器錯誤 C2308
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

正在結合不相符的字串  
  
 指定寬字元 \(Wide Character\) 字串和非寬字元字串進行串連。  您不能將寬字元字串和非寬字元字串串連。
