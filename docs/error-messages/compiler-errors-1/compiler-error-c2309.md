# <a name="compiler-error-c2309"></a>編譯器錯誤 C2309
catch 處理常式必須括號括住例外狀況宣告  
  
 Catch 處理常式有沒有括號括住型別。  
  
 下列範例會產生 C2309:  
  
```  
// C2309.cpp  
// compile with: /EHsc  
#include <eh.h>  
class C {};  
int main() {  
   try {  
      throw "ooops!";  
   }  
   catch C {}   // C2309  
   // try the following line instead  
   // catch( C ) {}  
}  
```
