# <a name="compiler-error-c2310"></a>編譯器錯誤 C2310
catch 處理常式必須指定一個類型  
  
 Catch 處理常式會指定沒有型別或多個類型。  
  
 下列範例會產生 C2310:  
  
```  
// C2310.cpp  
// compile with: /EHsc  
#include <eh.h>  
int main() {  
   try {  
      throw "Out of memory!";  
   }  
   catch( int ,int) {}   // C2310 two types  
   // try the following line instead  
   // catch( int)  {}  
}  
```
