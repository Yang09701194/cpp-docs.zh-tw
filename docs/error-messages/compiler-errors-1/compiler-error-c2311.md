# <a name="compiler-error-c2311"></a>編譯器錯誤 C2311
'exception': 被 '...' 行號  
  
 省略符號 （...） 的 catch 處理常式必須是擲回的最後一個處理常式。  
  
 下列範例會產生 C2311:  
  
```  
// C2311.cpp  
// compile with: /EHsc  
#include <eh.h>  
int main() {  
   try {  
      throw "ooops!";  
   }  
   catch( ... ) {}  
   catch( int ) {}   // C2311  ellipsis handler not last catch  
}  
```
