# <a name="compiler-error-c2312"></a>編譯器錯誤 C2312
'exception1': 是 'exception2' 在行號攔截到  
  
 兩個處理常式會捕捉相同的例外狀況類型。  
  
 下列範例會產生 C2312：  
  
```  
// C2312.cpp  
// compile with: /EHsc  
#include <eh.h>  
int main() {  
    try {  
        throw "ooops!";  
    }  
    catch( signed int ) {}  
    catch( int ) {}   // C2312  
}  
```
