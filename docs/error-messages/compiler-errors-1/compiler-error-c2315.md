# 編譯器錯誤 C2315
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type1' : 參考被 'type2' 攔截，行號:  
  
 先前的處理常式會處理例外狀況類型。 第二個 catch 的參考具有第一個 catch 的類型。
