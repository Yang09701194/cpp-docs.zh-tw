# <a name="compiler-error-c2317"></a>編譯器錯誤 C2317

從第 'number' 行開始的 'try' 區塊沒有 catch 處理常式

`try` 區塊必須至少一個 catch 處理常式。

下列範例會產生 C2317：

```cpp
// C2317.cpp
// compile with: /EHsc
#include <eh.h>
int main() {
   try {
      throw "throw an exception";
   }
   // C2317, no catch handler
}
```

可能的解決方案：

```cpp
// C2317b.cpp
// compile with: /EHsc
#include <eh.h>
int main() {
   try {
      throw "throw an exception";
   }
   catch(char*) {}
}
```
