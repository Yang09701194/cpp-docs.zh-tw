# <a name="compiler-error-c2319"></a>編譯器錯誤 C2319
'try/catch' 之後必須是複合陳述式。 遺漏 '{'  
  
 在 `try` 或 `catch` 陳述式後面，找不到 `try` 或 `catch` 區塊。 這個區塊必須以大括弧括住。  
  
 下列範例會產生 C2319：  
  
```  
// C2319.cpp  
// compile with: /EHsc  
#include <eh.h>  
class C {};  
int main() {  
   try {  
      throw "ooops!";  
   }  
   catch( C ) ;    // C2319  
   // try the following line instead  
   // catch( C ) {}  
}  
```
