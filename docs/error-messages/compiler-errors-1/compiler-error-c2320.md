# 編譯器錯誤 C2320
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

存取規範 'specifier' 的後面必須是 ':'  
  
 關鍵字 `public`、`protected` 或 `private` 的後面必須是冒號。
