# 編譯器錯誤 C2322
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : dllimport 'dllimport' 的位址不是靜態的  
  
 指定了非靜態值作為以 `dllimport` 宣告之函式的位址。
