# <a name="compiler-error-c2324"></a>編譯器錯誤 C2324

'identifier': 非預期的 'name' 的右邊

解構函式會呼叫使用不正確的識別碼。

下列範例會產生 C2324:

```
// C2324.cpp
class A {};
typedef A* pA_t;
int i;

int main() {
   pA_t * ppa = new pA_t;
   ppa->~i;   // C2324
   ppa->~pA_t();   // OK
}
```
