# <a name="compiler-error-c2325"></a>編譯器錯誤 C2325

'type': 非預期的類型，右邊的 'name'

類型不正確的解構函式進行呼叫。

下列範例會產生 C2325:

```
// C2325.cpp
// compile with: /c
class A {};
typedef A* pA_t;
void f() {
    A** ppa = new A *;
    ppa->~A*;   // C2325

   pA_t *ppa2 = new pA_t;
   ppa2->~pA_t();   // OK
}
```
