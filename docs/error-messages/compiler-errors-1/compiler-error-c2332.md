# 編譯器錯誤 C2332
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'typedef' : 遺漏標記名稱  
  
 編譯器 \(Compiler\) 發現不完整的型別定義。  
  
 下列範例會產生 C2332：  
  
```  
// C2332.cpp  
// compile with: /c  
struct S {  
   int i;  
};  
  
typedef struct * pS;   // C2332  
typedef struct S* pS;   // OK  
  
int get_S_i(pS p) {  
   return p->i;  
}  
```
