# <a name="compiler-error-c2334"></a>編譯器錯誤 C2334
未預期的語彙基元，上述 ': 或 {'; 略過函式主體  
  
 下列範例會產生 C2334。 錯誤 C2059 之後，就會發生此錯誤：  
  
```  
// C2334.cpp  
// compile with: /c  
// C2059 expected  
struct s1 {  
   s1   {}   // C2334  
   s1() {}   // OK  
};  
```
