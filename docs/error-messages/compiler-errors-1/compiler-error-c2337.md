# <a name="compiler-error-c2337"></a>編譯器錯誤 C2337
'attribute name': 找不到屬性  
  
 您已使用在這個版本的 Visual C++ 中不受支援的屬性。  
  
 下列範例會產生 C2337：  
  
```  
// C2337.cpp  
// compile with: /c  
[emitidl];  
[module(name="x")];  
[grasshopper]   // C2337, not a supported attribute  
class a{};  
```
