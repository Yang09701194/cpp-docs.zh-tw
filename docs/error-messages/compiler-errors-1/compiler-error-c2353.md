# <a name="compiler-error-c2353"></a>編譯器錯誤 C2353

不允許例外狀況規格

例外狀況規格不允許的 managed 類別成員函式。

下列範例會產生 C2353:

```
// C2353.cpp
// compile with: /clr /c
ref class X {
   void f() throw(int);   // C2353
   void f();   // OK
};
```
