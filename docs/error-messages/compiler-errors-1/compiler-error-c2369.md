# <a name="compiler-error-c2369"></a>編譯器錯誤 C2369
'array': 重複定義; 註標不相同  
  
 陣列已使用不同的註標宣告。  
  
 下列範例會產生 C2369：  
  
```  
// C2369.cpp  
// compile with: /c  
int a[10];  
int a[20];   // C2369  
int b[20];   // OK  
```
