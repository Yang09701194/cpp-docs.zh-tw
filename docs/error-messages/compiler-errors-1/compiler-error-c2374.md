# <a name="compiler-error-c2374"></a>編譯器錯誤 C2374
'identifier' : 重複定義; 多個初始設定  
  
 多次初始化識別項。  
  
 下例會產生 C2374：  
  
```  
// C2374.cpp  
// compile with: /c  
int i = 0;  
int i = 1;   // C2374  
int j = 1;   // OK  
```
