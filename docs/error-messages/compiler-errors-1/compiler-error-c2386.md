# <a name="compiler-error-c2386"></a>編譯器錯誤 C2386
'symbol': 目前範圍中已經有使用這個名稱的符號  
  
 您嘗試建立命名空間別名，但所選名稱已存在。  
  
 下列範例會產生 C2386：  
  
```  
// C2386.cpp  
namespace A {  
   int k;  
}  
  
int i;  
namespace i = A;   // C2386, i already exists  
```
