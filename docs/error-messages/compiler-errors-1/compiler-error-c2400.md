# <a name="compiler-error-c2400"></a>編譯器錯誤 C2400

內嵌組譯工具中發生語法錯誤 'context';找到 'token'

語彙基元會導致語法錯誤，在指定的內容。

下列範例會產生 C2400:

```
// C2400.cpp
// processor: x86
int main() {
   __asm {
      heh ax,bx;   // C2400, heh is not a valid x86 instruction
      mov ax,bx;   // OK
   }
}
```
