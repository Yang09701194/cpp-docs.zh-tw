# 編譯器錯誤 C2402
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 暫存器必須是 'context' 中的索引  
  
 用於間接記憶體運算元的暫存器必須是這個內容中的索引暫存器。
