# 編譯器錯誤 C2405
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

具有位移 \(offset\) 的 short 向前參考不合法  
  
 短距離的向前參考 \(Forward Reference\) 只能參考到標籤。  不能使用額外的位移。
