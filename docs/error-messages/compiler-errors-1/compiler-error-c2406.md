# 編譯器錯誤 C2406
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 'context' 中的名稱未定義  
  
 會搭配使用未定義的識別項與 `SIZE`、`LENGTH` 或 member\-selection \(.\) 運算子。
