# <a name="compiler-error-c2410"></a>編譯器錯誤 C2410

' identifier '： ' coNtext ' 中有不明確的成員名稱

此識別碼是此內容中多個結構或等位的成員。

在造成錯誤的運算元上使用結構或聯集規範。 結構或等位規範是類型的識別碼 `struct` 或 `union` （`typedef` 名稱或與所參考之結構或等位相同類型的變數）。 規範必須是第一個成員選取運算子（）的左運算元，才能使用運算元。
