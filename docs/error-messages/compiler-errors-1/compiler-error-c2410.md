# <a name="compiler-error-c2410"></a>編譯器錯誤 C2410
'identifier': 'context' 中的模稜兩可的成員名稱  
  
 識別碼是一個以上的結構或等位在此內容中的成員。  
  
 造成錯誤的運算元上使用的結構或等位的規範。 結構或等位的規範是類型的識別項`struct`或`union`(`typedef`名稱或結構或等位正在參考相同類型的變數)。 規範必須是第一個成員選取運算子 （.） 的運算元的左的運算元。
