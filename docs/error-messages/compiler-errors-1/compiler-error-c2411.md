# <a name="compiler-error-c2411"></a>編譯器錯誤 C2411
'identifier': 'context' 中的結構/等位成員不合法  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正  
  
1.  識別項不可見的結構或等位在此內容中的成員。  
  
2.  識別項不是結構或等位成員選取 （.） 運算子使用指定的成員。
