# 編譯器錯誤 C2411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 'context' 中的結構\/等位成員不合法  
  
### 若要修正，請檢查下列可能原因  
  
1.  識別項並非此內容中的可見結構或等位的成員。  
  
2.  識別項並非以成員選取 \(.\) 運算子所指定的結構或等位的成員。
