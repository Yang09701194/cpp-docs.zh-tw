# 編譯器錯誤 C2413
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'token' : 記憶體對齊 \(Align\) 的大小不合法  
  
 遺漏 `ALIGN` 指示詞所使用的大小，或大小超過有效範圍。
