# 編譯器錯誤 C2418
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法刪除瀏覽器檔案：filename  
  
 編譯器無法刪除瀏覽器檔案。  
  
### 若要修正，請檢查下列可能原因  
  
1.  檔案是唯讀檔。  
  
2.  檔案被其他處理序 \(Process\) 使用中。
