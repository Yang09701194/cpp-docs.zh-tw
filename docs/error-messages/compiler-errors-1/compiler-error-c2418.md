# <a name="compiler-error-c2418"></a>編譯器錯誤 C2418

無法刪除瀏覽器檔案：檔案名

編譯器無法刪除瀏覽器檔案。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正

1. 檔案是唯讀的。

1. 檔案正由另一個進程使用中。
