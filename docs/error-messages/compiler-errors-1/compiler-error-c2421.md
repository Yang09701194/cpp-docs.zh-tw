# 編譯器錯誤 C2421
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在 'context' 中 PTR 運算子和暫存器搭配使用  
  
 `PTR` 運算子無法與 `register` 運算元搭配使用。
