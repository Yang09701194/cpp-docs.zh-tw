# <a name="compiler-error-c2425"></a>編譯器錯誤 C2425
'token': 'context' 中的非常數運算式  
  
 語彙基元形成此內容中的一部分非常數運算式。  
  
 若要修正這個問題，請使用常數常值或計算取代語彙基元。  
  
 下列範例會產生 C2425：  
  
```  
// C2425.cpp  
// processor: x86  
int main() {  
   int i = 3;  
   __asm {  
      mov eax, [ebp - i]   // C2425  
      mov eax, [ebp - 3]   // OK  
   }  
}  
```
