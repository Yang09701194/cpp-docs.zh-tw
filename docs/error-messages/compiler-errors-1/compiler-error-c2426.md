# 編譯器錯誤 C2426
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'token': 'context' 中有不合法的運算子  
  
 語彙基元在此內容中無法做為運算子。  例如索引運算子無法為巢狀。
