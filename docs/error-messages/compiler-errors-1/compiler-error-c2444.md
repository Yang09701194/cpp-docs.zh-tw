# 編譯器錯誤 C2444
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 使用 ANSI 原型，找到 'type'，必須是 '{' 或 ';'  
  
 函式原型後面接著一個類型。  
  
 造成這個錯誤的原因是遺漏分號或大括號。
