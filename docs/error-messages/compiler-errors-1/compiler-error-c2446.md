# <a name="compiler-error-c2446"></a>編譯器錯誤 C2446
'operator': 沒有從 'type1' 到 'type2' 的轉換  
  
 編譯器無法轉換`type1`至`type2`。 轉換可能不會有意義，因為它違反 C/c + + 語意。
