# 編譯器錯誤 C2446
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator' : 沒有從 'type1' 至 'type2' 的轉換  
  
 編譯器無法在 `type1` 與 `type2` 之間進行轉換。  轉換不合理，因為它違反了 C\/C\+\+ 語意 \(Semantics\)。
