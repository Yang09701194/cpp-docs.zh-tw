# <a name="compiler-error-c2446"></a>編譯器錯誤 C2446

' operator '：無法從 ' type1 ' 轉換為 ' type2 '

編譯器無法將 `type1` 轉換成 `type2`。 轉換可能不合理，因為它違反 C/C++語義。
