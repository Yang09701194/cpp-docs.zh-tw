# <a name="compiler-error-c2465"></a>編譯器錯誤 C2465
不能在括號運算式內定義匿名類型  
  
 括號運算式內定義了匿名結構、等位或列舉類型。 這在 C++ 中無效，因為定義在函式範圍中沒有意義。
