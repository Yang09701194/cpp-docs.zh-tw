# 編譯器錯誤 C2471
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法更新程式資料庫 'file'  
  
 編譯器無法寫入資料庫檔案。  
  
### 若要修復這個問題  
  
-   請確定磁碟有足夠的可用空間。  
  
-   請確定檔案不是唯讀的。  
  
-   確定沒有共用違規。
