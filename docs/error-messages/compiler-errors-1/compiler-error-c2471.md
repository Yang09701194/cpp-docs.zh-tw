# <a name="compiler-error-c2471"></a>編譯器錯誤 C2471

無法更新程式資料庫 ' file '

編譯器無法寫入資料庫檔案。

### <a name="to-fix-the-problem"></a>修復問題

- 請確定磁碟有足夠的可用空間。

- 請確定檔案不是唯讀的。

- 確定沒有共用違規。
