# <a name="compiler-error-c2474"></a>編譯器錯誤 C2474
'keyword'：遺漏相鄰的分號，可能是關鍵字或識別項。  
  
 編譯器預期找到分號，且無法判定您的意圖。 加入分號以解決此錯誤。  
  
## <a name="example"></a>範例  
 下列範例會產生 C2474：  
  
```  
// C2474.cpp  
// compile with: /clr /c  
  
ref class A {  
   ref class B {}  
   property int i;   // C2474 error  
};  
  
// OK  
ref class B {  
   ref class D {};  
   property int i;  
};  
  
ref class E {  
   ref class F {} property;   
   int i;  
};  
```
