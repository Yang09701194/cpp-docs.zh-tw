# <a name="compiler-error-c2477"></a>編譯器錯誤 C2477
'member': 無法透過衍生類別初始化靜態資料成員  
  
 樣板類別的靜態資料成員已初始化不正確。 這是為了符合 ISO c + + 標準的版本 Visual Studio.NET 2003 中之前, 的 Visual c + + 編譯器的重大變更。  
  
 下列範例會產生 C2477:  
  
```  
// C2477.cpp  
// compile with: /Za /c  
template <class T>  
struct S {  
   static int n;  
};  
  
struct X {};  
struct A: S<X> {};  
  
int A::n = 0;   // C2477  
  
template<>  
int S<X>::n = 0;  
```
