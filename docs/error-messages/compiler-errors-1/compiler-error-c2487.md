# <a name="compiler-error-c2487"></a>編譯器錯誤 C2487
'identifier': dll 介面類別的成員可能無法以 dll 介面宣告  
  
 您可以宣告了整個類別或具有 DLL 介面的非 DLL 介面類別的特定成員。 您無法宣告具有 DLL 介面的類別，然後將宣告該類別以 DLL 介面的成員。
