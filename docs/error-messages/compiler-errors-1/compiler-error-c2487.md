# <a name="compiler-error-c2487"></a>編譯器錯誤 C2487

' identifier '： dll 介面類別別的成員不可以使用 dll 介面宣告

您可以使用 DLL 介面，宣告整個類別或非 DLL 介面類別別的特定成員。 您不能宣告具有 DLL 介面的類別，然後使用 DLL 介面宣告該類別的成員。
