# <a name="compiler-error-c2487"></a>編譯器錯誤 C2487

'identifier': dll 介面類別的成員不可以宣告以 dll 介面

您可以宣告了整個類別或非 DLL 介面類別，以 DLL 介面的特定成員。 您無法以 DLL 介面的類別宣告，然後將宣告 DLL 介面與該類別的成員。
