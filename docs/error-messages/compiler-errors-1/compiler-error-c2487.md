# 編譯器錯誤 C2487
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : dll 介面類別的成員可能無法以 dll 介面宣告  
  
 您可使用 DLL 介面宣告整個類別，或非 DLL 介面類別的某些成員。  您不能以 DLL 介面宣告一個類別，接著又以 DLL 介面宣告該類別的成員。
