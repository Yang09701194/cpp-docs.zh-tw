# <a name="compiler-error-c2495"></a>編譯器錯誤 C2495
'identifier': 'nothrow' 只能套用至函式宣告或定義  
  
 [Nothrow](../../cpp/nothrow-cpp.md)擴充的屬性可以套用至函式宣告或定義只。  
  
 下列範例會產生 C2495:  
  
```  
// C2495.cpp  
// compile with: /c  
__declspec(nothrow) class X {   // C2495  
   int m_data;  
} x;  
  
__declspec(nothrow) void test();   // OK  
```
