# <a name="fatal-error-c1002"></a>嚴重錯誤 C1002

編譯器於第二編譯階段堆積空間不足

在其第二個階段，可能是因為太多符號或複雜運算式的程式編譯器動態記憶體空間不足。

### <a name="to-fix-by-using-the-following-possible-solutions"></a>使用下列可能的解決方式來進行修正

1. 您可以將原始程式檔分成數個較小的檔案。

1. 分成較小的子運算式的運算式。

1. 移除其他程式或驅動程式所耗用的記憶體。
