# 嚴重錯誤 C1008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

沒有指定輸入檔案  
  
 未將 C 或 C\+\+ 原始程式檔提供給編譯器進行編譯。 檢查 **CL** 命令列和環境變數中的檔案名稱規格。
