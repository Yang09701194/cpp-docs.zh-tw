# <a name="fatal-error-c1009"></a>嚴重錯誤 C1009

編譯器限制：宏嵌套太深

編譯器嘗試同時展開太多宏。 編譯器有256層級的嵌套宏限制。 將嵌套的宏分割成較簡單的宏。
