# <a name="fatal-error-c1009"></a>嚴重錯誤 C1009

編譯器限制： 巨集巢狀太深

編譯器會嘗試同時展開太多的巨集。 編譯器有 256 個層級的巢狀的巨集的限制。 將巢狀的巨集分割成較簡單的巨集。
