# <a name="fatal-error-c1013"></a>嚴重錯誤 C1013

編譯器限制 : 左括號太多，請簡化運算式或分為數個陳述式

運算式中包含對單一運算式而言太多的括弧層級。 請簡化運算式，或將它切為多個陳述式。

在 Visual C++ 6.0 Service Pack 3 之前，對於單一運算式中的巢狀括弧限制是 59。 目前巢狀括弧的限制為 256。
