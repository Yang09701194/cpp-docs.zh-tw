# <a name="fatal-error-c1014"></a>嚴重錯誤 C1014

Include 檔太多: 深度 = 層級

`#include` 指示詞的巢狀太深。 巢狀指示詞可以包含開啟檔案。 包含指示詞的原始程式檔會計算為一個檔案。
