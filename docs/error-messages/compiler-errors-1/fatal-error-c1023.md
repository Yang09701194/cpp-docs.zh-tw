# <a name="fatal-error-c1023"></a>嚴重錯誤 C1023
'file' : pch 發生未預期的錯誤，請嘗試重建 pch  
  
 很多種問題都會引起 C1023 錯誤，解決方案是重建先行編譯標頭檔。  
  
 如果您嘗試在建置所在電腦以外的電腦上使用先行編譯標頭檔，可能會引發 C1023 錯誤。
