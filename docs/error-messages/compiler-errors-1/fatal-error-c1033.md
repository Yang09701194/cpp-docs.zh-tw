# <a name="fatal-error-c1033"></a>嚴重錯誤 C1033

無法開啟程式資料庫的 pdb

此錯誤可能因磁碟錯誤，防毒程式，已完全關機的情況下，前一個偵錯工具執行個體所建立的暫時鎖定，或者平行建置 mspdbsrv.exe 程序嘗試存取相同的檔案，在其他可能的原因。
