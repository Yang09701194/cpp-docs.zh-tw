# <a name="fatal-error-c1035"></a>嚴重錯誤 C1035
運算式太複雜; 請簡化運算式  
  
 編譯器無法產生複雜運算式的程式碼。 請將運算式分割成數個較簡單的運算式，然後重新編譯。
