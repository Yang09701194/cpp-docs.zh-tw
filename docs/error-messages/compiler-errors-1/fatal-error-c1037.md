# <a name="fatal-error-c1037"></a>嚴重錯誤 C1037

無法開啟目的檔名稱

找不到 [/Fo](../../build/reference/fo-object-file-name.md) 所指定的目的檔。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正

1. 檔案名稱無效。

1. 記憶體不足，無法開啟檔案。

1. 其他程序正在使用此檔案。

1. 唯讀檔具有相同的名稱。
