# <a name="fatal-error-c1038"></a>嚴重錯誤 C1038
編譯器限制：函式：控制流程狀態太複雜；請簡化函式  
  
 函式的控制流程狀態比編譯器所能處理的更多。 請簡化控制流程或將函式分割成較小的函式。
