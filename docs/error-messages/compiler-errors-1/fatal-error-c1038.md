# 嚴重錯誤 C1038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

編譯器限制：函式：控制流程狀態太複雜；請簡化函式  
  
 函式的控制流程狀態比編譯器所能處理的更多。 請簡化控制流程或將函式分割成較小的函式。
