# <a name="fatal-error-c1046"></a>嚴重錯誤 C1046

編譯器限制：結構嵌套太深

結構、等位或類別超過嵌套限制，也就是15個層級。 重寫定義以減少嵌套層級。 使用 `typedef` 定義一或多個嵌套結構，將結構、等位或類別分割成兩個或多個元件。
