# <a name="fatal-error-c1051"></a>嚴重錯誤 C1051

程式資料庫檔案 'pdbfile'，已過時的格式，將它刪除並重新編譯

編譯器無法更新程式資料庫檔案，其具有較舊的版本號碼。 刪除檔案，並重新編譯您的程式與 **/Zi**或是 **/ZI**。 如需詳細資訊，請參閱 [ /z7，/Zi，/ZI （偵錯資訊格式）](../../build/reference/z7-zi-zi-debug-information-format.md)
