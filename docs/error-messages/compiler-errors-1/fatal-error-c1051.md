# <a name="fatal-error-c1051"></a>嚴重錯誤 C1051
程式資料庫檔案 'pdbfile'，格式已過時，將其刪除並重新編譯  
  
 編譯器無法更新程式資料庫檔案，具有較舊的版本號碼。 刪除檔案，並重新編譯您的程式與**/Zi**或**/ZI**。 如需詳細資訊，請參閱[/Z7、 /Zi、 /ZI （偵錯資訊格式）](../../build/reference/z7-zi-zi-debug-information-format.md)
