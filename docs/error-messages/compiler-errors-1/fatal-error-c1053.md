# <a name="fatal-error-c1053"></a>嚴重錯誤 C1053
'\<識別項 >': 函式太大  
  
 函式太大而無法編譯。  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>使用下列可能的解決方式來進行修正  
  
1.  嘗試編譯但不進行最佳化。  
  
2.  將函式分割成較小的函式。  
  
3.  減少內嵌函式的呼叫。
