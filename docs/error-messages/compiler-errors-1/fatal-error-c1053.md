# 嚴重錯誤 C1053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\<識別項\>' : 函式太大  
  
 函式太大而無法編譯。  
  
### 使用下列可能的解決方式來進行修正  
  
1.  嘗試編譯但不進行最佳化。  
  
2.  將函式分割成較小的函式。  
  
3.  減少內嵌函式的呼叫。
