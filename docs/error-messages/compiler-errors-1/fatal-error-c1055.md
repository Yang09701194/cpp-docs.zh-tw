# <a name="fatal-error-c1055"></a>嚴重錯誤 C1055

編譯器限制： 索引鍵不足

來源檔案包含太多符號。 編譯器已用完雜湊索引鍵的符號表。

### <a name="to-fix-by-using-the-following-possible-solutions"></a>使用下列可能的解決方式來進行修正

1. 原始程式檔分成較小的檔案。

1. 消除不必要的標頭檔。

1. 重複使用而不是建立新的暫存和全域變數。
