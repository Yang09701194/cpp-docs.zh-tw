# 嚴重錯誤 C1057
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

巨集展開中未預期的檔案結尾  
  
 編譯器在收集巨集引動引數時遇到原始程式檔的結尾，這可能是遺失巨集引動的右括號所造成的。
