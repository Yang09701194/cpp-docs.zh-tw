# <a name="fatal-error-c1057"></a>嚴重錯誤 C1057

巨集展開中未預期的檔案結尾

編譯器遇到收集巨集引動過程的引數，可能是因為遺漏右括號中的巨集引動過程時的來源檔案的結尾。
