# 嚴重錯誤 C1065
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

編譯器限制：標記不足  
  
 原始程式檔 \(Source File\) 包含超過 65,523 個類別、結構、等位、命名空間，或列舉。  
  
 請將檔案分割成較小的檔案。
