# <a name="fatal-error-c1065"></a>嚴重錯誤 C1065
編譯器限制： 標記不足  
  
 原始程式檔包含多個 65,523 類別、 結構、 等位、 命名空間或列舉。  
  
 將檔案分割成較小的檔案。
