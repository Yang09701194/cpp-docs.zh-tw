# 嚴重錯誤 C1073
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

累加編譯時發生內部錯誤 \(編譯檔案名稱 'filename'，第 number 行\)  
  
 重新編譯檔案時沒有使用累加編譯。
