# 嚴重錯誤 C1075
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

檔案結尾的左語彙基元不相符  
  
 編譯器必須先比對上*語彙基元*，才能達到檔案結尾。  
  
 這個錯誤的成因為不相符的方括弧、括號或其他成對的字元。
