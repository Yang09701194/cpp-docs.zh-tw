# <a name="fatal-error-c1075"></a>嚴重錯誤 C1075

檔案結尾的左語彙基元不相符

編譯器必須先比對上 *語彙基元* ，才能達到檔案結尾。

這個錯誤的成因為不相符的方括弧、括號或其他成對的字元。
