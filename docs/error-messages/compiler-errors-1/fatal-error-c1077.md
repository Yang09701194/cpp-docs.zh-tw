# <a name="fatal-error-c1077"></a>嚴重錯誤 C1077

編譯器限制：不能有超過 <數字> 個的命令列選項

命令列選項的數目超過內部限制。

可能使用 [/D](../../build/reference/d-preprocessor-definitions.md)定義了太多符號。 (請改為將定義放在標頭檔。)
