# 嚴重錯誤 C1082
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法關閉 filetype 檔案：'file': 訊息  
  
 如果訊息指出「錯誤的檔案號碼」，則檔案可能已在前景中關閉，而在背景中編譯。
