# 嚴重錯誤 C1085
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法寫入 filetype 檔案: 'file': 訊息  
  
### 若要修正，請檢查下列可能原因  
  
1.  磁碟機是唯讀的。  
  
2.  磁碟機已滿。  
  
3.  共用違規。  
  
4.  如果訊息指出「錯誤的檔案編號」，檔案可能在背景編譯時，已於前景關閉。
