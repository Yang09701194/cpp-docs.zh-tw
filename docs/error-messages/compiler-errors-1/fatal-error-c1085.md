# <a name="fatal-error-c1085"></a>嚴重錯誤 C1085
無法寫入 filetype 檔案: 'file': 訊息  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正  
  
1.  磁碟機為唯讀。  
  
2.  磁碟機已滿。  
  
3.  共用違規。  
  
4.  如果訊息指出「錯誤的檔案號碼」，則檔案可能已在前景中關閉，而在背景中編譯。
