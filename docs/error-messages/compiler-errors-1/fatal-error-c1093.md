# 嚴重錯誤 C1093
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

API 呼叫 'function name' 失敗 'location of call' : 'text from run\-time'  
  
 呼叫 .NET 函式失敗。  COM Runtime 不一定會提供 ***text from run\-time***。  
  
 如需系統錯誤訊息的詳細資訊，請參閱 winerror.h 系統檔案和 [FormatMessage](http://msdn.microsoft.com/library/windows/desktop/ms679351)。
