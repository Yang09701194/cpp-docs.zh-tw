# <a name="fatal-error-c1093"></a>嚴重錯誤 C1093
API 呼叫 '函式名稱' 失敗 '呼叫的位置': '執行階段中的文字'  
  
 .NET 函式的呼叫失敗。 ***執行階段中的文字***可能或可能不需要提供由 COM 執行階段。  
  
 如需系統錯誤訊息的詳細資訊，請參閱 winerror.h 系統檔案，並[FormatMessage](http://msdn.microsoft.com/library/windows/desktop/ms679351)。
