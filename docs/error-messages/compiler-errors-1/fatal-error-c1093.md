# <a name="fatal-error-c1093"></a>嚴重錯誤 C1093

> API 呼叫 '*函式名稱*「 失敗 」*呼叫的位置*':'*從執行階段的文字*'

.NET 函式呼叫失敗。 *執行階段中的文字*字串可能會或可能不是 COM 執行階段提供的。

如需有關系統錯誤訊息的詳細資訊，請參閱 winerror.h 系統檔案，並[FormatMessage](/windows/desktop/api/winbase/nf-winbase-formatmessage)。
