# <a name="fatal-error-c1093"></a>嚴重錯誤 C1093

> API 呼叫 ' 函式*名稱*' 失敗 '*呼叫 ' 的位置*': '*執行時間的文字*'

呼叫 .NET 函數失敗。 *來自執行時間*字串的文字不一定由 COM 執行時間提供。

如需系統錯誤訊息的詳細資訊, 請參閱 winerror.h 系統檔案和[FormatMessage](/windows/win32/api/winbase/nf-winbase-formatmessage)。
