# <a name="fatal-error-c1098"></a>嚴重錯誤 C1098

版本與編輯後繼續程式引擎不符

您所使用的偵錯工具版本不符合用來建立可執行檔的編譯器。 如果重新編譯無法修正問題，則可能需要重新安裝 Visual C++，以確定偵錯工具和編譯器的版本正確。
