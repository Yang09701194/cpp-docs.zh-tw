# <a name="fatal-error-c1099"></a>嚴重錯誤 C1099
編輯後繼續程式引擎正在終止編譯  
  
 編輯後繼續已載入先行編譯的標頭檔來準備編譯程式碼變更，但後續動作 (例如，先行編譯的標頭 `#include` 陳述式之前的程式碼變更，或停止偵錯工具) 防止編輯後繼續使用該程序來完成編譯。 您不需要採取任何動作來修正這個錯誤。
