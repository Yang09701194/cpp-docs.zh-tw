# <a name="fatal-error-c1100"></a>嚴重錯誤 C1100

無法初始化 OLE：系統錯誤訊息

編譯器無法初始化元件物件模型（COM）程式庫。 請參閱[CoInitialize](/windows/win32/api/objbase/nf-objbase-coinitialize)。
