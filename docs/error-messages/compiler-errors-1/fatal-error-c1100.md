# 嚴重錯誤 C1100
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法初始化 OLE：系統錯誤訊息  
  
 編譯器無法初始化元件物件模型 \(Component Object Model，COM\) 程式庫。  請參閱 [CoInitialize](http://msdn.microsoft.com/library/windows/desktop/ms678543)。
