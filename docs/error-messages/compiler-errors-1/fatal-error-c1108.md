# <a name="fatal-error-c1108"></a>嚴重錯誤 C1108

> 找不到 DLL： '*dll name*'

在路徑中找不到指定的 DLL （*dll 名稱*）。

若要解決這個錯誤，請C++重新安裝視覺效果，或從安裝將適當的 .dll 檔案複製到您的電腦。
