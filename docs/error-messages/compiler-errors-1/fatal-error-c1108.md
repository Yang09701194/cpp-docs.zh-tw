# <a name="fatal-error-c1108"></a>嚴重錯誤 C1108

> 找不到 DLL: '*dll 名稱*'

指定的 DLL (*dll 名稱*) 找不到路徑中。

若要解決這個錯誤，請重新安裝 VisualC++或安裝適當的.dll 檔複製到您的電腦。
