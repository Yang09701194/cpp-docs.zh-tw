# <a name="fatal-error-c1113"></a>嚴重錯誤 C1113
\#using 'file'  
  
 只有 Microsoft Intermediate Language (MSIL) 格式的檔案可以傳遞至[#using](../../preprocessor/hash-using-directive-cpp.md)指示詞。 [/Clr](../../build/reference/clr-common-language-runtime-compilation.md)編譯器選項可讓您建立的 MSIL 的輸出檔。 其他[!INCLUDE[vsprvs](../../assembler/masm/includes/vsprvs_md.md)]語言也會產生 MSIL 檔案。
