# <a name="fatal-error-c1113"></a>嚴重錯誤 C1113

在 ' file ' 上使用失敗 \#

只有 Microsoft 中繼語言（MSIL）格式的檔案可以傳遞至[#using](../../preprocessor/hash-using-directive-cpp.md)指示詞。 [/Clr](../../build/reference/clr-common-language-runtime-compilation.md)編譯器選項可讓您建立 MSIL 輸出檔案。 其他 Visual Studio 語言也會產生 MSIL 檔案。
