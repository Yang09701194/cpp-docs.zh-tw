# 嚴重錯誤 C1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

呼叫 CryptoAPI 失敗  
  
 編譯器呼叫了 CryptoAPI，而呼叫已失敗。  重新安裝 Visual Studio，而且可能也要重新安裝作業系統。
