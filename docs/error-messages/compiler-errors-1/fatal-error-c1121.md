# <a name="fatal-error-c1121"></a>嚴重錯誤 C1121

呼叫 CryptoAPI 失敗，請重新安裝 Visual Studio 或作業系統

編譯器所做的 CryptoAPI 呼叫與呼叫失敗。 重新安裝 Visual Studio，並可能重新安裝作業系統。
