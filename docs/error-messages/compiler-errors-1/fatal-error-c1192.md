# 嚴重錯誤 C1192
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法在 'file' 上使用 \#using  
  
 只有一個 Microsoft Intermediate Language \(MSIL\) 格式的檔案可傳遞給 [\#using](../../preprocessor/hash-using-directive-cpp.md) 指示詞。  [\/clr](../../build/reference/clr-common-language-runtime-compilation.md) 編譯器選項可以讓您建立 MSIL 輸出檔。  其他的 Visual Studio 語言也會產生 MSIL 檔。
