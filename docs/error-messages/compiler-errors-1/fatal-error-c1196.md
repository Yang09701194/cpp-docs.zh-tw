# 嚴重錯誤 C1196
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 在類型程式庫 'typelib' 中找到的識別項不是有效的 C\+\+ 識別項  
  
 類型程式庫中的其中一個識別項不是有效的 C\+\+ 識別項。 類型程式庫無法與 [\#import](../../preprocessor/hash-import-directive-cpp.md) 搭配使用。
