# <a name="fatal-error-c1201"></a>嚴重錯誤 C1201
類別樣板定義中發生語法錯誤，無法繼續  
  
 剖析類別樣板定義時發生非預期的錯誤。  
  
 請修正任何其他錯誤，並重新編譯。 如果該失敗，請注意錯誤的情況下嘗試找出問題並建立可重現的測試案例，請連絡[Microsoft 產品支援服務](/visualstudio/ide/talk-to-us)。
