# 嚴重錯誤 C1201
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

類別樣板定義中發生語法錯誤，無法繼續  
  
 剖析類別樣板定義時發生非預期的錯誤。  
  
 請修正任何其他錯誤，並重新編譯。 如果失敗，請記錄錯誤發生的情況、設法找出問題癥結，並建立可重現的測試案例，然後連絡 [Microsoft 產品支援服務](../Topic/Talk%20to%20Us.md)。
