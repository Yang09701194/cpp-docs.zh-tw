# <a name="fatal-error-c1205"></a>嚴重錯誤 C1205
所安裝的執行階段版本不支援泛型  
  
 編譯器使用的 Common Language Runtime 版本不是目前編譯器所支援的執行階段版本。  例如，泛型功能需要與編譯器相對應的執行階段。  
  
 您可能需要修改路徑規格。
