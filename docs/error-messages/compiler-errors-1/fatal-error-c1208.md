# <a name="fatal-error-c1208"></a>嚴重錯誤 C1208
所安裝的執行階段版本不支援在堆疊上配置參考類別  
  
 如果您有目前版本的編譯器，但 Common Language Runtime 來自舊版本，就會發生 C1208。  
  
 編譯器的某個功能可能無法在執行階段的舊版本上運作。  
  
 請安裝要與編譯器搭配使用的 Common Language Runtime 版本。
