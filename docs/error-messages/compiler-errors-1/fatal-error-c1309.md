# <a name="fatal-error-c1309"></a>嚴重錯誤 C1309

不相符的 C2 版本。DLL 和 PGODB\<版本 >。URLMON.DLL

您用來建立和使用特性[指引優化](../../build/profile-guided-optimizations.md)的工具組包含不相符的元件。  如果您無法以手動方式解決這個錯誤，請重新安裝 Visual C++。
