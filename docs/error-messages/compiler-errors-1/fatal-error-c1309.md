# 嚴重錯誤 C1309
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

C2.DLL 與 PGODB\<version\>.DLL 的版本不符  
  
 您用來建置及使用[特性指引最佳化](../../build/reference/profile-guided-optimizations.md)的工具組包含不相符的元件。  如果您無法以手動方式解決這個錯誤，請重新安裝 Visual C\+\+。
