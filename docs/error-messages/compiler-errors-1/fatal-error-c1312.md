# <a name="fatal-error-c1312"></a>嚴重錯誤 C1312

函式中的條件式分支太多。 請簡化或重整原始程式碼。

程式碼太複雜，編譯器若要處理便會用盡堆疊記憶體。  請簡化您的程式碼。
