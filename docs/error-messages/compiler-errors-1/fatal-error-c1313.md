# <a name="fatal-error-c1313"></a>嚴重錯誤 C1313

編譯器限制：類型區塊的嵌套可能不能比數位層級更深

例外狀況處理（或結構化例外狀況處理）區塊的嵌套太深。  請簡化您的程式碼。
