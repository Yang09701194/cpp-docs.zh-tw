# <a name="fatal-error-c1313"></a>嚴重錯誤 C1313
編譯器限制： 區塊類型可能不巢狀結構深度數個層級  
  
 例外狀況處理 （或結構化例外狀況處理） 區塊巢狀太深。  請簡化您的程式碼。
