# 嚴重錯誤 C1313
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

編譯器限制 : type 區塊的巢狀結構深度不可超過 number 層  
  
 例外狀況處理 \(或結構化例外狀況處理\) 區塊巢狀結構太深。請簡化您的程式碼。
