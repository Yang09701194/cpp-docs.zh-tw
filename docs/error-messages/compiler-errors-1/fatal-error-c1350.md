# <a name="fatal-error-c1350"></a>嚴重錯誤 C1350
載入 dll 'dll' 時發生錯誤： 找不到 dll  
  
 找不到 DLL 支援所嘗試的作業。  
  
 這表示您的安裝問題，而且您應該從產品光碟重新安裝此 DLL。
