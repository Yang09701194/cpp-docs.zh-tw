# 嚴重錯誤 C1350
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

載入 dll 'dll' 時發生錯誤: 找不到 dll  
  
 找不到支援嘗試的作業之 DLL。  
  
 這個錯誤表示您的安裝出現問題，您應該從產品光碟重新安裝此 DLL。
