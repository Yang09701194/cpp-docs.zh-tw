# <a name="fatal-error-c1351"></a>嚴重錯誤 C1351

> 載入 dll 時發生錯誤 '*dll*': 不相容的版本

找不到錯誤的 dll 版本。

這表示您的安裝問題，您應該重新安裝 Visual Studio。
