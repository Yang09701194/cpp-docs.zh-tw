# <a name="fatal-error-c1351"></a>嚴重錯誤 C1351

> 載入 dll 時發生錯誤 '*dll*': 不相容版本  

找不到 DLL 的版本錯誤。

這表示您的安裝問題，而且您應該重新安裝 Visual Studio。
