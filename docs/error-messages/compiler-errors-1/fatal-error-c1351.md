# 嚴重錯誤 C1351
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

載入 dll 'dll' 時發生錯誤: 不相容的版本  
  
 找到錯誤的 DLL 版本。  
  
 這個錯誤表示您的安裝出現問題，您應該從產品光碟重新安裝此 DLL。
