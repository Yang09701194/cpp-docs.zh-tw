# <a name="fatal-error-c1351"></a>嚴重錯誤 C1351
載入 dll 'dll' 時發生錯誤： 版本不相容  
  
 找不到 DLL 的版本錯誤。  
  
 這表示您的安裝問題，而且您應該從產品光碟重新安裝此 DLL。
