# <a name="fatal-error-c1506"></a>嚴重錯誤 C1506

無法回復的區塊範圍錯誤

區塊太大，無法編譯。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正

1. 不成對的大括弧

1. 異常大的函式或類別
