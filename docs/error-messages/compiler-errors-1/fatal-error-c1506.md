# 嚴重錯誤 C1506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法回復的區塊範圍錯誤  
  
 區塊太大以致於無法編譯。  
  
### 若要修正，請檢查下列可能原因  
  
1.  括號不對稱  
  
2.  異常大型的函式或類別
