# 嚴重錯誤 C1508
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

編譯器限制：'function': 引數超過 65535 位元組上限  
  
 函式的正式參數超過 65535 個位元組的限制。
