# 嚴重錯誤 C1601
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不支援內嵌組譯碼的 opcode  
  
 這種錯誤指出編譯器 .exe 檔中出現不符合的狀況，這可能是安裝不完全所造成的。  例如，您可能已經安裝 Service Pack，但未安裝處理器的套件。  請安裝所有需要的軟體。
