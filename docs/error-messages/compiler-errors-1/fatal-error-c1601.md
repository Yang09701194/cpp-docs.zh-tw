# <a name="fatal-error-c1601"></a>嚴重錯誤 C1601

不支援內嵌組譯碼的 opcode

此錯誤表示您編譯器的.exe 檔案，因為未完成的安裝可能會發生不相符。 例如，您可能已安裝 service pack，但不是將處理器的組件。 安裝所有必要的產品。
