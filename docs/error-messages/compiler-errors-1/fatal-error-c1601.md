# <a name="fatal-error-c1601"></a>嚴重錯誤 C1601

不支援內嵌組譯碼的 opcode

此錯誤表示您的編譯器 .exe 檔案不相符，可能是因為安裝不完整所造成。 例如，您可能已安裝 Service Pack，但不是處理器套件。 安裝所有必要的產品。
