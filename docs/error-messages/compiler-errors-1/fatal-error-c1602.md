# <a name="fatal-error-c1602"></a>嚴重錯誤 C1602
要求的內建功能未支援  
  
 此錯誤表示編譯器.exe 檔，因為未完成的安裝可能發生的不相符。 例如，您可能已安裝 service pack，但不是將處理器的組件。 安裝所有必要的產品。
