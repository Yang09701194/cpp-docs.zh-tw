# <a name="fatal-error-c1603"></a>嚴重錯誤 C1603
內嵌組譯碼分支目標超過範圍 'number' 個位元組  
  
 JCXZ 或 JECXZ 指令和其指定的目標標籤的計算的距離超過 128 個位元組。 更新您的程式碼，讓標籤位於更接近指令。
