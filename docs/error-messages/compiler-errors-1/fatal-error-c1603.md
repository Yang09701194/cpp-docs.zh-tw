# <a name="fatal-error-c1603"></a>嚴重錯誤 C1603

內嵌組解碼分支目標超出 ' number ' 個位元組的範圍

JCXZ 或 JECXZ 指令與其指定目標標籤之間的計算距離大於128個位元組。 更新您的程式碼，讓標籤更接近指令。
