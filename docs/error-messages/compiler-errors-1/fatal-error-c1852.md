# <a name="fatal-error-c1852"></a>嚴重錯誤 C1852
'filename' 是無效的先行編譯標頭檔  
  
 檔案不是先行編譯標頭檔。  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正  
  
1.  使用 **/Yu** 或 **#pragma hdrstop**指定的檔案無效。  
  
2.  如果您未指定副檔名，編譯器會假設副檔名為 .pch。
