# <a name="fatal-error-c1904"></a>嚴重錯誤 C1904
錯誤的提供者互動：'file'  
  
 這個錯誤表示屬性提供者失敗。  
  
 請注意錯誤的情況下，嘗試找出問題並建立可重現的測試案例，然後再連絡[告訴我們](/visualstudio/ide/talk-to-us)。
