# Fatal Error C999
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不明訊息    請選擇 Visual C\+\+ \[說明\] 功能表上的 \[技術支援\] 命令或開啟技術支援說明檔以取得詳細資訊  
  
 此錯誤通常表示有一些來自不同版本的編譯器的混合檔案，或是編譯器安裝已損毀。  使用 \[控制台\] 中的 \[程式和功能\] 小程式來修復或重新安裝產品。
