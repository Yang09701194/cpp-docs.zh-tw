# <a name="fatal-error-c999"></a>嚴重錯誤 C999

不明訊息請選擇 視覺效果上的 技術支援 命令C++說明 功能表，或開啟技術支援說明檔，如需詳細資訊

此錯誤通常表示有一些來自不同版本的編譯器的混合檔案，或是編譯器安裝已損毀。 使用**程式和功能** 小程式來修復或重新安裝產品控制項台中。
