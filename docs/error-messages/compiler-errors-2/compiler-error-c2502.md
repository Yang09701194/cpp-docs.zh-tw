# <a name="compiler-error-c2502"></a>編譯器錯誤 C2502

'identifier': 基底類別上的太多存取修飾詞

基底類別有一個以上的存取修飾詞。 只有一個存取修飾詞 (`public`， `private`，或`protected`) 允許。

下列範例會產生 C2502:

```
// C2502.cpp
// compile with: /c
class A { };
class B { };
class C : private public A { };   // C2502

// OK
class D : private A {};
class E : public A, private B {};
```
