# 編譯器錯誤 C2510
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': '::' 的左邊必須是類別\/結構\/等位  
  
 類別、結構或等位名稱必須出現在範圍解析運算子 \(`::` 運算子\) 左邊。
