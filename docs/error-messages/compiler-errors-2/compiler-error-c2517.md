# <a name="compiler-error-c2517"></a>編譯器錯誤 C2517

' identifier '： '：： ' 的右邊未定義

範圍解析運算子（`::`）右邊的識別碼必須是左邊的類別、結構或等位的已定義成員。 如果沒有命名類別、結構或等位，則右邊的識別碼必須以全域範圍宣告。
