# <a name="compiler-error-c2517"></a>編譯器錯誤 C2517
'identifier': 右邊 ':: ' 未定義  
  
 範圍解析運算子右邊的識別項 (`::`) 必須是類別、 結構或等位在左側的已定義的成員。 如果不為任何類別、 結構或等位，在右邊的識別項必須宣告具有全域領域。
