# 編譯器錯誤 C2517
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : '::' 的右邊未定義  
  
 範圍解析運算子 \(`::`\) 右側的識別項，必須是左側類別、結構或等位的已定義成員。  如果未命名類別、結構或等位，則位於右側的識別項必須以全域範圍 \(Scope\) 宣告。
