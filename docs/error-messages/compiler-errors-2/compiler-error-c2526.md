# 編譯器錯誤 C2526
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier1' : C 連結函式無法傳回 C\+\+ 類別 'identifier2'  
  
 使用 C 連結定義的函式無法傳回使用者定義型別。
