# <a name="compiler-error-c2537"></a>編譯器錯誤 C2537
'specifier': 不合法的連結規格  
  
 可能的原因：  
  
1.  不支援的連結規範。 支援只"C"的連結規範。  
  
2.  一個以上的函式多載函式集合中指定"C"連結。 這是不允許的。  
  
 下列範例會產生 C2537:  
  
```  
// C2537.cpp  
// compile with: /c  
extern "c" void func();   // C2537  
extern "C" void func2();   // OK  
```
