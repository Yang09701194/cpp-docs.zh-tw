# <a name="compiler-error-c2541"></a>編譯器錯誤 C2541

'delete': 刪除： 無法刪除不是指標的物件

[刪除](../../cpp/delete-operator-cpp.md)運算子用於不是指標的物件。

下列範例會產生 C2541:

```
// C2541.cpp
int main() {
   int i;
   delete i;   // C2541 i not a pointer

   // OK
   int *ip = new int;
   delete ip;
}
```
