# <a name="compiler-error-c2542"></a>編譯器錯誤 C2542
'identifier': 類別物件已初始化沒有建構函式  
  
 沒有任何建構函式的參數清單，符合此初始化。  
  
 這個錯誤可能被因不正確的參數，初始化。
