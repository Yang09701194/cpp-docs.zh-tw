# <a name="compiler-error-c2542"></a>編譯器錯誤 C2542

' identifier '：類別物件沒有用於初始化的函式

沒有任何具有符合初始化之參數清單的函式。

此錯誤可能是因為初始化中的參數不正確所造成。
