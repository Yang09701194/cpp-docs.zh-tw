# 編譯器錯誤 C2542
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 類別物件沒有初始化所需的建構函式  
  
 沒有具有參數清單的建構函式符合此初始化。  
  
 這項錯誤可能是因初始化中不正確函式而造成。
