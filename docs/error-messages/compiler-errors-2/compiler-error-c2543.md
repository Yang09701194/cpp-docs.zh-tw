# 編譯器錯誤 C2543
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

運算子 '\[\]' 必須有 '\]'  
  
 註標運算子 \(Subscripting Operator\) 遺漏左括號。  
  
 錯誤發生的原因可能是巨集展開 \(Macro Expansion\)。
