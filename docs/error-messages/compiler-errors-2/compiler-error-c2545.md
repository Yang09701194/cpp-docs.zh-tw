# 編譯器錯誤 C2545
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator' : 找不到多載運算子  
  
 運算子無法與提供的運算元一起使用。  您必須提供一個配合所需運算元的多載運算子。  
  
 如果運算元的型別不正確，可能就會發生這項錯誤。  
  
 如果您定義轉換運算子或採用單一參數的建構函式，也許就能修正這項錯誤。
