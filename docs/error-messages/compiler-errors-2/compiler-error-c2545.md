# <a name="compiler-error-c2545"></a>編譯器錯誤 C2545
'operator': 找不到多載運算子  
  
 運算子不能與提供的運算元。 您必須提供必要的運算元多載的運算子。  
  
 如果運算元的類型不正確，可能造成這個錯誤。  
  
 如果您定義轉換運算子或建構函式採用單一參數，可能會修正此錯誤。
