# <a name="compiler-error-c2545"></a>編譯器錯誤 C2545

' operator '：找不到多載運算子

運算子不能與提供的運算元一起使用。 您必須提供具有所需運算元的多載運算子。

如果運算元的類型不正確，可能會導致此錯誤。

如果您定義轉換運算子或採用單一參數的函式，就可以修正這個錯誤。
