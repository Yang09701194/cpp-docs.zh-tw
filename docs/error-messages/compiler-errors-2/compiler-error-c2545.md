# <a name="compiler-error-c2545"></a>編譯器錯誤 C2545

'operator': 無法到多載運算子

運算子不適用於使用提供的運算元。 您必須提供具有必要的運算元多載的運算子。

如果運算元具有類型不正確，可能造成此錯誤。

如果您定義轉換運算子或建構函式接受單一參數，可能會修正此錯誤。
