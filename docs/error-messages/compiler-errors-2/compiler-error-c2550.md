# <a name="compiler-error-c2550"></a>編譯器錯誤 C2550

'identifier': 建構函式初始設定式清單只允許在建構函式定義

不是建構函式的函式定義上使用基底類別初始設定式清單。

下列範例會產生 C2550:

```
// C2550.cpp
// compile with: /c
class C {
public:
   C();
};

class D : public C {
public:
   D();
   void func();
};

void D::func() : C() {}  // C2550
D::D() : C() {}   // OK
```
