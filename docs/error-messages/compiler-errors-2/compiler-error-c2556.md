# <a name="compiler-error-c2556"></a>編譯器錯誤 C2556
'identifier': 多載函式只有傳回型別  
  
 多載函式會有不同的傳回類型，但相同的參數清單。 每個多載函式必須有不同的型式參數清單。  
  
 下列範例會產生 C2556:  
  
```  
// C2556.cpp  
// compile with: /c  
class C {  
   int func();  
   double func();   // C2556  
   int func(int i);   // ok parameter lists differ  
};  
```
