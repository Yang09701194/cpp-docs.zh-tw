# <a name="compiler-error-c2557"></a>編譯器錯誤 C2557

'identifier': 沒有建構函式無法初始化 private 和 protected 成員

只有成員和 Friend 才能將值指派給 private 和 protected 成員。 應該在類別建構函式中初始化非公用成員。
