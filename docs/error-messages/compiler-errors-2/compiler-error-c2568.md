# 編譯器錯誤 C2568
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier1' : 無法解析函式多載  
  
 編譯器無法判斷呼叫哪一個多載函式。  傳遞至函式的實質參數 \(Actual Parameter\) 必須轉換成與其中一個多載函式的型式參數相符，但是沒有任何一個比對結果比其他的比對結果更明確。
