# <a name="compiler-error-c2568"></a>編譯器錯誤 C2568
'identifier1': 無法解析函式多載  
  
 編譯器無法判斷哪一個多載函式呼叫。 傳遞至函式的實質參數必須轉換成符合多載函式，其中的型式參數，但沒有一個符合項目明確優於其他所有項目。
