# <a name="compiler-error-c2568"></a>編譯器錯誤 C2568

' identifier1 '：無法解析函數多載

編譯器無法判斷要呼叫哪個多載函數。 傳遞給函式的實際參數必須轉型，以符合其中一個多載函式的型式參數，但沒有任何相符專案明顯優於所有其他函數。
