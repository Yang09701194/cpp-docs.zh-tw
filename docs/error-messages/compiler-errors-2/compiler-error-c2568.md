# <a name="compiler-error-c2568"></a>編譯器錯誤 C2568

'identifier1': 無法解析函式多載

編譯器無法判斷哪一個多載函式呼叫。 傳遞至函式的實質參數必須轉換成符合的型式參數的其中一個多載的函式，但沒有一個相符項目是明確優於其他所有人。
