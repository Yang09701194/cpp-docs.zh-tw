# 編譯器錯誤 C2573
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class': 無法刪除指向此類型物件的指標; 類別沒有 'operator delete' 的未定位多載。  
  
 類別遺漏未定位的 delete 運算子。
