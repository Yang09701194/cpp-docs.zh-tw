# 編譯器錯誤 C2579
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法解析型別 'type' \(offset\)。它必須在 file 中  
  
 C2579 一定都是跟在 C4691 之後發生。  如需詳細資訊，請參閱[編譯器警告 \(層級 1\) C4691](../../error-messages/compiler-warnings/compiler-warning-level-1-c4691.md)。
