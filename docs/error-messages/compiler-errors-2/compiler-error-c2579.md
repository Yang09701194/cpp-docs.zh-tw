# <a name="compiler-error-c2579"></a>編譯器錯誤 C2579

無法解析類型 ' type ' （offset）。 預期會在檔案中

C2579 一律會遵循 C4691。 如需詳細資訊，請參閱[編譯器警告（層級1） C4691](../../error-messages/compiler-warnings/compiler-warning-level-1-c4691.md)。
