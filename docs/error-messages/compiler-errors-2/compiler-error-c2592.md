# 編譯器錯誤 C2592
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class': 'base\_class\_2' 繼承自 'base\_class\_1'，不能重新指定  
  
 您只能指定不是繼承自其他基底類別的基底類別。  在此情況下，`class` 的規格中只需要 `base_class_1`，因為 `base_class_1` 已經繼承 `base_class_2`。
