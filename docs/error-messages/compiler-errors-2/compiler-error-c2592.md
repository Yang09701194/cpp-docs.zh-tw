# <a name="compiler-error-c2592"></a>編譯器錯誤 C2592

'class': 'base_class_2' 繼承自 'base_class_1'，不能重新指定

您只能指定不是繼承自其他基底類別的基底類別。 在此情況下，`base_class_1` 的規格中只需要 `class`，因為 `base_class_1` 已經繼承 `base_class_2`。
