# <a name="compiler-error-c2614"></a>編譯器錯誤 C2614
'class1': 成員初始化不合法: 'class2' 不是基底或成員  
  
 只有成員或基底類別，可以出現在類別或結構的初始設定清單中。  
  
## <a name="example"></a>範例  
 下列範例會產生 C2614。  
  
```  
// C2614.cpp  
// compile with: /c  
struct A {  
   int i;  
   A( int ia ) : B( i ) {};   // C2614 B is not a member of A  
};  
  
struct A2 {  
   int B;  
   int i;  
   A2( int ia ) : B( i ) {};   // OK  
};  
```
