# 編譯器錯誤 C2614
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class1' : 不合法的成員初始化: 'class2' 不是基底或成員類別  
  
 類別或結構的初始化清單內只能出現一個成員或基底類別。  
  
## 範例  
 下列範例會產生 C2614。  
  
```  
// C2614.cpp  
// compile with: /c  
struct A {  
   int i;  
   A( int ia ) : B( i ) {};   // C2614 B is not a member of A  
};  
  
struct A2 {  
   int B;  
   int i;  
   A2( int ia ) : B( i ) {};   // OK  
};  
```
