# <a name="compiler-error-c2616"></a>編譯器錯誤 C2616
'conversion': 無法將非左值 'type1' 到 'type2' 隱含轉換不是常數  
  
 無法從非左值初始化的參考。  
  
 這是 ANSI 相容性錯誤 ([/Za](../../build/reference/za-ze-disable-language-extensions.md)) 是 Microsoft 擴充功能下的警告 (**/Ze**)。
