# <a name="compiler-error-c2616"></a>編譯器錯誤 C2616

' 轉換 '：無法將非左值 ' type1 ' 隱含轉換為不是 const 的 ' type2 '

無法從非左值初始化參考。

這是 ANSI 相容性（[/za](../../build/reference/za-ze-disable-language-extensions.md)）底下的錯誤，以及 Microsoft extensions （ **/ze**）底下的警告。
