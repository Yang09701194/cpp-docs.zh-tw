# 編譯器錯誤 C2616
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'conversion' : 無法將非左值的 'type1' 隱含轉換為不是常數的 'type2'  
  
 參考不能從非左值上初始化。  
  
 在 ANSI 相容性 \([\/Za](../../build/reference/za-ze-disable-language-extensions.md)\) 之下這是一項錯誤，在 Microsoft Extensions \(**\/Ze**\) 之下這是一項警告。
