# 編譯器錯誤 C2630
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在必須是逗號分隔清單中找到 'symbol'  
  
 這個符號出現在需要逗號的內容中。  
  
 下列範例會產生 C2630：  
  
```  
// C2630.cpp  
// compile with: /c  
struct D {  
   D(int);  
};  
  
struct E {  
   E(int);  
};  
  
class C : public D, public E {  
   C();  
};  
  
C::C() : D(0) ; E(0) { }   // C2630  
C::C() : D(0), E(0) {}   // OK  
```
