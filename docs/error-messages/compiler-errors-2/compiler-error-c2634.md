# <a name="compiler-error-c2634"></a>編譯器錯誤 C2634

' & 類別：： member '：參考成員的指標不合法

參考成員的指標已宣告。

下列範例會產生 C2634：

```cpp
// C2634.cpp
int mem;
struct S {
   S() : rf(mem) { }
   int &rf;
};
int (S::*pdm) = &S::rf;   // C2634
```
