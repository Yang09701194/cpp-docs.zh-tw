# <a name="compiler-error-c2635"></a>編譯器錯誤 C2635
無法轉換成 'identifier1 *'' identifier2\*'; 隱含轉換，從虛擬基底類別  
  
 轉換需要轉換從`virtual`基底類別衍生的類別，這不允許。  
  
 下列範例會產生 C2635:  
  
```  
// C2635.cpp  
class B {};  
class D : virtual public B {};  
class E : public B {};  
  
int main() {  
   B b;  
   D d;  
   E e;  
  
   D * pD = &d;  
   E * pE = &e;  
   pD = (D*)&b;   // C2635  
   pE = (E*)&b;   // OK  
}  
```
