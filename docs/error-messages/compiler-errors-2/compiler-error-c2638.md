# <a name="compiler-error-c2638"></a>編譯器錯誤 C2638
'identifier': __based 修飾詞不合法的成員指標上  
  
 `__based`修飾詞不能用於成員的指標。  
  
 下列範例會產生 C2638:  
  
```  
// C2638.cpp  
void *a;  
  
class C {  
public:  
   int i;  
   int j;  
   int func();  
};  
int __based (a) C::* cpi = &C::i;  // C2638  
int (__based (a) C::* cpf)() = &C::func; // c2638  
```
