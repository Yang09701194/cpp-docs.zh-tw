# 編譯器錯誤 C2640
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 在參考上的 \_\_based 修飾詞不合法  
  
 `__based` 修飾詞只能用在指標上。  
  
 下列範例會產生 C2640：  
  
```  
// C2640.cpp  
void f(int i) {  
    void *vp;  
    int _based(vp) &vr = I;  // C2640  
}  
```
