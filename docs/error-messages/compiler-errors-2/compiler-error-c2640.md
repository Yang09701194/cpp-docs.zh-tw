# <a name="compiler-error-c2640"></a>編譯器錯誤 C2640

'identifier': __based 修飾詞不合法的參考

`__based`可以只指標上使用修飾詞。

下列範例會產生 C2640:

```
// C2640.cpp
void f(int i) {
    void *vp;
    int _based(vp) &vr = I;  // C2640
}
```
