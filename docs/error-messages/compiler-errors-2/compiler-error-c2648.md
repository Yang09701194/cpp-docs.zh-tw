# <a name="compiler-error-c2648"></a>編譯器錯誤 C2648
'identifier': 做為成員的預設參數必須是靜態成員  
  
 非靜態成員做為預設參數。  
  
 下列範例會產生 C2648:  
  
```  
// C2648.cpp  
// compile with: /c  
class C {  
public:  
   int i;  
   static int j;  
   void func1( int i = i );  // C2648  i is not static  
   void func2( int i = j );  // OK  
};  
```
