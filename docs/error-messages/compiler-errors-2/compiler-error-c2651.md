# <a name="compiler-error-c2651"></a>編譯器錯誤 C2651

' 資料類型 '： ' operator ' 的左邊必須是類別、結構或等位

若要使用樣板參數，如同它是類別，請使用類別（而不是整數類型）將類別範本特殊化。
