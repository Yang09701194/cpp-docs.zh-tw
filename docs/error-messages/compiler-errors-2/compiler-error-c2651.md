# 編譯器錯誤 C2651
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'data type' : 'operator' 的左邊必須是類別、結構或等位  
  
 若要將樣板參數當做是一個類別使用，請使用類別來指定類別樣板，而不是整數型別 \(Integral Type\)。
