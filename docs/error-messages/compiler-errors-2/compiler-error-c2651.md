# <a name="compiler-error-c2651"></a>編譯器錯誤 C2651

'data type': 'operator' 的左邊必須是類別、 結構或等位

若要使用的樣板參數，如同它是類別，是專為類別樣板，而不是整數類資料類型的類別。
