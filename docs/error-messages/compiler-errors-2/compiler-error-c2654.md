# <a name="compiler-error-c2654"></a>編譯器錯誤 C2654
'identifier': 嘗試在成員函式之外存取成員  
  
 存取成員宣告中。 只有在成員函式，就可以存取成員資料。  
  
 嘗試初始化中宣告的變數時，可能造成這個錯誤。 針對此用途使用建構函式。
