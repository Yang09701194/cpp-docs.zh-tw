# 編譯器錯誤 C2654
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 嘗試在成員函式之外存取成員  
  
 在宣告中存取成員。  只能在成員函式中存取成員資料。  
  
 這項錯誤可能會因為嘗試在宣告中初始化變數而造成。  為此，請使用建構函式。
