# <a name="compiler-error-c2654"></a>編譯器錯誤 C2654

' identifier '：嘗試存取成員函式外部的成員

成員會在宣告中存取。 成員資料只能在成員函式中存取。

嘗試初始化宣告中的變數時，可能會造成此錯誤。 基於此目的，請使用函式。
