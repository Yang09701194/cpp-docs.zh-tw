# <a name="compiler-error-c2655"></a>編譯器錯誤 C2655

'identifier': 目前範圍中不合法的宣告或定義

識別項可以只在全域範圍內重新宣告。

下列範例會產生 C2655:

```
// C2655.cpp
class A {};
class B {
public:
   static int i;
};

int B::i;  // OK

int main() {
   A B::i;  // C2655
}
```
