# <a name="compiler-error-c2656"></a>編譯器錯誤 C2656

' function '：不允許使用函數做為位欄位

函式會宣告為位欄位的成員。

這個錯誤可能是由「函式初始化運算式」清單中的語法錯誤所造成。
