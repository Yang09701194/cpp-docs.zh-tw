# 編譯器錯誤 C2656
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 不允許將函式當做位元欄位  
  
 宣告函式為位元欄位 \(Bit Field\) 的成員。  
  
 這項錯誤可能是因為建構函式初始設定式清單中的語法錯誤所致。
