# <a name="compiler-error-c2656"></a>編譯器錯誤 C2656
'function': 函式不允許當做位元欄位  
  
 函式宣告為位元欄位的成員。  
  
 這個錯誤可能被因建構函式初始設定式清單中有語法錯誤。
