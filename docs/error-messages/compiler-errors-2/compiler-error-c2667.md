# <a name="compiler-error-c2667"></a>編譯器錯誤 C2667

'function': 沒有任何數字的多載具有最佳轉換

多載函式呼叫是模稜兩可，無法解析。

比對其中一個多載函式的函式呼叫中的實際參數所需的轉換必須是絕對優於所有其他多載函式所需的轉換。

在函式樣板的部分排序，請參閱知識庫文件 Q240869 如需詳細資訊。
