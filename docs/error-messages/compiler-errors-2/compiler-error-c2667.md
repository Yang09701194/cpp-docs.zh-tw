# 編譯器錯誤 C2667
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function': 沒有 number 多載具有最佳轉換  
  
 多載函式呼叫模稜兩可且無法被解析。  
  
 將函式呼叫中的實質參數符合成多載函式之一所需的轉換，必須比所有其他多載函式所需的轉換來的嚴格。  
  
 如需函式樣板的部分指定詳細資訊，請參閱知識庫文件 Q240869。
