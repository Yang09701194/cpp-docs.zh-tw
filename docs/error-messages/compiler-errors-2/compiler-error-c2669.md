# <a name="compiler-error-c2669"></a>編譯器錯誤 C2669
匿名等位中不允許的成員函式  
  
[匿名等位](../../cpp/unions.md#anonymous_unions)不能有成員函式。  
  
## <a name="example"></a>範例  
下列範例會產生 C2669:  
  
```cpp  
// C2669.cpp  
struct X {  
   union {  
      int i;  
      void f() {   // C2669, remove function  
         i = 0;   
      }  
   };  
};  
```
