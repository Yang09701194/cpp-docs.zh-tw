# 編譯器錯誤 C2670
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 函式樣板無法將參數 number 從型別 'type' 轉換  
  
 無法將參數轉換成所需的型別。  
  
 如果您建立明確轉換，可能會修正這項錯誤。
