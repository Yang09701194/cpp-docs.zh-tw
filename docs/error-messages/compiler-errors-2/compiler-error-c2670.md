# <a name="compiler-error-c2670"></a>編譯器錯誤 C2670
'identifier': 函式樣板無法從類型 'type' 轉換參數號碼  
  
 參數無法轉換為需的類型。  
  
 如果您建立的明確轉換，可能會修正此錯誤。
