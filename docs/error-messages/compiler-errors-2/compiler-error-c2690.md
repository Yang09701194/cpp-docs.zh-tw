# <a name="compiler-error-c2690"></a>編譯器錯誤 C2690

'operator'：無法在 Managed 或 WinRT 陣列上執行指標算術

不允許在 Managed 或 WinRT 陣列上使用指標算術。 請使用陣列索引標記法來周遊陣列。
