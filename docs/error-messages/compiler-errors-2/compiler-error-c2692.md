# <a name="compiler-error-c2692"></a>編譯器錯誤 C2692
'function_name': 完全原型函式中使用的 C 編譯器所需 '/ /clr' 選項  
  
 當編譯的.NET managed 程式碼時，C 編譯器需要 ANSI 函式宣告。 此外，如果函式可不接受任何參數，它必須明確宣告`void`做為參數類型。
