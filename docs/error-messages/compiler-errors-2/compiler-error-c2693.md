# <a name="compiler-error-c2693"></a>編譯器錯誤 C2693
'operator' : 對 Managed 或 WinRT 陣列參考的非法比較  
  
 您不能測試 Managed 或 WinRT 陣列是否有任何種類的不相等。 例如，您可以測試以查看 Managed 陣列是否相等，但是您不能測試以查看一個陣列是否大於或小於另一個陣列。
