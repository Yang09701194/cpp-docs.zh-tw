# <a name="compiler-error-c2698"></a>編譯器錯誤 C2698
using 宣告為 '宣告 1' 不能同時存在與 using 宣告為' 宣告 2'  
  
 一旦[using 宣告](../../cpp/using-declaration.md)資料成員，任何使用不允許使用相同的名稱和相同範圍中宣告，因為只有函式可以多載。  
  
 下列範例會產生 C2698:  
  
```  
// C2698.cpp  
struct A {  
   int x;  
};  
  
struct B {  
   int x;  
};  
  
struct C : A, B {  
   using A::x;  
   using B::x;   // C2698  
}  
```
