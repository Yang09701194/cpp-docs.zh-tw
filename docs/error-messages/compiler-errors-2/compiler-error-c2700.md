# <a name="compiler-error-c2700"></a>編譯器錯誤 C2700

'identifier': 無法擲回 (請使用 /W4 以取得更多資訊)

無法擲回物件。 請使用 [/W4](../../build/reference/compiler-option-warning-level.md) 編譯，以取得詳細的診斷資訊。
