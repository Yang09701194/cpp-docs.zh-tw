# <a name="compiler-error-c2701"></a>編譯器錯誤 C2701

' function '：函式樣板不可以是區域類別的 friend

區域類別不能有樣板函式做為 friend 函數。

下列範例會產生 C2701：

```cpp
// C2701.cpp
// compile with: /c
template<typename T>   // OK
void f1(const T &);

void MyFunction() {
   class MyClass {
      template<typename T> friend void f2(const T &);   // C2701
   };
}
```
