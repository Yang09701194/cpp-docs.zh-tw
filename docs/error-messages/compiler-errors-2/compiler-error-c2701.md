# 編譯器錯誤 C2701
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function': 函式樣板不能是區域類別的 friend  
  
 區域類別不能以樣板函式做為 friend 函式。  
  
 下列範例會產生 C2701：  
  
```  
// C2701.cpp  
// compile with: /c  
template<typename T>   // OK  
void f1(const T &);  
  
void MyFunction() {  
   class MyClass {  
      template<typename T> friend void f2(const T &);   // C2701  
   };  
}  
```
