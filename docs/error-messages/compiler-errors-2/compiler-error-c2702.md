# <a name="compiler-error-c2702"></a>編譯器錯誤 C2702

__except 不可以出現在終止區塊中

例外狀況處理常式 (`__try`/`__except`) nelze vnořit do`__finally`區塊。

下列範例會產生 C2702:

```
// C2702.cpp
// processor: x86 IPF
int Counter;
int main() {
   __try {}
   __finally {
      __try {}   // C2702
      __except( Counter ) {}   // C2702
   }
}
```
