# <a name="compiler-error-c2703"></a>編譯器錯誤 C2703
不合法的 __leave 陳述式  
  
 A`__leave`陳述式必須是內部`__try`區塊。  
  
 下列範例會產生 C2703:  
  
```  
// C2703.cpp  
int main() {  
   __leave;   // C2703  
   __try {  
      // try the following line instead  
      __leave;  
   }  
   __finally {}  
}  
```
