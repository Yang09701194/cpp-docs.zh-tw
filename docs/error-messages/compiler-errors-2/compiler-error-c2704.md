# 編譯器錯誤 C2704
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': \_\_va\_start 內建只允許在 varargs 中  
  
 `__va_start` 內建函式用在具有固定數目之引數的函式宣告中。
