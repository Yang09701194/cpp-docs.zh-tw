# <a name="compiler-error-c2706"></a>編譯器錯誤 C2706
不合法的 __except 沒有相符\_或 (遺漏 '}' 中\_或區塊？)  
  
 編譯器找不到的右大括號`__try`區塊。  
  
 下列範例會產生 C2706:  
  
```  
// C2706.cpp  
int main() {  
   __try {  
      void f();  
   // C2706  } missing here  
   __except(GetExceptionCode() == 0x0) {  
   }  
}  
```
