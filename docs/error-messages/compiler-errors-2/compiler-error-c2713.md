# 編譯器錯誤 C2713
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

每個函式只允許一種例外狀況處理型式  
  
 同一個函式中不能使用結構化例外狀況處理 \(`__try`\/`__except`\) 和 C\+\+ 例外狀況處理 \(`try`\/`catch`\)。
