# <a name="compiler-error-c2713"></a>編譯器錯誤 C2713
只有一種允許每個函式的例外狀況處理  
  
 您無法使用結構化例外狀況處理 (`__try`/`__except`) 和 c + + 例外狀況處理 (`try`/`catch`) 相同的函式中。
