# <a name="compiler-error-c2713"></a>編譯器錯誤 C2713

每個函式只允許一種例外狀況處理形式

您無法在相同的函式中使用結構化例外C++狀況處理（`__try`/`__except`）和例外狀況處理（`try`/`catch`）。
