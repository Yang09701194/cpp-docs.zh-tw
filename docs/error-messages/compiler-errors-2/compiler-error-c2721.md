# <a name="compiler-error-c2721"></a>編譯器錯誤 C2721
'specifier': 不合法的運算子關鍵字和型別之間的儲存類別規範  
  
 使用者定義型別轉換套用至所有的儲存類別，因此您無法指定儲存類別中的型別轉換。
