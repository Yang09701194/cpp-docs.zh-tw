# 編譯器錯誤 C2721
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'specifier' : 在運算子關鍵字和型別間的儲存類別規範不合法  
  
 使用者定義型別轉換套用至所有儲存類別，因此，在型別轉換中您無法指定儲存類別。
