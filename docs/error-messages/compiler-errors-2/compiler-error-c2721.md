# <a name="compiler-error-c2721"></a>編譯器錯誤 C2721

' 規範 '：運算子關鍵字和類型之間的儲存類別規範不合法

使用者定義型別轉換會套用至所有儲存類別，因此您無法在類型轉換中指定儲存類別。
