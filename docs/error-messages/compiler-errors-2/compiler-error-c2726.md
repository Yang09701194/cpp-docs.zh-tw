# <a name="compiler-error-c2726"></a>編譯器錯誤 C2726

'gcnew' 只可用來建立具有 Managed 或 WinRT 類型的物件

您無法在記憶體回收堆積上建立原生類型的執行個體。

下列範例會產生 C2726，並說明如何加以修正：

```
// C2726.cpp
// compile with: /clr
using namespace System;
class U {};
ref class V {};
value class W {};

int main() {
   U* pU = gcnew U;    // C2726
   U* pU2 = new U;   // OK
   V^ p2 = gcnew V;   // OK
   W p3;   // OK

}
```
