# <a name="compiler-error-c2736"></a>編譯器錯誤 C2736
轉換中不允許 'keyword' 關鍵字  
  
 關鍵字是無效的轉型。  
  
 下列範例會產生 C2736:  
  
```  
// C2736.cpp  
int main() {  
   return (virtual) 0;   // C2736  
   // try the following line instead  
   // return 0;  
}  
```
