# <a name="compiler-error-c2738"></a>編譯器錯誤 C2738

'declaration': 模稜兩可或不是 'type' 的成員

函式宣告不正確。

下列範例會產生 C2738:

```
// C2738.cpp
struct A {
   template <class T> operator T*();
   // template <class T> operator T();
};

template <>
A::operator int() {   // C2738

// try the following line instead
// A::operator int*() {

// or use the commented member declaration

   return 0;
}
```
