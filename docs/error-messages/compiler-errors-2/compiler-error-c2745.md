# <a name="compiler-error-c2745"></a>編譯器錯誤 C2745
'token': 這個語彙基元無法轉換為識別項  
  
 識別項必須同時包含合法的字元。  
  
 下列範例會產生 C2745:  
  
```  
// C2745.cpp  
// compile with: /clr  
int main() {  
   int __identifier([));   // C2745  
}  
```
