# <a name="compiler-error-c2752"></a>編譯器錯誤 C2752

' template '：有一個以上的部分特製化符合範本引數清單

具現化是不明確的。

下列範例會產生 C2752：

```cpp
// C2752.cpp
template<class T, class U>
struct A {};

template<class T, class U>
struct A<T*, U> {};

template<class T, class U>
struct A<T,U*> {};

// try the following line instead
// template<class T, class U> struct A<T*,U*> {};

int main() {
   A<char*,int*> a;   // C2752 an instantiation

   // OK
   A<char*,int> a1;
   A<char,int*> a2;
   A<char,int> a3;
}
```
