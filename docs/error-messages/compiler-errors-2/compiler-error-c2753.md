# 編譯器錯誤 C2753
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class' : 樣板類別已經定義過了  
  
 如果樣板引數清單符合參數清單，則編譯器將會視其為同一個樣板。  同一個樣板不允許定義兩次。  
  
 下列範例會產生 C2753：  
  
```  
// C2753.cpp  
template<class T>  
struct A {};  
  
template<class T>  
struct A<T> {};   // C2753  
// try the following line instead  
// struct A<int> {};  
  
template<class T, class U, class V, class W, class X>  
struct B {};  
```
