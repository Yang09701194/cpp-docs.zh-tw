# <a name="compiler-error-c2761"></a>編譯器錯誤 C2761
'function': 不允許成員函式重新宣告  
  
 您無法重新宣告成員函式。 您可以定義它，但不是能重新宣告。  
  
## <a name="example"></a>範例  
 下列範例會產生 C2761。  
  
```  
// C2761.cpp  
class a {  
   int t;  
   void test();  
};  
  
void a::a;     // C2761  
void a::test;  // C2761  
  
```  
  
## <a name="example"></a>範例  
 無法定義類別或結構的非靜態成員。  下列範例會產生 C2761。  
  
```  
// C2761_b.cpp  
// compile with: /c  
struct C {  
   int s;  
   static int t;  
};  
  
int C::s;   // C2761  
int C::t;   // OK  
```
