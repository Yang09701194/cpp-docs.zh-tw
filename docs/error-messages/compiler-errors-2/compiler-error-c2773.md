# <a name="compiler-error-c2773"></a>編譯器錯誤 C2773
\#匯入和 #using 只適用於 c + + 編譯器  
  
 C 編譯器無法辨識`#import`前置處理器指示詞。 將來源檔編譯為 c + +。 使用[/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md)如有必要。
