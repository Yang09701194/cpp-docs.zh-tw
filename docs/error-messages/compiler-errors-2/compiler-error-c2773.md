# <a name="compiler-error-c2773"></a>編譯器錯誤 C2773

\#匯入和 #using 僅適用C++于編譯器

C 編譯器無法辨識 `#import` 預處理器指示詞。 將來源編譯為C++。 如有必要，請使用[/tp](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) 。
