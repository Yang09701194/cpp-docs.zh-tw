# <a name="compiler-error-c2780"></a>編譯器錯誤 C2780
'declaration'：預期 N 引數 - 提供 M  
  
 函式樣板包含太少或太多引數。  
  
 下列範例會產生 C2780，並示範如何修正此問題：  
  
```  
// C2780.cpp  
template<typename T>  
void f(T, T){}  
  
int main() {  
   f(1);  // C2780  
   // try the following line instead  
   // f(1,2);  
}  
```
