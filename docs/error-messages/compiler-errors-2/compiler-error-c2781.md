# <a name="compiler-error-c2781"></a>編譯器錯誤 C2781
'declaration': 必須要有至少 value1 引數-提供 value2  
  
 使用變數的參數清單的函式樣板包含太少引數。  
  
 下列範例會產生 C2781:  
  
```  
// C2781.cpp  
template<typename T>  
void f(T, T, ...){}  
  
int main() {  
   f(1);   // C2781  
  
   // try the following line instead  
   f(1,1);  
}  
```
