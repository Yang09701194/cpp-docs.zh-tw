# 編譯器錯誤 C2781
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'declaration' : 必須至少有 value1 個引數 \- 已經提供 value2 個  
  
 使用可變個數參數清單的函式樣板的引數太少。  
  
 下列範例會產生 C2781：  
  
```  
// C2781.cpp  
template<typename T>  
void f(T, T, ...){}  
  
int main() {  
   f(1);   // C2781  
  
   // try the following line instead  
   f(1,1);  
}  
```
