# 編譯器錯誤 C2795
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'super::function' 不是成員函式  
  
 每當您試圖使用 [super](../../cpp/super.md) 來存取一個成員，而非一個成員函式時，就會出現這項錯誤訊息。
