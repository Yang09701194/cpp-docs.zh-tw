# <a name="compiler-error-c2802"></a>編譯器錯誤 C2802
靜態成員 'operator operator' 具有沒有型式參數  
  
 由運算子宣告`static`成員函式必須有至少一個參數。  
  
 下列範例會產生 C2802:  
  
```  
// C2802.cpp  
// compile with: /clr /c  
ref class A {  
   static operator+ ();   // C2802  
   static operator+ (A^, A^);   // OK  
};  
```
