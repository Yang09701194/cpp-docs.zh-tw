# <a name="compiler-error-c2805"></a>編譯器錯誤 C2805
二進位 '運算子 operator operator' 具有參數太少  
  
 二元運算子沒有任何參數。  
  
 下列範例會產生 C2805:  
  
```  
// C2805.cpp  
// compile with: /c  
class X {  
public:  
   X operator< ( void );   // C2805 must take one parameter  
   X operator< ( X );   // OK  
};  
```
