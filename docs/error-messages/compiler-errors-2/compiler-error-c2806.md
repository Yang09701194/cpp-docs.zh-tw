# <a name="compiler-error-c2806"></a>編譯器錯誤 C2806

'operator operator' 具有太多的型式參數

多載的運算子的參數太多。

下列範例會產生 C2806:

```
// C2806.cpp
// compile with: /c
class X {
public:
   X operator++ ( int, int );   // C2806 more than 1 parameter
   X operator++ ( int );   // OK
} ;
```
