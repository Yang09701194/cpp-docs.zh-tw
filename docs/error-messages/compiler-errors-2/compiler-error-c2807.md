# 編譯器錯誤 C2807
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

後置 'operator operator' 的第二個型式參數必須是 'int'  
  
 後置運算子的第二個參數的型別錯誤。  
  
 下列範例會產生 C2807：  
  
```  
// C2807.cpp  
// compile with: /c  
class X {  
public:  
   X operator++ ( X );   // C2807 nonvoid parameter  
   X operator++ ( int );   // OK, int parameter  
};  
```
