# <a name="compiler-error-c2808"></a>編譯器錯誤 C2808

一元 'operator operator' 具有太多的型式參數

一元運算子具有 void 參數清單。

下列範例會產生 C2808:

```
// C2808.cpp
// compile with: /c
class X {
public:
   X operator! ( X );   // C2808 nonvoid parameter list
   X operator! ( void );   // OK
};
```
