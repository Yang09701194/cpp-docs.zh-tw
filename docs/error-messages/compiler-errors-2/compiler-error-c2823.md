# <a name="compiler-error-c2823"></a>編譯器錯誤 C2823  
  
> typedef 範本是不合法  
  
範本中不允許`typedef`定義。  
  
## <a name="example"></a>範例  
  
下列範例會產生 C2823，並示範修正此問題的一種方法：  
  
```cpp  
// C2823.cpp  
template<class T>  
typedef struct x {  
   T i;   // C2823 can't use T, specify data type and delete template  
   int i;   // OK  
} x1;  
```
