# <a name="compiler-error-c2827"></a>編譯器錯誤 C2827

' operator operator ' 無法以一元形式全域覆寫

運算子不能在物件外具有一元形式。

### <a name="to-fix-by-using-the-following-possible-solutions"></a>使用下列可能的解決方式來進行修正

1. 將多載運算子設為物件的本機。

1. 選擇適當的一元運算子來多載。
