# <a name="compiler-error-c2828"></a>編譯器錯誤 C2828
'operator operator' 無法全域覆寫的二進位格式  
  
 運算子不能有外部物件的二進位格式。  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>使用下列可能的解決方式來進行修正  
  
1.  對物件進行多載的運算子本機。  
  
2.  選擇適當的一元運算子多載。
