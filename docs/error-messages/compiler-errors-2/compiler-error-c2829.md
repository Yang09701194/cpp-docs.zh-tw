# 編譯器錯誤 C2829
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator operator' 不可以擁有可變個數參數清單  
  
 只有兩個運算子可以使用可變個數參數清單：  
  
-   函式呼叫 `()`  
  
-   `new`
