# 編譯器錯誤 C2830
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

只有 'operator new' 的位置參數可以有預設值  
  
 [operator new](../Topic/operator%20new%20\(%3Cnew%3E\).md) 的標準型式參數不能有預設值。  只有使用者定義的位置參數可以指定預設值。
