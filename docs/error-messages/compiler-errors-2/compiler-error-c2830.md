# <a name="compiler-error-c2830"></a>編譯器錯誤 C2830

只有 'operator new' 的位置參數可以有預設值

標準的型式參數，如[new 運算子](../../standard-library/new-operators.md#op_new)不能有預設值。 只有使用者定義的位置參數可以指定預設值。
