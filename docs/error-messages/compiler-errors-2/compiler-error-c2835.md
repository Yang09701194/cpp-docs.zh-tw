# 編譯器錯誤 C2835
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

使用者定義轉換 'type' 沒有使用型式參數  
  
 使用者定義型別轉換不能使用型式參數。  
  
 下列範例會產生 C2835：  
  
```  
// C2835.cpp  
class A {  
public:  
   char v_char;  
  
   A() {   
      v_char = 'A';   
   };  
   operator char(char a) {   // C2835  
   // try the following line instead  
   // operator char() {     
      return v_char + 1;   
   };  
};  
  
int main() {  
   A a;  
}  
```
