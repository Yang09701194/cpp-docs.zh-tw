# 編譯器錯誤 C2839
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無效的傳回型別 'type' 針對多載的 'operator \-\>'  
  
 `->` 運算子必須傳回類別、結構或等位，或是對其中之一的參考。
