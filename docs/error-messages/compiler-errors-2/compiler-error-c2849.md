# <a name="compiler-error-c2849"></a>編譯器錯誤 C2849
'解構函式': 介面不可以有解構函式  
  
 Visual c + +[介面](../../cpp/interface.md)不能有解構函式。  
  
 下列範例會產生 C2849:  
  
```  
// C2849.cpp  
// compile with: /c  
__interface C {  
   ~C();   // C2849 destructor not allowed in an interface  
};  
```
