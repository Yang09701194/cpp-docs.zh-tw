# 編譯器錯誤 C2855
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

命令列選項 'option' 和先行編譯標頭檔不一致  
  
 給定的命令列選項與用來建立先行編譯標頭 \(Precompiled Header\) 的選項不同。  使用給定的命令列選項重新產生先行編譯標頭。
