# <a name="compiler-error-c2855"></a>編譯器錯誤 C2855

命令列選項 ' option ' 與先行編譯標頭檔不一致

指定的命令列選項與用來建立先行編譯標頭檔的選項不同。 使用指定的命令列選項重新產生先行編譯標頭檔。
