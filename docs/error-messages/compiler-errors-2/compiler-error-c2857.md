# <a name="compiler-error-c2857"></a>編譯器錯誤 C2857
'#include' 的原始程式檔中找不到與 /Ycfilename 命令列選項指定的陳述式  
  
 [/Yc](../../build/reference/yc-create-precompiled-header-file.md)選項指定未包含所要編譯原始程式檔中的包含檔案名稱。  
  
 這個錯誤可能因`#include`未編譯的條件式編譯區塊中的陳述式。
