# <a name="compiler-error-c2858"></a>編譯器錯誤 C2858
命令列選項 '/Yc (/ Fdfilename)' 與使用的先行編譯標頭不一致' / Fdfilename'  
  
 使用先行編譯標頭所指定的程式資料庫 ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) 不是先前建立先行編譯標頭指定的選項。 ([/Yc](../../build/reference/yc-create-precompiled-header-file.md)) 選項。
