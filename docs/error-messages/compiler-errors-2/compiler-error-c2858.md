# <a name="compiler-error-c2858"></a>編譯器錯誤 C2858

命令列選項 '/Yc （/Fdfilename） ' 與使用 '/Fdfilename ' 的先行編譯標頭檔不一致

[使用先行編譯標頭檔（[/yu](../../build/reference/yu-use-precompiled-header-file.md)）] 選項所指定的程式資料庫，不是先前的 [建立先行編譯標頭檔（[/yc](../../build/reference/yc-create-precompiled-header-file.md)）] 選項所指定的。
