# 編譯器錯誤 C2858
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

命令列選項 '\/Yc \(\/Fdfilename\)' 和先行編譯標頭檔不一致，它使用 '\/Fdfilename'  
  
 以使用先行編譯標頭 \([\/Yu](../../build/reference/yu-use-precompiled-header-file.md)\) 選項所指定的程式資料庫，和先前用建立先行編譯標頭 \([\/Yc](../../build/reference/yc-create-precompiled-header-file.md)\) 選項所指定的不同。
