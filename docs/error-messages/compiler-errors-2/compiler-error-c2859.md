# <a name="compiler-error-c2859"></a>編譯器錯誤 C2859

filename 不是建立此先行編譯標頭檔時所使用的類型檔案，請重新建立先行編譯的標頭。

專案資料庫和先行編譯標頭檔必須一起建立，以確保一致的資訊。 重建專案以重新建立先行編譯標頭檔。

如需先行編譯標頭檔的詳細資訊，請參閱[/y （先行編譯標頭檔）](../../build/reference/y-precompiled-headers.md)。
