# <a name="compiler-error-c2859"></a>編譯器錯誤 C2859

檔案名稱不是已建立此先行編譯標頭時所使用的型別檔案，請重新建立先行編譯標頭。

先行編譯標頭檔案與專案資料庫必須一起建立，以確保一致的資訊。 重建專案以重新建立先行編譯標頭。

如需先行編譯標頭資訊，請參閱[/Y （先行編譯標頭）](../../build/reference/y-precompiled-headers.md)。
