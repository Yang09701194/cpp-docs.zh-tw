# 編譯器錯誤 C2859
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

filename 不是建立此先行編譯標頭檔時使用的 type 檔案，請重新建立先行編譯標頭檔。  
  
 專案資料庫和先行編譯標頭檔必須一起建立，以確保資訊的一致性。  重建專案，以重新建立先行編譯標頭。  
  
 如需先行編譯標頭的詳細資訊，請參閱 [\/Y \(先行編譯標頭\)](../../build/reference/y-precompiled-headers.md)。
