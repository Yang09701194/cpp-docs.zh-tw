# <a name="compiler-error-c2867"></a>編譯器錯誤 C2867
'identifier': 不是命名空間  
  
 A`using`指示詞會套用至命名空間以外的項目。  
  
 下列範例會產生 C2867:  
  
```  
// C2867.cpp  
// compile with: /c  
namespace N {  
   class X {};  
}  
using namespace N::X;   // C2867  
```
