# <a name="compiler-error-c2873"></a>編譯器錯誤 C2873
'symbol': 無法在使用宣告中使用符號  
  
 A`using`指示詞缺少[命名空間](../../cpp/namespaces-cpp.md)關鍵字。 這會造成編譯器錯誤解譯為程式碼[using 宣告](../../cpp/using-declaration.md)而[using 指示詞](../../cpp/namespaces-cpp.md#using_directives)。
