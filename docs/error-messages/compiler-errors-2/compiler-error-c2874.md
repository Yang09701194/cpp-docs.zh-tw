# 編譯器錯誤 C2874
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

using 宣告造成 'symbol' 的多重宣告  
  
 此宣告使得同一個項目被定義兩次。  
  
 下列範例會產生 C2874：  
  
```  
// C2874.cpp  
namespace Z {  
   int i;  
}  
  
int main() {  
   int i;  
   using Z::i;   // C2874, i already declared  
}  
```
