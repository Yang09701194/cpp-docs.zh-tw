# <a name="compiler-error-c2874"></a>編譯器錯誤 C2874

using 宣告造成 'symbol' 的多重宣告

宣告會導致相同的項目會定義兩次。

下列範例會產生 C2874:

```
// C2874.cpp
namespace Z {
   int i;
}

int main() {
   int i;
   using Z::i;   // C2874, i already declared
}
```
