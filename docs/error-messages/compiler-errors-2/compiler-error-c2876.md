# <a name="compiler-error-c2876"></a>編譯器錯誤 C2876
'class::symbol': 不是所有的多載是否可存取  
  
 所有的多載的形式的基底類別中的函式必須是在衍生類別存取。  
  
 下列範例會產生 C2876:  
  
```  
// C2876.cpp  
// compile with: /c  
class A {  
public:     
   double a(double);  
private:  
   int a(int);  
};  
  
class B : public A {  
   using A::a;   // C2876 one overload is private in base class  
};  
```
