# <a name="compiler-error-c2888"></a>編譯器錯誤 C2888
'identifier': 無法在命名空間 'namespace' 中定義符號  
  
 屬於命名空間的符號必須定義在命名空間中含括 a。  
  
 下列範例會產生 C2888:  
  
```  
// C2888.cpp  
// compile with: /c  
namespace M {  
   namespace N {  
      void f1();  
      void f2();  
   }  
  
   void N::f1() {}   // OK: namspace M encloses N  
}  
  
namespace O {  
   void M::N::f2() {}   // C2888 namespace O does not enclose M  
}  
```
