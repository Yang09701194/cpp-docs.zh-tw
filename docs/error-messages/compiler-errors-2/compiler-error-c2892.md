# <a name="compiler-error-c2892"></a>編譯器錯誤 C2892
區域類別不應該有成員樣板  
  
 樣板化的成員函式不正確定義的函式中的類別中。  
  
 下列範例會產生 C2892:  
  
```  
// C2892.cpp  
int main() {  
   struct local {  
      template<class T>   // C2892  
      void f() {}  
   };  
}  
```
