# <a name="compiler-error-c2898"></a>編譯器錯誤 C2898

'declaration': 成員函式樣板不可為虛擬

下列範例會產生 C2898:

```
// C2898.cpp
// compile with: /c
class X {
public:
   template<typename T> virtual void f(T t) {}   // C2898
};
```
