# 編譯器錯誤 C2906
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'specialization' : 明確特製化必須有 'template \<\>'  
  
 您必須對樣板的明確特製化使用新語法。  
  
 下列範例會產生 C2906：  
  
```  
// C2906.cpp  
// compile with: /c  
template<class T> class X{};   // primary template  
class X<int> { }   // C2906  
template<> class X<int> { };   // new syntax  
```
