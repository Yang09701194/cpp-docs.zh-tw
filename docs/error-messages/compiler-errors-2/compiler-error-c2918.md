# 編譯器錯誤 C2918
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'name'：索引的屬性不能用在 WinRT 類型的已發行介面上  
  
 索引的屬性在 WinRT 類型的已發行介面上不受支援。
