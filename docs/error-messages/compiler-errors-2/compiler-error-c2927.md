# 編譯器錯誤 C2927
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function'：呼叫函式樣板必須至少有一個引數  
  
 您不能呼叫沒有引數的樣板函式。 樣板引數的類型可決定要產生之函式的版本。
