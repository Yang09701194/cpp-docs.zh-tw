# <a name="compiler-error-c2928"></a>編譯器錯誤 C2928
明確具現化; 'identifier' 不是函式或樣板類別 (Template-Class) 的靜態資料成員  
  
 您無法明確具現化不是函式或 `class` 變數之 `static` 的成員。
