# 編譯器錯誤 C2928
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

明確具現化; 'identifier' 不是函式或樣板類別 \(Template\-Class\) 的靜態資料成員  
  
 您無法明確具現化不是函式或 `static` 變數之 `class` 的成員。
