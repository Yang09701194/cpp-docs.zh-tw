# <a name="compiler-error-c2932"></a>編譯器錯誤 C2932
'class': type-class-id 重複定義為 'identifier' 的資料成員  
  
 您無法使用泛型或樣板類別作為資料成員。  
  
 下列範例會產生 C2932：  
  
```  
// C2932.cpp  
// compile with: /c  
template<class T>   
struct TC {};   
  
struct MyStruct {  
   int TC<int>;   // C2932  
   int TC;   // OK  
};  
```  
  
 使用泛型時，也會發生 C2932：  
  
```  
// C2932b.cpp  
// compile with: /clr /c  
generic<class T>  
ref struct GC {};  
  
struct MyStruct {  
   int GC<int>;   // C2932  
   int GC;   // OK  
};  
```
