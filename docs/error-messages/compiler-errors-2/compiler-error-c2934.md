# 編譯器錯誤 C2934
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class': type\-class\-id 已經重複定義為 'identifier' 的巢狀 'item'  
  
 您無法使用泛型或樣板類別作為巢狀項目。
