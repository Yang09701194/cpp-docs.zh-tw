# 編譯器錯誤 C2945
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

明確具現化 \(Explicit Instantiation\) 沒有參考至樣板類別 \(Template\-Class\) 特製化  
  
 您無法明確具現化不是樣板的某個項目。
