# <a name="compiler-error-c2947"></a>編譯器錯誤 C2947
必須以 '>' 結束 construct，但卻找到 'syntax'  
  
 使用泛型或樣板引數清單可能不正確地終止。  
  
 C2947 也可能會產生語法錯誤。  
  
 下列範例會產生 C2947:  
  
```  
// C2947.cpp  
// compile with: /c  
template <typename T>=   // C2947  
// try the following line instead  
// template <typename T>  
struct A {};  
```
