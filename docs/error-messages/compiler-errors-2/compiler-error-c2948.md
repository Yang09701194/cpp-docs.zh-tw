# 編譯器錯誤 C2948
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

明確具現化 \(Explicit Instantiation\)；在特製化上不允許儲存類別規範 'specifier'  
  
 您不能在樣板類別 \(先前已經明確具現化\) 的特製化中使用儲存類別規範 \(例如 `extern`\)。
