# <a name="compiler-error-c2948"></a>編譯器錯誤 C2948
明確具現化;儲存類別規範 'specifier' 特製化上不允許  
  
 您不能使用的儲存類別規範 (例如`extern`) 中的先前已明確具現化的樣板類別特製化。
