# <a name="compiler-error-c2953"></a>編譯器錯誤 C2953
'identifier': 類別樣板已經定義過了  
  
 請檢查原始程式碼檔並包含其他定義的檔案。  
  
 下列範例會產生 C2953：  
  
```  
// C2953.cpp  
// compile with: /c  
template <class T>  class A {};  
template <class T>  class A {};   // C2953  
template <class T>  class B {};   // OK  
```
