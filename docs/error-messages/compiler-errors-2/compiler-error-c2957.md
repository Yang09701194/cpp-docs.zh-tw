# <a name="compiler-error-c2957"></a>編譯器錯誤 C2957
'delim': 左分隔符號無效: 必須是 '<'  
  
 泛型類別的格式錯誤。  
  
 下列範例會產生 C2957：  
  
```  
// C2957.cpp  
// compile with: /clr /LD  
generic << class T>   // C2957  
// try the following line instead  
// generic < class T>  
gc class C {};  
```
