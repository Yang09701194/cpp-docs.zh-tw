# <a name="compiler-error-c2958"></a>編譯器錯誤 C2958
左的分隔符號，請參閱 'location' 沒有正確對應  
  
 分隔符號不完全相符。  
  
 這個錯誤可能被因函式樣板宣告中不相符的括號。
