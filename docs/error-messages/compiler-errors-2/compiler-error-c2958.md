# <a name="compiler-error-c2958"></a>編譯器錯誤 C2958

在 ' location ' 找到的左邊分隔符號未正確地相符

分隔符號的對應不正確。

此錯誤可能是因為函式樣板宣告中的括弧不相符所造成。
