# 編譯器錯誤 C2958
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

找到沒有正確對應的左 delimiter \(於 'location'\)  
  
 分隔符號 \(Delimiter\) 不對稱。  
  
 這項錯誤可能是因為函式樣板宣告中不對稱的括號所造成。
