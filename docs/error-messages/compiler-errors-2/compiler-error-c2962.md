# <a name="compiler-error-c2962"></a>編譯器錯誤 C2962
語法錯誤: 'token': 必須是以 '}' 結尾的樣板類別 成員函式定義  
  
 語彙基元會導致樣板宣告中的語法錯誤。  
  
 這個錯誤可能是不對稱的分隔符號所造成
