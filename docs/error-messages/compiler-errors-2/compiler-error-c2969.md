# <a name="compiler-error-c2969"></a>編譯器錯誤 C2969
語法錯誤：'symbol' : 必須是以 '}' 結尾的成員函式定義  
  
 樣板成員函式定義有不對稱的右大括弧。  
  
 下列範例會產生 C2969：  
  
```  
// C2969.cpp  
// compile with: /c  
class A {  
   int i;  
public:  
   A(int i) {}  
};  
  
A anA(1);  
  
class B {  
   A a;  
   B() : a(anA);   // C2969  
   // try the following line instead  
   // B() : a(anA) {}  
};  
```
