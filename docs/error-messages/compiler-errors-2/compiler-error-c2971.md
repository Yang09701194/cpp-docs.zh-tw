# <a name="compiler-error-c2971"></a>編譯器錯誤 C2971

'class': 樣板參數 'param': 'arg': 區域變數不能當做非類型引數

您無法使用的名稱或區域變數的位址，做為範本引數。

下列範例會產生 C2971:

```
// C2971.cpp
template <int *pi>
class Y {};

int global_var = 0;

int main() {
   int local_var = 0;
   Y<&local_var> aY;   // C2971
   // try the following line instead
   // Y<&global_var> aY;
}
```
