# <a name="compiler-error-c2977"></a>編譯器錯誤 C2977

'identifier' : 類型引數太多

泛型或樣板的實質引數太多。 請檢查泛型或樣板宣告，以找出正確的參數數目。

下列範例會產生 C2977：

```
// C2977.cpp
// compile with: /c
template<class T, int i>
class MyClass {};

template MyClass< int , 1, 1 >;   // C2977
template MyClass< int , 1 >;   // OK
```

使用泛型時，也會發生 C2977：

```
// C2977b.cpp
// compile with: /clr
// C2977 expected
generic <class T, class U>
void f(){}

generic <class T>
ref struct GC1 {};

int main() {
   // Delete the following 2 lines to resolve.
   GC1<int, char> ^ pgc1;
   f<int,int,int>();

   // OK
   GC1<int> ^ pgc1;
   f<int, int>();
}
```
