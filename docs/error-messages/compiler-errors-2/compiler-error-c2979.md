# <a name="compiler-error-c2979"></a>編譯器錯誤 C2979

不支援在泛型中進行明確特製化

泛型類別宣告不正確。  如需詳細資訊，請參閱[泛型](../../extensions/generics-cpp-component-extensions.md)。

## <a name="example"></a>範例

下列範例會產生 C2979：

```cpp
// C2979.cpp
// compile with: /clr /c
generic <>
ref class Utils {};   // C2979 error

generic <class T>
ref class Utils2 {};   // OK
```
