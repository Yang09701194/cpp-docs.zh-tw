# <a name="compiler-error-c2995"></a>編譯器錯誤 C2995
'function': 函式樣板已經定義過了  
  
 請確定樣板類別的每個成員函式都只有一個定義。  
  
 下列範例會產生 C2995：  
  
```  
// C2995.cpp  
// compile with: /c  
template <class T>  
void Test(T x){}  
  
template <class T> void Test(T x){}   // C2995  
template <class T> void Test2(T x){}   // OK  
```
