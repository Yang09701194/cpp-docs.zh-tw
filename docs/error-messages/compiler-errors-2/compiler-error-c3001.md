# <a name="compiler-error-c3001"></a>編譯器錯誤 C3001
'error_text': 需要 OpenMP 指示詞名稱  
  
 `omp` pragma 之後必須是指示詞。  
  
 下列範例會產生 C3001：  
  
```  
// C3001.c  
// compile with: /openmp  
int main()  
{  
   #pragma omp   // C3001 missing token  
}  
```
