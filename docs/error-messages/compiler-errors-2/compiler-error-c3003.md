# <a name="compiler-error-c3003"></a>編譯器錯誤 C3003

'directive': OpenMP 指示詞名稱不能在指示詞子句後面

OpenMP 指示詞名稱不能在 OpenMP 指示詞子句後面。

下列範例會產生 C3003：

```
// C3003.c
// compile with: /openmp
int main()
{
   int x, y, z;
   #pragma omp parallel shared(x, y, z) for   // C3003
}
```
