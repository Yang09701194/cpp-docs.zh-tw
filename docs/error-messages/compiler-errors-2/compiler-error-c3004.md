# <a name="compiler-error-c3004"></a>編譯器錯誤 C3004
'clause' : 子句對 OpenMP 'directive' 指示詞無效  
  
 OpenMP 子句用在未啟用的指示詞上。  
  
 下列範例會產生 C3004：  
  
```  
// C3004.c  
// compile with: /openmp  
int main()  
{  
   int x, y, z;  
  
   // Shared clause not allowed for 'single' directive.  
   #pragma omp single shared(x, y)   // C3004  
  
   x = y;  
}  
```
