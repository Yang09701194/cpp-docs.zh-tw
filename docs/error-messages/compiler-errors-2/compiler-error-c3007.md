# <a name="compiler-error-c3007"></a>編譯器錯誤 C3007
'arg' : OpenMP 'directive' 指示詞上的子句不接受引數  
  
 OpenMP 指示詞具有引數，但指示詞不接受引數。  
  
 下列範例會產生 C3007：  
  
```  
// C3007.c  
// compile with: /openmp  
int main()  
{  
   #pragma omp parallel for ordered(2)   // C3007  
}  
```
