# <a name="compiler-error-c3008"></a>編譯器錯誤 C3008
'arg': OpenMP 'directive' 指示詞上的引數遺漏右 ')'  
  
 使用引數的 OpenMP 指示詞沒有右括號。  
  
 下列範例會產生 C3008：  
  
```  
// C3008.c  
// compile with: /openmp  
int main()  
{  
   int x, y, z;  
   #pragma omp parallel shared(x   // C3008  
   // Try the following line instead:  
   #pragma omp parallel shared(x)  
   {  
   }  
}  
```
