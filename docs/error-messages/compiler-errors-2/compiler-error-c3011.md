# <a name="compiler-error-c3011"></a>編譯器錯誤 C3011

內嵌組譯碼不允許直接放在平行區域內

`omp` 平行區域不能包含內嵌組譯碼指令。

下列範例會產生 C3011：

```
// C3011.cpp
// compile with: /openmp
// processor: /x86
int main() {
   int   n = 0;

   #pragma omp parallel
   {
      _asm mov eax, n   // Delete this line to resolve this error.
   }   // C3011
}
```
