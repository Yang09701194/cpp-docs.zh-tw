# <a name="compiler-error-c3015"></a>編譯器錯誤 C3015

OpenMP 'for' 陳述式中的初始設定格式不當

OpenMP 陳述式中的 `for` 迴圈必須完整且明確地指定。

下列範例會產生 C3015：

```
// C3015.cpp
// compile with: /openmp
int main()
{
   int i = 0, j = 10;

   #pragma omp parallel
   {
      #pragma omp for
      for (; i < 0; i += j)   // C3015
      // Try the following line instead:
      // for (i = 0; i < 0; i++)
         --j;
   }
}
```
