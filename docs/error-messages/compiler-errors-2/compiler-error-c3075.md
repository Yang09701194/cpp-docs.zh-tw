# <a name="compiler-error-c3075"></a>編譯器錯誤 C3075
'instance' : 不可將參考類型的執行個體 'type' 嵌入實值類型中  
  
 實值類型不能包含參考類型的執行個體。  
  
 如需詳細資訊，請參閱[參考類型的 c + + 堆疊語意](../../dotnet/cpp-stack-semantics-for-reference-types.md)。  
  
## <a name="example"></a>範例  
 下列範例會產生 C3075。  
  
```  
// C3075.cpp  
// compile with: /clr /c  
ref struct U {};  
value struct X {  
   U y;   // C3075  
};  
  
ref struct Y {  
   U y;   // OK  
};  
```
