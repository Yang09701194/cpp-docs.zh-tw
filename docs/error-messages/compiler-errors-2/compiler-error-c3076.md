# <a name="compiler-error-c3076"></a>編譯器錯誤 C3076

'instance': 您無法將執行個體的參考類型 'type'，內嵌在原生類型

原生類型不能包含 CLR 型別的執行個體。

如需詳細資訊，請參閱 < [ C++的參考型別堆疊語意](../../dotnet/cpp-stack-semantics-for-reference-types.md)。

## <a name="example"></a>範例

下列範例會產生 C3076。

```
// C3076.cpp
// compile with: /clr /c
ref struct U {};

struct V {
   U y;   // C3076
};

ref struct W {
   U y;   // OK
};
```
