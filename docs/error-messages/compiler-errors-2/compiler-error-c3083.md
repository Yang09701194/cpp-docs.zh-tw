# <a name="compiler-error-c3083"></a>編譯器錯誤 C3083

'function': 左邊的符號 ':: ' 必須是類型

未正確地呼叫函式。

## <a name="example"></a>範例

下列範例會產生 C3083。

```
// C3083.cpp
// compile with: /c
struct N {
   ~N();
};

struct N1 {
   ~N1();
};

N::N::~N() {}   // C3083
N1::~N1() {}   // OK
```
