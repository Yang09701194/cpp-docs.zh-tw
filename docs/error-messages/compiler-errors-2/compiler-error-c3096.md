# <a name="compiler-error-c3096"></a>編譯器錯誤 C3096

'attribute': 屬性只能用於屬性類別的資料成員上

已不正確地套用屬性。

如需詳細資訊，請參閱 [User-Defined Attributes](../../windows/user-defined-attributes-cpp-component-extensions.md)。
