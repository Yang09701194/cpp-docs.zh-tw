# <a name="compiler-error-c3126"></a>編譯器錯誤 C3126

無法定義等位內受管理的類型 'type' 的 ' union'

等位不能定義於 managed 類型。

下列範例會產生 C3126:

```
// C3126_2.cpp
// compile with: /clr /c
ref class Test
{
   union x
   {   // C3126
      int a;
      int b;
   };
};
```
