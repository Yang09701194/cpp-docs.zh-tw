# 編譯器錯誤 C3126
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不可以在 Managed 型別 'type' 中定義等位 'union'  
  
 等位不可在 Managed 型別之內定義。  
  
 下列範例會產生 C3126：  
  
```  
// C3126_2.cpp  
// compile with: /clr /c  
ref class Test  
{  
   union x  
   {   // C3126  
      int a;  
      int b;  
   };  
};  
```  
  
 下列範例會產生 C3126：  
  
```  
// C3126.cpp  
// compile with: /clr:oldSyntax /c  
#using <mscorlib.dll>  
  
__gc class Test  
{  
   union x  
   {   // C3126  
      int a;  
      int b;  
   };  
};  
```
