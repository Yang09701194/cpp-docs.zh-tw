# <a name="compiler-error-c3126"></a>編譯器錯誤 C3126
無法定義等位內 managed 型別 'type' 的 ' 等位'  
  
 等位不能定義內的 managed 型別。  
  
 下列範例會產生 C3126:  
  
```  
// C3126_2.cpp  
// compile with: /clr /c  
ref class Test  
{  
   union x  
   {   // C3126  
      int a;  
      int b;  
   };  
};  
```
