# <a name="compiler-error-c3130"></a>編譯器錯誤 C3130

編譯器內部錯誤：無法將插入的程式碼區塊寫入 PDB

如果編譯器無法將插入的程式碼區塊寫入 .pdb 檔案，就會發生這個錯誤。 失敗的最常見原因是磁碟空間不足。
