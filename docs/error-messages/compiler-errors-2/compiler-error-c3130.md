# 編譯器錯誤 C3130
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

編譯器內部錯誤：無法寫入插入程式碼區塊至 PDB  
  
 如果編譯器無法將插入的程式碼區塊寫入 .pdb 檔，就會發生此錯誤。  發生此錯誤最常見的原因是磁碟空間不足。
