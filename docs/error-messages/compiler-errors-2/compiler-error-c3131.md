# <a name="compiler-error-c3131"></a>編譯器錯誤 C3131
專案必須要有 'name' 屬性為 'module' 屬性  
  
 [模組](../../windows/module-cpp.md)屬性必須具有 name 參數。  
  
 下列範例會產生 C3131:  
  
```  
// C3131.cpp  
[emitidl];  
[module];   // C3131  
// try the following line instead  
// [module (name="MyLib")];  
  
[public]  
typedef long int LongInt;  
```
