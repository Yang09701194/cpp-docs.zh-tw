# <a name="compiler-error-c3135"></a>編譯器錯誤 C3135

' property '：屬性不能有 ' const ' 或 ' volatile ' 類型

屬性上不允許[const](../../cpp/const-cpp.md)和[volatile](../../cpp/volatile-cpp.md)關鍵字。
