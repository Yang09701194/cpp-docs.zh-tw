# 編譯器錯誤 C3135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'property' : 屬性不可以有 'const' 或 'volatile' 型別  
  
 不允許在屬性上使用 [const](../../cpp/const-cpp.md) 和 [volatile](../../cpp/volatile-cpp.md) 關鍵字。
