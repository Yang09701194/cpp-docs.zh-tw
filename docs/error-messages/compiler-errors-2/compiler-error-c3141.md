# <a name="compiler-error-c3141"></a>編譯器錯誤 C3141

'interface_name': 介面只支援公用繼承

介面定義[介面 （或 __interface）](../../cpp/interface.md)關鍵字僅支援公用繼承。

下列範例會產生 C3141:

```
// C3141.cpp
__interface IBase {};
__interface IDerived1 : protected IBase {};  // C3141
__interface IDerived2 : private IBase {};    // C3141
```
