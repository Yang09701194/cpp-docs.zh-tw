# <a name="compiler-error-c3142"></a>編譯器錯誤 C3142
'property_name': 您無法取得屬性的位址  
  
 屬性的位址不適用於開發人員。  
  
 下列範例會產生 C3142:  
  
```  
// C3142_2.cpp  
// compile with: /clr  
using namespace System;  
ref class CSize {  
private:  
   property int Size {  
      int get();  
   }  
};  
  
int main() {  
    &CSize::Size; // C3142  
}  
```
