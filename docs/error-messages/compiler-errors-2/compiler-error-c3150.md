# <a name="compiler-error-c3150"></a>編譯器錯誤 C3150
'element': 'attribute' 只能套用至類別、 介面、 陣列或指標  
  
`__gc` 只有類別、 介面或陣列上使用。  
  
 C3150 才可使用過時的編譯器選項 **/clr:oldSyntax**。
