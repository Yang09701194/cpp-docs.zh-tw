# <a name="compiler-error-c3157"></a>編譯器錯誤 C3157
ParamArray 屬性只能套用至最後一個參數  
  
 [ParamArray](https://msdn.microsoft.com/en-us/library/system.paramarrayattribute.aspx)屬性已套用至錯誤的參數。
