# <a name="compiler-error-c3159"></a>編譯器錯誤 C3159
「 指標 」: 不可以宣告陣列的指標實值型別  
  
 不可以宣告陣列的指標實值型別。  
  
 C3159 才可使用過時的編譯器選項**/clr:oldSyntax**。
