# <a name="compiler-error-c3166"></a>編譯器錯誤 C3166
'pointer': 無法將內部 __gc 指標的指標宣告為 'type' 的成員  
  
編譯器發現無效的指標宣告 (`__nogc`指標`__gc`指標。)。 
  
C3166 才可使用過時的編譯器選項**/clr:oldSyntax**。
