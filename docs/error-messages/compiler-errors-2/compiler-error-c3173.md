# <a name="compiler-error-c3173"></a>編譯器錯誤 C3173

idl 合併中的版本不相符

當物件檔案包含使用舊版編譯器所產生的內嵌 idl 時，就會發生這個錯誤。 編譯器會編碼版本號碼，以確保用來產生 .obj 檔案中內嵌之 idl 內容的相同編譯器也是用來合併內嵌 idl 的相同編譯器。

更新您的C++視覺效果安裝，讓所有工具都來自最新發行的版本。
