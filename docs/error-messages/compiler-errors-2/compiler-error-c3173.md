# <a name="compiler-error-c3173"></a>編譯器錯誤 C3173
在 idl 合併中版本不符  
  
 當物件檔案包含與先前版本的編譯器產生的內嵌的 idl 時，就會發生此錯誤。 編譯器會將編碼的版本號碼，以確保用來產生.obj 檔中內嵌 idl 內容的相同編譯器也是用來合併內嵌的 idl 的相同編譯器。  
  
 更新您的 Visual c + + 安裝，使所有工具都都從最新版本。
