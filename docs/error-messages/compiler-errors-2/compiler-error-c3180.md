# <a name="compiler-error-c3180"></a>編譯器錯誤 C3180

' type name '：名稱超過中繼資料限制 ' limit ' 個字元

編譯器已截斷中繼資料中 managed 類型的名稱。 截斷會使型別無法與 `#using` 指示詞一起使用（或另一種語言的對等用法）。

類型名稱限制包括任何限定的命名空間。
