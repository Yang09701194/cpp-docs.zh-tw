# <a name="compiler-error-c3180"></a>編譯器錯誤 C3180
'type name': 名稱超過中繼資料 'limit' 個字元的限制  
  
 編譯器會截斷中繼資料中的 managed 類型的名稱。 截斷會造成型別無法與`#using`指示詞 （或另一種語言對等項目）。  
  
 類型名稱限制包含任何命名空間限定。
