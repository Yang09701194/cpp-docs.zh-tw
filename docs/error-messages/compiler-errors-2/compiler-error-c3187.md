# 編譯器錯誤 C3187
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\_\_func\_\_' : 只能用於函式主體中  
  
 無法在函式的主體之外使用預先定義的巨集。  
  
 若要更正這個錯誤，請將巨集移到函式的主體內。
