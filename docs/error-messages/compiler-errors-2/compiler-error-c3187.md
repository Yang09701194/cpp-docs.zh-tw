# <a name="compiler-error-c3187"></a>編譯器錯誤 C3187

> '&#95;&#95;&#95;func&#95;'：只能在函式的主體中使用

無法在函式的主體之外使用預先定義的巨集。

若要更正這個錯誤，請將巨集移到函式的主體內。
