# <a name="compiler-error-c3192"></a>編譯器錯誤 C3192
語法錯誤: '^' 不是前置運算子 (您是否是指' *'？)  
  
 控制代碼不當做取值運算子。  
  
 下列範例會產生 C3192:  
  
```  
// C3192.cpp  
// compile with: /clr  
using namespace System;  
  
ref class MyClass {  
public:  
   MyClass () {}  
   MyClass(MyClass%) {}  
};  
  
int main() {  
   MyClass ^ s = gcnew MyClass;   
   MyClass b = ^s;   // C3192  
  
   // OK  
   MyClass b2 = *s;  
}  
```
