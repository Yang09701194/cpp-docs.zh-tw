# <a name="compiler-error-c3196"></a>編譯器錯誤 C3196
'keyword': 使用超過一次  
  
 一次使用的關鍵字。  
  
 下列範例會產生 C3196:  
  
```  
// C3196.cpp  
// compile with: /clr  
ref struct R abstract abstract {};   // C3196  
ref struct S abstract {};   // OK  
```
