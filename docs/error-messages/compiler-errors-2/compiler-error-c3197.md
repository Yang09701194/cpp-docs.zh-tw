# <a name="compiler-error-c3197"></a>編譯器錯誤 C3197

'keyword': 只能用於定義

關鍵字用在宣告中，但只適用於在定義中。

下列範例會產生 C3197:

```
// C3197.cpp
// compile with: /clr /c
ref struct R abstract;   // C3197
ref struct R abstract {};   // OK

public ref class MyObject;   // C3197
ref class MyObject;   // OK
public ref class MyObject {};   // OK
```
