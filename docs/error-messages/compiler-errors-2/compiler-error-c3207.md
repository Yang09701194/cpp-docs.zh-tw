# <a name="compiler-error-c3207"></a>編譯器錯誤 C3207
'function': 'arg' 的無效樣板引數，必須是類別樣板  
  
 樣板函式定義為使用樣板類樣板引數。 不過，已傳遞樣板類型引數。  
  
 下列範例會產生 C3207：  
  
```  
// C3207.cpp  
template <template <class T> class TT>  
void f(){}  
  
template <class T>  
struct S  
{  
};  
  
void f1()  
{  
   f<S<int> >();   // C3207  
   // try the following line instead  
   // f<S>();  
}  
```
