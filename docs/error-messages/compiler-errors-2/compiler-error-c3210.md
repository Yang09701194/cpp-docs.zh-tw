# <a name="compiler-error-c3210"></a>編譯器錯誤 C3210
'type': 存取宣告只能套用至基底類別成員  
  
 A [using 宣告](../../cpp/using-declaration.md)指定不正確。  
  
## <a name="example"></a>範例  
 下列範例會產生 C3210。  
  
```  
// C3210.cpp  
// compile with: /c  
struct A {  
protected:  
   int i;  
};  
  
struct B {  
   using A::i;   // C3210  
};  
  
struct C : public A {  
   using A::i;   // OK  
};  
```
