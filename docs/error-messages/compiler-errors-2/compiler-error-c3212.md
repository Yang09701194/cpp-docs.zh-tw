# <a name="compiler-error-c3212"></a>編譯器錯誤 C3212

'specialization' : 樣板成員的明確特製化必須是明確特製化的成員

明確特製化的格式不正確。

下列範例會產生 C3212：

```cpp
// C3212.cpp
// compile with: /LD
template <class T>
struct S {
   template <class T1>
   struct S1;
};

template <class T>   // C3212
template <>
struct S<T>::S1<int> {};

/*
// try the following instead
template <>
template <>
struct S<int>::S1<int> {};
*/

/*
// or, the following
template <>
struct S<int> {
   template <class T1>
   struct S1;
};

template <>
struct S<int>::S1<int> {
};
*/
```
