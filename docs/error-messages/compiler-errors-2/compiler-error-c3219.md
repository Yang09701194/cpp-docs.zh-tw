# <a name="compiler-error-c3219"></a>編譯器錯誤 C3219
'param': 泛型參數不可受到多重非介面的條件約束: 'class'  
  
 不可以透過兩個以上的 Managed 類別來限制泛型參數。  
  
 下列範例會產生 C3219：  
  
```  
// C3219.cpp  
// compile with: /clr  
ref class A {};  
ref class B {};  
  
generic <class T>  
where T : A, B  
ref class E {};   // C3219  
```  
  
 下列範例示範可能的解決方式：  
  
```  
// C3219b.cpp  
// compile with: /clr /c  
ref class A {};  
  
interface struct C {};  
  
generic <class T>  
where T : A  
ref class E {};  
```
