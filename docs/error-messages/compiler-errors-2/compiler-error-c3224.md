# <a name="compiler-error-c3224"></a>編譯器錯誤 C3224
'type': 沒有任何多載泛型類別接受 'number' 個泛型類型引數  
  
 編譯器找不到適當的多載。  
  
 下列範例會產生 C3224：  
  
```  
// C3224.cs  
// compile with: /target:library  
public class C<T> {}  
public class C<T,U> {}  
```  
  
 然後，  
  
```  
// C3224b.cpp  
// compile with: /clr  
#using "C3224.dll"  
int main() {  
   C<int,int,int>^ c = gcnew C<int,int,int>();   // C3224  
   C<int,int>^ c2 = gcnew C<int,int>();   // OK  
}  
```
