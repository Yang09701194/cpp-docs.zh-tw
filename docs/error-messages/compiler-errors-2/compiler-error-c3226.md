# <a name="compiler-error-c3226"></a>編譯器錯誤 C3226
泛型宣告內不允許有樣板宣告，必須改為泛型宣告  
  
 請在泛型類別內使用泛型宣告。  
  
 下列範例會產生 C3226：  
  
```  
// C3226.cpp  
// compile with: /clr  
generic <class T>  
ref class C {  
   template <class T1>   // C3226  
   ref struct S1 {};  
};  
```  
  
 下列範例示範可能的解決方式：  
  
```  
// C3226b.cpp  
// compile with: /clr /c  
generic <class T>  
ref class C {  
   generic <class T1>  
   ref struct S1 {};  
};  
```
