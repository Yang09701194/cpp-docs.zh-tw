# <a name="compiler-error-c3230"></a>編譯器錯誤 C3230

'function' : 'template' 的樣板類型引數不可包含泛型類型參數: 'param'

樣板會在編譯時期具現化，但泛型會在執行階段具現化。 因此，您無法產生可呼叫樣板的泛型程式碼，因為當泛型類型最後確定時，無法於該執行階段具現化樣板。

下列範例會產生 C3230：

```cpp
// C3230.cpp
// compile with: /clr /LD
template <class S>
void f(S t);

generic <class U>
ref class C {
   void f1(U x) {
      f(x);   // C3230
   }
};
```
