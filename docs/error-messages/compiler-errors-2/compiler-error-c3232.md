# <a name="compiler-error-c3232"></a>編譯器錯誤 C3232
'param': 泛型類型參數不可使用在限定名稱中  
  
 泛型類型參數的使用方式錯誤。  
  
 下列範例會產生 C3232：  
  
```  
// C3232.cpp  
// compile with: /clr  
generic <class T>  
ref class C {  
   typename T::TYPE t;   // C3232  
};  
```
