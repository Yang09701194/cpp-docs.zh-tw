# <a name="compiler-error-c3233"></a>編譯器錯誤 C3233
'type': 泛型類型參數已經受到條件約束  
  
 不可以透過多個 `where` 子句來限制泛型參數。  
  
 下列範例會產生 C3233：  
  
```  
// C3233.cpp  
// compile with: /clr /LD  
  
interface struct C {};  
interface struct D {};  
  
generic <class T>  
where T : C  
where T : D  
ref class E {};   // C3233  
```
