# <a name="compiler-error-c3237"></a>編譯器錯誤 C3237
'generic_class'：泛型類別不可為自訂屬性  
  
 泛型類別不可為使用者定義的屬性。  
  
## <a name="example"></a>範例  
 下列範例會產生 C3237：  
  
```  
// C3237.cpp  
// compile with: /clr /c  
// C3237 expected  
using namespace System;  
  
generic <class T>  
// Delete the following line to resolve.  
[attribute(AttributeTargets::All, AllowMultiple=true)]  
public ref class GR {};  
```
