# <a name="compiler-error-c3247"></a>編譯器錯誤 C3247
'class1' : Coclass 無法繼承自其他 Coclass 'class2'  
  
 標記為 [coclass](../../windows/coclass.md) 屬性的類別無法繼承自另一個標記為 `coclass` 屬性的類別。  
  
 下列範例會產生 C3247：  
  
```  
// C3247.cpp  
[module(name="MyLib")];  
[coclass]  
class a {  
};  
  
[coclass]  
class b : public a {   // C3247  
};  
int main() {  
}  
```
