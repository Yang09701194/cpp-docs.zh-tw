# <a name="compiler-error-c3248"></a>編譯器錯誤 C3248
'function1': 宣告為 '__sealed' 的函式不可被 'function2' 覆寫  
  
衍生類別已嘗試覆寫 **__sealed** 虛擬方法。  
  
C3248 才可使用過時的編譯器選項**/clr:oldSyntax**。
