# <a name="compiler-error-c3251"></a>編譯器錯誤 C3251
無法在實值類型執行個體上叫用基底類別方法  
  
 發生下列錯誤，因為 `GetClass` 是 `Microsoft.Runtime.Object`的成員，而不是 `Microsoft.Runtime.Integer4`的成員。
