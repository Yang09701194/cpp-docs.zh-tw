# <a name="compiler-error-c3264"></a>編譯器錯誤 C3264
'class': 類別建構函式不可以有傳回類型  
  
類別建構函式不可以有傳回類型。  
  
下列範例會產生 C3264：  
  
```  
// C3264_2.cpp  
// compile with: /clr  
  
ref class X {  
   public:  
      static int X()   { // C3264  
      }  
  
      /* use the code below to resolve the error  
      static X() {  
      }  
      */  
};  
int main() {  
}  
```
