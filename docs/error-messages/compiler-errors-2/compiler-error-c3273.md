# <a name="compiler-error-c3273"></a>編譯器錯誤 C3273

__finally 不能用在 Unmanaged 程式碼的例外狀況區塊上。

下列範例會產生 C3273：

```
// C3273.cpp
// compile with: /GX
int main()
{
   try
   {
   }
   catch (int)
   {
   }
   __finally   // C3273, remove __finally clause
   {
   }
}
```
