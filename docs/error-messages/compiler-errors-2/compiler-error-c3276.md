# <a name="compiler-error-c3276"></a>編譯器錯誤 C3276

' 關鍵字 '：跳出 __finally/finally 組塊在終止處理期間有未定義的行為

此錯誤與[C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md)警告相同。 不過，當您使用/clr 時，無法使用[warning](../../preprocessor/warning.md) pragma 來停用此條件。
