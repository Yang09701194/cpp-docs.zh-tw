# <a name="compiler-error-c3276"></a>編譯器錯誤 C3276

'keyword': 跳出 __finally/finally 區塊未定義在終止處理期間的行為

此錯誤是相同[C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md)警告。 不過，當您使用 /clr，這種狀況無法停用與[警告](../../preprocessor/warning.md)pragma。
