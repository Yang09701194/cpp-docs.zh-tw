# 編譯器錯誤 C3276
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'keyword' : 跳出 \_\_finally\/finally 區塊在終止處理期間有未定義的行為  
  
 此錯誤與 [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md) 警告相同。  然而，當您使用 \/clr 時，無法用 [warning](../../preprocessor/warning.md) Pragma 避免此狀況發生。
