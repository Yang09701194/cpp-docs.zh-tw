# <a name="compiler-error-c3277"></a>編譯器錯誤 C3277
無法在受管理的 'type' 內部定義 unmanaged 的列舉 'enum'  
  
 列舉型別內的 managed 型別定義不正確。  
  
 下列範例會產生 C3277:  
  
```  
// C3277a.cpp  
// compile with: /clr  
ref class A  
{  
   enum E {e1,e2};   // C3277  
   // try the following line instead  
   // enum class E {e1,e2};  
};  
  
int main()  
{  
}  
```
