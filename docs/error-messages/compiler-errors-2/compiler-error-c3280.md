# <a name="compiler-error-c3280"></a>編譯器錯誤 C3280
'class' : 無法將 Managed 類型的成員函式編譯成 Unmanaged 函式  
  
 無法將 Managed 類別成員函式編譯成 Unmanaged 函式。  
  
 下列範例會產生 C3280：  
  
```  
// C3280_2.cpp  
// compile with: /clr  
ref struct A {  
   void func();  
};  
  
#pragma managed(push,off)  
  
void A::func()   // C3280  
{  
}  
  
#pragma managed(pop)  
```
