# <a name="compiler-error-c3322"></a>編譯器錯誤 C3322
'property': 不是屬性 (attribute) 'attribute' 的有效屬性 (property)  
  
 無效或無法辨識的屬性 (property) 或參數已傳遞給屬性 (attribute)。 請參閱屬性的文件。
