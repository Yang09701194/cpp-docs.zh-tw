# <a name="compiler-error-c3333"></a>編譯器錯誤 C3333
'type library': #import 陳述式中的類型程式庫損毀，無法執行作業  
  
 在 `#import` 陳述式中指定的類型程式庫無法由編譯器讀取。 您可能想要重新產生類型程式庫，或是可能的話，從您的提供者要求新的類型程式庫。 您可能想要使用 Visual C++ 中提供的 OLE 檢視器，來檢視類型程式庫檔案，看看它有什麼問題。
