# <a name="compiler-error-c3334"></a>編譯器錯誤 C3334

#import 陳述式中的類型程式庫損毀，無法執行作業

`#import` 陳述式中的程式庫檔案已損毀且無法由編譯器使用。  損毀可能來自不正確產生的模組、磁碟損毀，或二進位檔案已經以某種方式編輯或修改。

如需詳細資訊，請參閱 < [#import 指示詞](../../preprocessor/hash-import-directive-cpp.md)。
