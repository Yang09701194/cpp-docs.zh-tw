# 編譯器錯誤 C3356
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'attribute': 無法呼叫擁有完整限定名稱的多點傳送屬性  
  
 不正確地指定多個處理程序 \(例如編譯器和 ATL 提供者\) 所處理的屬性。
