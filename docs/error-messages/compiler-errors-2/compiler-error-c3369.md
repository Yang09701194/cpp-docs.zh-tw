# <a name="compiler-error-c3369"></a>編譯器錯誤 C3369
'module name': 已定義 idl_module  
  
 定義 DLL 的 [idl_module](../../windows/idl-module.md) 用法只能在程式中出現一次。  
  
 下列範例會產生 C3369：  
  
```  
// C3369.cpp  
// compile with: /c  
[idl_module(name="name1", dllname="x.dll")]; // C3369  
[idl_module(name="name1", dllname="x.dll")];  
```
