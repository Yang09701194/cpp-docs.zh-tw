# <a name="compiler-error-c3396"></a>編譯器錯誤 C3396
'attribute' : 在 'namespace' 中找不到自訂屬性  
  
 C3396 表示您未使用最新的 Common Language Runtime。  請重新安裝 CLR 來解決問題。
