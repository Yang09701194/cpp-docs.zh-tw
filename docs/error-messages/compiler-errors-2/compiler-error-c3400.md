# <a name="compiler-error-c3400"></a>編譯器錯誤 C3400
循環條件約束相依性，涉及 'constraint_1' 和 'constraint_2'  
  
 編譯器偵測到循環條件約束。  
  
 如需詳細資訊，請參閱[泛型型別參數的條件約束 (C + + /CLI)](../../windows/constraints-on-generic-type-parameters-cpp-cli.md)。  
  
## <a name="example"></a>範例  
 下列範例會產生 C3400。  
  
```  
// C3400.cpp  
// compile with: /clr /c  
generic<class T, class U>  
where T : U  
where U : T   // C3400  
public ref struct R {};  
```
