# <a name="compiler-error-c3452"></a>編譯器錯誤 C3452

列出不是常數的引數成員

引數已傳遞給預期有常數的屬性，可以在編譯時期評估值。

## <a name="example"></a>範例

下列範例會產生 C3452：

```
// C3452.cpp
// compile with: /c
int i;
[module( name="mod", type=dll, custom={i} ) ];   // C3452
// try the following line instead
// [module( name="mod", type=dll, custom={"a"} ) ];
```
