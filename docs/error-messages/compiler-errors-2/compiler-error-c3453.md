# <a name="compiler-error-c3453"></a>編譯器錯誤 C3453
'attribute': 未套用屬性，因為限定詞 'assembly' 不相符  
  
 只有組件或模組層級屬性才能指定為獨立指示。  
  
## <a name="example"></a>範例  
 下列範例會產生 C3453。  
  
```  
// C3453.cpp  
// compile with: /clr /c  
[assembly:System::CLSCompliant(true)]   // C3453  
// try the following line instead  
// [assembly:System::CLSCompliant(true)];  
ref class X {};  
```
