# <a name="compiler-error-c3455"></a>編譯器錯誤 C3455
'attribute': 沒有任何屬性建構函式與引數相符  
  
 用來宣告屬性的值無效。  如需詳細資訊，請參閱 [attribute](../../windows/attribute.md) 。  
  
## <a name="example"></a>範例  
 下列範例會產生 C3455。  
  
```  
// C3455.cpp  
// compile with: /clr /c  
using namespace System;  
  
[attribute("MyAt")]   // C3455  
// try the following line instead  
// [attribute(All)]  
ref struct MyAttr {  
   MyAttr() {}  
};  
```
