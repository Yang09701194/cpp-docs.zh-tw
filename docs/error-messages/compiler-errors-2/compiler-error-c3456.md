# Compiler Error C3456
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不能在 Managed 或 WinRT 類別宣告上使用 \[source\_annotation\_attribute\]  
  
 source\_annotation\_attribute 用來定義程式碼分析所要使用的自訂屬性。  
  
 必須有相對應的外掛程式了解屬性，這些自訂屬性才有用。
