# <a name="compiler-error-c3456"></a>編譯器錯誤 C3456

不能在 Managed 或 WinRT 類別宣告上使用 [source_annotation_attribute]

source_annotation_attribute 用來定義程式碼分析所要使用的自訂屬性。

必須有相對應的外掛程式了解屬性，這些自訂屬性才有用。
