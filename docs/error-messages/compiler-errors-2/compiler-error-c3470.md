# <a name="compiler-error-c3470"></a>編譯器錯誤 C3470
'type': 類別不能同時有索引子 (預設索引的屬性) 和運算子[]  
  
 類型無法定義預設索引子和運算子[]。  
  
## <a name="example"></a>範例  
 下列範例會產生 C3470：  
  
```  
// C3470.cpp  
// compile with: /clr  
using namespace System;  
  
ref class R {  
public:  
   property int default[int] {  
      int get(int i) {  
         return i+1;  
      }  
   }  
  
   int operator[](String^ s) { return Convert::ToInt32(s); }   // C3470  
};  
  
int main() {  
   R ^ r = gcnew R;  
   // return r[9] + r["32"] - 42;  
}  
```
