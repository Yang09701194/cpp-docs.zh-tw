# 編譯器錯誤 C3500
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無效的 ProgID 'progid'  
  
 以 `#import` 陳述式指定了無效的 Progid。  請檢查 Windows 登錄，確定您指定的是有效的 Progid。
