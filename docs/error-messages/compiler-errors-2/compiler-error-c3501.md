# 編譯器錯誤 C3501
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

沒有為 ProgID 'progid' 註冊的 typelib  
  
 Progid 的類別 ID 沒有相關的型別程式庫。  因此，您不能將此 Progid 傳至 `#import` 陳述式。
