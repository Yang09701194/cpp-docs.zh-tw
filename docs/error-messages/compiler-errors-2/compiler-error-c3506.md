# 編譯器錯誤 C3506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

沒有為 LIBID 'id' 註冊的 typelib  
  
 未正確註冊 typelib。  請使用 regsvr32.exe，註冊型別程式庫。
