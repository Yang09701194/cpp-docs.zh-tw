# <a name="compiler-error-c3553"></a>編譯器錯誤 C3553
decltype 必須是運算式而不是類型  
  
 `decltype()` 關鍵字需要運算式作為引數，而不是類型名稱。 例如，下列程式碼片段的最後一個陳述式會產生錯誤 C3553。  
  
 `int x = 0;`  
  
 `decltype(x+1);`  
  
 `decltype(int); // C3553`
