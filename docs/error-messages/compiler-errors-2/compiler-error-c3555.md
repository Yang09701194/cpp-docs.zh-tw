# 編譯器錯誤 C3555
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'decltype' 的引數不正確  
  
 `decltype(` `expression` `)` 類型規範的引數不是有效的運算式。  
  
> [!NOTE]
>  C3555 不太可能發生。 如果編譯器發出 C3555，可能是因為發生編譯器內部錯誤。
