# <a name="compiler-error-c3610"></a>編譯器錯誤 C3610
'valuetype': 實值型別必須為 'boxed' 才能呼叫方法 'method'  
  
 根據預設，實值型別不是 managed 堆積上。 您可以從呼叫方法.NET 執行階段類別，例如之前`Object`，您需要將實值型別移到受管理的堆積。  
  
 C3610 才可使用過時的編譯器選項**/clr:oldSyntax**。
