# <a name="compiler-error-c3619"></a>編譯器錯誤 C3619
無法在 Managed 或 WinRT 類型中宣告範本  
  
不允許在 Managed 或 WinRT 類別或介面中使用類別樣板。  
  
C3619 才可使用過時的編譯器選項**/clr:oldSyntax**。
