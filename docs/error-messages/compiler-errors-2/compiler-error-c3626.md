# 編譯器錯誤 C3626
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'keyword': '\_\_event' 關鍵字只能使用在本身是委派指標的 COM 介面、成員函式和資料成員上  
  
 關鍵字使用不正確。  
  
 下列範例會產生 C3626：  
  
```  
// C3626.cpp  
// compile with: /c  
struct A {  
   __event int i;   // C3626  
// try the following line instead  
// __event int i();  
};  
```
