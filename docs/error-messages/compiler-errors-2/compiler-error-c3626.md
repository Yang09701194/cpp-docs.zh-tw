# <a name="compiler-error-c3626"></a>編譯器錯誤 C3626
'keyword': '__event' 關鍵字只能用於 COM 介面、 成員函式和是委派指標的資料成員  
  
 關鍵字的使用方式錯誤。  
  
 下列範例會產生 C3626:  
  
```  
// C3626.cpp  
// compile with: /c  
struct A {  
   __event int i;   // C3626  
// try the following line instead  
// __event int i();  
};  
```
