# <a name="compiler-error-c3631"></a>編譯器錯誤 C3631
'function'：無法多載 Managed 或 WinRT 事件  
  
 無法多載 Managed 或 WinRT 事件。  
  
 C3631 才可使用過時的編譯器選項 **/clr:oldSyntax**。
