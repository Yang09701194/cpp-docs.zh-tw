# <a name="compiler-error-c3632"></a>編譯器錯誤 C3632

'event': 不合法的建構事件的樣式

[__event](../../cpp/event.md)宣告不適用於所有的建構。

C3632 才可使用過時的編譯器選項 **/clr: oldsyntax**。
