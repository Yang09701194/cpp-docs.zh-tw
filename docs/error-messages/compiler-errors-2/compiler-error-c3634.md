# <a name="compiler-error-c3634"></a>編譯器錯誤 C3634
'function': 無法定義 managed 或 WinRTclass 抽象方法  
  
 抽象方法可以在 Managed 或 WinRT 類別中宣告，但無法定義。  
  
## <a name="example"></a>範例  
下列範例會產生 C3634：  
  
```  
// C3634.cpp  
// compile with: /clr  
ref class C {  
   virtual void f() = 0;  
};  
  
void C::f() {   // C3634 - don't define managed class' pure virtual method  
}  
```
