# <a name="compiler-error-c3637"></a>編譯器錯誤 C3637
'function': friend 函式定義不能為函式類型的特製化  
  
 Friend 函式的樣板或泛型定義不正確。  
  
 下列範例會產生 C3637:  
  
```  
// C3637.cpp  
template <class T>  
void f();  
  
struct S {  
   friend void f<int>() {}   // C3637  
};  
```  
  
 可能的解決方式：  
  
```  
// C3637b.cpp  
// compile with: /c  
template <class T>  
void f();  
  
struct S {  
   friend void f() {}  
};  
```  
  
 使用泛型時，也會發生 C3637:  
  
```  
// C3637c.cpp  
// compile with: /clr  
  
generic <class T>  
void gf();  
  
struct S {  
   friend void gf<int>() {}   // C3637  
};  
```  
  
 可能的解決方式：  
  
```  
// C3637d.cpp  
// compile with: /clr /c  
generic <class T>  
void gf();  
  
struct S {  
   friend void gf() {}  
};  
```
