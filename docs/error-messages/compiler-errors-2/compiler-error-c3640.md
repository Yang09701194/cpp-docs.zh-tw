# <a name="compiler-error-c3640"></a>編譯器錯誤 C3640

' member '：必須定義區域類別的參考或虛擬成員函式

編譯器需要定義某些函數。

下列範例會產生 C3640：

```cpp
// C3640.cpp
void f()
{
   struct S
   {
      virtual void f1();   // C3640
      // Try the following line instead:
      // virtual void f1(){}
   };
}
```
