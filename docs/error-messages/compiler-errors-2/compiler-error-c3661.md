# <a name="compiler-error-c3661"></a>編譯器錯誤 C3661

明確覆寫清單找不到要覆寫任何方法

明確覆寫指定一或多個型別名稱。  不過，已使用所需的簽章符合覆寫的函式簽章的型別中的函式。  如果您嘗試覆寫根據型別名稱，在指定覆寫的函式簽章相符的類型必須有一或多個虛擬函式。

如需詳細資訊，請參閱 <<c0> [ 明確覆寫](../../extensions/explicit-overrides-cpp-component-extensions.md)。
