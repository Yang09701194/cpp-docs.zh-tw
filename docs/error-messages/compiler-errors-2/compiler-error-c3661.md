# <a name="compiler-error-c3661"></a>編譯器錯誤 C3661

明確覆寫清單找不到任何要覆寫的方法

明確覆寫指定了一或多個類型名稱。  不過，在符合覆寫函式簽章的類型中，沒有具有必要簽章的函式。  如果您嘗試根據型別名稱來覆寫，則指定的型別中必須有一或多個虛擬函數符合覆寫函式的簽章。

如需詳細資訊，請參閱[明確覆寫](../../extensions/explicit-overrides-cpp-component-extensions.md)。
