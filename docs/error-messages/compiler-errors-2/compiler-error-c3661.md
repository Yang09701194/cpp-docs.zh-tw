# <a name="compiler-error-c3661"></a>編譯器錯誤 C3661
明確覆寫清單找不到要覆寫任何方法  
  
 明確覆寫指定一或多個型別名稱。  不過，已使用所需的簽章符合覆寫函式簽章的型別中的函式。  如果您嘗試覆寫根據型別名稱，必須有一個或多個虛擬函式中指定的類型相符的覆寫的函式的簽章。  
  
 如需詳細資訊，請參閱[明確覆寫](../../windows/explicit-overrides-cpp-component-extensions.md)。
