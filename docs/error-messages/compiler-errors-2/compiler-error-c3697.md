# <a name="compiler-error-c3697"></a>編譯器錯誤 C3697

'限定詞': 不能使用這個限定詞上 ' ^'

追蹤控制代碼 (^) 已套用至未設計的限定詞。

下列範例會產生 C3697:

```
// C3697.cpp
// compile with: /clr
using namespace System;
int main() {
   String ^__restrict s;   // C3697
   String ^ s2;   // OK
}
```
