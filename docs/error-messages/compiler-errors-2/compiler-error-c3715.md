# <a name="compiler-error-c3715"></a>編譯器錯誤 C3715

' 指標 '：必須是 ' class ' 的指標

您在[__hook](../../cpp/hook.md)或[__unhook](../../cpp/unhook.md)中指定的指標並未指向有效的類別。 若要修正這個錯誤，請確定您的 `__hook` 和 `__unhook` 呼叫都指定了有效類別的指標。
