# <a name="compiler-error-c3715"></a>編譯器錯誤 C3715

'pointer': 必須是 'class' 的指標

指定中的指標[__hook](../../cpp/hook.md)或是[__unhook](../../cpp/unhook.md) ，並未指向有效的類別。 若要修正這個錯誤，請確認您`__hook`和`__unhook`呼叫指定有效的類別的指標。
