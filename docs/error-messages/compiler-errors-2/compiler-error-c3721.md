# <a name="compiler-error-c3721"></a>編譯器錯誤 C3721
「 簽章': 不相容的簽章的事件  
  
 事件宣告不正確。 如需詳細資訊，請參閱[__event](../../cpp/event.md)。  
  
 C3721 才可使用過時的編譯器選項**/clr:oldSyntax**。
