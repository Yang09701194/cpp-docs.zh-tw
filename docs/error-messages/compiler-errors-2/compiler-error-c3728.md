# <a name="compiler-error-c3728"></a>編譯器錯誤 C3728
'event': 事件沒有引發方法  
  
 中繼資料建立使用的語言，例如 C# 中，不允許外定義所在的類別引發的事件，隨附於[#using](../../preprocessor/hash-using-directive-cpp.md)指示詞，而是使用 CLR 程式設計嘗試的 Visual c + + 程式引發事件。  
  
 若要引發事件，例如 C# 語言開發的程式中，包含事件的類別必須也定義公用方法來引發事件。
