# <a name="compiler-error-c3762"></a>編譯器錯誤 C3762

無法處理屬性 ' attribute '

繼承自 `System.Security.Permissions.SecurityAttribute` 的使用者自訂屬性會用來定義安全性屬性。 這類屬性不能在其定義所在的相同元件中使用。
