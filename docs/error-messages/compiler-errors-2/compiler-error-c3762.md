# 編譯器錯誤 C3762
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法處理屬性 'attribute'  
  
 繼承自 `System.Security.Permissions.SecurityAttribute` 的使用者定義屬性正用來定義安全性屬性。  這樣的屬性不可以於定義它的同一個組件中使用。
