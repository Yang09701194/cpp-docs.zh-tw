# <a name="compiler-error-c3762"></a>編譯器錯誤 C3762

無法處理屬性 'attribute'

使用者定義的屬性繼承自`System.Security.Permissions.SecurityAttribute`用來定義安全性屬性。 這類屬性不能在相同的組件中定義的位置。
