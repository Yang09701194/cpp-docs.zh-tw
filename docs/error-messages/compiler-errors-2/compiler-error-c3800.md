# <a name="compiler-error-c3800"></a>編譯器錯誤 C3800
'declaration': 不能混合屬性和事件  
  
 您無法宣告為屬性和事件的建構。  
  
 C3800 才可使用過時的編譯器選項**/clr:oldSyntax**。
