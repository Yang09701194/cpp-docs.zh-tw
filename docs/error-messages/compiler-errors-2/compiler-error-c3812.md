# <a name="compiler-error-c3812"></a>編譯器錯誤 C3812

'property' 必須是屬性宣告中的第一個語彙基元

宣告屬性時`__property`關鍵字必須是該行的第一個語彙基元。

C3812 才可使用過時的編譯器選項 **/clr: oldsyntax**。
