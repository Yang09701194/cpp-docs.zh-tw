# <a name="compiler-error-c3815"></a>編譯器錯誤 C3815
方法 'get_accessor' 傳回型別必須符合 setter 最後一個參數的型別  
  
 宣告屬性的傳回值時`get_accessor`方法必須符合的 set 存取子方法宣告中的最後一個參數。  
  
 C3815 才可使用過時的編譯器選項 **/clr:oldSyntax**。
