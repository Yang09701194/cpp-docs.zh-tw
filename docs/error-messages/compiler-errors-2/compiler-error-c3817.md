# <a name="compiler-error-c3817"></a>編譯器錯誤 C3817
'declaration': 屬性只能套用至函式  
  
 `property`關鍵字只能套用至函式定義。  
  
 C3817 才可使用過時的編譯器選項**/clr:oldSyntax**。
