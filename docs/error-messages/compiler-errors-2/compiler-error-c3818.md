# <a name="compiler-error-c3818"></a>編譯器錯誤 C3818

陣列屬性宣告 'property1' 不應多載索引屬性 'property2'

當其中一個是索引子，另一個是陣列屬性時，不可能屬性的多載。

C3818 才可使用過時的編譯器選項 **/clr: oldsyntax**。
