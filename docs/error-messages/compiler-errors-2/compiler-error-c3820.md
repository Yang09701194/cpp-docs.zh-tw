# <a name="compiler-error-c3820"></a>編譯器錯誤 C3820

'name'：初始設定式必須是 Managed

Managed 類型的 *name* 初始設定式也必須是 Managed。

### <a name="to-correct-this-error"></a>若要改正這項錯誤

1. 指定 Managed 初始設定式方法。
