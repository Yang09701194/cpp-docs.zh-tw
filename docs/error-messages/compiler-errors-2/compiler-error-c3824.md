# <a name="compiler-error-c3824"></a>編譯器錯誤 C3824
'member': 這個類型不能出現在此內容 （函式參數、 傳回型別或靜態成員）  
  
 Pin 指標不可以是函式參數、 傳回型別，或宣告`static`。  
  
## <a name="example"></a>範例  
 下列範例會產生 C3824:  
  
```  
// C3824a.cpp  
// compile with: /clr /c  
void func() {  
   static pin_ptr<int> a; // C3824  
   pin_ptr<int> b; // OK  
}  
```
