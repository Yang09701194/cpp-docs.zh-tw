# <a name="compiler-error-c3832"></a>編譯器錯誤 C3832

「類型程式庫」：類型程式庫看起來就像是針對32位指標所建立的一樣。請變更 ' ptrsize ' 限定詞

[#Import](../../preprocessor/hash-import-directive-cpp.md)指示詞的 `ptrsize` 屬性所提供的明確資訊，不同意編譯器在類型程式庫中找到的內容。
