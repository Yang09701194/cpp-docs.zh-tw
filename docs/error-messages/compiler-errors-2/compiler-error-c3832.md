# <a name="compiler-error-c3832"></a>編譯器錯誤 C3832

'type library': 類型程式庫會如同它已針對 32 位元指標建置;請變更 'ptrsize' 限定詞

使用提供明確的資訊`ptrsize`的屬性[#import](../../preprocessor/hash-import-directive-cpp.md)指示詞沒有不一致的類型程式庫中找到的編譯器。
