# 編譯器錯誤 C3832
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type library' : 型別程式庫看來像為 32 位元指標建置的；請變更 'ptrsize' 限定詞  
  
 隨 [\#import](../../preprocessor/hash-import-directive-cpp.md) 指示詞的 `ptrsize` 屬性提供的明確資訊，與編譯器在型別程式庫中找到的資訊不一致。
