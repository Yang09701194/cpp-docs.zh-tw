# <a name="compiler-error-c3832"></a>編譯器錯誤 C3832
'type library': 類型程式庫看來像它專為 32 位元指標。請變更 'ptrsize' 限定詞  
  
 明確的資訊提供`ptrsize`屬性[#import](../../preprocessor/hash-import-directive-cpp.md)指示詞 clrtype 不相符的類型程式庫中找到的編譯器。
