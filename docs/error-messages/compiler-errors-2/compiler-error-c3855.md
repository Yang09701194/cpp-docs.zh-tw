# 編譯器錯誤 C3855
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class': type 參數 'param' 與宣告不相容  
  
 編譯器發現不同名稱的非型別樣板或泛型參數。  這可能發生在樣板特製化定義中所指定的樣板參數與其宣告不相容時。  
  
 下列範例會產生 C3855：  
  
```  
// C3855.cpp  
template <int N>  
struct C {  
   void f();  
};  
  
template <char N>  
void C<N>::f() {}   // C3855  
```  
  
 可能的解決方案：  
  
```  
// C3855b.cpp  
// compile with: /c  
template <int N>  
struct C {  
   void f();  
};  
  
template <int N>  
void C<N>::f() {}  
```  
  
 使用 Generics 時也可能發生 C3855：  
  
```  
// C3855c.cpp  
// compile with: /clr  
generic <class T>  
ref struct GC1 {  
   generic <class U>  
   ref struct GC2;  
};  
  
generic <class T>  
generic <class U>  
generic <class V>  
ref struct GC1<T>::GC2 { };   // C3855  
```  
  
 可能的解決方案：  
  
```  
// C3855d.cpp  
// compile with: /clr /c  
generic <class T>  
ref struct GC1 {  
   generic <class U>  
   ref struct GC2;  
};  
  
generic <class T>  
generic <class U>  
ref struct GC1<T>::GC2 { };  
```
