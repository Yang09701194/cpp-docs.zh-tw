# <a name="compiler-error-c3858"></a>編譯器錯誤 C3858

'type': 無法在目前的範圍內重新宣告

相同範圍中的兩次，就不能宣告類型。

下列範例會產生 C3858:

```
// C3858.cpp
// compile with: /LD
template <class T>
struct Outer
{
   struct Inner;
};

template <class T>
struct Outer<T>::Inner;   // C3858
// try the following line instead
// struct Outer<T>::Inner{};
```
