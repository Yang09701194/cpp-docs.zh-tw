# <a name="compiler-error-c3866"></a>編譯器錯誤 C3866
函式呼叫遺漏引數清單  
  
 非靜態成員函式內的解構函式或 finalizer 呼叫沒有引數清單。  
  
 下列範例會產生 C3866:  
  
```  
// C3866.cpp  
// compile with: /c  
class C {  
   ~C();  
   void f() {  
      this->~C;   // C3866  
      this->~C();   // OK  
   }  
};  
```
