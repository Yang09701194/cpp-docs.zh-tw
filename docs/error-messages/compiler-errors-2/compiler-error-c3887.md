# <a name="compiler-error-c3887"></a>編譯器錯誤 C3887
'var': 常值資料成員初始設定式必須是常數運算式  
  
 A[常值](../../windows/literal-cpp-component-extensions.md)資料成員只能與常數運算式初始化。  
  
 下列範例會產生 C3887:  
  
```  
// C3887.cpp  
// compile with: /clr  
ref struct Y1 {  
   static int i = 9;  
   literal  
   int staticConst = i;   // C3887  
};  
```  
  
 可能的解決方式：  
  
```  
// C3887b.cpp  
// compile with: /clr /c  
ref struct Y1 {  
   literal  
   int staticConst = 9;  
};  
```
