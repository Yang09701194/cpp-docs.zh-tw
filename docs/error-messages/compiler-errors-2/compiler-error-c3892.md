# <a name="compiler-error-c3892"></a>編譯器錯誤 C3892
'var': 您無法指派給是 const 變數  
  
 宣告並初始化後，就無法變更是 const 變數。  
  
 下列範例會產生 C3892:  
  
```  
// C3892.cpp  
// compile with: /clr  
ref struct Y1 {  
   static const int staticConst = 9;  
};  
  
int main() {  
   Y1::staticConst = 0;   // C3892  
}  
```
