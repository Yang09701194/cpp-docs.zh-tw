# 編譯器錯誤 C3910
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'event': 必須定義成員 'method'  
  
 已定義事件，但不包含已指定而必要的存取子方法。  
  
 如需詳細資訊，請參閱[事件](../../windows/event-cpp-component-extensions.md)。  
  
 下列範例會產生 C3910：  
  
```  
// C3910.cpp  
// compile with: /clr /c  
delegate void H();  
ref class X {  
   event H^ E {  
      // uncomment the following lines  
      // void add(H^) {}  
      // void remove( H^ h ) {}  
      // void raise( ) {}  
   };   // C3910  
  
   event H^ E2; // OK data member  
};  
```
