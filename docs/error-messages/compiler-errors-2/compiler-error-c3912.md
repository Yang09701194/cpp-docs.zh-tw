# 編譯器錯誤 C3912
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'event': 事件的型別必須是委派型別  
  
 宣告了事件，但型別不適當。  
  
 如需詳細資訊，請參閱[事件](../../windows/event-cpp-component-extensions.md)。  
  
 下列範例會產生 C3912：  
  
```  
// C3912.cpp  
// compile with: /clr  
delegate void H();  
ref class X {  
   event int Ev;   // C3912  
   event H^ Ev2;   // OK  
};  
```
