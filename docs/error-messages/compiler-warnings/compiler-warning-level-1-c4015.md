# <a name="compiler-warning-level-1-c4015"></a>編譯器警告（層級1） C4015

' identifier '：位欄位的類型必須是整數

位欄位未宣告為整數類型。 編譯器會假設位欄位的基底類型為不帶正負號。 位欄位必須宣告為不帶正負號的整數類型。
