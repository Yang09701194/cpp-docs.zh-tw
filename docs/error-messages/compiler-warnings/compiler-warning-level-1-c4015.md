# <a name="compiler-warning-level-1-c4015"></a>編譯器警告 （層級 1） C4015
'identifier': 位元欄位的類型必須是整數類資料  
  
 位元欄位不被宣告為整數型別。 編譯器會假設為不帶正負號位元欄位的基底類型。 位元欄位必須宣告為不帶正負號的整數類型。
