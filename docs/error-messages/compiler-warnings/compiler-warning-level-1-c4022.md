# <a name="compiler-warning-level-1-c4022"></a>編譯器警告（層級1） C4022

' function '：指標與實際參數 ' number ' 不相符

實際參數的指標類型與對應型式參數的指標類型不同。 會傳遞實際參數而不會變更。
