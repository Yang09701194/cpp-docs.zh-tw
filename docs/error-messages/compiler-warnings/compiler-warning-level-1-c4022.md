# 編譯器警告 (層級 1) C4022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 指標與實質參數 'number' 不相符  
  
 實質參數的指標型別與對應的型式參數的指標型別 \(Pointer Type\) 不同。  傳遞此實質參數，未作變更。
