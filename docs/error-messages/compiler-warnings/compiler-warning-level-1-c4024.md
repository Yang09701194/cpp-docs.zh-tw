# 編譯器警告 (層級 1) C4024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 型式和實質參數 'number' 不同的型別  
  
 對應的型式和實質參數具有不同的型別。  編譯器傳遞此參數，未作變更。  接收函式將參數型別轉換成預期的型別。
