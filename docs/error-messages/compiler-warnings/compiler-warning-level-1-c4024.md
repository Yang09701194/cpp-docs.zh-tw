# <a name="compiler-warning-level-1-c4024"></a>編譯器警告（層級1） C4024

' function '：型式和實際參數 ' number ' 的不同類型

對應的正式和實際參數具有不同的類型。 編譯器會傳遞實際參數而不會變更。 接收函式會將參數類型轉換成預期的類型。
