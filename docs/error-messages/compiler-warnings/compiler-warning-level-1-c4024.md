# <a name="compiler-warning-level-1-c4024"></a>編譯器警告 （層級 1） C4024
'function': 不同類型的型式和實質參數 'number'  
  
 對應的型式和實質參數具有不同的類型。 編譯器會傳遞不會變更實際的參數。 接收函式會將參數類型轉換成預期的類型。
