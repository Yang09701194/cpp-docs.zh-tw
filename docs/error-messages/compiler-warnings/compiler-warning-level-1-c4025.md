# <a name="compiler-warning-level-1-c4025"></a>編譯器警告 (層級 1) C4025
'number': 基底指標傳遞至擁有變數引數的函式: 參數號碼  
  
 將基底指標傳遞至具有變數引數的函式會導致指標正規化，但結果無法預期。 請不要將基底指標傳遞至具有變數引數的函式。
