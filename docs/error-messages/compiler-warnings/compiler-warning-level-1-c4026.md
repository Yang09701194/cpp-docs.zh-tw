# 編譯器警告 (層級 1) C4026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

宣告有型式參數清單的函式  
  
 函式宣告具有型式參數，但函式定義則沒有。 這個函式的後續呼叫假設函式不接受任何參數。
