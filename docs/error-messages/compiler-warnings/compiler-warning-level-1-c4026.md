# <a name="compiler-warning-level-1-c4026"></a>編譯器警告 (層級 1) C4026

宣告有型式參數清單的函式

函式宣告具有型式參數，但函式定義則沒有。 這個函式的後續呼叫假設函式不接受任何參數。
