# <a name="compiler-warning-level-1-c4027"></a>編譯器警告 (層級 1) C4027

宣告沒有型式參數清單的函式

函式宣告沒有型式參數，但在函式定義中有型式參數或在呼叫中有實質參數存在。 之後對這個函式的呼叫會假設函式使用在函式定義或呼叫中找到之類型的實質參數。
