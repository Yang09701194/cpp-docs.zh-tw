# 編譯器警告 (層級 1) C4029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

宣告的型式參數清單與定義不同  
  
 函式宣告中的型式參數型別與函式定義中的不符合。  編譯器會使用來自定義中的參數清單。
