# <a name="compiler-warning-level-1-c4030"></a>編譯器警告 (層級 1) C4030
**第一個型式參數清單比第二清單長**  
  
 已使用不同的型式參數重新宣告函式。 編譯器會使用第一個宣告中指定的型式參數。
