# <a name="compiler-warning-level-1-c4031"></a>編譯器警告 （層級 1） C4031
第二個型式參數清單比第一清單長  
  
 函式是使用不同的型式參數重新宣告。 編譯器會使用第一個宣告中指定的型式參數。
