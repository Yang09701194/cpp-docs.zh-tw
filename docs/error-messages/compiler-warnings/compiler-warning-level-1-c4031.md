# 編譯器警告 (層級 1) C4031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

第二型式參數清單比第一清單長  
  
 函式使用不同的型式參數重新宣告。  編譯器會使用第一個宣告中給定的型式參數。
