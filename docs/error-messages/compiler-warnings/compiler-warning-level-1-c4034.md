# 編譯器警告 (層級 1) C4034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sizeof 傳回 0  
  
 套用 `sizeof` 運算子至大小為零的運算元 \(空結構、等位、類別或列舉類型，或是 `void` 型別\)。
