# <a name="compiler-warning-level-1-c4034"></a>編譯器警告 （層級 1） C4034

sizeof 傳回 0

`sizeof`運算子會套用至運算元的大小為零 (空的結構、 等位、 類別或列舉的類型，或類型`void`)。
