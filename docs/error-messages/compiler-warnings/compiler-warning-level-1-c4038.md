# <a name="compiler-warning-level-1-c4038"></a>編譯器警告 (層級 1) C4038
'modifier' : 不合法的環境類別修飾詞  
  
 此修飾詞不能用於具有 **dllimport** 或 [dllexport](../../cpp/dllexport-dllimport.md) 屬性的類別。
