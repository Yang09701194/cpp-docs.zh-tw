# 編譯器警告 (層級 1) C4045
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'array' : 陣列界限溢位  
  
 陣列的初始設定式太多。 已忽略額外的初始設定式。  
  
 請確定陣列項目和初始設定式的大小和數量相符。
