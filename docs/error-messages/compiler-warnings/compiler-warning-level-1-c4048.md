# <a name="compiler-warning-level-1-c4048"></a>編譯器警告 （層級 1） C4048

不同的宣告陣列註標: 'identifier1' 和 'identifier2'

運算式包含不同大小的陣列的指標。 指標會使用不需要轉換。

如果您明確轉型為相同或對等型別陣列，則可能會修正這個警告。
