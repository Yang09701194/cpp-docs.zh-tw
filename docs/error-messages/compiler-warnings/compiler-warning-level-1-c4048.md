# 編譯器警告 (層級 1) C4048
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不同的陣列註標 : 'identifier1' 和 'identifier2'  
  
 運算式包含指向不同大小陣列的指標。  使用未經轉換的指標。  
  
 如果您明確地將陣列轉換成相同或相等的型別，就能修正這項警告。
