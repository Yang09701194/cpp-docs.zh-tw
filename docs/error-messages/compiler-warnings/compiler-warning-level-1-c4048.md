# <a name="compiler-warning-level-1-c4048"></a>編譯器警告（層級1） C4048

不同宣告的陣列注標： ' identifier1 ' 和 ' identifier2 '

運算式包含不同大小之陣列的指標。 使用指標而不進行轉換。

如果您明確地將陣列轉換成相同或對等的類型，就可以修正這個警告。
