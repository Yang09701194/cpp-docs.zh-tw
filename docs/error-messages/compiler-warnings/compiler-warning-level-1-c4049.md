# <a name="compiler-warning-level-1-c4049"></a>編譯器警告 （層級 1） C4049

編譯器限制： 結束行號發出

此檔案包含多個 16777215 (2<sup>24</sup>-1) 原始程式行。 編譯器會停止在 16777215 編號。

16777215 行的程式碼：

- 映像會包含行號不偵錯資訊。

- 某些診斷可能會報告有不正確的行號。

- .asm 清單 (/ FAs) 可能會有不正確的行號。
