# 編譯器警告 (層級 1) C4081
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

必須是 'token1'; 但找到 'token2'  
  
 編譯器必須是這個內容中的不同語彙基元。  
  
## 範例  
  
```  
// C4081.cpp // compile with: /W1 /LD #pragma optimize) "l", on )   // C4081  
```
