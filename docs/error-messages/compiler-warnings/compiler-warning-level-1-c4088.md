# 編譯器警告 (層級 1) C4088
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 指標與實質參數 'number' 中的不相符，型式參數 'number'  
  
 對應的型式和實質參數具有不同層級的間接取值。  傳遞此實質參數，未作變更。  所呼叫的函式將它的值解譯為指標。
