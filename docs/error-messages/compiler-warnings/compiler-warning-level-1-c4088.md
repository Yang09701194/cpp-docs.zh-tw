# <a name="compiler-warning-level-1-c4088"></a>編譯器警告 （層級 1） C4088
'function': 實質參數 'number'、 'number' 的型式參數中的指標不相符  
  
 對應的型式和實質參數具有不同層級的間接取值。 實際的參數會傳遞不會變更。 呼叫的函式會將該值解譯為指標。
