# <a name="compiler-warning-level-1-c4088"></a>編譯器警告（層級1） C4088

' function '：指標與實際參數 ' number '，型式參數 ' number ' 不相符

對應的正式和實際參數具有不同層級的間接取值。 會傳遞實際參數而不會變更。 被呼叫的函式會將其值解釋為指標。
