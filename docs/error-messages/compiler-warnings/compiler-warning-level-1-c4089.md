# 編譯器警告 (層級 1) C4089
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 實質參數 'number' 中的不同型別，型式參數 'number'  
  
 對應的型式和實質參數具有不同的型別。  傳遞此實質參數，未作變更。  此函式將實質參數轉換成函式定義中指定的型別。
