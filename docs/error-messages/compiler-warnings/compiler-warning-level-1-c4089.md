# <a name="compiler-warning-level-1-c4089"></a>編譯器警告 （層級 1） C4089

'function': 實質參數 'number'、 'number' 的型式參數中的不同類型

對應的型式和實質參數具有不同的類型。 實際的參數被傳遞不會變更。 此函式是函式定義中指定的型別將實質參數的轉換。
