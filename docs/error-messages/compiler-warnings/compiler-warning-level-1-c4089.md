# <a name="compiler-warning-level-1-c4089"></a>編譯器警告（層級1） C4089

' function '：實際參數 ' number ' 中的不同類型，型式參數 ' number '

對應的正式和實際參數具有不同的類型。 會傳遞實際參數而不會變更。 函式會將實際參數轉換成函式定義中指定的類型。
