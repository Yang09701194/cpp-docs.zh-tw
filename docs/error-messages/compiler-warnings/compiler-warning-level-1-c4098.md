# 編譯器警告 (層級 1) C4098
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 'void' 函式正在傳回值  
  
 宣告為 [void](../../cpp/void-cpp.md) 傳回型別的函式具有傳回值的 `return` 陳述式。  編譯器假設函式傳回 `int` 型別的值。
