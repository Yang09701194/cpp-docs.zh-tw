# <a name="compiler-warning-level-1-c4098"></a>編譯器警告（層級1） C4098

' function '： void 函數傳回值

以傳回類型[void](../../cpp/void-cpp.md)宣告的函式具有傳回值的 `return` 語句。 編譯器會假設函式會傳回 `int`類型的值。
