# 編譯器警告 (層級 1) C4113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier1' 與來自 'identifier2' 的參數清單不同  
  
 一個函式指標被指派成另一個函式指標，但是函式的型式參數清單 \(Formal Parameter List\) 不接受。  該指派會在不經修改情況下進行編譯。
