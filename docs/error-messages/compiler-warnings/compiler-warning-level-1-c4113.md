# <a name="compiler-warning-level-1-c4113"></a>編譯器警告（層級1） C4113

' identifier1 ' 與 ' identifier2 ' 中的參數清單不同

函式指標已指派給另一個函式指標，但函式的型式參數清單不同意。 此指派會在不修改的情況下進行編譯。
