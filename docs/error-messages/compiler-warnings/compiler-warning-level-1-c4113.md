# <a name="compiler-warning-level-1-c4113"></a>編譯器警告 （層級 1） C4113

'identifier1' 與 'identifier2' 的參數清單中

函式指標指派給另一個函式指標，但不是同意函式的型式參數清單。 指派會編譯而不需修改。
