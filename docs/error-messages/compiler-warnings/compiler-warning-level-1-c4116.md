# <a name="compiler-warning-level-1-c4116"></a>編譯器警告（層級1） C4116

括弧中未命名的類型定義

不含名稱的結構、等位或列舉類型是在括號運算式中定義。 類型定義沒有意義。

在 C 函式呼叫中，此定義具有全域範圍。 在C++函式呼叫中，定義的範圍與所呼叫的函式相同。
