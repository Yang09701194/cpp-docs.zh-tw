# <a name="compiler-warning-level-1-c4116"></a>編譯器警告 （層級 1） C4116
在括號中的未命名的類型定義  
  
 結構、 等位或列舉型別沒有名稱會放在括號運算式中定義。 類型定義是沒有意義。  
  
 在 C 函式呼叫中，此定義具有全域範圍。 在 c + + 函式呼叫中，定義具有相同的範圍和被呼叫的函式。
