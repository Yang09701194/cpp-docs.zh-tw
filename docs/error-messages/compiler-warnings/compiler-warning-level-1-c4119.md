# 編譯器警告 (層級 1) C4119
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

指定的基底 'base1' 和 'base2' 不同  
  
 兩個基底指標不相容，因為它們的基底不同。 編譯器無法轉換它們。
