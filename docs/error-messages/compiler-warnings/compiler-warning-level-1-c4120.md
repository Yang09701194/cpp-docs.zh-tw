# 編譯器警告 (層級 1) C4120
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

兩者分別為基底與非基底，無法轉換  
  
 無法在兩個指標之間轉換編譯器，因為其中一個是基底指標，另一個不是。
