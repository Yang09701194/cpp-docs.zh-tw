# <a name="compiler-warning-level-1-c4120"></a>編譯器警告 (層級 1) C4120
兩者分別為基底與非基底，無法轉換  
  
 無法在兩個指標之間轉換編譯器，因為其中一個是基底指標，另一個不是。
