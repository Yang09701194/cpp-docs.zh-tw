# <a name="compiler-warning-level-1-c4122"></a>編譯器警告 (層級 1) C4122

'function': alloc_text 僅適用使用 C 連結的函式

**alloc_text** pragma 只適用於以 **extern c**宣告的函式。 它不能搭配外部 C++ 函式使用。

忽略 pragma。
