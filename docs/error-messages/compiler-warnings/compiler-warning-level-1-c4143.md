# 編譯器警告 (層級 1) C4143
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

pragma 'same\_seg' 不支援; 使用 \_\_based 配置  
  
 不再支援 **\#pragma same\_seg**。 請改用 [\_\_based](../../cpp/based-pointers-cpp.md) 關鍵字。
