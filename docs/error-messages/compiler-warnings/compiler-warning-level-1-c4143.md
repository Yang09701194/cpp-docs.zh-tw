# <a name="compiler-warning-level-1-c4143"></a>編譯器警告 (層級 1) C4143

pragma 'same_seg' 不支援; 使用 __based 配置

不再支援 **#pragma same_seg** 。 請改用 [__based](../../cpp/based-pointers-cpp.md) 關鍵字。
