# <a name="compiler-warning-level-1-c4153"></a>編譯器警告 (層級 1) C4153
運算式中函式/資料的指標轉換  
  
 函式指標會轉換成資料指標，反之亦然。 Microsoft 擴充功能 (/Ze) 下允許這項轉換，但 ANSI C 下則不允許。
