# <a name="compiler-warning-level-1-c4158"></a>編譯器警告 (層級 1) C4158
假設使用 #pragma pointers_to_members(full_generality, inheritance)  
  
 A **#pragma pointers_to_members (***單一 | 多個 | 虛擬***)**已發出但未伴隨**#pragma pointers_to_members**。
