# <a name="compiler-warning-level-1-c4166"></a>編譯器警告 (層級 1) C4166
**對建構函式/解構函式的不合法呼叫慣例**  
  
 建構函式和解構函式不能有平台預設值以外的呼叫慣例 (您明確指定 **__clrcall**時除外)。
