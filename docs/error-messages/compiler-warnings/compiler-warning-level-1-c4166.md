# 編譯器警告 (層級 1) C4166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**對建構函式\/解構函式的不合法呼叫慣例**  
  
 建構函式和解構函式不能有平台預設值以外的呼叫慣例 \(您明確指定 **\_\_clrcall** 時除外\)。
