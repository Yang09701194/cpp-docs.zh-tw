# 編譯器警告 (層級 1) C4168
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

編譯器限制: 偵錯工具類型不足，必須刪除程式資料庫 'database' 並重新建置  
  
 您必須重建程式資料庫檔案，以容納程式中的所有類型。
