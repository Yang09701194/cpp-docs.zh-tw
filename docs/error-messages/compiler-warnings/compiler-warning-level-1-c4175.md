# <a name="compiler-warning-level-1-c4175"></a>編譯器警告 (層級 1) C4175

\#pragma component(browser, on)： 必須在命令列上指定瀏覽器資訊

若要使用 [component](../../preprocessor/component.md) pragma，您必須在編譯期間產生瀏覽資訊 ([/FR](../../build/reference/fr-fr-create-dot-sbr-file.md))。
