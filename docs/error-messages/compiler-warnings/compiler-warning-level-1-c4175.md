# <a name="compiler-warning-level-1-c4175"></a>編譯器警告 (層級 1) C4175

 若要使用[元件](../../preprocessor/component.md)pragma，您必須產生瀏覽資訊編譯期間 ([/FR](../../build/reference/fr-fr-create-dot-sbr-file.md))。
