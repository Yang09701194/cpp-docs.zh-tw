# <a name="compiler-warning-level-1-c4178"></a>編譯器警告 (層級 1) C4178

對 switch 運算式的類型而言，case 常數 'constant' 太大

`switch` 運算式中的 case 常數不符合指派給它的類型。

## <a name="example"></a>範例

```
// C4178.cpp
// compile with: /W1
int main()
{
    int i;  // maximum size of unsigned long int is 4294967295
    switch( i )
    {
        case 4294967295:   // OK
            break;
        case 4294967296:   // C4178
            break;
    }
}
```
