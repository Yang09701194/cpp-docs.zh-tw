# <a name="compiler-warning-level-1-c4179"></a>編譯器警告 (層級 1) C4179

> '//* '：剖析為 '/' 和 '/\*'：與標準的 '//' 批註混淆

__//\*__ 是不正確的註解分隔符號。 請改用 __//__ 或 __/\*__ 。
