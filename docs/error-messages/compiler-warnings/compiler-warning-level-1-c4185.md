# <a name="compiler-warning-level-1-c4185"></a>編譯器警告 (層級 1) C4185
忽略未知的 #import 屬性 'attribute'  
  
 屬性不是 `#import`的有效屬性。 會忽略此項。  
  
## <a name="example"></a>範例  
  
```  
// C4185.cpp  
// compile with: /W1 /c  
#import "stdole2.tlb" no_such_attribute   // C4185  
```
