# 編譯器警告 (層級 1) C4187
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#import 屬性 'attribute1' 和 'attribute2' 不相容; 兩者都已忽略  
  
 [\#import](../../preprocessor/hash-import-directive-cpp.md) 陳述式已指定 `no_implementation` 和 `implementation_only` 屬性。 兩者都會予以忽略。
