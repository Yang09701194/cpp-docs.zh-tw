# <a name="compiler-warning-level-1-c4187"></a>編譯器警告 (層級 1) C4187
\#匯入的屬性 'attribute1' 和 'attribute2' 不相容。兩者都已忽略  
  
 A [#import](../../preprocessor/hash-import-directive-cpp.md)陳述式指定`no_implementation`和`implementation_only`屬性。 兩者都會予以忽略。
