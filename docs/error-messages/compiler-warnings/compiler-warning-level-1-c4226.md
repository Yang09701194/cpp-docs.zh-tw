# <a name="compiler-warning-level-1-c4226"></a>編譯器警告 (層級 1) C4226

> 使用非標準的擴充： '*關鍵字*' 是過時的關鍵字

目前的 Visual C++版本不會使用此關鍵字。

此警告會自動升級為錯誤。
