# 編譯器警告 (層級 1) C4264
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'virtual\_function': 基底類別 'class' 的虛擬成員函式沒有覆寫; 函式已被隱藏  
  
 C4264 一定會在 [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md) 之後產生。  
  
 此警告在預設情況下為關閉的。  如需詳細資訊，請參閱[預設為關閉的編譯器警告](../../preprocessor/compiler-warnings-that-are-off-by-default.md)。
