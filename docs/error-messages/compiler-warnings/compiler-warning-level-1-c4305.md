# 編譯器警告 (層級 1) C4305
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 從 'type1' 到 'type2' 發生截斷狀況  
  
 識別項被轉換成較小的型別，造成資訊遺漏。  
  
 如果您使用 ATL 實作連接點精靈，請參閱知識庫 \(Knowledge Base\) 文件：  
  
-   Q250847 : 具 VARIANT 引數之事件的 ATL 連接點精靈產生程式碼提供 C4305 警告
