# <a name="compiler-warning-level-1-c4305"></a>編譯器警告 (層級 1) C4305
'identifier': 從 'type1' 到 'type2' 截斷  
  
 識別項會轉換成較小的類型，進而導致遺失的資訊。  
  
 如果您使用 ATL 連接點精靈時，請參閱知識庫文件：  
  
-   Q250847: ATL 連接點精靈產生的程式碼的事件具有變數引數提供 C4305 警告
