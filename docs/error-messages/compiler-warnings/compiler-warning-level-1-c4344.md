# <a name="compiler-warning-level-1-c4344"></a>編譯器警告 (層級 1) C4344
行為變更： 使用明確樣板引數會導致呼叫 'function'  
  
 使用明確樣板引數的函式的呼叫會比從未指定明確的引數呼叫不同的函式
