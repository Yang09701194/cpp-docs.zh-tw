# <a name="compiler-warning-level-1-c4344"></a>編譯器警告 (層級 1) C4344

行為變更：使用明確範本引數會導致呼叫 ' function '

使用明確樣板引數呼叫函式，會呼叫不同的函式，而不是未指定明確引數的函數。
