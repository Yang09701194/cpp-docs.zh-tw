# 編譯器警告 (層級 1) C4344
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

行為變更: 使用明確樣板引數會導致對 'function' 的呼叫  
  
 如果沒有指定明確引數，則使用明確樣板引數的函式將會呼叫不同於原本呼叫的函式
