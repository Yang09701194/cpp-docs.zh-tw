# 編譯器警告 (層級 1) C4375
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

非公用方法 'method2' 不會覆寫 'method1'  
  
 實作另一種型別的型別定義了覆寫方法，但覆寫不是公用的。  因此，方法不會覆寫基底型別方法。
