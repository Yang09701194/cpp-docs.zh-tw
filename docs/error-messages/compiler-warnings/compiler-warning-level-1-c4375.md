# <a name="compiler-warning-level-1-c4375"></a>編譯器警告 (層級 1) C4375

非公用方法 'method2' 未覆寫 'method1'

實作另一種類型的類型定義覆寫方法，但公用覆寫。 因此，方法不會覆寫基底型別方法。
