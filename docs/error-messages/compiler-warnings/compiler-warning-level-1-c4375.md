# <a name="compiler-warning-level-1-c4375"></a>編譯器警告 (層級 1) C4375
非公用方法 'method2' 不會覆寫 'method1'  
  
 實作 另一種類型的類型定義覆寫方法，但覆寫不是公用。 因此，此方法不會覆寫基底類型方法。
