# 編譯器警告 (層級 1) C4379
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這個編譯器不支援 version 版的 Common Language Runtime。使用這個版本會造成無法預期的結果。  
  
 您的電腦上有舊版的 Common Language Runtime，而不是最新版的。  若要解決 C4379 的問題，請安裝編譯器交貨時隨附的 Common Language Runtime。
