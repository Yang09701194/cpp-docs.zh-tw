# <a name="compiler-warning-level-1-c4379"></a>編譯器警告 (層級 1) C4379
此編譯器不支援版本的 common language runtime 版本。 使用這個版本可能會導致非預期的結果。  
  
 您有舊版的 common language runtime 上您的電腦，但不是目前的版本。 若要解決 C4379，安裝隨附的 common language runtime 的版本與您的編譯器。
