# <a name="compiler-warning-level-1-c4379"></a>編譯器警告 (層級 1) C4379

這個編譯器不支援 common language runtime 版本。 使用此版本可能會導致非預期的結果。

您的電腦上有舊版通用語言執行時間，但目前的版本不存在。 若要解決 C4379，請安裝編譯器隨附的 common language runtime 版本。
