# <a name="compiler-warning-level-1-c4381"></a>編譯器警告 (層級 1) C4381
'function1': 非公用方法 'function2' 將不會實作介面方法  
  
 類別必須實作介面中的所有函式。 如果其中一個基底類別實作的函式類別，能滿足此條件。 不過，此函式必須實作為公用函式。
