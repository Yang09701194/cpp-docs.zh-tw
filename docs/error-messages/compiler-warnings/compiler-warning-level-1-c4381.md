# <a name="compiler-warning-level-1-c4381"></a>編譯器警告 (層級 1) C4381

' function1 '：介面方法不會由非公用方法 ' function2 ' 執行

類別必須實作用介面中的所有函式。 如果類別的其中一個基類會執行此函式，則可以滿足此條件。 不過，函數必須實作為公用函式。
