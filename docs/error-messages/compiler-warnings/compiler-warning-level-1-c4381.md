# 編譯器警告 (層級 1) C4381
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function1' : 介面方法將無法由非公用方法 'function2' 實作  
  
 類別必須實作介面中所有函式。  如果其中一個基底類別實作函式，類別可以滿足這些條件。  但是函式必須實作為公用函式。
