# 編譯器警告 (層級 1) C4402
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

必須使用 PTR 運算子  
  
 在內嵌組譯程式碼中參考或轉換型別時，在運算元上使用型別而未使用 PTR 運算子。  
  
 編譯器假設使用 PTR 運算子。
