# <a name="compiler-warning-level-1-c4402"></a>編譯器警告 (層級 1) C4402
必須使用 PTR 運算子  
  
 類型用不含參照，或內嵌組譯碼中的型別轉型時 PTR 運算子的運算元上。  
  
 編譯器會假設 PTR 運算子。
