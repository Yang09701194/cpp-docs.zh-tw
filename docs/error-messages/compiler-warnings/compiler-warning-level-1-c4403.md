# <a name="compiler-warning-level-1-c4403"></a>編譯器警告 (層級 1) C4403
不合法的 PTR 運算子  
  
 PTR 運算子是不當使用內嵌組合語言程式碼中。  
  
 編譯器會忽略 PTR 運算子。
