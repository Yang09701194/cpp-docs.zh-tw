# 編譯器警告 (層級 1) C4403
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不合法的 PTR 運算子  
  
 PTR 運算子在內嵌組譯程式碼中的使用不正確。  
  
 編譯器忽略 PTR 運算子。
