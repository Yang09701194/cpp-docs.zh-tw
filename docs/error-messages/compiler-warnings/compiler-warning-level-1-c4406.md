# <a name="compiler-warning-level-1-c4406"></a>編譯器警告 (層級 1) C4406
已忽略指示詞上的運算元  
  
 指示詞不接受任何運算元，但並未指定的運算元。  
  
 編譯器會忽略指定的運算元。
