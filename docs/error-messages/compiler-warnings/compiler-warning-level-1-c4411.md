# 編譯器警告 (層級 1) C4411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 符號解析至替代登錄  
  
 識別項是解析為替代登錄的區域符號，因此在運算元上只能使用其他符號。
