# <a name="compiler-warning-level-1-c4440"></a>編譯器警告 (層級 1) C4440
呼叫慣例 'calling_convention2' 略過從 'calling_convention1' 重複定義  
  
 嘗試變更的呼叫慣例已忽略。  
  
 下列範例會產生 C4440:  
  
```  
// C4440.cpp  
// compile with: /W1 /LD /clr  
typedef void __clrcall F();  
typedef F __cdecl *PFV;   // C4440  
```
