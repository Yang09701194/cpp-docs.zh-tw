# <a name="compiler-warning-level-1-c4445"></a>編譯器警告 (層級 1) C4445
'function'：在 WinRT 或 Managed 類型中，虛擬方法不可為私用  
  
 若是私用虛擬函式，它無法衍生類型存取。 若要修正此錯誤，請將虛擬成員函式的存取範圍變更為受保護或公用。
