# <a name="compiler-warning-level-1-c4502"></a>編譯器警告 (層級 1) C4502
'連結規格' 必須使用關鍵字 'extern'，必須在所有其他規範之前  
  
 指定連結時未`extern`關鍵字。 連結至非外部型別無關。  
  
 編譯器假設`extern`關鍵字。
