# <a name="compiler-warning-level-1-c4502"></a>編譯器警告 (層級 1) C4502

'連結規格' 必須使用關鍵字 'extern'，前面必須加上所有其他的規範

指定連結時，如果未`extern`關鍵字。 連結至非外部類型無關。

編譯器假設`extern`關鍵字。
