# <a name="compiler-warning-level-1-c4506"></a>編譯器警告 (層級 1) C4506

沒有內嵌函數 ' function ' 的定義

已宣告指定的函式，並將其標示為內嵌，但未定義。

編譯器未內嵌函數。

請確定要內嵌的外部函式會使用 `extern` 關鍵字來宣告。
