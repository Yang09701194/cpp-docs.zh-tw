# <a name="compiler-warning-level-1-c4506"></a>編譯器警告 (層級 1) C4506

沒有內嵌函式 'function' 的定義

指定的函式已宣告，並標示為內嵌，但未定義。

編譯器未內嵌函式。

請確定內嵌的外部函式宣告為具有`extern`關鍵字。
