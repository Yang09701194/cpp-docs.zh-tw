# 編譯器警告 (層級 1) C4506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

沒有內嵌函式 'function' 的定義  
  
 函式被宣告並標示為內嵌，但並未被定義。  
  
 編譯器不內嵌該函式。  
  
 請確認要內嵌的外部函式是用 `extern` 關鍵字宣告的。
