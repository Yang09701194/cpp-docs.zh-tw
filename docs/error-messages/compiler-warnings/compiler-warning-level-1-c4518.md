# <a name="compiler-warning-level-1-c4518"></a>編譯器警告 (層級 1) C4518

'specifier': 儲存類別或; 型別未預期的規範已忽略

下列範例會產生 C4518:

```
// C4518.cpp
// compile with: /c /W1
_declspec(dllexport) extern "C" void MyFunction();   // C4518

extern "C" void MyFunction();   // OK
```
