# <a name="compiler-warning-level-1-c4533"></a>編譯器警告 (層級 1) C4533
'variable' 的初始化會被 '指令' 略過  
  
 指令程式中的變更控制項的流量，使得初始化變數的指令，所以未執行。 下列範例會產生 C4533:  
  
```  
// C4533.cpp  
// compile with: /W1  
#include <stdio.h>  
  
struct A  
{  
   int m_data;  
};  
  
int main()  
{  
   if (1)  
   {  
      goto Label;  
   }  
  
   A a = { 100 };  
  
   Label:   // C4533  
      printf("\n%d", a.m_data);   // prints an uninitialized value  
}  
```
