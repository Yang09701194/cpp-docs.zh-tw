# <a name="compiler-warning-level-1-c4541"></a>編譯器警告 (層級 1) C4541

多型類型 'type' 與 /GR-; ' identifier'可能會造成無法預期的行為

您嘗試使用的功能，需要執行階段類型資訊，而不啟用執行階段類型資訊。 重新編譯[/GR](../../build/reference/gr-enable-run-time-type-information.md)。
