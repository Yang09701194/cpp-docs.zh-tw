# <a name="compiler-warning-level-1-c4541"></a>編譯器警告 (層級 1) C4541

在具有/GR-的多型類型 ' type ' 上使用 ' identifier ';可能導致無法預期的行為

您嘗試使用需要執行時間類型資訊的功能，而不需啟用執行時間類型資訊。 使用[/GR](../../build/reference/gr-enable-run-time-type-information.md)重新編譯。
