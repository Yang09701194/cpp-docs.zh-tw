# 編譯器警告 (層級 1) C4541
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' 搭配 \/GR\- 於多型型別 'type' 上使用; 可能發生無法預期的行為  
  
 您試著使用需要執行階段型別資訊的功能，但沒有啟用執行階段型別資訊。  請用 [\/GR](../../build/reference/gr-enable-run-time-type-information.md) 重新編譯。
