# <a name="compiler-warning-level-1-c4550"></a>編譯器警告 (層級 1) C4550

運算式評估為遺漏引數清單的函式

已取值的指標，函式為遺漏引數清單。

## <a name="example"></a>範例

```
// C4550.cpp
// compile with: /W1
bool f()
{
   return true;
}

typedef bool (*pf_t)();

int main()
{
   pf_t pf = f;
   if (*pf) {}  // C4550
   return 0;
}
```
