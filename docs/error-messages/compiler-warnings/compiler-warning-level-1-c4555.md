# <a name="compiler-warning-level-1-c4555"></a>編譯器警告（層級1） C4555

運算式無效; 必須是具有副作用的運算式

當運算式沒有任何作用時，這個警告就會通知您。

此警告預設為關閉。 如需詳細資訊，請參閱 [預設為關閉的編譯器警告](../../preprocessor/compiler-warnings-that-are-off-by-default.md) 。

例如:

```cpp
// C4555.cpp
// compile with: /W1
#pragma warning(default:4555)

void func1()
{
   1;   // C4555
}

void func2()
{
   int x;
   x;   // C4555
}

int main()
{
}
```
