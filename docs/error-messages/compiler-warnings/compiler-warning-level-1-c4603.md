# <a name="compiler-warning-level-1-c4603"></a>編譯器警告 (層級 1) C4603

'\<識別項 >': 未定義巨集，或使用先行編譯標頭之後，定義是不同

*識別項* 預留位置所指定的巨集不同，或在先行編譯器標頭之後不再定義。
