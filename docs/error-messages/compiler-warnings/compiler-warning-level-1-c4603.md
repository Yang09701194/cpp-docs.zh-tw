# 編譯器警告 (層級 1) C4603
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\<識別項\>': 巨集未定義，或是定義在使用先行編譯標頭後變成不同  
  
 *識別項* 預留位置所指定的巨集不同，或在先行編譯器標頭之後不再定義。
