# 編譯器警告 (層級 1) C4613
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'segment': 無法變更區段類別  
  
 您嘗試建立之區段的類別名稱與編譯器所使用的區段相同。 未建立任何新的區段類別。
