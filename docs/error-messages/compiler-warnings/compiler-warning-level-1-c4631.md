# <a name="compiler-warning-level-1-c4631"></a>編譯器警告 (層級 1) C4631

MSXML 或 XPath 無法使用，將不會處理 XML 文件註解。 reason

您的 Common Language Runtime 安裝沒有必要的檔案，無法支援處理文件註解。 請重新安裝 Common Language Runtime。
