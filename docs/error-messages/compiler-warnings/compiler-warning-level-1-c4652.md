# <a name="compiler-warning-level-1-c4652"></a>編譯器警告 (層級 1) C4652
編譯器選項 'option' 與先行編譯標頭; 不一致目前的命令列選項會覆寫先行編譯標頭中定義  
  
 將指定的命令列選項的差異在於建立先行編譯標頭 (.pch) 時提供。 使用目前的命令列中指定的選項。  
  
 重新產生先行編譯標頭，以指定的命令列選項，可以避免此警告。
