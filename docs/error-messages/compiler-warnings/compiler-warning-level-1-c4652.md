# 編譯器警告 (層級 1) C4652
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

編譯器選項 'option' 與先行編譯標頭檔不相容；目前的命令列選項會覆寫先行編譯標頭檔中所定義的選項  
  
 提供的命令列選項與建立先行編譯標頭 \(.pch\) 時提供的不同。  會使用目前命令列指定的選項。  
  
 以提供的命令列選項重新產生先行編譯標頭，可避免此警告。
