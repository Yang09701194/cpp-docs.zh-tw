# <a name="compiler-warning-level-1-c4652"></a>編譯器警告 (層級 1) C4652

編譯器選項 ' option ' 與先行編譯標頭檔不一致;目前的命令列選項會覆寫在先行編譯標頭檔中定義的

當建立先行編譯標頭檔（.pch）時，指定的命令列選項與指定的不相同。 使用目前命令列中指定的選項。

藉由使用指定的命令列選項重新產生先行編譯標頭檔，可以避免這個警告。
