# <a name="compiler-warning-level-1-c4679"></a>編譯器警告 (層級 1) C4679

> '*成員*': 無法匯入成員

編譯器會遇到一種建構，它無法支援，無法從中繼資料匯入。

請勿嘗試使用建構。
