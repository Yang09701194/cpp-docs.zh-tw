# <a name="compiler-warning-level-1-c4811"></a>編譯器警告 (層級 1) C4811

pragma conform(forScope, show) 的值 == value

當您使用 **conform** pragma 的 [show](../../preprocessor/conform.md) 選項時，會發出這個警告。 *value* 是目前符合的值。
