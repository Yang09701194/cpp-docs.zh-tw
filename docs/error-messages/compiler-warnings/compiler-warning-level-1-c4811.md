# 編譯器警告 (層級 1) C4811
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

pragma conform\(forScope, show\) 的值 \=\= value  
  
 當您使用 [conform](../../preprocessor/conform.md) pragma 的 **show** 選項時，會發出這個警告。*value* 是目前符合的值。
