# 編譯器警告 (層級 1) C4821
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法判斷 Unicode 編碼類型，請用簽章 \(BOM\) 儲存檔案  
  
 編譯器無法判斷檔案的編碼類型。  若要解決這個警告，請用位元組順序標記 \(Byte Order Marker\) 儲存檔案。  如需詳細資訊，請參閱[管理具編碼方式的檔案](http://msdn.microsoft.com/zh-tw/ab3a74da-5c8f-4680-8882-f1def4909e82)。
