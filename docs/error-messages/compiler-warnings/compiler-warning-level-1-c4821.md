# <a name="compiler-warning-level-1-c4821"></a>編譯器警告 (層級 1) C4821

> 無法判斷 Unicode 編碼類型，請將檔案儲存使用簽章 (BOM)

編譯器無法判斷檔案的編碼類型。 若要解決這個警告，請將檔案儲存使用位元組順序標記。 請參閱[管理的檔案，以編碼方式](/sql/ssms/solution/manage-files-with-encoding)如需詳細資訊。
