# <a name="compiler-warning-level-1-c4821"></a>編譯器警告 (層級 1) C4821

> 無法判斷 Unicode 編碼類型，請以簽章（BOM）儲存檔案

編譯器無法判斷檔案的編碼類型。 若要解決這個警告，請以位元組順序標記儲存檔案。 如需詳細資訊，請參閱[使用編碼方式管理](/sql/ssms/solution/manage-files-with-encoding)檔案。
