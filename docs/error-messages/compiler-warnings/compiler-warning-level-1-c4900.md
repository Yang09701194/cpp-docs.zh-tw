# 編譯器警告 (層級 1) C4900
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在 'tool1' 'version1' 版和 'tool2' 'version2' 版之間發生中繼語言不符  
  
 *tool1* 和 *tool2* 使用的中繼語言不相符。 請檢查是否已安裝這兩種工具的最新版本。
