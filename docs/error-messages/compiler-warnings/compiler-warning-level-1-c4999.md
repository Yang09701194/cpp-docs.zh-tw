# <a name="compiler-warning-level-1-c4999"></a>編譯器警告 (層級 1) C4999

未知的警告 從 [說明] 功能表中，選擇 [技術支援] 命令或開啟技術支援說明檔以取得詳細資訊

請記錄錯誤發生的情況、設法找出問題癥結，並建立可重現的測試案例，然後連絡 [Microsoft 產品支援服務](/visualstudio/ide/talk-to-us)。
