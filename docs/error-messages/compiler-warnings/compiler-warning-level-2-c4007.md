# <a name="compiler-warning-level-2-c4007"></a>編譯器警告 （層級 2） C4007

'function': 必須是 '屬性'

未明確指定函式的必要的屬性。 例如，函式**主要**必須要有`__cdecl`屬性。 編譯器會強制屬性。
