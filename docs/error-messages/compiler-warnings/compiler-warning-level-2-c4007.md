# <a name="compiler-warning-level-2-c4007"></a>編譯器警告（層級2） C4007

' function '：必須是 ' attribute '

未明確陳述函式的必要屬性。 例如，函數**main**必須具有 `__cdecl` 屬性。 編譯器會強制執行屬性。
