# 編譯器警告 (層級 2) C4007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 必須為 'attribute'  
  
 未明確指定函式必要的屬性。  例如，**main** 函式必須具 `__cdecl` 屬性。  編譯器強制這個屬性。
