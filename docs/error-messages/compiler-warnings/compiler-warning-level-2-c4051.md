# <a name="compiler-warning-level-2-c4051"></a>編譯器警告 (層級 2) C4051

轉換類型，可能導致資料遺失

運算式包含兩個具有不同基底類型的資料項目。 轉換其中一個類型會使資料項目被截斷。

如果您將資料項目轉換為適當類型，即可修正這個警告。
