# <a name="compiler-warning-level-2-c4309"></a>編譯器警告 (層級 2) C4309
'conversion': 常值的截斷  
  
 類型轉換會導致常數超過配置給它的空間。 您可能需要使用較大的類型常數。  
  
 下列範例會產生 C4309:  
  
```  
// C4309.cpp  
// compile with: /W2  
int main()  
{  
   char c = 128;   // C4309  
}  
```
