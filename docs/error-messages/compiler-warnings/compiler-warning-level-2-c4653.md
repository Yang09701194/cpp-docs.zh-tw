# <a name="compiler-warning-level-2-c4653"></a>編譯器警告 (層級 2) C4653

編譯器選項 'option' 與先行編譯標頭; 不一致忽略目前的命令列選項

使用先行編譯標頭所指定的選項 ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) 選項是建立先行編譯標頭時所指定的選項不一致。 此編譯使用先行編譯標頭建立時指定的選項。

組件結構選項的不同值時，可能會發生這個警告 ([/Zp](../../build/reference/zp-struct-member-alignment.md)) 已先行編譯標頭的編譯期間指定。
