# <a name="compiler-warning-level-2-c4653"></a>編譯器警告 (層級 2) C4653

編譯器選項 ' option ' 與先行編譯標頭檔不一致;已忽略目前的命令列選項

以 [使用先行編譯標頭檔（[/yu](../../build/reference/yu-use-precompiled-header-file.md)）] 選項指定的選項與建立先行編譯標頭檔時指定的選項不一致。 這個編譯會使用建立先行編譯標頭檔時所指定的選項。

編譯預先編譯的標頭期間指定了不同的套件結構選項（[/Zp](../../build/reference/zp-struct-member-alignment.md)）值時，可能會出現這個警告。
