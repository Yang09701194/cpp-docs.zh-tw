# <a name="compiler-warning-level-2-c4948"></a>編譯器警告 (層級 2) C4948
'accessor' 的傳回型別不符合對應 setter 的最後一個參數類型  
  
編譯器發現不相符的資料型別是目前所取得，且設定為索引的屬性。  
  
C4948 才可使用過時的編譯器選項**/clr:oldSyntax**。
