# 編譯器警告 (層級 3) C4013
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' 未定義; 假設 extern 傳回整數  
  
 編譯器遇到未定義函式的呼叫。  
  
### 若要修正，請檢查下列可能原因  
  
1.  函式名稱拼字不正確  
  
2.  外部函式的原型宣告不是 `extern`
