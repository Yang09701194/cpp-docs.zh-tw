# <a name="compiler-warning-level-3-c4013"></a>編譯器警告 （層級 3） C4013
' function' 未定義。假設 extern 傳回整數  
  
 編譯器遇到未定義的函式的呼叫。  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正  
  
1.  函式名稱的拼字錯誤  
  
2.  沒有原型做為外部函式 `extern`
