# <a name="compiler-warning-level-3-c4133"></a>編譯器警告 （層級 3） C4133
'type': 不相容的類型-'type1' 到 'type2'  
  
 這個警告可能因嘗試減去兩個不同類型的指標。  
  
 若要避免這個警告，提供適當的型別轉型。
