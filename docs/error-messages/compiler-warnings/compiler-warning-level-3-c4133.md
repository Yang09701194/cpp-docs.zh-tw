# <a name="compiler-warning-level-3-c4133"></a>編譯器警告（層級3） C4133

' type '：不相容的類型-從 ' type1 ' 到 ' type2 '

這項警告可能是嘗試減去不同類型的兩個指標所造成。

若要避免這個警告，請提供適當的類型轉換。
