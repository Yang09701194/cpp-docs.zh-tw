# 編譯器警告 (層級 3) C4133
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type' : 不相容的型別 \- 從 'type1' 至 'type2'  
  
 這項警告可能會因嘗試減去不同型別的兩個指標而發生。  
  
 若要避免出現這個警告，請提供適當的型別轉換 \(Type Cast\)。
