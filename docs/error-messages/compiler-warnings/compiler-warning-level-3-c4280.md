# 編譯器警告 (層級 3) C4280
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type'，'operator \-\>' 為透過型別的自我遞迴  
  
 您的程式碼錯誤地允許 **operator–\>** 呼叫其本身。  
  
 下列範例會產生 C4280：  
  
```  
// C4280.cpp  
// compile with: /W3 /WX  
struct A  
{  
   int z;  
   A& operator ->();  
};  
  
void f(A y)  
{  
   int i = y->z; // C4280  
}  
```
