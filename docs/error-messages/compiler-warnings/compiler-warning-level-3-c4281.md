# <a name="compiler-warning-level-3-c4281"></a>編譯器警告 (層級 3) C4281

'operator->' 的遞迴發生透過類型 'type'

您的程式碼可讓**operator->** 呼叫本身。

下列範例會產生 C4281:

```
// C4281.cpp
// compile with: /W3 /WX
struct A;
struct B;
struct C;

struct A
{
   int z;
   B& operator->();
};

struct B
{
   C& operator->();
};

struct C
{
   A& operator->();
};

void f(A p)
{
   int i = p->z; // C4281
}
```
