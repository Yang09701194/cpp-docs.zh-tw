# <a name="compiler-warning-level-3-c4283"></a>編譯器警告 (層級 3) C4283

也透過類型 'type'

警告的接續[C4281](../../error-messages/compiler-warnings/compiler-warning-level-3-c4281.md)會顯示**operator->** 呼叫本身透過`type`。
