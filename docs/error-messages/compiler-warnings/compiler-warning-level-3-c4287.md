# <a name="compiler-warning-level-3-c4287"></a>編譯器警告 (層級 3) C4287
'operator': unsigned 和負常數不相符  
  
 以負數作業中使用的不帶正負號的變數。  
  
 此警告預設為關閉。 如需詳細資訊，請參閱 [預設為關閉的編譯器警告](../../preprocessor/compiler-warnings-that-are-off-by-default.md) 。  
  
## <a name="example"></a>範例  
 下列範例會產生 C4287:  
  
```  
// C4287.cpp  
// compile with: /W3  
#pragma warning(default : 4287)  
#include <stdio.h>  
  
int main()  
{  
    unsigned int u = 1;  
    if (u < -1)   // C4287  
        printf_s("u LT -1");  
    else  
        printf_s("u !LT -1");  
    return 0;  
}  
```
