# 編譯器警告 (層級 3) C4306
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**'**   
 ***identifier* ' : 將 '**   
 ***type1* ' 轉換為較大的 '**   
 ***type2* '**  
  
 識別項被型別轉換為較大的指標。  新型別中未被填入的高位元會由零填補。  
  
 此警告可能表示非預期的轉換。  結果指標可能無法使用。
