# <a name="compiler-warning-level-3-c4306"></a>編譯器警告 (層級 3) C4306
**'**   
 ***識別項*': 從 '**   
 ***type1* 'to'**   
 ***type2* ' 更大的**  
  
 識別項是型別轉型為較大的指標。 中空的高位新類型會是零填滿。  
  
 此警告可能表示非預期的轉換。 產生的指標可能無效。
