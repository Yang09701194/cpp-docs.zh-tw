# <a name="compiler-warning-level-3-c4306"></a>編譯器警告 (層級 3) C4306

> '*identifier*'：從 '*type1*' 轉換為較大的 '*type2*'

識別碼是轉換成較大指標的類型。 新類型未填入的高位將會填滿零。

此警告可能表示不想要的轉換。 產生的指標可能無效。
