# <a name="compiler-warning-level-3-c4306"></a>編譯器警告 (層級 3) C4306

> '*識別碼*': 從轉換'*type1*'to'*type2*' 更大的

識別項是型別轉換為較大的指標。 新的型別未填入高的位元會是零填滿。

這個警告可能表示非預期的轉換。 產生的指標可能無效。
