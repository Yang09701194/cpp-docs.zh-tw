# 編譯器警告 (層級 3) C4310
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

轉換截斷常數值  
  
 常數的值轉換為較小的型別。  編譯器會進行截斷資料的轉換。  下列範例會產生 C4310：  
  
```  
// C4310.cpp  
// compile with: /W4  
int main() {  
   long int a;  
   a = (char) 128;   // C4310, use value 0-127 to resolve  
}  
```
