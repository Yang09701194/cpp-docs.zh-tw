# 編譯器警告 (層級 3) C4511
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class' : 無法產生複製建構函式  
  
 編譯器無法為類別建立預設的複製建構函式 \(Copy\-Constructor\)；基底類別可以有私用的複製建構函式。
