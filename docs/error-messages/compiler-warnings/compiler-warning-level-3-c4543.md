# <a name="compiler-warning-level-3-c4543"></a>編譯器警告 (層級 3) C4543
屬性 'no_injected_text' 隱藏插入的文字  
  
 [No_injected_text](../../windows/no-injected-text.md)屬性出現在原始程式碼中，這表示編譯器將會防止屬性插入程式碼。  
  
 這項警告是，用意在提醒您屬性不能插入程式碼。
