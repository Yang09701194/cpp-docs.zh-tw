# 編譯器警告 (層級 3) C4543
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

屬性 'no\_injected\_text' 隱藏插入的文字  
  
 [no\_injected\_text](../../windows/no-injected-text.md) 屬性出現在來源程式檔，這表示編譯器會阻止屬性插入程式碼中。  
  
 此警告是要提醒您無法將屬性插入程式碼中。
