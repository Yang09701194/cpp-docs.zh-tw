# <a name="compiler-warning-level-3-c4622"></a>編譯器警告 (層級 3) C4622
覆寫在目的檔中建立先行編譯標頭檔期間產生的偵錯資訊: 'file'  
  
 指定的檔案中的 CodeView 資訊已遺失與編譯時[/Yu](../../build/reference/yu-use-precompiled-header-file.md) （使用先行編譯標頭） 選項。  
  
 重新命名物件檔案 (使用[/Fo](../../build/reference/fo-object-file-name.md)) 時建立或使用先行編譯標頭檔，並使用新的物件檔案。
