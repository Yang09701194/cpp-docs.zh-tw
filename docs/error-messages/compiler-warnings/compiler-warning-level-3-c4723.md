# <a name="compiler-warning-level-3-c4723"></a>編譯器警告 (層級 3) C4723

可能除以0

除法運算中的第二個運算元在編譯時期評估為零，提供未定義的結果。

只有在使用[/og](../../build/reference/og-global-optimizations.md)或表示/Og. 的優化選項時，才會發出此警告。

編譯器可能產生零運算元。
