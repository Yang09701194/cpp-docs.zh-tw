# <a name="compiler-warning-level-3-c4723"></a>編譯器警告 (層級 3) C4723
可能除以 0  
  
 評估為編譯時期零未定義的結果除以作業中的第二個運算元。  
  
 只有在使用時，會發出這個警告[/Og](../../build/reference/og-global-optimizations.md)或 /og 的最佳化選項。  
  
 編譯器可能會產生零個運算元。
