# 編譯器警告 (層級 3) C4724
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

MOD 的模數可能為 0  
  
 在編譯時期，會將餘數運算中的第二個運算元評估為零，因而產生未定義的結果。
