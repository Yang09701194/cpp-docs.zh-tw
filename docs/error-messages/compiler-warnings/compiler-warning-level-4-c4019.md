# <a name="compiler-warning-level-4-c4019"></a>編譯器警告 (層級 4) C4019
在全域範圍有空的陳述式  
  
 全域範圍的分號前面沒有陳述式。  
  
 如果您移除多餘的分號，可能即會修正這個警告。  
  
## <a name="example"></a>範例  
  
```  
// C4019.c  
// compile with: /Za /W4  
#define declint( varname ) int varname;  
declint( a );   // C4019, int a;;  
declint( b )   // OK, int b;  
  
int main()  
{  
}  
```
