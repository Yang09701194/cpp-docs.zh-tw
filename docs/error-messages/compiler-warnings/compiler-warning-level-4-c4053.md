# 編譯器警告 (層級 4) C4053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'?:' 有一類型為 void 的運算元  
  
 提供了類型 `void` 的運算式給 `?:` 運算子。 未定義 `void` 運算元的值。
