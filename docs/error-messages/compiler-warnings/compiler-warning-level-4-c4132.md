# 編譯器警告 (層級 4) C4132
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'object' : 應初始化常數物件  
  
 常數並未初始化。 具有 **const** 屬性的符號值無法在定義後進行變更，您必須針對該值提供初始化數值。
