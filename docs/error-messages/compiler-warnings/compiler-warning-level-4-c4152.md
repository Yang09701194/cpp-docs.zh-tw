# <a name="compiler-warning-level-4-c4152"></a>編譯器警告 (層級 4) C4152

非標準的擴充，運算式中函式/資料的指標轉換

函式指標會轉換成資料指標，反之亦然。 Microsoft 擴充功能 (/Ze) 下允許這項轉換，但 ANSI C 下則不允許。
