# <a name="compiler-warning-level-4-c4202"></a>編譯器警告 (層級 4) C4202
使用非標準擴充: '...': 不合法的名稱清單中的原型參數  
  
 舊式函式定義包含變數引數。 這些定義產生 ANSI 相容性錯誤 ([/Za](../../build/reference/za-ze-disable-language-extensions.md))。  
  
## <a name="example"></a>範例  
  
```  
// C4202.c  
// compile with: /W4  
void func( a, b, ...)   // C4202  
int a, b;  
{}  
  
int main()  
{  
}  
```
