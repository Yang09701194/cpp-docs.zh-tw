# <a name="compiler-warning-level-4-c4206"></a>編譯器警告 (層級 4) C4206

**使用非標準擴充： 轉譯單位是空白**

在前置處理之後，檔案是空的。

此延伸模組可以防止您的程式碼移植到其他編譯器。 它會產生 ANSI 相容性錯誤 ([/Za](../../build/reference/za-ze-disable-language-extensions.md)) 和僅適用於 C 原始程式碼。
