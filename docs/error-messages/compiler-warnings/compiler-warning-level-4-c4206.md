# <a name="compiler-warning-level-4-c4206"></a>編譯器警告 (層級 4) C4206

**使用非標準的擴充：轉譯單位是空的**

前置處理之後，檔案是空的。

此延伸模組可防止您的程式碼可移植到其他編譯器。 它會在 ANSI 相容性（[/za](../../build/reference/za-ze-disable-language-extensions.md)）下產生錯誤，而且僅適用于 C 原始程式碼。
