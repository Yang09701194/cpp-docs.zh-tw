# <a name="compiler-warning-level-4-c4324"></a>編譯器警告 (層級 4) C4324
'struct_name': 因為 __declspec(align()) 而獲得填補的結構  
  
 填補已加入結尾的結構，因為指定[有](../../cpp/align-cpp.md)值。  
  
 例如，下列程式碼會產生 C4324:  
  
```  
// C4324.cpp  
// compile with: /W4  
struct __declspec(align(32)) A  
{  
   char a;  
};   // C4324  
  
int main()  
{  
}  
```
