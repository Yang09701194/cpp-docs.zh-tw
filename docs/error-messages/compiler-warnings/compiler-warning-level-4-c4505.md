# <a name="compiler-warning-level-4-c4505"></a>編譯器警告 (層級 4) C4505

' function '：已移除未參考的區域函式

給定的函式在本機，而且不會在模組主體中參考;因此，函式是無作用程式碼。

編譯器未產生這個無效函式的程式碼。
