# 編譯器警告 (層級 4) C4505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 已移除未參考的區域函式  
  
 函式為區域性函式，而且在模組本體中未被參考；因此，該函式是無作用程式碼 \(Dead Code\)。  
  
 編譯器不會為此無作用函式產生程式碼。
