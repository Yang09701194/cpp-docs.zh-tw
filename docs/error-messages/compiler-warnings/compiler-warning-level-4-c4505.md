# <a name="compiler-warning-level-4-c4505"></a>編譯器警告 (層級 4) C4505

'function': 已移除未參考本機函式

指定的函式位於本機並不受參考的模組; 主體中因此，此函式是無作用程式碼。

編譯器不會產生這個無作用的函式程式碼。
