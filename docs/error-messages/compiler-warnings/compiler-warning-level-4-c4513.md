# <a name="compiler-warning-level-4-c4513"></a>編譯器警告 (層級 4) C4513

'class': 無法產生解構函式

編譯器無法產生指定的類別; 預設解構函式建立沒有解構函式。 解構函式是不是在衍生類別存取基底類別中。 如果基底類別的私用解構函式，讓公用或受保護。
