# <a name="compiler-warning-level-4-c4513"></a>編譯器警告 (層級 4) C4513

' class '：無法產生析構函式

編譯器無法為指定的類別產生預設的析構函式;未建立任何析構函式。 此析構函式位於無法供衍生類別存取的基類中。 如果基類具有私用的析構函式，請將它設為公用或受保護。
