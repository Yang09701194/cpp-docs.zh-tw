# <a name="compiler-warning-level-4-c4517"></a>編譯器警告 (層級 4) C4517

不再支援存取宣告;成員 using 宣告代替

ANSI c + + 委員會已宣告存取宣告 (變更而不需要在衍生類別中成員的存取權[使用](../../cpp/using-declaration.md)關鍵字) 為過期。 C + + 的未來版本可能不支援存取宣告。
