# <a name="compiler-warning-level-4-c4517"></a>編譯器警告 (層級 4) C4517
不再支援存取宣告;成員 using 宣告提供較佳替代方式  
  
 ANSI c + + committee 已宣告存取宣告 (變更而不在衍生類別中成員的存取權[使用](../../cpp/using-declaration.md)關鍵字) 為過期。 C + + 的未來版本可能不支援存取宣告。
