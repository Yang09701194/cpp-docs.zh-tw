# <a name="compiler-warning-level-4-c4517"></a>編譯器警告 (層級 4) C4517

存取宣告已被取代;使用宣告的成員可以提供更好的替代方案

ANSI C++委員會已宣告存取宣告（在不[使用](../../cpp/using-declaration.md)關鍵字的情況下變更衍生類別中成員的存取權）會過時。 未來的C++版本可能不支援存取宣告。
