# 編譯器警告 (層級 4) C4517
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

鼓勵使用存取宣告；建議使用成員 using 宣告代替  
  
 The ANSI C\+\+ 委員會聲明存取宣告 \(未用 [using](../../cpp/using-declaration.md) 關鍵字而更改衍生類別中成員的存取\) 已過期。  未來的 C\+\+ 版本可能不支援存取宣告。
