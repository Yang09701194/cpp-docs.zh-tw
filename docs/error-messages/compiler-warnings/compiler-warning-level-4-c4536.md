# 編譯器警告 (層級 4) C4536
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type name' : 型別名稱超出中繼資料 'limit' 個字元的限制  
  
 如果是 Managed 型別，型別的名稱在中繼資料會被截斷。  如需詳細資訊，請參閱 [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md)。  
  
 此警告在預設情況下為關閉的。  如需詳細資訊，請參閱[預設為關閉的編譯器警告](../../preprocessor/compiler-warnings-that-are-off-by-default.md)。
