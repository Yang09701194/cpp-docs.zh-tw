# <a name="compiler-warning-level-4-c4629"></a>編譯器警告 (層級 4) C4629
使用了雙拼詞，字元順序 'digraph' 被解譯為語彙基元 'char' (如果這不是您想要的，請在兩個字元之間插入一個空格)  
  
 在 [/Za](../../build/reference/za-ze-disable-language-extensions.md)下，偵測到雙拼詞時，編譯器會發出警告。  
  
 下列範例會產生 C4629：  
  
```  
// C4629.cpp  
// compile with: /Za /W4  
int main()  
<%   // C4629 <% digraph for {  
}  
```
