# <a name="compiler-warning-level-4-c4639"></a>編譯器警告 (層級 4) C4639

MSXML 錯誤，將不會處理 XML 檔批註。 reason

發生此警告的原因有很多。

若要解決此警告：

- 重新編譯.

- 重新安裝 common language runtime 來重新安裝 MSXML。

- 編輯或移除造成警告和重新編譯的檔批註。

發出 C4639 時，會停用所有進一步的 XML 批註處理，而且不會產生 .xdc 檔案。
