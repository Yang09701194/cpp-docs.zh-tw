# <a name="compiler-warning-level-4-c4639"></a>編譯器警告 (層級 4) C4639

MSXML 錯誤，將不會處理註解的 XML 文件。 原因

任意數目的原因可能會發生這個警告。

若要解決這個警告：

- 重新編譯。

- 重新安裝 common language runtime，重新安裝 MSXML。

- 編輯或移除文件註解會導致警告，然後重新編譯。

C4639 時，會停用所有進一步的 XML 註解處理，並不會產生.xdc 檔案。
