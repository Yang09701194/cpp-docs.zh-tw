# 編譯器警告 (層級 4) C4672
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier1' : 模稜兩可。 第一個看到的當做 'identifier2'  
  
 要在 **try** 區塊中擲回的指定物件模稜兩可。 如果擲回物件，則無法明確地區分物件。
