# 編譯器警告 (層級 4) C4931
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

我們假設已針對 number 位元指標建置類型程式庫  
  
 [\#import](../../preprocessor/hash-import-directive-cpp.md) 指示詞的 **ptrsize** 屬性並未提供明確資訊；編譯器會認為型別程式庫的指標大小為 ***number***。  
  
 此警告在預設情況下為關閉的。  如需詳細資訊，請參閱[預設為關閉的編譯器警告](../../preprocessor/compiler-warnings-that-are-off-by-default.md)。
