# <a name="compiler-warning-levels-1-and-4-c4223"></a>編譯器警告 (層級 1 和 4) C4223

使用非標準的擴充：轉換成指標的非左值陣列

在標準 C 中，您無法將非左值陣列轉換成指標。 使用預設的 Microsoft 擴充功能（[/ze](../../build/reference/za-ze-disable-language-extensions.md)）時，您可以。
