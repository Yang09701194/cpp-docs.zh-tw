# <a name="compiler-warning-levels-1-and-4-c4223"></a>編譯器警告 (層級 1 和 4) C4223

使用非標準擴充： 非左值陣列轉換為指標

在標準 C 中，您無法轉換為非左值陣列的指標。 使用預設的 Microsoft 擴充功能 ([/Ze](../../build/reference/za-ze-disable-language-extensions.md))，您可以。
