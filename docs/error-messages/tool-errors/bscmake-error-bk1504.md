# BSCMAKE 錯誤 BK1504
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法在檔案 'filename' 中定位 \[: reason\]  
  
 BSCMAKE 無法移到檔案的某個位置。  
  
### 若要修正，請檢查下列可能原因  
  
1.  磁碟已滿。  
  
2.  硬體錯誤。  
  
3.  因為在建立 .sbr 檔案時磁碟空間不足或編譯器中斷，導致檔案截斷。
