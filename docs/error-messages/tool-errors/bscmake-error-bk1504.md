# <a name="bscmake-error-bk1504"></a>BSCMAKE 錯誤 BK1504

無法在檔案 'filename' 中的位置 [: 原因]

BSCMAKE 無法移動到檔案中的位置。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正

1. 磁碟已滿。

1. 硬體錯誤。

1. 由於磁碟空間不足或中斷的編譯器建立.sbr 檔案時的檔案截斷。
