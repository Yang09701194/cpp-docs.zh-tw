# BSCMAKE 錯誤 BK1505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法讀取檔案 'filename' \[: reason\]  
  
 BSCMAKE 無法讀取檔案。  
  
### 若要修正，請檢查下列可能原因  
  
1.  檔案損毀。  
  
2.  因為在建立 .sbr 檔案時，編譯器造成磁碟空間不足或被中斷，導致檔案截斷。
