# <a name="bscmake-error-bk1505"></a>BSCMAKE 錯誤 BK1505

無法讀取檔案 'filename' [: 原因]

BSCMAKE 無法從檔案讀取。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正

1. 檔案損毀。

1. 檔案給編譯器用盡磁碟空間或中斷時建立的.sbr 檔的截斷。
