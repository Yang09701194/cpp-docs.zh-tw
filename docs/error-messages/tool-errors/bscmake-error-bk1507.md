# BSCMAKE 錯誤 BK1507
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法開啟暫存檔 'filename' \[: reason\]  
  
 BSCMAKE 無法開啟暫存檔。  
  
### 若要修正，請檢查下列可能原因  
  
1.  **TMP** 環境變數 \(Environment Variable\) 未被設定為有效的磁碟機和目錄。  
  
2.  磁碟已滿。
