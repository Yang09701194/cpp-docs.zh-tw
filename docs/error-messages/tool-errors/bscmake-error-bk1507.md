# <a name="bscmake-error-bk1507"></a>BSCMAKE 錯誤 BK1507

無法開啟暫存檔 'filename' [: 原因]

BSCMAKE 無法開啟暫存檔案。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正

1. **TMP**環境變數未設為有效的磁碟機和目錄。

1. 磁碟已滿。
