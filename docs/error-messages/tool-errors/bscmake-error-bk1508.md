# <a name="bscmake-error-bk1508"></a>BSCMAKE 錯誤 BK1508

無法刪除暫存檔案 ' filename ' [： reason]

BSCMAKE 無法刪除其中一個暫存檔案。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正

1. 另一個進程開啟了檔案。

1. 硬體錯誤。
