# BSCMAKE 錯誤 BK1508
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法刪除暫存檔 'filename'\[: reason\]  
  
 BSCMAKE 無法刪除暫存檔。  
  
### 若要修正，請檢查下列可能原因  
  
1.  其他處理序已經開啟此檔案。  
  
2.  硬體錯誤。
