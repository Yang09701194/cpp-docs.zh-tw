# BSCMAKE 錯誤 BK1512
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

filename: 超出容量  
  
 BSCMAKE 無法建置瀏覽資訊檔 \(Browse Information File\)，因為定義、參考、模組或其他資訊的數量超過限制。  
  
### 若要採用下列可能解決方式以進行修正  
  
1.  使用 \/Em、\/Es 或 \/Ei 排除某些資訊。  
  
2.  省略 \/Iu 選項。
