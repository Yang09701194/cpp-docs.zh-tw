# <a name="bscmake-error-bk1514"></a>BSCMAKE 錯誤 BK1514
所有。截斷 SBR 檔案，檔名中找不到  
  
 指定要更新的.sbr 檔案沒有原始的瀏覽資訊 (.bsc) 檔的一部分。 若要尋找導致此錯誤的.sbr 檔案名稱，請參閱[BK4502](../../error-messages/tool-errors/bscmake-warning-bk4502.md)前面的警告。  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正  
  
1.  .Sbr 或.bsc 指定的檔案名稱錯誤。  
  
2.  .Bsc 檔損毀所需 BSCMAKE 重建它。
