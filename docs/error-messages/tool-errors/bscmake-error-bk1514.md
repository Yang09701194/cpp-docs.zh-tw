# <a name="bscmake-error-bk1514"></a>BSCMAKE 錯誤 BK1514

這.已截斷 .SBR 檔案，檔案名中沒有找到

不是針對更新指定的任何 .sbr 檔案都屬於原始流覽資訊（.bsc）檔案的一部分。 若要尋找造成此錯誤的 .sbr 檔案名，請閱讀其前面的[BK4502](../../error-messages/tool-errors/bscmake-warning-bk4502.md)警告。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正

1. 為 .sbr 或 .bsc 指定了錯誤的檔案名。

1. 已損毀的 .bsc 檔案需要 BSCMAKE 來重建它。
