# BSCMAKE 錯誤 BK1515
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'bscfile': 不相容的版本；無法累加更新  
  
 此 .bsc 檔案是由其他版本的 BSCMAKE 所建立，  請重新建立此 .bsc 檔案。
