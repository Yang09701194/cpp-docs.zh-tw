# <a name="bscmake-error-bk1515"></a>BSCMAKE 錯誤 BK1515
bscfile： 版本不相容： 無法以累加方式更新  
  
 .Bsc 檔是以另一個版本的 BSCMAKE 建立。 重新建立.bsc 檔案。
