# <a name="bscmake-error-bk1516"></a>BSCMAKE 錯誤 BK1516
bscfile 損毀。無法以累加方式更新  
  
 .Bsc 檔已損毀，可能是因為在建置期間的系統失敗。 刪除.bsc 檔，請重建所有.sbr 檔，然後重新都建置.bsc 檔。
