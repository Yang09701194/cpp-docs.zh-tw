# BSCMAKE 錯誤 BK1516
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

bscfile 損毀; 無法累加更新  
  
 此 .bsc 檔案已損毀，可能是建置檔案時發生系統錯誤。  請刪除此 .bsc 檔案，重建所有的 .sbr 檔案，然後再重建 .bsc 檔案。
