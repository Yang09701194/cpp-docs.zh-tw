# <a name="bscmake-error-bk1516"></a>BSCMAKE 錯誤 BK1516

bscfile 已損毀;無法以累加方式更新

.Bsc 檔案已損毀，可能是因為組建期間發生系統失敗。 刪除 .bsc 檔案，重建所有 .sbr 檔，然後重建 .bsc 檔案。
