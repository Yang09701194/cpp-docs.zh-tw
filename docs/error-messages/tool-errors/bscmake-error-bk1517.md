# <a name="bscmake-error-bk1517"></a>BSCMAKE 錯誤 BK1517
sbrfile 以 /Yc 和 /Yu 編譯原始程式檔  
  
 .Sbr 檔案參照了自己。 它可能已使用 /Yc 編譯之後重新編譯使用 /Yu。 /Yc，重設原始程式檔的編譯器選項，然後選取 **重建**以產生新的.sbr 檔案。 不重新編譯 /Yu 的原始程式檔。
