# <a name="bscmake-error-bk1517"></a>BSCMAKE 錯誤 BK1517

sbrfile 以 /Yc 和 /Yu 編譯的原始程式檔

.Sbr 檔案是指本身。 它可能已使用 /Yc 編譯之後重新編譯使用 /Yu。 重新設定原始程式檔的編譯器選項 /Yc，然後選取**重建**以產生新的.sbr 檔案。 不重新編譯原始程式檔與 /Yu。
