# <a name="bscmake-error-bk1517"></a>BSCMAKE 錯誤 BK1517

以/Yc 和/Yu 編譯之 sbrfile 的原始程式檔

.Sbr 檔案是指本身。 在使用/Yc. 編譯之後，可能會使用/Yu 重新編譯它 將原始程式檔的編譯器選項重設為/Yc，然後選取 [**重建**] 以產生新的 .sbr 檔案。 請勿使用/Yu. 重新編譯來源檔案
