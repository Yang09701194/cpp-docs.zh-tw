# <a name="bscmake-warning-bk4503"></a>BSCMAKE 警告 BK4503
已忽略 .SBR 檔案檔名中的次要錯誤  
  
 .sbr 檔案中包含的錯誤未中止組建，但產生的 .bsc 檔案可能不正確。 若要修正這個問題，請重新編譯以建立新的 .sbr。
