# <a name="c-runtime-error-r6019"></a>C 執行階段錯誤 R6019

無法開啟主控台裝置

> [!NOTE]
> 如果您在執行應用程式時遇到這個錯誤訊息，應用程式已關閉，因為它嘗試存取主控台，但它沒有足夠的許可權。 此錯誤有幾個可能的原因，但通常是因為程式必須以系統管理員的身分執行，或程式中有錯誤。
>
> 您可以嘗試進行下列步驟來修正這個錯誤：
>
> - 以系統管理員身分執行程式。
> - 使用 [**控制台**] 中的 [**應用程式和功能**] 或 [**程式和功能**] 頁面修復或重新安裝程式。
> - 在 [**控制台**] 中選取 [軟體更新] **Windows Update** 。
> - 檢查應用程式的更新版本。 如果問題持續發生，請洽詢應用程式廠商。

**程式設計人員的資訊**

發生此錯誤的原因是應用程式呼叫主控台功能，但作業系統並未授與主控台的存取權。 除了在偵錯工具模式中，主控台函式通常不允許用於 Microsoft Store 應用程式。 如果您的應用程式需要系統管理員許可權才能執行，請確定它已安裝為預設以系統管理員身分執行。
