# C 執行階段錯誤 R6019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法開啟主控台裝置  
  
 程式呼叫 CONIO.h 中宣告的主控台 \(Console\) 函式，但是作業系統不允許存取此主控台。
