# C 執行階段錯誤 R6024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\_onexit\/atexit 表格空間不足  
  
 沒有足夠的記憶體供 `_onexit` 或 `atexit` 函式使用。  此錯誤是因記憶體不足所造成。
