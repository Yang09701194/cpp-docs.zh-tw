# C 執行階段錯誤 R6026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

空間不足以供 stdio 初始化使用  
  
 當記憶體不足，無法初始化標準 I\/O 支援時，就會發生這個錯誤。
