# C 執行階段錯誤 R6027
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

空間不足以供 lowio 初始化使用  
  
 當沒有足夠的記憶體可用來初始化低階 I\/O 支援時，就會出現這個錯誤。
