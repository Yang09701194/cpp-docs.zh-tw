# C 執行階段錯誤 R6031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

嘗試初始化 CRT 一次以上。這表示應用程式中有錯誤。  
  
 這項診斷表示在載入器鎖定期間執行了 MSIL 指令。  如需詳細資訊，請參閱[混合組件的初始化](../../dotnet/initialization-of-mixed-assemblies.md)。
