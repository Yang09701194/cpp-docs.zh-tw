# <a name="c-runtime-error-r6032"></a>C 執行階段錯誤 R6032

空間不足，無法用於地區設定資訊

> [!NOTE]
> 如果您在執行應用程式時遇到此錯誤訊息，則應用程式已關閉，因為它有內部記憶體問題。 此錯誤有幾個可能的原因，但通常是因為記憶體不足的狀況或程式中的錯誤所造成。
>
> 您可以嘗試進行下列步驟來修正這個錯誤：
>
> - 關閉其他正在執行的應用程式，或重新開機您的電腦以釋放記憶體。
> - 使用 [**控制台**] 中的 [**應用程式和功能**] 或 [**程式和功能**] 頁面修復或重新安裝程式。
> - 在 [**控制台**] 中選取 [軟體更新] **Windows Update** 。
> - 檢查應用程式的更新版本。 如果問題持續發生，請洽詢應用程式廠商。

**程式設計人員的資訊**

執行時間會維護每個執行緒的地區設定相關資訊，讓它可以處理區分地區設定函式的呼叫。 如果這項資訊的記憶體配置失敗，執行時間將無法繼續，因為它的基本功能太多而無法執行。
