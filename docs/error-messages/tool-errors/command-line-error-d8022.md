# <a name="command-line-error-d8022"></a>命令列錯誤 D8022

無法開啟 'messagefile'

指定的檔案已不在目前的目錄，或在 PATH 環境變數所指定的目錄中。 訊息檔案包含編譯器命令列語法和選項的簡短摘要。

此檔案移至目前的目錄或目前路徑中的目錄。 如果找不到此檔案，執行安裝程式，以將它從發佈磁碟複製。
