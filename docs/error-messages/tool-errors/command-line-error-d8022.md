# 命令列錯誤 D8022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法開啟 'messagefile'  
  
 指定的檔案不在目前的目錄或 PATH 環境變數所指定的目錄下。  訊息檔案包含編譯器命令列語法和選項的簡短摘要。  
  
 將此檔案移至目前的目錄或目前路徑下的目錄。  如果找不到此檔案，請執行安裝程式從安裝磁片中複製。
