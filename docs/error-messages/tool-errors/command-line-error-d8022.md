# <a name="command-line-error-d8022"></a>命令列錯誤 D8022

無法開啟 ' messagefile '

指定的檔案不在目前目錄中，或在 PATH 環境變數中指定的目錄中。 訊息檔案包含編譯器命令列語法和選項的簡短摘要。

將此檔案移至目前目錄或目前路徑中的目錄。 如果找不到這個檔案，請執行安裝程式，從散發磁碟複製該檔案。
