# <a name="command-line-error-d8036"></a>命令列錯誤 D8036

'/option ' 不允許使用多個原始檔

這些編譯器選項不能與多個來源檔案搭配使用：

- 命名元件檔案清單（/Fa）

- 重新命名物件檔案（/Fo）

- 建立不含本機變數的來源瀏覽器資訊（/Fr）

- 建立包含本機變數的來源瀏覽器資訊（/FR）
