# 命令列錯誤 D8036
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\/option' 不允許有多個原始程式檔  
  
 這些編譯器選項不能與多個原始程式檔一起使用：  
  
-   指定組譯碼清單檔的檔名 \(\/Fa\)  
  
-   重新指定目的檔 \(Object File\) 檔名 \(\/Fo\)  
  
-   建立不包含區域變數的原始程式瀏覽資訊 \(Browser Information，\/Fr\)  
  
-   建立包含區域變數的原始程式瀏覽資訊 \(\/FR\)
