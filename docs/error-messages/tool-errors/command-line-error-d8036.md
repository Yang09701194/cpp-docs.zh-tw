# <a name="command-line-error-d8036"></a>命令列錯誤 D8036
'/option' 不允許有多個原始程式檔  
  
 這些編譯器選項不能搭配多個原始程式檔：  
  
-   名稱的組件檔案清單 (/ Fa)  
  
-   重新命名物件檔案 (/ Fo)  
  
-   建立來源未含區域變數的瀏覽器資訊 (/ Fr)  
  
-   建立包含本機變數的來源瀏覽器資訊 (/ FR)
