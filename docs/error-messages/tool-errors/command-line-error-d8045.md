# <a name="command-line-error-d8045"></a>命令列錯誤 D8045

無法編譯 C 檔案 'file' 與 /clr 選項

只有 c + + 原始程式檔可以傳遞至使用編譯 **/clr**。  使用 **/TP**到.c 檔案編譯為.cpp 檔案，請參閱[/Tc、 /Tp、 /TC、 /TP （指定原始程式檔類型）](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md)如需詳細資訊。

如需詳細資訊，請參閱 [/clr (Common Language Runtime 編譯)](../../build/reference/clr-common-language-runtime-compilation.md)。

如果您使用 Visual c + + ATL 應用程式編譯，也會發生 D8045。 請參閱[如何： 移轉至 /clr](../../dotnet/how-to-migrate-to-clr.md)如需詳細資訊。
