# 命令列警告 D9024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法辨認的原始程式檔類型 'filename'，假設是目的檔  
  
 無法辨認指定檔案的副檔名。  假設此檔案為目的檔並被傳遞至連結器。  
  
 下列為可辨認的副檔名：  
  
-   .c \(C 原始程式檔）  
  
-   .cxx \(C\+\+ 原始程式檔\)  
  
-   .cpp \(C\+\+ 原始程式檔\)  
  
-   .obj \(目的檔\)  
  
-   .lib \(程式庫檔案\)  
  
-   .def \(模組定義檔\)  
  
-   .exp \(由 LINK \/LIB 建立的連結器匯出檔\)
