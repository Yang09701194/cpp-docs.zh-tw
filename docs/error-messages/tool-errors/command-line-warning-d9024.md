# <a name="command-line-warning-d9024"></a>命令列警告 D9024

無法辨識的來源檔案類型 ' filename '，物件檔已被假設

無法辨識指定檔案的副檔名。 檔案已假設為物件檔案，並已傳遞至連結器。

可辨識下列延伸模組：

- .c （C 原始檔）

- . .cxx （C++原始檔）

- .cpp （C++原始檔）

- .obj （物件檔案）

- .lib （程式庫檔案）

- .def （模組定義檔）

- exp （連結/LIB 所建立的連結器匯出檔案）
