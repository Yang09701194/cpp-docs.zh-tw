# <a name="command-line-warning-d9024"></a>命令列警告 D9024

無法辨識的來源檔案類型 'filename'、 物件檔案假設

無法辨識指定的檔案的副檔名。 檔案已假設為目的檔，並傳遞給連結器。

會辨識下列延伸模組：

- .c （C 原始程式檔）

- .cxx (C++原始程式檔)

- .cpp (C++原始程式檔)

- .obj （目的檔）

- .lib （程式庫檔案）

- .def （模組定義檔）

- .exp （連結器會匯出連結 /LIB 所建立的檔案）
