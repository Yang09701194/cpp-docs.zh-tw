# <a name="command-line-warning-d9024"></a>命令列警告 D9024
無法識別的來源檔案類型 'filename'，檔案假設是 object  
  
 無法辨識指定的檔案的副檔名。 檔案已假設為目的檔，而且已傳遞至連結器。  
  
 會辨識下列延伸項目：  
  
-   .c （C 原始程式檔）  
  
-   .cxx （c + + 原始程式檔）  
  
-   .cpp （c + + 原始程式檔）  
  
-   .obj （目的檔）  
  
-   .lib （程式庫檔案）  
  
-   .def （模組定義檔）  
  
-   .exp （連結器匯出連結 /LIB 所建立的檔案）
