# <a name="command-line-warning-d9028"></a>命令列警告 D9028

> 最小重建失敗，還原為一般組建

專案 .idb 檔案已損毀。 刪除檔案並重建。

如需最小重建和 .idb 檔案的詳細資訊，請參閱已淘汰的[/gm 編譯器選項](../../build/reference/gm-enable-minimal-rebuild.md)。
