# 命令列警告 D9028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

最少重建失敗，還原成一般建置  
  
 專案 .idb 檔案已損毀。  請刪除檔案並重建。  
  
 如需最少重建 \(Minimal Rebuild\) 以及 .idb 檔案的詳細資訊，請參閱 [\/Gm 編譯器選項](../../build/reference/gm-enable-minimal-rebuild.md)。
