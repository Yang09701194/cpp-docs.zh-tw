# <a name="command-line-warning-d9035"></a>命令列警告 D9035
選項 'option' 已被取代，未來版本將移除  
  
 您指定的編譯器選項，將 Visual c + + 編譯器的未來版本中移除。 這個警告後面警告[D9036](../../error-messages/tool-errors/command-line-warning-d9036.md)如果沒有建議的替代*選項*。  
  
 指定的選項仍能運作，但您應該更新現在的組建組態。 如此一來，您的專案是較容易升級編譯器時繼續建置。
