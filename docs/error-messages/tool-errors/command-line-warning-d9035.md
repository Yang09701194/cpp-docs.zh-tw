# 命令列警告 D9035
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

選項 'option' 已被取代，在未來的發行版本中將會移除  
  
 您指定了在未來版本 Visual C\+\+ 編譯器中會被移除的編譯器選項。  這項警告後面接著警告 [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md)，將提供您建議的替代 *option*。  
  
 指定的選項仍然可以用，但您應立即更新組建組態。  如此，您的專案在編譯器升級後才能繼續建置。
