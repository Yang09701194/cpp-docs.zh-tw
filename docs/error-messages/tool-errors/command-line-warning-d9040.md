# <a name="command-line-warning-d9040"></a>命令列警告 D9040

忽略選項 '/analyze';程式碼分析警告不適用於此版本的編譯器

**/Analyze**命令列選項不適用於所有版本的 Visual Studio。 若要補救這個警告，請切換到支援的 Visual Studio 版本，或移除命令列選項。
