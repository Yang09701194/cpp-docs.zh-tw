# 命令列警告 D9040
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

忽略選項 '\/analyze'，程式碼分析警告在這個版本的編譯器中無法使用  
  
 在 Visual Studio 的所有版本中 **\/analyze** 命令列選項是無法使用。  若要補救這個警告，請切換到 Visual Studio 所支援的版本，或移除命令列選項。
