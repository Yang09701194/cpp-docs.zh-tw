# <a name="command-line-warning-d9040"></a>命令列警告 D9040

忽略選項 '/analyze ';程式碼分析警告在這個版本的編譯器中無法使用

Visual Studio 的所有版本都無法使用 **/analyze**命令列選項。 若要補救這個警告，請切換到支援的 Visual Studio 版本，或移除命令列選項。
