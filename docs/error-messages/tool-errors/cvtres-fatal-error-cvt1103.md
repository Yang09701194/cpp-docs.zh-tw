# <a name="cvtres-fatal-error-cvt1103"></a>CVTRES 嚴重錯誤 CVT1103

無法讀取檔案名

CVTRES 嘗試讀取指定的檔案時發生無法復原的錯誤。

此錯誤可能是檔案損毀所造成。
