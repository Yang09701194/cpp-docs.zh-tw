# CVTRES 嚴重錯誤 CVT1103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法讀取檔案  
  
 當 CVTRES 嘗試讀取給定的檔案時發生無法回復的錯誤。  
  
 錯誤的原因可能是檔案損毀。
