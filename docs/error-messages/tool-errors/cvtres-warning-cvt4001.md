# <a name="cvtres-warning-cvt4001"></a>CVTRES 警告 CVT4001

> 未指定的機器類型假設*類型*

CVTRES 找不到電腦的規格。 它假設指定的電腦類型。 如果預設為不正確，然後重新執行 CVTRES 使用 /MACHINE 選項。
