# CVTRES 警告 CVT4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**未指定電腦類型; 假設其類型為**   
 ***type***  
  
 CVTRES 找不到電腦規格。  它會假設電腦為給定的電腦類型。  如果預設類型不正確，請使用 \/MACHINE 選項重新執行 CVTRES。
