# <a name="cvtres-warning-cvt4001"></a>CVTRES 警告 CVT4001

> 未指定電腦類型;假設的*類型*

CVTRES 找不到電腦規格。 它會假設指定的電腦類型。 如果預設值不正確，請使用/MACHINE 選項重新執行 CVTRES。
