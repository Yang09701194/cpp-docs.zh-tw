# <a name="expression-evaluator-error-cxx0002"></a>運算式評估工具錯誤 CXX0002
存取使用者記憶體時發生錯誤  
  
 運算式嘗試將參考未配置到偵錯程式的記憶體。  
  
 這個錯誤是與 can0002 相同。
