# 運算式評估工具錯誤 CXX0002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

存取使用者記憶體時發生錯誤  
  
 此運算式嘗試參考未配置給被偵錯程式的記憶體。  
  
 此錯誤與 CAN0002 相同。
