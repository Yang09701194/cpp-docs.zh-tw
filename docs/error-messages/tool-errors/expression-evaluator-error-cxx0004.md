# <a name="expression-evaluator-error-cxx0004"></a>運算式評估工具錯誤 CXX0004
語法錯誤  
  
 運算式的語法不正確。  
  
 重新輸入運算式以正確的語法。  
  
 這個錯誤是與 can0004 相同。
