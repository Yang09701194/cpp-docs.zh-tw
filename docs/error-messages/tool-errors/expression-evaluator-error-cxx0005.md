# 運算式評估工具錯誤 CXX0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不支援運算子  
  
 運算式中指定了不支援的 C 運算子。  使用受支援的 C 運算子來編寫相同的運算式。  
  
 此錯誤與 CAN0005 相同。
