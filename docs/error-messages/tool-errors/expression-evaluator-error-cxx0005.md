# <a name="expression-evaluator-error-cxx0005"></a>運算式評估工具錯誤 CXX0005

不支援的運算子

在運算式中指定了不支援的 C 運算子。 撰寫使用支援的 C 運算子的相等運算式。

此錯誤是與 can0005 相同。
