# <a name="expression-evaluator-error-cxx0006"></a>運算式評估工具錯誤 CXX0006

遺漏左括號

找不到運算式中的不對稱的括號。 重新輸入與對稱的括號運算式。

此錯誤是與 can0006 相同。
