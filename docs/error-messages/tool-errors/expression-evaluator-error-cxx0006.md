# 運算式評估工具錯誤 CXX0006
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

遺漏左括號  
  
 運算式中發現不對稱的括號。  重新以對稱的括號輸入此運算式。  
  
 此錯誤與 CAN0006 相同。
