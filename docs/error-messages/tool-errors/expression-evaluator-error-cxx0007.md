# 運算式評估工具錯誤 CXX0007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

遺漏右括號  
  
 運算式中發現不對稱的括號。  重新以對稱的括號輸入此運算式。  
  
 此錯誤與 CAN0007 相同。
