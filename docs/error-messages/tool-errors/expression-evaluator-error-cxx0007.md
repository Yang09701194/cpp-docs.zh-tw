# <a name="expression-evaluator-error-cxx0007"></a>運算式評估工具錯誤 CXX0007

缺少右括弧

運算式中找到不對稱的括弧。 以對稱的括弧重新輸入運算式。

此錯誤與 CAN0007 相同。
