# <a name="expression-evaluator-error-cxx0008"></a>運算式評估工具錯誤 CXX0008

**「 在字串結尾遺失**

缺少預期結尾的字串常值雙引號。 重新輸入運算式，字串常值雙引號括住。

此錯誤是與 can0008 相同。
