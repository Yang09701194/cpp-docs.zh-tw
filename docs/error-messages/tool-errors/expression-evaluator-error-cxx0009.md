# <a name="expression-evaluator-error-cxx0009"></a>運算式評估工具錯誤 CXX0009

遺漏 ' after 字元常數

遺漏字元常數結尾所預期的單引號。 重新輸入運算式，並以單引號括住字元常數。

此錯誤與 CAN0009 相同。
