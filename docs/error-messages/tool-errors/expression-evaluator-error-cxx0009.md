# <a name="expression-evaluator-error-cxx0009"></a>運算式評估工具錯誤 CXX0009
遺漏 ' 字元常數結尾處  
  
 遺漏結尾的字元常數必須是單引號。 請重新輸入運算式，並以單引號括住字元常數。  
  
 這個錯誤是與 can0009 相同。
