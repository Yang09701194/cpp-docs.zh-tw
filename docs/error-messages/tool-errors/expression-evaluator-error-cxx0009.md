# 運算式評估工具錯誤 CXX0009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

字元常數結尾處遺漏單引號 \('\)  
  
 預期出現在字元常數結尾處的單引號遺漏。  重新輸入運算式，將字元常數置於單引號之間。  
  
 此錯誤與 CAN0009 相同。
