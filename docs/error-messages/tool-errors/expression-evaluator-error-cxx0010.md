# 運算式評估工具錯誤 CXX0010
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

遺漏左方括弧  
  
 運算式中包含不對稱的方括弧。  重新以對稱的方括弧輸入運算式。  
  
 此錯誤與 CAN0010 相同。
