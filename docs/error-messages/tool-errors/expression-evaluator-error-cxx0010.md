# <a name="expression-evaluator-error-cxx0010"></a>運算式評估工具錯誤 CXX0010
遺漏左括號  
  
 運算式包含不對稱的方括號。 請重新輸入具有平衡方括號運算式。  
  
 這個錯誤是與 can0010 相同。
