# <a name="expression-evaluator-error-cxx0012"></a>運算式評估工具錯誤 CXX0012
遺漏左大括號  
  
 運算式包含不對稱的大括號。 請重新輸入具有對稱的大括號運算式。  
  
 這個錯誤是與 can0012 相同。
