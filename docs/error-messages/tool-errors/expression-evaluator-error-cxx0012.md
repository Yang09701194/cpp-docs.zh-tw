# 運算式評估工具錯誤 CXX0012
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

遺漏左大括號  
  
 運算式中包含不對稱的大括號。  重新以對稱的大括號輸入運算式。  
  
 此錯誤與 CAN0012 相同。
