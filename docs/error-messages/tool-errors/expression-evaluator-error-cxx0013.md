# <a name="expression-evaluator-error-cxx0013"></a>運算式評估工具錯誤 CXX0013
遺漏運算子  
  
 運算子必須在運算式中，但找不到。 請檢查運算式的語法。  
  
 這個錯誤是與 can0013 相同。
