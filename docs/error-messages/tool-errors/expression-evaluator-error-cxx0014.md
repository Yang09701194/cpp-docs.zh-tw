# <a name="expression-evaluator-error-cxx0014"></a>運算式評估工具錯誤 CXX0014
缺少運算元  
  
 必要的運算元不被指定的運算子。 請檢查運算式的語法。  
  
 這個錯誤是與 can0014 相同。
