# 運算式評估工具錯誤 CXX0014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

遺漏運算元  
  
 沒有為所指定的運算子提供需要的運算元。  請檢查運算式的語法。  
  
 此錯誤與 CAN0014 相同。
