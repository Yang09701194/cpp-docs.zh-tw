# <a name="expression-evaluator-error-cxx0015"></a>運算式評估工具錯誤 CXX0015

運算式太複雜 （堆疊溢位）

輸入的運算式是太複雜或巢狀層數太多量 C 運算式評估工具的可用儲存空間。

溢位，通常會發生因為太多暫止的計算。

重新排列的運算式，以便可以評估運算式的每個元件，會發現，而無需等待計算運算式的其他部分。

將運算式分割為多個命令。

此錯誤是與 can0015 相同。
