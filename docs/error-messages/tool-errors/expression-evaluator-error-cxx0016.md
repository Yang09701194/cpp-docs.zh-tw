# <a name="expression-evaluator-error-cxx0016"></a>運算式評估工具錯誤 CXX0016
常數太大  
  
 C 運算式評估工具無法接受大於 4294967295 (0FFFFFFFF 十六進位)，將不帶正負號的整數常數或浮點常數，其值必須大於大約 1.8 e + 308。  
  
 這個錯誤是與 can0016 相同。
