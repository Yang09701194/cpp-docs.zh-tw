# 運算式評估工具錯誤 CXX0016
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

常數太大  
  
 C 運算式評估工具無法接受大於 4,294,967,295 \(0FFFFFFFF 十六進位\) 的沒有帶正負號整數常數，或是大小超過約 1.8E\+308 的浮點常數。  
  
 此錯誤與 CAN0016 相同。
