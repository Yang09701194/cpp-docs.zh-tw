# <a name="expression-evaluator-error-cxx0016"></a>運算式評估工具錯誤 CXX0016

常數太大

C 運算式評估工具無法接受大於4294967295（0FFFFFFFF 十六進位）的不帶正負號整數常數，或其大小大於約 1.8 E + 308 的浮點常數。

此錯誤與 CAN0016 相同。
