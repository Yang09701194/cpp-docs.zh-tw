# <a name="expression-evaluator-error-cxx0018"></a>運算式評估工具錯誤 CXX0018

不正確的註冊名稱

指定的暫存器不存在或無法顯示。

監看式視窗可以顯示下列的暫存器：

AX 預存程序 DS IP

BX BP ES 奧蘭多市

CX SI SS GS

DX DI CS SS

EAX ESP DS EIP

EBX EBP ES EFL

ECX ESI FS

EDX EDI CS

此錯誤是與 can0018 相同。
