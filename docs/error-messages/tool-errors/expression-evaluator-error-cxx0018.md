# <a name="expression-evaluator-error-cxx0018"></a>運算式評估工具錯誤 CXX0018

不正確的暫存器名稱

指定的暫存器不存在或無法顯示。

監看式視窗可以顯示下列暫存器：

AX SP DS IP

BX BP ES FL

CX SI SS GS

DX DI CS SS

EAX ESP DS EIP

EBX 的 EBP ES EFL

ECX ESI FS

EDX EDI CS

此錯誤與 CAN0018 相同。
