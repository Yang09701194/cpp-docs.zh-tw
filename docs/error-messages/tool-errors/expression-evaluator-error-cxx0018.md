# 運算式評估工具錯誤 CXX0018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

暫存器名稱錯誤  
  
 指定的暫存器不存在或無法顯示。  
  
 \[監看式\] 視窗可以顯示下列暫存器：  
  
 AX    SP    DS    IP  
  
 BX    BP    ES    FL  
  
 CX    SI    SS    GS  
  
 DX    DI    CS    SS  
  
 EAX    ESP    DS    EIP  
  
 EBX    EBP    ES    EFL  
  
 ECX    ESI    FS  
  
 EDX    EDI    CS  
  
 此錯誤與 CAN0018 相同。
