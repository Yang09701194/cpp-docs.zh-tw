# <a name="expression-evaluator-error-cxx0020"></a>運算式評估工具錯誤 CXX0020

這種運算的運算元類型不正確

運算子已套用至具有該運算子之無效類型的運算式。

例如，取得暫存器位址或使用浮點運算式為數組加上標時，是不正確。

此錯誤與 CAN0020 相同。
