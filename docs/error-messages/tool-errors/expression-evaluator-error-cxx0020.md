# <a name="expression-evaluator-error-cxx0020"></a>運算式評估工具錯誤 CXX0020

這項作業的運算元類型錯誤

運算子套用到具有無效類型的運算式，該運算子。

比方說，不正確的註冊或下標陣列的浮點運算式的位址。

此錯誤是與 can0020 相同。
