# 運算式評估工具錯誤 CXX0020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這個運算的運算元類型錯誤  
  
 運算子套用至含有對該運算子而言無效的運算式。  
  
 例如，取得登錄的位址或是註標 \(Subscript\) 具有浮點運算式的陣列。  
  
 此錯誤與 CAN0020 相同。
