# <a name="expression-evaluator-error-cxx0020"></a>運算式評估工具錯誤 CXX0020
這項作業的運算元類型錯誤  
  
 運算子套用到具有無效的類型的運算式，該運算子。  
  
 例如，它不正確登錄或具有浮點運算式的陣列註標的位址。  
  
 這個錯誤是與 can0020 相同。
