# 運算式評估工具錯誤 CXX0021
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

當做純量使用 struct 或 union  
  
 運算式中使用了結構或等位，但是沒有指定項目。  
  
 操作結構或等位變數時，變數的名稱可能會自己出現，而沒有欄位限定詞。  如果運算式中使用了結構或等位，必須以希望的特定項目加以限定。  
  
 指定其值將用於運算式中的項目。  
  
 此錯誤與 CAN0021 相同。
