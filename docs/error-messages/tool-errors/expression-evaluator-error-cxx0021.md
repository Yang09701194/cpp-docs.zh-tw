# <a name="expression-evaluator-error-cxx0021"></a>運算式評估工具錯誤 CXX0021

當做純量使用的 struct 或 union

運算式中使用了結構或等位，但未指定任何元素。

當操作結構或等位變數時，變數的名稱本身可能會出現，而不會有欄位辨識符號。 如果運算式中使用了結構或等位，它必須以所需的特定元素來限定。

指定要在運算式中使用其值的元素。

此錯誤與 CAN0021 相同。
