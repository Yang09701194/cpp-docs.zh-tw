# <a name="expression-evaluator-error-cxx0021"></a>運算式評估工具錯誤 CXX0021
結構或等位做為純量  
  
 結構或等位已使用的運算式中，但是沒有任何項目指定。  
  
 當操作結構或等位變數，可能本身不含欄位限定詞會出現的變數名稱。 如果在運算式中使用結構或等位，它必須被限定所需的特定項目。  
  
 指定的值會在運算式中使用的項目。  
  
 這個錯誤是與 can0021 相同。
