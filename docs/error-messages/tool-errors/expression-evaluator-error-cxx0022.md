# <a name="expression-evaluator-error-cxx0022"></a>運算式評估工具錯誤 CXX0022
_main 之前呼叫函式  
  
 C 運算式評估工具無法評估函式，偵錯工具已進入函式之前**_main**。 程式未正確初始化直到**_main**已呼叫。  
  
 這個錯誤是與 can0022 相同。
