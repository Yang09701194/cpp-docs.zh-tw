# <a name="expression-evaluator-error-cxx0022"></a>運算式評估工具錯誤 CXX0022

_main 之前的函式呼叫

C 運算式評估工具在偵錯工具進入函數 **_main**之前，無法評估函式。 在呼叫 **_main**之前，程式未正確初始化。

此錯誤與 CAN0022 相同。
