# <a name="expression-evaluator-error-cxx0022"></a>運算式評估工具錯誤 CXX0022

函式呼叫之前 （_m）

C 運算式評估工具無法評估的函式，才能偵錯工具進入函式 **_main**。 程式未正確初始化直到 **_main**已呼叫。

此錯誤是與 can0022 相同。
