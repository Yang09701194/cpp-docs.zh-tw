# 運算式評估工具錯誤 CXX0022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在 \_main 之前呼叫函式  
  
 C 運算式評估工具無法在偵錯工具進入 **\_main** 函式前評估函式。  呼叫 **\_main** 前，程式都無法正確地初始化。  
  
 此錯誤與 CAN0022 相同。
