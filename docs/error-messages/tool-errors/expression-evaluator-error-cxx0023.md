# <a name="expression-evaluator-error-cxx0023"></a>運算式評估工具錯誤 CXX0023

不正確的基數

C 運算式評估工具無法辨識指定的基數。 只有十進位、 十六進位和八進位的基數是有效的。

此錯誤是與 can0023 相同。
