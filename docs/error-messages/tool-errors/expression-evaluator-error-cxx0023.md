# 運算式評估工具錯誤 CXX0023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

基數錯誤  
  
 C 運算式評估工具無法辨識指定的基數 \(Radix\)。  只有十進位、十六進位和八進位基數是有效的基數。  
  
 此錯誤與 CAN0023 相同。
