# <a name="expression-evaluator-error-cxx0025"></a>運算式評估工具錯誤 CXX0025
運算子必須結構/等位  
  
 會使用運算式的運算子`struct`或**union**型別已套用至運算式不是`struct`或**union**。  
  
 元件的類別、 結構或等位變數必須完整限定的名稱。 元件無法輸入沒有完整規格。  
  
 這個錯誤是與 can0025 相同。
