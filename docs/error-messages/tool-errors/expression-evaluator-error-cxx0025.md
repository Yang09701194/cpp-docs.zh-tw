# 運算式評估工具錯誤 CXX0025
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

運算子必須有 struct\/union  
  
 使用 `struct` 或 **union** 型別運算式的運算子，套用至不是 `struct` 或 **union** 的運算式中。  
  
 類別、結構或等位變數應該具有完整名稱。  不可輸入沒有完整規格的元件。  
  
 此錯誤與 CAN0025 相同。
