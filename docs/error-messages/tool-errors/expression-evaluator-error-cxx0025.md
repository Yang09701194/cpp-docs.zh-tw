# <a name="expression-evaluator-error-cxx0025"></a>運算式評估工具錯誤 CXX0025

運算子需要結構/等位

採用 `struct` 或聯**集**類型運算式的運算子，已套用至不是 **`struct` 或等**位的運算式。

Class、structure 或 union 變數的元件必須具有完整名稱。 無法在沒有完整規格的情況下輸入元件。

此錯誤與 CAN0025 相同。
