# <a name="expression-evaluator-error-cxx0026"></a>運算式評估工具錯誤 CXX0026

不正確的格式字串

未正確指定格式字串。 請檢查運算式的語法。

此錯誤是與 can0026 相同。

如需詳細資訊，請參閱 <<c0> [ 格式規範，在C++ ](/visualstudio/debugger/format-specifiers-in-cpp)。</c0>
