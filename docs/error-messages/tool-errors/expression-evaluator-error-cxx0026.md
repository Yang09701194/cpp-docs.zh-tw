# 運算式評估工具錯誤 CXX0026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

格式字串錯誤  
  
 所指定的格式字串 \(Format String\) 不正確。  請檢查運算式的語法。  
  
 此錯誤與 CAN0026 相同。  
  
 如需詳細資訊，請參閱[C\+\+ 中的格式規範](../Topic/Format%20Specifiers%20in%20C++.md)。
