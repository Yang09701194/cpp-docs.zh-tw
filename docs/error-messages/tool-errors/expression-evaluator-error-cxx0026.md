# <a name="expression-evaluator-error-cxx0026"></a>運算式評估工具錯誤 CXX0026

格式不正確的字串

未正確指定格式字串。 請檢查運算式的語法。

此錯誤與 CAN0026 相同。

如需詳細資訊，請參閱[中C++的格式](/visualstudio/debugger/format-specifiers-in-cpp)規範。
