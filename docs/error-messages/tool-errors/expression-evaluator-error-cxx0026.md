# <a name="expression-evaluator-error-cxx0026"></a>運算式評估工具錯誤 CXX0026

此錯誤是與 can0026 相同。

如需詳細資訊，請參閱 < [c + + 中的格式規範](/visualstudio/debugger/format-specifiers-in-cpp)。
