# 運算式評估工具錯誤 CXX0028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不是 struct\/union 項目  
  
 指定了 "Struct.Member" 或 "pStruct\-\>Member" 形式的運算式，但是 \<member\> 並不是該結構的項目。  
  
 運算式的括號可能不正確。  
  
 此錯誤與 CAN0028 相同。
