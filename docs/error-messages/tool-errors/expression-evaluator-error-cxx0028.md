# <a name="expression-evaluator-error-cxx0028"></a>運算式評估工具錯誤 CXX0028

not 結構/等位元素

已指定 ' Struct. Member ' 或 ' pStruct > Member ' 格式的運算式，但 \<成員 > 不是結構的元素。

運算式可能未正確地加上括弧。

此錯誤與 CAN0028 相同。
