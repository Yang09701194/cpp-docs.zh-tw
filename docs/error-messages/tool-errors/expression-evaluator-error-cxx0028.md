# <a name="expression-evaluator-error-cxx0028"></a>運算式評估工具錯誤 CXX0028
不 struct/union 項目  
  
 指定表單 'Struct.Member' 或 'Pstruct->member-成員 >' 的運算式，但\<成員 > 不是結構的項目。  
  
 運算式可能不正確括號括住。  
  
 這個錯誤是與 can0028 相同。
