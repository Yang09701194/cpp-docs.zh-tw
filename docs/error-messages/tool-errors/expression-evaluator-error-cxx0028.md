# <a name="expression-evaluator-error-cxx0028"></a>運算式評估工具錯誤 CXX0028

無法結構/等位的項目

指定表單 '了' 或 'pStruct-成員 >' 的運算式，但\<成員 > 不是結構的項目。

運算式不可能正確加上括號。

此錯誤是與 can0028 相同。
