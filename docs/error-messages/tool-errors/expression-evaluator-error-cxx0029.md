# 運算式評估工具錯誤 CXX0029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不是 struct 指標  
  
 成員選取運算子 \(**\-\>**\)套用至不是指向結構的運算式。  
  
 檢查整個運算式的括號是否正確，或是將位址運算式的型別轉換為適當的結構指標型別。  
  
 此錯誤與 CAN0029 相同。
