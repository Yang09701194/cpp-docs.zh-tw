# <a name="expression-evaluator-error-cxx0029"></a>運算式評估工具錯誤 CXX0029

not 結構指標

成員選取運算子（ **->** ）已套用至不是結構指標的運算式。

請檢查整個運算式是否已正確地加上括弧，或輸入將位址運算式轉換成適當的結構指標類型。

此錯誤與 CAN0029 相同。
