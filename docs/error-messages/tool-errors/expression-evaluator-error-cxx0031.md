# <a name="expression-evaluator-error-cxx0031"></a>運算式評估工具錯誤 CXX0031

無法評估運算式

運算式評估工具發生內部錯誤。

您可以撰寫可正確評估的對等運算式。

此錯誤是與 can0031 相同。
