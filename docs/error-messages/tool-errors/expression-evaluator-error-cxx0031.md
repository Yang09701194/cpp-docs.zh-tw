# 運算式評估工具錯誤 CXX0031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法評估運算式  
  
 運算式評估工具遇到內部錯誤。  
  
 您可以撰寫另一個可正確評估的相同運算式。  
  
 此錯誤與 CAN0031 相同。
