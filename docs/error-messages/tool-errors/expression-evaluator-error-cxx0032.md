# <a name="expression-evaluator-error-cxx0032"></a>運算式評估工具錯誤 CXX0032

除以0

運算式包含零的除數，這是不合法的。 這個除數可能是常值零，也可能是評估為零的運算式。

此錯誤與 CAN0032 相同。
