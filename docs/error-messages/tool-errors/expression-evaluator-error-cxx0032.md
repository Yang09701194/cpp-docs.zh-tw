# 運算式評估工具錯誤 CXX0032
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

除以 0  
  
 運算式中包含是零的除數，這是不合法的。  除數可能是常值數值零，或是可評估為零的運算式。  
  
 此錯誤與 CAN0032 相同。
