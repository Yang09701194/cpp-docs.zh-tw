# <a name="expression-evaluator-error-cxx0032"></a>運算式評估工具錯誤 CXX0032
除以 0  
  
 運算式包含除數為零，也就是不合法。 這個除數可以是常值的數字是零，或它可能是運算式評估為零。  
  
 這個錯誤是與 can0032 相同。
