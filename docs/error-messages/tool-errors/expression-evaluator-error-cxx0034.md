# 運算式評估工具錯誤 CXX0034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**型別與運算子不相容**  
  
 指定的運算元型別對此運算而言是不合法的型別。  
  
 例如，指標不可以與任何數值相乘。  
  
 您可能需要將運算元的型別轉換為與運算子相容的型別。
