# <a name="expression-evaluator-error-cxx0034"></a>運算式評估工具錯誤 CXX0034
**類型與運算子不相容**  
  
 指定的運算元類型不合法的操作。  
  
 例如，指標無法乘以任何值。  
  
 您可能需要輸入運算元與運算子相容的型別轉型。
