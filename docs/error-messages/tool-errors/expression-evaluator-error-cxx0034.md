# <a name="expression-evaluator-error-cxx0034"></a>運算式評估工具錯誤 CXX0034

**類型與運算子不相容**

指定的運算元類型對作業而言是不合法的。

例如，指標不能乘以任何值。

您可能需要輸入將運算元轉換成與運算子相容的類型。
