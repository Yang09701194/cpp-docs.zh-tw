# <a name="expression-evaluator-error-cxx0038"></a>運算式評估工具錯誤 CXX0038

函式引數的計數和/或類型不符

所指定的函式呼叫與函式原型不相符。

重新輸入正確的引數數目的呼叫。 類型轉換來比對的原型，視每個引數。

此錯誤是與 can0038 相同。
