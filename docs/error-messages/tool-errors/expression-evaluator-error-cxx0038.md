# <a name="expression-evaluator-error-cxx0038"></a>運算式評估工具錯誤 CXX0038

函數引數計數和/或類型不符

指定的函式呼叫與函數的原型不相符。

請以正確的引數數目重新輸入呼叫。 必要時，類型會轉換每個引數以符合原型。

此錯誤與 CAN0038 相同。
