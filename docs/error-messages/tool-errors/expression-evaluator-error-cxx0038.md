# 運算式評估工具錯誤 CXX0038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

函式引數計數和 \(或\) 型別不符  
  
 所指定的函式呼叫與函式的原型 \(Prototype\) 不符。  
  
 重新以正確數目的引數呼叫函式。  必要時，將每一個引數的型別轉換為與原型相符的型別。  
  
 此錯誤與 CAN0038 相同。
