# 運算式評估工具錯誤 CXX0040
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

函式要求隱含轉換  
  
 C 運算式評估工具不支援有關建構函式 \(Constructor\) 呼叫的隱含轉換。  
  
 此錯誤與 CAN0040 相同。
