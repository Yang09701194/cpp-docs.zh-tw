# <a name="expression-evaluator-error-cxx0040"></a>運算式評估工具錯誤 CXX0040

函式要求隱含轉換

C 運算式評估工具不支援包含建構函式呼叫的隱含轉換。

此錯誤是與 can0040 相同。
