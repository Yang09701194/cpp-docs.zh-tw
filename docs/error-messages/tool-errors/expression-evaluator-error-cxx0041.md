# <a name="expression-evaluator-error-cxx0041"></a>運算式評估工具錯誤 CXX0041

class 元素必須是靜態成員或成員函式

未指定要使用之類別的具現化時，會使用類別（或結構或等位）的非靜態成員。

只有靜態資料成員或成員函式可以使用，而不需指定具現化。

此錯誤與 CAN0041 相同。
