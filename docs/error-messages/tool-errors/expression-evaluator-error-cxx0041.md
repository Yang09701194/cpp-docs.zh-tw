# <a name="expression-evaluator-error-cxx0041"></a>運算式評估工具錯誤 CXX0041
類別項目必須是靜態成員或成員函式  
  
 非靜態成員的類別 （或結構或等位） 已使用但未指定的具現化類別的使用。  
  
 只有靜態資料成員或成員函式可以用於未指定具現化。  
  
 這個錯誤是與 can0041 相同。
