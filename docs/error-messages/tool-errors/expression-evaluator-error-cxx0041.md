# 運算式評估工具錯誤 CXX0041
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

類別項目必須是靜態成員或成員函式  
  
 使用類別 \(或結構或等位\) 的非靜態成員，但沒有指定使用類別的那一個執行個體化 \(Instantiation\)。  
  
 只有使用靜態資料成員或成員函式才可以不指定執行個體化。  
  
 此錯誤與 CAN0041 相同。
