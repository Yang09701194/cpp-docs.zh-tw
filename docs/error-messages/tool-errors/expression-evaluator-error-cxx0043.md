# 運算式評估工具錯誤 CXX0043
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在成員函式外部使用 this 指標  
  
 **this** 指標只能用於非靜態成員函式 \(Nonstatic Member Function\)。  
  
 此錯誤與 CAN0043 相同。
