# <a name="expression-evaluator-error-cxx0044"></a>運算式評估工具錯誤 CXX0044

使用 _based(void) 指標必須有 :> 運算子

指標根據`void`不能直接使用。 您必須形成完整的指標，使用 **: >** 運算子。

此錯誤是與 can0044 相同。
