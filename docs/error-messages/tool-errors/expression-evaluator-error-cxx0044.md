# 運算式評估工具錯誤 CXX0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

使用 \_based\(void\) 指標必須有 :\> 運算子  
  
 以 `void` 為基礎的指標不可以直接使用。  您必須使用 **:\>** 運算子，才是完整的指標。  
  
 此錯誤與 CAN0044 相同。
