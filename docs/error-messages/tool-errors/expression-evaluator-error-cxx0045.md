# <a name="expression-evaluator-error-cxx0045"></a>運算式評估工具錯誤 CXX0045

不是函式

在程式中為符號提供的引數清單不是函式的名稱。

## <a name="example"></a>範例

```
queue( alpha, beta )
```

當 `queue` 不是函數時。

此錯誤與 CAN0045 相同。
