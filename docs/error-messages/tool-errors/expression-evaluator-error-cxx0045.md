# <a name="expression-evaluator-error-cxx0045"></a>運算式評估工具錯誤 CXX0045

不是函式

符號不是函式名稱的程式中提供的引數清單。

## <a name="example"></a>範例

```
queue( alpha, beta )
```

當`queue`不是函式。

此錯誤是與 can0045 相同。
