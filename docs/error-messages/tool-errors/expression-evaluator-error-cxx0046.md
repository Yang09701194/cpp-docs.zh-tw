# 運算式評估工具錯誤 CXX0046
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

成員函式所需的引數清單  
  
 運算式呼叫成員函式，但是沒有指定任何實質參數。  
  
 此錯誤與 CAN0046 相同。
