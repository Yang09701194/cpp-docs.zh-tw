# <a name="expression-evaluator-error-cxx0046"></a>運算式評估工具錯誤 CXX0046
所需的成員函式的引數清單  
  
 運算式呼叫成員函式，但未指定任何實際的參數。  
  
 這個錯誤是與 can0046 相同。
