# <a name="expression-evaluator-error-cxx0047"></a>運算式評估工具錯誤 CXX0047

引數清單不符合函式

呼叫具有不符合任何函式的型式參數清單與程式中定義的相同名稱的實質參數清單的函式運算式。

只有當參數完全符合或不需要在物件建構的相符項目時，才可以呼叫多載函式。

此錯誤是與 can0047 相同。
