# 運算式評估工具錯誤 CXX0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

引數清單與函式不符  
  
 運算式以真實的參數清單呼叫函式，但是該參數清單與程式中定義之任何相同名稱函式的正規參數清單不符。  
  
 只有當參數完全符合或是配對時不需要物件建構時，才可以呼叫多載函式。  
  
 此錯誤與 CAN0047 相同。
