# <a name="expression-evaluator-error-cxx0047"></a>運算式評估工具錯誤 CXX0047

引數清單與函數不相符

運算式，其使用的實際參數清單不符合程式中所定義之任何函式的正式參數清單。

只有在有完全相符的參數，或不需要結構物件的對應時，才可以呼叫多載函式。

此錯誤與 CAN0047 相同。
