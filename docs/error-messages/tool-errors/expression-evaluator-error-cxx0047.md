# <a name="expression-evaluator-error-cxx0047"></a>運算式評估工具錯誤 CXX0047
引數清單不符合函式  
  
 具有相同名稱定義在程式中不符合任何函式的型式參數清單的實質參數清單的函式呼叫運算式。  
  
 沒有參數完全符合或不需要物件的建構的相符項目時，才可以呼叫多載函式。  
  
 這個錯誤是與 can0047 相同。
