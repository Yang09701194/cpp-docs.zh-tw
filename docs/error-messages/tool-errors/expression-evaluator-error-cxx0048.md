# <a name="expression-evaluator-error-cxx0048"></a>運算式評估工具錯誤 CXX0048

不支援的呼叫順序

運算式中指定的函式會使用以 C 運算式評估工具不支援的呼叫順序。 您無法呼叫此函式中的監看式 視窗的運算式。

此錯誤是與 can0048 相同。
