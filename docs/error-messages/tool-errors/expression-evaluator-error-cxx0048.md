# 運算式評估工具錯誤 CXX0048
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不支援呼叫順序  
  
 運算式中指定的函式使用 C 運算式評估工具不支援的呼叫順序 \(Calling Sequence\)。  您無法在 \[監看式\] 視窗中呼叫此函式。  
  
 此錯誤與 CAN0048 相同。
