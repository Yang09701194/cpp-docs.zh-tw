# 運算式評估工具錯誤 CXX0049
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

過時的 OMF \- 請重新連結程式  
  
 此程式使用舊的 OMF \(物件模組格式\)。  
  
 使用目前版本的連結器 \(Linker\) 重新連結程式。  
  
 此錯誤與 CAN0049 相同。
