# <a name="expression-evaluator-error-cxx0049"></a>運算式評估工具錯誤 CXX0049

過時的 OMF-請重新連結程式

程式使用了舊的 OMF （物件模組格式）。

使用目前的連結器版本重新連結程式。

此錯誤與 CAN0049 相同。
