# <a name="expression-evaluator-error-cxx0049"></a>運算式評估工具錯誤 CXX0049

過時的 OMF-請重新連結程式

程式會使用舊的 OMF （物件模組格式）。

重新連結使用目前的連結器版本的程式。

此錯誤是與 can0049 相同。
