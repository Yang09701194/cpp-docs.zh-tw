# 運算式評估工具錯誤 CXX0050
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

:: 的左方必須是 class\/struct\/union 型別  
  
 範圍解析運算子 \(`::`\) 左方的符號不是類別、結構或等位。  
  
 此錯誤與 CAN0050 相同。
