# <a name="expression-evaluator-error-cxx0050"></a>運算式評估工具錯誤 CXX0050
左邊的:: 必須是類別/結構/等位  
  
 範圍解析運算子左邊的符號 (`::`) 不是類別、 結構或等位。  
  
 這個錯誤是與 can0050 相同。
