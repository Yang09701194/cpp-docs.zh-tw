# 運算式評估工具錯誤 CXX0051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

中斷點指定一個以上的多載符號  
  
 運算式評估工具無法決定使用那一個多載符號做為中斷點。  
  
 此錯誤與 CAN0051 相同。
