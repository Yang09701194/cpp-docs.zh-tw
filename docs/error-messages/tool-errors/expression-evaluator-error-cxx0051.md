# <a name="expression-evaluator-error-cxx0051"></a>運算式評估工具錯誤 CXX0051

在中斷點中指定了一個以上的多載符號

運算式評估工具無法判斷要使用一個以上多載符號做為中斷點。

此錯誤與 CAN0051 相同。
