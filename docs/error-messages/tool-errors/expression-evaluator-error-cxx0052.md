# <a name="expression-evaluator-error-cxx0052"></a>運算式評估工具錯誤 CXX0052

成員函式不存在

已將成員函式指定為中斷點，但找不到。 在已內嵌的函式上設定中斷點，可能會導致此錯誤。

使用內嵌強制關閉（/Ob0）來重新編譯檔案，以在此函式中設定中斷點。

運算式，稱為未定義的函式。

此錯誤與 CAN0052 相同。
