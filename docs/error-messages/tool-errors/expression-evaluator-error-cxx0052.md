# <a name="expression-evaluator-error-cxx0052"></a>運算式評估工具錯誤 CXX0052

不存在的成員函式

成員函式為中斷點指定了，但找不到。 在已內嵌函式設定中斷點，可能會導致此錯誤。

重新編譯檔案內嵌強制關閉 (/ Ob0) 在此函式設定中斷點。

呼叫未定義的函式運算式。

此錯誤是與 can0052 相同。
