# 運算式評估工具錯誤 CXX0052
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

沒有顯示成員函式  
  
 將成員函式指定為中斷點，但是找不到此成員函式。  將中斷點設定在內嵌 \(Inline\) 的函式可能會造成此錯誤。  
  
 強制關閉內嵌 \(\/Ob0\) 將中斷點設定在此函式，重新編譯程式。  
  
 運算式呼叫的函式未定義。  
  
 此錯誤與 CAN0052 相同。
