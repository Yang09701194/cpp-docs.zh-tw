# <a name="expression-evaluator-error-cxx0053"></a>運算式評估工具錯誤 CXX0053
找到符號符合繫結中斷點時  
  
 做為中斷點的符號不是函式。 為中斷點指定的資料成員可能會造成這個錯誤。  
  
 這個錯誤是與 can0053 相同。
