# <a name="expression-evaluator-error-cxx0053"></a>運算式評估工具錯誤 CXX0053

系結中斷點時，nonfunction 符號相符

當做中斷點使用的符號不是函數。 將資料成員指定為中斷點可能會造成此錯誤。

此錯誤與 CAN0053 相同。
