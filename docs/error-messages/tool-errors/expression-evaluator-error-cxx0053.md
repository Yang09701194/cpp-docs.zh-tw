# 運算式評估工具錯誤 CXX0053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

繫結中斷點時找到相符的非函式符號  
  
 做為中斷點的符號不是函式。  將資料成員指定為中斷點會造成此錯誤。  
  
 此錯誤與 CAN0053 相同。
