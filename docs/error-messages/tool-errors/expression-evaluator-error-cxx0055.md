# <a name="expression-evaluator-error-cxx0055"></a>運算式評估工具錯誤 CXX0055
內容運算子中有模稜兩可的符號  
  
 內容運算子中的符號 (**{}**) 程式中的多個符號參考。  
  
 範圍解析運算子 (`::`) 可能無法解析模稜兩可。  
  
 這個錯誤是與 can0055 相同。
