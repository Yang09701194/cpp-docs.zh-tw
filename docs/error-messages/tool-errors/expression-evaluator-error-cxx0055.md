# <a name="expression-evaluator-error-cxx0055"></a>運算式評估工具錯誤 CXX0055

內容運算子中的符號不明確

內容運算子（ **{}** ）中的符號，參考程式中的一個以上的符號。

範圍解析運算子（`::`）可能能夠解決不明確的問題。

此錯誤與 CAN0055 相同。
