# 運算式評估工具錯誤 CXX0055
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

內容運算中模稜兩可的符號  
  
 在內容運算子中的符號 \(**{}**\) 參考程式中一個以上的符號。  
  
 範圍解析運算子 \(`::`\) 可能可以解決這種模稜兩可 \(Ambiguity\)。  
  
 此錯誤與 CAN0055 相同。
