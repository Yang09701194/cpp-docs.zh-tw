# <a name="expression-evaluator-error-cxx0057"></a>運算式評估工具錯誤 CXX0057

行號沒有程式碼

未針對指定的行號產生任何程式碼。 它不能當做中斷點使用。

此錯誤與 CAN0057 相同。
