# 運算式評估工具錯誤 CXX0057
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

此行號位置上沒有程式碼  
  
 指定的行號沒有產生程式碼。  不可以做為中斷點。  
  
 此錯誤與 CAN0057 相同。
