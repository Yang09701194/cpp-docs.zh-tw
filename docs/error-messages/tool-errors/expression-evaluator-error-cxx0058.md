# 運算式評估工具錯誤 CXX0058
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

找不到多載運算子  
  
 運算式中將類別型別指定為左運算元，但是該類別並沒有定義多載運算子 \(Overload Operator\)。  
  
 此錯誤與 CAN0058 相同。
