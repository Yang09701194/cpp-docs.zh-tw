# <a name="expression-evaluator-error-cxx0059"></a>運算式評估工具錯誤 CXX0059

左運算元的類別不是函數名稱

函數呼叫的左運算元是類別名稱，無法解析為函式呼叫。 省略運算式中的成員函式名稱可能會造成這個錯誤。

此錯誤與 CAN0059 相同。
