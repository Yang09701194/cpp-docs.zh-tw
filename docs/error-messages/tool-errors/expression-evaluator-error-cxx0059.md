# 運算式評估工具錯誤 CXX0059
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

左運算元是類別名稱，而不是函式名稱  
  
 函式呼叫的左運算元是類別名稱，但是無法解析為函式呼叫 \(Function Call\)。  省略運算式的成員函式名稱會造成此錯誤。  
  
 此錯誤與 CAN0059 相同。
