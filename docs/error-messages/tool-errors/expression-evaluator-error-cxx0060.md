# <a name="expression-evaluator-error-cxx0060"></a>運算式評估工具錯誤 CXX0060
無法使用暫存器  
  
 指定的暫存器不能使用的運算式。 這個錯誤可能被因嘗試存取不存在執行的電腦的暫存器。  
  
 這個錯誤是與 can0060 相同。
