# 運算式評估工具錯誤 CXX0060
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法使用暫存器  
  
 運算式指定了不可使用的暫存器。  企圖存取不存在於執行機器中的暫存器會造成此錯誤。  
  
 此錯誤與 CAN0060 相同。
