# <a name="expression-evaluator-error-cxx0060"></a>運算式評估工具錯誤 CXX0060

註冊無法使用

運算式指定了無法使用的暫存器。 嘗試存取不存在於執行之電腦上的暫存器，可能會導致此錯誤。

此錯誤與 CAN0060 相同。
