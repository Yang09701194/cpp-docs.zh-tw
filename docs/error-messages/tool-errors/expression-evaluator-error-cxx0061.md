# 運算式評估工具錯誤 CXX0061
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

超過函式巢狀結構的深度  
  
 運算式包含的函式巢狀結構深度超過限制。  修改運算式以減少巢狀結構深度。  
  
 此錯誤與 CAN0061 相同。
