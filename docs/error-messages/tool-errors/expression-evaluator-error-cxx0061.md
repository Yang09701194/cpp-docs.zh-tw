# <a name="expression-evaluator-error-cxx0061"></a>運算式評估工具錯誤 CXX0061

超過函數的嵌套深度

運算式包含的函數嵌套深度大於限制。 修改運算式以減少嵌套深度。

此錯誤與 CAN0061 相同。
