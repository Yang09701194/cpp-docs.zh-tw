# <a name="expression-evaluator-error-cxx0062"></a>運算式評估工具錯誤 CXX0062

不支援的函式呼叫

運算式呼叫了函數。 運算式無法對函式進行明確的呼叫，或進行需要呼叫某個函數的轉換。

此錯誤與 CAN0062 相同。
