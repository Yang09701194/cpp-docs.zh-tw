# 運算式評估工具錯誤 CXX0062
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不支援建構函式呼叫  
  
 運算式呼叫建構函式。  運算式無法明確呼叫建構函式，或進行需要呼叫建構函式的轉換。  
  
 此錯誤與 CAN0062 相同。
