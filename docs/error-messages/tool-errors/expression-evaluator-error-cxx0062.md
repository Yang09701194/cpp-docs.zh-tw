# <a name="expression-evaluator-error-cxx0062"></a>運算式評估工具錯誤 CXX0062

不支援的建構函式呼叫

運算式所做的建構函式的呼叫。 運算式無法進行明確呼叫建構函式，或進行需要建構函式呼叫的轉換。

此錯誤是與 can0062 相同。
