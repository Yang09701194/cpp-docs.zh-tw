# <a name="expression-evaluator-error-cxx0063"></a>運算式評估工具錯誤 CXX0063
不支援多載運算子 ->  
  
 運算式使用多載的類別成員存取運算子 (**->**)。  
  
 這個錯誤是與 can0063 相同。
