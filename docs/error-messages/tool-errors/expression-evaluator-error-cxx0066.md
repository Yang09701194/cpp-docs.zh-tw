# 運算式評估工具錯誤 CXX0066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

沒有顯示靜態成員  
  
 找不到類別的靜態成員，或是未定義類別的靜態成員。  已宣告但未定義的靜態類別成員，或是只在不包含偵錯資訊的模組中定義及參考的靜態類別成員都會造成此錯誤。  
  
 此錯誤與 CAN0066 相同。
