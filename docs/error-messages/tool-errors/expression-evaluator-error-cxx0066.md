# <a name="expression-evaluator-error-cxx0066"></a>運算式評估工具錯誤 CXX0066

靜態成員不存在

找不到類別的靜態成員，或未定義。 此錯誤可能會造成從靜態類別成員已宣告但未定義，或只定義和參考模組中，並包含偵錯資訊。

此錯誤是與 can0066 相同。
