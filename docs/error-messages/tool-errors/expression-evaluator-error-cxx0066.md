# <a name="expression-evaluator-error-cxx0066"></a>運算式評估工具錯誤 CXX0066

靜態成員不存在

找不到或未定義類別的靜態成員。 這個錯誤可能是由已宣告但未定義的靜態類別成員所造成，或是只在不包含偵錯工具資訊的模組中定義和參考。

此錯誤與 CAN0066 相同。
