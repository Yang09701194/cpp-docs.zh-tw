# 運算式評估工具錯誤 CXX0067
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不支援函式評估  
  
 運算式中包含函式呼叫。  某些運算式評估工具不支援函式呼叫。  
  
 此錯誤與 CAN0067 相同。
