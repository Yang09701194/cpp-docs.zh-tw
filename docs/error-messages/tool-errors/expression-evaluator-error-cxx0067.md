# <a name="expression-evaluator-error-cxx0067"></a>運算式評估工具錯誤 CXX0067

不支援函數評估

運算式包含函式呼叫。 某些運算式評估工具不支援函式呼叫。

此錯誤與 CAN0067 相同。
