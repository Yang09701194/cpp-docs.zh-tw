# <a name="expression-evaluator-error-cxx0069"></a>運算式評估工具錯誤 CXX0069

變數需要堆疊框架

運算式評估工具無法評估變數，因為它不會發生在堆疊框架中。 這可能是因為宣告為內嵌函式一部分的變數所造成。
