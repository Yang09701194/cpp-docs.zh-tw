# 運算式評估工具錯誤 CXX0069
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

變數必須有堆疊框架  
  
 運算式評估工具無法評估變數，因為它沒有出現在堆疊框架中。  可能是因為將變數宣告為內嵌 \(Inline\) 函式的一部分所造成。
