# 運算式評估工具錯誤 CXX0072
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

遺漏型別資訊或未知的型別資訊  
  
 無法連結 .pch 檔案，或是程式碼參考到未以 \/Zi 編譯之模組中的型別。  
  
 此錯誤與 CAN0067 相同。
