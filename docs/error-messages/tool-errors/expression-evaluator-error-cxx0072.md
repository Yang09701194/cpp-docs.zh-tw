# <a name="expression-evaluator-error-cxx0072"></a>運算式評估工具錯誤 CXX0072
錯誤： 類型資訊遺漏或未知  
  
 .Pch 檔案無法連結中，或是程式碼的參考不是使用 /Zi 編譯模組中的類型。  
  
 這個錯誤是與 can0067 相同。
