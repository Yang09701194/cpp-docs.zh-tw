# <a name="expression-evaluator-error-cxx0072"></a>運算式評估工具錯誤 CXX0072

錯誤：遺漏或未知的類型資訊

未在中連結 .pch 檔案，或程式碼參考到不是使用/Zi. 編譯之模組中的類型

此錯誤與 CAN0067 相同。
