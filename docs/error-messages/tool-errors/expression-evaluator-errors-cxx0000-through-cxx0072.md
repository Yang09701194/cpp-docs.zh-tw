# 運算式評估工具錯誤 CXX0000 至 CXX0072
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

本章節提供組建工具所產生的錯誤之參考資訊。  如需取得特定錯誤訊息的說明，請在 \[輸出\] 視窗中按一下錯誤代碼並按下 F1，或在 \[索引\] 的 \[尋找\] 方塊中輸入錯誤代碼。
