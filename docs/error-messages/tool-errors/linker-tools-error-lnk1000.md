# 連結器工具錯誤 LNK1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未知的錯誤；請查閱文件以取得技術支援選項的相關資訊  
  
 記錄錯誤發生的情況，設法找出問題癥結，並建立可重現的測試案例，然後再聯繫 `Microsoft Product Support Services`。  如需調查並回報錯誤的詳細資訊，請參閱[http:\/\/support.microsoft.com\/default.aspx?scid\=kb;en\-us;134650](http://support.microsoft.com/default.aspx?scid=kb;en-us;134650)。  
  
 如果混合標準的標頭檔 \(例如 dos.h\) 和您自己的檔案，也可能會收到這項錯誤訊息。  請先以 `#include` 包含標準的標頭，後面再接您自己的標頭檔。
