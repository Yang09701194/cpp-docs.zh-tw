# <a name="linker-tools-error-lnk1000"></a>連結器工具錯誤 LNK1000
未知的錯誤。如需技術支援選項，請參閱文件  
  
 請注意錯誤的情況下，嘗試找出問題並建立可重現的測試案例，然後再連絡`Microsoft Product Support Services`。 如何調查，並報告這些錯誤的資訊，請參閱[ http://support.microsoft.com/default.aspx?scid=kb; en-us-我們; 134650](http://support.microsoft.com/default.aspx?scid=kb;en-us;134650)。  
  
 如果您混用標準標頭檔 (例如，dos.h) 與您自己的檔案，您可能會收到這個錯誤。 `#include` 標準標頭，接著再設定您自己的標頭檔。
