# <a name="linker-tools-error-lnk1103"></a>連結器工具錯誤 LNK1103

調試資訊已損毀;重新編譯模組

造成此錯誤的原因可能是編譯在建立有效的物件檔案之前終止。

重新編譯指定的物件檔案。 如果重新編譯無法更正問題，請注意錯誤的狀況，嘗試找出問題所在，並建立可重現的測試案例。 然後參閱技術支援說明檔，或其中一個手冊中的技術支援一節。
