# 連結器工具錯誤 LNK1103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

偵錯資訊損毀；請重新編譯模組  
  
 這項錯誤可能會因編譯在建立有效目的檔以前被結束而引發。  
  
 重新編譯指定的目的檔。  如果重新編譯仍無法解決問題，請記下出現該錯誤的情況，嘗試隔離問題並且建立可重現的測試案例。  然後參閱技術支援說明檔，或手冊中的技術支援章節。
