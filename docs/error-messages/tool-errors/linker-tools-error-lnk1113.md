# <a name="linker-tools-error-lnk1113"></a>連結器工具錯誤 LNK1113
無效的電腦類型  
  
 物件的標頭中指定的電腦類型無效。  
  
 如果檔案已損毀，可能造成這個錯誤。 重建物件。
