# 連結器工具錯誤 LNK1113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無效的電腦類型 type  
  
 目的檔標頭中的電腦指定無效。  
  
 如果檔案已損毀，就可能會造成這項錯誤。  請重建該目的檔。
