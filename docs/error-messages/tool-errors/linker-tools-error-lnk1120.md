# 連結器工具錯誤 LNK1120
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

number 個無法在外部程式庫中找到或連結所需之資料型別或函式  
  
 錯誤 LNK1120 讓您知道這個連結無法在外部之程式庫中找到或連結所需之資料型別或函式的計數 \(`number`\)。  造成這種情況的條件由出現在這個錯誤訊息 \(每次錯誤就會出現一次\) 之前的錯誤 [LNK2001](../../error-messages/tool-errors/linker-tools-error-lnk2001.md) 說明。  
  
 如需詳細資訊，請參閱 [http:\/\/support.microsoft.com\/default.aspx?scid\=kb;en\-us;815650](http://support.microsoft.com/default.aspx?scid=kb;en-us;815650)。
