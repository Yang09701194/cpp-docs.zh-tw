# <a name="linker-tools-error-lnk1120"></a>連結器工具錯誤 LNK1120
*數字*無法解析外部符號  
  
錯誤 LNK1120 報告計數 (*數目*) 的未解析外部符號錯誤，此連結作業。 大部分無法解析外部符號錯誤，會個別地報告[連結器工具錯誤 LNK2001](../../error-messages/tool-errors/linker-tools-error-lnk2001.md)和[連結器工具錯誤 LNK2019](../../error-messages/tool-errors/linker-tools-error-lnk2019.md)，這在之前的這個錯誤訊息，針對每個無法解析的外部執行一次符號時發生錯誤。  
  
若要修正這個錯誤，更正所有的其他未解析外部的錯誤或其他組建輸出中前面的連結器錯誤。 任何未解析的外部錯誤不存在時，不會報告此錯誤。
