# 連結器工具錯誤 LNK1136
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

檔案無效或已損毀  
  
 輸入檔案可能出現標頭損毀、大小為零或異常地小等情形。  
  
 如果檔案不正確地指定為其他相依性，LNK1136 也有可能會發生。
