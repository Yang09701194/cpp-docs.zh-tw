# <a name="linker-tools-error-lnk1136"></a>連結器工具錯誤 LNK1136
檔案無效或損毀  
  
 輸入的檔標頭損毀或大小為零或異常地小。  
  
 如果檔案未正確指定做為其他相依性，也可能會造成 LNK1136。
