# <a name="linker-tools-error-lnk1136"></a>連結器工具錯誤 LNK1136

檔案無效或損毀

輸入檔的標頭損毀，或大小為零或不太小。

如果檔案不正確地指定為額外的相依性，也可能會導致 LNK1136。
