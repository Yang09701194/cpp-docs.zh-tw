# 連結器工具錯誤 LNK1141
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

建置匯出檔時失敗  
  
 [LINK](../../build/reference/linker-command-line-syntax.md) 無法建置匯出 \(.exp\) 檔案。  
  
 發生這種錯誤的原因，包括記憶體不足以及和選項或模組定義陳述式中含有語法錯誤等等。  
  
 在這個錯誤出現之前，會先出現一個提供其他資訊的錯誤。
