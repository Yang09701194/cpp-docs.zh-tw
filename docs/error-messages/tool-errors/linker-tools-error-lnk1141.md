# <a name="linker-tools-error-lnk1141"></a>連結器工具錯誤 LNK1141

建置匯出檔期間發生失敗

[連結](../../build/reference/linker-command-line-syntax.md)無法建立匯出 (.exp) 檔案。

此錯誤的原因包括選項或模組定義陳述式中的系統問題，例如沒有足夠的記憶體和語法錯誤。

此錯誤被加上另一個可提供其他資訊的錯誤。
