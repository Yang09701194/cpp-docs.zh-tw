# <a name="linker-tools-error-lnk1141"></a>連結器工具錯誤 LNK1141

匯出檔案的組建期間失敗

[連結](../../build/reference/linking.md)無法建立匯出（.exp）檔案。

造成此錯誤的原因包括系統問題，例如選項或模組定義語句中的記憶體不足和語法錯誤。

此錯誤前面會有另一個錯誤，提供其他資訊。
