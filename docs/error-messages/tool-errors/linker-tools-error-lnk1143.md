# 連結器工具錯誤 LNK1143
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

檔案無效或損毀：沒有 COMDAT 區段 number 的符號  
  
 如果物件檔損毀，可能造成這個錯誤。  
  
 請重建專案。
