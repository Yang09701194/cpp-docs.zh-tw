# <a name="linker-tools-error-lnk1143"></a>連結器工具錯誤 LNK1143
檔案無效或損毀： 沒有 COMDAT 區段數字的符號  
  
 如果目的檔已損毀，可能造成這個錯誤。  
  
 重建該檔案。
