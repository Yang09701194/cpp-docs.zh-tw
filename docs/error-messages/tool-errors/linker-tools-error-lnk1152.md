# 連結器工具錯誤 LNK1152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法解析一個或多個未裝飾符號  
  
 在這個錯誤之前，會有一個針對每個無法解析的未裝飾符號所發出的 [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) 警告，和至少兩個針對未裝飾符號之重複符號所發出的 [LNK4002](../../error-messages/tool-errors/linker-tools-warning-lnk4002.md) 警告。
