# <a name="linker-tools-error-lnk1152"></a>連結器工具錯誤 LNK1152

無法解析一或多個未修飾符號

這個錯誤前面會有一個警告[LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) ，用於無法解析的每個未修飾符號，以及至少有兩個警告[LNK4002](../../error-messages/tool-errors/linker-tools-warning-lnk4002.md)針對未修飾符號找到的重複符號。
