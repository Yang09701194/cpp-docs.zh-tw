# <a name="linker-tools-error-lnk1164"></a>連結器工具錯誤 LNK1164

區段區段對齊（數位）大於/ALIGN 值

物件檔中給定區段的對齊大小超過以[/align](../../build/reference/align-section-alignment.md)選項指定的值。 **/Align**值必須是2的乘冪，而且必須等於或超過物件檔中指定的區段對齊。

請以較小的區段對齊方式重新編譯，或增加 **/align**值。
