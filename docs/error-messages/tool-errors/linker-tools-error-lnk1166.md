# <a name="linker-tools-error-lnk1166"></a>連結器工具錯誤 LNK1166
無法調整程式碼在位移 = 位移，va = 值  
  
 連結器無法填補所需的程式碼。  
  
 某些指示不允許某些處理器跨頁面界限。 新增填補更正這種情況下嘗試連結。 在此情況下，連結可能無法解決此問題。
