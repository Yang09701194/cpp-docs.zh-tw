# <a name="linker-tools-error-lnk1166"></a>連結器工具錯誤 LNK1166

無法調整位移的程式碼 = 位移，va = value

連結器無法填補所需的程式碼。

若要跨頁面界限在某些處理器上不允許特定指示。 新增填補，若要更正這種情況下嘗試連結。 在此情況下，連結可能無法解決此問題。
