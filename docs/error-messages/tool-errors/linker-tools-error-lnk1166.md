# <a name="linker-tools-error-lnk1166"></a>連結器工具錯誤 LNK1166

無法調整位移 = offset，va = 值的程式碼

連結無法視需要填補程式碼。

某些指示不允許在某些處理器上交叉分頁界限。 連結嘗試新增 pad 以更正這種情況。 在此情況下，連結無法解決此問題。
