# <a name="linker-tools-error-lnk1169"></a>連結器工具錯誤 LNK1169

一或多個已定義找到的符號

組建失敗，因為多個定義的一或多個符號。 此錯誤會加上錯誤[LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md)。

[/Force](../../build/reference/force-force-file-output.md)或是 **/force: multiple 都會**選項會覆寫這個錯誤。
