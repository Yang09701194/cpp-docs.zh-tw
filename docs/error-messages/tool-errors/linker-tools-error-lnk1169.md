# 連結器工具錯誤 LNK1169
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

找到一或多個已定義的符號  
  
 因為一個或數個符號有多個定義而使得該版本失敗。  在這個錯誤之前會出現 [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md) 錯誤。  
  
 [\/FORCE](../../build/reference/force-force-file-output.md) 或 **\/FORCE:MULTIPLE** 選項將覆寫這個錯誤。
