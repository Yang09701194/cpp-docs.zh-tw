# <a name="linker-tools-error-lnk1169"></a>連結器工具錯誤 LNK1169
一或多個已定義的符號  
  
 建置失敗，因為多個定義的一個或多個符號。 這個錯誤加上錯誤[發生 LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md)。  
  
 [/強制](../../build/reference/force-force-file-output.md)或 **/force: multiple 都會**選項會覆寫這個錯誤。
