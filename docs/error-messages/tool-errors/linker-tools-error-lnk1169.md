# <a name="linker-tools-error-lnk1169"></a>連結器工具錯誤 LNK1169

找到一個或多個已定義的符號

組建失敗，因為一個或多個符號有多個定義。 此錯誤前面會出現錯誤[LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md)。

[/Force](../../build/reference/force-force-file-output.md)或 **/force： MULTIPLE**選項會覆寫這個錯誤。
