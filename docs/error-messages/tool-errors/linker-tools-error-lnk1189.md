# <a name="linker-tools-error-lnk1189"></a>連結器工具錯誤 LNK1189
LIBTOOMANYMEMBERS::  已超出程式庫的物件數目限制  
  
 已超出程式庫中的 65535 個物件或成員的限制。  
  
 若要修正這個問題，請將程式庫分成兩個以上具有較少物件或成員的程式庫。
