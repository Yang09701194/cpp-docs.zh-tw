# 連結器工具錯誤 LNK1215
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

中繼資料作業失敗 \(HRESULT\)：error  
  
 連結器在嘗試透過 .NET Runtime 進行中繼資料 \(Metadata\) 更新時接收到 .NET Runtime 發出的錯誤訊息。  
  
 `HRESULT` 是 .NET 執行階段方法的 HRESULT。  `error` 是 .NET 提供的文字。  
  
 您可能擁有不相符的連結器和 .NET Runtime；請重新安裝 Visual C\+\+。
