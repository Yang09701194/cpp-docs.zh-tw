# <a name="linker-tools-error-lnk1215"></a>連結器工具錯誤 LNK1215

中繼資料作業失敗（HRESULT）：錯誤

連結器在嘗試透過 .NET 執行時間進行中繼資料更新時，從 .NET 執行時間收到錯誤。

`HRESULT` 是 .NET 執行時間方法中的 HRESULT。 `error` 是。NET 提供的文字。

您可能有不相符的連結器和 .NET 執行時間;重新安裝C++視覺效果。
