# 連結器工具錯誤 LNK1218
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

警告視為錯誤處理，未產生輸出檔  
  
 當您以 **\/WX** 連結時，任何連結器警告都將視為錯誤，而且將不會建立輸出檔。  
  
 如需詳細資訊，請參閱[\/WX \(將連結器警告視為錯誤\)](../../build/reference/wx-treat-linker-warnings-as-errors.md)。
