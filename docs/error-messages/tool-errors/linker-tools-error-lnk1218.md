# <a name="linker-tools-error-lnk1218"></a>連結器工具錯誤 LNK1218

將警告視為錯誤;未產生輸出檔

當您使用 **/wx**連結時，任何連結器警告都會被視為錯誤，而且不會建立任何輸出檔。

如需詳細資訊，請參閱[/wx （將連結器警告視為錯誤）](../../build/reference/wx-treat-linker-warnings-as-errors.md) 。
