# <a name="linker-tools-error-lnk1218"></a>連結器工具錯誤 LNK1218

將警告視為錯誤;未產生輸出檔

當您連結 **/WX**，會將任何連結器警告視為錯誤，將會建立任何輸出檔案。

請參閱[/WX （連結器警告視為錯誤）](../../build/reference/wx-treat-linker-warnings-as-errors.md)如需詳細資訊。
