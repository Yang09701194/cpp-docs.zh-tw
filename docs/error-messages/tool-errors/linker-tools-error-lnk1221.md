# <a name="linker-tools-error-lnk1221"></a>連結器工具錯誤 LNK1221

無法推斷子系統，且必須加以定義

連結器沒有足夠的資訊來推斷您想要作為目標的子系統。

若要修正此錯誤，請使用[/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)。
