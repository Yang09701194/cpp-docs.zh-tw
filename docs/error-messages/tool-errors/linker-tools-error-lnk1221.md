# 連結器工具錯誤 LNK1221
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法推斷子系統且必須定義子系統  
  
 連結器沒有足夠的資訊來推斷您要將哪個子系統當做目標。  
  
 若要修復這個錯誤，請使用[\/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)。
