# <a name="linker-tools-error-lnk1221"></a>連結器工具錯誤 LNK1221
無法推斷子系統，而且必須定義  
  
 連結器沒有足夠的資訊來推斷哪個子系統做為目標。  
  
 若要修正這個錯誤，請使用[/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)。
