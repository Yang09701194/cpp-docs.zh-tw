# <a name="linker-tools-error-lnk1224"></a>連結器工具錯誤 LNK1224

無效的映像的基底位址

您指定無效的基底映像位址。 基底位址必須是 64KB 對齊 （最後四個十六進位數字必須是零），基底映像必須納入 32 位元帶正負號或不帶正負號值。
