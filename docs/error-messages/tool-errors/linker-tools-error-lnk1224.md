# <a name="linker-tools-error-lnk1224"></a>連結器工具錯誤 LNK1224

不正確映射基底位址

您指定了不正確映射基底位址。 基底位址必須對齊64KB （最後四個十六進位數位必須為零），而且映射基底必須符合32位帶正負號或不帶正負號的值。
