# <a name="linker-tools-error-lnk1224"></a>連結器工具錯誤 LNK1224
無效的映像基底地址  
  
 您指定的映像的基底位址。 基底位址必須是 64 KB 對齊 （最後四個十六進位數字必須是零） 和 32 位元帶正負號或不帶正負號值必須符合基底映像。
