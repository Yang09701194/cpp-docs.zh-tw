# <a name="linker-tools-error-lnk1240"></a>連結器工具錯誤 LNK1240
無法編譯 IDL 內容  
  
 連結器產生 MIDL 編譯內嵌的 IDL 但時發生問題。 請檢查由 MIDL 所指定的錯誤。
