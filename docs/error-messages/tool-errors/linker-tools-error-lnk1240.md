# 連結器工具錯誤 LNK1240
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法編譯 IDL 內容  
  
 連結器繁衍 \(Spawn\) MIDL 來編譯內嵌的 IDL，但發生了問題。  請檢查 MIDL 所指出的錯誤。
