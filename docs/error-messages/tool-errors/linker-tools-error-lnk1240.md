# <a name="linker-tools-error-lnk1240"></a>連結器工具錯誤 LNK1240

無法編譯 IDL 內容

連結器繁衍編譯內嵌的 IDL MIDL，但發生問題。 檢查由 MIDL 所指定的錯誤。
