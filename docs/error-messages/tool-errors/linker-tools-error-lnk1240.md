# <a name="linker-tools-error-lnk1240"></a>連結器工具錯誤 LNK1240

無法編譯 IDL 內容

連結器產生 MIDL 以編譯內嵌 IDL，但發生問題。 檢查 MIDL 指定的錯誤。
