# <a name="linker-tools-error-lnk1241"></a>連結器工具錯誤 LNK1241

資源檔資源檔案已指定

如果您執行，則會產生這個錯誤**cvtres**手動從命令列，如果您將已產生的.obj 檔案連結器除了其他.res 檔案。

若要指定多個.res 檔案，將它們全部加入.res 檔作為連結器，不從.obj 檔案建立**cvtres**。
