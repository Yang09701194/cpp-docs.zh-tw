# <a name="linker-tools-error-lnk1241"></a>連結器工具錯誤 LNK1241
資源檔已指定的資源檔  
  
 如果您執行會產生這個錯誤**cvtres**以手動方式從命令列，如果您再傳遞產生.obj 檔案連結器除了其他.res 檔案。  
  
 若要指定多個.res 檔案，將其傳遞至為.res 檔案連結器，所有不從.obj 檔案所建立**cvtres**。
