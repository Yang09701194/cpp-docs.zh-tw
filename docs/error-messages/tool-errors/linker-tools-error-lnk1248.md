# <a name="linker-tools-error-lnk1248"></a>連結器工具錯誤 LNK1248

> 影像大小 ('*大小*') 超過最大容許大小 (80000000)

連結器會決定輸出檔的大小會超過最大可能的 32 位元程式映像的大小。 若要讓程式到多個 Dll。
