# <a name="linker-tools-error-lnk1248"></a>連結器工具錯誤 LNK1248

> 影像大小（「*大小*」）超過允許的大小上限（80000000）

連結器判定輸出檔案的大小會超過32位程式映射的最大可能大小。 您可能想要將您的程式設為多個 Dll。
