# <a name="linker-tools-error-lnk1282"></a>連結器工具錯誤 LNK1282

無法/REBASE 檔案;已簽署

您嘗試使用[editbin](../../build/reference/editbin-reference.md)的/REBASE 選項來變更已簽署元件的基底位址。 若要這麼做，請先變更基底位址，然後簽署元件。
