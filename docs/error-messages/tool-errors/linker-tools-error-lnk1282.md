# 連結器工具錯誤 LNK1282
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法 \/REBASE 檔案，已經簽署  
  
 您嘗試使用 [editbin](../../build/reference/editbin-reference.md) 的 \/REBASE 選項變更已簽署組件的基底位址。  若要如此做，請先變更基底位址，然後再對組件簽署。
