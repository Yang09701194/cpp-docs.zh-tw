# <a name="linker-tools-error-lnk1282"></a>連結器工具錯誤 LNK1282
無法 /REBASE 檔案;已簽署  
  
 您嘗試要變更的 /REBASE 選項簽署的組件的基底位址[editbin](../../build/reference/editbin-reference.md)。 若要這樣做，請先變更基底地址，然後簽署組件。
