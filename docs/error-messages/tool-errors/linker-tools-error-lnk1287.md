# <a name="linker-tools-error-lnk1287"></a>連結器工具錯誤 LNK1287

無效的 managed 的進入點函式

進入點的受管理的映像無效。 Managed 的進入點函式的傳回型別只能`void`或`int`。 Managed 的進入點函式參數的型別只能`void`或`String []`。
