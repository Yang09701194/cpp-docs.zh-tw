# <a name="linker-tools-error-lnk1287"></a>連結器工具錯誤 LNK1287

不正確受控進入點函式

進入點對受控映射無效。 Managed 進入點函式的傳回型別只能 `void` 或 `int`。 Managed 進入點函式的參數類型只能 `void` 或 `String []`。
