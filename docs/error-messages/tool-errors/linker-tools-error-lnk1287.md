# 連結器工具錯誤 LNK1287
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無效的 Managed 進入點 function  
  
 進入點對 Managed 影像無效。  Managed 進入點函式的傳回型別只能為 `void` 或 `int`。  Managed 進入點函式的參數型別只能為 `void` 或 `String []`。
