# 連結器工具錯誤 LNK1296
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法載入 filename  
  
 無法使用指定的 DLL。  
  
 DLL 的可能位置是目前目錄、系統目錄、Windows 目錄和 PATH 環境變數所指定的目錄。
