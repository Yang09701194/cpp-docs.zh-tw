# <a name="linker-tools-error-lnk1296"></a>連結器工具錯誤 LNK1296

無法載入檔案名

給定的 DLL 無法使用。

DLL 的可能位置為目前目錄、系統目錄、Windows 目錄，以及 PATH 環境變數中指定的目錄。
