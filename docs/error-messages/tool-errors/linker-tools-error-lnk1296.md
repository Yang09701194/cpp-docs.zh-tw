# <a name="linker-tools-error-lnk1296"></a>連結器工具錯誤 LNK1296
無法載入檔案名稱  
  
 無法取得所指定的 DLL。  
  
 DLL 的可能位置是目前的目錄、 系統目錄、 Windows 目錄中和在 PATH 環境變數中指定的目錄。
