# <a name="linker-tools-error-lnk1314"></a>連結器工具錯誤 LNK1314

COFF 符號表已損毀或無效（未定義的靜態或標籤符號）

指定物件中的區段之編譯器產生的內容似乎已損毀。

此錯誤可能表示編譯器失敗。
