# 連結器工具錯誤 LNK1314
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

COFF 符號表損毀或無效 \(未定義的靜態或標籤符號\)  
  
 指定目的檔內由編譯器產生之區段內容顯示已損毀。  
  
 這項錯誤可能是代表編譯器失敗。
