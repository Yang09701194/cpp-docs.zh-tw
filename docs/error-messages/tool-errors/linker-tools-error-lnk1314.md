# <a name="linker-tools-error-lnk1314"></a>連結器工具錯誤 LNK1314

損毀或無效 COFF 符號表 （未定義的靜態或標籤符號）

編譯器產生的內容 > 一節中指定的物件似乎已損毀。

此錯誤可能表示編譯器失敗。
