# 連結器工具錯誤 LNK1332
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

偵測到 \<count\> Windows 執行階段類型已匯入一個模組並定義於另一個模組  
  
 當它會產生目前目標， 連結器偵測到 \<`count`\> Windows 執行階段型別，每一個模組在另一個模組匯入同時定義。  
  
### 更正這個錯誤  
  
-   根據錯誤訊息的建議修正每個組建的 LNK2039 錯誤。
