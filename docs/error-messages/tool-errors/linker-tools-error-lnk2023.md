# <a name="linker-tools-error-lnk2023"></a>連結器工具錯誤 LNK2023

錯誤的 dll 或進入點 \<dll 或進入點 >

連結器載入了不正確的 msobj90 版本。 請確定您路徑中的連結 .exe 和 msobj90 具有相同的版本。

可能不存在 msobj90 的相依性。 Msobj90 的相依性清單如下：

- Msvcr90 .dll

- Kernel32.dll

檢查您的電腦是否有任何其他可能已過期的 msobj90 複本。
