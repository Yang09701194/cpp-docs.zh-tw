# <a name="linker-tools-error-lnk2026"></a>連結器工具錯誤 LNK2026
模組對 SAFESEH 映像不安全  
  
 [/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md)已指定，但模組不相容於安全例外狀況處理功能。 如果您想要使用此模組具有**/SAFESEH**，則您必須重新編譯的模組。
