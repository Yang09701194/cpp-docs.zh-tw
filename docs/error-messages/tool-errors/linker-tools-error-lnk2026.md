# <a name="linker-tools-error-lnk2026"></a>連結器工具錯誤 LNK2026

模組不安全的 SAFESEH 映射

已指定[/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) ，但模組與安全例外狀況處理功能不相容。 如果您想要將此模組與 **/SAFESEH**搭配使用，則必須重新編譯模組。
