# 連結器工具警告 LNK4002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

symbol 已定義於 object 之中  
  
 以裝飾形式顯示的符號在 `object` 中指定了未裝飾形式，但是找不到唯一符合的裝飾符號。  在這個警告之前通常會出現一個警告 [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md)，之後則會出現嚴重錯誤 [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md)。
