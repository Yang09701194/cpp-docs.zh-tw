# <a name="linker-tools-warning-lnk4002"></a>連結器工具警告 LNK4002

在物件中定義的符號

以其裝飾形式，顯示的符號已指定未裝飾的形式在`object`，但找不到唯一的裝飾符號的相符項目。 這個警告會一律加上警告[LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md)且後面接著嚴重錯誤[LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md)。
