# <a name="linker-tools-warning-lnk4002"></a>連結器工具警告 LNK4002

物件中定義的符號

符號（以其裝飾形式顯示）在 `object`中以未修飾的形式指定，但找不到符合裝飾符號的唯一相符項。 此警告的前面一律會出現警告[LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) ，後面接著嚴重錯誤[LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md)。
