# <a name="linker-tools-warning-lnk4002"></a>連結器工具警告 LNK4002
物件中定義的符號  
  
 中的未裝飾形式中指定的符號，以其裝飾形式，顯示`object`，但找不到唯一的裝飾符號的相符項目。 這項警告一律加上警告[LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md)和嚴重錯誤[LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md)。
