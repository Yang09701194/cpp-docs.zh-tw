# <a name="linker-tools-warning-lnk4010"></a>連結器工具警告 LNK4010

不正確子系統版本號碼號碼;假設是預設子系統版本

您可以指定映射子系統的版本（[/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)）。 每個子系統都具有最低版本需求。 如果指定的版本低於最小值，則會發生此警告，而且連結器只會使用預設子系統。
