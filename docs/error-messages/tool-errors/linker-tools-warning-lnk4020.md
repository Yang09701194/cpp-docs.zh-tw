# <a name="linker-tools-warning-lnk4020"></a>連結器工具警告 LNK4020

> 中的類型記錄 '*filename*' 已損毀; 某些符號和類型可能無法從偵錯工具

PDB 檔案*filename*已損毀的類型記錄。

此問題通常是次要其他組建問題;這是第一個報告的組建問題，除非處理其他錯誤和警告第一次。 如果這是第一個回報的問題，您可能需要清除組建目錄並重建您的專案。 如果您使用平行建置程序，請參閱 < 當您序列化您的組建時，如果持續發生錯誤。
