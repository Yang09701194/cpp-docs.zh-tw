# <a name="linker-tools-warning-lnk4039"></a>連結器工具警告 LNK4039

區段以 /SECTION 選項指定的 ' name' 不存在

[DUMPBIN](../../build/reference/dumpbin-reference.md)或是[EDITBIN](../../build/reference/editbin-reference.md)找不到一個小節稱為`name`輸入檔中。

若要查看檔案中的區段，執行與 DUMPBIN [/HEADERS](../../build/reference/headers.md)選項。
