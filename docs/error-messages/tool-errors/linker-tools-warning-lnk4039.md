# 連結器工具警告 LNK4039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

以 \/SECTION 選項指定的區段 'name' 不存在  
  
 [DUMPBIN](../../build/reference/dumpbin-reference.md) 或 [EDITBIN](../../build/reference/editbin-reference.md) 在輸入檔中找不到名為 `name` 的區段。  
  
 若要檢視檔案中的區段，請使用 [\/HEADERS](../../build/reference/headers.md) 選項執行 DUMPBIN。
