# <a name="linker-tools-warning-lnk4044"></a>連結器工具警告 LNK4044
無法辨認的選項 'option';忽略  
  
 將指定的選項不是有效的選項，這項工具。 此工具會忽略選項。  
  
 若要查看有效的選項清單，請執行不含引數或選項的工具。
