# <a name="linker-tools-warning-lnk4044"></a>連結器工具警告 LNK4044

無法辨識的選項 ' option ';忽略

指定的選項不是此工具的有效選項。 此工具已忽略選項。

若要查看有效選項的清單，請執行不含引數或選項的工具。
