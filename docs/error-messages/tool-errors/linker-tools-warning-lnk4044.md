# 連結器工具警告 LNK4044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法辨認的選項 'option'；已忽略  
  
 指定的選項不是工具的有效選項。  工具已忽略該選項。  
  
 若要檢視有效選項的清單，請在沒有引數或選項的狀態下執行該工具。
