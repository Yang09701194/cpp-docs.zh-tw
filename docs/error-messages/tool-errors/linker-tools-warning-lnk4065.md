# <a name="linker-tools-warning-lnk4065"></a>連結器工具警告 LNK4065

' function ' 無法排序;忽略

指定的函式未編譯為封裝的函式。 使用[/gy](../../build/reference/gy-enable-function-level-linking.md)進行重新編譯。
