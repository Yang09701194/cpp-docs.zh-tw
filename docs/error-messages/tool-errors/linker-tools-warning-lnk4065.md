# 連結器工具警告 LNK4065
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法對 'function' 排序；已忽略  
  
 指定函式並未編譯成封裝函式。  請使用 [\/Gy](../../build/reference/gy-enable-function-level-linking.md) 重新編譯。
