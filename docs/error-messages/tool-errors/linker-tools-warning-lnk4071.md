# <a name="linker-tools-warning-lnk4071"></a>連結器工具警告 LNK4071

無法以累加方式連結於在後續的連結

連結中找到多個定義一或多個符號，但[/force](../../build/reference/force-force-file-output.md)或 **/force: multiple 都會**用來建立輸出檔，不論錯誤為何。 刪除連結的累加狀態 (.ilk) 檔案。
