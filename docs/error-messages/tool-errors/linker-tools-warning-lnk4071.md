# <a name="linker-tools-warning-lnk4071"></a>連結器工具警告 LNK4071

在後續的連結上無法以累加方式連結

連結找到一個或多個符號的多個定義，但是[/force](../../build/reference/force-force-file-output.md)或 **/FORCE：多個**是用來建立輸出檔，而不是錯誤。 連結已刪除累加狀態（. .ilk）檔案。
