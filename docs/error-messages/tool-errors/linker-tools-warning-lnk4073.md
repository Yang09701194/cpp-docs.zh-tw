# 連結器工具警告 LNK4073
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法建立 .ILK 檔的映像；非累加連結中  
  
 共用記憶體沒有足夠的連續空間，無法供 LINK 建立累加狀態 \(.ilk\) 檔。  LINK 會進行非累加建置。
