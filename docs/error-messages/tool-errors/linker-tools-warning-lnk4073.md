# <a name="linker-tools-warning-lnk4073"></a>連結器工具警告 LNK4073
無法建立.ilk 檔; 的對應非累加連結中  
  
 沒有大型足夠的連續空間中的連結，以建立累加狀態 (.ilk) 檔案的共用記憶體。 連結會執行非累加建置。
