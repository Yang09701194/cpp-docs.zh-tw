# <a name="linker-tools-warning-lnk4073"></a>連結器工具警告 LNK4073

無法建立 .ilk 檔案的對應;非增量連結

共用記憶體中的連續空間不足，無法建立增量狀態（. .ilk）檔案。 連結已執行非累加組建。
