# <a name="linker-tools-warning-lnk4076"></a>連結器工具警告 LNK4076
無效的累加狀態檔 'filename';非累加連結中  
  
 累加狀態 (.ilk) 檔案無法寫入連結。 任一`filename`損毀或不是累加連結的資料庫。 請移除檔案後重新連結。
