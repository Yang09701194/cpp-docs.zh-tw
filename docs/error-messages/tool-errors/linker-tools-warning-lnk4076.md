# <a name="linker-tools-warning-lnk4076"></a>連結器工具警告 LNK4076

增量狀態檔案 ' filename ' 無效;非增量連結

連結無法寫入增量狀態（. .ilk）檔案。 可能是 `filename` 已損毀，或它不是增量連結資料庫。 移除檔案並重新連結。
