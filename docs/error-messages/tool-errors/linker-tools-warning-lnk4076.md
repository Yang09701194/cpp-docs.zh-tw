# 連結器工具警告 LNK4076
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無效的累加狀態檔 'filename'；非累加連結中  
  
 LINK 無法寫入累加狀態 \(.ilk\) 檔。  `filename` 已損毀或並非累加連結資料庫。  請移除檔案並重新連結。
