# <a name="linker-tools-warning-lnk4076"></a>連結器工具警告 LNK4076

無效的累加狀態檔 'filename';非累加連結

連結是以累加的狀態 (.ilk) 檔案無法寫入。 任一`filename`損毀，或它不是累加連結的資料庫。 移除檔案並重新連結。
