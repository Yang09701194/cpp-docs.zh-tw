# <a name="linker-tools-warning-lnk4096"></a>連結器工具警告 LNK4096

/BASE 值 "number" 對 Windows 95 和 Windows 98 無效;映射可能無法執行

您指定的基底位址無效。 Windows 95 和 Windows 98 可執行檔的基底位址必須大於0x400000。 如需基底位址的詳細資訊，請參閱[/base](../../build/reference/base-base-address.md)連結器選項。
