# 連結器工具警告 LNK4096
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/BASE 值 'number' 對 Windows 95 和 Windows 98 無效；映像可能無法執行  
  
 您指定的基底位址無效。  Windows 95 和 Windows 98 可執行檔的基底位址必須大於 0x400000。  如需基底位址的詳細資訊，請參閱 [\/BASE](../../build/reference/base-base-address.md) 連結器選項。
