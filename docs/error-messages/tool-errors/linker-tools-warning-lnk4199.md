# <a name="linker-tools-warning-lnk4199"></a>連結器工具警告 LNK4199

/DELAYLOAD:dllname 忽略;沒有從 dllname 的匯入

連結器會忽略`dllname`因為不需要任何函式，`dllname`匯出。
