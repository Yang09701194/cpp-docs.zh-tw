# 連結器工具警告 LNK4199
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/DELAYLOAD:dllname 已忽略；找不到來自 dllname 的匯入  
  
 連結器會忽略 `dllname`，因為不需要 `dllname` 所匯出的任何函式。
