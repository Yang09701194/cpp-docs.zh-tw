# <a name="linker-tools-warning-lnk4199"></a>連結器工具警告 LNK4199

/DELAYLOAD：已忽略 dllname;從 dllname 找不到任何匯入

連結器會忽略 `dllname`，因為它不需要任何 `dllname` 匯出的函式。
