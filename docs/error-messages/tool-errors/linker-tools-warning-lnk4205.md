# <a name="linker-tools-warning-lnk4205"></a>連結器工具警告 LNK4205

'filename' 遺失用於參考模組; 目前的偵錯資訊如同沒有偵錯資訊般連結物件

.Pdb 檔案中有過時資訊。 連結器會繼續連結物件，而不偵錯資訊。 您可能想要重新編譯物件檔案使用[/Zi](../../build/reference/z7-zi-zi-debug-information-format.md)選項。
