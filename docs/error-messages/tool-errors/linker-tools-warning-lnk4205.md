# <a name="linker-tools-warning-lnk4205"></a>連結器工具警告 LNK4205

' filename ' 缺少參考模組目前的偵錯工具資訊;連結化物件，如同沒有任何調試資訊

.Pdb 檔案具有過期的資訊。 連結器將會繼續連結化物件，而不會有任何 debug 資訊。 您可能想要使用[/zi](../../build/reference/z7-zi-zi-debug-information-format.md)選項重新編譯物件檔案。
