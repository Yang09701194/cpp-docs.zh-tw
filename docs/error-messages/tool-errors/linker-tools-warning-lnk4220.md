# <a name="linker-tools-warning-lnk4220"></a>連結器工具警告 LNK4220

無效的 '連結器選項' 值 'value';假設預設值

利用指定的範圍外的值[/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md)選項。 預設值 **/TLBID**為 1。
