# 連結器工具警告 LNK4220
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無效的 'linker option' 值 'value'；假設預設值  
  
 [\/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md) 選項指定了超出範圍的值。  **\/TLBID** 的預設值為 1。
