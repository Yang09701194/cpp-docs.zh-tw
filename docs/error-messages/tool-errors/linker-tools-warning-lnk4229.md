# <a name="linker-tools-warning-lnk4229"></a>連結器工具警告 LNK4229

找到不正確指示詞/directive;忽略

透過[批註](../../preprocessor/comment-c-cpp.md)pragma 傳遞的指示詞無效。 連結器會忽略 `/directive`。
