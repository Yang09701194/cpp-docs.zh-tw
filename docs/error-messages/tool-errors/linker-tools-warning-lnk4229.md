# 連結器工具警告 LNK4229
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

出現無效的指示詞 \/directive；已忽略  
  
 透過 [comment](../../preprocessor/comment-c-cpp.md) pragma 傳遞的指示詞無效。  連結器已忽略 `/directive`。
