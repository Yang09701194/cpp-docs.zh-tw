# <a name="linker-tools-warning-lnk4229"></a>連結器工具警告 LNK4229

找到無效的指示詞 /directive已忽略

透過傳遞指示詞[註解](../../preprocessor/comment-c-cpp.md)pragma 無效。 連結器會忽略`/directive`。
