# <a name="linker-tools-warning-lnk4229"></a>連結器工具警告 LNK4229
找到無效的指示詞 /directive忽略  
  
 透過傳遞指示詞[註解](../../preprocessor/comment-c-cpp.md)pragma 不正確。 連結器會忽略`/directive`。
