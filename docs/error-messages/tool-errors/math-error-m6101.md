# <a name="math-error-m6101"></a>運算錯誤 M6101
無效  
  
 無效的作業。  
  
 一個運算元為 NaN （不是數字） 或無限大時，就可能導致這個錯誤。  
  
 程式終止，結束代碼為 129。
