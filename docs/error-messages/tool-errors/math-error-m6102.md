# <a name="math-error-m6102"></a>運算錯誤 M6102

異常

作業會產生非常小的浮點數，這是因為無效的精確度遺失。 異常的浮點例外狀況通常會遮罩，造成設陷及運算。

程式結束，結束代碼為 130。
