# <a name="math-error-m6102"></a>運算錯誤 M6102
異常  
  
 作業會產生非常小的浮點數，這是由於無效的精確度遺失。 異常的浮點例外狀況通常已遮罩，讓他們截取，在其上作業。  
  
 程式終止，結束代碼 130。
