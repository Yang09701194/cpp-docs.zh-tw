# <a name="math-error-m6102"></a>運算錯誤 M6102

denormal

作業產生非常小的浮點數，這是不正確重要性遺失。 Denormal 的浮點例外狀況通常會進行遮罩，使其被攔截和操作。

程式終止，結束代碼為130。
