# 運算錯誤 M6102
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

反正規化  
  
 作業產生非常小的浮點數 \(Floating\-Point Number\)，因數字精確度喪失 \(Loss of Significance\) 而無效。  反正規化 \(Denormal\) 浮點例外狀況 \(Exception\) 通常都會被遮罩，讓它們可以被截取及操作。  
  
 程式以結束代碼 130 結束。
