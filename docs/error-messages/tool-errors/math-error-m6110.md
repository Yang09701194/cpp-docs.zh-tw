# <a name="math-error-m6110"></a>運算錯誤 M6110

堆疊溢位

浮點運算式造成浮點堆疊溢位。

堆疊溢位浮點例外狀況會限制上限是七個層級，除了通常受到 8087/287/387 副處理器的八個層級。

程式結束，結束代碼為 138。
