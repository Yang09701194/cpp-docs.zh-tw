# <a name="math-error-m6110"></a>運算錯誤 M6110

堆疊溢位

浮點運算式造成浮點堆疊溢位。

除了8087/287/387 副處理器所支援的八個層級以外，堆疊溢位的浮點例外狀況會被攔截到七個層級的限制。

程式終止，結束代碼為138。
