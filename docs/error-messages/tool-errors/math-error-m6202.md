# <a name="math-error-m6202"></a>運算錯誤 M6202

'function': 錯誤

指定的函式的引數是此函式的獨一性值。 該引數未定義的函式。

這個錯誤呼叫`_matherr`函式名稱、 其引數，與錯誤類型的函式。 您可以重新撰寫`_matherr`函式來自訂特定執行階段浮點數學錯誤的處理。
