# <a name="math-error-m6205"></a>運算錯誤 M6205
'function': _TLOSS 錯誤  
  
 發生總遺失精確度 （有效位數）。  
  
 這個錯誤可能是因為 sin、 運算元，提供非常大量 cos 或 tan 因為運算元必須減少為介於 0 到 2 之間的數字 * pi。
