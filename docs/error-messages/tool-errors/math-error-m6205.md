# <a name="math-error-m6205"></a>運算錯誤 M6205

' function '： _TLOSS 錯誤

總損失（有效位數）已發生。

這項錯誤的原因可能是因為運算元必須縮減為介於0和 2 * pi 之間的數位，所以會提供非常大的數位做為 sin、cos 或 tan 的運算元。
