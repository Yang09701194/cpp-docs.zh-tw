# <a name="nmake-fatal-error-u1000"></a>NMAKE 嚴重錯誤 U1000

> 語法錯誤: ')' 巨集引動過程中遺失

左括號 **(**，沒有相符的右括號中，已出現 **)**，巨集引動過程中。 正確的格式是 **$(**<em>名稱</em>**)**;**$** <em>n</em>允許一個字元的名稱。
