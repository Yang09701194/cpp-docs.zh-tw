# <a name="nmake-fatal-error-u1000"></a>NMAKE 嚴重錯誤 U1000
語法錯誤: ')' 巨集引動過程中遺失  
  
 左括號， **(**、 出現但沒有相符的右括號， **)**，巨集引動過程中。 正確的格式是**$(***名稱***)**;`$`  *n* 所允許之字元的名稱。
