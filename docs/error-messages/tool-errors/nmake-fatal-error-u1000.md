# NMAKE 嚴重錯誤 U1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

語法錯誤: 巨集引動過程中遺漏 '\)'  
  
 在巨集引動過程中出現一個左括號 **\(**，但沒有對稱的右括號 **\)**。  正確的格式是 **$\(***name***\)**；`$`*n* 可以是單一字元的名稱。
