# <a name="nmake-fatal-error-u1000"></a>NMAKE 嚴重錯誤 U1000

> 語法錯誤：宏調用中遺漏 '） '

在宏調用中，左**括弧（** ）出現時，沒有對應的右括弧， **）** 。 正確的格式為 **$ （** <em>name</em> **）** ;單一字元名稱允許 **$** <em>n</em> 。
