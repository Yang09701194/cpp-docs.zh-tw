# <a name="nmake-fatal-error-u1007"></a>NMAKE 嚴重錯誤 U1007

名稱中不允許使用雙引號

指定的目標名稱或檔案名包含雙引號（ **"** ）。

雙引號可以括住檔案名，但不能包含在其中。
