# NMAKE 嚴重錯誤 U1007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不允許在名稱中使用雙引號  
  
 指定的目標名稱或檔名中包含雙引號 \(**"**\)。  
  
 雙引號可以在檔名前後，但是不可以包含在檔名中。
