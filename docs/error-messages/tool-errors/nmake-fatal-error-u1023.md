# NMAKE 嚴重錯誤 U1023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

運算式語法錯誤  
  
 運算式無效。  
  
 檢查允許的運算子及運算子優先順序 \(Operator Precedence\)。
