# <a name="nmake-fatal-error-u1052"></a>NMAKE 嚴重錯誤 U1052

> 找不到檔案 '*filename*'

NMAKE 找不到使用下列其中一項所指定的檔案：

- **/F**選項

- **!INCLUDE**前置處理指示詞

- 回應檔的 At sign （ **\@** ）規範

檢查檔案是否存在，以及檔案名是否拼寫正確。
