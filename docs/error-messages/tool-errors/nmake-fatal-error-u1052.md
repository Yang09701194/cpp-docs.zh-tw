# <a name="nmake-fatal-error-u1052"></a>NMAKE 嚴重錯誤 U1052

> 檔案 '*filename*' 找不到

NMAKE 找不到與下列其中一項指定的檔案：

- **/F**選項

- **!包含**前置處理器指示詞

- At 符號 (**\@**) 規範的回應檔案

請確認檔案存在，檔案名稱的拼字正確。
