# NMAKE 嚴重錯誤 U1052
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

找不到檔案 'filename'  
  
 NMAKE 找不到下列指定的檔案：  
  
-   **\/F** 選項  
  
-   **\!INCLUDE** 前置處理器指示詞  
  
-   回應檔 \(Response File\) 的 **@** 符號規範  
  
 檢查檔案是否存在及檔名的拼字是否正確。
