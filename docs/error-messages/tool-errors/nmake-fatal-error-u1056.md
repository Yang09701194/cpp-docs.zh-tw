# <a name="nmake-fatal-error-u1056"></a>NMAKE 嚴重錯誤 U1056

找不到命令處理器

中指定的路徑不是命令處理器**COMSPEC**或是**路徑**環境變數。

NMAKE 使用 COMMAND.COM 或 cmd。為執行命令時在命令處理器的 EXE。 它會尋找命令處理器先在設定的路徑**COMSPEC**。 如果**COMSPEC**不存在，在指定的目錄的 NMAKE 搜尋**路徑**。
