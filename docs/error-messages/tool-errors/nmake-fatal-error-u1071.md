# <a name="nmake-fatal-error-u1071"></a>NMAKE 嚴重錯誤 U1071

目標 'targetname' 的相依性樹狀結構中循環

指定之目標的相依性樹狀結構中，有循環相依性。 指定的目標是其中一個從屬參照的指定之目標的相依性。 循環的相依性無效。
