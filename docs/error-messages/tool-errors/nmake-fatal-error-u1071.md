# NMAKE 嚴重錯誤 U1071
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

目標 'targetname' 的相依樹狀結構產生循環  
  
 在相依樹狀結構中存在著給定目標的循環相依性。  給定的目標是給定目標相依的相依之一。  循環相依性是無效的。
