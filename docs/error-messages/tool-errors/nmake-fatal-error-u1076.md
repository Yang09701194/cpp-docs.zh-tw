# NMAKE 嚴重錯誤 U1076
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

名稱太長  
  
 字串超過下列限制之一：  
  
-   巨集名稱超過 1024 個字元。  
  
-   目標路徑名稱超過 256 個字元。  
  
-   命令超過 2048 個字元。
