# <a name="nmake-fatal-error-u1076"></a>NMAKE 嚴重錯誤 U1076

名稱太長

字串超過下列限制：

- 巨集名稱的 1024年個字元。

- 目標路徑名稱超過 256 個字元。

- 命令的 2048年個字元。
