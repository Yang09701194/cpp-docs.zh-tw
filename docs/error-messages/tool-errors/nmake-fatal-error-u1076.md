# <a name="nmake-fatal-error-u1076"></a>NMAKE 嚴重錯誤 U1076

名稱太長

字串超過下列其中一項限制：

- 適用于宏名稱的1024個字元。

- 目標路徑名稱為256個字元。

- 命令的2048個字元。
