# <a name="nmake-fatal-error-u1077"></a>NMAKE 嚴重錯誤 U1077
[程式] 5d： 傳回碼 'value'  
  
 指定的命令或程式失敗，傳回指定的結束代碼的 NMAKE 所呼叫。  
  
 若要抑制此錯誤並繼續 NMAKE 工作階段，請使用 /I 選項中， **。忽略**點指示詞或破折號 (**-**) 命令修飾詞。 若要繼續 NMAKE 工作階段不相關的組件的相依性樹狀結構，請使用 [/K] 選項。
