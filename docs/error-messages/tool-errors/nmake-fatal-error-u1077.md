# <a name="nmake-fatal-error-u1077"></a>NMAKE 嚴重錯誤 U1077

[程式]: 傳回碼 'value'

指定的命令或程式呼叫 NMAKE 失敗，傳回指定的結束代碼。

若要隱藏此錯誤並繼續 NMAKE 工作階段，請使用 /I 選項中， **。略過**點指示詞或虛線 (**-**) 命令修飾詞。 若要繼續不相關的組件的相依性樹狀結構的 NMAKE 工作階段，請使用 [/K] 選項。
