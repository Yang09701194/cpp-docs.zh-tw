# NMAKE 嚴重錯誤 U1077
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'program' : 傳回碼 'value'  
  
 給定的命令或 NMAKE 呼叫的程式執行失敗，並傳回給定的結束代碼 \(Exit Code\)。  
  
 若要隱藏此錯誤並繼續執行 NMAKE 工作階段 \(Session\)，請使用 \/I 選項、**.IGNORE** 點指示詞或破折號 \(**\-**\) 命令修飾詞 \(Modifier\)。  若要對相依樹狀結構中其他無關的部分繼續執行 NMAKE 工作階段，請使用 \/K 選項。
