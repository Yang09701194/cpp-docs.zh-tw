# <a name="nmake-fatal-error-u1077"></a>NMAKE 嚴重錯誤 U1077

' program '：傳回碼 ' value '

NMAKE 所呼叫的指定命令或程式失敗，並傳回指定的結束代碼。

若要隱藏此錯誤並繼續 NMAKE 會話，請使用/I 選項 **。忽略**點指示詞或虛線（ **-** ）命令修飾詞。 若要針對相依性樹狀結構中不相關的部分繼續 NMAKE 會話，請使用/K 選項。
