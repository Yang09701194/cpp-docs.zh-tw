# <a name="nmake-fatal-error-u1078"></a>NMAKE 嚴重錯誤 U1078

' expression ' 的常數溢位

給定運算式包含的常數超過範圍-2147483648 到2147483647。 常數出現在下列其中一種情況：

- 使用前置處理指示詞指定的運算式

- 以破折號（ **-** ）命令修飾詞指定的錯誤層級
