# <a name="nmake-fatal-error-u1078"></a>NMAKE 嚴重錯誤 U1078

在 'expression' 的常數溢位

指定的運算式中包含超過，範圍從-2,147,483,648 到 2,147,483,647 的常數。 常數會出現在下列情況的其中一個：

- 前置處理器指示詞指定的運算式

- 指定虛線的錯誤層級 (**-**) 命令修飾詞
