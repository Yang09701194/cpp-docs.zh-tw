# NMAKE 嚴重錯誤 U1078
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在 'expression' 發生常數溢位  
  
 給定的運算式中包含超出 2,147,483,648 到 2,147,483,647 範圍的常數。  下列情形之一會出現這樣的常數：  
  
-   前置處理器指示詞指定的運算式  
  
-   破折號 \(**\-**\) 命令修飾詞指定的錯誤層級
