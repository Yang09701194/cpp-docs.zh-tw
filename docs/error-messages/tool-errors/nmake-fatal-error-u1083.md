# <a name="nmake-fatal-error-u1083"></a>NMAKE 嚴重錯誤 U1083
目標巨集 'target' 展開為無  
  
 給定的目標是未定義或 null 值的巨集引動過程。 NMAKE 無法處理 null 的目標。
