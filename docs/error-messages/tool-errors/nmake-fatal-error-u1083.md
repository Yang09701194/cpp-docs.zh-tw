# <a name="nmake-fatal-error-u1083"></a>NMAKE 嚴重錯誤 U1083

目標巨集 'target' 展開為無

指定的目標是未定義，或具有 null 值的巨集的引動過程。 NMAKE 無法處理 null 的目標。
