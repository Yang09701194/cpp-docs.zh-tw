# NMAKE 嚴重錯誤 U1083
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

目標巨集 'target' 展開為無  
  
 給定的目標是未定義或為 Null 值的巨集引動過程。  NMAKE 無法處理 Null 目標。
