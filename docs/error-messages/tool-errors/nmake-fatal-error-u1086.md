# <a name="nmake-fatal-error-u1086"></a>NMAKE 嚴重錯誤 U1086

**推斷規則不能有相依性**

冒號 (**:**) 中的推斷規則後面必須接著其中一種：

- 新行字元

- 以分號 (**;**)，它後面可以接著命令

- 數字符號 (**#**)，這可以後面加上註解
