# NMAKE 嚴重錯誤 U1086
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**推斷規則不能有相依性**  
  
 推斷規則中的冒號 \(**:**\) 後必須跟隨下列之一：  
  
-   新行字元 \(Newline Character\)  
  
-   分號 \(**;**\)，其後可以是命令  
  
-   數字符號 \(**\#**\)，其後可以是註解
