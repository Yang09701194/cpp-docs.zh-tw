# <a name="nmake-fatal-error-u1086"></a>NMAKE 嚴重錯誤 U1086

**推斷規則不能有相依性**

推斷規則中的冒號（ **：** ）後面必須接著下列其中一項：

- 新行字元 (Newline Character)

- 分號（ **;** ），後面可以接著命令

- 數位記號（ **#** ），後面可以加上批註
