# <a name="nmake-fatal-error-u1087"></a>NMAKE 嚴重錯誤 U1087
不能有： 和:: 同一個目標的相依性  
  
 目標不能指定兩個單冒號 (**:**) 和雙冒號 (`::`) 相依性。  
  
 若要在多個描述區塊中指定的目標，使用`::`每個相依性列中。
