# <a name="nmake-fatal-error-u1087"></a>NMAKE 嚴重錯誤 U1087

相同的目標不能有：和：：相依項

無法同時在單一冒號（ **：** ）和雙冒號（`::`）相依性中指定目標。

若要指定多個描述區塊中的目標，請在每個相依性行中使用 `::`。
