# <a name="nmake-fatal-error-u1087"></a>NMAKE 嚴重錯誤 U1087

不能有： 和:: 同一個目標的相依性

不指定目標，在這兩個在單一冒號 (**:**) 和雙冒號 (`::`) 相依性。

多重描述區塊中指定的目標，請使用`::`在一行的相依性。
