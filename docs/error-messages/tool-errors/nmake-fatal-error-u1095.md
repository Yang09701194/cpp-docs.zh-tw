# <a name="nmake-fatal-error-u1095"></a>NMAKE 嚴重錯誤 U1095

展開的命令列 '命令列' 太長

巨集展開後指定的命令列會超過作業系統的命令列的長度限制。

MS-DOS 命令列上允許最多可有 128 個字元。

如果命令可以接受從檔案的命令列輸入的程式，變更命令，並提供從磁碟上的檔案或內嵌檔案的輸入。 例如，連結和 LIB 接受輸入，是從回應檔。
