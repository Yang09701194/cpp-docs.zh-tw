# <a name="nmake-fatal-error-u1095"></a>NMAKE 嚴重錯誤 U1095

擴充的命令列 ' 命令列 ' 太長

在宏展開之後，指定的命令列超過作業系統的命令列長度限制。

MS-DOS 在命令列上允許最多128個字元。

如果命令適用于可接受檔案中命令列輸入的程式，請變更命令，並從磁片或內嵌檔案的檔案提供輸入。 例如，LINK 和 LIB 接受來自回應檔的輸入。
