# NMAKE 嚴重錯誤 U1097
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

檔名部分語法需有相依性  
  
 目前的相依性不具有明確相依或隱含相依。  使用百分比 \(`%`\) 規範的檔名部分語法代表目前目標的第一個相依元件。
