# <a name="nmake-fatal-error-u1097"></a>NMAKE 嚴重錯誤 U1097

檔案名-元件語法需要相依

目前的相依性沒有明確相依或隱含相依。 檔案名-部分語法（使用百分比（`%`）規範）代表目前目標的第一個相依元件。
