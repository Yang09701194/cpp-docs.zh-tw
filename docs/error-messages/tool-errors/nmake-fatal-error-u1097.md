# <a name="nmake-fatal-error-u1097"></a>NMAKE 嚴重錯誤 U1097
檔名部分語法需有相依性  
  
 目前的相依性並沒有明確的相依性或隱含相依。 使用百分比的檔名部分語法 (`%`) 規範，表示目前目標的第一個相依元件。
