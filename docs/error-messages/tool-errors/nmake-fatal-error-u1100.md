# <a name="nmake-fatal-error-u1100"></a>NMAKE 嚴重錯誤 U1100

巨集 'macroname' 是在批次規則 'rule' 的內容中不合法

NMAKE 就會產生此錯誤的批次模式規則的命令區塊直接或間接參考的特殊檔案巨集不是 $<。

$< 唯一允許的批次模式規則的巨集。
