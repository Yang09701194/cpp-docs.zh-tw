# <a name="nmake-warning-u4001"></a>NMAKE 警告 U4001

> 可以叫用命令檔案，只能從命令列

命令檔，來叫用方式是 @ 記號 (**\@**) 規範，不能包含其他的命令檔的規格。 不允許這類的巢狀結構。 已忽略的規格。
