# NMAKE 警告 U4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

只能從命令列叫用命令檔  
  
 **@** 符號規範叫用 \(Invoke\) 的命令檔不可以包含其他的命令檔規格。  不允許這種巢狀運算。  此規格將被忽略。
