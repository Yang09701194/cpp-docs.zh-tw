# <a name="nmake-warning-u4001"></a>NMAKE 警告 U4001

> 只能從命令列叫用命令檔

@ Sign （ **\@** ）規範所叫用的命令檔，不能包含另一個命令檔的規格。 不允許這種嵌套。 已忽略規格。
