# <a name="nmake-warning-u4007"></a>NMAKE 警告 U4007

檔案名 ' filename ' 太長;截斷至8。3

指定檔案的基底名稱超過八個字元，或副檔名包含三個以上的字元。 NMAKE 將名稱截斷為八個字元的基底和三個字元的副檔名。

如果您的檔案系統支援長檔名，請將名稱括在雙引號（ **"** ）中。
