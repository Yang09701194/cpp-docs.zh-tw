# <a name="nmake-warning-u4007"></a>NMAKE 警告 U4007

檔名 'filename' 太長;截斷為 dos 8.3 格式

指定檔案的基底名稱超過八個字元，或延伸模組有三個以上的字元。 NMAKE 截斷了八個字元的基底和三個字元的延伸模組的名稱。

如果您的檔案系統所支援的長檔名，將名稱括雙引號括住 (**"**)。
