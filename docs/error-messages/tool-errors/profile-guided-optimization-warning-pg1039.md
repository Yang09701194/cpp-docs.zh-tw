# <a name="profile-guided-optimization-warning-pg1039"></a>特性指引最佳化警告 PG1039

無法等待靜音時間進行清除。

應用程式不允許執行[pgosweep](../../build/pgosweep.md) 。  可能的話，請在應用程式處於閒置狀態時執行 pgosweep。
