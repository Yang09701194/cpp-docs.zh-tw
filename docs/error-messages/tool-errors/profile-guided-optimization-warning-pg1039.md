# <a name="profile-guided-optimization-warning-pg1039"></a>特性指引最佳化警告 PG1039

無法以掃掠的空閒時間等候。

不允許應用程式[pgosweep](../../build/reference/pgosweep.md)執行。  當應用程式處於閒置狀態時，可能的話，請執行 pgosweep。
