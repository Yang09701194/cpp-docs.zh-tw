# 特性指引最佳化警告 PG1039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法等候較空閒的時間來進行清除。  
  
 應用程式不允許 [pgosweep](../../build/reference/pgosweep.md) 執行。可能的話，在應用程式處於閒置狀態時執行 pgosweep。
