# <a name="profile-guided-optimization-warning-pg1087"></a>特性指引最佳化警告 PG1087
PGC 檔案 'file' 不是有效的。  
  
 .Pgc 檔案已損毀，可能是因為程式碼剖析工作階段不正常結束。  重新執行此程式碼剖析的案例，並正常結束。
