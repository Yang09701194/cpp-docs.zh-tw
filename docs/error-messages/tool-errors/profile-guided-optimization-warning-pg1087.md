# 特性指引最佳化警告 PG1087
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

PGC 檔 'file' 無效。  
  
 .pgc 檔案已損毀，可能是由於分析工作階段不正常結束而造成。請傳回分析案例，並正常結束。
