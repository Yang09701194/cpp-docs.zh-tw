# <a name="project-build-error-prj0004"></a>專案建置錯誤 PRJ0004

> 無法產生命令列 '*工具*' 工具。

在這種方式進行的呼叫語法指定了一個或多個內容*工具*不合法：

- 您可能已指定格式錯誤或未知的巨集。

- 您的電腦可能位於可用磁碟空間不足。

您可能想要檢閱此工具的設定來看看[命令列屬性頁](../../ide/command-line-property-pages.md)。
