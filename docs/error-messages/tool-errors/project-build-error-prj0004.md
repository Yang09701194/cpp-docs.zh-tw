# <a name="project-build-error-prj0004"></a>專案建置錯誤 PRJ0004

> 無法產生 '*工具*' 工具的命令列。

指定了一或多個屬性，這種方式可讓對*工具*的呼叫語法不合法：

- 您可能指定了格式不正確或未知的宏。

- 您的電腦的可用磁碟空間可能不足。

您可能想要查看 [[命令列] 屬性頁](../../build/reference/command-line-property-pages.md)來檢查工具的設定。
