# 專案建置錯誤 PRJ0004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法產生工具 'tool' 的命令列。  
  
 以此種方法指定了一或多個屬性，使呼叫 ***tool*** 的語法不合法：  
  
-   您可能指定了錯誤的格式或未知的巨集。  
  
-   您電腦的可用磁碟空間不足。  
  
 您應查看[命令列屬性頁](../../ide/command-line-property-pages.md)來檢視工具的設定。
