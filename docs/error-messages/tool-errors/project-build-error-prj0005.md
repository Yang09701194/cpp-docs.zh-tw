# <a name="project-build-error-prj0005"></a>專案建置錯誤 PRJ0005
無法建立暫存檔目錄 'directory' 中。  
  
 無法建立暫存檔的呼叫。 失敗的原因包括：  
  
-   已用完暫存檔名稱。  
  
-   暫存目錄為唯讀。  
  
-   沒有 TMP 環境變數或暫存目錄。  
  
-   可用磁碟空間不足，是您的電腦。
