# <a name="project-build-error-prj0005"></a>專案建置錯誤 PRJ0005

無法建立暫存檔中目錄 'directory'。

若要建立暫存檔的呼叫失敗。 失敗的原因包括：

- 已用盡暫存檔案名稱。

- 暫存目錄是唯讀的。

- 沒有暫存目錄或 TMP 環境變數。

- 可用磁碟空間不足，是您的電腦。
