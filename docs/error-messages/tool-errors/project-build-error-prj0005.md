# <a name="project-build-error-prj0005"></a>專案建置錯誤 PRJ0005

無法在目錄 ' directory ' 中建立暫存檔案。

建立暫存檔案的呼叫失敗。 失敗的原因包括：

- 暫存檔案名稱已用盡。

- 臨時目錄是唯讀的。

- 沒有臨時目錄或 TMP 環境變數。

- 您的電腦的可用磁碟空間不足。
