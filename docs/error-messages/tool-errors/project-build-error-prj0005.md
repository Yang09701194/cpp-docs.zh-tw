# 專案建置錯誤 PRJ0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法在目錄 'directory' 中建立暫存檔案。  
  
 建立暫存檔案的呼叫失敗。  可能的失敗原因包括：  
  
-   用盡暫存檔案名稱。  
  
-   暫存目錄是唯讀的。  
  
-   沒有暫存目錄或 TMP 環境變數。  
  
-   電腦的磁碟空間不足。
