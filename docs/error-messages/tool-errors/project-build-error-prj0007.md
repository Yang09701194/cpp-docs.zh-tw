# <a name="project-build-error-prj0007"></a>專案建置錯誤 PRJ0007

無法建立輸出目錄 ' directory '。

視覺C++效果無法建立輸出目錄。 可能的原因包括：

- 要在其中建立輸出目錄的目錄是唯讀的。

- 電腦已用盡磁碟空間。

- 您指定了不正確目錄路徑。

- 使用者權限不足以建立目錄。
