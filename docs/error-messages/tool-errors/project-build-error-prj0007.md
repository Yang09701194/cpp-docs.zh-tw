# 專案建置錯誤 PRJ0007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法建立輸出目錄 'directory'。  
  
 Visual C\+\+ 無法建立輸出目錄。  可能的原因包括：  
  
-   要建立輸出目錄的目錄為唯讀。  
  
-   電腦的磁碟空間不足。  
  
-   指定的目錄路徑無效。  
  
-   使用者權限不足，無法建立該目錄。
