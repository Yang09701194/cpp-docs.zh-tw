# <a name="project-build-error-prj0007"></a>專案建置錯誤 PRJ0007

無法建立輸出目錄 'directory'。

Visual c + +，就無法建立輸出目錄。 可能的原因包括：

- 在其中顯示已建立的輸出目錄的目錄是唯讀的。

- 電腦已用完磁碟空間。

- 您指定的目錄路徑無效。

- 使用者權限並不足以建立目錄。
