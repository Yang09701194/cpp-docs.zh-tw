# 專案建置錯誤 PRJ0014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

用來控制繁衍 \(Spawn\) 的處理序之工作物件已失敗，建置無法繼續。  
  
 在開發環境中發生了錯誤。  
  
 若要解決這個錯誤，請關閉專案，然後重新載入。  必要時，結束 Visual Studio 之後，再重新啟動。
