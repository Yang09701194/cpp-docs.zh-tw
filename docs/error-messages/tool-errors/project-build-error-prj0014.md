# <a name="project-build-error-prj0014"></a>專案建置錯誤 PRJ0014

用來控制繁衍的處理序的工作物件失敗。  建置無法繼續。

在開發環境中發生錯誤。

若要解決這個錯誤，請關閉並重新載入專案。 如有必要，結束並重新啟動 Visual Studio。
