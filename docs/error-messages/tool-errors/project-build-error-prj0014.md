# <a name="project-build-error-prj0014"></a>專案建置錯誤 PRJ0014

用來控制衍生進程的工作物件失敗。  組建無法繼續。

開發環境發生錯誤。

若要解決這個錯誤，請關閉並重載專案。 如有必要，請結束並重新啟動 Visual Studio。
