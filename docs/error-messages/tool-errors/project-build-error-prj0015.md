# <a name="project-build-error-prj0015"></a>專案建置錯誤 PRJ0015

您的系統中遺漏了 Null 裝置。 我們無法啟動組建。

這項錯誤可能是因為系統資源不足所造成。 關閉部分應用程式或重新開機。

如果您對電腦上的 Null 裝置沒有足夠的許可權，也可能會造成此問題。
