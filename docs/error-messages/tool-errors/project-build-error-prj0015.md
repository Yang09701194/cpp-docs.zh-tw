# 專案建置錯誤 PRJ0015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

系統遺漏 NULL 裝置，無法啟動組建。  
  
 這個錯誤可能因系統資源不足所導致。  請關閉部分應用程式或重新啟動。  
  
 也可能是因為您沒有足夠的權限使用電腦上的 NULL 裝置所導致。
