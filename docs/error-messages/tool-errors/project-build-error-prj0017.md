# 專案建置錯誤 PRJ0017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

目前的工作目錄無效。  
  
 目前工作目錄的路徑長度超過 \_MAXPATH。  若要解決這個錯誤，請不要在如此深的層次中建立您的專案。
