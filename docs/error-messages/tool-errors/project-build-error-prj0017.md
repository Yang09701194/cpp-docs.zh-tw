# <a name="project-build-error-prj0017"></a>專案建置錯誤 PRJ0017

目前的工作目錄無效。

目前的工作目錄的路徑長度超過 _MAXPATH。 若要解決這個錯誤，請勿在這類深度的層級建立您的專案。
