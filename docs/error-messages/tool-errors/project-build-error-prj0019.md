# <a name="project-build-error-prj0019"></a>專案建置錯誤 PRJ0019
工具傳回錯誤碼  
  
 錯誤層級為非零代表自訂建置步驟或建置事件。  
  
 工具傳回錯誤碼，但沒有任何錯誤訊息時，也會看到 PRJ0019。 此情形，例如，如果您將 MIDL 的輸出重新導向到 NUL。  
  
 請參閱[疑難排解自訂建置步驟和建置事件](../../ide/troubleshooting-build-customizations.md)如需詳細資訊。  
  
 當您正在執行的 「 使用者 」 群組成員身分，並且需要系統管理存取權時，也會發生這個錯誤。 如需詳細資訊，請參閱[Users 群組的成員身分執行](../../security/running-as-a-member-of-the-users-group.md)。
