# <a name="project-build-error-prj0021"></a>專案建置錯誤 PRJ0021

> 工具 '*tool*'，屬性 '*property*' 包含不正確檔案名。

工具*工具*的屬性*屬性*中指定的檔案名無效。

您可能使用了未知或不正確宏。

如需如何存取專案屬性的詳細資訊，請參閱[設定視覺C++專案屬性](../../build/working-with-project-properties.md)。
