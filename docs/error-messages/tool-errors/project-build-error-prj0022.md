# <a name="project-build-error-prj0022"></a>專案建置錯誤 PRJ0022

> 無法辨識的工具，屬性 '*屬性*'包含無效的檔名'*檔案*'。

檔名*檔案*屬性中指定*屬性*無效。

您可能已經使用了未知或無效的巨集。

請參閱[設定 Visual c + + 專案屬性](../../build/working-with-project-properties.md)如需如何存取您的專案屬性的詳細資訊。
