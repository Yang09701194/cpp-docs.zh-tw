# 專案建置錯誤 PRJ0022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未知工具的屬性 'property' 含有無效的檔名 'file'。  
  
 在屬性 ***property*** 中指定的檔名 ***file*** 無效。  
  
 您可能使用了未知或無效的巨集。  
  
 如需存取專案屬性的詳細資訊，請參閱[設定 Visual C\+\+ 專案屬性](../../ide/working-with-project-properties.md)。
