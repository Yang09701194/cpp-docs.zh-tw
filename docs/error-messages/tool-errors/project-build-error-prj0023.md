# <a name="project-build-error-prj0023"></a>專案建置錯誤 PRJ0023

> 工具 '*工具*'，未知的屬性包含無效的檔名'*檔案*'。

檔名*檔案*中指定給工具*工具*無效。

您可能已經使用了未知或無效的巨集。

請參閱[設定 Visual c + + 專案屬性](../../ide/working-with-project-properties.md)如需如何存取您的專案屬性的詳細資訊。
