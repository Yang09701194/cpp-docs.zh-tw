# 專案建置錯誤 PRJ0023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

工具 'tool' 的「未知的屬性」所含的檔案名稱 'file' 無效。  
  
 對工具 ***tool*** 所指定的檔名 ***file*** 無效。  
  
 您可能使用了未知或無效的巨集。  
  
 如需存取專案屬性的詳細資訊，請參閱[設定 Visual C\+\+ 專案屬性](../../ide/working-with-project-properties.md)。
