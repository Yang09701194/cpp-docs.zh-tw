# <a name="project-build-error-prj0026"></a>專案建置錯誤 PRJ0026
回應檔 'file' 包含無法轉譯為使用者 ANSI 字碼頁的 Unicode 內容。  
  
 ***UNICODE 檔案的內容***  
  
 專案系統無法正確轉譯成使用者的目前 ANSI 字碼頁的回應檔案中找到 Unicode 內容。  
  
 此錯誤的解決方法是，更新來使用 ANSI，或是在電腦上安裝的字碼頁，並將它設為系統預設值的回應檔案的內容。
