# 專案建置錯誤 PRJ0026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

回應檔案 'file' 包含無法轉譯成使用者 ANSI 字碼頁的 Unicode 內容。  
  
 ***UNICODE 檔案內容***  
  
 專案系統發現：回應檔中的 Unicode 內容無法適當地轉譯為使用者的現行 ANSI 字碼頁。  
  
 這個錯誤的解決方法是，更新回應檔的內容來使用 ANSI，或在您的電腦上安裝該字碼頁，並將它設為系統預設值。
