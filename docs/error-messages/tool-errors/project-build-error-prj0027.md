# 專案建置錯誤 PRJ0027
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unicode 記錄訊息 'contents' 含有無法轉譯為使用者 ANSI 字碼頁的 Unicode 內容。  
  
 您通常只會在建立批次和 \(或\) 回應檔時發生的錯誤中一起看到這則警告。  
  
 這個錯誤的解決方法是，更新組建記錄的內容來使用 ANSI，或在您的電腦上安裝該字碼頁，並將它設為系統預設值。
