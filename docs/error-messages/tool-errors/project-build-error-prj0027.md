# <a name="project-build-error-prj0027"></a>專案建置錯誤 PRJ0027
Unicode 記錄訊息 '內容' 包含無法轉譯為使用者 ANSI 字碼頁的內容。  
  
 您通常只會看到搭配這個警告，但發生錯誤，建立批次和/或回應檔案。  
  
 此錯誤的解決方法是，更新組建記錄檔來使用 ANSI，或是在電腦上安裝的字碼頁，並將它設為系統預設值的內容。
