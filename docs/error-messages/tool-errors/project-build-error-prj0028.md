# <a name="project-build-error-prj0028"></a>專案建置錯誤 PRJ0028

暫存檔案 ' file ' 包含無法轉譯為使用者 ANSI 字碼頁的 Unicode 內容。

以[/MIDL （指定 MIDL 命令列選項）](../../build/reference/midl-specify-midl-command-line-options.md)連結器選項指定的值，無法由系統字碼頁解析。

當您指定 MIDL 命令（輸入字碼頁）時所用的字碼頁必須與系統字碼頁相同。
