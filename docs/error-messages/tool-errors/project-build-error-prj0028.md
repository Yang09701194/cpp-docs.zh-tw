# 專案建置錯誤 PRJ0028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

暫存檔案 'file' 包含無法轉譯成使用者 ANSI 字碼頁的 Unicode 內容。  
  
 以 [\/MIDL \(指定 MIDL 命令列選項\)](../../build/reference/midl-specify-midl-command-line-options.md) 連結器選項指定了一個系統字碼頁無法加以解析的值。  
  
 指定 MIDL 命令時所使用的字碼頁 \(輸入字碼頁\) 必須與系統字碼頁相同。
