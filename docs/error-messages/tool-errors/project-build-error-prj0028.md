# <a name="project-build-error-prj0028"></a>專案建置錯誤 PRJ0028

暫存檔案 'file' 包含無法轉譯為使用者 ANSI 字碼頁的 Unicode 內容。

值，指定具有[/MIDL （指定 MIDL 命令列選項）](../../build/reference/midl-specify-midl-command-line-options.md)無法解析的系統字碼頁的連結器選項。

當您指定 MIDL 命令 （輸入程式碼頁面） 時所使用的字碼頁必須是系統字碼頁相同。
