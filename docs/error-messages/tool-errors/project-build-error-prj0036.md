# <a name="project-build-error-prj0036"></a>專案建置錯誤 PRJ0036
Web Deployment Tool 的 '其他檔案' 屬性包含無效的項目。  
  
 Web 部署的 [屬性] 頁面上的其他檔案屬性包含錯誤，可能是因為巨集評估問題。 這項錯誤也可能表示的格式不正確的路徑，是包含字元或檔案路徑中不合法的字元的組合。  
  
 若要解決這個錯誤，請修正巨集，或修正路徑規格。 評估的路徑是從專案目錄的絕對路徑。  
  
 這項錯誤也可能表示其中一個所參考的檔案不存在。
