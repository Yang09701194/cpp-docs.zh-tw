# <a name="project-build-error-prj0040"></a>專案建置錯誤 PRJ0040
建置發生內部錯誤。 無法繼續。 請重新載入專案，然後再試一次。  
  
 專案系統無法處理您的專案。  
  
 請嘗試重新載入專案。
