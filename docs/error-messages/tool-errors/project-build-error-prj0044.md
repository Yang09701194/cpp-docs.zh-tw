# <a name="project-build-error-prj0044"></a>專案建置錯誤 PRJ0044
自訂建置規則 'rule' 指派給檔案 'file' 的 '其他相依性' 屬性無效。 這個屬性包含評估為 'value' 的 ' string'。  
  
 **其他相依性**屬性評估為空字串，或包含無效的字元 （無法在檔案或目錄名稱中任何字元） 的字串。 自訂建置規則需要建置動作的輸出。  
  
 如需詳細資訊，請參閱[指定自訂建置工具](../../ide/specifying-custom-build-tools.md)。
