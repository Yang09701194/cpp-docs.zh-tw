# 專案建置錯誤 PRJ0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

指派給檔案 'file' 的自訂建置規則 'rule' 之 'Additional Dependencies' 屬性無效，此屬性所含的字串評估結果為 'value'。  
  
 **Additional Dependencies** 屬性評估結果為空字串，或是含有無效字元 \(不能做為檔案或目錄名稱的字元\) 的字串。  自訂建置規則需要建置動作的輸出。  
  
 如需詳細資訊，請參閱[指定自訂建置工具](../../ide/specifying-custom-build-tools.md)。
