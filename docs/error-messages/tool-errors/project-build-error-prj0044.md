# <a name="project-build-error-prj0044"></a>專案建置錯誤 PRJ0044

已指派給檔案 ' file ' 之自訂群組建規則 ' rule ' 的 [其他相依性] 屬性無效。 這個屬性包含評估為 'value' 的 ' string'。

**其他**相依性屬性會評估為空字串，或包含無效字元（不在檔案或目錄名稱中的任何字元）的字串。 自訂建置規則需要建置動作的輸出。

如需詳細資訊，請參閱[指定自訂群組建工具](../../build/specifying-custom-build-tools.md)。
