# 專案建置錯誤 PRJ0046
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法繁衍 \(Spawn\) 命令列，因為指定的命令列是空的。  
  
 makefile 組態中含有空白的命令列，但實際上必須指定命令列才行。
