# <a name="project-build-error-prj0047"></a>專案建置錯誤 PRJ0047

無法繼續暫停的處理序。  建置已失敗。

在開發環境中發生錯誤。

若要解決這個錯誤，請關閉並重新載入專案。 如有必要，結束並重新啟動 Visual Studio。
