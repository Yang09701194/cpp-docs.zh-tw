# 專案建置錯誤 PRJ0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法繼續暫停的程序。建置已失敗。  
  
 在開發環境中發生了錯誤。  
  
 若要解決這個錯誤，請關閉專案，然後重新載入。  必要時，結束 Visual Studio 之後，再重新啟動。
