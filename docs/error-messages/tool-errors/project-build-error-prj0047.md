# <a name="project-build-error-prj0047"></a>專案建置錯誤 PRJ0047

無法繼續暫停的進程。  組建失敗。

開發環境發生錯誤。

若要解決這個錯誤，請關閉並重載專案。 如有必要，請結束並重新啟動 Visual Studio。
