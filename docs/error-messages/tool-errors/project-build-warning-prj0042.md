# <a name="project-build-warning-prj0042"></a>專案建置警告 PRJ0042

> 未設定檔案 '*file*' 之自訂群組建步驟的 ' output ' 屬性。 將略過自訂群組建步驟。

未執行自訂群組建步驟，因為未指定任何輸出。

若要解決這個錯誤，請執行下列其中一個動作：

- 從組建中排除自訂的組建步驟。

- 新增輸出。

- 刪除自訂群組建步驟之命令的內容。
