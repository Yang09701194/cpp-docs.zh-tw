# <a name="resource-compiler-error-rc2015"></a>資源編譯器錯誤 RC2015
常數中包含太多字元  
  
 字元常數中包含兩個以上的字元。  
  
 字元常數會限制為一個字元 （標準字元常數） 或兩個字元 （長的字元常數）。  
  
 請注意逸出序列 (例如，\t 索引標籤) 會轉換成單一字元。
