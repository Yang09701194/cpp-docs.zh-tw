# <a name="resource-compiler-error-rc2015"></a>資源編譯器錯誤 RC2015

常數中有太多字元

字元常數包含兩個以上的字元。

字元常數限制為一個字元（標準字元常數）或兩個字元（長字元常數）。

請注意，escape 序列（例如，tab 的 \t）會轉換成單一字元。
