# 資源編譯器錯誤 RC2015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

常數中包含太多字元  
  
 字元常數包含兩個以上的字元。  
  
 字元常數僅限一個字元 \(標準字元常數\) 或是兩個字元 \(長字元常數\)。  
  
 請注意，逸出序列 \(例如代表 Tab 字元的 \\t\) 會轉換成單一字元。
