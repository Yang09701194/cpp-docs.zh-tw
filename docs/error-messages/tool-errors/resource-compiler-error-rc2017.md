# 資源編譯器錯誤 RC2017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不合法的逸出序列  
  
 出現未預期的逸出序列。  
  
 反斜線 \(**\\**\) 之後跟著數字或字母的逸出序列，只能出現在字元或是字串常數之中。
