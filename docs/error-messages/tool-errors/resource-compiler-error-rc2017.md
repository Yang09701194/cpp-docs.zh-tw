# <a name="resource-compiler-error-rc2017"></a>資源編譯器錯誤 RC2017

不合法的轉義順序

出現一個非預期的逸出序列。

Escape 序列（反斜線（ **\\** ）後面接著數位或字母）可能只會出現在字元或字串常數中。
