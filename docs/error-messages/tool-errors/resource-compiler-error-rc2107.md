# 資源編譯器錯誤 RC2107
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**必須有數字命令值**  
  
 資源編譯器在 **ACCELERATORS** 陳述式中必須有數字 *idvalue* 欄位。  請確定是使用 `#define` 常數來指定此值，且常數的拼字正確。
