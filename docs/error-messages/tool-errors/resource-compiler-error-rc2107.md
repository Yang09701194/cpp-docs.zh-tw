# <a name="resource-compiler-error-rc2107"></a>資源編譯器錯誤 RC2107

**預期的數值命令值**

資源編譯器在**快速鍵**語句中預期會有數值*idvalue*欄位。 請確定您已使用 `#define` 常數來指定值，且常數的拼寫正確。
