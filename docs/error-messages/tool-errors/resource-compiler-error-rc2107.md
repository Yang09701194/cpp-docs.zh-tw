# <a name="resource-compiler-error-rc2107"></a>資源編譯器錯誤 RC2107
**必須是數字命令值**  
  
 資源編譯器需要數值*idvalue*欄位**加速器**陳述式。 請確定您已使用`#define`常數，以指定的值和常數的拼字正確。
