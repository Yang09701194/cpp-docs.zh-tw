# <a name="resource-compiler-error-rc2109"></a>資源編譯器錯誤 RC2109
必須是數值對話常數  
  
 A**對話方塊**陳述式會需要整數值*x、 y、 寬度*，和*高度*欄位。 請確定這些值都會包含在之後**對話方塊**關鍵字，而且它們不是負數。
