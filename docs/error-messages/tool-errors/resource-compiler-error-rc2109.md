# <a name="resource-compiler-error-rc2109"></a>資源編譯器錯誤 RC2109

預期的數值對話方塊常數

DIALOG 語句需要 [x]、[ *y]、[width*] 和 [ *height* **]** 欄位的整數值。 請確定這些值包含在**DIALOG**關鍵字之後，而且不是負數。
