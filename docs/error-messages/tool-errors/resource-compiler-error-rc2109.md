# 資源編譯器錯誤 RC2109
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

必須是數值對話常數  
  
 **DIALOG** 陳述式在 *x*、*y*、寬度 \(width\) 和高度 \(height\) 欄位需要整數值。  請確定在 **DIALOG** 關鍵字之後包含有這些值，且都不是負值。
