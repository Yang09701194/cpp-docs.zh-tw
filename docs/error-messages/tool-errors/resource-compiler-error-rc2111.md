# <a name="resource-compiler-error-rc2111"></a>資源編譯器錯誤 RC2111

不正確控制項類型

**DIALOG**語句中的每個控制語句都必須是下列其中一項：3STATE、AUTO3、AUTOCHECK、AUTORADIO、BEDIT、CHECKBOX、COMBOBOX、CONTROL、CTEXT、DEFPUSHBUTTON、EDITTEXT、群組方塊、HEDIT、ICON、IEDIT、LISTBOX、LTEXT、PUSHBOX、按鈕、選項按鈕、RTEXT、捲軸、USERBUTTON。

請確定這些控制語句的拼寫正確。
