# <a name="resource-compiler-error-rc2111"></a>資源編譯器錯誤 RC2111

無效的控制項類型

在每個控制項陳述式** 對話方塊**陳述式必須是下列其中之一： 3STATE AUTO3、 AUTOCHECK、 AUTORADIO、 BEDIT、 核取方塊、 下拉式方塊、 控制項、 SYSCOMMENTS、 DEFPUSHBUTTON，EDITTEXT、 GROUPBOX、 HEDIT、 圖示、 IEDIT、 LISTBOX、 LTEXT，PUSHBOX、 按鈕、 RADIOBUTTON、 RTEXT、 捲軸、 USERBUTTON。

請確定這些控制陳述式的拼字正確。
