# 資源編譯器錯誤 RC2111
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無效的控制項型別  
  
 **DIALOG** 陳述式中的每個 CONTROL 陳述式都必須是下列其中之一：3STATE、AUTO3、AUTOCHECK、AUTORADIO、BEDIT、CHECKBOX、COMBOBOX、CONTROL、CTEXT、DEFPUSHBUTTON、EDITTEXT、GROUPBOX、HEDIT、ICON、IEDIT、LISTBOX、LTEXT、PUSHBOX、PUSHBUTTON、RADIOBUTTON、RTEXT、SCROLLBAR、USERBUTTON。  
  
 請確定這些 CONTROL 陳述式拼字是正確的。
