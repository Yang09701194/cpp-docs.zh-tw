# <a name="resource-compiler-error-rc2113"></a>資源編譯器錯誤 RC2113

在對話方塊中預期為 END

**結束**關鍵字必須出現在結尾**對話方塊**陳述式。 請確定沒有任何從前面的陳述式的左引號。
