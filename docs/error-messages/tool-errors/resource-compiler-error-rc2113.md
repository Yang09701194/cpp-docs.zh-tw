# 資源編譯器錯誤 RC2113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

對話方塊中必須有 END  
  
 **END** 關鍵字必須出現在 **DIALOG** 陳述式的結尾。  請確定先前陳述式未留下任何左引號。
