# <a name="resource-compiler-error-rc2114"></a>資源編譯器錯誤 RC2114

需要的控制項類別名稱

**DIALOG**語句中 CONTROL 語句的 `class` 欄位必須是下列其中一種類型：按鈕、COMBOBOX、編輯、LISTBOX、捲軸、靜態或使用者定義。 請確定類別的拼寫正確。
