# 資源編譯器錯誤 RC2114
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

必須是控制項類別名稱  
  
 **DIALOG** 陳述式中 CONTROL 陳述式的 `class` 欄位必須是下列其中一種類型：BUTTON、COMBOBOX、EDIT、LISTBOX、SCROLLBAR、STATIC 或使用者自訂。  請確定類別的拼字正確。
