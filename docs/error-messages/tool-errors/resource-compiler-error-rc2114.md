# <a name="resource-compiler-error-rc2114"></a>資源編譯器錯誤 RC2114
預期的控制項類別名稱  
  
 `class`欄位中的控制陳述式**對話方塊**陳述式必須是下列類型之一： 按鈕、 下拉式方塊、 編輯、 清單方塊、 捲軸、 靜態，或使用者定義。 請確定類別的拼法正確。
