# <a name="resource-compiler-error-rc2116"></a>資源編譯器錯誤 RC2116

識別碼必須是數字

預期的數字`id`欄位中的控制陳述式**對話方塊**陳述式。 請確定您有許多或`#define`陳述式的控制項 id。
