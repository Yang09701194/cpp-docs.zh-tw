# 資源編譯器錯誤 RC2116
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ID 必須是數字  
  
 **DIALOG** 陳述式中控制項陳述式的 `id` 欄位必須有數字。  請確定控制項 ID 具有數字或 `#define` 陳述式。
