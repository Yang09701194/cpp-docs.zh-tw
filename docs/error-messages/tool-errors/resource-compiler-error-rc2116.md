# <a name="resource-compiler-error-rc2116"></a>資源編譯器錯誤 RC2116

需要 ID 的數位

在**DIALOG**語句中，控制項語句的 `id` 欄位應為數字。 請確定您有控制項識別碼的數位或 `#define` 語句。
