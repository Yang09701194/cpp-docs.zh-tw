# 資源編譯器錯誤 RC2122
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未知的功能表子型別  
  
 **MENU** 陳述式的 *item\-definition* 欄位只能包含 **MENUITEM** 和 **POPUP** 陳述式。
