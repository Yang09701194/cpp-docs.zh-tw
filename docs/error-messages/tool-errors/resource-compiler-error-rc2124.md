# <a name="resource-compiler-error-rc2124"></a>資源編譯器錯誤 RC2124

不允許空的功能表

在**功能表**語句中定義任何功能表項目之前，會出現**END**關鍵字。 資源編譯器不允許空的功能表。 請確定您在**MENU**語句中沒有任何左引號。
