# 資源編譯器錯誤 RC2124
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不允許空的功能表  
  
 **END** 關鍵字出現在 **MENU** 陳述式中定義的功能表項目之前。  資源編譯器不允許功能表是空的。  請確定 **MENU** 陳述式中沒有任何左引號。
