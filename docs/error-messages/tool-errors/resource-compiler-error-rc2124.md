# <a name="resource-compiler-error-rc2124"></a>資源編譯器錯誤 RC2124
不允許空的功能表  
  
 **結束**關鍵字出現在定義所有功能表項目之前**功能表**陳述式。 資源編譯器不允許空的功能表。 請確定您沒有任何開啟的引號內**功能表**陳述式。
