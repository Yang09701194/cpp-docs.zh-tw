# <a name="resource-compiler-error-rc2135"></a>資源編譯器錯誤 RC2135

找不到檔案：filename

找不到資源編譯器命令列中所指定的檔案。 檢查是否已將檔案移至另一個目錄，以及檔案名和路徑是否輸入正確。

使用**include**環境變數或視覺效果C++ **INCLUDE**設定搜尋檔案。
