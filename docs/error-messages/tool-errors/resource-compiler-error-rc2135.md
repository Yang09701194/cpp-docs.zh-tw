# <a name="resource-compiler-error-rc2135"></a>資源編譯器錯誤 RC2135

找不到檔案：filename

找不到資源編譯器命令列中所指定的檔案。 請檢查檔案是否移動到另一個目錄，以及是否輸入正確的檔案名稱和路徑。

使用搜尋檔案**INCLUDE**環境變數或 Visual c + + **INCLUDE**設定。
