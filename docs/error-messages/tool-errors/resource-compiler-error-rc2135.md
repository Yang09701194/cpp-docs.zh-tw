# 資源編譯器錯誤 RC2135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

找不到檔案: 檔案名稱  
  
 找不到資源編譯器命令列中指定的檔案。  請檢查檔案是否移到其他目錄，以及是否輸入正確的檔名或路徑。  
  
 使用 **INCLUDE** 環境變數或 Visual C\+\+ **INCLUDE** 設定來搜尋檔案。
