# <a name="resource-compiler-error-rc2147"></a>資源編譯器錯誤 RC2147

子語言識別項不是數位

[子語言識別項] 值必須是數位。

**LANGUAGE** 陳述式必須使用下列語法：

**LANGUAGE** *primary_language_ID*，*secondary_language_ID*

有效的子語言識別項會定義為 WINNT. h 檔案中的**SUBLANG_** 常數。
