# <a name="resource-compiler-error-rc2151"></a>資源編譯器錯誤 RC2151

無法重複使用字串常數

在**STRINGTABLE**語句中，您會使用相同的值兩次。 請確定您不會混用重迭的十進位和十六進位值。

**STRINGTABLE**中的每個識別碼都必須是唯一的。 為了達到最高效率，請使用以16的倍數開頭的連續常數。
