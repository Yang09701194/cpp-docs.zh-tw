# 資源編譯器錯誤 RC2152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無效的控制字元  
  
 **ACCELERATORS** 陳述式中的控制字元無效。  有效的控制字元是由 \(僅限\) 單一字母跟隨著插入號 \(**^**\) 所組成。
