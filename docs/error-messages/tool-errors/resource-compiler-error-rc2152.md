# <a name="resource-compiler-error-rc2152"></a>資源編譯器錯誤 RC2152

不正確控制字元

**ACCELERATORS** 陳述式中的控制字元無效。 有效的控制字元是由插入號（ **^** ）後面的一個字母（僅限）所組成。
