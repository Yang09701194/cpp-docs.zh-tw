# 資源編譯器錯誤 RC2162
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

必須是巨集型式參數  
  
 跟在字串化運算子 \(**\#**\) 之後的語彙基元不是型式參數 \(Formal Parameter\) 名稱。
