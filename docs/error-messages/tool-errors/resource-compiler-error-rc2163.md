# <a name="resource-compiler-error-rc2163"></a>資源編譯器錯誤 RC2163
快速鍵類型需要 [ASCII 或 VIRTKEY]  
  
 `type` ACCELERATORS **陳述式中的** 欄位必須包含 ASCII 或 VIRTKEY 值。
