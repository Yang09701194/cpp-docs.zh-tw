# 資源編譯器錯誤 RC2163
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

快速鍵類別必須 \[ASCII 或 VIRTKEY\]  
  
 **ACCELERATORS** 陳述式中的 `type` 欄位必須包含 ASCII 或 VIRTKEY 值。
