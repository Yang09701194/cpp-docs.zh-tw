# 資源編譯器錯誤 RC2164
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

RCDATA 中出現未預期的值  
  
 **RCDATA** 陳述式中的 *raw\-data* 值必須是以逗號分隔的整數或字串。  請確定您沒有遺漏逗號或者遺漏字串前後的引號。
