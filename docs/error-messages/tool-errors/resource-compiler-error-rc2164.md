# <a name="resource-compiler-error-rc2164"></a>資源編譯器錯誤 RC2164

RCDATA 中有未預期的值

*RCDATA* 陳述式中的 **raw-data** 值必須是整數或字串，並以逗號區隔。 請確定您未遺漏逗號或字串周圍的引號。
