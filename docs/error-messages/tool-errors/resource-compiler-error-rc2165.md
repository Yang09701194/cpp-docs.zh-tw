# <a name="resource-compiler-error-rc2165"></a>資源編譯器錯誤 RC2165

在 DLGINCLUDE 語句中找不到字串

語句未指定有效的 include 檔案。

**DLGINCLUDE**語句必須使用下列語法：

**DLGINCLUDE "** <em>filename</em> **. h"**
