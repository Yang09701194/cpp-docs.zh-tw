# <a name="resource-compiler-error-rc2165"></a>資源編譯器錯誤 RC2165
DLGINCLUDE 陳述式中找不到字串  
  
 陳述式未指定有效的包含檔案。  
  
 **DLGINCLUDE**陳述式必須使用下列語法：  
  
 **DLGINCLUDE"*filename*.h 」 * *
