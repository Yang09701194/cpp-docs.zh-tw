# <a name="resource-compiler-error-rc2167"></a>資源編譯器錯誤 RC2167

無法辨識的 VERSIONINFO 欄位;需要開始或逗號

在**VERSIONINFO**結構宣告的**固定**部分中找到無法辨識的欄位。 **VERSIONINFO**欄位必須是以逗點分隔的 dword。
