# <a name="resource-compiler-error-rc2167"></a>資源編譯器錯誤 RC2167

無法辨認的 VERSIONINFO 欄位;BEGIN 或必須有逗號

無法辨認的欄位中找到**FIXED**屬於**VERSIONINFO**結構宣告。 A **VERSIONINFO**欄位必須是以逗號分隔的 DWORD。
