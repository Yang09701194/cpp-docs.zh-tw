# 資源編譯器錯誤 RC2167
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無法辨認的 VERSIONINFO 欄位; 必須是 BEGIN 或逗號  
  
 在 **VERSIONINFO** 結構宣告的 **FIXED** 部分找到無法辨識的欄位。  **VERSIONINFO** 欄位必須是以逗號分隔的 DWORDS。
