# <a name="resource-compiler-error-rc2169"></a>資源編譯器錯誤 RC2169

資源檔檔案名不是2.03 格式

指定的資源所使用的格式早于2.03 版。 資源檔必須使用3.00 版或更新版本的格式進行轉換或重新建立。
