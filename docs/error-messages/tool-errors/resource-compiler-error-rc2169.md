# <a name="resource-compiler-error-rc2169"></a>資源編譯器錯誤 RC2169

資源檔的名稱不是 2.03 格式

指定的資源 2.03 版使用的格式。 資源檔必須轉換或重新建立 3.00 或更新版本使用的格式。
