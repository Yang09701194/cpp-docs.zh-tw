# <a name="resource-compiler-error-rc2170"></a>資源編譯器錯誤 RC2170
點陣圖檔的檔案名稱格式不是 3.00  
  
 使用 Windows 2.x 版格式的點陣圖不能用於 3.x 版資源檔。  
  
 若要修正這個問題，請重繪或將點陣圖轉換成 3.x 格式。
