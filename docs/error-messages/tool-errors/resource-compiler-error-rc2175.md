# <a name="resource-compiler-error-rc2175"></a>資源編譯器錯誤 RC2175
資源檔的名稱不是 3.00 格式  
  
 指定的資源使用 3.00 版之前的格式。 資源檔必須轉換或重新建立使用 3.00 或更新版本的格式。
