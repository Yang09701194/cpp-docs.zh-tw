# <a name="resource-compiler-error-rc2175"></a>資源編譯器錯誤 RC2175

資源檔的名稱不是 3.00 格式

指定的資源比 3.00 版更早使用的格式。 資源檔必須轉換或重新建立 3.00 或更新版本使用的格式。
