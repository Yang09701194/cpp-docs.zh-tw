# 資源編譯器錯誤 RC2175
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

資源檔的名稱不是 3.00 格式  
  
 指定的資源使用了 3.00 版之前版本的格式。  必須利用 3.00 或更新版本的格式來轉換或重新建立資源檔。
