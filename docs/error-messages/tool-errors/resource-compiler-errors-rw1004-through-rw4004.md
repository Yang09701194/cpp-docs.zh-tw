# <a name="resource-compiler-errors-rw1004-through-rw4004"></a>資源編譯器錯誤s RW1004 through RW4004

本章節提供組建工具所產生的錯誤之參考資訊。 若要取得特定的錯誤訊息的說明，請 [輸出] 視窗中的錯誤代碼上按一下滑鼠和按下 f1 鍵，或輸入中的錯誤號碼**尋找**方塊中的索引。
