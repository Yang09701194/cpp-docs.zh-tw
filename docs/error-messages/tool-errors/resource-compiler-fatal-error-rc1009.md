# 資源編譯器嚴重錯誤 RC1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

編譯器限制: 巨集 'macro' 巢狀結構太深  
  
 檔案超出資源編譯器的巨集巢狀層級上限。  修訂指定的原始程式檔，減少檔案中的巨集巢狀層級深度。
