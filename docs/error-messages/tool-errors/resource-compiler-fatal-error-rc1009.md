# <a name="resource-compiler-fatal-error-rc1009"></a>資源編譯器嚴重錯誤 RC1009

編譯器限制： 巨集太深的巢狀 'macro'

檔案超過巨集巢狀資源編譯器限制。 修改指定的原始程式檔，以減少其巨集巢狀結構深度。
