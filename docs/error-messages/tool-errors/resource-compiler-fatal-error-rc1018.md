# <a name="resource-compiler-fatal-error-rc1018"></a>資源編譯器嚴重錯誤 RC1018

非預期的 ' #elif '

`#elif` 指示詞並未出現在 `#if`、 **#ifdef**或 **#ifndef**結構中。

請確定在此語句之前，`#if`、 **#ifdef**或 **#ifndef**語句生效。
