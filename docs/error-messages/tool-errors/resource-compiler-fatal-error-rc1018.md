# <a name="resource-compiler-fatal-error-rc1018"></a>資源編譯器嚴重錯誤 RC1018

未預期的 ' #elif'

`#elif`指示詞沒有出現在`#if`， **#ifdef**，或 **#ifndef**建構。

請確定沒有`#if`， **#ifdef**，或 **#ifndef**此陳述式的作用中之前的陳述式。
