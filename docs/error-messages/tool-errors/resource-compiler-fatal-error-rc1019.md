# <a name="resource-compiler-fatal-error-rc1019"></a>資源編譯器嚴重錯誤 RC1019

非預期 ' #else '

`#else`指示詞沒有出現在`#if`， **#ifdef**，或 **#ifndef**建構。

請確定沒有`#if`， **#ifdef**，或 **#ifndef**此陳述式的作用中之前的陳述式。
