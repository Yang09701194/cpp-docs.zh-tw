# <a name="resource-compiler-fatal-error-rc1019"></a>資源編譯器嚴重錯誤 RC1019
未預期 ' #else '  
  
 `#else`指示詞未出現在`#if`， **#ifdef**，或**#ifndef**建構。  
  
 請確定沒有`#if`， **#ifdef**，或**#ifndef**這個陳述式前的作用中陳述式。
