# <a name="resource-compiler-fatal-error-rc1019"></a>資源編譯器嚴重錯誤 RC1019

非預期的 ' #else '

`#else` 指示詞並未出現在 `#if`、 **#ifdef**或 **#ifndef**結構中。

請確定在此語句之前，`#if`、 **#ifdef**或 **#ifndef**語句生效。
