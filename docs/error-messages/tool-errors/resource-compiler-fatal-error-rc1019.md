# 資源編譯器嚴重錯誤 RC1019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未預期的 '\#else'  
  
 `#else` 指示詞沒有出現在 `#if`、**\#ifdef**，或 **\#ifndef** 建構中。  
  
 請確定在這個陳述式之前有指定的 `#if`、**\#ifdef** 或 **\#ifndef** 陳述式。
