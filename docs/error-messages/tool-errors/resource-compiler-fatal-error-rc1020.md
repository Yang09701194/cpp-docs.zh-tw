# 資源編譯器嚴重錯誤 RC1020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未預期的 '\#endif'  
  
 出現 `#endif` 指示詞，但無對應的 `#if`、**\#ifdef** 或 **\#ifndef** 指示詞。  
  
 請確定每項 `#if`、**\#ifdef** 以及 **\#ifndef** 陳述式都有對應的 `#endif`。
