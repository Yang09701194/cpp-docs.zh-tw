# <a name="resource-compiler-fatal-error-rc1020"></a>資源編譯器嚴重錯誤 RC1020

非預期的 ' #endif '

出現 `#endif` 指示詞，但沒有相符的 `#if`、 **#ifdef**或 **#ifndef**指示詞。

請確定每個 `#if`、 **#ifdef**和 **#ifndef**語句都有相符的 `#endif`。
