# <a name="resource-compiler-fatal-error-rc1020"></a>資源編譯器嚴重錯誤 RC1020
未預期 ' #endif'  
  
 `#endif`指示詞出現但沒有對應的`#if`， **#ifdef**，或**#ifndef**指示詞。  
  
 請確定已符合`#endif`每個`#if`， **#ifdef**，和**#ifndef**陳述式。
