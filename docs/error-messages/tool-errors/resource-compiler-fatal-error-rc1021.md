# 資源編譯器嚴重錯誤 RC1021
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無效的前置處理器命令 'string'  
  
 跟著數字符號 \(**\#**\) 之後的字元未能形成有效的前置處理器指示詞 \(Preprocessor Directive\)。  
  
 數字符號不能是識別項的第一個字元。
