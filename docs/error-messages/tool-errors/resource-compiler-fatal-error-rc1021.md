# <a name="resource-compiler-fatal-error-rc1021"></a>資源編譯器嚴重錯誤 RC1021
無效的前置處理器命令 'string'  
  
 後面的數字符號字元 (**#**) 並未形成有效的前置處理器指示詞。  
  
 數字的符號不能當做識別項中的第一個字元。
