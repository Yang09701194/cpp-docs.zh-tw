# 資源編譯器嚴重錯誤 RC1022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

必須是 '\#endif'  
  
 `#if`、**\#ifdef** 或 **\#ifndef** 指示詞不是以 `#endif` 指示詞結束。  
  
 請確定在這個陳述式之前有指定的 `#if`、**\#ifdef** 或 **\#ifndef** 陳述式。
