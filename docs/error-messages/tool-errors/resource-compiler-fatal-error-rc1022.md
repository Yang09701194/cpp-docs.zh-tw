# <a name="resource-compiler-fatal-error-rc1022"></a>資源編譯器嚴重錯誤 RC1022

必須是 ' #endif '

`#if`、 **#ifdef**或 **#ifndef**指示詞未以 `#endif` 指示詞結束。

請確定在此語句之前，`#if`、 **#ifdef**或 **#ifndef**語句生效。
