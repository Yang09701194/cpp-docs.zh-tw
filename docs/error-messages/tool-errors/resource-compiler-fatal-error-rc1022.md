# <a name="resource-compiler-fatal-error-rc1022"></a>資源編譯器嚴重錯誤 RC1022

預期的 ' #endif'

`#if`， **#ifdef**，或 **#ifndef**指示詞未終止與`#endif`指示詞。

請確定沒有`#if`， **#ifdef**，或 **#ifndef**此陳述式的作用中之前的陳述式。
