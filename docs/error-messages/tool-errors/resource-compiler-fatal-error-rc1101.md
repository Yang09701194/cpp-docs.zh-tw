# <a name="resource-compiler-fatal-error-rc1101"></a>資源編譯器嚴重錯誤 RC1101

> 未指定資源二進位檔檔案名

[重新命名輸出（/fo）] 選項後面沒有檔案名。

請針對/fo 選項使用下列語法：

> **Rc/fo** *outfilename* <em>infilename</em> **.rc**
