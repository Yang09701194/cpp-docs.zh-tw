# 資源編譯器嚴重錯誤 RC1101
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

沒有指定二進位的資源檔名  
  
 \[重新命名輸出\] \(\/fo\) 選項後未接著檔名。  
  
 針對 \/fo 選項，請使用下列語法：  
  
 **RC \/fo**   
 ***outfilename infilename.* rc**
