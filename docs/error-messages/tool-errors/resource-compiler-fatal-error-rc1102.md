# <a name="resource-compiler-fatal-error-rc1102"></a>資源編譯器嚴重錯誤 RC1102

內部錯誤： RCPP 的引數太多

傳遞太多引數至資源編譯器預處理器。 藉由在來源中定義符號，減少使用 [定義符號（/d）] 選項定義的符號數目。 使用 Include 搜尋路徑選項（/i）指定太多 include 檔案搜尋路徑，也可能會導致此錯誤。
