# <a name="resource-compiler-fatal-error-rc1102"></a>資源編譯器嚴重錯誤 RC1102

內部錯誤： RCPP 引數太多

資源編譯器前置處理器時傳遞引數太多。 減少的符號定義的符號定義 (/d) 藉由定義這些來源中的選項。 這項錯誤也可能造成指定了過多的包含檔搜尋路徑使用 Include 搜尋路徑選項 (/ 我)。
