# <a name="resource-compiler-fatal-error-rc1102"></a>資源編譯器嚴重錯誤 RC1102
內部錯誤： RCPP 引數太多  
  
 資源編譯器前置處理器時傳遞引數太多。 請減少使用定義的符號定義的符號 (/d) 選項，您在來源中定義它們。 指定了過多的包含檔搜尋路徑，使用包含搜尋路徑選項也會造成這個錯誤 (/ 我)。
