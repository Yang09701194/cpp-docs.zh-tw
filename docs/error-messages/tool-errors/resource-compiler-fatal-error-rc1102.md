# 資源編譯器嚴重錯誤 RC1102
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

內部錯誤 : RCPP 的引數太多  
  
 傳送到資源編譯器前置處理器的引數過多。  在來源中定義這些符號，來減少 \[定義符號 \(\/d\)\] 選項所定義的符號數目。  造成這項錯誤的原因也可能是由於使用 \[Include 搜尋路徑\] 選項 \(\/i\) 指定了過多的包含檔案搜尋路徑。
