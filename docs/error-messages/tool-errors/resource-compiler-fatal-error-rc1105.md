# 資源編譯器嚴重錯誤 RC1105
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無效的參數、選項: 太多 \/d 參數  
  
 使用「定義符號 \(\/d\)」選項定義的符號過多。  在來源中定義一些符號，然後重新編譯。
