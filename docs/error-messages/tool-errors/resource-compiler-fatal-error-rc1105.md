# <a name="resource-compiler-fatal-error-rc1105"></a>資源編譯器嚴重錯誤 RC1105

不正確參數，選項：/d 參數太多

使用 [定義符號（/d）] 選項定義了太多符號。 在您的來源中定義一些符號，然後重新編譯。
