# 資源編譯器嚴重錯誤 RC1109
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

建立 resource\-name 時發生錯誤  
  
 無法建立指定的 .res 檔案。  請確定不是在唯讀磁碟機上建立檔案。  使用 \/V 選項來找出是否已建立檔案。
