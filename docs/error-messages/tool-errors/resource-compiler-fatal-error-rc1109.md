# <a name="resource-compiler-fatal-error-rc1109"></a>資源編譯器嚴重錯誤 RC1109

建立資源名稱時發生錯誤

無法建立指定的.res 檔案。 請確定不在唯讀的磁碟機上建立。 使用 /V 選項，以找出是否建立檔案。
