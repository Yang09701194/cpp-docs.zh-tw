# <a name="resource-compiler-fatal-error-rc1121"></a>資源編譯器嚴重錯誤 RC1121

讀取檔案時發生 I/O 錯誤

資源編譯器無法讀取檔案。 請檢查包含檔案的磁碟機是否可用，以及檔案是否有效。
