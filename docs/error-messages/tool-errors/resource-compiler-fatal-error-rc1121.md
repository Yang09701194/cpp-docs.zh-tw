# 資源編譯器嚴重錯誤 RC1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

讀取檔案時發生 I\/O 錯誤  
  
 資源編譯器無法讀取檔案。  請檢查包含檔案的磁碟是否可用，以及檔案是否有效。
