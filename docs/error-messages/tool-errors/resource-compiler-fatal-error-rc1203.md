# 資源編譯器嚴重錯誤 RC1203
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

指定了無效的十六進位做為預設語言 ID。  
  
 「指定預設語言 \(\/l\)」選項之後接著無效的十六進位語言 ID。  如需有效語言 ID 的清單，請參閱*執行階段程式庫參考*中的[語言和國家\/地區字串](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)。
