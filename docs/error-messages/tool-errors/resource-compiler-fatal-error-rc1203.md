# <a name="resource-compiler-fatal-error-rc1203"></a>資源編譯器嚴重錯誤 RC1203

指定了不正確十六進位預設語言識別項。

指定 Default Language （/l）選項後面接著不正確十六進位語言識別項。 如需有效語言識別項的清單，請參閱《*執行時間程式庫參考*》中的[語言和國家/地區字串](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)。
