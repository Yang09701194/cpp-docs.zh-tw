# <a name="resource-compiler-fatal-error-rc1205"></a>資源編譯器嚴重錯誤 RC1205
無效的字碼頁  
  
 指定字碼頁 (/c) 選項後面無效的字碼頁。 請參閱[字碼頁](../../c-runtime-library/code-pages.md)中*執行階段程式庫參考*如需詳細資訊。
