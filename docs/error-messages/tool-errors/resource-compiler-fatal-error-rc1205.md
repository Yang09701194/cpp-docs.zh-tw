# <a name="resource-compiler-fatal-error-rc1205"></a>資源編譯器嚴重錯誤 RC1205

字碼頁無效

指定字碼頁（/c）選項後面接著不正確字碼頁。 如需詳細資訊，請參閱 [執行階段程式庫參考](../../c-runtime-library/code-pages.md) 中的 *字碼頁* 。
