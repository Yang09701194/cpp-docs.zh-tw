# 資源編譯器嚴重錯誤 RC1205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無效的字碼頁  
  
 「指定字碼頁 \(\/c\)」選項之後接著無效的字碼頁。  如需詳細資訊，請參閱《執行階段程式庫參考》中的[字碼頁](../../c-runtime-library/code-pages.md)。
