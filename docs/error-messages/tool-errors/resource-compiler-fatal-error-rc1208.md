# <a name="resource-compiler-fatal-error-rc1208"></a>資源編譯器嚴重錯誤 RC1208

輸入的檔案具有。RES 延伸模組

。RES 延伸模組用於資源編譯器的輸出。 。RC 的擴充功能應該用於資源編譯器指令碼。
