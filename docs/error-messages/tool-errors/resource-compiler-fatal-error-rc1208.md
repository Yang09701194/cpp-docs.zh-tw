# 資源編譯器嚴重錯誤 RC1208
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

輸入檔使用 .RES 做為副檔名  
  
 .RES 副檔名使用在資源編譯器的輸出。  .RC 副檔名則使用在資源編譯器指令碼。
