# <a name="resource-compiler-fatal-error-rw1009"></a>資源編譯器嚴重錯誤 RW1009

建立資源名稱時發生錯誤

無法建立指定的.res 檔案。 請確定不在唯讀的磁碟機上建立。 您可以使用 /V 來找出是否建立檔案。
