# 資源編譯器嚴重錯誤 RW1016
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

資源編譯器在前置處理器發生錯誤後結束  
  
 資源編譯器因其他錯誤而中止。  請修正這些其他錯誤，然後重新編譯。
