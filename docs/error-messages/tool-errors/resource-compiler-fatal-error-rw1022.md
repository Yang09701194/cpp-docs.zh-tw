# 資源編譯器嚴重錯誤 RW1022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**寫入檔案時發生 I\/O 錯誤**  
  
 資源編譯器無法寫入檔案。  
  
### 若要修正，請檢查下列可能原因  
  
1.  磁碟空間不足。  可用空間必須至少是相當於您所建立可執行檔的兩倍大小。  
  
2.  磁碟區為唯讀。  
  
3.  錯誤磁區。  
  
4.  共用違規。
