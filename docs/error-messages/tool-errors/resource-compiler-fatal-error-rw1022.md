# <a name="resource-compiler-fatal-error-rw1022"></a>資源編譯器嚴重錯誤 RW1022

**寫入檔案時發生 i/o 錯誤**

資源編譯器無法寫入檔案中。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>透過檢查下列可能原因進行修正

1. 磁碟空間不足。 可用空間必須等於您所建立之可執行檔案大小的兩倍以上。

1. 磁碟區是唯讀的。

1. 磁區損壞。

1. 共用違規。
