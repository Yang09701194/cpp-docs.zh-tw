# <a name="resource-compiler-fatal-error-rw1023"></a>資源編譯器嚴重錯誤 RW1023
寫入檔案，磁碟機已滿的 I/O 錯誤  
  
 可用空間必須等於至少兩次所建立的可執行檔的大小。
