# <a name="resource-compiler-fatal-error-rw1030"></a>資源編譯器嚴重錯誤 RW1030

輸出錯誤

如果資源編譯器無法建立目的地檔案，就可能造成此錯誤。 請確定沒有足夠的磁碟空間，而且您具有寫入權限，在磁碟區上。
