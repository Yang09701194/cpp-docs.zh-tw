# 資源編譯器嚴重錯誤 RW1030
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

輸出錯誤  
  
 如果資源編譯器無法建立目的檔案，就可能會發生這個錯誤。  請確定磁碟空間足夠，且您擁有磁碟的寫入許可。
