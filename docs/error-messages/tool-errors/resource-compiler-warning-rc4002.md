# 資源編譯器警告 RC4002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

巨集 'identifier' 的實質參數太多  
  
 以給定識別項指定的實質參數 \(Actual Parameter\) 數量超出識別項巨集定義中所給定的型式參數 \(Formal Parameter\) 數量。  
  
 已收集的其他實質參數在巨集展開時被忽略。
