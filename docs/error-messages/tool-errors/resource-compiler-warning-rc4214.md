# <a name="resource-compiler-warning-rc4214"></a>資源編譯器警告 RC4214
不是有效的字碼頁： 已忽略  
  
 .Rc 檔案所包含的字碼頁引數，並以指定的字碼頁不正確。 請參閱[IsValidCodePage](/windows/desktop/api/winnls/nf-winnls-isvalidcodepage)如需詳細資訊。
