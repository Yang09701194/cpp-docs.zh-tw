# <a name="resource-compiler-warning-rc4214"></a>資源編譯器警告 RC4214
不是有效的字碼頁： 忽略  
  
 .Rc 檔包含字碼頁引數，而且指定的字碼頁不正確。 請參閱[IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674)如需詳細資訊。
