# <a name="resource-compiler-warning-rc4214"></a>資源編譯器警告 RC4214

字碼頁無效：已忽略

.Rc 檔案包含字碼頁引數，且指定的字碼頁無效。 如需詳細資訊，請參閱[IsValidCodePage](/windows/win32/api/winnls/nf-winnls-isvalidcodepage) 。
