# 資源編譯器警告 RC4214
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無效的字碼頁 : 已忽略  
  
 .rc 檔包含字碼頁引數，而所指定的字碼頁無效。  如需詳細資訊，請參閱 [IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674)。
