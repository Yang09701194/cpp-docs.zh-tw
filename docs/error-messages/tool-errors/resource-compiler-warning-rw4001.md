# <a name="resource-compiler-warning-rw4001"></a>資源編譯器警告 RW4001
.EXE 處理選項 (/L /M/P /T /K /E /31 或/30)  
  
 指定 EXE 處理選項，但沒有任何要處理的可執行檔。 不要使用這些選項在.res 檔。
