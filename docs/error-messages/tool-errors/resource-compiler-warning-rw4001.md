# 資源編譯器警告 RW4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

.EXE 處理選項 \(\/L \/M \/P \/T \/K \/E \/31 或 \/30\)  
  
 您指定了 EXE 處理選項，但沒有要處理的可執行檔。  請不要在.res 檔使用這些選項。
