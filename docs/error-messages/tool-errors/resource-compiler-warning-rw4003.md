# <a name="resource-compiler-warning-rw4003"></a>資源編譯器警告 RW4003

SHIFT 或 CONTROL 使用時沒有 VIRTKEY

在快速鍵對應表資源中，SHIFT 或 CONTROL 需要 VIRTKEY。 SHIFT 和 CONTROL 在 VIRTKEY 類型加速器中以旗標位元表示，因此它們不能獨立於 VIRTKEY 而存在。
