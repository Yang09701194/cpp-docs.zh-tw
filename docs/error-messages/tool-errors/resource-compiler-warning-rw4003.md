# 資源編譯器警告 RW4003
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

使用的 SHIFT 或 CONTROL 缺少 VIRTKEY  
  
 在快速鍵對應表資源中，SHIFT 或 CONTROL 需要 VIRTKEY。  由於將 SHIFT 和 CONTROL 指定為 VIRTKEY 類型快速鍵的旗標位元，因此不能缺少 VIRTKEY。
