# 加入泛型 C++ 類別
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

您可以使用 \[**類別檢視**\] 加入泛型 C\+\+ 類別。  泛型 C\+\+ 類別 \(Class\) 是您定義的類別，或是衍生自您定義的類別。  
  
### 若要將泛型 C\+\+ 類別加入至專案  
  
1.  在 \[**類別檢視**\] 中，以滑鼠右鍵按一下您要加入新類別的專案，然後依序按一下 \[**加入**\] 和 \[**類別**\]。  
  
2.  在[加入類別](../ide/add-class-dialog-box.md)對話方塊的範本窗格中，按一下 \[**C\+\+ 類別**\]。  按一下 \[**加入**\] 顯示[泛型 C\+\+ 類別精靈](../ide/generic-cpp-class-wizard.md)。  
  
3.  在精靈中，提供類別名稱，然後定義設定或接受預設值。  
  
4.  若要關閉精靈並在專案中檢視新的泛型 C\+\+ 類別，按一下 \[**完成**\]。
