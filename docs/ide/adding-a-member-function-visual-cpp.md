# <a name="adding-a-member-function-visual-c"></a>加入成員函式 (Visual C++)
在**類別檢視**，您可以加入任何類別成員函式。 當您這樣做，宣告會加入至標頭檔，而 Stub 成員函式主體會加入至該類別的實作檔 (可以修改這個檔案)。  
  
### <a name="to-add-a-member-function-to-a-class"></a>若要將成員函式加入至類別  
  
1.  在**類別檢視**，展開專案節點，以顯示專案中的類別。 (若要開啟**類別檢視**，在功能表列上選擇 **檢視**，**類別檢視**。)  
  
2.  開啟您想要加入的成員函式，然後選擇 類別的捷徑功能表**新增**，**加入函式**。  
  
3.  提供成員函式的詳細資訊。 如需詳細資訊，請參閱[加入成員函式精靈](../ide/add-member-function-wizard.md)。  
  
4.  選擇**完成**按鈕以產生成員函式程式碼。
