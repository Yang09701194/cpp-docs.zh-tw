# 加入成員函式 (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在 **類別檢視** 中，您可以將成員函式 \(Member Function\) 加入至任何類別 \(Class\) 之中。  當您做這件動作時，宣告會加入至標頭檔 \(Header File\) 當中，然後將 Stub 成員函式主體 \(Member Function Body\) 加入至該類別的實作檔 \(Implementation File\) 當中，並可以修改這個檔案。  
  
### 將成員函式加入至類別  
  
1.  在**類別檢視**中，將專案節點展開以顯示專案中的類別。\(若要開啟 \[**類別檢視**\]，請在功能表列上，選擇 \[**檢視**\]、 \[**類別檢視**\] \)。  
  
2.  開啟要加入成員函式類別的捷徑功能表，然後選取 \[**加入**\]、 \[**加入函式**\]。  
  
3.  提供成員函式的詳細資訊。  如需詳細資訊，請參閱[加入成員函式精靈](../ide/add-member-function-wizard.md)。  
  
4.  選擇 \[**結束**\] 按鈕並產生成員函式程式碼。
