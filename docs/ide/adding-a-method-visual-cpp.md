# 加入方法 (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

您可以使用[加入方法精靈](../ide/add-method-wizard.md)來將方法加入到專案內的介面。  如果專案中包含與介面關聯的類別 \(Class\)，則精靈也會修改該類別。  
  
### 若要將方法加入至專案  
  
1.  在 \[類別檢視\] 中將專案節點展開，以顯示要加入方法的介面。  
  
    > [!NOTE]
    >  您也可以將方法加入分配介面 \(Dispinterface\)，除非專案已經屬性化，否則分配介面位於程式庫節點下。  
  
2.  在介面名稱上按一下滑鼠右鍵。  
  
3.  在捷徑功能表中按一下 \[加入\]，再按一下 \[加入方法\]。  
  
4.  在加入方法精靈中提供建立方法的資訊。  
  
5.  在精靈的 [IDL 屬性](../ide/idl-attributes-add-method-wizard.md)頁面上，對此方法指定任何介面定義語言設定。  
  
6.  按一下 \[完成\] 以加入方法。
