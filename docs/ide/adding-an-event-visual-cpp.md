# <a name="adding-an-event-visual-c"></a>加入事件 (Visual C++)
類別檢視中，您可以加入事件使用[加入事件精靈](../ide/add-event-wizard.md)只至控制項中的類別程式[MFC ActiveX 控制項](../mfc/reference/creating-an-mfc-activex-control.md)專案。 如果您想要將事件加入至專案的另一種，使用**事件**按鈕[屬性 視窗](/visualstudio/ide/reference/properties-window)。  
  
### <a name="to-add-an-event-to-your-mfc-activex-control-project"></a>若要將事件加入至 MFC ActiveX 控制項專案  
  
1.  在 類別檢視中，展開專案節點，顯示專案中的類別。  
  
2.  以滑鼠右鍵按一下專案的控制項類別。  
  
3.  在捷徑功能表，按一下 **新增**，然後按一下 **加入事件**顯示加入事件精靈。  
  
4.  提供適當的精靈方塊中的事件資訊。  
  
5.  按一下**完成**將事件加入至專案。
