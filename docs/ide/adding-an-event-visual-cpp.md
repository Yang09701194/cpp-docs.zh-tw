# 加入事件 (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

從 \[類別檢視\]，您只能使用[加入事件精靈](../ide/add-event-wizard.md)將事件加入至 [MFC ActiveX 控制項](../mfc/reference/creating-an-mfc-activex-control.md)專案中的控制項類別。  如果要將事件加入至其他類型的專案，請使用[屬性視窗](../Topic/Properties%20Window.md)中的 \[事件\] 按鈕。  
  
### 若要將事件加入至您的 MFC ActiveX 控制項專案  
  
1.  在 \[類別檢視\] 中，將專案節點展開以顯示專案中的類別。  
  
2.  以滑鼠右鍵按一下專案的控制項類別。  
  
3.  在捷徑功能表上按一下 \[加入\]，接著按一下 \[**加入事件**\] 以顯示加入事件精靈。  
  
4.  在適當的精靈方塊中提供事件資訊。  
  
5.  按一下 \[完成\] 以將事件加入至專案。
