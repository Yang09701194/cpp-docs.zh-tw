# <a name="adding-an-event-visual-c"></a>加入事件 (Visual C++)

從 [類別檢視]，您可以使用[新增事件精靈](../ide/add-event-wizard.md)，將事件只新增至 [MFC ActiveX 控制項](../mfc/reference/creating-an-mfc-activex-control.md)專案中的控制項類別。 如果您想要將事件新增至另一種專案，請使用[屬性視窗](/visualstudio/ide/reference/properties-window)中的 [事件] 按鈕。

### <a name="to-add-an-event-to-your-mfc-activex-control-project"></a>將事件新增至您的 MFC ActiveX 控制項專案

1. 在 [類別檢視] 中，展開專案節點，顯示專案中的類別。

1. 以滑鼠右鍵按一下專案的控制項類別。

1. 在捷徑功能表上，按一下 [新增]，然後按一下 [新增事件] 顯示 [新增事件精靈]。

1. 在適當的精靈方塊中提供事件資訊。

1. 按一下 [完成] 將事件新增至專案。
