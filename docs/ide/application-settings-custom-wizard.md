# 自訂精靈、應用程式設定
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使用自訂精靈的這個頁面來設計精靈。  
  
 **精靈的易記名稱**  
 設定您設計的精靈名稱。  這個名稱會顯示在 ReadMe.txt 檔案中，且會成為 HTML 檔案的標題。  
  
 **使用者介面**  
 預設值。  指定該自訂精靈顯示使用者介面。  如果您要讓使用者選取指定您的精靈會產生何種程式碼的選項，則該精靈必須要有一個使用者介面。  如果精靈的目的是產生不含使用者選項的樣板程式碼，便不需要有使用者介面。  
  
 **頁碼**  
 設定您的自訂精靈所需的頁數。  這個選項只適用於有使用者介面的精靈。
