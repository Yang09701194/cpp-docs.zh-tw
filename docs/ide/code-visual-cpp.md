# &lt;code&gt; (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

\<code\> 標記讓您可以指出一行或多行做為程式碼。  
  
## 語法  
  
```  
<code>content</code>  
```  
  
#### 參數  
 `content`  
 您要標記為程式碼的文字。  
  
## 備註  
 應該將使用 [\<c\>](../ide/c-visual-cpp.md) 指定文字的部分程式碼。  
  
 使用 [\/doc](../build/reference/doc-process-documentation-comments-c-cpp.md) 進行編譯，將文件註解處理為檔案。  
  
## 範例  
 如需取得使用 \<code\> 標記的範例，請參閱 [\<example\>](../ide/example-visual-cpp.md) 主題。
