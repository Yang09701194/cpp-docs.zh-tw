# 命令列屬性頁
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

多數屬性頁資料夾都包含 \[**命令列**\] 屬性頁。  這個屬性頁顯示要在資料夾中設定哪些屬性。  \[**命令列**\] 屬性頁也包含 \[**其他選項**\] 方塊，讓您能夠指定對工具有效但不存在於資料夾的屬性。  
  
 您在編輯方塊中輸入的任何命令都會為資料夾傳遞至工具。  輸入不會執行任何驗證或檢查，也不會有任何相依性檢查。  
  
 如需如何存取 \[**命令列**\] 屬性頁的詳細資訊，請參閱 [HOW TO：使用屬性頁指定專案屬性](../misc/how-to-specify-project-properties-with-property-pages.md)。
