# 建立自訂精靈
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

您可以在需要建立新的自訂精靈時，使用 Visual C\+\+ 自訂精靈。  建立自訂精靈最簡單的方式就是使用[自訂精靈](../ide/custom-wizard.md)。  
  
> [!TIP]
>  自訂精靈範本是在 Express 版中無法使用。  
  
### 若要使用自訂精靈建立精靈  
  
1.  請依照說明主題[使用 Visual C\+\+ 應用程式精靈建立專案](../ide/creating-desktop-projects-by-using-application-wizards.md)中的指示操作。  
  
2.  在 \[**新的專案**\] 對話方塊中，選取 \[**Visual C\+\+&#124;概觀**\] 窗格中的 **Custom Wizard** 圖示以開啟自訂精靈。  
  
3.  使用此自訂精靈來定義您的[應用程式設定](../ide/application-settings-custom-wizard.md)。  
  
4.  按一下 \[完成\] 以關閉精靈，並在開發環境中開啟新專案。  
  
 在建立專案之後，您可在**方案總管**中檢視建立的檔案。  如需自訂精靈的建立檔案之詳細資訊，請參閱[您的精靈所建立的檔案](../ide/files-created-for-your-wizard.md)。
