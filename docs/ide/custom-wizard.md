# 自訂精靈
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Visual C\+\+ 自訂精靈是一種特定類型的精靈，專門設計用來協助您建立屬於您自己的精靈。  一旦設定好之後，精靈就會產生您專案所需的起始檔案 \(Starter File\)。  
  
## 概觀  
 這個精靈頁面將會描述您正在建立的自訂精靈專案之目前應用程式設定。  在預設情況下，精靈會如下建立專案：  
  
-   精靈的易記名稱將會設為您在**新增專案對話方塊**中所指定的專案名稱。  
  
-   精靈內有一個使用者介面。  
  
-   而精靈使用者介面內有一個頁面。  
  
 您可在精靈的[應用程式設定](../ide/application-settings-custom-wizard.md)頁面變更這些預設值。
