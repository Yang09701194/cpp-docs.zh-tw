# <a name="general-property-page-file"></a>一般屬性頁 (檔案)

檔案中的選取時**方案總管 中**、**一般**下的 屬性頁**組態屬性**節點包含下列屬性：

**從組建排除**  
指定是否應在目前組態的組建檔案。

若要以程式設計方式存取此屬性，請參閱 <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.ExcludedFromBuild%2A>。

**工具**  
將用來建立此檔案的工具。 請參閱[指定自訂建置工具](../ide/specifying-custom-build-tools.md)如需詳細資訊。

若要以程式設計方式存取此屬性，請參閱 <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.Tool%2A>。

如需有關如何存取詳細**一般**下的 屬性頁**組態屬性** 節點，請參閱[使用專案屬性](../ide/working-with-project-properties.md)。

對於非 Windows 的專案，請參閱[Linux c + + 屬性頁面參考](../linux/prop-pages-linux.md)<!-- or [C++ Cross Platform Property Page Reference](../linux/prop-pages-linux.md)-->。
