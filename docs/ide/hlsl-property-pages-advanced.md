# <a name="hlsl-property-pages-advanced"></a>HLSL 屬性頁：進階
若要設定下列屬性的 HLSL 編譯器 (fxc.exe)，使用其**進階**屬性頁。 如需有關如何存取資訊**進階**屬性頁的 HLSL 資料夾中，請參閱[使用專案屬性](../ide/working-with-project-properties.md)。  
  
## <a name="uielement-list"></a>UIElement 清單  
 **隱藏程式啟始資訊**  
 **是 (/ nologo)**來隱藏程式啟始資訊及著作權訊息; 否則**否**。 根據預設，此值是**是 (/ nologo)**。  
  
 **將警告視為錯誤**  
 **是 (/WX)**將所有的編譯器警告視為錯誤; 否則**否**。
