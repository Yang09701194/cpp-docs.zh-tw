# 如何：從現有程式碼建立 C++ 專案
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在 [!INCLUDE[vsPro](../ide/includes/vspro_md.md)] 和 [!INCLUDE[vsUltShort](../ide/includes/vsultshort_md.md)] 中，您可以使用 \[**從現有程式碼檔建立新專案**\] 精靈將現有的程式碼檔案移植到 Visual C\+\+ 專案。  在 Express 版中無法使用此精靈。  
  
 只要將現有的程式碼檔移植到 Visual C\+\+ 專案中，您就可以使用 IDE 功能，例如 [IntelliSense](../Topic/Using%20IntelliSense.md)、\[**呼叫瀏覽器**\] 及[屬性管理員](../misc/property-manager.md)。  
  
### 若要從現有程式碼建立 C\+\+ 專案  
  
1.  在 \[**檔案**\] 功能表上，指向 \[**新增**\]，再按一下 \[**現有程式碼中的專案**\]。  
  
2.  在 \[**從現有的程式碼檔建立新專案**\] 精靈的第一頁上，於 \[**您要建立的專案類型為何**\] 清單中選取 \[**Visual C\+\+**\]。  
  
3.  在精靈的後續頁面中，為您要建立的新專案指定設定值。  
  
4.  按一下 \[**完成**\] 產生新專案。
