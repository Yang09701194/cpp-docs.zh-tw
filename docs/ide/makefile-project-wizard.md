# <a name="makefile-project-wizard"></a>Makefile 專案精靈
若要建立的建置引擎，以執行命令列使用這個精靈。 一旦您建立專案時，您可以檢視及編輯專案的屬性，在專案的[屬性頁](../ide/working-with-project-properties.md)。  
  
## <a name="overview"></a>總覽  
 這個精靈頁面說明目前的專案設定為您建立的專案。 根據預設，已設定專案建置這兩個偵錯和發行組態的*ProjName*.exe。  
  
 若要變更這些預設值，請按一下[應用程式設定](../ide/application-settings-makefile-project-wizard.md)精靈左欄中索引標籤，然後進行所需的變更。
