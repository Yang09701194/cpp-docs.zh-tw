# Makefile 專案精靈
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使用本精靈建立命令列供建置引擎執行。  一旦建立了專案，您就可以在專案的[屬性頁](../ide/working-with-project-properties.md)檢視和編輯專案的屬性。  
  
## 概觀  
 本精靈頁說明您正在建立的專案中目前的專案設定。  根據預設，專案的設定是同時建置 *ProjName*.exe 的偵錯組態和發行組態。  
  
 若要變更這些預設值，請按一下精靈左欄的[應用程式設定](../ide/application-settings-makefile-project-wizard.md)索引標籤來進行所需的變更。
