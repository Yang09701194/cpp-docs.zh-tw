# <a name="managed-resources-property-page"></a>Managed 資源屬性頁
可讓資源編譯器設定。  
  
 **受管理的資源**屬性頁包含下列屬性：  
  
 **資源的邏輯名稱**  
 指定*邏輯名稱*與產生的中繼.resources 檔。 邏輯名稱是用來載入資源的名稱。 如果未不指定任何邏輯名稱，則資源 (.resx) 檔案名稱會使用做為邏輯名稱。  
  
 **輸出檔名稱**  
 指定的資源 (.resx) 檔案所提供的最後輸出檔的名稱。  
  
 **預設的當地語系化資源**  
 指定在給定的.resx 檔作為預設的資源或附屬 dll。  
  
 如需有關如何存取詳細**受管理的資源**屬性頁上，請參閱[使用專案屬性](../ide/working-with-project-properties.md)。
