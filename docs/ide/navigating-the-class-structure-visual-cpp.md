# <a name="navigate-the-class-structure"></a>巡覽類別結構

您可以使用 [類別檢視] 或 [物件瀏覽器] 來巡覽專案類別結構。 專案類別結構可能包含函式、繼承和介面。 如需詳細資訊，請參閱[檢視程式碼的結構](/visualstudio/ide/viewing-the-structure-of-code)。

若要移至函式宣告，請在 [類別檢視] 中的函式名稱上按兩下。 若要移至類別定義，請在 [類別檢視] 中的函式名稱上按兩下。
