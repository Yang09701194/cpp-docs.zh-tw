# <a name="redistributing-database-support-files"></a>轉散發資料庫支援檔案
資料存取物件 (DAO) 以及 Microsoft Data Access SDK 中的資料庫技術，可以轉散發的支援檔案。  
  
## <a name="installing-dao-support-files"></a>安裝 DAO 支援檔案  
 若要取得 DAO 的最新版本，請參閱[文章 239114： 濆爧髍孮 Microsoft Jet 4.0 資料庫引擎的最新 service pack](http://go.microsoft.com/fwlink/p/?linkid=198014) Microsoft 技術支援網站上。  
  
## <a name="installing-microsoft-data-access-sdk-support-files"></a>安裝 Microsoft Data Access SDK 支援檔案  
 若要安裝的 ODBC、 OLE DB、 ActiveX Data Objects (ADO) 和遠端資料服務 (RDS) 支援使用 Mdac_typ.exe。 Mdac_typ.exe 位於在 Visual Studio 安裝媒體中的 ..\WCU\MDAC28\ 資料夾。 您也可以下載 Mdac_typ.exe 從[Microsoft Download Center](http://go.microsoft.com/fwlink/p/?linkid=198015)網站。  
  
 如需詳細資訊，在[MSDN](http://go.microsoft.com/fwlink/p/?linkid=198016)網站中，搜尋 < 轉散發 MDAC 2.8 SP1 >。 如果您使用 Visual Studio 安裝程式專案來部署您的應用程式，請參閱[部署和相依性](http://msdn.microsoft.com/en-us/49e9b84d-bd6a-4388-b9ac-46ea79cf0733)。
