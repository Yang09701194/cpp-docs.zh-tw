# 轉散發 Web 用戶端應用程式
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

如果您的應用程式使用 MFC 類別實作 WebBrowser 控制項 \(例如，`CHtmlView` 或 `CHtmlEditView`\)，則目標電腦上至少必須安裝 Microsoft Internet Explorer 4.0 或更新版本。  
  
 安裝最新版的 Internet Explorer 也會確保目標電腦具有最新的通用控制項檔案。  
  
 下列知識庫文件中載有安裝最少 Internet Explorer 元件的相關資訊：  
  
-   Q185375＜HOWTO: Create a Single EXE Install of Internet Explorer＞\([http:\/\/support.microsoft.com\/support\/kb\/articles\/q185\/3\/75.asp](http://support.microsoft.com/support/kb/articles/q185/3/75.asp)\)  
  
 您可以在 MSDN Library 或在 [http:\/\/support.microsoft.com](http://support.microsoft.com/) 網站上找到知識庫文件。
