# 資源屬性頁
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

如需使用資源編譯器的詳細資訊，請參閱[使用 RC \(RC 命令列\)](http://msdn.microsoft.com/library/windows/desktop/aa381055)。  
  
 如需存取 \[資源\] 屬性頁的詳細資訊，請參閱 [HOW TO：使用屬性頁指定專案屬性](../misc/how-to-specify-project-properties-with-property-pages.md)。  
  
 若要以程式設計的方式存取這些屬性，請參閱 <xref:Microsoft.VisualStudio.VCProjectEngine.VCResourceCompilerTool>。
