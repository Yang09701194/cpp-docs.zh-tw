# <a name="xml-data-generator-tool-property-page"></a>XML 資料產生器工具屬性頁
**XML 資料產生器工具**當您將資料集加入至專案屬性頁面會變成可用。  
  
 **XML 資料產生器工具**屬性頁包含下列屬性：  
  
 **輸出檔**  
 指定要使用的輸出檔案名稱。  
  
 **隱藏程式啟始資訊**  
 隱藏程式啟始資訊及著作權訊息。  
  
 **產生的 Proxy 語言**  
 決定要發出 managed 程式碼。  
  
 如需有關如何存取詳細**XML 資料產生器工具**屬性頁上，請參閱[使用專案屬性](../ide/working-with-project-properties.md)。  
  
 如需如何以程式設計方式存取此物件的成員資訊，請參閱<xref:Microsoft.VisualStudio.VCProjectEngine.VCXMLDataGeneratorTool>
