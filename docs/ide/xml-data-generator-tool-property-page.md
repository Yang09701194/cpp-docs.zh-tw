# <a name="xml-data-generator-tool-property-page"></a>XML 資料產生器工具屬性頁
[XML 資料產生器工具] 屬性頁會在您將資料集新增至專案時變成可用。  
  
[XML 資料產生器工具] 屬性頁包含下列屬性：  
  
- **輸出檔案**

   指定要使用的輸出檔名稱。  
  
- **隱藏啟動橫幅**

   隱藏程式啟始資訊和資訊訊息的畫面。  
  
- **產生的 Proxy 語言**

   指定是否要發出受控碼。  
  
如需如何存取 [XML 資料產生器工具] 屬性頁的詳細資訊，請參閱[使用專案屬性](../ide/working-with-project-properties.md)。  
  
如需如何以程式設計方式存取此物件成員的資訊，請參閱 <xref:Microsoft.VisualStudio.VCProjectEngine.VCXMLDataGeneratorTool>
