# XML 資料產生器工具屬性頁
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

當您將資料集加入至專案時，就可以使用 \[XML 資料產生器工具\] 屬性頁。  
  
 \[XML 資料產生器工具\] 屬性頁包含下列屬性：  
  
 **輸出檔案**  
 指定要使用的輸出檔名稱。  
  
 **隱藏程式啟始資訊**  
 隱藏程式啟始資訊和資訊訊息的畫面。  
  
 **產生的 Proxy 語言**  
 決定是否發出 Managed 程式碼。  
  
 如需如何存取 \[XML 資料產生器工具\] 屬性頁的詳細資訊，請參閱 [HOW TO：使用屬性頁指定專案屬性](../misc/how-to-specify-project-properties-with-property-pages.md)。  
  
 如需如何以程式設計方式存取此物件成員的詳細資訊，請參閱 <xref:Microsoft.VisualStudio.VCProjectEngine.VCXMLDataGeneratorTool>。
