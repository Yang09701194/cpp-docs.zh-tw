# <a name="inbytestring"></a>__inbytestring
**Microsoft 特定的**  
  
 讀取資料，從指定的連接埠使用`rep insb`指令。  
  
## <a name="syntax"></a>語法  
  
```  
void __inbytestring(  
   unsigned short Port,  
   unsigned char* Buffer,  
   unsigned long Count  
);  
```  
  
#### <a name="parameters"></a>參數  
 [輸入] `Port`  
 要讀取的連接埠。  
  
 [輸出] `Buffer`  
 從連接埠讀取的資料會寫入這裡。  
  
 [輸入] `Count`  
 要讀取之資料的位元組數目。
