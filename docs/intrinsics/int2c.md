# <a name="int2c"></a>__int2c
**Microsoft 專屬**  
  
 會產生`int 2c`觸發程序的指令`2c`插斷。  
  
## <a name="syntax"></a>語法  
  
```  
void __int2c(void);  
```  
  
## <a name="requirements"></a>需求  
  
|內建|架構|  
|---------------|------------------|  
|`__int2c`|x86、x64|  
  
 **標頭檔** \<intrin.h >  
  
**結束 Microsoft 專屬**
