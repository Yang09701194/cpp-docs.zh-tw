# <a name="inwordstring"></a>__inwordstring
**Microsoft 特定的**  
  
 讀取資料，從指定的連接埠使用`rep insw`指令。  
  
## <a name="syntax"></a>語法  
  
```  
void __inwordstring(  
   unsigned short Port,  
   unsigned short* Buffer,  
   unsigned long Count  
);  
```  
  
#### <a name="parameters"></a>參數  
 [輸入] `Port`  
 要讀取的連接埠。  
  
 [輸出] `Buffer`  
 從連接埠讀取的資料會寫入這裡。  
  
 [輸入] `Count`  
 要讀取之資料的字組數目。
