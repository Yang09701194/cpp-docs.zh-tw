# <a name="outdword"></a>__outdword
**Microsoft 特定的**  
  
 會產生`out`指令，以便傳送 doubleword`Data`輸出連接埠`Port`。  
  
## <a name="syntax"></a>語法  
  
```  
void __outdword(   
   unsigned short Port,   
   unsigned long Data   
);  
```  
  
#### <a name="parameters"></a>參數  
 [輸入] `Port`  
 將資料傳送至連接埠。  
  
 [輸入] `Data`  
 要傳送 doubleword。
