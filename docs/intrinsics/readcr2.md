# <a name="readcr2"></a>__readcr2
**Microsoft 特定的**  
  
 讀取 CR2 暫存器，並傳回其值。  
  
## <a name="syntax"></a>語法  
  
```  
unsigned __int64 __readcr2(void);  
```  
  
## <a name="return-value"></a>傳回值  
 CR2 暫存器中的值。

 **標頭檔** \<intrin.h >  
  
## <a name="remarks"></a>備註  
 此內建只適用於核心模式，且此常式僅可作為內建常式使用。
