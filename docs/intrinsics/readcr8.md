# <a name="readcr8"></a>__readcr8
**Microsoft 特定的**  
  
 讀取 CR8 暫存器，並傳回其值。  
  
## <a name="syntax"></a>語法  
  
```  
unsigned __int64 __readcr8(void);  
```  
  
## <a name="return-value"></a>傳回值  
 CR8 暫存器中的值。
