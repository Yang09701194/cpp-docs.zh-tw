# <a name="readeflags"></a>__readeflags
讀取程式狀態和控制 (EFLAGS) 註冊。  
  
## <a name="syntax"></a>語法  
  
```  
unsigned     int __readeflags(void);  
unsigned __int64 __readeflags(void);  
```  
  
## <a name="return-value"></a>傳回值  
 EFLAGS 暫存器的值。 傳回值為 32 位元長 32 位元平台以及 64 位元長時間上的 64 位元平台。  
  
## <a name="remarks"></a>備註  
 這些常式會只提供內建函式。
