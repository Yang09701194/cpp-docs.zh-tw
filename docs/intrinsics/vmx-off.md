# <a name="vmxoff"></a>__vmx_off
**Microsoft 專屬**  
  
 停用處理器中的虛擬機器擴充功能 (VMX) 作業。  
  
## <a name="syntax"></a>語法  
  
```  
void __vmx_off();  
```  
  
## <a name="remarks"></a>備註  
 `__vmx_off`函式相當於`VMXOFF`機器指令。 這個函式支援主機虛擬機器監視器與客體作業系統及其應用程式的互動。 如需詳細資訊，搜尋文件 < Intel 虛擬化技術規格的 IA-32 Intel 架構 >、 文件編號 C97063-002，位於[Intel Corporation](http://go.microsoft.com/fwlink/p/?linkid=127)站台。

|---------------|------------------|  
|`__vmx_off`|x86、x64|
