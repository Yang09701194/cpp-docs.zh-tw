# <a name="writecr8"></a>__writecr8
**Microsoft 特定的**  
  
 將值寫入`Data`CR8 註冊。  
  
## <a name="syntax"></a>語法  
  
```  
void writecr8(   
   unsigned __int64 Data   
);  
```  
  
#### <a name="parameters"></a>參數  
 [輸入] `Data`  
 要寫入至 CR8 暫存器值。
