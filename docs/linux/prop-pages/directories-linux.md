# <a name="vc-directories-linux-c"></a>VC++ 目錄 (Linux C++)

屬性 | 描述
--- | ---
Include 目錄 | 建置 VC++ 專案期間，搜尋內含檔案時要使用的路徑。  相當於環境變數 INCLUDE。
程式庫目錄 | 建置 VC++ 專案期間，搜尋程式庫檔案時要使用的路徑。  相當於環境變數 LIB。
來源目錄 | 搜尋用於 Intellisense 的來源檔案時要使用的路徑。
排除目錄 | 搜尋掃描相依性時要略過的路徑。
