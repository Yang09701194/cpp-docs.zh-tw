# Accelerator Keys for the Dialog Editor
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下列是 \[對話方塊\] 編輯器預設的快速鍵 \(Accelerator Key\) 命令。  若要變更快速鍵，請按一下 \[**工具**\] 功能表上的 \[**選項**\]，然後在 \[**環境**\] 資料夾底下選擇 \[**鍵盤**\]。  如需詳細資訊，請參閱 [識別及自訂鍵盤快速鍵](../Topic/Identifying%20and%20Customizing%20Keyboard%20Shortcuts%20in%20Visual%20Studio.md)。  
  
> [!NOTE]
>  根據您目前使用的設定或版本，您所看到的對話方塊可用選項，以及功能表命令的名稱和位置，可能會與 \[說明\] 中描述的有所不同。  若要變更設定，請從 \[**工具**\] 功能表中選擇 \[**匯入和匯出設定**\]。  如需詳細資訊，請參閱 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/zh-tw/22c4debb-4e31-47a8-8f19-16f328d7dcd3)。  
  
|Command|索引鍵|描述|  
|-------------|---------|--------|  
|Format.AlignBottoms|CTRL \+ SHIFT \+ 向下鍵|以主控項對齊已選取控制項的下邊緣|  
|Format.AlignCenters|SHIFT \+ F9|以主控項對齊已選取控制項的垂直置中|  
|Format.AlignLefts|CTRL \+ SHIFT \+ 向左鍵|以主控項對齊選取控制項的左邊緣|  
|Format.AlignMiddles|F9|以主控項對齊已選取控制項的水平置中|  
|Format.AlignRights|CTRL \+ SHIFT \+ 向右鍵|以主控項對齊已選取控制項的右邊緣|  
|Format.AlignTops|CTRL \+ SHIFT \+ 向上鍵|以主控項對齊已選取控制項的上邊緣|  
|Format.ButtonBottom|CTRL \+ B|將選取按鈕沿著對話方塊的底部中間放置|  
|Format.ButtonRight|CTRL \+ R|將選取的按鈕置於對話方塊的右上角|  
|Format.CenterHorizontal|CTRL \+ SHIFT \+ F9|在對話方塊內將控制項水平置中|  
|Format.CenterVertical|CTRL \+ F9|在對話方塊內將控制項垂直置中|  
|Format.CheckMnemonics|CTRL \+ M|檢查助憶鍵的唯一性|  
|Format.SizeToContent|SHIFT \+ F7|重新調整選取的控制項的大小以配合標題文字|  
|Format.SpaceAcross|ALT \+ 向左鍵|均等選取控制項的水平間距|  
|Format.SpaceDown|ALT \+ 向下鍵|均等選取控制項的垂直間距|  
|Format.TabOrder|CTRL \+ D|在對話方塊內設定控制項的順序|  
|Format.TestDialog|CTRL \+ T|執行對話方塊來測試外觀及行為|  
|Format.ToggleGuides|CTRL \+ G|在對話方塊的無格線、導線和格線之間的切換|  
  
 如需將資源加入至 Managed 專案的詳細資訊，請參閱《.NET Framework 開發人員手冊》中的[應用程式中的資源](../Topic/Resources%20in%20Desktop%20Apps.md)。 如需手動將資源加入至 Managed 專案、存取資源、顯示靜態資源和指定屬性的資源字串等詳細資訊，請參閱[逐步解說：將 Windows Form 當地語系化](http://msdn.microsoft.com/zh-tw/9a96220d-a19b-4de0-9f48-01e5d82679e5)和[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)。
