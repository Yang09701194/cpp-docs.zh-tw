# Adding, Editing, or Deleting Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使用對話方塊編輯器，您可以加入、調整大小、編輯和刪除對話方塊中的控制項。  您也可以編輯控制項的屬性，例如，ID 或者是否在執行階段初始化時是可見的。  
  
 您可以在對話方塊編輯器中使用捷徑功能表，快速的將登錄的 ActiveX 控制項加入對話方塊，而且您可以將 ActiveX 控制項加入工具箱以便快速存取 \(如需詳細資訊，請參閱[自訂工具箱對話方塊](http://msdn.microsoft.com/zh-tw/bd07835f-18a8-433e-bccc-7141f65263bb)\)。  
  
-   [在對話方塊中加入控制項](../mfc/adding-a-control-to-a-dialog-box.md)  
  
-   [在您加入控制項時調整它的大小](../mfc/sizing-a-control-while-you-add-it.md)  
  
-   [加入多個控制項](../mfc/adding-multiple-controls.md)  
  
-   [加入 ActiveX 控制項至對話方塊](../mfc/viewing-and-adding-activex-controls-to-a-dialog-box.md)  
  
-   [編輯控制項的屬性](../mfc/editing-control-properties.md)  
  
-   [刪除控制項](../mfc/deleting-a-control.md)  
  
-   [顯示或隱藏工具箱視窗](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md)  
  
 如需將資源加入至 Managed 專案的詳細資訊，請參閱《.NET Framework 開發人員手冊》中的[應用程式中的資源](../Topic/Resources%20in%20Desktop%20Apps.md)。 如需手動將資源加入至 Managed 專案、存取資源、顯示靜態資源和指定屬性的資源字串等詳細資訊，請參閱[逐步解說：將 Windows Form 當地語系化](http://msdn.microsoft.com/zh-tw/9a96220d-a19b-4de0-9f48-01e5d82679e5)和[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)。
