# 應用程式架構
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Microsoft Foundation Class \(MFC\) 程式庫的核心是 Windows API 的一個主要部分的套件以 C\+\+ 形式。  程式庫類別代表視窗、對話方塊、裝置內容的 Windows 控制代碼、通用 GDI 物件 \(例如筆刷、畫筆，控制項和其他標準視窗項目。  這些類別提供方便的 C\+\+ 成員函式介面將它們封裝之視窗的結構。  如需更多關於使用這些類別的詳細資訊，請參閱 [視窗物件主題](../mfc/window-objects.md)。  
  
 但是， MFC 程式庫也提供在 Windows 應用程式開發介面的 C\+\+ 套件建立的其他應用程式功能層。  此圖層是提供大部分通用使用者介面 \(UI\) 預期 Windows 程式，包括工具列、狀態列、列印、列印預覽、資料庫支援和 ActiveX 支援視窗中運作的應用程式架構。  [使用撰寫的類別的 Windows 應用程式](../mfc/using-the-classes-to-write-applications-for-windows.md) 詳細說明架構。
