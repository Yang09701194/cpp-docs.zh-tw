# Arrangement of Controls on Dialog Boxes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

對話方塊編輯器提供配置工具，可自動對齊控制項和調整控制項的大小。  對於大多數工作，您可以使用[對話方塊編輯器工具列](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md)。  \[格式\] 功能表中也包含了所有的對話方塊編輯器工具列命令，而且大多數都有[快速鍵](../mfc/accelerator-keys-for-the-dialog-editor.md)。  
  
 許多對話方塊的配置命令只有在選取一個以上的控制項時才能使用。  您可以選取一個控制項或多個控制項，而且選取一個以上的控制項時，依據預設您選取的第一個是「主」控項。  如需選取控制項和主控項的詳細資訊，請參閱[選取控制項](../mfc/selecting-controls.md)。  
  
 目前控制項的位置、高度和寬度會顯示在狀態列的右下角。  選取整個對話方塊時，狀態列會顯示整個對話方塊的位置，以及它的高度和寬度。
