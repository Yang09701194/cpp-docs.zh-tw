# <a name="class-library-overview"></a>類別庫概觀

程式庫的類別是以下按照下列類別：

   - [檢視類別 (架構)](../mfc/view-classes-architecture.md)

   - [框架視窗類別 (架構)](../mfc/frame-window-classes-architecture.md)

   - [文件樣板類別](../mfc/document-template-classes.md)
