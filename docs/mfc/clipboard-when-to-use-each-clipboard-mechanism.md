# 剪貼簿：何時使用每個剪貼簿機制
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

遵循在使用剪貼簿的方針:  
  
-   現在使用 OLE 剪貼簿機制以啟用新功能。  當標準剪貼簿 API 會維護時，這個 OLE 機制是未來的資料傳輸。  
  
-   請使用 OLE 剪貼簿機制，如果您正在撰寫 OLE 應用程式或您想要任何 OLE 功能，例如拖放。  
  
-   如果您提供 OLE 格式，請使用 OLE 剪貼簿機制。
