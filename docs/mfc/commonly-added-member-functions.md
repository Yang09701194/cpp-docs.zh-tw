# 常加入的成員函式
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

除了確定或取消之外，如果您的對話方塊包含按鈕，可以在您的對話方塊類別需要寫入訊息處理常式會產生的成員函式回應控制項通知訊息。  如需範例，請參閱 [Scribble](../top/visual-cpp-samples.md) 範例程式。  您也可以處理來自其他控制項通知訊息在您的對話方塊。
