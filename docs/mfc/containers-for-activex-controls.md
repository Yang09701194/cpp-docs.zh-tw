# <a name="containers-for-activex-controls"></a>ActiveX 控制項的容器

您可以在其他應用程式中使用 Visual C++ 中開發的 ActiveX 控制項,只要它們支援 ActiveX 控制控制控制。 許多 Microsoft 應用程式(從列出的版本開始)支援 ActiveX 控制控制。

>[!IMPORTANT]
> ActiveX 是一種不應用於新開發的傳統技術。 有關取代 ActiveX 的現代技術的詳細資訊,請參閱[ActiveX 控制件](activex-controls.md)。

> [!NOTE]
> 以下清單不是支援 ActiveX 控制項但表示測試中最常用的集的應用程式的完整清單:

支援 ActiveX 控制控制包含的應用程式包括:

- 網際網路瀏覽器 3.x 及更高

- 視覺基本 4.x 及更高

- 視覺C++ 4.x 及更高

- 存取 95 及更高

- Excel 97 及更高

- 字 97 和更高

- 存取 97 及更高

- 前頁 97 及更高

- PowerPoint 97 及更高

- 視覺 InterDev 97 及更高

以下是支援 ActiveX 控制遏制的非 Microsoft 應用程式:

- PowerBuilder

- 德爾 福

- C++建設者

- 網景導覽器的 NCompass 外掛程式
