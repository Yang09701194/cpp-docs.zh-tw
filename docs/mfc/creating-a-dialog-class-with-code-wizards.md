# 使用程式碼精靈建立對話方塊類別
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下表列出程式碼精靈會協助您管理的對話方塊相關的工作。  
  
### 對話方塊相關的工作。  
  
|工作|套用至..|  
|--------|-----------|  
|[建立新的 CDialog 衍生的類別](../mfc/creating-your-dialog-class.md) 以管理您的對話方塊中。|每個對話方塊。|  
|[對應 Windows 訊息](../mfc/handling-windows-messages-in-your-dialog-box.md) 至您的對話方塊類別。|您要管理的每個訊息。|  
|宣告類別成員變數以表示對話方塊的控制項。|產生文字或數值的每個控制項要從程式存取。|  
|於控制項和成員變數之間指定 [資料如何交換](../mfc/dialog-data-exchange-and-validation.md) 。|您要從程式存取的每個控制項。|  
|為成員變數指定 [驗證規則](../mfc/dialog-data-exchange-and-validation.md) 。|產生文字或數值的每個控制項，如果需要的話。|
