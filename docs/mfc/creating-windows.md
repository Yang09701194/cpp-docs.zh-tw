# 建立視窗
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

架構會自動建立您在框架配置所需的大部分視窗。  [文件\/檢視建立](../mfc/document-view-creation.md) 會顯示架構如何建立框架視窗與文件和檢視。  但是，應用程式可以建立自己的視窗—包含您的子視窗框架視窗或檢視—刪除架構提供的視窗以外。
