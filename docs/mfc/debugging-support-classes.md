# <a name="debugging-support-classes"></a>偵錯支援類別
MFC 提供下列類別，以協助您偵錯動態記憶體配置問題。  
  
 [CDumpContext](../mfc/reference/cdumpcontext-class.md)  
 提供的目的地，以診斷傾印。  
  
 [CMemoryState](../mfc/reference/cmemorystate-structure.md)  
 提供記憶體使用量的快照集的結構。 也可用來比較之前和以後記憶體快照。
