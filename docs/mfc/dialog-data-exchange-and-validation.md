# 對話方塊資料交換和驗證
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

對話資料交換 \(Dialog Data Exchange，DDX\) 是一個簡單的方法會初始化在對話方塊的控制項和使用者收集資料輸入。  對話資料驗證 \(DDV\) 是一種簡單方式驗證在對話方塊中的資料項目。  若要使用 DDX 和 DDV 在對話方塊中，請使用 [加入成員變數精靈](../ide/add-member-variable-wizard.md) 建立資料成員和設定其資料型別和指定驗證規則。
