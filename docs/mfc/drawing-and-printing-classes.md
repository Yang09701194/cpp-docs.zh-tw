# 繪圖和列印類別
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在 Windows 中，所有圖形輸出在呼叫裝置內容的真實繪圖區繪製 \(DC\)。  MFC 提供 Windows 繪圖工具的類別封裝 DC 的各種型別，以及封裝如 Bitmap、筆刷、畫筆調色盤。
