# <a name="drawing-and-printing-classes"></a>繪圖和列印類別

在 Windows 中，所有圖形輸出是在稱為裝置內容 (DC) 的虛擬繪圖區繪製。 MFC 提供封裝各種類型 DC 的類別，以及 Windows 繪圖工具的封裝，例如點陣圖、筆刷、調色盤和畫筆。
