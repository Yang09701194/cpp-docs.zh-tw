# Editing Properties for an ActiveX Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

不同廠商提供的 ActiveX 控制項可能有其自身的屬性和特性。  ActiveX 控制項的屬性會顯示在屬性視窗。  此外，任何由 ActiveX 控制項的作者所撰寫的屬性頁會在 \[屬性頁\] 對話方塊 \(若要檢視特定 ActiveX 控制項的屬性頁，按一下[屬性視窗](../Topic/Properties%20Window.md)中的 \[屬性頁\] 按鈕\)。  
  
 各種索引標籤會顯示在 ActiveX 控制項的屬性頁中，端賴於本身是 ActiveX 控制項一部分的屬性工作表 \(Property Sheet\)。  
  
> [!NOTE]
>  下列程序適用於使用屬性頁來編輯 ActiveX 控制項。  您也可以在新的 \[屬性視窗\] 瀏覽和編輯 ActiveX 屬性。  
  
### 若要編輯 ActiveX 控制項的屬性  
  
1.  選取 **ActiveX** 控制項。  
  
2.  在 \[檢視\] 功能表上，按一下 \[屬性頁\] 並且檢視屬性。  
  
3.  在 \[屬性頁\] 中根據需要做變更。  
  
 如需將資源加入至 Managed 專案的詳細資訊，請參閱《.NET Framework 開發人員手冊》中的[應用程式中的資源](../Topic/Resources%20in%20Desktop%20Apps.md)。 如需手動將資源加入至 Managed 專案、存取資源、顯示靜態資源和指定屬性的資源字串等詳細資訊，請參閱[逐步解說：將 Windows Form 當地語系化](http://msdn.microsoft.com/zh-tw/9a96220d-a19b-4de0-9f48-01e5d82679e5)和[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)。
