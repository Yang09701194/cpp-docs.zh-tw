# 檔案和資料庫類別
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

這些類別可讓您將資訊儲存到資料庫或磁碟檔案。  有三個集合資料庫類別 \(例如 OLE DB、ODBC 和 DAO \) 提供類似功能。  OLE DB 群組使用 OLE DB 實作並與 OLE DB 消費者樣板一起工作，DAO 群組使用資料存取物件實作，而 ODBC 群組使用開方資料庫連接實作。  也有一組作業的標準資料類別中，使用資料流和 HTML 資料流。  
  
 下列類別的分類支援資料永續性。  
  
-   [檔案 I\/O 類別](../mfc/file-i-o-classes.md)  
  
-   [OLE DB 類別](../mfc/ole-db-classes.md)  
  
-   [DAO 類別](../mfc/dao-classes.md)  
  
-   [ODBC 類別](../mfc/odbc-classes.md)
