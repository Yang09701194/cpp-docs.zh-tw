# 格線設定對話方塊 (圖示影像編輯器)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

可讓您指定影像的格線設定，以及在所編輯的影像上顯示格線。  格線對於編輯影像非常有用，但它並不儲存為影像本身的一部分。  
  
 **像素格線**  
 當選取時，會在影像編輯器中的每一個像素周圍顯示格線。  此格線僅在 4× 或以上的解析度顯示。  
  
 **磚狀格線**  
 當選取時，會在影像編輯器中的像素區塊周圍顯示格線 \(由格線間距值指定\)。  
  
 **Width**  
 指定每一個磚狀區塊的寬度。  這在繪製包含多個以一定間距排列之影像的點陣圖時非常有用。  
  
 **Height**  
 指定每一個磚狀區塊的高度。  這在繪製包含多個以一定間距排列之影像的點陣圖時非常有用。
