# <a name="how-the-framework-searches-message-maps"></a>架構如何搜尋訊息對應
架構會搜尋相符項目與內送訊息中的訊息對應資料表。 一旦您撰寫您希望類別處理，並寫入對應的處理常式的每個訊息的訊息對應項目時，架構會自動呼叫您的處理常式。 下列主題將說明訊息對應搜尋：  
  
-   [哪裡可以找到訊息對應](../mfc/where-to-find-message-maps.md)  
  
-   [衍生的訊息對應](../mfc/derived-message-maps.md)  
  
-   [將訊息、 命令 Id 或控制項 Id 的範圍對應至一個處理常式](../mfc/handlers-for-message-map-ranges.md)
