# <a name="making-owner-drawn-tabs"></a>建立主控描繪索引標籤

您可以將索引標籤控制項（[CTabCtrl](../mfc/reference/ctabctrl-class.md)）的個別專案定義為主控描繪專案。 如需詳細資訊，[請參閱 Windows SDK 中的主控](/windows/win32/Controls/tab-controls)描繪索引標籤。
