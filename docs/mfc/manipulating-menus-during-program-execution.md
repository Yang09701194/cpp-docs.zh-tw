# 程式執行期間管理功能表
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使用 `CMenu` 類別以即時操作功能表和功能表項目。  `CMenu` 會封裝 Windows `HMENU` 控制代碼並把功能表提供工作的成員函式。  
  
 參閱 [CMenu](../mfc/reference/cmenu-class.md) 類別概觀的詳細資料。
