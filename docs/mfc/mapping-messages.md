# 對應訊息
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

可以接收訊息或命令的每個架構類別有其訊息對應。架構會使用訊息對應連接訊息和命令到它們的處理函式。  衍生自類別的任何類別 `CCmdTarget` 可能有訊息對應。  其他文章詳細說明訊息對應並說明用法。  
  
 儘管名稱「訊息對應」，訊息對應處理訊息和命令，在 [訊息的類別](../mfc/message-categories.md)中所列訊息的三個分類。
