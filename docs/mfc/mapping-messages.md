# <a name="mapping-messages"></a>對應訊息

每個可以接收訊息或命令的架構類別皆有其自己的「訊息對應」。 架構會使用訊息對應將訊息和命令連接到它們的處理函式。 任何衍生自類別 `CCmdTarget` 的類別皆可有訊息對應。 其他文件會詳細解釋訊息對應並說明用法。

儘管名稱 「 訊息對應 」 訊息對應會處理訊息和命令，所有三個訊息中列出分類[訊息分類](../mfc/message-categories.md)。
