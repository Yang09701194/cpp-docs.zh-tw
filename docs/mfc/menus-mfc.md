# 功能表 (MFC)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC 提供可協助您使用功能表一起使用的兩個項目:  
  
-   在執行階段管理程式功能表的 [CMenu](../mfc/reference/cmenu-class.md) 類別。  使用`CMenu` 文件和這個範例學習如何有效使用 `CMenu` 。  
  
-   更新的功能表和工具列按鈕的機制:即時啟用或停用它們以符合目前的程式。  
  
 Visual C\+\+ 為了建立及編輯程式的功能表資源也提供了 [功能表編輯器](../mfc/menu-editor.md) 。
