# <a name="messages-and-commands-in-the-framework"></a>架構中的訊息和命令

針對 Microsoft Windows 所撰寫的應用程式都是 「 訊息導向 」。 為了回應事件，例如滑鼠點按的按鍵動作、 視窗移動等等，Windows 會將訊息傳送至適當的視窗。 Framework 應用程式處理 Windows 訊息，就像 Windows 的任何其他應用程式。 但是 framework 也提供一些增強功能可讓處理訊息更方便、 更易於維護，且更佳封裝。
