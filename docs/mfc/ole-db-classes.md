# <a name="ole-db-classes"></a>OLE DB 類別

MFC 中的 OLE DB 支援目前包含類別[COLEDBRecordView](../mfc/reference/coledbrecordview-class.md)。 `COleDBRecordView` 會透過直接連接至[CRowset](../data/oledb/crowset-class.md)物件的表單檢視，在控制項中顯示資料庫記錄。 如需 OLE DB 取用者範本的詳細資訊，請參閱[OLE DB 取用者範本清單](../data/oledb/ole-db-consumer-templates-reference.md)。
