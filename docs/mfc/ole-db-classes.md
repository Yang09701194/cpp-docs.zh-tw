# <a name="ole-db-classes"></a>OLE DB 類別

MFC 中的 OLE DB 支援目前類別所組成[COLEDBRecordView](../mfc/reference/coledbrecordview-class.md)。 `COleDBRecordView` 在控制項中，透過直接連線到 [表單] 檢視會顯示資料庫記錄[CRowset](../data/oledb/crowset-class.md)物件。 如需有關 OLE DB 消費者範本的詳細資訊，請參閱 <<c0> [ 清單中的 OLE DB 消費者樣板](../data/oledb/ole-db-consumer-templates-reference.md)。
