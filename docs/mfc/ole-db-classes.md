# OLE DB 類別
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在 MFC 的 OLE DB 支援目前包含 [COLEDBRecordView](../mfc/reference/coledbrecordview-class.md)類別。  `COleDBRecordView` 透過表單檢視控制項中顯示的資料庫記錄，直接連接至 [CRowset](../data/oledb/crowset-class.md) 物件。  如需 OLE DB 消費者樣板的詳細資訊，請參閱 [OLE DB 消費者樣板清單。](../data/oledb/ole-db-consumer-templates-reference.md)。
