# OLE (MFC)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在您的程式中實作 OLE 功能會影響使用者介面使用幾種方式：  
  
-   視覺編輯 \(就地啟動\) 顯示其他程式使用者介面在程式中的視窗並從其他程式修改您的項目的程式功能表。  
  
-   拖放功能可讓使用者拖曳物件在視窗之間甚至程式之間。  
  
-   在視覺化編輯和拖放作業期間，追蹤提供視覺提示給物件狀態。
