# <a name="ole-mfc"></a>OLE (MFC)

在您的程式中實作 OLE 功能會影響您的使用者介面，以數種方式：

- 視覺化編輯 （就地啟用） 會在您的程式視窗中顯示其他程式的使用者介面，並修改您的程式功能表與另一個程式中的項目。

- 拖放允許使用者拖曳物件內和 windows 之間及程式甚至之間。

- 追蹤器視覺化編輯期間提供的物件狀態的視覺提示和拖放。
