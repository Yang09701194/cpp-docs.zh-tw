# <a name="ole-mfc"></a>OLE (MFC)
在您的程式中實作 OLE 功能，會影響您的使用者介面，幾種方式：  
  
-   視覺化編輯 （就地啟用） 在您的程式視窗中顯示其他程式的使用者介面，並修改您的程式功能表與其他程式中的項目。  
  
-   拖放允許使用者拖曳物件內，在視窗間之間，甚至程式。  
  
-   追蹤器提供視覺化編輯期間的視覺提示以物件的狀態和拖放。
