# <a name="adding-an-mfc-class"></a>加入 MFC 類別
若要新增至您的專案從 Microsoft Foundation Class (MFC) 程式庫類別衍生的類別，請使用**加入類別**命令可從[類別檢視](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925)。 指定新類別的名稱、 選取的基底類別，並選取 [與相關聯，所以 （如果有的話）] 對話方塊中的識別碼。 程式碼精靈建立標頭檔與實作檔，並將它們加入至您的專案。  
  
> [!NOTE]
>  您可以將 MFC 類別加入 ATL COM 應用程式，如果您一開始[建立應用程式與 MFC 支援](../../atl/reference/mfc-support-in-atl-projects.md)。 您也可以加入至 Win32 專案 MFC 支援的 MFC 類別。  
  
### <a name="to-add-an-mfc-class-to-your-project"></a>若要加入您的專案中的 MFC 類別  
  
1.  從 類別檢視，以滑鼠右鍵按一下專案名稱。 按一下**新增**，然後按一下 [**加入類別**開啟[加入類別](../../ide/add-class-dialog-box.md)] 對話方塊。  
  
2.  在 範本 窗格中，選取**MFC 類別**按**新增** 按鈕。  
  
3.  定義新的類別中的設定[MFC 類別精靈](../../mfc/reference/mfc-add-class-wizard.md) 對話方塊。  
  
4.  按一下**完成**来關閉精靈並在 類別檢視中檢視新的類別。 您也可以檢視中的精靈所建立的檔案**方案總管 中**。
