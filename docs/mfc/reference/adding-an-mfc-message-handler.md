# 加入 MFC 訊息處理常式
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

您可以[使用屬性視窗加入訊息處理常式](../../mfc/reference/mapping-messages-to-functions.md) \(一種用來處理 Windows 訊息的成員 \(Member\) 函式\) 至類別中，並將 Windows 訊息對應到訊息處理常式 \(Message Handler\)。  您也可以加入[任何對話方塊控制項的事件處理常式](../../mfc/adding-event-handlers-for-dialog-box-controls.md)。  
  
 藉由使用 \[屬性\] 視窗定義訊息和事件處理函式，可以自動更新訊息分派表格 \(或訊息對應 \(Message Map\)\) 和類別標頭檔。  
  
> [!NOTE]
>  您可以使用 \[屬性\] 視窗將訊息處理常式加入至 ATL 類別；然而，有些結果可能有所不同。  如需詳細資訊，請參閱 ATL 主題的[加入 ATL 訊息處理常式](../../atl/adding-an-atl-message-handler.md)。
