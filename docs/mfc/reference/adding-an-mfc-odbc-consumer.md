# 加入 MFC ODBC 消費者
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

MFC ODBC 消費者是由 ODBC 資料錄集 \(Recordset\) 類別 \(Class\) 和存取資料來源所需的資料繫結 \(Data Binding\) 所組成的。  
  
### 若要加入 MFC ODBC 消費者  
  
1.  在 \[類別檢視\] 中以滑鼠右鍵按一下專案。  在捷徑功能表中，按一下 \[加入\]，然後再按一下 \[加入類別\]。  
  
2.  在 \[Visual C\+\+\] 資料夾中按兩下 \[MFC ODBC 消費者\] 圖示，或選取它並按一下 \[開啟\]。  
  
     開啟 MFC ODBC 消費者精靈。  
  
3.  如 [MFC ODBC 消費者精靈](../../mfc/reference/mfc-odbc-consumer-wizard.md)中的描述來定義設定。  
  
4.  按一下 \[完成\] 以關閉精靈，如此一來就會將剛建立的 ODBC 消費者程式碼插入至您的專案中。
