# <a name="adding-an-mfc-odbc-consumer"></a>加入 MFC ODBC 消費者
MFC ODBC 消費者包含 ODBC 資料錄集類別和資料繫結的必要存取資料來源。  
  
### <a name="to-add-an-mfc-odbc-consumer"></a>若要加入 MFC ODBC 消費者  
  
1.  在 類別檢視，以滑鼠右鍵按一下專案。 在捷徑功能表，按一下 **新增**，然後按一下**加入類別**。  
  
2.  在 Visual c + + 資料夾中，按兩下**MFC ODBC 消費者**圖示或選取它，然後按一下 **開啟**。  
  
     MFC ODBC 消費者精靈 隨即開啟。  
  
3.  定義設定中所述[MFC ODBC 消費者精靈](../../mfc/reference/mfc-odbc-consumer-wizard.md)。  
  
4.  按一下**完成**關閉精靈，將您的專案中插入新建立的 ODBC 消費者程式碼。
