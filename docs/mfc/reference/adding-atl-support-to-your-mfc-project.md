# 將 ATL 支援加入至 MFC 專案
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

如果己經建立 MFC 架構應用程式，您就可藉由「將 ATL 支援加入 MFC 專案精靈」來輕易地加入 Active Template Library \(ATL\) 支援。  
  
> [!NOTE]
>  ATL 和 MFC 在 Visual Studio 的 Express 版本通常不支援。  
  
> [!NOTE]
>  這個支援只適用於加入至 MFC 可執行檔或 DLL 專案中的簡單 COM 物件。  您可將其他 COM 物件 \(包括 ActiveX 控制項\) 加入至 MFC 專案中，但是物件可能無法如預期運作。  
  
### 若要將 ATL 支援加入至您的 MFC 專案中  
  
1.  在 \[方案總管\] 中，以滑鼠右鍵按一下要加入 ATL 支援的專案。  
  
2.  在捷徑功能表中按一下 \[加入\]，然後按一下 \[加入類別\]。  
  
3.  選取 \[將 ATL 支援加入至 MFC 專案\] 圖示。  
  
    > [!NOTE]
    >  這個圖示位於 \[**分類**\] 窗格的 \[ATL\] 資料夾中。  
  
4.  當出現提示時，按一下 \[**是**\] 以加入 ATL 支援。  
  
 如需有關加入 ATL 支援如何變更 MFC 專案程式碼的詳細資訊，請參閱 [ATL 精靈加入 ATL 支援的詳細資訊](../../mfc/reference/details-of-atl-support-added-by-the-atl-wizard.md)。
