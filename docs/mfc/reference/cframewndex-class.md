# <a name="cframewndex-class"></a>CFrameWndEx 類別

實作 Windows 單一文件介面 (SDI) 重疊或快顯框架視窗的功能，並提供管理視窗的成員。 它會擴充[CFrameWnd](../../mfc/reference/cframewnd-class.md)類別。

## <a name="syntax"></a>語法

```
class CFrameWndEx : public CFrameWnd
```

##  <a name="enablefullscreenmode"></a>CFrameWndEx：： EnableFullScreenMode

啟用框架視窗的全螢幕模式。

```
void EnableFullScreenMode(UINT uiFullScreenCmd);
```

### <a name="parameters"></a>參數

*uiFullScreenCmd*<br/>
在啟用和停用全螢幕模式之命令的識別碼。

### <a name="remarks"></a>備註

在全螢幕模式中，所有停駐控制列、工具列和功能表都會隱藏起來，而使用中的視圖會調整大小以佔用全螢幕。

當您啟用全螢幕模式時，您必須指定命令的識別碼，以啟用或停用全螢幕模式。 您可以從主框架的 `OnCreate` 函數呼叫 `EnableFullScreenMode`。 當框架視窗切換至全螢幕模式時，架構會建立一個具有指定命令識別碼的浮動工具列，其中包含一個按鈕。

如果您想要在螢幕上保留主功能表，請呼叫[CFrameWndEx：： EnableFullScreenMainMenu](#enablefullscreenmainmenu)。
