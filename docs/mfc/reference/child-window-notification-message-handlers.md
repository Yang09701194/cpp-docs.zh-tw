有五個類別的子視窗通知訊息：  
  
|分類|描述|  
|--------------|-----------------|  
|[泛型控制處理常式](../../mfc/reference/generic-control-handler.md)|如需一般控制項通知碼的處理常式。|  
|[使用者按鈕處理常式](../../mfc/reference/user-button-handlers.md)|使用者按鈕通知碼的處理常式。|  
|[下拉式方塊處理常式](../../mfc/reference/combo-box-handlers.md)|下拉式方塊通知碼的處理常式。|  
|[編輯控制處理常式](../../mfc/reference/edit-control-handlers.md)|如需編輯控制項通知碼的處理常式。|  
|[清單方塊處理常式](../../mfc/reference/list-box-handlers.md)|清單方塊通知碼的處理常式。|
