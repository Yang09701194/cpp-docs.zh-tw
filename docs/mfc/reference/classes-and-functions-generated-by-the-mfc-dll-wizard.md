# <a name="classes-and-functions-generated-by-the-mfc-dll-wizard"></a>MFC DLL 精靈產生的類別和函式

MFC DLL 精靈產生的代碼取決於要建立的 DLL 類型和您選擇的選項。 MFC DLL 精靈為兩種形式的常規 MFC DLL 生成相同的代碼。

|Kind of DLL|選項|類別|函式|
|-----------------|------------|-------------|---------------|
|[分機](../../build/extension-dlls-overview.md)|None|None|`DllMain`|
|[定期](../../build/regular-dlls-dynamically-linked-to-mfc.md)|None|派生自`CWinApp`|None|
|[定期](../../build/regular-dlls-dynamically-linked-to-mfc.md)|自動化|派生自`CWinApp`|`DllGetClassObject` `DllCanUnloadNow` `DllRegisterServer`|
|[分機](../../build/extension-dlls-overview.md)|視窗通訊端|None|`DllMain`|
|[定期](../../build/regular-dlls-dynamically-linked-to-mfc.md)|視窗通訊端|派生自`CWinApp`|`InitInstance`包含對`AfxSocketInit`|
