# <a name="cmfcribboncategory-class"></a>CMFCRibbonCategory 類別

```  
class CMFCRibbonCategory : public CObject  
```

## <a name="remarks"></a>備註  
 一般而言，您建立功能區分類間接藉由呼叫[CMFCRibbonBar::AddCategory](../../mfc/reference/cmfcribbonbar-class.md#addcategory)，這會傳回新建立的功能區類別的指標。 將面板新增至類別目錄的藉由呼叫[CMFCRibbonCategory::AddPanel](#addpanel)。  
  
 `CMFCRibbonTab`類別繪製功能區分類。 它衍生自[CMFCRibbonBaseElement 類別](../../mfc/reference/cmfcribbonbaseelement-class.md)。  
  
 下列範例示範如何建立功能區分類和面板新增至它。  
  
 `// Create a new ribbon category and get a pointer to it`  
  
 `CMFCRibbonCategory* pCategory = m_wndRibbonBar.AddCategory`  
  
 `(_T("&Write"),           // Category name`  
  
 `IDB_WRITE,              // Category small images (16 x 16)`  
  
 `IDB_WRITE_LARGE);   // Category large images (32 x 32)`  
  
 `// Add a panel to the new category`  
  
 `CMFCRibbonPanel* pPanel = pCategory->AddPanel (`  
  
 `_T("Clipboard"),                       // Panel name`  
  
 `m_PanelIcons.ExtractIcon (0));  // Panel icon`

##  <a name="addpanel"></a>  CMFCRibbonCategory::AddPanel  
 建立功能區面板的功能區分類。  
  
```  
CMFCRibbonPanel* AddPanel(
    LPCTSTR lpszPanelName,  
    HICON hIcon = 0,  
    CRuntimeClass* pRTI = NULL);
```  
  
### <a name="parameters"></a>參數  
*lpszPanelName*<br/>
[in]新的功能區面板名稱的指標。  
  
*hIcon*<br/>
[in]新的功能區面板的預設圖示的控制代碼。  
  
*pRTI*<br/>
[in]自訂功能區面板的執行階段類別資訊指標。  
  
### <a name="return-value"></a>傳回值  
 如果方法成功; 的新功能區面板的指標如果不建立 [面板] 中，否則為 NULL。  
  
### <a name="remarks"></a>備註  
 如果您想要建立自訂功能區面板，您必須指定在其執行階段類別資訊*pRTI*。 自訂功能區面板類別必須衍生自`CMFCRibbonPanel`類別。  
  
 空間不足，無法顯示功能區項目時，會顯示功能區面板的預設圖示。

##  <a name="cmfcribboncategory"></a>  CMFCRibbonCategory::CMFCRibbonCategory  
 建構並初始化[CMFCRibbonCategory](../../mfc/reference/cmfcribboncategory-class.md)物件。  
  
```  
CMFCRibbonCategory(
    CMFCRibbonBar* pParenrRibbonBar,  
    LPCTSTR lpszName,  
    UINT uiSmallImagesResID,  
    UINT uiLargeImagesResID,  
    CSize sizeSmallImage = CSize(16,
    16),  
    CSize sizeLargeImage = CSize(32,
    32));
```  
  
### <a name="parameters"></a>參數  
*pParenrRibbonBar*<br/>
[in]功能區分類的父功能區列的指標。  
  
*lpszName*<br/>
[in]功能區分類的名稱。  
  
*uiSmallImagesResID*<br/>
[in]在 功能區類別的功能區項目所使用的小型影像的影像清單的資源識別碼。  
  
*uiLargeImagesResID*<br/>
[in]功能區類別中的功能區項目所使用的大型映像的映像清單的資源識別碼。  
  
*sizeSmallImage*<br/>
[in]預設的功能區中的項目功能區分類的小型影像的大小。  
  
*sizeLargeImage*<br/>
[in]預設的功能區中的項目功能區分類的大型映像的大小。

##  <a name="setcollapseorder"></a>  CMFCRibbonCategory::SetCollapseOrder  
 定義功能區面板的功能區分類的摺疊的順序。  
  
```  
void SetCollapseOrder(const CArray<int,int>& arCollapseOrder);
```  
  
### <a name="parameters"></a>參數  
*arCollapseOrder*<br/>
[in]指定的摺疊的順序。 陣列包含的功能區面板之以零起始的索引。  
  
### <a name="remarks"></a>備註  
 程式庫定義摺疊的順序。 不過，您可以自訂此行為，藉由提供類別，指定的摺疊順序清單中的索引。  
  
 當類別偵測到它已摺疊功能區面板時，它會尋找指定的清單中的下一個項目。 如果清單是空的或您沒有指定足夠的項目，類別目錄會使用內部演算法。  
  
 例如，類別有三個功能區面板，並且可以摺疊數次直到所有面板完全摺疊狀態。 您可以設定下列的摺疊順序： 0、 0、 2、 2。 在此情況下，類別目錄會摺疊面板 0 兩次，面板 2 2 倍。 索引為 1 的面板會保持展開項目。
