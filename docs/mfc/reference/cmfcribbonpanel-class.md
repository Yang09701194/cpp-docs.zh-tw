# <a name="cmfcribbonpanel-class"></a>CMFCRibbonPanel 類別

實作包含一組功能區項目的面板。 繪製面板時，會在指定的面板大小下，顯示盡可能多的項目。

如需詳細資訊，請參閱位於**VC\\atlmfc\\src\\** Visual Studio 安裝的 mfc 資料夾中的原始程式碼。

## <a name="syntax"></a>語法

```
class CMFCRibbonPanel : public CObject
```

## <a name="remarks"></a>備註

功能區面板是您在功能區類別中建立之相關工作的邏輯分組。 當功能區的大小變更時，面板版面配置會自動調整，盡可能顯示最多的元素。

您可以藉由呼叫[CMFCRibbonCategory：： GetPanel](../../mfc/reference/cmfcribboncategory-class.md#getpanel)方法，取得包含在功能區分類中的功能區面板。

##  <a name="add"></a>CMFCRibbonPanel：： Add

將指定的功能區專案附加至功能區面板中包含的功能區元素陣列。

```
virtual void Add(CMFCRibbonBaseElement* pElem);
```

### <a name="parameters"></a>參數

*pElem*<br/>
[in、out]功能區元素的指標。

### <a name="remarks"></a>備註

##  <a name="addseparator"></a>CMFCRibbonPanel：： AddSeparator

將分隔符號加入至功能區面板。

```
virtual void AddSeparator();
```

### <a name="remarks"></a>備註

呼叫這個方法，將分隔符號加入功能區面板。 分隔符號會加入至先前呼叫[CMFCRibbonPanel：： Add](#add)所加入的功能區專案旁邊。 若要在指定的位置插入分隔符號，請呼叫[CMFCRibbonPanel：： InsertSeparator](#insertseparator)。

##  <a name="addtoolbar"></a>CMFCRibbonPanel：： AddToolBar

將工具列加入至功能區面板。

```
CMFCRibbonButtonsGroup* AddToolBar(
UINT uiToolbarResID,
UINT uiColdResID = 0,
UINT uiHotResID = 0,
UINT uiDisabledResID = 0);
```

### <a name="parameters"></a>參數

*uiToolbarResID*<br/>
在指定要加入之工具列的資源識別碼。

*uiColdResID*<br/>
在指定工具列冷影像的資源識別碼。

*uiHotResID*<br/>
在指定工具列的熱影像的資源識別碼。

*uiDisabledResID*<br/>
在指定工具列已停用映射的資源識別碼。

### <a name="return-value"></a>傳回值

呼叫這個方法，將工具列加入至功能區面板。 工具列將會新增至先前呼叫[CMFCRibbonPanel：： Add](#add)所新增的功能區元素旁邊。

### <a name="remarks"></a>備註

如需工具列、熱影像、冷映射和已停用映射的詳細資訊，請參閱[CMFCToolBar 類別](../../mfc/reference/cmfctoolbar-class.md)。

*pPaletteButton*<br/>
