# <a name="cmfcribbonpanel-class"></a>CMFC 剪彩面板類

實作包含一組功能區項目的面板。 繪製面板時，會在指定的面板大小下，顯示盡可能多的項目。

有關詳細資訊,請參閱位於 Visual Studio 安裝的**VC\\\\\\atlmfc src mfc**資料夾中的原始程式碼。

## <a name="syntax"></a>語法

```
class CMFCRibbonPanel : public CObject
```

## <a name="remarks"></a>備註

功能區面板是您在功能區類別中創建的相關任務的邏輯分組。 隨著功能區大小的更改,面板佈局會自動調整以顯示盡可能多的元素。

通過調用[CMFC 功能區類別::getPanel](../../mfc/reference/cmfcribboncategory-class.md#getpanel)方法,可以獲取功能區類別中包含的功能區面板。

## <a name="cmfcribbonpaneladdtoolbar"></a><a name="addtoolbar"></a>CMFC 功能面板:新增工具列

向功能區面板添加工具列。

```
CMFCRibbonButtonsGroup* AddToolBar(
UINT uiToolbarResID,
UINT uiColdResID = 0,
UINT uiHotResID = 0,
UINT uiDisabledResID = 0);
```

### <a name="parameters"></a>參數

*uiToolbarResID*<br/>
[在]指定要新增的工具列的資源識別碼。

*烏伊庫德雷斯ID*<br/>
[在]指定工具列冷影像的資源識別碼。

*烏霍特雷斯ID*<br/>
[在]指定工具列熱影像的資源識別碼。

*ui禁用雷斯代碼*<br/>
[在]指定工具列禁用影像的資源識別碼。

### <a name="return-value"></a>傳回值

調用此方法以向功能區面板添加工具列。 工具列將新增到上一次呼叫[CMFCRibbonPanel 新增](#add)的功能區元素旁邊::新增 。

### <a name="remarks"></a>備註

有關工具列、熱影像、冷圖像和禁用影像的詳細資訊,請參閱[CMFCToolBar 類別](../../mfc/reference/cmfctoolbar-class.md)。

CRuntimeClass* pRTC);
```

### <a name="parameters"></a>參數

*烏伊CmdID*<br/>
[在]指定要新增的功能區元素的命令 ID。

*pRTC*<br/>
[進出]指向與添加到功能區面板的功能區元素關聯的運行時類資訊的指標。

如果要向功能區面板添加自定義元素(例如顏色按鈕),則必須指定自定義元素的運行時類資訊。 功能區存儲此資訊、創建自定義元素並替換由指定命令 ID 定位的現有元素。 然後,它返回指向新創建元素的指標。

### <a name="example"></a>範例

下面的範例展示如何使用方法`SetElementRTCByID`:

```

// Load and add toolbar with standard buttons. This toolbar
// should display a custom color button with id ID_CHAR_COLOR:

pPanel->AddToolBar(IDR_MAINFRAME,
    IDB_MAINFRAME256);

CMFCRibbonColorButton* pColorButton =
(CMFCRibbonColorButton*)pPanel->SetElementRTCByID(
ID_CHAR_COLOR,
    RUNTIME_CLASS (CMFCRibbonColorButton));

// SetElementRTCByID sets runtime class and returns a pointer
// to the newly created custom button,
    which can be set up immediately:
pColorButton->EnableAutomaticButton(_T("Automatic"),
    RGB (0,
    0,
