# <a name="cmfctabctrl-class"></a>CMFCTabCtrl Class

`CMFCTabCtrl`類別會提供索引標籤控制項的功能。 索引標籤控制項會顯示頂端或底部有一般或 3D 索引標籤的可停駐視窗。 索引標籤可以顯示文字和影像，當在使用中狀態時，也可變更色彩。

## <a name="syntax"></a>語法

```
class CMFCTabCtrl : public CMFCBaseTabCtrl
```

## <a name="remarks"></a>備註

`CMFCTabCtrl`類別支援:

- 包含3D、平面和平面且具有共用水準捲軸的索引標籤控制項樣式。

- 位於視窗頂端或底部的索引標籤。

- 顯示文字、影像或文字和影像的索引標籤。

- 當索引標籤為作用中時, 會變更色彩的索引標籤。

- 可調整索引標籤的框線大小變更。

- 可分離的索引標籤式視窗。

`CMFCTabCtrl`類別可以搭配對話方塊使用, 但適用于使用固定控制列 (例如 Microsoft Excel 和 Visual Studio) 的應用程式。 如需詳細資訊, 請參閱[CDockablePane 類別](../../mfc/reference/cdockablepane-class.md)。

請遵循下列步驟, 在您的應用程式中新增可調整大小的停駐索引標籤控制項:

1. 建立[CTabbedPane 類別](../../mfc/reference/ctabbedpane-class.md)的實例。

1. 呼叫[CDockablePane:: Create](../../mfc/reference/cdockablepane-class.md#create)。

1. 使用[CBaseTabbedPane:: AddTab](../../mfc/reference/cbasetabbedpane-class.md#addtab)或[CMFCBaseTabCtrl:: InsertTab](../../mfc/reference/cmfcbasetabctrl-class.md#inserttab)來加入新的索引標籤。

1. 呼叫[CBasePane:: EnableDocking](../../mfc/reference/cbasepane-class.md#enabledocking) , 讓目前的停駐索引標籤控制項可以在主框架視窗中停駐。

1. 呼叫[CFrameWndEx::D ockpane](../../mfc/reference/cframewndex-class.md#dockpane) , 將索引標籤式視窗停駐在主要畫面格上。

如需如何建立索引標籤式視窗做為停駐控制列的範例, 請參閱[CTabbedPane 類別](../../mfc/reference/ctabbedpane-class.md)。 若要`CMFCTabCtrl`使用做為非固定的控制項, 請`CMFCTabCtrl`建立物件, 然後呼叫[CMFCTabCtrl:: create](#create)。
