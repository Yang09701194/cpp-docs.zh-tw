# <a name="cmfctabctrl-class"></a>CMFCTabCtrl Class

類`CMFCTabCtrl`提供選項卡控制項的功能。 索引標籤控制項會顯示頂端或底部有一般或 3D 索引標籤的可停駐視窗。 索引標籤可以顯示文字和影像，當在使用中狀態時，也可變更色彩。

## <a name="syntax"></a>語法

```
class CMFCTabCtrl : public CMFCBaseTabCtrl
```

## <a name="remarks"></a>備註

該`CMFCTabCtrl`類別支援:

- 選項卡控件樣式,包括 3D、平面和平面,帶有共用水平滾動條。

- 位於視窗頂部或底部的選項卡。

- 顯示文字、圖像或文字和圖像的選項卡。

- 選項卡處於活動狀態時更改顏色的選項卡。

- 可調整選項卡的邊框大小更改。

- 可拆卸的選項卡式視窗。

該`CMFCTabCtrl`類可與對話框一起使用,但適用於使用銜接站控制列(如Microsoft Excel和 Visual Studio)的應用程式。 有關詳細資訊,請參閱[可多克窗格類別](../../mfc/reference/cdockablepane-class.md)。

依以下步驟在應用程式中新增可調整大小的停靠選項卡控制項:

1. 創建[CTabbedPane 類](../../mfc/reference/ctabbedpane-class.md)的實例。

1. 呼叫[可嵌入的單窗格: 建立](../../mfc/reference/cdockablepane-class.md#create)。

1. 使用[CBaseTabbed 窗格::添加 Tab](../../mfc/reference/cbasetabbedpane-class.md#addtab)或[CMFCBaseTabCtrl::插入選項卡](../../mfc/reference/cmfcbasetabctrl-class.md#inserttab)以添加新選項卡。

1. 呼叫[CBasePane:啟用停靠](../../mfc/reference/cbasepane-class.md#enabledocking),以便當前停靠選項卡控制項可以停靠在主框架視窗。

1. 調用[CFrameWndEx::DockPane](../../mfc/reference/cframewndex-class.md#dockpane)將選項卡式視窗停靠在主框架上。

有關如何建立選項卡式視窗作為停靠控制項列的範例,請參閱[CTabbedPane 類別](../../mfc/reference/ctabbedpane-class.md)。 要用作`CMFCTabCtrl`非停靠控制項,請建立一`CMFCTabCtrl`個 物件,然後呼叫[CMFCTabCtrl::建立](#create)。

## <a name="cmfctabctrlondragenter"></a><a name="ondragenter"></a>CMFCTabCtrl::OnDragenter

    COleDataObject* pDataObject,
    DWORD dwKeyState,
    CPoint point);
```

### <a name="parameters"></a>參數

[在]指向包含使用者拖動的數據的數據物件。

virtual BOOL SetImageList(HIMAGELIST hImageList);
```

### <a name="parameters"></a>參數

*uiID*<br/>
[在]包含影像清單的點陣圖資源的識別碼。
