 A`CMFCToolBarMenuButton`可以顯示為功能表項目具有子功能表功能表、 按鈕，或是執行命令會顯示功能表或顯示功能表上的按鈕。 您的功能表按鈕的外觀與行為來指定參數，例如影像、 文字、 功能表的控制代碼，然後命令建構函式中的按鈕相關聯的識別碼`CMFCToolbarMenuButton::CMFCToolbarMenuButton`。  
  
 自訂的類別衍生自`CMFCToolbarMenuButton`類別必須使用[DECLARE_SERIAL](run-time-object-model-services.md#declare_serial)巨集。 [DECLARE_DYNCREATE](run-time-object-model-services.md#declare_dyncreate)巨集產生錯誤，應用程式關閉時。

##  <a name="cmfctoolbarmenubutton"></a>  CMFCToolBarMenuButton::CMFCToolBarMenuButton  
 建構 `CMFCToolBarMenuButton` 物件。  
  
```  
CMFCToolBarMenuButton();
CMFCToolBarMenuButton(const CMFCToolBarMenuButton& src);

CMFCToolBarMenuButton(
    UINT uiID,  
    HMENU hMenu,  
    int iImage,  
    LPCTSTR lpszText=NULL,  
    BOOL bUserButton=FALSE);
```  
  
### <a name="parameters"></a>參數  
 [in]*src*  
 現有`CMFCToolBarMenuButton`物件複製到這個`CMFCToolBarMenuButton`物件。  
  
 [in]*uiID*  
 當使用者按一下按鈕; 若要執行的命令識別碼或 ( `UINT`) 不會直接執行命令的功能表按鈕的-1。  
  
 [in]*hMenu*  
 功能表; 的控點或`NULL`如果按鈕沒有功能表。  
  
 [in]*iImage*  
 按鈕; 映像的索引則為-1，如果此按鈕，並沒有圖示，或針對所指定的命令會使用圖示*uiID*。 索引是針對每個相同`CMFCToolBarImages`應用程式中的物件。  
  
 [in]*lpszText*  
 工具列功能表按鈕的文字。  
  
 [in]*bUserButton*  
 `TRUE` 如果按鈕顯示的使用者定義的映像。`FALSE`如果按鈕會顯示與所指定的命令相關聯的預先定義的映像*uiID*。  
  
### <a name="remarks"></a>備註  
 如果*uiID*是有效的命令識別碼，當使用者按一下它時，按鈕會執行該命令。 如果*hMenu*是有效的功能表的控制代碼，當它出現在功能表工具列或子功能表中出現時，按鈕會提供下拉式選單。 如果兩個*uiID*和*hMenu*有效，是將執行的命令，當使用者按一下部分與向下箭號，將下拉功能表與部分與分割按鈕時的使用者當它。 不過，如果*hMenu*有效，使用者不能按一下 插入 功能表按鈕時執行的指令 按鈕。
