# <a name="cmfcvisualmanager-class"></a>CMFCVisualManager 類別
提供在全域層級上變更應用程式外觀的支援。 `CMFCVisualManager` 類別會使用一致的樣式，與提供繪製應用程式 GUI 控制項指令的類別搭配使用。 這些其他類別稱為視覺管理員，並且繼承自 `CMFCBaseVisualManager`。  
  
## <a name="syntax"></a>語法  
  
```  
class CMFCVisualManager : public CMFCBaseVisualManager  
```

## <a name="remarks"></a>備註  
 因為`CMFCVisualManager`類別會控制應用程式的 GUI，每個應用程式可擁有的任一個一個執行個體`CMFCVisualManager`，或從衍生類別的一個執行個體`CMFCVisualManager`。 您的應用程式也可以運作，不需要`CMFCVisualManager`。 使用靜態方法`GetInstance`以取得目前的指標`CMFCVisualManager`-衍生物件。  
  
 若要變更您的應用程式的外觀，您必須使用其他類別可提供繪製應用程式的視覺元素的所有方法。 這些類別的一些範例包括[CMFCVisualManagerOfficeXP 類別](../../mfc/reference/cmfcvisualmanagerofficexp-class.md)， [CMFCVisualManagerOffice2003 類別](../../mfc/reference/cmfcvisualmanageroffice2003-class.md)，和[CMFCVisualManagerOffice2007 類別](../../mfc/reference/cmfcvisualmanageroffice2007-class.md)。 當您想要變更您的應用程式的外觀時，將這些視覺管理員的其中一個傳遞至方法`SetDefaultManager`。 如需示範如何應用程式可以模擬 Microsoft Office 2003 的外觀的範例，請參閱[CMFCVisualManagerOffice2003 類別](../../mfc/reference/cmfcvisualmanageroffice2003-class.md)。  
  
 所有的繪圖方法都是虛擬。 這可讓您建立自訂的視覺化樣式的應用程式的 GUI。 如果您想要建立您自己的視覺化樣式，從其中一個視覺管理員類別衍生類別並覆寫繪製您想要變更的方法。

 使用`CMFCVisualManager`類別和任何衍生以自訂您的應用程式外觀的視覺管理員。 設定預設的視覺管理員之後，這個方法來重新繪製您的應用程式使用新的視覺管理員。 如需如何使用視覺管理員的詳細資訊，請參閱[視覺化管理員](../../mfc/visualization-manager.md)。  
  
 若要變更應用程式所使用的視覺管理員使用這個方法。  
  
##  <a name="setembossdisabledimage"></a>CMFCVisualManager::SetEmbossDisabledImage  
 啟用或停用已停用的工具列影像的浮凸的模式。  
  
```  
void SetEmbossDisabledImage (BOOL bEmboss = TRUE);
```  
  
### <a name="parameters"></a>參數  
 [in] `bEmboss`  
 布林值的參數，指出是否要啟用已停用的工具列影像的浮凸的模式。  
  
### <a name="remarks"></a>備註  
 使用函數[CMFCVisualManager::IsEmbossDisabledImage](#isembossdisabledimage)來判斷是否要啟用浮凸的模式。  
  
##  <a name="setfadeinactiveimage"></a>CMFCVisualManager::SetFadeInactiveImage  
 啟用或停用功能表或工具列上的非作用中的映像光源效果。  
  
```  
void SetFadeInactiveImage(BOOL bFade = TRUE);
```  
  
### <a name="parameters"></a>參數  
 [in] `bFade`  
 布林值的參數，指定是否要啟用光源效果。  
  
### <a name="remarks"></a>備註  
 此功能用於控制功能表或工具列上顯示非作用中的影像是否取消。 使用方法[CMFCVisualManager::IsFadeInactiveImage](#isfadeinactiveimage)來判斷是否已啟用此功能。  
  
##  <a name="setmenuflatlook"></a>CMFCVisualManager::SetMenuFlatLook  
 設定旗標，指出是否出現一般功能表按鈕。 否則，它們會出現&3;d。  
  
```  
void SetMenuFlatLook(BOOL bMenuFlatLook = TRUE);
```  
  
### <a name="parameters"></a>參數  
 [in] `bMenuFlatLook`  
 布林值的參數，指出是否出現一般功能表按鈕。  
  
### <a name="remarks"></a>備註  
 根據預設，不會啟用此功能。  
  
##  <a name="setmenushadowdepth"></a>CMFCVisualManager::SetMenuShadowDepth  
 設定寬度和高度功能表陰影。  
  
```  
void SetMenuShadowDepth(int nDepth);
```  
  
### <a name="parameters"></a>參數  
 [in] `nDepth`  
 像素為單位指定的功能表陰影深度的整數。  
  
### <a name="remarks"></a>備註  
 必須是相同的高度和寬度功能表陰影。 預設值是 7 個像素。  
  
##  <a name="setshadowhighlightedimage"></a>CMFCVisualManager::SetShadowHighlightedImage

```  
void SetShadowHighlightedImage(BOOL bShadow = TRUE);
```  
  
### <a name="parameters"></a>參數
