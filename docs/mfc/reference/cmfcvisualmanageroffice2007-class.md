# CMFCVisualManagerOffice2007 Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

## 語法  
  
```  
class CMFCVisualManagerOffice2007 : public CMFCVisualManagerOffice2003  
```

## 備註  
 使用 `CMFCVisualManagerOffice2007` 變更應用程式的視覺外觀類似 Microsoft Office 2007。  在使用它之前，這個視覺管理員都需要您設定樣式。  當您透過這個視覺管理員為 `CMFCVisualManager::SetDefaultManager`之前，必須先呼叫靜態函式 `CMFCVisualManagerOffice2007::SetStyle`。
