# CMFCVisualManagerOfficeXP Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

`CMFCVisualManagerOfficeXP` 寫入應用程式一次 Microsoft Office XP 外觀。  
  
## 語法  
  
```  
class CMFCVisualManagerOfficeXP : public CMFCVisualManager  
```  
  
## Members  
 這個類別的所有成員都是自這個類別的祖系衍生的虛擬函式， [CMFCVisualManager Class](../../mfc/reference/cmfcvisualmanager-class.md)。  
  
## 備註  
 您可以使用 `CMFCVisualManagerOfficeXP` 類別變更應用程式的視覺外觀類似 Microsoft Office XP。
