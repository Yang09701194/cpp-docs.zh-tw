# <a name="cmfcvisualmanagerofficexp-class"></a>CMFCVisualManagerOfficeXP 類別
`CMFCVisualManagerOfficeXP`為應用程式提供 Microsoft Office XP 外觀。  
  
## <a name="syntax"></a>語法  
  
```  
class CMFCVisualManagerOfficeXP : public CMFCVisualManager  
```  
  
## <a name="members"></a>成員  
 這個類別的成員都衍生自這個類別的祖系的虛擬函式[CMFCVisualManager 類別](../../mfc/reference/cmfcvisualmanager-class.md)。  
  
## <a name="remarks"></a>備註  
 您使用`CMFCVisualManagerOfficeXP`類別，以變更您的應用程式類似 Microsoft Office XP 的視覺外觀。
