# CString 格式和訊息方塊顯示
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

提供一些函式格式化和剖析 `CString` 物件。  您可以使用這些函式，每當您必須管理 `CString` 物件，但是，用於格式化要顯示在訊息方塊文字的字串特別有用。  
  
 函式的這個群組也包含說明的訊息方塊全域常式。  
  
### CString 函式  
  
|||  
|-|-|  
|[AfxExtractSubString](../Topic/AfxExtractSubString.md)|從指定的來源字串擷取單一字元分隔的子字串。|  
|[AfxFormatString1](../Topic/AfxFormatString1.md)|以格式字元「替代特定字串在字串資料表內包含的字串。|  
|[AfxFormatString2](../Topic/AfxFormatString2.md)|替代的兩個字串的格式字元「和「在字串資料表中包含了。|  
|[AfxMessageBox](../Topic/AfxMessageBox.md)|顯示訊息方塊。|
