# <a name="data-types-mfc"></a>資料類型 (MFC)

本主題列出最常在 MFC 程式庫中使用的資料類型。 大部分的資料類型與 Platform Software Development Kit (SDK) 中的相同，其他的則是 MFC 獨有。

如需 Windows SDK 和 MFC 中使用之資料類型的詳細資訊, 請參閱[Windows 資料類型](/windows/win32/WinProg/windows-data-types)。

MFC 程式庫獨有的資料類型如下：

- 放置用來表示集合中專案位置的值;由 MFC 集合類別使用。

- LPCRECT 常數 (不可變) `RECT`結構的32位指標。
