# DHTML 編輯命令對應
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

下列巨集可以用來將在 [CHtmlEditView](../../mfc/reference/chtmleditview-class.md)中進行編輯的 DHTML 命令衍生類別。  如需其用法的範例，請參閱 [HTMLEdit 範例](../../top/visual-cpp-samples.md)。  
  
### 編輯命令對應巨集的 DHTML  
  
|||  
|-|-|  
|[DECLARE\_DHTMLEDITING\_CMDMAP](../Topic/DECLARE_DHTMLEDITING_CMDMAP.md)|宣告編輯類別的 DHTML 命令對應。|  
|[BEGIN\_DHTMLEDITING\_CMDMAP](../Topic/BEGIN_DHTMLEDITING_CMDMAP.md)|開始編輯類別內的 DHTML 的定義命令對應。|  
|[END\_DHTMLEDITING\_CMDMAP](../Topic/END_DHTMLEDITING_CMDMAP.md)|標記編譯命令對應的 DHTML 的結束。|  
|[DHTMLEDITING\_CMD\_ENTRY](../Topic/DHTMLEDITING_CMD_ENTRY.md)|將命令 ID 的編輯命令的 HTML。|  
|[DHTMLEDITING\_CMD\_ENTRY\_FUNC](../Topic/DHTMLEDITING_CMD_ENTRY_FUNC.md)|將命令 ID 的編輯命令和訊息處理常式的 HTML。|  
|[DHTMLEDITING\_CMD\_ENTRY\_TYPE](../Topic/DHTMLEDITING_CMD_ENTRY_TYPE.md)|將命令 ID 的編輯命令和使用者介面項目的 HTML。|  
|[DHTMLEDITING\_CMD\_ENTRY\_FUNC\_TYPE](../Topic/DHTMLEDITING_CMD_ENTRY_FUNC_TYPE.md)|將命令 ID 的編輯命令、訊息處理常式和使用者介面項目的 HTML。|
