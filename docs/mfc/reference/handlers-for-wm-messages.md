|[P-R](../../mfc/reference/wm-messages-p-r.md)|**ON_WM_PAINT**透過**ON_WM_RENDERFORMAT**|  
|[S](../../mfc/reference/wm-messages-s.md)|**ON_WM_SETCURSOR**透過**ON_WM_SYSKEYUP**|
