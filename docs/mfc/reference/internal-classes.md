# <a name="internal-classes"></a>內部類別

下列類別可在 MFC 中的內部。 為求完整起見，本節會說明這些內部類別，但是它們並不適合直接在您的程式碼中使用。

## <a name="in-this-section"></a>本節內容

|類別|描述|
|-----------|-----------------|
|`CCommandManager Class`|`CCommandManager`類別會管理命令和其關聯與映像。|
|`CDialogImpl Class (MFC)`|提供實作詳細資料 對話方塊方塊為基礎的類別，例如`CDialogEx`。|
|`CDocumentAdapter Class`|搜尋和組織的處理常式所需的實作 IDocument 介面。|
|`CFrameImpl Class`|`CFrameImpl`類別會處理下列的框架視窗類別的工具列自訂：[CFrameWndEx 類別](../../mfc/reference/cframewndex-class.md)， [CMDIFrameWndEx 類別](../../mfc/reference/cmdiframewndex-class.md)，以及[COleIPFrameWndEx 類別](../../mfc/reference/coleipframewndex-class.md)。|
|`CFullScreenImpl Class`|實作全螢幕功能通用於 SDI 和 MDI 應用程式。|
|`CMDIClientAreaWnd Class`|`CMDIClientAreaWnd`類別是一個 helper 類別，可簡化 MDI 索引標籤和 MDI 索引群組的實作。|
|`CMemDC Class`|協助程式類別記憶體裝置內容。 記憶體裝置內容支援螢幕的繪圖。|
|`CMenuHash Class`|讀取的狀態`CMFCToolBar`從封存物件或狀態寫入封存。|
|`CMenuImages Class`|提供的功能，以顯示預先定義的映像，例如 [關閉] 按鈕、 最大化按鈕、 選項按鈕和箭號。|
|`CMFCCaptionButtonEx Class`|保存系統標題按鈕的非工作區。|
|`CMFCCaptionMenuButton Class`|代表標題區域的工作窗格或工具列中的功能表按鈕。|
|`CMFCColorPropertySheet Class`|供[CMFCColorDialog 類別](../../mfc/reference/cmfccolordialog-class.md)提供索引標籤式的對話方塊。|
|`CMFCControlBarImpl Class`|提供實作詳細資料，以控制列類別，例如`CMFCReBar`和`CMFCToolBar`。|
|`CMFCControlRenderer Class`|協助程式類別來處理映像轉譯。|
|`CMFCControlRendererInfo Class`|協助程式類別來傳遞參數至`CMFCControlRenderer`類別。|
|`CMFCCustomizeButton Class`|支援功能表命令或工具列按鈕會在自訂對話方塊。|
|`CMFCCustomizeMenuButton Class`|表示自訂功能表上的單一按鈕。|
|`CMFCDropDownListBox Class`|提供功能區下拉式方塊的下拉式清單方塊功能。|
|`CMFCMousePropertyPage Class`|可讓使用者自訂的滑鼠行為的屬性頁面。|
|`CMFCOutlookBarPaneAdapter Class`||
|`CMFCOutlookBarPaneButton Class`|實作 framework 中新增的按鈕`CMFCOutlookBarPane`物件。|
|`CMFCOutlookBarPaneList Class`|擴充`CMFCOutlookBarPane`類別，以提供 Outlook 視覺化樣式。|
|`CMFCOutlookBarScrollButton Class`|提供捲動功能`CMFCOutlookBarTabCtrl`類別。|
|`CMFCOutlookBarToolBar Class`|會實作具有 Outlook 2003 的視覺化樣式的工具列。|
