# 網際網路 URL 剖析全域
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

當用戶端將查詢傳送至網際網路伺服器時，您可以使用 URL 剖析全域來擷取與用戶端相關的資訊。  
  
### 網際網路 URL 剖析全域  
  
|||  
|-|-|  
|[AfxParseURL](../Topic/AfxParseURL.md)|剖析 URL 字串並傳回服務型別及其元件。|  
|[AfxParseURLEx](../Topic/AfxParseURLEx.md)|剖析 URL 字串並傳回服務型別及其元件，並提供使用者名稱和密碼。|
