# <a name="mapping-messages-to-functions"></a>將訊息對應到函式

[類嚮導](mfc-class-wizard.md)使您能夠將消息處理程式(MFC 用戶介面類的成員函數)綁定到應用程式資源生成的消息。 它們使用[MFC 消息映射](../../mfc/messages-and-commands-in-the-framework.md)創建綁定。

當您使用 Class View 創建派生自其中一個框架類的新類時,它會自動在指定的標頭 (.h) 和實現 (.cpp) 檔中放置一個完整的函數類。

> [!NOTE]
> 要添加不處理消息的新類,請直接在文字編輯器中創建該類。

### <a name="to-define-or-remove-a-message-handler-using-the-class-wizard"></a>使用類別精靈定義或移除訊息處理程式

1. 在**類視圖中**,右鍵單擊類。

1. 在上下文選單中,選擇[類別精靈](mfc-class-wizard.md)。
