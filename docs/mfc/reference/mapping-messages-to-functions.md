# <a name="mapping-messages-to-functions"></a>將訊息對應到函式

[類別 Wizard](mfc-class-wizard.md)可讓您將訊息處理常式（MFC 使用者介面類別別的成員函式）系結至應用程式資源所產生的訊息。 它們會使用[MFC 訊息對應](../../mfc/messages-and-commands-in-the-framework.md)來建立系結。

當您使用類別檢視建立衍生自其中一個架構類別的新類別時，它會自動將完整的功能類別放在您指定的標頭（.h）和執行檔（.cpp）中。

> [!NOTE]
>  若要加入不會處理訊息的新類別，請直接在文字編輯器中建立類別。

### <a name="to-define-or-remove-a-message-handler-using-the-class-wizard"></a>若要使用類別 Wizard 定義或移除訊息處理常式

1. 在**類別檢視**中，以滑鼠右鍵按一下類別。

1. 在內容功能表中，選擇 [[類別] [Wizard]](mfc-class-wizard.md)。
