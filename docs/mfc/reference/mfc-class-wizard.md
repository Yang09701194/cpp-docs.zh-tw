# <a name="mfc-class-wizard"></a>MFC 類別精靈

使用 [**類別 Wizard]** 建立新的 MFC 類別，或將訊息和訊息處理常式加入至專案中的現有類別。

有三種方式可以開啟**類別嚮導**：

- 在 [**專案**] 功能表上, 選擇 [**類別 Wizard]** 。
- 輸入**Ctrl**  > ShiftX > 。
- 在**類別檢視**中，以滑鼠右鍵按一下類別或專案節點，然後選擇 [**類別] [Wizard]** 。

![類別 Wizard](media/class-wizard.png "MFC 類別 Wizard")

## <a name="uielement-list"></a>UIElement 清單

- **Project**

   方案中專案的名稱。

   您可以從下拉式清單方塊中選取方案中的其他專案。

- **類別名稱**

   專案中的類別名稱。

   當您在 [**類別名稱**] 清單中選取類別時，類別中的資料會填入**MFC 類別 Wizard**中的控制項。 當您變更控制項的值時，所選類別中的資料會受到影響。

- **新增類別**

   可讓您將新的類別加入至 MFC 專案。

- **基底類別**

   類別**名稱**中顯示之類別的基類。

- **類別宣告**

   宣告**類別名稱**類別的類別。

   只有在其中的名稱與**類別實**中的名稱不同時，才會顯示 [**類別**宣告] 方塊。

- **Resource**

   **類別名稱**中資源的識別碼（如果有的話）。 否則，**資源**箱會是空的。

- **類別執行**

   檔案的名稱，該檔案包含**類別名稱**中的類別的執行。

   您可以按一下箭號來選取不同的執行檔。 下表列出可用的選項。

   |選項|說明|
   |------------|-----------------|
   |**開啟檔案**|結束類別 wizard 並開啟目前的類別執行檔。|
   |**開啟包含資料夾**|開啟包含目前類別執行檔案的資料夾。|
   |**將完整路徑複製到剪貼簿**|將目前的執行檔路徑複製到剪貼簿。|

- **指令**

   可讓您新增、刪除、編輯或搜尋命令和其訊息處理常式。

   若要加入處理常式，請按一下 [**加入處理常式**]，或按兩下 [**物件識別碼**] 清單或 [**訊息**] 清單中的專案。 產生的函數名稱、識別碼和訊息會顯示在**成員**函式清單中。

   若要刪除處理常式，請在 [**成員**函式] 清單中選取一個專案，然後按一下 [**刪除處理常式**]。

   若要修改處理常式，請按兩下 [**成員**函式] 清單中的對應專案。 或者，在清單方塊中選取一個專案，然後按一下 [**編輯程式碼**]。

- **訊息**

   可讓您加入、刪除、編輯或搜尋訊息和其訊息處理常式。

   若要加入處理常式，請按一下 [**加入處理常式**]，或按兩下 [**訊息**] 清單中的專案。

   若要加入自訂訊息，請按一下 [**新增自訂訊息**] 或按 enter 鍵，然後在 [**新增自訂訊息**] 對話方塊中指定值。 在該對話方塊中，您也可以選取 [**已註冊的訊息**] 來處理 windows 訊息，保證在整個作業系統中都是唯一的。

- **虛擬函式**

   可讓您加入、刪除、編輯或搜尋虛擬函式，或是覆寫的虛擬函式。

- **成員變數**

   可讓您加入、刪除、編輯或搜尋成員變數。

- **方法**

   可讓您新增、刪除或搜尋方法，也可以移至方法的定義或宣告。

   若要新增方法，請按一下 [**加入方法**]，然後在 [**加入方法**] 對話方塊中指定值。

   若要刪除方法，請在 [**方法**] 清單中選取一個專案，然後按一下 [**刪除方法**]。

   若要顯示宣告，請在 [**方法**] 清單中選取一個專案，然後按一下 [**移至**宣告]。

   若要顯示定義，請按兩下 [**方法**] 清單中的專案。 或者，在 [**方法**] 清單中選取一個專案，然後按一下 [**移至定義**] 按鈕。
