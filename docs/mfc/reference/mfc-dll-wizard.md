# <a name="mfc-dll-wizard"></a>MFC DLL 精靈
當您使用 MFC DLL 精靈建立 MFC DLL 專案時，您會得到工作起始應用程式具有內建功能，，編譯時，將實作的基本功能[DLL](../../build/dlls-in-visual-cpp.md)。 MFC 入門程式包括 c + + 來源 (.cpp) 檔案、 資源 (.rc) 檔，以及專案 (.vcxproj) 檔案。 起始檔案中產生的程式碼是以 MFC 為基礎。 如需詳細資訊，請參閱您的專案在 Visual Studio 中，針對所產生的 Readme.txt 中的檔案詳細資料和[類別和 MFC DLL 精靈所產生的函式](../../mfc/reference/classes-and-functions-generated-by-the-mfc-dll-wizard.md)  
  
## <a name="overview"></a>總覽  
 這個精靈頁面說明目前[MFC DLL 專案的應用程式設定](../../mfc/reference/application-settings-mfc-dll-wizard.md)您所建立。 根據預設，為規則 （MFC 共用） 的 MFC DLL 專案的任何其他設定建立專案。  
  
 若要變更這些預設值，請按一下**應用程式設定**左側的 MFC DLL 精靈頁面中的精靈並變更資料行中。  
  
 建立 MFC DLL 專案之後，您可以將物件或控制項加入您專案中使用 Visual c + +[程式碼精靈](../../ide/adding-functionality-with-code-wizards-cpp.md)。  
  
 您可以執行下列工作和類型的基本的 MFC DLL 專案的增強功能：  
  
-   [從 DLL 匯出](../../build/exporting-from-a-dll.md)  
  
-   [可執行檔連結至 DLL](../../build/linking-an-executable-to-a-dll.md)  
  
-   [初始化 DLL](../../build/run-time-library-behavior.md#initializing-a-dll)
