# <a name="mfc-dll-wizard"></a>MFC DLL 精靈

當您使用 MFC DLL 精靈建立 MFC DLL 專案時，得到工作起始應用程式的內建功能，當編譯時，會實作的基本功能[DLL](../../build/dlls-in-visual-cpp.md)。 MFC 入門版計畫包含 c + + 原始檔 (.cpp) 檔案、 資源 (.rc) 檔，以及專案 (.vcxproj) 檔案。 這些入門檔案中產生的程式碼是以 MFC 為基礎。 如需詳細資訊，請參閱檔案中的詳細資料為您的專案在 Visual Studio 中產生的 Readme.txt 和[類別和 MFC DLL 精靈所產生的函式](../../mfc/reference/classes-and-functions-generated-by-the-mfc-dll-wizard.md)

## <a name="overview"></a>總覽

這個精靈頁面說明目前[MFC DLL 專案的應用程式設定](../../mfc/reference/application-settings-mfc-dll-wizard.md)您所建立。 預設情況下，專案會建立為一般 （MFC 共用） 的 MFC DLL 專案的任何其他設定。

若要變更這些預設值，請按一下**應用程式設定**精靈並進行變更的 MFC DLL 精靈頁面中的左資料行中。

建立 MFC DLL 專案之後，您可以將物件或控制項加入您的專案使用 Visual c + +[程式碼精靈](../../ide/adding-functionality-with-code-wizards-cpp.md)。

您可以執行下列工作和類型的基本 MFC DLL 專案的增強功能：

- [從 DLL 匯出](../../build/exporting-from-a-dll.md)

- [連結至 DLL 的可執行檔](../../build/linking-an-executable-to-a-dll.md)

- [初始化 DLL](../../build/run-time-library-behavior.md#initializing-a-dll)
