# Remote Automation 執行緒模型
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

遠端 Automation 支援 COM 單一執行緒 Apartment 和無限制執行緒模型。  在 32 位元和 64 位元平台上，是 Visual C\+\+ 支援，應該將執行緒模型為「Apartment」。如需執行這項作業的詳細資訊，方式和位置。請參閱 [建立使用遠端自動化的程式](../mfc/creating-programs-that-use-remote-automation.md)。
