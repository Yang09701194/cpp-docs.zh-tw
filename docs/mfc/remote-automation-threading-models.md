# <a name="remote-automation-threading-models"></a>Remote Automation 執行緒模型
Remote Automation 支援 COM 的單一執行緒模型和 Apartment 執行緒模型。 在 Visual C++ 所支援的 32 位元和 64 位元平台上，應該將執行緒模型設定為「Apartment」。 如需有關如何及如何執行這項操作，請參閱[建立程式使用 Remote Automation 的](../mfc/creating-programs-that-use-remote-automation.md)。
