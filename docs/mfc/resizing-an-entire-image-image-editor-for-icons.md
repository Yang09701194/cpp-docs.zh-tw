# Resizing an Entire Image (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### 若要使用屬性視窗調整整個影像的大小  
  
1.  開啟要變更屬性的影像。  
  
2.  在[屬性視窗](../Topic/Properties%20Window.md)中的 \[寬度\] 和 \[高度\] 方塊內，輸入您要的大小。  
  
     如果您要增加影像的大小，影像編輯器會向右、向下 \(或兩者\) 擴充影像，並以目前的背景色彩填滿新的區域。  影像不會自動縮放。  
  
     如果您要縮小影像的大小，影像編輯器會裁剪影像的右邊或底部 \(或兩者\)。  
  
    > [!NOTE]
    >  您可使用寬度和高度屬性，只調整整個影像的大小，而不調整部分選取範圍的大小。
