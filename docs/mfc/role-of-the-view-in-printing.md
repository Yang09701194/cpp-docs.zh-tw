# 檢視在列印中的角色
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

您要在列印相關的文件也兩扮演重要角色。  
  
 檢視：  
  
-   在印表機上使用相同的 [OnDraw](../Topic/CView::OnDraw.md) 程式碼繪製在螢幕上繪製。  
  
-   將文件的 Manages 成列印的頁面。  
  
 如需列印的詳細資訊以及列印的檢視的作用，請參閱 [列印和預覽](../mfc/printing-and-print-preview.md)。
