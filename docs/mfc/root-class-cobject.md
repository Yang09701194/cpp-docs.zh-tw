# 根類別：CObject
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Microsoft Foundation Class \(MFC\) 程式庫中的大部分的類別從類別階層架構的根的單一基底類別衍生。  `CObject` 以非常低的額外負荷提供從它衍生的任何類別許多有用的功能。  如需 `CObject` 及其功能的詳細資訊，請參閱 [使用 CObject](../mfc/using-cobject.md)。  
  
 [CObject](../mfc/reference/cobject-class.md)  
 大部分的 MFC 類別的基底類別。  序列化資料和取得類別支援的執行階段資訊。  
  
 [CRuntimeClass](../mfc/reference/cruntimeclass-structure.md)  
 用於判斷物件的實際類別在執行階段的結構。
