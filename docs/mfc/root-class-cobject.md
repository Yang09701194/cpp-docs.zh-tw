# <a name="root-class-cobject"></a>根類別：CObject

Microsoft Foundation Class （MFC）程式庫中的大部分類別都是衍生自類別階層根目錄的單一基類。 `CObject` 為所有衍生自它的類別提供許多有用的功能，而且額外負荷極低。 如需 `CObject` 及其功能的詳細資訊，請參閱[使用 CObject](../mfc/using-cobject.md)。

[CObject](../mfc/reference/cobject-class.md)<br/>
大部分 MFC 類別的最終基類。 支援序列化資料並取得有關類別的執行時間資訊。

[CRuntimeClass](../mfc/reference/cruntimeclass-structure.md)<br/>
用來在執行時間判斷物件之確切類別的結構。
