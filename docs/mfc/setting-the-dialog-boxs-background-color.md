# <a name="setting-the-dialog-boxs-background-color"></a>設定對話方塊的背景色彩

您可以藉由處理對話方塊視窗的 WM_CTLCOLOR 訊息，來設定對話方塊的背景色彩。 設定的色彩只會用於指定的對話方塊。

如需範例，請參閱[codexpert blog](https://codexpert.ro/blog/2013/03/13/painting-the-dialog-backround/) 。
