# 設定對話方塊的背景色彩
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

您可以藉由處理對話方塊視窗的 `WM_CTLCOLOR` 訊息，設定自己的對話方塊的背景色彩。  色彩設定為僅於指定的對話方塊中使用。
