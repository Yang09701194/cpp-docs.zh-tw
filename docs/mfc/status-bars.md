# <a name="status-bars"></a>狀態列

狀態列會提供您的應用程式可向使用者顯示訊息和有用的資訊，而不會中斷使用者的工作。 通常顯示在視窗底部，狀態列會有 「 窗格 」 包括 「 指示器 」 和 「 訊息列 」。 指標可讓 SCROLL LOCK 的這類事件的狀態是否巨集錄製已開啟，依此類推。 [狀態] 列上的訊息列可以顯示使用者指向上方的工具列按鈕或功能表項目或程式狀態相關資訊。

選取您的程式中建立狀態列**初始狀態列**MFC 應用程式精靈中的選項。
