# 索引標籤控制項和屬性工作表
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

屬性工作表是多頁對話方塊或「可以顯示 24 對話方塊樣板資源給使用者的選項對話方塊」。  以屬性工作表的範例，請參閱 Windows 說明屬性對話方塊或下列 MFC 範例應用程式:  
  
 [CMNCTRL1:示範通用控制項類別，第 1 部分](../top/visual-cpp-samples.md)  
  
 [CMNCTRL2:示範通用控制項類別，第 2 部分](../top/visual-cpp-samples.md)  
  
 使用 MFC 類別 [CPropertySheet](../mfc/reference/cpropertysheet-class.md)，屬性工作表可以輕易地實作。
