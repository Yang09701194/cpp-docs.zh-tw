# 陣列、清單和對應的樣板類別
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

這些集合類別是參數所識別的存放的物件類型的範本。  `CArray`、 `CMap`和 `CList` 類別使用通常必須自訂的全域 Helper 函式。  如需這些 Helper 函式的詳細資訊，請參閱 [集合類別 Helper](../mfc/reference/collection-class-helpers.md)。  具型別指標類別是在類別庫中其他類別的包裝函式。  使用這些包裝函式，您協助編譯器的型別檢查可讓您避免錯誤。  如需使用這些類別的詳細資訊，請參閱 [集合](../mfc/collections.md)。  
  
 這些類別提供可用來建立陣列，清單和對應使用想要的任何型別的範本。  
  
 [CArray](../mfc/reference/carray-class.md)  
 製造任意型別的陣列的樣板類別。  
  
 [CList](../mfc/reference/clist-class.md)  
 製造任意型別的列表的樣板類別。  
  
 [CMap](../mfc/reference/cmap-class.md)  
 製造具有任意索引鍵和值型別的對應的樣板類別。  
  
 [CTypedPtrArray](../mfc/reference/ctypedptrarray-class.md)  
 型別安全的指標陣列的樣板類別。  
  
 [CTypedPtrList](../mfc/reference/ctypedptrlist-class.md)  
 型別安全的指標列表的樣板類別。  
  
 [CTypedPtrMap](../mfc/reference/ctypedptrmap-class.md)  
 型別安全的對應及指標的樣板類別。
