# <a name="tn036-using-cformview-with-appwizard-and-classwizard"></a>TN036： 搭配搭配使用 CFormView 與 AppWizard 及 ClassWizard

本技術提示說明如何修改 AppWizard 產生應用程式，以便使用`CFormView`而非預設`CView`作為主要檢視類別。 這支援直接使用這個版本的 Visual C++。
