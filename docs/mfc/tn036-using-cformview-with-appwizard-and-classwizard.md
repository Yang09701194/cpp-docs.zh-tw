# <a name="tn036-using-cformview-with-appwizard-and-classwizard"></a>TN036：搭配使用 CFormView 與 AppWizard 及 ClassWizard

本技術提示說明如何修改「應用程式精靈」產生的應用程式，使其使用 `CFormView` 而非預設的 `CView` 作為其主要 view 類別。 這是此版本的 Visual C++的直接支援。
