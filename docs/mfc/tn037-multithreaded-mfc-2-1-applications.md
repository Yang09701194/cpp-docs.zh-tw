# <a name="tn037-multithreaded-mfc-21-applications"></a>TN037:多執行緒的 MFC 2.1 應用程式

此技術提示原本描述在 Windows NT 適用的 Visual C++ 1.0 原先所提供的 MFC 2.1 中，多執行緒應用程式的限制。 MFC 3.0 直接支援多執行緒並有文件記錄可供參考 如需詳細資訊，請參閱該參考。
