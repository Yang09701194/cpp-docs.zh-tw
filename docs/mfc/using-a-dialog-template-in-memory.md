而不是使用所提供的方法[ 對話方塊建立資料表](../mfc/creating-a-dialog-class-with-code-wizards.md)，您可以從對話方塊範本，在記憶體中間接建立對話方塊中的其中一種。 如需詳細資訊，請參閱類別[CDialog](../mfc/reference/cdialog-class.md)中*MFC 參考 》*。
