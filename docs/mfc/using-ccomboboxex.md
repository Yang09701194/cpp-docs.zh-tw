# 使用 CComboBoxEx
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

展開的下拉式方塊控制項是項目影像提供原生支援標準下拉式方塊控制項的擴充功能。  這些影像可以用來表示個別項目的狀態在下拉式方塊中，例如目前選取或未選取的項目。  若要使項目影像容易存取，控制項會提供影像清單支援。  
  
 使用這個控制項提供下拉式方塊的功能，而不必手動繪製項目圖形。
