# <a name="using-ccomboboxex"></a>使用 CComboBoxEx

擴充的下拉式方塊控制項是標準下拉式方塊控制項的提供項目影像的原生支援的延伸模組。 這些映像可用來表示下拉式方塊，例如目前的選取或未選取的項目中的個別項目的狀態。 若要讓項目影像更容易存取，控制會提供映像清單支援。

使用此控制項來提供下拉式方塊的功能，而不必以手動方式繪製項目的圖形。
