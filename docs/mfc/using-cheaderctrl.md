# 使用 CHeaderCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使用標題控制項，以類別 [CHeaderCtrl](../mfc/reference/cheaderctrl-class.md)，顯示一份分欄式清單的資料行行首。  例如，標題控制項為實作的控制項在報表。  
  
 標題控制項通常分為部分，稱為標題項目、」每個軸繼承文字關聯資料行的標題或數字。  根據樣式設定，您可以為使用者提供數種方式直接操作標頭項目。  
  
> [!NOTE]
>  [CListCtrl](../mfc/reference/clistctrl-class.md) 提供內嵌標題控制項和 [CListView](../mfc/reference/clistview-class.md) 封裝 `CListCtrl` 在 MFC 類別。  一般而言，會視為使用 `CHeaderCtrl` 對標誌目錄要繪製其本身。
