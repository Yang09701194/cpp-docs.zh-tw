# <a name="using-cheaderctrl"></a>使用 CHeaderCtrl
使用標題控制項，類別所代表[CHeaderCtrl](../mfc/reference/cheaderctrl-class.md)，以顯示資料行標頭，如需單欄式清單。 例如，標題控制項會是適用於實作試算表中的資料行控制。  
  
 標題控制項通常分為部分，稱為 「 標頭項目，「 每個具有相關聯的資料行的文字或數字的標題。 根據您設定的樣式，您可以提供直接的方式，讓使用者操作標頭項目的數目。  
  
> [!NOTE]
>  [CListCtrl](../mfc/reference/clistctrl-class.md)提供內嵌的標題控制項，以及[CListView](../mfc/reference/clistview-class.md)封裝`CListCtrl`MFC 類別中。 一般情況下，將使用`CHeaderCtrl`至您想要繪製自己的標籤清單。
