# 使用 CMonthCalCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

月曆控制項 \([CMonthCalCtrl](../mfc/reference/cmonthcalctrl-class.md)\) 實作類似行事曆的使用者介面。  這提供使用者輸入或選取日期非常容易了解和可辨識的方法。  使用現有的資料型別，控制項也提供應用程式以方法取得和設定控制項中的日期資訊。  根據預設，月曆控制項會顯示目前的日期和月份。  不過，使用者可以移動至之前和下月份和選取特定月份和年份。  
  
> [!NOTE]
>  若要使用 `CMonthCalCtrl` 類別在您的專案中，您可以在 STDAFX.H. 必須通常是包含 AFXDTCTL.H。
