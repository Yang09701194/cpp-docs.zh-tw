# 使用 CProgressCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

您可以使用進度列控制項來指示長時間作業的進度。  這是一個矩形，隨著作業的進度，會逐漸填滿系統的醒目提示色彩。  
  
 進度控制項在 MFC 類別 [CProgressCtrl](../mfc/reference/cprogressctrl-class.md) 表示。  
  
 當您一開始建立進度控制項時，您會指定它的大小和位置、父視窗 \(通常為對話方塊\) 和 ID。.  使用 `dwStyle` 參數，您可以為控制項和填滿的樣式指定各種視窗樣式。
