# <a name="using-cprogressctrl"></a>使用 CProgressCtrl

您可以使用進度列控制項來指示長時間作業的進度。 它是一個矩形，會隨著作業的進度逐漸填滿系統的醒目提示色彩。

進度控制項類別來表示在 MFC [CProgressCtrl](../mfc/reference/cprogressctrl-class.md)。

當您開始建立進度控制項時，您會指定它的大小和位置、父視窗 (通常為對話方塊) 和 ID。 藉由使用*cheaderctrl:: Create*參數，您也可以指定各種視窗樣式控制項和其填滿的樣式。
