# <a name="using-cricheditctrl"></a>使用 CRichEditCtrl

Rich edit 控制項是一個視窗，讓使用者輸入及編輯文字。 文字字元和段落格式，可指派，而且可以包含內嵌的 OLE 物件。 Rich edit 控制項台中由 MFC [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md)類別。
