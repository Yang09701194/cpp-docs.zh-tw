# 使用 CRichEditCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Rich Edit 控制項是使用者可以輸入和編輯文字的視窗。  文字是指定字元和段落格式，並且可以包含內嵌 OLE 物件。  Rich Edit 控制項在 MFC 表示由 [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md) 類別。
