# 使用 CTabCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

索引標籤控制項類似於筆記本裡的分隔頁或檔案櫃中的標籤資料夾。  使用以 [CTabCtrl](../mfc/reference/ctabctrl-class.md)類別表示的索引標籤控制項，在建構階段對等或邏輯關聯性各網頁間的格式時，一次顯示資訊或控制多頁給一個使用者。  
  
 如需選項的詳細資訊，請參閱 [!INCLUDE[winSDK](../atl/includes/winsdk_md.md)]的 [索引標籤控制項](http://msdn.microsoft.com/library/windows/desktop/bb760550) 。
