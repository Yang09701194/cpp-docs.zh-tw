# 使用 CToolBarCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

您可以使用工具列控制項 \([CToolBarCtrl](../mfc/reference/ctoolbarctrl-class.md)\) 會包含按鈕和選擇性空間控制項視窗。  在工具列控制項視窗中的每個按鈕會傳送命令訊息到父視窗，使用者選取它。  通常，在工具列的按鈕對應於應用程式的功能表項目，提供其他更直接的方式為使用者存取應用程式的命令。
