# 使用 CTreeCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

由 [CTreeCtrl](../mfc/reference/ctreectrl-class.md)類別表示的樹狀目錄控制項是顯示項目的階層式清單，例如中的文件的標題、在索引中輸入或檔案和目錄在磁碟上的視窗。  每個項目都包含標籤和選擇性點陣圖影像，因此，每個項目都可以有與自己相關的子項目清單。  您可以按一下項目以展開和摺疊子項目關聯的清單。  在檔案總管左方窗格中的目錄樹狀結構是樹狀目錄控制項的範例。
