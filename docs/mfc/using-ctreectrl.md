# <a name="using-ctreectrl"></a>使用 CTreeCtrl

樹狀目錄控制項（由類別[CTreeCtrl](../mfc/reference/ctreectrl-class.md)表示）是顯示專案階層式清單的視窗，例如檔中的標題、索引中的專案，或磁片上的檔案和目錄。 每個項目都包含標籤和選擇性點陣圖影像，因此，每個項目都可以擁有與自己相關的子項目清單。 使用者可以按一下項目以展開和摺疊子項目的關聯清單。 [檔案總管] 位於左方窗格中的目錄樹狀即是一種樹狀目錄控制項的範例。
