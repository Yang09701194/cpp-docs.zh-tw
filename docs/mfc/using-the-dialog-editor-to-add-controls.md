# <a name="using-the-dialog-editor-to-add-controls"></a>使用對話方塊編輯器加入控制項

當您建立您的對話方塊範本資源，與[對話方塊編輯器](../windows/dialog-editor.md)，您將控制項拖曳控制項調色盤中，放到對話方塊。 這樣會加入該控制項類型的規格的對話方塊範本資源。 當您建構對話方塊物件並呼叫其`Create`或`DoModal`成員函式，架構會建立 Windows 控制項，並將它放在螢幕上的 [對話方塊] 視窗中。

您可以改用[以手動方式建立控制項](../mfc/adding-controls-by-hand.md)如果您想要。 這是更多工作。
