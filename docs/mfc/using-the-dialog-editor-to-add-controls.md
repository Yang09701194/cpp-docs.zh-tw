# 使用對話方塊編輯器加入控制項
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

當您以 [對話方塊編輯器](../mfc/dialog-editor.md)時的對話方塊範本資源，將項目從色板控制項將控制項拖曳到對話方塊。  這會將該控制項型別的規格到對話方塊範本資源。  當您建構對話物件和呼叫其 **Create** 或 `DoModal` 成員函式時，架構會建立視窗控制項並將它放在對話方塊視窗的螢幕。  
  
 如果需要，您可以 [手動建立控制項](../mfc/adding-controls-by-hand.md) 。  這是更多工作。
