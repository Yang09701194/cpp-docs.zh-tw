# <a name="walkthroughs-mfc"></a>逐步解說 (MFC)
本節包含逐步解說與新的 MFC 程式庫功能相關聯的各種工作的文件。  
  
## <a name="in-this-section"></a>本節內容  
 [逐步解說：使用新的 MFC Shell 控制項](../mfc/walkthrough-using-the-new-mfc-shell-controls.md)  
 在此逐步解說中，您會建立類似檔案總管的應用程式。 您將建立包含兩個窗格的視窗。 左的窗格將包含[CMFCShellTreeCtrl](../mfc/reference/cmfcshelltreectrl-class.md)將桌面顯示階層式檢視中的物件。 在右窗格會包含[CMFCShellListCtrl](../mfc/reference/cmfcshelllistctrl-class.md) ，會在左窗格中選取的資料夾中顯示的檔案。  
  
 [逐步解說：將控制項放在工具列上](../mfc/walkthrough-putting-controls-on-toolbars.md)  
 現代 MFC 工具列可以裝載簡單按鈕之外的控制項。 本文說明如何操作。  
  
 [逐步解說：將 D2D 物件新增至 MFC 專案](../mfc/walkthrough-adding-a-d2d-object-to-an-mfc-project.md)  
 示範如何將 D2D 物件加入至專案，並介紹如何使用 D2D。  
  
 [逐步解說：將動畫新增至 MFC 專案](../mfc/walkthrough-adding-animation-to-an-mfc-project.md)  
 示範如何將圖形化的物件加入至專案，並介紹如何建立它的動畫。
