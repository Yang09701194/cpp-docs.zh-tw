# Win32 網際網路類別
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC 包裝 Win32 Internet \(WinInet\) 和 ActiveX 技術可以讓網際網路程式設計更容易。  
  
 [CInternetSession](../mfc/reference/cinternetsession-class.md)  
 建立並初始化一個網際網路工作階段或幾個同時網際網路工作階段和，如果需要，，描述與 Proxy 伺服器的連接。  
  
 [CInternetConnection](../mfc/reference/cinternetconnection-class.md)  
 管理您與網際網路伺服器的連接。  
  
 [CInternetFile](../mfc/reference/cinternetfile-class.md)  
 這個類別和其衍生類別允許對檔案的存取在使用網際網路通訊協定的遠端系統。  
  
 [CHttpConnection](../mfc/reference/chttpconnection-class.md)  
 管理您與 HTTP 伺服器的連接。  
  
 [CHttpFile](../mfc/reference/chttpfile-class.md)  
 在 HTTP 伺服器提供功能尋找和讀取檔案。  
  
 [CGopherFile](../mfc/reference/cgopherfile-class.md)  
 提供在 Gopher 伺服器上尋找和讀取檔案的功能。  
  
 [CFtpConnection](../mfc/reference/cftpconnection-class.md)  
 管理您與 FTP 伺服器的連接。  
  
 [CGopherConnection](../mfc/reference/cgopherconnection-class.md)  
 管理與 Gopher 伺服器。  
  
 [CFileFind](../mfc/reference/cfilefind-class.md)  
 執行本機和網際網路檔案搜尋。  
  
 [CFtpFileFind](../mfc/reference/cftpfilefind-class.md)  
 協助 FTP 伺服器的網際網路檔案搜尋。  
  
 [CGopherFileFind](../mfc/reference/cgopherfilefind-class.md)  
 協助網際網路檔案搜尋 Gopher 伺服器。  
  
 [CGopherLocator](../mfc/reference/cgopherlocator-class.md)  
 從 Gopher 伺服器取得 Gopher「定位器」，判斷定位器的類型，並將定位器提供給 `CGopherFileFind`。  
  
 [CInternetException](../mfc/reference/cinternetexception-class.md)  
 表示與網際網路作業相關的例外狀況。
