# <a name="win32-internet-extensions-wininet"></a>Win32 網際網路擴充功能 (WinInet)
網際網路用戶端應用程式是可從網路資料來源 （伺服器） 使用網際網路通訊協定，例如 gopher、 FTP 或 HTTP 存取資訊的程式。 網際網路用戶端應用程式可能會存取要擷取資料，例如對應天氣、 股票價格或頭條，例如的伺服器。 網際網路用戶端可以透過外部網路 （網際網路） 或 （有時候稱為內部網路） 的內部網路存取伺服器。  
  
 MFC 會包含 Win32 網際網路擴充功能或 WinInet，用於建立網際網路用戶端應用程式。 MFC 封裝一組標準的方便使用的類別中的這些網際網路擴充功能。 您可以藉由呼叫 Win32 函式直接或使用 MFC WinInet 類別來撰寫 WinInet 用戶端應用程式。  
  
 Microsoft Win32 網際網路功能 (WinInet) 協助您進行網際網路的任何應用程式中不可或缺的一部分。 WININET 中包含的新函數。DLL，簡化存取使用超文字傳輸通訊協定 （） 的 HTTP、 FTP （檔案傳輸通訊協定） 和 gopher 的網際網路。
