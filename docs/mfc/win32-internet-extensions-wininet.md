# <a name="win32-internet-extensions-wininet"></a>Win32 網際網路擴充功能 (WinInet)

網際網路用戶端應用程式是從網路資料來源 （伺服器） 使用網際網路通訊協定，例如 gopher、 FTP 或 HTTP 存取資訊的程式。 網際網路用戶端應用程式可能需要存取的伺服器擷取資料，例如氣象地圖、 股票價格、 或頭條，例如。 網際網路用戶端可以透過外部網路 （網際網路） 或 （有時候稱為內部網路） 的內部網路存取伺服器。

MFC 包含 Win32 網際網路擴充功能或 WinInet，建立網際網路用戶端應用程式。 MFC 會封裝一組標準且容易使用的類別中的這些網際網路擴充功能。 藉由呼叫 Win32 函式直接或使用 MFC WinInet 類別，您可以撰寫 WinInet 用戶端應用程式。

Microsoft Win32 Internet 函式 (WinInet) 可協助您進行網際網路的任何應用程式中不可或缺的一部分。 WININET 中包含的新函數。DLL，簡化存取網際網路，使用 HTTP （超文字傳輸通訊協定）、 檔案傳輸通訊協定 （FTP) 和 gopher。
