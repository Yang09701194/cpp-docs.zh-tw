# 視窗、對話方塊和控制項類別
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

將 `CWnd` ，其衍生類別封裝 `HWND`，視窗的控制代碼。視窗。  `CWnd` 可以單獨或做為基底來衍生新的類別。  類別庫提供的衍生類別代表不同類型的視窗。  
  
 [CWnd](../mfc/reference/cwnd-class.md)  
 所有視窗的基底類別。  您可以使用衍生自 `CWnd` 的其中一個類別或直接從它衍生自己的類別。
