# Windows Sockets
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

本文章系列涵蓋 Windows Sockets 的 MFC 實作。  MFC 提供兩個類別加入至支援 Web 應用程式將 Windows Sockets API。  將 [CAsyncSocket](../mfc/reference/casyncsocket-class.md) 封裝分類 Windows Sockets API 一對一，提供進階的 Web 程式設計人員這種大部分的功能和彈性。  [CSocket](../mfc/reference/csocket-class.md) 類別為序列化資料提供簡化的介面加入至 [CArchive](../mfc/reference/carchive-class.md) 物件。
