# <a name="working-with-a-tab-control"></a>使用索引標籤控制項

最簡單的方式，使用索引標籤控制項 ([CTabCtrl](../mfc/reference/ctabctrl-class.md))，請將它新增至對話方塊範本資源中，使用對話方塊編輯器。 您也可以單獨使用索引標籤控制項。 MFC 呼叫`InitCommonControls`您。 主要工作如下：

- [建立索引標籤控制項](../mfc/creating-the-tab-control.md)

- [將索引標籤加入至索引標籤控制項](../mfc/adding-tabs-to-a-tab-control.md)

- [處理索引標籤控制項通知訊息](../mfc/processing-tab-control-notification-messages.md)

如果索引標籤控制項物件嵌入父檢視或對話方塊類別中，則當父代終結時控制項也會終結。
