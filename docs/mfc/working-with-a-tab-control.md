# 使用索引標籤控制項
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

最簡單的方式就是使用索引標籤控制項 \([CTabCtrl](../mfc/reference/ctabctrl-class.md)\) 是透過加入至對話方塊編輯器的對話方塊範本資源。  您可以單獨也使用索引標籤控制項。  您的 MFC 呼叫 **InitCommonControls** 。  主要工作如下:  
  
-   [建立索引標籤控制項](../mfc/creating-the-tab-control.md)  
  
-   [將索引標籤加入至索引標籤控制項](../mfc/adding-tabs-to-a-tab-control.md)  
  
-   [處理索引標籤控制項通知訊息](../mfc/processing-tab-control-notification-messages.md)  
  
 如果選項控制物件在父檢視或對話方塊類別內嵌，終結控制終結時，父代。
