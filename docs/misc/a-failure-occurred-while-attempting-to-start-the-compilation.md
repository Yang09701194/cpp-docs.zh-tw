# A failure occurred while attempting to start the compilation
向 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 或 [!INCLUDE[csprcs](../ide/includes/csprcs_md.md)] 編譯器發出**開始建置**的要求時，發生未知的錯誤。  
  
 **若要更正這個錯誤**  
  
-   重新啟動 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]。  
  
     這種錯誤將導致建置處理序失敗。
