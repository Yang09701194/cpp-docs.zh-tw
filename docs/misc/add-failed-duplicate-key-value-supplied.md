# 加入失敗。 提供了重複的機碼值
`Add` 作業失敗，因為索引鍵的值與另一個索引鍵的值相同。 索引鍵的值必須是唯一的。  
  
### 更正這個錯誤  
  
-   請確保索引鍵的值是唯一的。
