# Add-in &lt;name&gt; caused an access violation. Remove it from the Add-in Manager?
此增益集已毀損，而且可能會在整合性開發環境中造成其他問題。  
  
### 若要更正這個錯誤  
  
1.  請選擇 \[是\] 來移除可用增益集清單中的增益集。  
  
     \-或\-  
  
     請選擇 \[否\]。  
  
    > [!NOTE]
    >  請連絡增益集的作者以報告問題。
