# 如果任何一個參數具有明確類型，則所有參數都必須具有明確類型
您在函式中混用了不同類型的類型宣告。  
  
 **錯誤 ID︰**BC30529  
  
### 更正這個錯誤  
  
-   確定所有參數都具有明確類型。
