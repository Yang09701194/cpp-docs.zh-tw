# 嘗試建立應用程式事件檔時發生錯誤。&#39;&lt;error&gt;&#39;
當您建立 WPF 瀏覽器應用程式，而 Visual Studio 無法產生應用程式事件檔 \(Application.xaml.vb 或 App.xaml.cs\)，就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
-   儲存並關閉方案。
