# An invalid cookie was passed to UnregisterExtenderProvider().
您指定了已經無效的 Cookie。  
  
### 若要更正這個錯誤  
  
1.  請傳回與您從呼叫 RegisterExtenderProvider\(\) 取得的相同 Cookie 值。
