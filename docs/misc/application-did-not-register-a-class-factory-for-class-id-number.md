# Application did not register a class factory for class ID &lt;number&gt;.
為應用程式實作 IClassFactory 介面的 COM 物件並未登錄。  這個是作業系統錯誤。  
  
### 若要更正這個錯誤  
  
1.  請重新安裝 Visual Studio 或受影響的應用程式，然後再試一次。
