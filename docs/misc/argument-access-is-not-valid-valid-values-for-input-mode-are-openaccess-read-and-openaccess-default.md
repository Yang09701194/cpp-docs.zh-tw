# 引數 &#39;Access&#39; 無效。 Input 模式的有效值是 &#39;OpenAccess.Read&#39; 和 &#39;OpenAccess.Default&#39;
存取值在 `Input` 模式中無效。  
  
### 更正這個錯誤  
  
-   將此值變更為 `OpenAccess.Read` 或 `OpenAccess.Default`。
