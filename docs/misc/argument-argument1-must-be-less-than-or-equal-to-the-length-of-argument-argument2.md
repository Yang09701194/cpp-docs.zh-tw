# 引數 &#39;&lt;引數1&gt;&#39; 必須小於或等於引數 &#39;&lt;引數2&gt;&#39; 的長度
引數的長度超過允許的限制 \(透過第二個引數的長度所決定\)。  
  
### 更正這個錯誤  
  
1.  請確定引數值的長度落在有效範圍內。  
  
2.  如果透過計算來提供引數，請檢查計算所處理的值。
