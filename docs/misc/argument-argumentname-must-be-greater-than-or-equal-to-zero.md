# 引數 &lt;引數名稱&gt; 必須大於或等於零
引數值小於零。 引數值必須大於或等於零。  
  
### 更正這個錯誤  
  
1.  請確定引數值大於或等於零。  
  
2.  如果透過計算來提供引數，請檢查計算所處理的值。
