# 與參數 &#39;&lt;參數名稱&gt;&#39; 對應的引數從 &#39;&lt;類型1&gt;&#39; 縮小到 &#39;&lt;類型2&gt;&#39;
您已呼叫多載的方法，但編譯器找不到可以呼叫且不用縮小轉換的方法。 縮小轉換會將值變更為資料類型，而可能無法精確保留一些可能的值。  
  
 **錯誤 ID：**BC30520  
  
### 更正這個錯誤  
  
-   請指定 `Option Strict Off`。
