# 引數 &#39;Path&#39; 為 Nothing 或空白
路徑名稱無效，因為它已初始化為 `Nothing` 或為空白。 路徑必須具有特定資料類型和有效的名稱。  
  
### 更正這個錯誤  
  
-   加入有效的路徑名稱。
