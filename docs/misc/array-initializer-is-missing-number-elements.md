# 陣列初始設定式遺漏 &lt;數字&gt; 個元素
您尚未在初始設定式中提供陣列的足夠項目。  
  
 **錯誤 ID︰**BC30567  
  
### 更正這個錯誤  
  
-   請提供遺漏的引數。  
  
-   使用 `ReDim` 變更陣列的大小。
