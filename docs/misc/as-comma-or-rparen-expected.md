# 必須是 &#39;As&#39;、逗號或 &#39;)&#39;
提供的類型參數缺少 `As`、逗號 \(`,`\) 或右括弧 \(`)`\)。  
  
 **錯誤 ID︰**BC32100  
  
### 更正這個錯誤  
  
-   請提供遺漏 `As`、逗號或右括弧。
