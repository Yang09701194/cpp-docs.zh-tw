# 必須有 &#39;As&#39;
出現的宣告未提供一個或多個所宣告項目的 `As` 子句。  
  
 **錯誤 ID︰**BC30197  
  
### 更正這個錯誤  
  
-   請視需要完成 `As` 子句，或開啟 `Option Strict Off`。
