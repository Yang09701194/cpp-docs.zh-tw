# 屬性 &#39;&lt;屬性名稱&gt;&#39; 無法被套用至帶有選擇性參數的方法
屬性只可以與使用必要引數或沒有引數的方法一起使用。  
  
 **錯誤 ID：**BC30645  
  
### 更正這個錯誤  
  
1.  請定義不含選擇性參數的方法。  
  
2.  請使用可以與具有選擇性參數的方法一起使用的屬性。  
  
3.  請定義可在此內容中使用的自訂屬性。
