# 屬性 &#39;Conditional&#39; 只在 &#39;Sub&#39; 宣告中有效
<xref:System.Diagnostics.ConditionalAttribute> 屬性已套用至 `Sub` 程序以外的方法宣告。  
  
 **錯誤 ID︰**BC41007  
  
### 更正這個錯誤  
  
1.  請從無效的宣告中移除 <xref:System.Diagnostics.ConditionalAttribute>。
