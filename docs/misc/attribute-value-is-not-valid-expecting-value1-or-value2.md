# 屬性值無效，必須是 &#39;value1&#39; 或 &#39;value2&#39;
XML 文件常值中指定了無效的屬性值。 XML 文件常值的有效屬性值是以 XML 1.0 規格為基礎。  
  
 **錯誤識別碼：**BC31182  
  
### 更正這個錯誤  
  
-   根據 XML 1.0 規格，將屬性值變更為有效的值。
