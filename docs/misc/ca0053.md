# CA0053
無法載入規則組件。  
  
 這表示反映 \(Reflection\) 無法載入規則組件。  如果使用者具有未正確建置的自訂規則，或分析工具因為其他原因 \(如檔案系統存取遭到拒絕\) 而無法載入組件，就可能會發生這種情形。
