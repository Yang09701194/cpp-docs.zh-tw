# CA0055
無法載入檔案: \<path\>。  
  
 程式碼分析工具無法載入指定要用於分析的目標檔案。  如果找不到該檔案或存取該檔案時遭拒，則會發生此狀況。  這通常會發生在使用者選擇要在 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 覆寫輸入組件 \(Assembly\) 屬性 \(Property\)，或是手動叫用 fxcopcmd.exe 的時候。
