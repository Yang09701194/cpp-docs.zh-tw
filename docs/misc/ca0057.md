# CA0057
違規無法對應至目前的目標集和規則集。  
  
 如果以專案檔或匯入的 XML 報告 \(含有不能對應到目前之目標集與規則集的違規情形\) 手動叫用 \(Invoke\) FxCopCmd.exe，則會發生這個錯誤。  
  
 只有在使用命令列版本的工具進行分析才會發生。
