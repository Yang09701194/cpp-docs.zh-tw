# CA0062
找不到規則集 '*RuleSetName*' 中參考到的規則 '*RuleId*'。  
  
 此錯誤表示無法找到指定的規則。  
  
 此錯誤通常會因為手動編輯規則集檔案而發生。  若要避免這個錯誤，請使用 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 規則集編輯器來指定規則。  如需詳細資訊，請參閱[建立自訂規則集](../Topic/Creating%20Custom%20Code%20Analysis%20Rule%20Sets.md)。  
  
 若要解決這個問題，請務必在您的規則集檔案中指定的任何檢查 ID 有效。如果您要使用非標準的規則檔，請建立，並適當的規則提示路徑中的規則集檔案中指定。
