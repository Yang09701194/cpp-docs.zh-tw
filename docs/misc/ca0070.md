# CA0070
如果程式碼分析對組件進行分析時遇到權限屬性，而且至少出現下列其中一個情況時，就會發生這個錯誤：  
  
-   程式碼分析找不到包含該屬性的組件。  
  
-   程式碼分析沒有載入組件的權限。  
  
-   程式碼分析載入的組件不包含該屬性。  
  
 在大部分情況中，只要確定擁有最新版的程式碼分析工具，就可以解決問題。
