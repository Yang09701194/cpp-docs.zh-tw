# CA0501
無法讀取輸出報告。  
  
 當輸出報告因下列其中一個原因而無法寫入時，就會發生這個問題：  
  
-   磁碟已滿  
  
-   沒有寫入目錄的使用權限  
  
-   目錄不存在
