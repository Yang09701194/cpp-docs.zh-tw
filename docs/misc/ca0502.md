# CA0502
不支援的語言。  
  
 當使用者嘗試分析 ASP.NET 專案 \(包含一個或多個以 C\# 或 Visual Basic 以外語言所撰寫的頁面或類別\) 時，就會發生這個錯誤。  若要更正這個錯誤，請從專案中移除檔案或停用程式碼分析。
