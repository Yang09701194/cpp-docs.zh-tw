# CA0505
系統會忽略已被取代的屬性，因為已經定義了取代屬性。  
  
 當指定的被取代和取代程式碼分析屬性都在專案或組建組態檔中包含值時，就會產生警告 CA505。  若要解決這個問題，請移除被取代的屬性。
