# Cannot find component &#39;file&#39; of multifile assembly &#39;assembly&#39;
找不到多檔案組件中的檔案。  一個多檔案組件參考一或一個以上的檔案，例如模組或資源檔。  
  
 複製的組件可以是直接參考 \(可在 \[方案總管\] 的 \[**參考**\] 節點中看到\)，或者也可以是與某一直接參考相依的組件。  
  
 **若要更正這個錯誤**  
  
-   重新安裝元件。  
  
     這個專案可能可以建置，但也可能無法執行。
