# Cannot find main menu commandbar in resources.
當整合式開發環境在啟動時找不到 MenuBar 命令列時會發生此錯誤。  此 .dll 已被刪除、移動或是損毀。  
  
### 若要更正這個錯誤  
  
-   請使用 \[**新增或移除程式**\] 修復或重新安裝產品。
