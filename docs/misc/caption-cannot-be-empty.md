# Caption cannot be empty.
當沒有為 \[工具箱\] 索引標籤輸入名稱時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請輸入 \[工具箱\] 索引標籤的名稱。
