# 即時運算視窗中的 &#39;Case&#39; 陳述式無效
`Case` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID︰**BC30719  
  
### 更正這個錯誤  
  
-   請從 \[即時運算\] 視窗中移除 `Case` 陳述式。
