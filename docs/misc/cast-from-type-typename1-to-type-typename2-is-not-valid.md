# 從類型 &#39;&lt;類型名稱1&gt;&#39; 到類型 &#39;&lt;類型名稱2&gt;&#39; 的轉換無效
嘗試將變數從一種類型轉換成另一種類型不正確。  
  
### 更正這個錯誤  
  
1.  請檢查轉換中的類型，並確定您嘗試轉換成有意義的類型。
