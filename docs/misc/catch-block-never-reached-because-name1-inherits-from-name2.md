# &#39;Catch&#39; 區塊永遠不會執行到，因為 &#39;&lt;名稱1&gt;&#39; 繼承自 &#39;&lt;名稱2&gt;&#39;。
永遠不會執行到程式碼內的 `Catch` 區塊。  
  
 根據預設，這個訊息是一個警告。 如需隱藏警告或將警告視為錯誤的詳細資訊，請參閱[在 Visual Basic 中設定警告](../Topic/Configuring%20Warnings%20in%20Visual%20Basic.md)。  
  
 **錯誤 ID︰**BC42029  
  
### 更正這個錯誤  
  
-   請移除多餘的 `Catch` 區塊。
