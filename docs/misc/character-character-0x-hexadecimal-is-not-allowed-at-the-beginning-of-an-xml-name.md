# XML 名稱的開頭不可為字元 &#39;character&#39; (0x&lt;hexadecimal&gt;)
XML 常值或 XML 軸屬性名稱開頭包含不支援的字元。  
  
 **錯誤 ID︰**BC31169  
  
### 更正這個錯誤  
  
-   請移除 XML 常值或 XML 軸屬性名稱開頭中不支援的字元。
