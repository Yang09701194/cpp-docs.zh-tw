# XML 名稱中不允許 &#39;character&#39; 字元 (0x&lt;十六進位&gt;)
XML 常值或 XML 軸屬性的名稱包含不支援的字元 。  
  
 **錯誤 ID：**BC31170  
  
### 更正這個錯誤  
  
-   移除 XML 常值或 XML 軸屬性的名稱中不支援的字元。
