# Character &lt;character&gt; is not valid for alias names.
當輸入的不正確字元是別名名稱的一部分時，一般會發生此錯誤。  別名不能含有空白、\=、\[、\]、\/、\- 或 "。  
  
### 若要更正這個錯誤  
  
1.  請重新輸入別名名稱，並省略不正確的字元。
