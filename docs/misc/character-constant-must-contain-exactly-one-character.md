# 字元常數必須剛好包含一個字元
字元常數無效，因為它含有一個以上的字元或為 null 字元 \(所有位元都是零\)。  
  
 **錯誤識別碼：**BC30004  
  
### 更正這個錯誤  
  
-   請將字元常數變更為單一字元，或將 null 字元變更為有效的字元。
