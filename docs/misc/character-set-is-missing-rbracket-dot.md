# Character set is missing &#39;]&#39;.
在尋找或取代字串中遺漏規則運算式所指定的右括弧 \(`]`\) 時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  要尋找常值字元 `]`，請使用 `\]`。  
  
2.  要符合字元集，請輸入遺漏的括弧 `]`。
