# 無法對類別 &#39;&lt;類別名稱&gt;&#39; 編製索引，因為其無預設屬性
您無法提供類別的預設屬性。  
  
 **錯誤 ID︰**BC30367  
  
### 更正這個錯誤  
  
-   指定屬性作為預設屬性，方法是使用 `Default` 關鍵字來開始它的宣告。
