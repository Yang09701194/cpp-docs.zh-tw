# 清除覆寫的屬性對話方塊
> [!NOTE]
>  [!INCLUDE[vs_dev10_long](../build/includes/vs_dev10_long_md.md)] 不支援此對話方塊。  
  
 當您在專案組態中加入使用者定義屬性工作表 \(.props file\) 時，根據預設，專案組態會繼承屬性工作表中定義的所有屬性。  使用本對話方塊來清除專案組態中意外繼承的屬性。
