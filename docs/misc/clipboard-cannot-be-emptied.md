# Clipboard cannot be emptied.
當已複製項目，但卻因為剪貼簿已包含無法移除的資訊而無法將它放置在剪貼簿上時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請關閉其他正在使用剪貼簿的程式，然後再試一次。
