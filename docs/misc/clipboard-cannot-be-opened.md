# Clipboard cannot be opened.
當您試圖剪下、複製或貼上項目，但卻無法存取剪貼簿時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請關閉其他正在使用剪貼簿的程式，然後再試一次。
