# Clipboard format is not valid.
剪貼簿只接受某些指定格式的資料。  此錯誤通常發生在嘗試從 \[剪貼簿\] 中的元件放置資料，但是該資料格式卻與 \[剪貼簿\] 不相容時。  
  
### 若要更正這個錯誤  
  
1.  請參閱元件的文件，以判斷是否能用剪貼簿從元件中轉移資料。  
  
     \-或\-  
  
2.  選取另一個項目，然後再試一次。
