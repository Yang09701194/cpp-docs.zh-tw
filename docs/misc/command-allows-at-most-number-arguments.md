# Command allows at most &lt;number&gt; arguments.
當輸入太多引數給命令，或因為使用不正確的引數組合時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請檢查文件中命令的正確語法，然後再試一次。
