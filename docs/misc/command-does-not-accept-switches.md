# Command does not accept switches.
將參數提供給只能辨識引數的命令時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請檢視命令的語法需求。  如需詳細資訊，請參閱[使用引數的 Visual Studio 命令](../Topic/Visual%20Studio%20Commands.md)。
