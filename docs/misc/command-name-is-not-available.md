# Command &lt;name&gt; is not available.
在 \[**命令**\] 視窗或 \[**尋找\/命令**\] 方塊中所輸入的命令根據 Visual Studio 目前的狀態若為無效，通常就會發生這個錯誤。  例如，如果 Visual Studio 中目前沒有開啟任何方案，則 `File.CloseSolution`  命令無效。  
  
### 若要更正這個錯誤  
  
1.  請選擇不同命令。  
  
     \-或\-  
  
     建立不同的選項讓命令作業。
