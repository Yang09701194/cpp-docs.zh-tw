# Command &lt;name&gt; is not valid.
在 \[**尋找\/命令**\] 方塊或 \[**命令**\] 視窗中輸入 Visual Studio 無法辨認的命令或命令別名時，通常就會發生這個錯誤。  如果已輸入別名，在錯誤訊息中會顯示完整的命令名稱，而非別名。  
  
### 若要更正這個錯誤  
  
1.  請重新輸入命令名稱或選擇有效的命令名稱。
