# Command was not specified.
在 \[**命令**\] 視窗或 \[**尋找\/命令**\] 方塊中輸入空字串時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請輸入有效的命令。  如需詳細資訊，請參閱[Visual Studio 命令](../Topic/Visual%20Studio%20Commands.md)。
