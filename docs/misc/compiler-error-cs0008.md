# 編譯器錯誤 CS0008
從檔案 'file' 讀取中繼資料時發生未預期的錯誤 — 'description'  
  
 已順利開啟 DLL 來擷取中繼資料但損毀，因此無法從中讀取資料。 如需詳細資訊，請參閱[\/reference \(Import Metadata\)](../Topic/-reference%20\(C%23%20Compiler%20Options\).md)。
