# 編譯器錯誤 CS0009
無法開啟中繼資料檔 'file' — 'description'  
  
 使用 [\/reference](../Topic/-reference%20\(C%23%20Compiler%20Options\).md) 編譯器選項指定的檔案未包含有效的中繼資料。
