# 編譯器錯誤 CS0013
將中繼資料寫入檔案 'ModelStore.dll' 時發生未預期的錯誤 \-\- 沒有剩餘的邏輯空間可以用來建立更多使用者字串。  
  
 .NET Framework Common Language Runtime 未發出中繼資料。 請確認路徑正確，而且磁碟未滿。 如果問題持續發生，您可能必須修復或重新安裝 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 及\/或 .NET Framework。
