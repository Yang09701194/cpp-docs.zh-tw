# 編譯器錯誤 CS0076
列舉程式名稱 'value\_\_' 為保留字而且無法使用  
  
 [列舉](../Topic/enum%20\(C%23%20Reference\).md)中的項目不能有稱為 **value\_\_** 的識別項。
