# 編譯器錯誤 CS0167
委派 'delegate' 遺漏 Invoke 方法  
  
 您已匯入和使用以其他編譯器所建立的受管理程式 \(使用 .NET Framework Common Language Runtime\)。 該編譯器允許格式不正確的[委派](../Topic/delegate%20\(C%23%20Reference\).md)。 因此，無法使用 `Invoke` 方法。 如需詳細資訊，請參閱[委派](../Topic/Delegates%20\(C%23%20Programming%20Guide\).md)。
