# 編譯器錯誤 CS0501
'member function' 不是標記成 abstract、extern 或 partial，因此必須宣告主體  
  
 非抽象方法必須具有實作。  
  
 下列範例會產生 CS0501：  
  
```  
// CS0501.cs // compile with: /target:library public class clx { public void f();   // CS0501 declared but not defined public void g() {}   // OK }  
```
