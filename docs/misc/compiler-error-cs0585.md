# 編譯器錯誤 CS0585
編譯器內部錯誤：階段 'stage'  
  
 請嘗試判斷編譯器失敗的原因是否為無法剖析未預期的語法。 如果不是這種狀況，請嘗試[告訴我們](../Topic/Talk%20to%20Us.md)。
