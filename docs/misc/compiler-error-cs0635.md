# 編譯器錯誤 CS0635
'attribute': System.Interop.UnmanagedType.CustomMarshaller 需要具名引數 ComType 和 Marshal  
  
 當封送處理格式為 **System.Interop.UnmanagedType.CustomMarshaller** 時，必須指定 **ComType** 和 **Marshal** 引數。
