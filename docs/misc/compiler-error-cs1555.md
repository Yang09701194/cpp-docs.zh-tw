# 編譯器錯誤 CS1555
找不到為 Main 方法所指定的 'class'  
  
 已指定類別給 [\/main](../Topic/-main%20\(C%23%20Compiler%20Options\).md) 編譯器選項，但在原始碼中找不到類別名稱。
