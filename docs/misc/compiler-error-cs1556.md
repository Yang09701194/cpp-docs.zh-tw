# 編譯器錯誤 CS1556
為 Main 方法所指定的 'construct' 必須為有效的類別或結構  
  
 已將不是類別名稱的識別項傳遞給 [\/main](../Topic/-main%20\(C%23%20Compiler%20Options\).md) 編譯器選項。
