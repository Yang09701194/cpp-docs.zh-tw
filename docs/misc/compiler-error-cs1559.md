# 編譯器錯誤 CS1559
'object' 是一個匯入物件，無法使用其 Main 方法  
  
 向 [\/main](../Topic/-main%20\(C%23%20Compiler%20Options\).md) 編譯器選項指定的類別無效；類別不能當成 [Main](../Topic/Main\(\)%20and%20Command-Line%20Arguments%20\(C%23%20Programming%20Guide\).md) 方法的位置使用。
