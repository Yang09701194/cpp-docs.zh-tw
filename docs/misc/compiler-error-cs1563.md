# 編譯器錯誤 CS1563
輸出 'output file' 沒有任何原始程式檔  
  
 已指定 [\/out](../Topic/-out%20\(C%23%20Compiler%20Options\).md) 編譯器選項，但之後沒有接著任何原始程式碼檔。 您可以不使用 **\/out**，或接著 **\/out** 指定原始程式碼檔。
