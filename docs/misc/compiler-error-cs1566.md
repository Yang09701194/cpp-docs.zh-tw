# 編譯器錯誤 CS1566
讀取資源檔 'file' 時發生錯誤 — 'reason'  
  
 編譯器無法將檔名傳遞給 [\/resource](../Topic/-resource%20\(C%23%20Compiler%20Options\).md) 編譯器選項。
