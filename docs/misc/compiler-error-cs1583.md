# 編譯器錯誤 CS1583
'file' 不是有效的 Win32 資源檔  
  
 當您使用 [\/win32res](../Topic/-win32res%20\(C%23%20Compiler%20Options\).md) 編譯器選項指定檔案名稱且它不是有效或格式正確的 Win32 資源檔時，就會發生這個錯誤。 在 Visual Studio 中，檔案名稱是在專案設計工具的 \[應用程式\] 窗格中指定。
