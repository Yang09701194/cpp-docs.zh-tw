# 編譯器錯誤 CS1713
為類型 Typename1 建置中繼資料名稱時發生未預期的錯誤 \-\- 'Reason'  
  
 此錯誤通常由編譯器內部錯誤所致。 對您的程式碼進行一些微幅變更，例如縮短名稱的長度，然後重新編譯，即可能解決此錯誤。
