# 編譯器錯誤 CS1727
未經授權無法自動傳送錯誤報告。 請造訪 '' 以取得傳送錯誤報告的授權。  
  
 錯誤文字中所列的網站說明如何啟用 [!INCLUDE[vsprvslong](../error-messages/compiler-errors-1/includes/vsprvslong_md.md)] 命令列工具的自動錯誤報告。  
  
## 範例  
 下列範例會產生 CS1727：  
  
```  
// CS1727.cs // compile with: /errorreport:send // CS1727 expected class Test { static void Main(){} }  
```
