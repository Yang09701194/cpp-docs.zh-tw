# 編譯器錯誤 CS2022
選項 '\/out' 和 '\/target' 必須出現於原始程式檔名稱之前  
  
 [\/out \(設定輸出檔名稱\)](../Topic/-out%20\(C%23%20Compiler%20Options\).md) 和 [\/target \(指定輸出檔格式\)](../Topic/-target%20\(C%23%20Compiler%20Options\).md) 編譯器選項 \(指定於命令列上時\) 必須在原始程式碼檔的前面。
