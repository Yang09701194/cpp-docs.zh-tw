# 編譯器錯誤 CS2035
命令列語法錯誤: 遺漏 'compiler\_option' 選項的 ':\<數字\>'  
  
 有一些編譯器選項需要值。  
  
## 範例  
 下列範例會產生 CS2035。  
  
```  
// CS2035.cs // compile with: /baseaddress // CS2035 expected  
```
