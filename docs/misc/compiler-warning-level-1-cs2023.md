# 編譯器警告 (層級 1) CS2023
因為在回應檔中已指定 \/noconfig 選項，所以將會忽略該選項。  
  
 在回應檔中指定了 [\/noconfig](../Topic/-noconfig%20\(C%23%20Compiler%20Options\).md) 編譯器選項，但不允許這樣做。  
  
 [\/nowarn](../Topic/-nowarn%20\(C%23%20Compiler%20Options\).md) 選項無法隱藏這個警告。
