# 編譯器警告 (層級 1) CS3004
混合和分解的 Unicode 字元不符合 CLS 標準  
  
 為了符合 Common Language Specification \(CLS\) 標準，只能在 [public](../Topic/public%20\(C%23%20Reference\).md)、[protected](../Topic/protected%20\(C%23%20Reference\).md) 或 `protected``internal` 識別項中使用組成的 UNICODE 字元。 如需 CLS 符合性的詳細資訊，請參閱[撰寫符合 CLS 標準的程式碼](http://msdn.microsoft.com/zh-tw/4c705105-69a2-4e5e-b24e-0633bc32c7f3)和[語言獨立性以及與語言無關的元件](../Topic/Language%20Independence%20and%20Language-Independent%20Components.md)。
