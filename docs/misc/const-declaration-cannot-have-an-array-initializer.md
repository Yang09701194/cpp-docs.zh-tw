# Const 宣告不可以有陣列初始設定式
您嘗試在 `Const` 宣告中使用陣列初始設定式。  
  
 **錯誤 ID︰**BC30445  
  
### 更正這個錯誤  
  
-   請移除兩側的大括弧，將初始設定式變更為有效的運算式。
