# 常數不可以當做指派的目標
常數發生在指派其值的內容中。 在執行期間，只會指派可寫入變數、屬性和陣列項目的值。  
  
 **錯誤 ID︰**BC30074  
  
### 更正這個錯誤  
  
-   請將常數取代為單一可寫入的變數、屬性或陣列項目。
