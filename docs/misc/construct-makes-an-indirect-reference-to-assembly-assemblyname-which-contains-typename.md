# 建構會間接參考含有 &#39;&lt;類型名稱&gt;&#39; 的組件 &#39;&lt;組件名稱&gt;&#39;
建構會間接參考含有 \<類型名稱\> 的組件 '\<組件名稱\>'。 請將 \<檔案名稱\> 的檔案參考加入您的專案中。  
  
 運算式使用類別、結構、介面、列舉或委派這類類型，但您的組件沒有定義該類型之組件的專案參考。  
  
 **錯誤 ID︰**BC31528  
  
### 更正這個錯誤  
  
-   在專案屬性中，加入包含組件之檔案的參考，而這個組件定義所使用的類型。
