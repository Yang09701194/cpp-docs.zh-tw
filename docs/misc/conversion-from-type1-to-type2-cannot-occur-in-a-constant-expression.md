# 常數運算式中不可進行從 &#39;&lt;類型1&gt;&#39; 到 &#39;&lt;類型2&gt;&#39; 的轉換
`Const` 陳述式中的初始化運算式評估為無法轉換成常數之宣告類型的資料類型。  
  
 **錯誤 ID︰**BC30060  
  
### 更正這個錯誤  
  
1.  請使用可轉換成針對常數所宣告之類型的資料類型的運算式，來初始化常數。
