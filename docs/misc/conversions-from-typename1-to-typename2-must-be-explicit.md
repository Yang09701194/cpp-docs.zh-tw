# 必須明確從 &#39;&lt;類型名稱1&gt;&#39; 轉換為 &#39;&lt;類型名稱2&gt;&#39;
無法以隱含方式完成這項轉換。  
  
 **錯誤 ID：**BC30664  
  
### 更正這個錯誤  
  
-   使用 `CType` 或其中一個特定轉換函式，例如 `CInt` 或 `CDec`。
