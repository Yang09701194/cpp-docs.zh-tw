# 無法產生二進位資源。 無法開啟輸出檔 &#39;file&#39;。 &lt;原因&gt;
專案系統無法刪除過期的 .resources 檔案複本。 準備資源時，專案系統會將副檔名為 .resx 的 XML 檔案轉換成 .NET 資源管理員的二進位 .resources 檔案。  
  
 錯誤訊息文字結尾所提供的作業系統特定原因將在這裡提供最佳資訊。  
  
 **更正這個錯誤**  
  
-   重新啟動 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]。  
  
     此錯誤會導致建置處理序失敗。
