# 無法產生二進位授權檔。 無法開啟輸出檔 &#39;file&#39;。 &lt;原因&gt;
專案系統無法刪除過期的授權檔複本。 在準備授權時，專案系統會將 .licx 文字檔轉換為 .NET 授權系統可解讀的二進位授權檔。  
  
 **更正這個錯誤**  
  
-   重新啟動 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]。  
  
     如果發生此錯誤，則建置處理序會失敗。
