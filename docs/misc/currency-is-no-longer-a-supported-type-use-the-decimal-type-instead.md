# 不再支援 &#39;Currency&#39; 類型。請使用 &#39;Decimal&#39; 類型代替
`Decimal` 資料類型取代了 `Currency` 資料類型。  
  
 **錯誤 ID︰**BC30803  
  
### 更正這個錯誤  
  
-   使用 `Decimal` 資料類型取代 `Currency`。
