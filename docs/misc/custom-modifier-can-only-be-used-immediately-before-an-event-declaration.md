# &#39;Custom&#39; 修飾詞只能緊接地用於 &#39;Event&#39; 宣告的前面
在自訂事件宣告中，`Custom` 關鍵字必須在 `Event` 關鍵字前面。  
  
 **錯誤 ID︰**BC31140  
  
### 更正這個錯誤  
  
-   請將 `Custom` 關鍵字移到 `Event` 關鍵字前面。  
  
     \-或\-  
  
-   請移除 `Custom` 關鍵字。
