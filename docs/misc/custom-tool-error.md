# Custom Tool Error
自訂工具產生錯誤或警告。  自訂工具是一種元件，可用來在設計階段將檔案從一種類型轉換成另一種類型。  例如，自訂工具可能是資料集程式碼產生器，用來讀取 XML 結構描述 \(.xsd\) 檔並且在以程式設計方式公開 \(Expose\) 其資料表和資料行的程式碼檔中產生類別。  
  
 **若要更正這個錯誤**  
  
-   請查閱自訂工具廠商提供的文件。
