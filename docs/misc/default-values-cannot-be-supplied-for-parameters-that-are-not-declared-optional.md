# 無法提供預設值給未宣告為 &#39;Optional&#39; 的參數。
呼叫程序時必須提供必要的引數，而且它們在宣告中不採用預設值。  
  
 **錯誤識別碼：**BC32024  
  
### 更正這個錯誤  
  
-   宣告引數為 `Optional` 或移除預設值。
