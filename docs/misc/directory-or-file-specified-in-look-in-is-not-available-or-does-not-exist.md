# Directory or file specified in Look In is not available or does not exist.
當找不到 \[瀏覽\] 清單中指定的路徑或檔名時，一般會發生此錯誤。  
  
### 若要更正這個錯誤  
  
1.  請重新輸入路徑和檔名。  
  
     \-或\-  
  
2.  使用 \[瀏覽\] 清單和 \[可用的項目\] 方塊來巡覽至正確的檔案或資料夾。
