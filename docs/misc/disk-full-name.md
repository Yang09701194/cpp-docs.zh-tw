# Disk full: &lt;name&gt;
當程式關閉時，含有程式的磁碟機沒有包含足夠的空間來儲存程式檔案，例如 toolbox.tbd。  
  
### 若要更正這個錯誤  
  
1.  請從磁碟機中移除檔案，然後再試著關閉程式。
