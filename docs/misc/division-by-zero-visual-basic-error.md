# 除數為零 (Visual Basic 錯誤)
作為除數的運算式值為零。  
  
### 更正這個錯誤  
  
1.  請檢查運算式中的變數拼法。 拼錯的變數可能會隱含地建立初始化為零的數值變數。  
  
2.  請檢查運算式中先前的變數作業，特別是那些作為引數從其他程序傳遞至程序的變數。
