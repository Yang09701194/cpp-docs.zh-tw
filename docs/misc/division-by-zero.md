# Division by zero.
除數為零在數學上是不可能發生的。  當運算式的值做為除數是零時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請檢查運算式的拼字、邏輯或大小寫錯誤。
