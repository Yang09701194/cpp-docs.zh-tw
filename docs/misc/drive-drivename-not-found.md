# 找不到磁碟機 &#39;&lt;磁碟機名稱&gt;&#39;
找不到磁碟機。 您的網路連線已遺失，或是您嘗試存取的磁碟機離線或不存在。  
  
### 更正這個錯誤  
  
1.  請檢查磁碟機的電源。  
  
2.  請確定已適當地連接所有將您的電腦連線到網路的纜線。  
  
3.  若要透過網路存取磁碟機，請確定您的電腦與磁碟機之間存在邏輯連線。  
  
4.  重新連線到網路，然後再試一次。
