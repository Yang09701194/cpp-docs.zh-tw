# 重複的 XML 屬性 &#39;attributeName&#39;
為 XML 元素常值指定的屬性與為 XML 元素常值預先定義的屬性相符。 XML 元素常值中的屬性名稱必須是唯一的。  
  
 **錯誤識別碼：**BC31149  
  
### 更正這個錯誤  
  
-   將屬性重新命名為對 XML 元素常值是唯一的名稱。
