# 省略符號 (...)
省略符號運算子 \(`...`\) 在 C 和 C\+\+ 中有許多用途，視內容而定。  原本在 C 中當做可變函式參數清單的抽象宣告子，也可以透過 variadic 巨集的形式用於前置處理器巨集程式設計。  在 C\+\+ 中，`...` 會用於例外狀況處理 `catch` 區塊，而在 C\+\+11 中則用在 variadic 樣板中。
