# 已指定空白字元集。
這個錯誤通常在已選取規則運算式選項，且包含不完整語法的字串已指定為**字元集 \[\]** 或**不包含字元集 \[^\]** 的模式時發生。  例如，`[}`。  
  
### 若要更正這個錯誤  
  
1.  檢視規則運算式主題來取得該模式的正確語法資訊，然後再一次輸入該字串。
