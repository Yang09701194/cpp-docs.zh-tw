# &#39;End AddHandler&#39; 必須是該行的第一個陳述式
`End AddHandler` 陳述式接在冒號 \(:\) 陳述式分隔符號的後面。`End AddHandler` 必須是其原始程式碼行上的唯一陳述式。  
  
 **錯誤 ID︰**BC31118  
  
### 更正這個錯誤  
  
-   請將多個陳述式分成不同的行。
