# &#39;End Enum&#39; 之前必須搭配相對應的 &#39;Enum&#39;
`End Enum` 陳述式沒有相對應的 `Enum` 陳述式。`End Enum` 之前必須搭配相對應的 `Enum` 陳述式。  
  
 **錯誤 ID︰**BC30184  
  
### 更正這個錯誤  
  
1.  確定前面的 `Enum` 陳述式沒有拼錯字，否則將會無效。  
  
2.  請確定已正確地格式化 `Enum` 成員。
