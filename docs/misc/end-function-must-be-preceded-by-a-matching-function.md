# &#39;End Function&#39; 之前必須搭配相對應的 &#39;Function&#39;
`End Function` 陳述式出現在您的程式碼中，但前面沒有對應的 `Function` 程序定義。  
  
 **錯誤 ID︰**BC30430  
  
### 更正這個錯誤  
  
1.  若 `End Function` 陳述式為多餘，請予以移除。  
  
2.  提供遺漏的 `Function` 程序 \(遺漏時\)。  
  
3.  將 `End Function` 移至程式碼中的適當位置。
