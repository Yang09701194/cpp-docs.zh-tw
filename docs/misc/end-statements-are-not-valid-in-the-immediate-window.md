# 即時運算視窗中的 &#39;End&#39; 陳述式無效
`Stop` 和 `End` 陳述式在偵錯內容中暫停執行且不允許使用。  
  
 **錯誤 ID︰**BC30123  
  
### 更正這個錯誤  
  
-   請不要在 \[即時運算\] 視窗中發出 `End` 或 `Stop` 陳述式。
