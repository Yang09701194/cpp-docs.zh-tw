# 列舉 &#39;&lt;列舉名稱&gt;&#39; 至少必須包含一個成員
發生空的列舉。 在 `Enum` 陳述式與 `End Enum` 陳述式之間至少必須宣告一個成員。  
  
 **錯誤 ID︰**BC30280  
  
### 更正這個錯誤  
  
1.  請將成員宣告加入列舉。
