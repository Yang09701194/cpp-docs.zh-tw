# Error getting Codezone Community sites.
當找不到連接到線上 Web 服務的連線時會發生此錯誤。  Web 服務會將資訊填入 \[**選項**\] 對話方塊之 \[**說明**\]、\[**環境**\] 下的 \[**線上**\] 頁面。  
  
### 若要更正這個錯誤  
  
1.  請確認您有作用中的網際網路連線。  
  
     \-或\-  
  
     如果是透過 Proxy 伺服器連接到網際網路，請檢查 Proxy 伺服器的設定。  
  
2.  請嘗試再一次存取 \[線上\] 頁面。
