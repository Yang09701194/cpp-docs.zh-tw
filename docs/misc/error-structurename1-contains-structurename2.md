# &lt;錯誤&gt;: &#39;&lt;結構名稱1&gt;&#39; 包含 &#39;&lt;結構名稱2&gt;&#39;
結構將其本身指定為它自己的其中一個成員時，就會發生循環結構定義。  
  
 **錯誤 ID︰**BC30293  
  
### 更正這個錯誤  
  
-   變更結構成員的名稱。
