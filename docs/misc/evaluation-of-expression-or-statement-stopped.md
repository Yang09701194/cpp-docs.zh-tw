# 運算式或陳述式評估已經停止
已選取 \[停止評估\] 功能表項目，進而導致評估停止。  
  
 **錯誤 ID︰**BC30721  
  
### 更正這個錯誤  
  
-   允許繼續執行，而不選取 \[停止評估\]。
