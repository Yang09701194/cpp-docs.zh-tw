# Evaluation stack overflow. The pattern may be too complex.
這個錯誤通常發生在搜尋字串中使用複雜的規則運算式，而整合式開發環境 \(IDE\) 無法處理。  
  
### 若要更正這個錯誤  
  
1.  減少封閉區段 \(Closure\) 和子句的數目然後再試一次。
