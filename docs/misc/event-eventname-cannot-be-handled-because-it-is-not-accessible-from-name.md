# 無法處理事件 &#39;&lt;事件名稱&gt;&#39;，因為它無法從 &#39;&lt;名稱&gt;&#39; 存取
您嘗試處理無法存取的事件。 例如，如果共用 `Handles` 變數，也必須共用處理事件的方法。  
  
 **錯誤 ID︰**BC30585  
  
### 更正這個錯誤  
  
1.  確定事件可供存取。
