# 內嵌運算式必須以 &#39;%&gt;&#39; 結尾
XML 常值中的內嵌運算式遺漏必要的結尾 `%>` 字元。  
  
 **錯誤 ID︰**BC31159  
  
### 更正這個錯誤  
  
-   將必要的結尾 `%>` 字元加入內嵌運算式的結尾。
