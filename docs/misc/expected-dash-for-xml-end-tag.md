# XML 結束標記必須是 &#39;/&#39;
XML 常值的 XML 結束標記遺漏必要的 '\/' 字元。  
  
 **錯誤 ID︰**BC31167  
  
### 更正這個錯誤  
  
-   在 XML 常值的 XML 結束標記開頭加入 '\/' 字元。
