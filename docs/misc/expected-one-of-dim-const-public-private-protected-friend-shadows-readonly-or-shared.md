# 必須是 &#39;Dim&#39;、&#39;Const&#39;、&#39;Public&#39;、&#39;Private&#39;、&#39;Protected&#39;、&#39;Friend&#39;、&#39;Shadows&#39;、&#39;ReadOnly&#39; 或 &#39;Shared&#39; 其中之一
宣告陳述式遺漏宣告關鍵字。 其中一個可能的原因是屬性宣告呼叫方法。  
  
 **錯誤 ID︰**BC30195  
  
### 更正這個錯誤  
  
1.  確認已在屬性宣告內宣告方法。  
  
2.  使用適當的宣告關鍵字作為陳述式的開頭。
