# 運算式評估工具錯誤 CXX0042
行號錯誤  
  
 這個錯誤應該永遠不會發生。  
  
 請記錄發生錯誤的情況，並通知 Microsoft Corporation。 您可以在 [http:\/\/support.microsoft.com\/support\/visualc\/report\/](http://support.microsoft.com/support/visualc/report/) 線上報告錯誤，也可以連絡 Microsoft 產品支援服務。 如需詳細資訊，請按一下 Visual C\+\+ 中 \[說明\] 功能表上的 \[技術支援\]。  
  
 這個錯誤與 CAN0042 相同。
