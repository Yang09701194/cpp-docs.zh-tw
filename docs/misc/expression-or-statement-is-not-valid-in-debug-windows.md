# 偵錯視窗中的運算式或陳述式無效
`Try...Catch...Finally` 陳述式不能用於偵錯內容中。  
  
 **錯誤 ID︰**BC30706  
  
### 更正這個錯誤  
  
1.  請從偵錯程式碼中移除 `Try...Catch...Finally` 陳述式。
