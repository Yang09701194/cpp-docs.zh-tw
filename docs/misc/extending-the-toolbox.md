# 擴充工具箱
[!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] **工具箱** 提供物件集合，以透過 IDE 的拖放機制，來為編輯器和設計工具提供功能。  
  
 VSPackage 在使用 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]**工具箱**方面有兩種基本方式：  
  
-   VSPackage 可以將新的資料項目和控制項加入**工具箱**中。  
  
-   VSPackage 可以是現有**工具箱**功能的目標或取用者，並支援拖放作業及設定**工具箱**的外觀。
