# Failed to generate resource files
專案系統讀取 .resx 檔案做為建立 .NET 資源管理員可瞭解的二進位 .resources 檔處理序一部分時，發生一般失敗。  
  
 這個錯誤通常是因為錯誤的 .resx 檔案所產生。  
  
 這個警告不會導致建置失敗。  但是，造成出現這個錯誤的 .resx 檔案將不會被包含在專案中做為組件資訊清單資源。
