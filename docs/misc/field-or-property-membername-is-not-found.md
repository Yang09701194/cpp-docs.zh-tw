# 找不到欄位或屬性 &#39;&lt;成員名稱&gt;&#39;
屬性參考了未定義的欄位或屬性。  
  
 **錯誤 ID：**BC30661  
  
### 更正這個錯誤  
  
1.  請檢查欄位或屬性的拼字。  
  
2.  請確認您的專案可以存取正確的屬性類別，並視需要將它加入專案參考中。
