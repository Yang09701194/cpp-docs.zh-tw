# File &lt;filename&gt; cannot be created.
這個錯誤通常在嘗試建立程式碼結構 Web 報告，而目前的使用者沒有寫入權限時發生。  
  
### 若要更正這個錯誤  
  
1.  請檢查您擁有寫入權限。  
  
2.  請檢查有足夠的空間儲存網頁。
