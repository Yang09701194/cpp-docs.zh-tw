# 檔案格式無效
可能是您嘗試將檔案載入元件中，但檔案中的資料格式與元件不相容，或您嘗試將元件資料儲存至檔案，但資料格式與檔案格式不相容。  
  
### 更正這個錯誤  
  
1.  請參閱元件的文件，以決定磁碟檔案資料的正確格式。  
  
2.  請參閱元件的文件，以判斷它是否支援不同格式的轉換。
