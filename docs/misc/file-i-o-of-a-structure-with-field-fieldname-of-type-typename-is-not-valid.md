# 類型為 &#39;&lt;類型名稱&gt;&#39; 之欄位 &#39;&lt;欄位名稱&gt;&#39; 的結構的檔案 I/O 無效
您嘗試寫入檔案的結構包含不支援的類型。  
  
### 更正這個錯誤  
  
-   請檢查結構以決定不受支援類型的發生位置，然後將它變更為有效的類型。
