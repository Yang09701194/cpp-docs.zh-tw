# File is not a valid .NET Framework or COM component
您正試圖檢視在 \[物件瀏覽器\] 中不含有效物件的檔案。  \[物件瀏覽器\] 中只能檢視 [!INCLUDE[dnprdnshort](../error-messages/tool-errors/includes/dnprdnshort_md.md)] 和 COM 元件。  
  
### 若要更正這個錯誤  
  
1.  如果是選取了錯誤的檔案，請選取正確檔案後再試一次。  
  
2.  如果誤選了 \[**開啟方式**\] 對話方塊中的 \[物件瀏覽器\]，請選擇另一個程式開啟檔案。
