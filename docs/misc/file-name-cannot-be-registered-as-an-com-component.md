# File &lt;name&gt; cannot be registered as an COM component.
無法將檔案加入至專案的參考。  當檔案已損毀、不包含型別程式庫、使用不相容的檔案類型或當系統資源不足無法完成作業時，就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請確認檔案包含有效的型別程式庫，然後再試一次。  
  
     \-或\-  
  
     重新安裝已安裝檔案的產品。  
  
    > [!NOTE]
    >  已安裝檔案的產品可能不是 Visual Studio。
