# 檔名不存在、無效或是包含無法驗證的參數。 要變更您的輸入嗎?
當 **\[加入程式\]** 對話方塊中輸入的路徑或檔案名稱不正確時，通常會發生此錯誤。  
  
### 更正這個錯誤  
  
1.  選擇 **\[瀏覽\]** 以巡覽至正確的檔案。  
  
     \-或\-  
  
2.  重新輸入路徑和檔案名稱。
