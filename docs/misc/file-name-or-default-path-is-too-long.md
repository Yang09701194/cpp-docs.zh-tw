# File name or default path is too long.
這個錯誤通常在嘗試建立程式碼結構 Web 報告，但在 \[建置註解 Web 網頁\] 對話方塊中之 \[儲存 Web 網頁\] 方塊中所指定的檔案名稱或路徑太長時發生。  
  
### 若要更正這個錯誤  
  
1.  請指定小於 260 字元的檔案名稱或路徑。
