# File &lt;name&gt; size is incorrect.
這個檔案的位元組比預期還要少，可能是因為檔案已被截斷或已損毀。  
  
### 若要更正這個錯誤  
  
1.  請關閉並重新啟動 Visual Studio。  
  
2.  如果這個錯誤持續發生，請重新安裝 Visual Studio，以修復損毀的檔案。
