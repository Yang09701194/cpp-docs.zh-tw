# 無法參考 &#39;&lt;檔案名稱&gt;&#39;，因為它不是組件
已對不是 .NET 組件的檔案進行參考。  
  
 **錯誤 ID︰**BC31519  
  
### 更正這個錯誤  
  
1.  請確認組件的名稱和位置。  
  
2.  請確認組件未損毀。
