# 在命名空間中類型的完整限定名稱不能超過 &lt;數字&gt; 個字元
類型的完整限定名稱不能超過指定的字元數。  
  
 **錯誤 ID︰**BC30031  
  
### 更正這個錯誤  
  
-   請縮短類型的名稱。
