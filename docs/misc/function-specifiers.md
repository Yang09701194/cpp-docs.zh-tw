# 函式規範
您可以使用 [inline](../misc/inline-inline-forceinline.md) 和 [virtual](../cpp/virtual-specifier.md) 關鍵字做為函式宣告中的指定名稱。**virtual** 的這種用法不同於在類別定義的基底類別指定名稱中的用法。
