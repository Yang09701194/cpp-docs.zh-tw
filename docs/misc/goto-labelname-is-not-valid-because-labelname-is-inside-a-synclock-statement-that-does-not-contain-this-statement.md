# &#39;GoTo &lt;標籤名稱&gt;&#39; 無效，因為 &#39;&lt;標籤名稱&gt;&#39; 位於不包含這個陳述式的 &#39;SyncLock&#39; 陳述式內部
您無法分支到 `SyncLock` 區塊。  
  
 **錯誤 ID：**BC30755  
  
### 更正這個錯誤  
  
-   重建您的程式碼，讓標籤位於 `SyncLock` 區塊之前。
