# Grouped expression is missing &#39;)&#39;.
在尋找字串中已指定左括弧 "`(`"，但省略右括弧 "`)`" 時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  若要搜尋常值 \(Literal\) 字元 "`(`"，請使用 "`\(`"。  
  
2.  若要群組運算式，則請輸入遺漏的 "`)`"。
