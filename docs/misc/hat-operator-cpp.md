# ^ 運算子 (C++)
`^` 運算子根據其內容，在 Visual C\+\+ 中具有兩種意義。  如需詳細資訊，請參閱：  
  
-   [位元互斥 OR 運算子：^](../cpp/bitwise-exclusive-or-operator-hat.md) 用於 C\+\+ 的所有層面。  
  
-   [物件控制代碼運算子 \(^\)](../windows/handle-to-object-operator-hat-cpp-component-extensions.md) 用於 [!INCLUDE[cppwrt_short](../build/reference/includes/cppwrt_short_md.md)] 和 [!INCLUDE[cppcli](../build/reference/includes/cppcli_md.md)]。
