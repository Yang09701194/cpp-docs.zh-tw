# 如何：回應 UML 模型中的變更
可以撰寫程式碼，在 Visual Studio 中的 UML 模型發生變更時執行。  它會平均回應直接由使用者，以及其他 Visual Studio 擴充功能所做的變更。  若要查看哪些版本的 Visual Studio 支援 UML 模型，請參閱 [Architecture and Modeling Tools 的版本支援](../Topic/What's%20new%20for%20design%20in%20Visual%20Studio.md#VersionSupport)。  
  
> [!WARNING]
>  UML API 不支援這些技術。  它們可能無法在未來版本的 Visual Studio 運作。  
  
 範例程式碼位於 [UML：使用事件與規則回應 UML 模型中的變更](http://code.msdn.microsoft.com/UML-Responding-to-changes-c024cd4b)
