# Imports 別名 &#39;&lt;名稱&gt;&#39; 和根命名空間中宣告的 &#39;&lt;名稱2&gt;&#39; 衝突。
`Imports` 陳述式中所使用的別名名稱與根命名空間中所宣告的另一個名稱相衝突。  
  
 **錯誤 ID︰**BC31403  
  
### 更正這個錯誤  
  
-   為 `Imports` 陳述式所使用的別名，選擇唯一的名稱。
