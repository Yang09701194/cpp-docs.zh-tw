# 必須有整數常數
發生 `#ExternalSource` 指示詞，其中的第二個引數不是整數常值。 在這個內容中，只有整數常值有效。 具名的常數或列舉成員無效。  
  
 **錯誤 ID︰**BC30204  
  
### 更正這個錯誤  
  
1.  使用具名的常數或列舉成員，而不要使用常值。  
  
2.  請提供整數常值作為 `#ExternalSource` 指示詞的第二個引數。
