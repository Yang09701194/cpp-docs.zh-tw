# &#39;&lt;介面名稱1&gt;&#39; 無法實作 &#39;&lt;方法名稱&gt;&#39;，因為在介面 &#39;&lt;介面名稱2&gt;&#39; 上沒有對應的 &lt;方法&gt;
您嘗試在從中衍生第一個介面的介面上沒有對應的方法時，於該介面上實作方法。  
  
 **錯誤識別碼︰**BC30401  
  
### 更正這個錯誤  
  
-   確定您沒有拼錯方法的名稱，或提供了遺漏的方法。
