# Internal error processing pattern.
這個錯誤通常發生在一般運算式已包括在搜尋字串中，但該運算式的語法含有錯誤。  
  
### 若要更正這個錯誤  
  
1.  在該搜尋字串中使用正確的規則運算式語法重新輸入。  如需詳細資訊，請參閱[規則運算式](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md)。
