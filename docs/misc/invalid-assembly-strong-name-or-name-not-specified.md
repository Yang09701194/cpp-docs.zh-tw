# Invalid assembly strong name or name not specified
專案系統物件模型中的專案屬性 \(Property\) 是空白的，該物件屬性 \(Property\) 是用來讓您為建置到強式名稱組件 \(已簽名的組件\) 中的專案指定了金鑰名稱或金鑰容器。  如果這個訊息顯示，則建置不會成功。  
  
### 若要更正這個錯誤  
  
1.  確定程式碼中有指定金鑰容器名稱或金鑰檔案。
