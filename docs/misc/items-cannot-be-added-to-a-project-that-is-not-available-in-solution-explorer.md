# 無法將項目加入至 [方案總管] 中無法使用的專案。
無法在方案總管中開啟指定的專案時，通常會發生此錯誤。 Visual Studio 無法將項目加入方案總管中目前沒有的專案。  
  
### 更正這個錯誤  
  
1.  將指定的專案加入目前的方案中，或開啟包含所指定專案的方案。
