# 關鍵字無法為類型命名
需要類型名稱的位置出現關鍵字。 這個內容呼叫基本資料類型或代表類型的已宣告項目名稱。  
  
 **錯誤 ID︰**BC30180  
  
### 更正這個錯誤  
  
-   請將關鍵字取代為有效的類型名稱。
