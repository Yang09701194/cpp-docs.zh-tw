# 無法讀取第 &lt;數字&gt; 行，因為超過行數上限。
無法讀取這行，因為超過行數上限。 這通常表示檔案中有左引號，但沒有右引號。  
  
### 更正這個錯誤  
  
-   請在檔案中插入右引號。
