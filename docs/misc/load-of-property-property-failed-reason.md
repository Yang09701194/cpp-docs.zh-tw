# Load of property &#39;property&#39; failed. &lt;reason&gt;
屬性 \(Property\) 值超出範圍。  
  
 最可能導致這個錯誤的原因是以手動方式編輯專案檔。  
  
 **若要更正這個錯誤**  
  
-   重新編輯專案檔案並且手動修正錯誤。  
  
     受影響的屬性 \(Property\) 將會設定為預設值 ─ 也就是專案建立時的屬性 \(Property\) 值。
