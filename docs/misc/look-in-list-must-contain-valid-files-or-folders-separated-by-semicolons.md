# Look In list must contain valid files or folders, separated by semicolons.
這個錯誤通常在 \[查詢\] 清單的目錄或檔案中包含無效的字元時發生。  
  
### 若要更正這個錯誤  
  
1.  請使用 \[查詢\] 對話方塊以瀏覽欲搜尋的目錄或檔案名稱，而不是自行輸入路徑或檔案名稱。  您可以經由選擇在 \[查詢\] 清單旁的省略按鈕來存取 \[查詢\] 對話方塊。
