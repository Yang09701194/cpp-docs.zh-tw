# Managed VSPackage
下列主題將說明如何建立 VSPackage。  VSPackage 是一種軟體模組延伸[!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]藉由提供使用者介面 \(UI\) 項目、 服務、 專案、 編輯器和設計工具的整合式的開發環境 \(IDE\)。  如需詳細資訊，請參閱 [Vspackage](../Topic/VSPackages.md)。  
  
## 在本節中  
 [使用 Visual Studio Interop 組件](../Topic/Using%20Visual%20Studio%20Interop%20Assemblies.md)  
 描述之功能和位置的[!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] interop 組件和它們所提供的命名空間。  
  
 [在 Managed 程式碼的 HRESULT 資訊](../misc/hresult-information-in-managed-code.md)  
 將告訴您如何轉譯成擲回的例外狀況的 HRESULT 資訊和`int`在 managed 程式碼會傳回值。  
  
 [Visual Studio Interop 組件參數封送處理](../misc/visual-studio-interop-assembly-parameter-marshaling.md)  
 討論之間的互通性問題[!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] interop 組件和 COM 介面。  
  
 [VSPackage 和 Managed 封裝架構](../misc/vspackages-and-the-managed-package-framework.md)  
 說明和列出的受管理的封裝 \(MPF\) 架構類別命名空間和 DLL 檔，並示範如何使用它們來建立 VSPackage。  
  
 [在 Vspackage 中的資源](../Topic/Resources%20in%20VSPackages.md)  
 說明在受管理的 VSPackages 中的 managed 和 unmanaged 資源的使用。
