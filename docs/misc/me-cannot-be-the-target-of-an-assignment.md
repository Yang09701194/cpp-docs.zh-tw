# &#39;Me&#39; 不可以是指派的目標
`Me` 關鍵字一律參照目前執行程式碼之類別的特定執行個體。 其值不能變更。  
  
 **錯誤 ID︰**BC30062  
  
### 更正這個錯誤  
  
-   請移除將值指派給 `Me` 的陳述式。
