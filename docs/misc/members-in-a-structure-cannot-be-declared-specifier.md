# 結構中的成員不可以宣告為 &#39;&lt;規範&gt;&#39;
您已經在 `Structure` 陳述式內使用成員上無效的規範。  
  
 **錯誤 ID：**BC30435  
  
### 更正這個錯誤  
  
-   請移除規範。
