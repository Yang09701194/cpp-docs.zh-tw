# Menu captions cannot contain separators.
當在功能表標題的名稱使用虛線字元 \(\-\) 時，一般會發生此錯誤。  虛線字元為功能表分隔符號。  
  
### 若要更正這個錯誤  
  
1.  請輸入不使用虛線字元的標題名稱。
