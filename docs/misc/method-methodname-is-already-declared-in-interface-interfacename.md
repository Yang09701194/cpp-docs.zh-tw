# 已經在介面 &#39;&lt;介面名稱&gt;&#39; 中宣告方法 &#39;&lt;方法名稱&gt;&#39;
您已第二次宣告介面方法，該介面方法重複。  
  
 **錯誤 ID︰**BC30377  
  
### 更正這個錯誤  
  
-   請移除重複的宣告。
