# &#39;&lt;方法名稱&gt;&#39; 只能實作一次
您嘗試實作已經實作的方法。  
  
 **錯誤 ID︰**BC30583  
  
### 更正這個錯誤  
  
1.  請檢查方法名稱以確定拼字正確。  
  
2.  如果方法多餘，請移除額外的宣告。
