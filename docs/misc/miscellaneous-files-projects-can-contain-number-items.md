# Miscellaneous Files projects can contain &lt;number&gt; items.
其他檔案專案最多可儲存 256 項目。  當在 \[其他檔案的專案最後儲存 \<number\> 個項目\] 方塊 \(在 \[選項\] 對話方塊中 \[環境\] 選項的 \[文件\] 窗格上\) 中輸入的數字小於 1 或大於 256 時，一般會發生此錯誤。  
  
### 若要更正這個錯誤  
  
1.  請在方塊中輸入 1 和 256 之間的數字。
