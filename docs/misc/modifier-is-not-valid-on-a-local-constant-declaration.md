# &#39;&lt;修飾詞&gt;&#39; 在區域常數宣告中無效
程序內的 `Const` 陳述式包含無效的關鍵字，例如 `Overloads`。  
  
 **錯誤 ID︰**BC30246  
  
### 更正這個錯誤  
  
1.  請從 `Const` 陳述式中移除無效的關鍵字。
