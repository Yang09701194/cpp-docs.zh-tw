# &#39;Module&#39; 陳述式之後必須搭配相對應的 &#39;End Module&#39;
模組區塊的結尾必須是 `End``Module` 陳述式。  
  
 **錯誤 ID：**BC30625  
  
### 更正這個錯誤  
  
-   將 `End Module` 陳述式加入 `Module` 區塊的結尾。
