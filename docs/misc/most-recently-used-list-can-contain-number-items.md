# Most recently used list can contain &lt;number&gt; items.
MRU 清單最多可包含 24 個項目。  當在 \[MRU 清單包含 \<number\> 個項目\] 方塊 \(在 \[選項\] 對話方塊中 \[環境\] 選項的 \[一般\] 窗格上\) 中輸入的數字小於 1 或大於 24 時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請在方塊中輸入 1 和 24 之間的數字。
