# MSBuild 錯誤 MSB1001
**未知的參數。**  
  
 命令列上指定了未知的參數。  
  
### 若要更正這個錯誤  
  
1.  檢查參數拼字是否正確。  若要顯示有效參數的清單，請輸入 `msbuild /help`。
