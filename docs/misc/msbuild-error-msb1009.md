# MSBuild 錯誤 MSB1009
**專案檔不存在。**  
  
 指定的目錄中沒有這個專案檔，或者，如果沒有指定目錄，就表示目前的工作目錄中沒有這個專案檔。  
  
### 若要更正這個錯誤  
  
1.  請檢查檔名和路徑的拼字。
