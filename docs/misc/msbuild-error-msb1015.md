# MSBuild 錯誤 MSB1015
**MSBuild 無法在這個版本的作業系統上執行。  只有 Windows 2000、Windows XP 以及更新的版本才支援 MSBuild。**  
  
 這個版本的作業系統不支援 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)]。  
  
### 若要更正這個錯誤  
  
1.  在其中一個支援的作業系統版本中，安裝及執行 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)]。
