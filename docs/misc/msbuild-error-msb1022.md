# MSBuild 錯誤 MSB1022
**回應檔 \(Response File\) 不存在。**  
  
 指定的位置上沒有回應檔。  
  
### 若要更正這個錯誤  
  
1.  如果您已指定回應檔的絕對或相對位置，請檢查回應檔是否存在於指定的目錄中。
