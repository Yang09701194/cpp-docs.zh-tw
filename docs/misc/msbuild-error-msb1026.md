# MSBuild 錯誤 MSB1026
**沒有結構描述檔。**  
  
 在指定的位置上找不到結構描述檔。  
  
### 若要更正這個錯誤  
  
1.  如果您指定結構描述檔的絕對或相對位置，請檢查結構描述檔是否存在於指定的目錄中。
