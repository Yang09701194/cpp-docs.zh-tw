# MSBuild 錯誤 MSB1040
**MSB1040: ToolsVersion 無效。**  
  
 與 **\/ToolsVersion** 切換一起指定的 `ToolsVersion` 值無效。  
  
### 若要更正這個錯誤  
  
-   當您使用命令列上的 **\/ToolsVersion** 切換時，請指定有效的 `ToolsVersion` 值。
