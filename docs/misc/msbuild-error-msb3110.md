# MSBuild 錯誤 MSB3110
**MSB3110: 組件 '\<assembly\>' 的識別 '\<AssemblyIdentity\>' 不相符，必須是檔名: '\<file\>'。**  
  
 當從組件識別取得的組件檔名和實際檔名不相符時，就會產生這個錯誤訊息。  
  
### 若要更正這個錯誤  
  
-   更正組件檔名，讓它符合組件識別。
