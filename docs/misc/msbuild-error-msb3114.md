# MSBuild 錯誤 MSB3114
**MSB3114: 找不到組件 '\<assembly\>' 參考的檔案 '\<file\>'。**  
  
 如果在現有的資訊清單中找到無法解析的參考，就會產生這個錯誤訊息。  
  
### 若要更正這個錯誤  
  
-   檢查資訊清單中是否有檔案參考衝突。
