# MSBuild 錯誤 MSB3123
**MSB3123: 檔案關聯數目超過限制 \<limit\>。**  
  
 每個檔案都只能有固定數目的關聯副檔名。  當您嘗試關聯的副檔名數目超過目標作業系統容許的數量時，就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
-   將檔案關聯的數目限制在指定的數目之內。
