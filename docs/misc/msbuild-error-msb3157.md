# MSBuild 錯誤 MSB3157
**MSB3157: 無法與項目 '\<package\>' 的文化特性 '\<culture\>' 相符。  使用文化特性 '\<culture\>' 代替。**  
  
 當指定的產品找不到使用指定之文化特性的套件資訊清單檔 \(package.xml\) 時，就會產生這個警告訊息。  
  
### 若要更正這個錯誤  
  
-   提供適當的套件資訊清單檔。
