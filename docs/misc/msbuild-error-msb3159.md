# MSBuild 錯誤 MSB3159
**MSB3159: 檔案 '\<file\>' 中發生 Xml 驗證錯誤: '\<error\>'**  
  
 當產品或 package.xml 檔案不符合 XML 結構描述時，就會產生這個錯誤訊息。  這個錯誤訊息的後半部是某特定錯誤訊息。  
  
### 若要更正這個錯誤  
  
-   更新 XML 檔案以解決此驗證錯誤。
