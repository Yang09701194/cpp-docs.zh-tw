# MSBuild 錯誤 MSB3160
**MSB3160: 檔案 '\<file\>' 中發生 Xml 驗證警告: '\<error\>'**  
  
 當產品或 package.xml 檔案不符合 XML 結構描述時，就會產生這個警告訊息。  這個警告訊息的後半部是某特定警告訊息。  
  
### 若要更正這個錯誤  
  
-   更新 XML 檔案以解決此驗證錯誤。
