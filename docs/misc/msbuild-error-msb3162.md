# MSBuild 錯誤 MSB3162
**MSB3162: 項目 '\<package\>' 找不到相依項目 '\<package\>'。**  
  
 如果某個啟動載入器套件和另一個啟動載入器套件相依，但找不到相依啟動載入器必要條件的資訊清單，就會發生這個錯誤訊息。
