# MSBuild 錯誤 MSB3166
**MSB3166: 找不到項目 '\<package\>' 需要的檔案 '\<file\>'。**  
  
 當使用者授權合約 \(EULA\) 或外部檢查檔遺失時，就會產生這個錯誤訊息。  請確認該檔案存在於磁碟上。  
  
### 若要更正這個錯誤  
  
-   重新安裝 Visual Studio SDK  
  
     \-或\-  
  
-   將所需的檔案複製到適當的位置
