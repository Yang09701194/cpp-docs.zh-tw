# MSBuild 錯誤 MSB3171
**MSB3171: 產生資訊清單時發生問題。 '  \<error\>'**  
  
 在取得參考檔案或組件的相關資訊時，可能會產生這個未預期的錯誤訊息。  在產生應用程式資訊清單或部署資訊清單時，可能會產生這個訊息。
