# MSBuild 錯誤 MSB3173
**MSB3173: 無法寫入資訊清單 '\<file\>'。 '  \<error\>'**  
  
 如果建置處理序在寫入資訊清單檔時遇到了一般性的問題，就會產生這個錯誤訊息。
