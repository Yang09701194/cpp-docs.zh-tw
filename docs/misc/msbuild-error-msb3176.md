# MSBuild 錯誤 MSB3176
**MSB3176: 指定的最低必要版本高於目前的發行版本。  請指定低於或等於目前發行版本的版本。**  
  
 如果必要的最低應用程式版本高於目前部署資訊清單的版本，就會產生這個錯誤訊息。  
  
### 若要更正這個錯誤  
  
-   指定低於或等於目前發行版本的最低必要版本。
