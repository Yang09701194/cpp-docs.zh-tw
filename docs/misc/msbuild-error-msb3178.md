# MSBuild 錯誤 MSB3178
**MSB3178: 誤將組件 '\<assembly\>' 指定為檔案。**  
  
 如果建置處理序偵測到某個檔案參考實際上是 \(Managed 或原生\) 組件，就會在應用程式資訊清單產生期間產生這個警告訊息。
