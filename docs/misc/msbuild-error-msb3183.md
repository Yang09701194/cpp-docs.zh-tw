# MSBuild 錯誤 MSB3183
**MSB3183: 參考 '\<assembly\>' 是需要完全信任的 Interop 組件。**  
  
 如果在產生部分信任應用程式的資訊清單時，在參考中遇到了互通性組件，就會產生這個警告訊息。
