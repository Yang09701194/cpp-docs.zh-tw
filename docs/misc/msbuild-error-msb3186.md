# MSBuild 錯誤 MSB3186
**MSB3186: 無法從工作輸入參數推斷產生之資訊清單的組件識別。**  
  
 如果建置處理序無法推斷應用程式或部署資訊清單的組件名稱，就會產生這個錯誤訊息。  組件名稱並不是以明確的方式提供；基底資訊清單中沒有識別，也沒有指定進入點識別。
