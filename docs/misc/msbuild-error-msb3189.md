# MSBuild 錯誤 MSB3189
**MSB3189: 這個應用程式的更新位置是本機路徑。**  
  
 只有部署資訊清單會產生這個警告訊息。  
  
### 若要更正這個錯誤  
  
-   確保更新位置是有效的 URL。
