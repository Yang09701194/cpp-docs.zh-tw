# MSBuild 錯誤 MSB3484
**MSB3484: 找不到簽署目標 '\<file\>'。**  
  
 如果找不到您試圖簽署的檔案，就會產生這個錯誤訊息。  
  
### 若要更正這個錯誤  
  
-   確認指定了有效的檔案以進行簽署。
