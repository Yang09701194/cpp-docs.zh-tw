# MSBuild 錯誤 MSB4137
**MSB4137: 位於 "\<x\>" 的組態檔中指定了無效的值。  屬性名稱或工具版本名稱是空字串。**  
  
 工具組定義包含無效的屬性名稱或值。  
  
### 若要更正這個錯誤  
  
-   確定工具組定義沒有包含無效或空白屬性名稱或值。
