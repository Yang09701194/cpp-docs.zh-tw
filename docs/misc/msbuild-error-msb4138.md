# MSBuild 錯誤 MSB4138
**MSB4138: 在登錄位置 "'\<key\>'" 指定了非字串資料。**  
  
 自訂工具組組態中所指定的登錄位置無效。  
  
### 若要更正這個錯誤  
  
-   確定工具組定義正確，且指定的登錄機碼存在。
