# MSBuild 錯誤 MSB8012
*{0}\(\\{1\\}\)* 不符合 *\\{2\\}* 的 OutputFile 屬性值 \(*\\{3\\}*\)。 這可能導致您的專案無法正確建立。 若要更正這個問題，請確定 $\(OutDir\)、$\(TargetName\) 和 $\(TargetExt\) 屬性值符合 %\(\\{4\\}.OutputFile\) 中指定的值。
