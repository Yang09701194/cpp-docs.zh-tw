# MSBuild 錯誤 MSB8020
找不到 *\<平台\>* 的建置工具 \(平台工具集 \= '*\<版本\>*'\)。 若要使用 *\<版本\>* 建置工具進行建置，請安裝 *\<平台\>* 建置工具。  或者，您可以選取 \[專案\] 功能表，或在方案上按一下滑鼠右鍵，然後選取 \[重定方案目標\]，升級成最新版的 Visual Studio 工具。
