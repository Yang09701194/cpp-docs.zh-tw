# 無法以方法 &#39;&lt;方法名稱&gt;&#39; 使用 &#39;MyBase&#39;，因為它被宣告為 &#39;MustOverride&#39;
您嘗試搭配使用 `MyBase` 與已宣告 `MustOverride` 的方法。  
  
 **錯誤 ID︰**BC30399  
  
### 更正這個錯誤  
  
-   請移除 `MustOverride` 宣告。
