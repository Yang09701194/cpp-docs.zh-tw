# &#39;MyClass&#39; 無法在類別外使用
您已在類別結構外使用 `MyClass` 關鍵字。  
  
 **錯誤 ID︰**BC30470  
  
### 更正這個錯誤  
  
-   請將 `MyClass` 宣告移至您的類別結構。
