# &#39;&lt;名稱&gt;&#39; 無法將其實作之事件的基礎委派類型 &#39;&lt;委派類型&gt;&#39; 中所用的類型 &#39;&lt;類型&gt;&#39;，透過 &lt;指定名稱2&gt; &#39;&lt;類型2&gt;&#39; 公開給 &lt;指定名稱1&gt; &#39;&lt;類型1&gt;&#39;
這個程式碼會在公用類別外部公開私用類型。  
  
 **錯誤 ID：**BC30923  
  
### 更正這個錯誤  
  
-   宣告 `Public` 類型或使用另一種類型。
