# &lt;name&gt; is not a valid project name.
從 \[**命令**\] 視窗或 \[**尋找\/命令**\] 方塊發出 `File.NewProject` 命令，且針對 *projectname* 輸入不正確的專案名稱時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請確認專案名稱沒有包含額外的空格，並且不是保留字，例如 NUL、CON 或 AUX。  
  
     \-或\-  
  
     重新輸入命令，並省略專案名稱。
