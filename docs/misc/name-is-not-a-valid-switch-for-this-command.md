# &lt;name&gt; is not a valid switch for this command.
當輸入給命令的參數或單一字元參數組合不正確時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請檢查文件中命令的正確語法，然後再試一次。
