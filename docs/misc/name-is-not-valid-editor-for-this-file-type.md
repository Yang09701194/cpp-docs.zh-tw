# &lt;name&gt; is not valid editor for this file type.
在 \[命令\] 視窗或 \[尋找\/命令\] 方塊中輸入 \/e:*editorname* 引數的不正確值時，通常就會發生這項錯誤。  
  
### 若要更正這個錯誤  
  
1.  請檢查文件中正確的 \/e:*editorname* 語法，並重新輸入命令。
