# 無法在類別 &#39;&lt;類別名稱&gt;&#39; 上使用 &#39;New&#39;，因為它包含一個尚未被覆寫的 &#39;MustOverride&#39; 成員
您已嘗試在包含尚未覆寫之 `MustOverride` 成員的類別上使用 `New`。  
  
 **錯誤 ID︰**BC30376  
  
### 更正這個錯誤  
  
-   請移除 `New` 陳述式。
