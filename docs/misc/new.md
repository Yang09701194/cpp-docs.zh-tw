# new
簡要描述 `new` 關鍵字的用法和詳細資訊的連結。  
  
|標題|說明|  
|--------|--------|  
|[operator new](../c-runtime-library/operator-new-crt.md)|從堆積配置記憶體區塊。|  
|[operator new](../Topic/operator%20new%20\(%3Cnew%3E\).md)|new\-expression 配置個別物件儲存區時呼叫的函式。|  
|[new 運算子](../cpp/new-operator-cpp.md)|從可用存放區針對物件或物件陣列配置記憶體，並傳回具適當類型的非零物件指標。|  
|[new \(new slot in vtable\)](../windows/new-new-slot-in-vtable-cpp-component-extensions.md)|在 C\+\+\/CLI 專案中，表示虛擬成員在 vtable 中取得新位置。  換句話說，函式並不會覆寫基底類別方法。|  
|[ref new, gcnew](../windows/ref-new-gcnew-cpp-component-extensions.md)|在 [!INCLUDE[cppwrt_short](../build/reference/includes/cppwrt_short_md.md)] \(`ref new`\) 或 C\+\+\/CLI \(`gcnew`\) 專案中，這個複合關鍵字會建立參考計數的物件。|
