# 無法評估 &#39;Nothing&#39;
已在偵錯視窗中使用 `Nothing` 關鍵字。  
  
 **錯誤 ID︰**BC30701  
  
### 更正這個錯誤  
  
-   請不要在偵錯時使用 `Nothing`。
