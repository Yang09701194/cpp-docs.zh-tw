# &lt;number&gt; power closure argument &#39;n&#39; must be from 1 to 256.
這個錯誤通常在數字超過 1 到 256 的可接受範圍時發生。  
  
### 若要更正這個錯誤  
  
1.  請輸入介於 1 到 256 的數字，然後再試一次。  如需語法的詳細資訊，請參閱[規則運算式](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md)。
