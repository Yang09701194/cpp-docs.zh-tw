# 不再支援 &#39;On GoTo&#39; 和 &#39;On GoSub&#39; 陳述式
搭配使用變數或運算式的值與 `GoTo` 和 `GoSub` 陳述式來控制程式流程，不再有效。  
  
 **錯誤 ID︰**BC30817  
  
### 更正這個錯誤  
  
-   請重構您的應用程式，以使用 `If...Then...Else` 或 `Case` 陳述式。
