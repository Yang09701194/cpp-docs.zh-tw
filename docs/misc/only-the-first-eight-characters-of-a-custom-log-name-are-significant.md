# 只有自訂記錄檔名稱的前八個字元才有意義
檢查事件記錄檔名稱的唯一性時，只會考慮前八個字元。 共用前八個字元的事件記錄檔可能會產生衝突。  
  
### 更正這個錯誤  
  
-   提供給事件記錄檔的名稱中其前八個字元為唯一。
