# &#39;Option Explicit&#39; 之後只能是 &#39;On&#39; 或 &#39;Off&#39;
`Option Explicit` 陳述式必須指定 `On` 或 `Off`。 所有其他值都無效。  
  
 **錯誤 ID︰**BC30640  
  
### 更正這個錯誤  
  
-   請指定 `Option Explicit On` 或 `Option Explicit Off`。
