# Option is not currently valid.
這個錯誤通常在命令已指定的引數或參數目前無法使用時發生。  例如，如果您輸入 `>Edit.Find test /sel` 且檔案中沒有選取範圍，則 \/sel 參數將無法使用。  
  
### 若要更正這個錯誤  
  
1.  請驗證所輸入的引數和參數目前可用。  如需詳細資訊，請參閱[使用引數的 Visual Studio 命令](../Topic/Visual%20Studio%20Commands.md)。
