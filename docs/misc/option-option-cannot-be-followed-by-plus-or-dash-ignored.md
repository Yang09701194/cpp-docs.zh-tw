# 選項 &#39;&lt;選項&gt;&#39; 後面不能跟隨著 &#39;+&#39; 或 &#39;-&#39;; 已忽略
編譯器選項已指定值 `+` 或 `-` 不支援這些值。  
  
 如需編譯器選項和其所支援值的詳細資訊，請參閱 [Visual Basic Compiler Options Listed Alphabetically](../Topic/Visual%20Basic%20Compiler%20Options%20Listed%20Alphabetically.md)。  
  
 **錯誤 ID︰**BC2028  
  
### 更正這個錯誤並使用選項  
  
-   指定沒有任何值的編譯器選項。  
  
### 更正這個錯誤但不使用選項  
  
-   移除編譯器選項。
