# 多載解析失敗，因為沒有可存取的 &#39;&lt;方法
您已呼叫多載的方法，但編譯器找不到具有相符引數清單的方法。  
  
 **錯誤 ID︰**BC30516  
  
### 更正這個錯誤  
  
-   請檢查您提供的引數清單。
