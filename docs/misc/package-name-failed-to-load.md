# Package &lt;name&gt; failed to load.
無法開啟套件。  當未正確登錄套件、套件遺漏 DLL、DLL 已損毀或遺漏套件由其他元件所取得的 DLL 時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請重新安裝提供套件的產品。
