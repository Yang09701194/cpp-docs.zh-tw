# ParamArray 參數必須是一維陣列
`ParamArray` 引數必須宣告為一維陣列類型。 宣告指定 `ParamArray` 引數的多個維度。  
  
 **錯誤 ID︰**BC30051  
  
### 更正這個錯誤  
  
-   請移除 `ParamArray` 引數名稱後面之括弧內的所有逗號。
