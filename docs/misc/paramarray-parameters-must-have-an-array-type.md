# ParamArray 參數必須有陣列類型
存取 MSCorlib 組件時發生問題，造成無效的類型中有 `ParamArray` 參數。  
  
 **錯誤 ID︰**BC31092  
  
### 更正這個錯誤  
  
1.  再次編譯程式，看看錯誤是否重複發生。  
  
2.  如果錯誤重複發生，請儲存您的工作，然後重新啟動 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]。  
  
3.  如果問題持續發生，請重新安裝 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)]。  
  
4.  如果錯誤在重新安裝之後持續發生，請通知 Microsoft 產品支援服務。
