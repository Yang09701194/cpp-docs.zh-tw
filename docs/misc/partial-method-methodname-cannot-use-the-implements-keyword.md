# 部分方法 &#39;&lt;方法名稱&gt;&#39; 不可以使用 &#39;Implements&#39; 關鍵字
部分方法定義無法實作介面或介面成員。  
  
 **錯誤 ID︰**BC31439  
  
### 更正這個錯誤  
  
1.  從部分方法定義中移除 `Implements` 陳述式。
