# Path or drive is not valid.
通常當 \[查詢\] 對話方塊中的 \[瀏覽\] 方塊為空白，而您卻按下 ENTER 鍵時，便會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請輸入路徑或目錄然後按 ENTER。
