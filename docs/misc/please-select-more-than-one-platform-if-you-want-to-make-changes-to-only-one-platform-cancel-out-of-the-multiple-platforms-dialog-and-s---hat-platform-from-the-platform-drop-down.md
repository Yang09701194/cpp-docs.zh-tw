# 請選取一個以上的平台。 若只想變更一個平台，請取消 [多重平台] 對話方塊，再從下拉式清單選取該平台。
當 **\[多重平台\]** 對話方塊中已選取單一平台時，通常會發生此錯誤。  
  
### 更正這個錯誤  
  
1.  選取其他平台。  
  
     \-或\-  
  
2.  如果您想要更新單一平台，請選擇 \[取消\]，然後從 \[平台\] 清單中選取正確的平台。
