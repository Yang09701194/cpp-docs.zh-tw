# Please specify valid search string.
在 \[尋找符號\] 對話方塊的 \[尋找目標\] 中試圖不輸入字串而進行搜尋時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請在 \[尋找目標\] 方塊中輸入正確的文字，然後再選擇 \[尋找\]。
