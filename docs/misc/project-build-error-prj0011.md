# 專案建置錯誤 PRJ0011
系統資源可能會非常低。 無法建立啟動建置所需的事件。  
  
 如果系統資源不足：  
  
-   關閉一些執行中應用程式。  
  
-   重新啟動 Visual Studio 以釋出資源。  
  
-   重新開機。
