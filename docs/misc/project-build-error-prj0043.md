# 專案建置錯誤 PRJ0043
指派給檔案 'file' 之自訂建置規則 'rule' 的 'Outputs' 屬性無效。 這個屬性包含評估為 'value' 的 ' string'。  
  
 Outputs 屬性評估為空字串。 自訂建置規則需要建置動作的輸出。
