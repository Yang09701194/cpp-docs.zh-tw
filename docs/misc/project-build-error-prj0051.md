# 專案建置錯誤 PRJ0051
無法將 'file' 複製到目標目錄。  請確定目的地檔案不在使用中。  
  
 建置系統無法複製檔案。  
  
 請確定電腦未開啟此檔案，或其他程式未使用此檔案。
