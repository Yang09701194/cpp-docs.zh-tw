# 專案建置警告 PRJ0045
未設定指派給檔案 'file' 之自訂建置規則 'rule' 的 'Outputs' 屬性。 會略過執行規則。  
  
 未定義自訂建置規則的 Outputs 屬性。 自訂建置規則需要建置動作的輸出。
