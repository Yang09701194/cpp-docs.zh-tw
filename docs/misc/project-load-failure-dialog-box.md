# 專案載入失敗對話方塊
如果無法載入專案檔，就會顯示這個對話方塊。  
  
 **請修正指定的錯誤並重新載入專案。**  
 會提供錯誤的相關資訊。  請使用這個對話方塊中的資訊找出及解決專案檔中的錯誤。  儲存專案，然後在 IDE 中重新載入專案。  
  
 **不要再於此工作階段顯示這個對話方塊**  
 若選取這個選項，直到您下次開啟 IDE 之前 \[**專案載入失敗**\] 對話方塊都不會再出現。
