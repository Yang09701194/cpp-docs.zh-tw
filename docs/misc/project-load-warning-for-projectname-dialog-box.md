# &lt;專案名稱&gt; 的專案載入警告對話方塊
如果 IDE 中開啟的專案其設定載入不正確，便會出現這個對話方塊。  
  
 **請更正指定的錯誤並重新載入專案**  
 提供有關無效路徑或檔名這類複雜問題的資訊。  若要解決錯誤，請移除或取代專案屬性頁中任何無效的參考。  儲存專案，然後在 IDE 中重新載入專案。  
  
 **不要再於此工作階段顯示這個對話方塊**  
 若選取這個選項，直到您下次開啟 IDE 之前，\<Projectname\> 專案載入警告對話方塊都不會再出現。
