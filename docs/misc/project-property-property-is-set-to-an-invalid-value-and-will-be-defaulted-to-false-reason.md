# Project property &#39;property&#39; is set to an invalid value and will be defaulted to FALSE. &lt;reason&gt;
`Boolean` 屬性保存的值不是下列其中一個值：  
  
-   1 或 0  
  
-   On 或 Off  
  
-   true 或 false  
  
 最可能導致這個錯誤的原因是以手動方式編輯專案檔。  
  
 **若要更正這個錯誤**  
  
-   再次編輯專案檔案並且將專案檔案還原成令人滿意的條件。  
  
     受影響的屬性值將設定為 false。
