# Project type is not valid.
輸入不正確值給 \[**加入新的專案**\] 命令的引數 *templatename* 時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請檢查文件中的 \[**加入新的專案**\] 命令是否為正確語法，然後再試一次。
