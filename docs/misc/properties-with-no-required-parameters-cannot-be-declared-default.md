# 沒有要求參數的屬性不可以宣告為 &#39;Default&#39;
指定的屬性沒有必要參數，但具有 `Default` 修飾詞。  
  
 **錯誤 ID︰**BC31048  
  
### 更正這個錯誤  
  
-   宣告預設屬性的必要參數。  
  
-   從屬性定義中移除 `Default` 修飾詞。
