# 屬性或欄位 &#39;&lt;名稱&gt;&#39; 沒有有效的屬性類型
您使用的屬性建構函式可能不接受有效的屬性類型，即使屬性可有其他適當的建構函式。  
  
 **錯誤 ID：**BC30659  
  
### 更正這個錯誤  
  
1.  請檢查這個屬性的使用方式。
