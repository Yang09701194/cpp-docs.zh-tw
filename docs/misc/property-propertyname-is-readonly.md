# 屬性 &#39;&lt;屬性名稱&gt;&#39; 是 &#39;ReadOnly&#39;
您已嘗試指派值給宣告為 `ReadOnly` 的屬性。  
  
 **錯誤識別碼：**BC30526  
  
### 更正這個錯誤  
  
-   請移除屬性宣告的 `ReadOnly` 指定名稱。
