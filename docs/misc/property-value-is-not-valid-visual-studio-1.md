# Property value is not valid. (Visual Studio) (1)
當輸入對屬性無效的值時，通常就會發生這個錯誤。  例如，如果屬性是有效值為 TRUE 和 FALSE 值的 Boolean 屬性，那麼對這個屬性而言 MAYBE 值是無效的。  
  
### 若要更正這個錯誤  
  
1.  請輸入對屬性有效的值。  
  
    > [!TIP]
    >  請在線上文件中搜尋屬性名稱，以找出有效值。
