# Property value is not valid. (Visual Studio) (2)
當輸入對屬性無效的值時，通常就會發生這個錯誤。  例如，輸入整數值的錯誤格式，或對短值輸入過長的值。  
  
### 若要更正這個錯誤  
  
1.  請輸入對屬性有效的值。  
  
    > [!TIP]
    >  請搜尋線上文件，以找出對屬性有效的值。
