# &#39;&lt;屬性名稱&gt;&#39; 隱含定義 &#39;&lt;成員名稱&gt;&#39;，其與 &lt;類型&gt; &#39;&lt;類型名稱&gt;&#39; 中同名的成員互相衝突
類型成員名稱與為屬性隱含建立的成員名稱相衝突。 屬性會隱含建立數個與已定義成員發生衝突的隱含變數。  
  
 **錯誤 ID︰**BC31060  
  
### 更正這個錯誤  
  
-   請重新命名明確宣告的成員，以移除命名衝突。
