# 無法在此內容中存取 &#39;&lt;完整方法&gt;&#39;，因為它是 &#39;&lt;修飾詞&gt;&#39;
您嘗試存取因已套用修飾詞而無法在此內容中存取的方法。  
  
 **錯誤 ID︰**BC30390  
  
### 更正這個錯誤  
  
-   請移除修飾詞。
