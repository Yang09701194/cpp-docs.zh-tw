# &#39;For&#39; 迴圈控制變數已被封閉式 &#39;For&#39; 迴圈所使用
巢狀 `For` 迴圈必須全部使用唯一的控制變數。  
  
 **錯誤 ID︰**BC30975  
  
### 更正這個錯誤  
  
-   變更其中一個控制變數名稱。
