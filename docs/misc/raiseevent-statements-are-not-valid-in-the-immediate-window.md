# 即時運算視窗中的 &#39;RaiseEvent&#39; 陳述式無效
\[即時運算\] 視窗中不能使用 `RaiseEvent` 陳述式。  
  
 **錯誤 ID︰**BC30739  
  
### 更正這個錯誤  
  
-   在 \[程式碼編輯器\] 中，透過程式碼測試 `RaiseEvent` 陳述式。
