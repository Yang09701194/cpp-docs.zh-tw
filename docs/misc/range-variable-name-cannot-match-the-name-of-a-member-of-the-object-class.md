# 範圍變數名稱不符合 &#39;Object&#39; 類別成員的名稱
LINQ 查詢中的範圍變數符合 <xref:System.Object> 類別成員的名稱。 這會產生與 Visual Basic 針對 LINQ 查詢所建立之物件的衝突。  
  
 **錯誤 ID：**BC36606  
  
### 更正這個錯誤  
  
1.  請為不符合任何 <xref:System.Object> 類別成員名稱的範圍變數，選擇新名稱。
