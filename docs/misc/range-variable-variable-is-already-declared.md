# 範圍變數 &lt;變數&gt; 已宣告
`Select` 子句中指定的範圍變數名稱，或 `Aggregate`、`Group By` 或 `Group Join` 子句的 `Into` 部分，會重複已經在查詢子句中指定之範圍變數的名稱。  
  
 **錯誤 ID︰**BC36600  
  
### 更正這個錯誤  
  
1.  請確定查詢子句中的所有範圍變數都有唯一的名稱。
