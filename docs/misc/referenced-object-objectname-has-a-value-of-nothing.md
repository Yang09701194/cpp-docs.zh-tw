# 參考的物件 &#39;&lt;物件名稱&gt;&#39; 具有 &#39;Nothing&#39; 值
參考的物件尚未設定，或已設定為 `Nothing` 值。  
  
 **錯誤 ID︰**BC30705  
  
### 更正這個錯誤  
  
-   請確認物件名稱拼字正確，且已設定該物件。
