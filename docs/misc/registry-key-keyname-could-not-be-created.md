# 無法建立登錄機碼 &#39;&lt;機碼名稱&gt;&#39;
無法建立指定的登錄機碼，原因可能是安全性權限不足。  
  
### 更正這個錯誤  
  
-   請確定您有足夠的安全性權限可以存取機碼。
