# &#39;RemoveHandler&#39; 宣告之後必須搭配相對應的 &#39;End RemoveHandler&#39;
`RemoveHandler` 宣告的結尾必須是 `End RemoveHandler` 陳述式。  
  
 **錯誤 ID︰**BC31116  
  
### 更正這個錯誤  
  
-   請檢查 `RemoveHandler` 宣告的結尾是否為 `End RemoveHandler` 陳述式。
