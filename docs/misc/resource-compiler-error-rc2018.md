# 資源編譯器錯誤 RC2018
未知的字元 'hexnumber'  
  
 對應至所指定十六進位數字的 ASCII 字元出現在原始程式檔中，但為不合法的字元。  
  
 這個錯誤可能是原始程式檔損毀所造成。
