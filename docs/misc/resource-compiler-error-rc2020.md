# 資源編譯器錯誤 RC2020
基底基數的數字不合法  
  
 指定的 `digit` 不是 `radix` 所指定之基底的有效數字。 可能是數字或基數不正確。  
  
 八進位數字必須是 0 到 7 的數字，而十六進位數字必須是 0 到 9 的數字或 A 到 E 的字母。
