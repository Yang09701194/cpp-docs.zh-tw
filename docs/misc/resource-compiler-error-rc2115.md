# 資源編譯器錯誤 RC2115
控制項中必須有文字字串或序數  
  
 **DIALOG** 陳述式中 CONTROL 陳述式的 *text* 欄位必須是參考控制項類型的文字字串或序數。 如果使用序數，請確定控制項具有 `#define` 陳述式。
