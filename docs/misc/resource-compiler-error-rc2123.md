# 資源編譯器錯誤 RC2123
功能表中預期為 END  
  
 **END** 關鍵字必須放在 **MENU** 陳述式結尾。 請確定您沒有不完整的引號或不成對的 **BEGIN** 和 **END** 陳述式。
