# 資源編譯器錯誤 RC2126
必須是功能表字串  
  
 每個 **MENUITEM** 和 **POPUP** 陳述式皆必須包含「文字」欄位，該欄位為以雙引號括住並指定功能表項目或快顯功能表名稱的字串。**MENUITEM SEPARATOR** 陳述式需要不含引號的字串。
