# 資源編譯器錯誤 RC2132
必須有 VALUE、BLOCK 或 END 關鍵字  
  
 **VERSION** 資源中的區塊未正確地結束，或新的區塊未正確地開始。 這會導致不是有效區塊的 **VERSION** 陳述式。
