# 資源編譯器錯誤 RC2138
關閉註解 '\/\*' 不對稱  
  
 開啟註解順序 \(`/*`\) 與關閉註解順序 \(`*/`\) 不對稱。這個錯誤可能是巢狀註解所造成。
