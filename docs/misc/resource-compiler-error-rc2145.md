# 資源編譯器錯誤 RC2145
PRIMARY LANGUAGE ID 太大  
  
 PRIMARY LANGUAGE ID 必須是有效的十六進位語言 ID。 請參閱《執行階段程式庫參考》[](../c-runtime-library/locale-names-languages-and-country-region-strings.md "Locale Names, Languages, and Country-Region Strings")中的*語言和國家\/地區字串*，以取得有效語言 ID 的清單。
