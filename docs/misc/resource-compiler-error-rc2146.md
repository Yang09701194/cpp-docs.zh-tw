# 資源編譯器錯誤 RC2146
LANGUAGE 陳述式中遺漏逗號  
  
 遺漏用來分隔主要語言和次要語言的逗點。  
  
 **LANGUAGE** 陳述式必須使用下列語法：  
  
 **LANGUAGE** *primary\_language\_ID*,*secondary\_language\_ID*
