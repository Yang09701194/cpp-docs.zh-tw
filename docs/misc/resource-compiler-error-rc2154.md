# 資源編譯器錯誤 RC2154
控制字元超出範圍 \[^A \- ^Z\]  
  
 **ACCELERATORS** 陳述式中的控制字元無效。 插入號 \(**^**\) 後的字元必須介於 A 與 Z \(含\) 之間。
