# 資源編譯器錯誤 RC2159
無效的快速鍵類型 \[ASCII 或 VIRTKEY\]  
  
 **ACCELERATORS** 陳述式中的 `type` 欄位必須包含 ASCII 或 VIRTKEY 值。
