# 資源編譯器錯誤 RC2174
點陣圖檔的檔案名稱格式不是 2.03  
  
 點陣圖使用的是 2.03 版之前的格式。 必須使用 3.00 版或更新版本的格式來轉換或重繪點陣圖。
