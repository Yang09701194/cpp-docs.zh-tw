# 資源編譯器錯誤 RC2176
**檔名中有舊的 DIB。 透過 SDKPAINT 傳遞它。**  
  
 在指定的檔案中找到舊格式的裝置獨立點陣圖。 它應該轉換成目前的格式。 您可以使用 Windows 3.0 SDK 提供的 SDKPAINT 應用程式或同等的應用程式，執行這項操作。
