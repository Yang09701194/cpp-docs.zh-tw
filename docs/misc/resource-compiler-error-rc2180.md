# 資源編譯器錯誤 RC2180
無法開啟暫存檔案  
  
 資源編譯器\/Visual C\+\+ 無法開啟暫存檔。 可能的原因是您沒有目錄的寫入權，或目錄不存在。 資源編譯器\/Visual C\+\+ 嘗試在 **TMP** 環境變數所指定的目錄或目前的目錄 \(如果未指定\) 中使用這些檔案。
