# 資源編譯器嚴重錯誤 RC1005
找不到檔案：filename  
  
 找不到資源編譯器命令列中所指定的檔案。 請確認是否已將檔案移至另一個目錄，以及是否正確地輸入檔名或路徑。  
  
 使用 **INCLUDE** 環境變數或 Visual C\+\+ 目錄設定來搜尋檔案。
