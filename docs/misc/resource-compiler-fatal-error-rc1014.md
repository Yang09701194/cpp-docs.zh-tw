# 資源編譯器嚴重錯誤 RC1014
Include 檔太多：深度 \= 'level'  
  
 `#include` 指示詞的巢狀深度太大。  
  
 請巢狀指示詞包含開啟的檔案。 包含指示詞的原始程式檔也算做檔案之一。
