# 資源編譯器嚴重錯誤 RC1023
無法開啟原始程式檔 'filename'  
  
 指定的檔案不存在、無法開啟或找不到。  
  
 請確定環境設定有效，而且指定的檔案路徑正確。  
  
 如果發生這個錯誤但未出現錯誤訊息，表示資源編譯器已用完檔案控制代碼。
