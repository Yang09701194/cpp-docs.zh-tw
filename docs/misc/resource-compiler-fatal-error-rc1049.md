# 資源編譯器嚴重錯誤 RC1049
無效的數值引數 'string'  
  
 資源編譯器必須要有數值引數，但卻收到指定的字串。  
  
 如果指定沒有必要 \\x 前置詞的十六進位數字，或浮點數的格式不正確，則可能會造成這個錯誤。
