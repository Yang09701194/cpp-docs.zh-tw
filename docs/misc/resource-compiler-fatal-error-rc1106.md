# 資源編譯器嚴重錯誤 RC1106
無效的參數: option  
  
 這個選項不是有效的資源編譯器選項。  
  
 有效的選項為：  
  
 \/r  
  
 發出 .res 檔案  
  
 \/v  
  
 詳細資訊 \(列印進度訊息\)  
  
 \/d  
  
 定義符號  
  
 \/fo  
  
 重新命名 .res 檔案  
  
 \/l  
  
 十六進位的預設語言 ID  
  
 \/I  
  
 加入 **INCLUDE** 搜尋的路徑  
  
 \/x  
  
 忽略 **INCLUDE** 環境變數  
  
 \/c  
  
 定義 NLS 轉換所使用的字碼頁
