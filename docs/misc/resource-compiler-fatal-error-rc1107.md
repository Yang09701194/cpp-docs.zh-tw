# 資源編譯器嚴重錯誤 RC1107
使用方式無效; 使用 RC \/? 針對說明  
  
 指定的資源編譯器選項無效。 有效的選項為 \/r、\/v、\/d、\/fo、\/l、\/i、\/x 和 \/c。
