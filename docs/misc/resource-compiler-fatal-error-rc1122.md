# 資源編譯器嚴重錯誤 RC1122
寫入檔案時發生 I\/O 錯誤  
  
 資源編譯器無法寫入檔案中。  
  
### 透過檢查下列可能原因進行修正  
  
1.  磁碟空間不足。 可用空間必須大約等於所建立可執行檔大小的兩倍。  
  
2.  磁碟區是唯讀的。  
  
3.  磁區損壞。  
  
4.  共用違規。
