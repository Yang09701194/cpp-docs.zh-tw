# 資源編譯器嚴重錯誤 RC1206
命令列上所指定的字碼頁不存在於登錄中  
  
 指定字碼頁 \(\/c\) 選項之後接著未在登錄中指定其位置的字碼頁。 執行 SETUP 以安裝指定的字碼頁。
