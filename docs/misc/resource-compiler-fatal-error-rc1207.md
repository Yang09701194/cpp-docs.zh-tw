# 資源編譯器嚴重錯誤 RC1207
預設字碼頁不正確  
  
 **RCCODEPAGE** 環境變數所指定的字碼頁不正確。 如需詳細資訊，請參閱*執行階段程式庫參考*中的[字碼頁](../c-runtime-library/code-pages.md)。
