# 資源編譯器警告 RC4003
巨集 'identifier' 的實質參數不足  
  
 使用指定識別項指定的實質參數數目，小於識別項之巨集定義中所指定的型式參數數目。  
  
 當定義中參考了型式參數，但尚未提供對應的實質參數時，便會在巨集展開中以空白文字取代。
