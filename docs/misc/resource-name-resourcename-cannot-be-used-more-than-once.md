# 資源名稱 &#39;&lt;資源名稱&gt;&#39; 不可以重複使用
指定的資源名稱在動態連結程式庫或可執行檔中重複使用。  
  
 **錯誤 ID︰**BC31502  
  
### 更正這個錯誤  
  
-   為資源指定唯一的名稱。
