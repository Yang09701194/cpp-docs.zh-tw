# 選取程式設計語言, Visual Studio 封裝精靈
這個步驟可讓您：  
  
-   選取您想要用來建立 VSPackage 的程式設計語言。  
  
-   產生或使用現有的金鑰檔簽署 VSPackage 組件。  
  
 **Visual C\#**  
 以 C\# 建立 VSPackage。  
  
 **Visual Basic**  
 以 Visual Basic 建立 VSPackage。  
  
 **Visual C\+\+**  
 以 Visual C\+\+ 建立 VSPackage。  
  
## 取得金鑰以簽署組件  
 **產生新金鑰檔以簽署組件**  
 導致精靈為您產生金鑰檔。 產生的金鑰檔稱為 key.snk。  
  
 **使用下列金鑰檔簽署組件：**  
 讓您輸入現有金鑰檔案的路徑，或按一下 \[瀏覽\] 取得檔案。 精靈會建立一份檔案，並將它稱為 key.snk。  
  
 **取消**  
 取消建立 VSPackage。  
  
 **上一步**  
 移至精靈的上一頁。  
  
 **下一步**  
 移至精靈的下一頁。  
  
 **完成**  
 建立 VSPackage。
