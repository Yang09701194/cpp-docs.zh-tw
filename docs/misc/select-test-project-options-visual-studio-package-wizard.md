# 選取測試專案選項, Visual Studio 封裝精靈
這個步驟可讓您：  
  
-   產生 VSPackage 的整合測試專案。  
  
-   產生 VSPackage 的單元測試專案。  
  
 **整合測試專案**  
 建立整合測試專案。 整合測試會在 Visual Studio 的實驗執行個體中執行，並使用 Automation 模型來測試封裝在 Visual Studio 中的整合程度。  
  
 **單元測試專案**  
 建立單元測試專案。 單元測試是可用來測試封裝中之個別類別和成員的小型 MSTest 執行個體。  
  
 **取消**  
 取消並停止新增專案命令。  
  
 **上一頁**  
 移至上一個步驟。  
  
 **完成**  
 建立 VSPackage。
