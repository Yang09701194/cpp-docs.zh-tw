# 在執行階段不支援 Set
您嘗試設定或變更其值只能在設計階段設定的屬性。  
  
### 更正這個錯誤  
  
1.  從您的程式碼中移除屬性的參考。  
  
2.  變更參考，使其僅在執行階段傳回屬性的值。
