# 共用的 &#39;Sub New&#39; 不可以有任何參數
您已在共用 `Sub New` 陳述式中加入一個或多個參數，這是無效的作業。  
  
 **錯誤 ID︰**BC30479  
  
### 更正這個錯誤  
  
-   請移除一個或多個參數。
