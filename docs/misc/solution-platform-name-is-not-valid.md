# Solution platform name is not valid.
方案平台名稱不能包含下列字元：  
  
-   井字號 \(\#\)  
  
-   百分比 \(%\)  
  
-   連字號 \(&\)  
  
-   星號 \(\*\)  
  
-   垂直線 \(&#124;\)  
  
-   反斜線 \(\\\)  
  
-   冒號 \(:\)  
  
-   雙引號 \("\)  
  
-   小於 \(\<\)  
  
-   大於 \(\>\)  
  
-   句號 \(.\)  
  
-   問號 \(?\)  
  
-   正斜線 \(\/\)  
  
-   前置空格或尾端空格 \(' '\)  
  
-   Windows 或 DOS 的保留名稱  \(nul、aux、con、com1、lpt1 等等\)  
  
### 若要更正這個錯誤  
  
1.  請輸入不包含上方所列字元的方案平台名稱。
