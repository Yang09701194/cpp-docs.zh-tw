# Some Unicode characters in this file will not be saved.
這項錯誤通常發生於檔案含有只能藉由套用 Unicode 字碼頁 \(Code Page\) 儲存的字元。  
  
### 以 Unicode 字元儲存檔案  
  
1.  按一下 \[**取消**\]。  
  
2.  從 \[**檔案**\] 功能表中，選擇 \[**進階儲存選項**\]。  
  
3.  在 \[**進階儲存選項**\] 對話方塊中，選擇適合您檔案的編碼方式。
