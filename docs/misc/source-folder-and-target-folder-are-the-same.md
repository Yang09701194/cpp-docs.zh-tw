# 來源資料夾和目標資料夾相同
嘗試複製或移動檔案，但指定的來源目錄和目標目錄相同。 這可能是檔案路徑剖析不正確的結果。  
  
### 更正這個錯誤  
  
-   請確定您提供正確的來源和目標目錄。
