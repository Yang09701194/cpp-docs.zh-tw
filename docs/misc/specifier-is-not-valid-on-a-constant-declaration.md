# &#39;&lt;規範&gt;&#39; 在常數宣告中無效
`Const` 陳述式包含無效的關鍵字。`Const` 陳述式只能包括 `Friend`、`Private`、`Protected`、`Public` 或 `Shadows` 關鍵字。  
  
 **錯誤 ID︰**BC30233  
  
### 更正這個錯誤  
  
1.  請確定關鍵字拼寫正確。  
  
2.  請從 `Const` 陳述式中移除無效的關鍵字。
