# &#39;&lt;規範&gt;&#39; 在 Structure 宣告中無效。
您使用了對 `Structure` 宣告無效的指定名稱，例如 `Dim` 或 `Override`。  
  
 **錯誤識別碼：**BC30395  
  
### 更正這個錯誤  
  
-   請移除規範。
