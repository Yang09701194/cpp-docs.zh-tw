# &#39;&lt;規範&gt;&#39; 在 Enum 宣告中無效
您使用了對 `Enum` 宣告無效的規範。  
  
 **錯誤 ID：**BC30396  
  
### 更正這個錯誤  
  
1.  請移除規範。
