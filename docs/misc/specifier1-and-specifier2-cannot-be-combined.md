# 無法結合 &#39;&lt;規範1&gt;&#39; 和 &#39;&lt;規範2&gt;&#39;
同時使用兩個不相容的規範 \(例如 `Shadows` 和 `Overrides`\)。  
  
 **錯誤 ID︰**BC31408  
  
### 更正這個錯誤  
  
-   請移除其中一個或另一個規範。
