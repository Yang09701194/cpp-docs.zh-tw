# 在 &#39;Namespace&#39; 陳述式上的規範及屬性無效。
命名空間用於程式區塊和程式設計項目的階層式組織。 它們不採用任何修飾詞或屬性。  
  
 **錯誤識別碼：**BC32026  
  
### 更正這個錯誤  
  
-   移除 `Namespace` 陳述式所有的關鍵字和屬性區塊。
