# 在此陳述式上的規範和屬性無效
無法修改的宣告 \(例如 `Inherits`\) 包含一或多個屬性，或是包含一或多個關鍵字，例如 `Public` 或 `ReadOnly`。 只有可修改的宣告可以包含這類關鍵字或屬性。  
  
 **錯誤 ID︰**BC30193  
  
### 更正這個錯誤  
  
-   從宣告中移除屬性或關鍵字。
