# 結構不可出現 &#39;Inherits&#39; 陳述式
與類別不同的是，結構不支援繼承。  
  
 **錯誤 ID︰**BC30628  
  
### 更正這個錯誤  
  
-   從結構中移除 `Inherits` 陳述式。  
  
-   重新設計您的應用程式，以改用類別。
