# Delegate&#39; 之後必須有 &#39;Sub&#39; 或 &#39;Function&#39;
`Delegate` 陳述式未指定 `Sub` 或 `Function` 程序。`Sub` 或 `Function` 關鍵字必須緊接在`Delegate` 關鍵字之後。  
  
 **錯誤 ID︰**BC30278  
  
### 更正這個錯誤  
  
1.  在 `Delegate` 之後緊接加入 `Sub` 或 `Function` 關鍵字。  
  
2.  適當地指定程序名稱、引數清單和傳回類型。
