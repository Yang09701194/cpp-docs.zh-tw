# Switch &lt;name&gt; specified with conflicting values.
當您在 \[命令\] 視窗輸入了兩次參數，但在每一個執行個體中指定的參數值不一樣時，就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
-   重新輸入命令，但每一個參數只使用一次。
