# Syntax error in pattern.
這個錯誤通常在搜尋字串中包含有二或多個規則運算式，但其中至少有一個規則運算式含有語法錯誤時發生。  
  
### 若要更正這個錯誤  
  
1.  請再一次輸入該字串。  如需語法的詳細資訊，請參閱[規則運算式](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md)。
