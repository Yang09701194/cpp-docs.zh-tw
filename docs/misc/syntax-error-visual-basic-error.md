# 語法錯誤 (Visual Basic 錯誤)
原始程式碼行產生不明確的語法錯誤。  
  
 **錯誤 ID︰**BC30035  
  
### 更正這個錯誤  
  
1.  檢查文件中用於原始程式碼行的每個關鍵字。 將文件中的範例程式碼與產生這個錯誤的原始程式行進行比較。  
  
2.  如果無法指出錯誤的原因，請收集情況的相關資訊，並通知 Microsoft 產品支援服務。
