# System registry could not be accessed.
當作業系統找不到登錄檔、檔案已損毀或試圖讀取或寫入檔案失敗時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請重新安裝 Visual Studio。
