# Tab name &lt;name&gt; already exists.
在工具箱上的每一個索引標籤都必須具有唯一的名稱。  當您為 \[工具箱\] 索引標籤所輸入的名稱與現有 \[工具箱\] 上的索引標籤同名時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請為索引標籤輸入唯一的名稱。
