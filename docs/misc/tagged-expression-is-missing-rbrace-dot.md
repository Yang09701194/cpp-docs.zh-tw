# Tagged expression is missing &#39;}&#39;.
如果尋找字串中已指定左括號 \(`{`\) 但卻省略右括號 \(`}`\) 時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  若要搜尋常值 \(Literal\) 字元 `{`，請使用 `\{`。  
  
2.  若要標記運算式，請輸入遺漏的 `}`。
