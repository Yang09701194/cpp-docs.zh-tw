# Tagged expression number is out of range.
在取代字串中使用標記運算式的參考，但指定的數字超出 0 \- 9 範圍時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請輸入 0 和 9 之間的數字。
