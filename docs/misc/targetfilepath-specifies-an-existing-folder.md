# TargetFilePath 指定現有資料夾
`TargetFilePath` 參數指定現有目錄，但必須指定新目錄的名稱。  
  
### 更正這個錯誤  
  
-   請變更 `TargetFilePath` 來指定新的目錄，而非現有目錄。
