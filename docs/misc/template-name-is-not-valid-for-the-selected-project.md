# Template &lt;name&gt; is not valid for the selected project.
在為 \[**新增檔案**\] 命令 \(`File.NewFile`\) 或 \[**新增方案項目**\] 命令 \(`File.AddNewSolutionItem`\) 輸入未知的範本名稱時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請重新輸入命令和命令的引數。  
  
     \-或\-  
  
2.  查閱「新增檔案命令」或「加入新的方案項目命令」主題中的正確語法。
