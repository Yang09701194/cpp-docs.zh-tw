# TextFieldParser 不支援包含行尾字元的分隔符號
已提供包含行尾字元的分隔符號，但是 `TextFieldParse`r 不支援包含行尾字元的分隔符號。  
  
### 更正這個錯誤  
  
-   請提供正確的分隔符號。
