# The application cannot run.
沒有足夠的記憶體可供 Visual Studio 執行。  
  
### 若要更正這個錯誤  
  
1.  請確認電腦滿足執行 Visual Studio 的最低需求。  
  
2.  關閉其他在電腦中執行的程式，以釋放記憶體。  
  
     \-或\-  
  
     請重新安裝 Visual Studio。
