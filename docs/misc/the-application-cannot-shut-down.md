# The application cannot shut down.
當 Visual Studio 仍有開啟的強制回應對話方塊，而外部程式 \(例如工作管理員\) 卻嘗試關閉 Visual Studio 時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請關閉強制回應對話方塊，然後再試一次。
