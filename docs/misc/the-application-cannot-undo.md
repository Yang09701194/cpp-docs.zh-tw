# The application cannot undo.
當試圖為某一項目 \(含有其他檔案的連結復原\) 執行復原作業時，通常就會發生這個錯誤。  如果在連結修改之後執行任何動作，Visual Studio 將無法復原該變更。  
  
### 若要更正這個錯誤  
  
1.  請手動復原變更。
