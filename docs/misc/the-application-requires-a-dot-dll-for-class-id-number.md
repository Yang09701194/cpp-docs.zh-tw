# The application requires a .dll for class ID &lt;number&gt;.
Visual Studio 找不到指定類別的 DLL。  這個是作業系統錯誤。  
  
### 若要更正這個錯誤  
  
1.  請重新安裝 Visual Studio 或受影響的應用程式，然後再試一次。
