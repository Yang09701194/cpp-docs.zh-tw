# 尚未正確安裝 .NET Framework 偵錯服務。 最可能的原因是未正確登錄 mscordbi.dll。 如需如何修復 .NET Framework 安裝程式的詳細資訊，請按一下 [說明]。
當 Visual Studio 找不到隨 Common Language Runtime 偵錯服務一起安裝的 mscordbi.dll 檔案時，通常就會發生這個錯誤。  
  
### 更正這個錯誤  
  
-   修復電腦的 .NET Framework 安裝。
