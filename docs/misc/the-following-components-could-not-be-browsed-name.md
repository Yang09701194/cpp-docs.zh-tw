# The following components could not be browsed: &lt;name&gt;
在 \[元件選取器\] 對話方塊中試圖加入不支援物件瀏覽的檔案類型 \(例如 .htm 和 .txt 檔\) 時，一般會發生此錯誤。  物件瀏覽器支援 .bsc、.olb、.tlb、.dll、.exe 和 .ocx 等檔案類型。  
  
### 若要更正這個錯誤  
  
1.  請從 \[選取的元件\] 清單中移除不支援的檔案類型，然後選取其他檔案。
