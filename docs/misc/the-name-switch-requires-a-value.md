# The &lt;name&gt; switch requires a value.
當指定的參數沒有要求值時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請重新輸入命令，並提供參數一個值。  
  
     \-或\-  
  
     重新輸入命令，並省略參數。
