# The project file is missing the &#39;subsection&#39; sub-section of the &#39;section&#39; section
子區段 \(Subsection\) 是 Include 而區段 \(Section\) 則是 Files。  
  
 最可能導致這個錯誤的原因是以手動方式編輯專案檔。  
  
 **若要更正這個錯誤**  
  
-   在 \[**顯示所有檔案**\] 模式中工作 \(按一下 \[方案總管\] 工作列中的按鈕\) 並且加入相關檔案。  
  
     如果您發生這個錯誤，專案將會開啟，但沒有檔案會被加入此專案中。
