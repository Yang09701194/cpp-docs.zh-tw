# The project &#39;project&#39; cannot be referenced. &lt;reason&gt;
無法製作特定專案的專案對專案間參考。  
  
 發生此錯誤最常見的原因是參考了建置 .exe 檔的專案。  這個錯誤也表示專案參考了輸出不是組件的專案 \(例如，原生 \(Native\) C\+\+ 應用程式\)。
