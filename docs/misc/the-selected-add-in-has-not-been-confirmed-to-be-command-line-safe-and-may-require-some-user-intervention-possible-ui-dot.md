# The selected Add-In has not been confirmed to be &#39;Command Line Safe&#39;, and may require some user intervention (possible UI).
這個錯誤通常是在增益集已經選取，以便從命令提示字元 \(DOS 提示字元\) 使用時發生，而此增益集不表示增益集本身可安全地在命令列使用。  增益集可能會顯示會防礙命令列使用的 UI。  
  
### 若要更正這個錯誤  
  
1.  在 \[**工具**\] 功能表中，選擇 \[**增益集管理員**\]。  
  
2.  在 \[**增益集管理員**\] 對話方塊中，清除此增益集的 \[**命令列**\]。
