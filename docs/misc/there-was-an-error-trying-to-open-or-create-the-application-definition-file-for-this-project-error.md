# 嘗試開啟或建立此專案的應用程式定義檔案時發生錯誤。&#39;&lt;error&gt;&#39;
當您嘗試開啟無法使用的應用程式定義檔時，就會發生這個錯誤。  它可能已被移動或刪除。  
  
### 若要更正這個錯誤  
  
-   重新建立應用程式定義檔。
