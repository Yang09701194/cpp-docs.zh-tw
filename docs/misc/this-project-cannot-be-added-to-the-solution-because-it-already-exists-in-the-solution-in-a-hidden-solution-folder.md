# This project cannot be added to the solution because it already exists in the solution in a hidden Solution Folder.
方案中的專案只能存在於一個位置。  
  
### 若要更正這個錯誤  
  
1.  若要顯示隱藏的 \[解決方案\] 資料夾，請選取方案或包含隱藏 \[方案\] 資料夾 的 \[方案\] 資料夾，然後在 \[**專案**\] 功能表中選取 \[**解除隱藏資料夾**\]。
