# This tool could not be executed. Please ensure that it points to a valid executable.
當您試圖啟動外部工具但卻無法執行工具時，通常就會發生這個錯誤。  可能會發生這個錯誤，因為指定的檔案並非是有效的可執行檔或批次檔。  
  
### 若要更正這個錯誤  
  
1.  請使用 \[**外部工具**\] 對話方塊 \(可從 \[**工具**\] 功能表中找到\) 來檢查外部工具是否為有效的可執行檔或批次檔。
