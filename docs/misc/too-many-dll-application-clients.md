# 太多 DLL 應用程式用戶端
[!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 的動態連結程式庫 \(DLL\) 只能容納有限數目的主應用程式所存取。 您的應用程式及其他 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 主機應用程式 \(其中有些可能由您的應用程式存取\) 全都同時嘗試存取 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] DLL。  
  
### 更正這個錯誤  
  
-   請減少存取 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 的開放應用程式數目。
