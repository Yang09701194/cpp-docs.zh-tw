# 工具必須有標題。 如果您不要儲存這項工具，請按一下 [外部工具] 對話方塊中的 [刪除]。
這個錯誤通常在 \[外部工具\] 對話方塊中加入新工具，但沒有指定名稱時發生。  
  
### 若要更正這個錯誤  
  
1.  如果您想要儲存該工具，按一下 \[**確定**\] 後，在 \[**外部工具**\] 對話方塊中輸入該工具的標題。  
  
2.  如果您不想儲存該工具，按一下 \[**確定**\] 後，再按一下 \[**外部工具**\] 對話方塊中的 \[**刪除**\]。
