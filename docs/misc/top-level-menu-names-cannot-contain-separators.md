# Top-level menu names cannot contain separators.
當試圖將包含分隔符號字元如虛線 \(\-\) 的文字拖曳到功能表名稱時，一般會發生此錯誤。  最上層的功能表不能包含這個字元，此字元是保留給功能表項目的分隔設定。  
  
### 若要更正這個錯誤  
  
1.  請輸入沒有使用分隔字元的文字。
