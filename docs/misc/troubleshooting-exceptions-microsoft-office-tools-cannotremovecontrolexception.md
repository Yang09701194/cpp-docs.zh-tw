# 疑難排解例外狀況：Microsoft.Office.Tools.CannotRemoveControlException
當嘗試以程式設計的方式，移除不是用程式設計方式加入的控制項時，就會擲回 <xref:Microsoft.Office.Tools.CannotRemoveControlException> 例外狀況。  
  
## 相關秘訣  
 您只能夠以程式設計的方式，移除用程式設計方式加入的控制項。  
 -   您只能夠以程式設計的方式，移除用程式設計方式加入的控制項。 如果是在設計階段而不是執行階段將控制項加入至文件，則無法移除控制項。
