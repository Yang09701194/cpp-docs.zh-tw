# 疑難排解例外狀況：Microsoft.Office.Tools.ControlNameAlreadyExistsException
當您嘗試以程式設計方式，使用與專案中另一個控制項相同的名稱來建立控制項時，就會擲回 <xref:Microsoft.Office.Tools.ControlNameAlreadyExistsException> 例外狀況。  
  
## 相關秘訣  
 **其中一個控制項使用不同的名稱。**  
 控制項名稱必須是唯一的。
