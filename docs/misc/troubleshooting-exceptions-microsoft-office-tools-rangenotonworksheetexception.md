# 疑難排解例外狀況：Microsoft.Office.Tools.RangeNotOnWorksheetException
當您以程式方式建立檢視控制項，其所使用的範圍並非來自對應控制項集合的工作表時，就會擲回 <xref:Microsoft.Office.Tools.RangeNotOnWorksheetException> 例外狀況。  
  
## 相關秘訣  
 請確定指定的範圍對應到正確的工作表。
