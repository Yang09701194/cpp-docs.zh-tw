# 疑難排解例外狀況：Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException
當程式碼嘗試存取文件中不存在的控制項時，就會擲回 <xref:Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException> 例外狀況。 使用者或其他程式碼可能已經移除這個控制項。 在嘗試存取控制項之前，無法檢查它，如果控制項已經移除，方案將無法運作。
