# 疑難排解例外狀況：Microsoft.VisualStudio.Tools.Applications.Runtime.NonCacheableTypeException
嘗試快取的類型如果不是 <xref:System.Data.DataSet>、<xref:System.Data.DataTable>，或符合 [Caching Data](http://msdn.microsoft.com/zh-tw/e60dc731-2cb0-4de6-a861-0f0d131d780a) 中所指定之需求的類型時，所擲回的例外狀況。
