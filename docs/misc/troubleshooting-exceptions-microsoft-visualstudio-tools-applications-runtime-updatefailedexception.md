# 疑難排解例外狀況：Microsoft.VisualStudio.Tools.Applications.Runtime.UpdateFailedException
當部署資訊清單所參考的應用程式資訊清單與文件中應用程式資訊清單的版本不同，且文件中的應用程式資訊清單無法更新時，所擲回的例外狀況。
