# 疑難排解例外狀況：System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException
當管線目錄結構遺漏區段目錄時，就會擲回 <xref:System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException> 例外狀況。  
  
## 備註  
 啟動增益集時，同時會建構通往主應用程式的管線。 增益集系統要求管線區段應置於指定的目錄結構內，並使用指定的目錄名稱，如[管線開發需求](http://msdn.microsoft.com/zh-tw/ef9fa986-e80b-43e1-868b-247f4c1d9da5)中所描述。
