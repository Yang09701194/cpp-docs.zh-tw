# 疑難排解例外狀況：System.ArrayTypeMismatchException
系統無法將元素轉換成為陣列宣告的型別時，就會擲回 <xref:System.ArrayTypeMismatchException> 例外狀況。  
  
## 相關秘訣  
 **請確定該物件類型可轉換為陣列類型。**  
 這兩種型別必須能夠轉換。  
  
 如需詳細資訊，Visual Basic 使用者請參閱 [陣列](../Topic/Arrays%20in%20Visual%20Basic.md)。
