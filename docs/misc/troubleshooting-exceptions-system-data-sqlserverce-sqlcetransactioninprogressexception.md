# 疑難排解例外狀況：System.Data.SqlServerCe.SqlCeTransactionInProgressException
當應用程式嘗試修改正在進行其他交易的資料庫，就會擲回 `System.Data.SqlServerCe.SqlCeTransactionInProgressException` 例外狀況 \(Exception\)。 如需詳細資訊，請參閱 [SQL Server Compact 3.5 類別庫](http://go.microsoft.com/fwlink/?LinkID=102595)。
