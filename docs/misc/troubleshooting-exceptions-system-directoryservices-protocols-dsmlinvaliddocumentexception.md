# 疑難排解例外狀況：System.DirectoryServices.Protocols.DsmlInvalidDocumentException
當 DSML \(目錄服務標記語言\) 要求或回應文件並非語式正確 \(Well\-Formed\) 的 XML，或者無法以 DSML V2 結構描述進行驗證時，所擲回的例外狀況。
