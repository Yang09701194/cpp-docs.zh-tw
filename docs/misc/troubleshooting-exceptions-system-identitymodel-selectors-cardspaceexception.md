# 疑難排解例外狀況：System.IdentityModel.Selectors.CardSpaceException
當 CardSpace 服務層上發生了一個或多個例外狀況 \(Exception\) 時，會擲回 <xref:System.IdentityModel.Selectors.CardSpaceException> 例外狀況。 錯誤的發生原因會列於事件記錄中。  
  
## 備註  
 導致此例外狀況的條件包括在載入存放區或 Ledger 時發生錯誤。
