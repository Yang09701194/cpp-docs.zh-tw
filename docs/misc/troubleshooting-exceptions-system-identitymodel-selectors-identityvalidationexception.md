# 疑難排解例外狀況：System.IdentityModel.Selectors.IdentityValidationException
當收件者憑證無效時，就會擲回 <xref:System.IdentityModel.Selectors.IdentityValidationException> 例外狀況 \(Exception\)。 如果憑證未成功地向上鏈結至根憑證，也會擲回此例外狀況。
