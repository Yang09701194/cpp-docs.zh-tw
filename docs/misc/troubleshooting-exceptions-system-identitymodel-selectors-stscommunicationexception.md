# 疑難排解例外狀況：System.IdentityModel.Selectors.StsCommunicationException
與安全性權杖服務的通訊有問題時，就會擲回 <xref:System.IdentityModel.Selectors.StsCommunicationException> 例外狀況 \(Exception\)。  
  
 這個問題可能是因為下列情況所造成：  
  
-   網路故障  
  
-   無法到達 MEX 端點  
  
-   安全性權杖服務沒有有效的結束點識別 \(Identity\)
