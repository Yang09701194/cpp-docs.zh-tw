# 疑難排解例外狀況：System.IdentityModel.Selectors.UntrustedRecipientException
當使用者檢閱來自要求語彙基元 \(Token\) 之實體 \(Entity\) 的資訊，並決定不信任該實體時，會擲回 <xref:System.IdentityModel.Selectors.UntrustedRecipientException> 例外狀況 \(Exception\)。 使用者要在 CardSpace 使用者介面上按一下 \[**停用 Windows CardSpace**\]。
