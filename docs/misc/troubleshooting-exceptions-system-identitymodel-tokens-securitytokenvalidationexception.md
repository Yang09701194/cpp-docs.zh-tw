# 疑難排解例外狀況：System.IdentityModel.Tokens.SecurityTokenValidationException
當接收的安全性權杖無效時，就會擲回 <xref:System.IdentityModel.Tokens.SecurityTokenValidationException> 例外狀況 \(Exception\)。 這個例外狀況通常是由 <xref:System.IdentityModel.Selectors.SecurityTokenAuthenticator.ValidateToken%2A> 方法擲回。
