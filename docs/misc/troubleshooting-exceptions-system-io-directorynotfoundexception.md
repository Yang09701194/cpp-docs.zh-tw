# 疑難排解例外狀況：System.IO.DirectoryNotFoundException
無法找到部分的檔案路徑或目錄時，就會擲回 <xref:System.IO.DirectoryNotFoundException> 例外狀況。  
  
## 相關秘訣  
 **請確定該目錄出現在指定位置中。**  
 請檢查該目錄存在於指定位置中。  
  
 **使用相對路徑時，請確定目前的目錄正確。**  
 如果您假設目前的路徑不正確，則這個相對路徑也可能不正確。
