# 疑難排解例外狀況：System.MulticastNotSupportedException
嘗試非可組合之委派型別 \(Delegate Type\) 的兩個執行個體組合在一起時，所擲回的例外狀況。 如果其中一個運算元是 Null 參考 \(Visual Basic 中的 `Nothing`\)，就不會擲回例外狀況。
