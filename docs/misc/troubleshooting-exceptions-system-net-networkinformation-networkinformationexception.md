# 疑難排解例外狀況：System.Net.NetworkInformation.NetworkInformationException
擷取網路資訊期間發生錯誤時擲回的例外狀況。  
  
## 備註  
 對 Win32 函式的呼叫失敗時，<xref:System.Net.NetworkInformation> 命名空間中的型別便會擲回這個例外狀況。<xref:System.Net.NetworkInformation.NetworkInformationException.ErrorCode%2A> 屬性包含由函式傳回的結果。
