# 疑難排解例外狀況：System.Printing.PrintCommitAttributesException
當錯誤狀況防止 <xref:System.Printing.PrintCommitAttributesException> 將屬性認可至物件所代表的實際電腦、印表機或裝置時，就會擲回 <xref:System.Printing.PrintSystemObject> 例外狀況 \(Exception\)。
