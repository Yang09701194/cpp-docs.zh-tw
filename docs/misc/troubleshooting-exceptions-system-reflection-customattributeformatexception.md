# 疑難排解例外狀況：System.Reflection.CustomAttributeFormatException
自訂屬性 \(Attribute\) 的二進位格式無效時所擲回的例外狀況。  
  
## 備註  
 這可能是工具中的錯誤所造成的結果，而該工具是用來建立資料型別擲回錯誤。
