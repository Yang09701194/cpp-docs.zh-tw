# 疑難排解例外狀況：System.Runtime.Serialization.InvalidDataContractException
當 <xref:System.Runtime.Serialization.InvalidDataContractException> 或 <xref:System.Runtime.Serialization.DataContractSerializer> 在序列化 \(Serialization\) 和還原序列化 \(Deserialization\) 期間遇到無效的資料合約時，就會擲回 <xref:System.Runtime.Serialization.NetDataContractSerializer> 例外狀況 \(Exception\)。
