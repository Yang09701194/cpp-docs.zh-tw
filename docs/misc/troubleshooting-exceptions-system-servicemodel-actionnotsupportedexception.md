# 疑難排解例外狀況：System.ServiceModel.ActionNotSupportedException
用戶端所叫用之作業的相關動作和伺服器上作業的所有動作都不符時，通常在用戶端會擲回 <xref:System.ServiceModel.ActionNotSupportedException> 例外狀況 \(Exception\)。
