# 疑難排解例外狀況：System.ServiceModel.PoisonMessageException
當訊息被視為是「滯礙」訊息時，便會擲回 <xref:System.ServiceModel.PoisonMessageException> 例外狀況 \(Exception\)。 如果嘗試重複傳送訊息失敗，該訊息便會被視為是滯礙訊息。
