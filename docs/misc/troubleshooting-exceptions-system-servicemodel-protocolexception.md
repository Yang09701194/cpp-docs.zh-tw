# 疑難排解例外狀況：System.ServiceModel.ProtocolException
因為資料傳輸通訊協定不符而無法與遠端的一方通訊時，用戶端就會發生 <xref:System.ServiceModel.ProtocolException> 例外狀況。  
  
## 備註  
 <xref:System.ServiceModel.ProtocolException> 可能是由於端點間不相容的繫結、使用不正確的遠端端點位址，或類似的不正確組態所造成。
