# 疑難排解例外狀況：System.Windows.Automation.ElementNotAvailableException
<xref:System.Windows.Automation.ElementNotAvailableException> 例外狀況 \(Exception\) 包含了例外狀況的相關資訊，該例外狀況是在嘗試存取對應至已無法使用之使用者介面部分的 UI <xref:System.Windows.Automation.AutomationElement> 時所引發。  
  
## 備註  
 如果項目在已關閉的對話方塊內，或已關閉或結束應用程式，則可能會引發例外狀況。
