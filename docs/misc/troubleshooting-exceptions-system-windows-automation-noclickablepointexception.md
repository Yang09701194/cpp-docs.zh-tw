# 疑難排解例外狀況：System.Windows.Automation.NoClickablePointException
<xref:System.Windows.Automation.NoClickablePointException> 例外狀況包含例外狀況的相關資訊，這個例外狀況會在針對沒有可按的點之 UI <xref:System.Windows.Automation.AutomationElement.GetClickablePoint%2A> 呼叫 <xref:System.Windows.Automation.AutomationElement> 時引發。  
  
## 備註  
 當應用程式視窗最小化或項目超出螢幕時，可能會引發此例外狀況。
