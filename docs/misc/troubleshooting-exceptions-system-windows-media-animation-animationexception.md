# 疑難排解例外狀況：System.Windows.Media.Animation.AnimationException
當建立屬性動畫時發生錯誤，便會擲回 <xref:System.Windows.Media.Animation.AnimationException> 例外狀況 \(Exception\)。  
  
## 備註  
 這個類別 \(Class\) 不可用於記時相關的錯誤，例如建立具有負 <xref:System.Windows.Media.Animation.Timeline.Duration%2A> 值或無效 <xref:System.Windows.Media.Animation.Timeline.AccelerationRatio%2A> 和 <xref:System.Windows.Media.Animation.Timeline.DecelerationRatio%2A> 設定的動畫。
