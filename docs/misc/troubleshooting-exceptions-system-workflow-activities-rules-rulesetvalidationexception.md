# 疑難排解例外狀況：System.Workflow.Activities.Rules.RuleSetValidationException
當程序因為無法驗證規則集而無法繼續時，就會擲回 <xref:System.Workflow.Activities.Rules.RuleSetValidationException> 例外狀況 \(Exception\)。 如果使用的 <xref:System.Workflow.Activities.Rules.RuleEngine> 驗證錯誤，則 <xref:System.Workflow.Activities.Rules.RuleSet> 建構函式會擲回例外狀況。
