# 疑難排解例外狀況：System.Workflow.Activities.WorkflowAuthorizationException
當角色驗證因指定的識別 \(Identity\) 未包含在 <xref:System.Workflow.Activities.WorkflowAuthorizationException> 內而失敗時，就會擲回 <xref:System.Workflow.Activities.WorkflowRoleCollection> 例外狀況 \(Exception\)。
