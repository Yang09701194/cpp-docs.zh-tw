# 在模組中的類型無法宣告為 &#39;&lt;規範&gt;&#39;
已在模組內宣告具有無效修飾詞的類型。  
  
 **錯誤 ID︰**BC30735  
  
### 更正這個錯誤  
  
-   請不要對模組內宣告的類型使用 `Protected` 或 `Protected Friend` 修飾詞。
