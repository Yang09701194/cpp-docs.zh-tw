# Type library &lt;name&gt; was not found.
Visual Studio 若沒有 DTE.olb 和 vsext.tlb 型別程式庫，則無法執行。  當已移除或刪除這兩個檔案時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請重新安裝 Visual Studio。
