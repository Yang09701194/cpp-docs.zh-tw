# 引數類型 &#39;&lt;引數名稱&gt;&#39; 為 &#39;&lt;類型名稱&gt;&#39;，不是數值
在必須要有數值引數的情況下，會發生非數值引數。  
  
### 更正這個錯誤  
  
-   請確定引數是數值。
