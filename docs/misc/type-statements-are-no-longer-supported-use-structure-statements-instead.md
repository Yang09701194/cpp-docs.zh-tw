# 不再支援 &#39;Type&#39; 陳述式。請使用 &#39;Structure&#39; 陳述式代替
您無法再使用 `Type` 關鍵字來定義使用者定義類型。  
  
 **錯誤 ID︰**BC30802  
  
### 更正這個錯誤  
  
-   使用 `Structure` 陳述式來定義新的類型。
