# &lt;類型1&gt; &#39;&lt;類型名稱&gt;&#39; 和 &lt;類型2&gt; &#39;&lt;類型名稱&gt;&#39; 在 &lt;命名空間&gt; &#39;&lt;命名空間名稱&gt;&#39; 中發生衝突
兩個宣告的項目 \(例如類別和模組\) 在相同的命名空間中同名。  
  
 **錯誤 ID：**BC30179  
  
### 更正這個錯誤  
  
-   將其中一個項目的宣告移至不同的命名空間。  
  
-   變更其中一個項目的名稱。
