# 在 &lt;名稱空間&gt; &#39;&lt;命名空間名稱&gt;&#39; 中，&lt;類型1&gt; &#39;&lt;類型名稱&gt;&#39; 與 &lt;類型2&gt; &#39;&lt;類型名稱&gt;&#39; (在 &#39;&lt;名稱&gt;&#39; 中宣告) 衝突
兩個宣告的項目 \(例如類別和模組\) 在相同的命名空間中同名。  
  
 **錯誤 ID︰**BC30175  
  
### 更正這個錯誤  
  
-   將其中一個項目的宣告移至不同的命名空間。  
  
-   變更其中一個項目的名稱。
