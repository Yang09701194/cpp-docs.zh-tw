# &#39;&lt;類型名稱&gt;&#39; 必須宣告為 &#39;MustInherit&#39;，因為它包含宣告為 &#39;MustOverride&#39; 的方法
包含 `MustOverride` 成員的類型必須宣告為 `MustInherit`。  
  
 **錯誤識別碼：**BC31411  
  
### 更正這個錯誤  
  
-   在類型中加入 `MustInherit` 修飾詞。
