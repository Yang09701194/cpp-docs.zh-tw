# Unable to customize.
當偵錯應用程式時，嘗試自訂功能表或工具列就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
-   請等到完成偵錯後再自訂工具列或功能表。  
  
     \-或\-  
  
     停止偵錯，然後自訂工具列或功能表。
