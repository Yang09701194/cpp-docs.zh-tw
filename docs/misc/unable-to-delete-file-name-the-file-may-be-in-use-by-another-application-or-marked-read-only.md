# Unable to delete file &lt;name&gt;. The file may be in use by another application or marked read-only.
當選取要刪除的鍵盤配置正在使用中或是唯讀時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  如果另一個應用程式正在使用指定的檔案，請關閉該應用程式。  
  
     \-或\-  
  
     如果檔案是唯讀，請移除唯讀屬性。  您可以修改在 \[屬性\] 對話方塊中的唯讀屬性的檔案，在檔案總管中使用。
