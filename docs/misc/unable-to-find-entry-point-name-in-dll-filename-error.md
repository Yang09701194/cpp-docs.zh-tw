# 找不到 DLL &#39;&lt;檔案名稱&gt;&#39; 中的進入點 &#39;&lt;名稱&gt;&#39;: &lt;錯誤&gt;
在指定的 DLL 中找不到程序名稱。  
  
 **錯誤 ID︰**BC31014  
  
### 更正這個錯誤  
  
1.  確認程序名稱拼字正確，而且大小寫符合 DLL 中函式的大小寫。
