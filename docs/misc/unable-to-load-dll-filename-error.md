# 無法載入 DLL &#39;&lt;檔案名稱&gt;&#39;：&lt;錯誤&gt;
無法載入這個專案所參考的 DLL。  
  
 **錯誤 ID：**BC31013  
  
### 更正這個錯誤  
  
1.  請確認檔案位於指定的位置，且未被其他應用程式鎖定。
