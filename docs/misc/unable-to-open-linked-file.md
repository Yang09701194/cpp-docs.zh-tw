# Unable to Open Linked File
專案中加入一個做為連結的檔案。  但是，當您開啟該專案時，專案系統卻找不到該檔案。  
  
 **若要更正這個錯誤**  
  
-   從 \[方案總管\] 刪除連結的檔案。  
  
     \-或\-  
  
-   將檔案置回專案系統預期尋找的位置。
