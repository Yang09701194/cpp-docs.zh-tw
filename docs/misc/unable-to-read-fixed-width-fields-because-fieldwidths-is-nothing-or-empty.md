# 無法讀取固定寬度的欄位，因為 FieldWidths 為 Nothing 或空白
`TextFieldParser` 無法讀取固定寬度的欄位，因為 `FieldWidths` 屬性設定為 `Nothing` 或空白。  
  
### 更正這個錯誤  
  
-   將 `FieldWidths` 設定為有效值。
