# Unable to read the resource information from the resx file
專案系統無法讀取 .resx 檔案。  
  
 作業系統特定原因 \(如果有的話\) 會被附加到此訊息的結尾。  還會附加上無法讀取的檔案的檔名。  
  
 這個錯誤的一些可能原因，包含缺乏讀取檔案的權限，或是失去與 Web 伺服器的通訊 \(對 Web 專案而言\)。  
  
 這種錯誤將導致建置處理序失敗。
