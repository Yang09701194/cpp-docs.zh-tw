# Unable to remove the last generated output
被視為是最後產生的輸出不會從磁碟和 \(或\) 專案中移除。  
  
 如果自訂工具傳回錯誤，最後產生的輸出 \(如果有的話\) 將從磁碟或專案中移除。  
  
 如果所產生的輸出鎖定在磁碟上就會發生這個錯誤。  
  
 **若要更正這個錯誤**  
  
-   關閉並重新啟動 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]。
