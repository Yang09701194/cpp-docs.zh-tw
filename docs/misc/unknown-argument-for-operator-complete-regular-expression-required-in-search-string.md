# Unknown argument for &#39;:&#39; operator. Complete Regular Expression required in search string.
在搜尋或取代期間於搜尋字串中使用規則運算式或萬用字元時，通常就會發生這個錯誤。  已輸入運算子冒號 \(`:`\) 但後隨的引數不是有效的引數。  
  
> [!NOTE]
>  冒號 \(`:`\) 運算子的引數是有大小寫區分的。  
  
### 若要更正這個錯誤  
  
1.  請查閱「規則運算式」主題中的正確規則運算式語法。  
  
2.  仔細檢查使用正確的大小寫的引數。  
  
3.  如果使用輸入法 \(IME\)，請確認不是使用全形字元來給定引數。
