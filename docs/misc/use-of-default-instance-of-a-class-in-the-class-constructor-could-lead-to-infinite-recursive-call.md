# 在類別建構函式中使用類別的「預設執行個體」，可能會導致無限遞迴呼叫。
類別的預設執行個體已用於類別的建構函式中。 這可能會導致無限遞迴呼叫 \(也稱為無限迴圈\)。  
  
### 更正這個錯誤  
  
-   從類別建構函式中移除預設執行個體。
