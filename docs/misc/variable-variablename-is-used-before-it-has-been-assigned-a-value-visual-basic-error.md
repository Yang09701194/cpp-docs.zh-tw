# 為變數 &#39;&lt;變數名稱&gt;&#39; 指派值之前，已使用該變數 (Visual Basic 錯誤)
為變數指派值之前，已使用該變數，導致 <xref:System.NullReferenceException>  
  
 **錯誤 ID︰**BC42109  
  
### 更正這個錯誤  
  
-   請確定已為變數指派值。
