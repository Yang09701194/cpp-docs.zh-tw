# Visual Basic 編譯器無法從下列錯誤復原: &lt;錯誤&gt;
[!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 編譯器無法從下列錯誤復原: \<`error`\>。  
  
 已停用某些 IDE 功能。  
  
 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 編譯器中發生內部錯誤。  
  
 **錯誤 ID︰**BC31021  
  
### 更正這個錯誤  
  
1.  儲存您的工作，並重新啟動 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]。  
  
2.  如果問題重複發生，請重新安裝 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]。  
  
3.  如果錯誤在重新安裝之後持續發生，請通知 Microsoft 產品支援服務。
