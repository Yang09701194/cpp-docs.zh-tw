# Visual Studio has encountered an unexpected error.
此錯誤通常是在 Visual Studio 接收到非預期的格式或接收到來自非預期來源的資料時發生。  
  
### 若要更正這個錯誤  
  
-   請關閉並且重新啟動 Visual Studio。
