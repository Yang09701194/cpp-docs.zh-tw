# Visual Studio requires Windows 2000 or above.
Visual Studio 需要 Windows NT 4.0 \(組建 1381\) 含 Service Pack 3，或 Windows 95 \(組建 708\) 或更新的版本。  
  
### 若要更正這個錯誤  
  
1.  請將您目前 Windows NT 或 Windows 版本升級到上述所列的版本。
