# VsRegEx 命令列公用程式
在 Visual Studio 2010 和更新版本中，VsRegEx 公用程式已取代為 [CreateExpInstance 公用程式](../Topic/CreateExpInstance%20Utility.md)。  
  
## 備註  
 在 Visual Studio 2008 中，IDE 的實驗執行個體的組態資訊是儲存在系統登錄中。 在 Visual Studio 2010 和更新版本中，則是儲存在檔案系統中。
