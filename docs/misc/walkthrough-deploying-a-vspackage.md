# 逐步解說：部署 VSPackage
從 Visual Studio 2010 開始，您應該使用 VSIX 技術來部署 VSPackage。 如需 VSIX 的詳細資訊，請參閱 [傳送 Visual Studio 擴充功能](../Topic/Shipping%20Visual%20Studio%20Extensions.md)。
