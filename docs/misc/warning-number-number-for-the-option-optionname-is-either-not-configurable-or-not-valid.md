# 選項 &#39;&lt;選項名稱&gt;&#39; 的警告編號 &#39;&lt;編號&gt;&#39; 無法設定或無效
警告 ID 無效。 ID 必須是大於 0 的整數值。  
  
 **錯誤 ID︰**BC2026  
  
### 更正這個錯誤  
  
-   請檢查提供的警告 ID，確認它有效。
