# 警告視為錯誤處理：&lt;error&gt;
開啟 \[警告視為錯誤\] 選項執行組件編譯時，出現警告。  
  
 **錯誤 ID：**BC31072  
  
### 更正這個錯誤  
  
1.  請更正這個警告。  
  
2.  關閉 \[警告視為錯誤\] 選項再編譯組件。
