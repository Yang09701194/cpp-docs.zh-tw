# 不再支援 &#39;Wend&#39; 陳述式；請改用 &#39;End While&#39; 陳述式。
`End While` 建構會取代 `Wend` 關鍵字。  
  
 **錯誤 ID︰**BC30809  
  
### 更正這個錯誤  
  
-   使用 `End While` 結束 `While` 區塊。
