# Window menu can contain &lt;number&gt; items.
視窗功能表中的視窗清單最多只能顯示 24 個視窗項目。  在 \[**\<number\> 個項目顯示在 \[視窗\] 功能表中**\] 方塊 \(在 \[選項\] 對話方塊中 \[環境\] 選項的 \[一般\] 窗格上\) 中輸入小於 1 或大於 24 的數字時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請在方塊中輸入 1 和 24 之間的數字。
