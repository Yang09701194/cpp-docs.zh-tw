# &#39;WithEvents&#39; 變數必須有 &#39;As&#39; 子句
您未提供具有關鍵字 `WithEvents` 的 `As` 子句。 請將變數宣告為可引發事件的特定類別。  
  
 **錯誤 ID︰**BC30412  
  
### 更正這個錯誤  
  
1.  加入指定可引發事件之類別的 `As` 子句。
