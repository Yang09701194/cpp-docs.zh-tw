# Wizard file &lt;name&gt; format is not recognized.
無法開啟精靈。  這個錯誤通常是由於精靈檔損毁或精靈與 Visual Studio 版本號碼不符所致。  
  
### 若要更正這個錯誤  
  
1.  請確認精靈檔的版本號碼與 Visual Studio 版本相同，它即是 VSWIZARD 的值，例如 `VSWIZARD 7.0`。  
  
     \-或\-  
  
     請重新安裝 Visual Studio，以修復損毁的檔案。
