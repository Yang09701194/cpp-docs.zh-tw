# Wizard file &lt;name&gt; is corrupted.
精靈的 .vsz 檔案沒有擁有正確的格式。  當 .vsz 檔案已損毀或該檔來自 Visual Studio 以外的產品時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請確認精靈是 Visual Studio 精靈。  
  
     \-或\-  
  
     請重新安裝 Visual Studio，以取代損毀的檔案。
