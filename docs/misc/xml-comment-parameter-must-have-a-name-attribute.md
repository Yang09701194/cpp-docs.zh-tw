# XML 註解參數必須要有 &#39;name&#39; 屬性
XML 註解參數必須要有 'name' 屬性。 將會忽略 XML 註解。  
  
 XML 標記遺漏必要的 `name` 屬性。  
  
 **錯誤 ID︰**BC42308  
  
### 更正這個錯誤  
  
-   為標記提供必要的 `name` 屬性。
