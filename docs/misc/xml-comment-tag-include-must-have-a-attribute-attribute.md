# XML 註解標記 &#39;include&#39; 必須有 &#39;&lt;屬性&gt;&#39; 屬性
XML 註解標記 'include' 必須有 \<屬性\> 屬性。 將會忽略 XML 註解。  
  
 傳遞給 `<`Include`>` 標記的 `file` 或 `path` 屬性遺漏或不完整。  
  
 **錯誤 ID︰**BC42310  
  
### 更正這個錯誤  
  
-   加入或完成傳遞給 \<Include\> 標記的必要屬性。
