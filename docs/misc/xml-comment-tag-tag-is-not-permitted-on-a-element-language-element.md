# &#39;&lt;項目&gt;&#39; 語言項目上不得有 XML 註解標記 &#39;&lt;標記&gt;&#39;
'\<項目\>' 項目上不得有 XML 註解標記 \<標記\>。 將會忽略 XML 註解。  
  
 語言項目不支援套用它的 XML 註解標記。  
  
 **錯誤 ID︰**BC42306  
  
### 更正這個錯誤  
  
-   請移除不支援的 XML 註解標記。
