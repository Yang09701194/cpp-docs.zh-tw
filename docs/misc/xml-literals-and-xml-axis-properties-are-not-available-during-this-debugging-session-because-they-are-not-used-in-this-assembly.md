# 這個偵錯工作階段中無法使用 XML 常值和 XML 軸屬性，因為它們不用在這個組件中
已在 XML in [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 功能無法使用的偵錯工作階段期間，於 \[監看式\] 或 \[即時運算\] 視窗中參考 XML 常值或 XML 軸屬性。 這是針對未使用 XML in [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 功能或其為發行組建的組件。  
  
 **錯誤 ID︰**BC31196  
  
### 更正這個錯誤  
  
-   請使用組件的偵錯組建來啟動新的偵錯工作階段。
