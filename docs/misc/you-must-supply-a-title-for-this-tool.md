# You must supply a title for this tool.
當 \[**外部工具**\] 對話方塊內的 \[**標題**\] 欄位保留空白時，通常就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  請在 \[**標題**\] 文字方塊中輸入新外部工具的名稱。
