# <a name="concurrencyprecise_math-namespace"></a>Concurrency::precise_math 命名空間

`precise_math` 命名空間中的函式符合 C99 規範。 其中包含每個函式的單一有效位數和雙精確度版本。 例如，`acos` 是雙精確度版本，`acosf` 是單精確度版本。 這些函式（包括單精確度函數）需要加速器的擴充雙精確度支援。 您可以使用[快速鍵：： supports_double_precision](accelerator-class.md#supports_double_precision)來判斷是否可以在特定的加速器上執行這些函數。

## <a name="syntax"></a>語法

```cpp
namespace precise_math;
```
