# <a name="concurrencyprecisemath-namespace"></a>Concurrency::precise_math 命名空間

中的函式`precise_math`命名空間會符合 C99 標準。 單一有效位數，而包含雙精確度的每個函式的版本。 例如，`acos`是雙精度版本和`acosf`是單精確度版本。 這些函式包括單精確度函式中，需要在加速器上的擴充的雙精確度支援。 您可以使用[accelerator:: supports_double_precision](accelerator-class.md#supports_double_precision)來判斷您可以在特定加速器上執行這些函式。

## <a name="syntax"></a>語法

```cpp
namespace precise_math;
```
