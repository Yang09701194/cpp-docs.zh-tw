# <a name="2724-shared"></a>2.7.2.4 shared
這個子句共用變數出現在*變數清單*小組中的所有執行緒之間。 在小組內的所有執行緒都存取相同的儲存區的**共用**變數。  
  
 語法**共用**子句如下所示：  
  
```  
shared(variable-list)  
```
