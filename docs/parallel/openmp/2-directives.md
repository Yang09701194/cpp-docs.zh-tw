# 2. Directives
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

指示詞以 **\# pragma** c 和 C\+\+ 標準中定義的指示詞。  編譯器支援 OpenMP C 和 C\+\+ API 將包含命令列選項，會啟動，並讓所有 OpenMP 編譯器指示詞的解譯。
