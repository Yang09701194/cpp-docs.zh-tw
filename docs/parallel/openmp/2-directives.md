# <a name="2-directives"></a>2.指示詞

指示詞會根據 **#pragma** C 和 c + + 標準中所定義的指示詞。  支援 OpenMP C 和 c + + API 的編譯器會包含命令列選項，就會啟動，並允許所有 OpenMP 編譯器指示詞的解譯。
