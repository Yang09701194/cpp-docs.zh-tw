-   **omp_set_num_threads**函式。  
  
-   **omp_get_num_threads**函式。  
  
-   **omp_get_max_threads**函式。  
  
-   **omp_get_thread_num**函式。  
  
-   **omp_get_num_procs**函式。  
  
-   **omp_in_parallel**函式。  
  
-   **omp_set_dynamic**函式。  
  
-   **omp_get_dynamic**函式。  
  
-   **omp_set_nested**函式。  
  
-   **omp_get_nested**函式。
