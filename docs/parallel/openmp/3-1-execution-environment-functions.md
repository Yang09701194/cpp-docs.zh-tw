# 3.1 Execution Environment Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

這一節所述的函式會影響，並監視執行緒、 處理器和平行的環境：  
  
-   **omp\_set\_num\_threads** 函式。  
  
-   **omp\_get\_num\_threads** 函式。  
  
-   **omp\_get\_max\_threads** 函式。  
  
-   **omp\_get\_thread\_num** 函式。  
  
-   **omp\_get\_num\_procs** 函式。  
  
-   **omp\_in\_parallel** 函式。  
  
-   **omp\_set\_dynamic** 函式。  
  
-   **omp\_get\_dynamic** 函式。  
  
-   **omp\_set\_nested** 函式。  
  
-   **omp\_get\_nested** 函式。
