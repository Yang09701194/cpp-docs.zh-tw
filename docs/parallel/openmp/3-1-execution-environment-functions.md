# <a name="31-execution-environment-functions"></a>3.1 執行環境函式
本節中所述的函式會影響，並監視執行緒、 處理器和平行環境：  
  
-   **omp_set_num_threads**函式。  
  
-   **omp_get_num_threads**函式。  
  
-   **omp_get_max_threads**函式。  
  
-   **omp_get_thread_num**函式。  
  
-   **omp_get_num_procs**函式。  
  
-   **omp_in_parallel**函式。  
  
-   **omp_set_dynamic**函式。  
  
-   **omp_get_dynamic**函式。  
  
-   **omp_set_nested**函式。  
  
-   **omp_get_nested**函式。
