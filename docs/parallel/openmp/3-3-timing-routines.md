# <a name="33-timing-routines"></a>3.3 計時常式
本節中所述的函式支援可攜式時鐘計時器：  
  
-   `omp_get_wtime`函式會傳回耗用的時鐘時間。  
  
-   `omp_get_wtick`函式會傳回後續時鐘刻度之間的秒數。
