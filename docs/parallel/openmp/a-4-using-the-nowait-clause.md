# <a name="a4---using-the-nowait-clause"></a>A.4 使用 nowait 子句
如果有多個獨立的迴圈的平行區域內，您可以使用`nowait`子句 ([區段 2.4.1](../../parallel/openmp/2-4-1-for-construct.md) 11 頁面上) 若要避免隱含的屏障結尾`for`指示詞，如下所示：  
  
```  
#pragma omp parallel  
{  
    #pragma omp for nowait  
        for (i=1; i<n; i++)  
             b[i] = (a[i] + a[i-1]) / 2.0;  
    #pragma omp for nowait  
        for (i=0; i<m; i++)  
            y[i] = sqrt(z[i]);  
}  
```
