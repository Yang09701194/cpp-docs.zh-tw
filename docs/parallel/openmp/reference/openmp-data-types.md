# OpenMP Data Types
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

提供連結至 OpenMP API 所使用的資料型別。  
  
 Visual C\+\+ 實作標準 OpenMP 包括下列資料類型。  
  
|資料型別|描述|  
|----------|--------|  
|[omp\_lock\_t](../../../parallel/openmp/reference/omp-lock-t.md)|保留的鎖定、 鎖定是否可用，或如果執行緒擁有的鎖定狀態的類型。|  
|[omp\_nest\_lock\_t](../../../parallel/openmp/reference/omp-nest-lock-t.md)|保留鎖定的資訊將下列媒體其中一項的類型： 是否鎖定可供使用，以及執行緒的識別所擁有的鎖定及巢狀的計數。|
