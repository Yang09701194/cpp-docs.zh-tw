# auto_rename
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 專有的**  
  
 藉由對變數名稱附加兩個底線 \(\_\_\) 來重新命名 C\+\+ 保留字，以解決可能發生的名稱衝突。  
  
## 語法  
  
```  
auto_rename  
```  
  
## 備註  
 當匯入使用一個或多個 C\+\+ 保留字 \(關鍵字或巨集\) 做為變數名稱的類型程式庫時，會使用這個屬性。  
  
 **END C\+\+ 特定的**
