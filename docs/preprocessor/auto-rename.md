# <a name="auto_rename-import-attribute"></a>auto_rename 匯入屬性

**C++特殊**

藉由對變數名稱附加兩個底線 (__) 來重新命名 C++ 保留字，以解決可能發生的名稱衝突。

## <a name="syntax"></a>語法

> **#import***類型-程式庫***auto_rename**

## <a name="remarks"></a>備註

當匯入使用一個或多個 C++ 保留字 (關鍵字或巨集) 做為變數名稱的類型程式庫時，會使用這個屬性。

**結束C++特定**
