# <a name="auto_search-import-attribute"></a>auto_search 匯入屬性

**C++特殊**

指定當類型程式庫`#import`參考, 且其本身參考另一個類型程式庫時, 編譯器可以針對其他類型程式庫執行隱含`#import`的。

## <a name="syntax"></a>語法

> **#import***類型-程式庫***auto_search**
