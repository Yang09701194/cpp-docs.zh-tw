# <a name="autosearch"></a>auto_search
**C + + 特定的**  
  
 指定時，當類型程式庫是使用 #import 加以參考，而且其本身參考又另一個類型程式庫，則編譯器可以對另一個類型程式庫執行隱含的 #import。  
  
## <a name="syntax"></a>語法  
  
```  
auto_search  
```  
  
## <a name="remarks"></a>備註  
 **END c + + 特定的**
