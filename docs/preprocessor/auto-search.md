# <a name="autosearch"></a>auto_search

**C++ 專有的**

指定，當使用參考的型別程式庫`#import`和本身參考另一個型別程式庫，編譯器可以隱含`#import`其他型別程式庫。

## <a name="syntax"></a>語法

```
auto_search
```

## <a name="remarks"></a>備註

**END C++ 特定的**
