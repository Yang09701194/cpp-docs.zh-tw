# auto_search
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 專有的**  
  
 指定時，當類型程式庫是使用 \#import 加以參考，而且其本身參考又另一個類型程式庫，則編譯器可以對另一個類型程式庫執行隱含的 \#import。  
  
## 語法  
  
```  
auto_search  
```  
  
## 備註  
 **END C\+\+ 特定的**
