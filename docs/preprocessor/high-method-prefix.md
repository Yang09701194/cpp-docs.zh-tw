# <a name="highmethodprefix"></a>high_method_prefix
**C + + 特定的**  
  
 指定用來命名高階屬性和方法的前置詞。  
  
## <a name="syntax"></a>語法  
  
```  
high_method_prefix("Prefix")  
```  
  
#### <a name="parameters"></a>參數  
 `Prefix`  
 要使用的前置詞。  
  
## <a name="remarks"></a>備註  
 根據預設，高階錯誤處理屬性和方法會由名稱不含前置詞的成員函式公開。 其名稱來自於類型程式庫。  
  
 **END c + + 特定的**
