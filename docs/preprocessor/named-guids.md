# named_guids
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 專有的**  
  
 告知編譯器以 **LIBID\_MyLib**、**CLSID\_MyCoClass**、**IID\_MyInterface** 和 **DIID\_MyDispInterface** 形式的舊樣式定義和初始化 GUID 變數。  
  
## 語法  
  
```  
named_guids  
```  
  
## 備註  
 **END C\+\+ 特定的**
