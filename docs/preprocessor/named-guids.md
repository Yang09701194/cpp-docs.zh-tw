# <a name="namedguids"></a>named_guids
**C++ 專有的**

告知編譯器来定義和初始化 GUID 變數中表單的舊樣式`LIBID_MyLib`， `CLSID_MyCoClass`， `IID_MyInterface`，和`DIID_MyDispInterface`。

## <a name="syntax"></a>語法

```
named_guids
```

## <a name="remarks"></a>備註

**END C++ 特定的**
