# <a name="named_guids-import-attribute"></a>named_guids 匯入屬性

**C++特殊**

告訴編譯器以舊樣式定義和初始化 GUID `LIBID_MyLib`變數, 格式為、 `CLSID_MyCoClass`、 `IID_MyInterface`和`DIID_MyDispInterface`。

## <a name="syntax"></a>語法

> **#import***類型-程式庫***named_guids**
