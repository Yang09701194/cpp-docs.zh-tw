# <a name="nosearchnamespace"></a>no_search_namespace
**C + + 特定**

具有相同的功能[no_namespace](../preprocessor/no-namespace.md)屬性，但會使用您所使用的型別程式庫`#import`指示詞搭配[auto_search](../preprocessor/auto-search.md)屬性。

## <a name="syntax"></a>語法

```
no_search_namespace
```
