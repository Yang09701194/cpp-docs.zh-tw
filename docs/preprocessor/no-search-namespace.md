# <a name="nosearchnamespace"></a>no_search_namespace
**C + + 特定的**  
  
 具有相同的功能為[no_namespace](../preprocessor/no-namespace.md)屬性，但適用於使用 #import 指示詞中的型別程式庫[auto_search](../preprocessor/auto-search.md)屬性。  
  
## <a name="syntax"></a>語法  
  
```  
no_search_namespace  
```
