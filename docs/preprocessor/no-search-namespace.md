# no_search_namespace
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 專有的**  
  
 與 [no\_namespace](../preprocessor/no-namespace.md) 屬性具有相同功能，但是會在您使用 \#import 指示詞搭配 [auto\_search](../preprocessor/auto-search.md) 屬性的類型程式庫上使用。  
  
## 語法  
  
```  
no_search_namespace  
```  
  
## 備註  
 **END C\+\+ 特定的**
