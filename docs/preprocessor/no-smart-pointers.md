# <a name="nosmartpointers"></a>no_smart_pointers
**C + + 特定的**  
  
 不為類型程式庫中的所有介面建立智慧型指標。  
  
## <a name="syntax"></a>語法  
  
```  
no_smart_pointers  
```  
  
## <a name="remarks"></a>備註  
 根據預設，當您使用 `#import` 時，會得到類型程式庫中所有介面的智慧型指標宣告。 這些智慧型指標是屬於型別[_com_ptr_t 類別](../cpp/com-ptr-t-class.md)。  
  
 **END c + + 特定的**
