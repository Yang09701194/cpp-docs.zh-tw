# <a name="omp"></a>omp
接受一個或多個 OpenMP 指示詞，以及選擇性的指示詞子句。

## <a name="syntax"></a>語法

```
#pragma omp directive
```

請參閱[OpenMP 指示詞](../parallel/openmp/reference/openmp-directives.md)如需詳細資訊。
