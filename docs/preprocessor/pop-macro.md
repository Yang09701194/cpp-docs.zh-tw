# <a name="popmacro"></a>pop_macro
設定的值*macro_name*這個巨集的堆疊頂端值的巨集。

## <a name="syntax"></a>語法

```
#pragma pop_macro("
macro_name
")
```

## <a name="remarks"></a>備註

您必須先發出[push_macro](../preprocessor/push-macro.md) for *macro_name*您可以執行之前**pop_macro**。

## <a name="example"></a>範例

```cpp
// pragma_directives_pop_macro.cpp
// compile with: /W1
#include <stdio.h>
#define X 1
#define Y 2

int main() {
   printf("%d",X);
   printf("\n%d",Y);
   #define Y 3   // C4005
   #pragma push_macro("Y")
   #pragma push_macro("X")
   printf("\n%d",X);
   #define X 2   // C4005
   printf("\n%d",X);
   #pragma pop_macro("X")
   printf("\n%d",X);
   #pragma pop_macro("Y")
   printf("\n%d",Y);
}
```

```Output
1
2
1
2
1
3
```
