# <a name="pushmacro"></a>push_macro
儲存的值*macro_name*這個巨集的堆疊頂端的巨集。  
  
## <a name="syntax"></a>語法  
  
```  
#pragma push_macro("  
macro_name  
")  
```  
  
## <a name="remarks"></a>備註  
 
您可以擷取的值*macro_name*使用`pop_macro`。  
  
請參閱[pop_macro](../preprocessor/pop-macro.md)如需相關範例。
