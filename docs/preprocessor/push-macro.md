# <a name="push_macro-pragma"></a>push_macro pragma

將*宏名稱*宏的值儲存在這個宏的堆疊頂端。

## <a name="syntax"></a>語法

> **#pragma push_macro ("** _宏名稱_ **")**

## <a name="remarks"></a>備註

您可以使用`pop_macro`來取出*宏名稱*的值。

如需範例, 請參閱[pop_macro](../preprocessor/pop-macro.md) 。
