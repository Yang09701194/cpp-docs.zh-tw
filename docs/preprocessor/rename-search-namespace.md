# <a name="rename_search_namespace-import-attribute"></a>rename_search_namespace 匯入屬性

**C++特殊**

具有與[rename_namespace](../preprocessor/rename-namespace.md)屬性相同的功能, 但會在您搭配使用`#import`指示詞與[auto_search](../preprocessor/auto-search.md)屬性的類型程式庫上使用。

## <a name="syntax"></a>語法

> **#import***類型-程式庫***rename_search_namespace (** "*NewName*" **)**

### <a name="parameters"></a>參數

*NewName*\
命名空間的新名稱。
