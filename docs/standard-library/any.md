# <a name="ltanygt"></a>&lt;any&gt;

定義類別 any 和數個支援的函式和類別。

## <a name="requirements"></a>需求

**標頭:** \<任何 >

**命名空間：** std

## <a name="members"></a>成員

### <a name="functions"></a>函式

|||
|-|-|
|[any_cast](../standard-library/any-functions.md#any_cast)|使物件成為任何。|
|[make_any](../standard-library/any-functions.md#make_any)|接受值並建立任何物件。|
|[swap](../standard-library/any-functions.md#swap)|將兩個物件的元素交換。|

### <a name="classes"></a>類別

|||
|-|-|
|[any](../standard-library/any-class.md)|儲存符合此函式需求的任何類型, 或沒有任何值。|
|[bad_any_cast](../standard-library/bad-any-cast-class.md)|所擲回的物件`any_cast`失敗。|
