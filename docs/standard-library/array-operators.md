# <a name="ltarraygt-operators"></a>&lt;array&gt; 運算子

數位\<>标头包括这些**陣列**非成員比較範本函數。

||||
|-|-|-|
|[操作員!](#op_neq)|[算子&gt;](#op_gt)|[算子&gt;=](#op_gt_eq)|

## <a name="operator"></a><a name="op_neq"></a>操作員!

陣列比較，不相等。

```cpp
template <Ty, std::size_t N>
bool operator!=(
    const array<Ty, N>& left,
    const array<Ty, N>& right);
```

### <a name="parameters"></a>參數

*泰*\
項目的類型。

*N*\
陣列的大小。

*離開*\
要比較的左容器。

*對*\
要比較的右容器。

### <a name="remarks"></a>備註

此範本函式會傳回 `!(left == right)`。

### <a name="example"></a>範例

```cpp
// std__array__operator_ne.cpp
// compile with: /EHsc
#include <array>
#include <iostream>

typedef std::array<int, 4> Myarray;
int main()
    {
    Myarray c0 = {0, 1, 2, 3};

// display contents " 0 1 2 3"
    for (Myarray::const_iterator it = c0.begin();
        it != c0.end(); ++it)
        std::cout << " " << *it;
    std::cout << std::endl;

    Myarray c1 = {4, 5, 6, 7};

// display contents " 4 5 6 7"
    for (Myarray::const_iterator it = c1.begin();
        it != c1.end(); ++it)
        std::cout << " " << *it;
    std::cout << std::endl;

// display results of comparisons
    std::cout << std::boolalpha << " " << (c0 != c0);
    std::cout << std::endl;
    std::cout << std::boolalpha << " " << (c0 != c1);
    std::cout << std::endl;

    return (0);
    }
```

```Output
0 1 2 3
4 5 6 7
false
true
```

## <a name="operatorlt"></a><a name="op_lt"></a>算子&lt;

陣列比較，小於。

```cpp
template <Ty, std::size_t N>
bool operator<(
    const array<Ty, N>& left,
    const array<Ty, N>& right);
```

### <a name="parameters"></a>參數

*泰*\
項目的類型。

*N*\
陣列的大小。

*離開*\
要比較的左容器。

*對*\
要比較的右容器。

### <a name="remarks"></a>備註

範本函數重載`operator<`以比較類範本陣[組 Class](../standard-library/array-class-stl.md)的兩個物件。 函式會傳回 `lexicographical_compare(left.begin(), left.end(), right.begin())`。

### <a name="example"></a>範例

```cpp

// compile with: /EHsc
#include <array>

    {
    Myarray c0 = {0, 1, 2, 3};

    for (Myarray::const_iterator it = c0.begin();
        it != c0.end(); ++it)
        std::cout << " " << *it;
    std::cout << std::endl;

    Myarray c1 = {4, 5, 6, 7};

// display contents " 4 5 6 7"
    for (Myarray::const_iterator it = c1.begin();
        it != c1.end(); ++it)
        std::cout << " " << *it;
    std::cout << std::endl;

// display results of comparisons
    std::cout << std::boolalpha << " " << (c0 < c0);
    std::cout << std::endl;
    std::cout << std::boolalpha << " " << (c0 < c1);
    std::cout << std::endl;

    return (0);
    }
```

```Output
4 5 6 7
false
true
```
