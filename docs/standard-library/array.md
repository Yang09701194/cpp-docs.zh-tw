# <a name="ltarraygt"></a>&lt;array&gt;

定義容器範本類別**陣列**以及數個支援的範本。

## <a name="requirements"></a>需求

**標頭：** \<array>

**命名空間：** std

> [!NOTE]
> \<陣列 > 程式庫也會使用`#include <initializer_list>`陳述式。
