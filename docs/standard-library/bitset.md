# &lt;bitset&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

定義樣板類別位元集，以及兩個代表和操作固定大小位元序列的支援樣板函式。  
  
## 語法  
  
```  
  
#include <bitset>  
  
```  
  
### 運算子  
  
|||  
|-|-|  
|[operator&](../Topic/operator&%20\(%3Cbitset%3E\).md)|在兩個位元集之間執行位元 AND。|  
|[運算子 \<\<](../Topic/operator%3C%3C%20\(%3Cbitset%3E\).md)|將位元序列的文字表示插入標準輸出資料流。|  
|[運算子 \>\>](../Topic/operator%3E%3E%20\(%3Cbitset%3E\).md)|將位元序列的文字表示插入標準輸入資料流。|  
|[operator^](../Topic/operator%5E%20\(%3Cbitset%3E\).md)|在兩個位元集之間執行位元 EXCLUSIVE\-OR。|  
|[operator&#124;](../Topic/operator%7C%20\(%3Cbitset%3E\).md)|在兩個位元集之間執行位元 OR。|  
  
### 類別  
  
|||  
|-|-|  
|[bitset 類別](../standard-library/bitset-class.md)|這個樣板類別所描述的物件類型，可儲存由固定位元數所組成的序列，以提供精簡的方式，來保留一組項目或條件的旗標。|
