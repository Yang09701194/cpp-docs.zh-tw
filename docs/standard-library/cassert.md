# <a name="ltcassertgt"></a>&lt;cassert&gt;

包含標準 C 程式庫標頭 \<assert.h>，並將相關名稱新增至 `std` 命名空間。

## <a name="syntax"></a>語法

```cpp
#include <cassert>

```

## <a name="remarks"></a>備註

包含此標頭可保證，透過使用 Standard C 程式庫標頭中的外部連結所宣告的名稱會在 `std` 命名空間中宣告。
