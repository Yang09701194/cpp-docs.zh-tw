# <a name="ltcerrnogt"></a>&lt;cerrno&gt;

包含標準 C 程式庫標頭\<errno.h> >，並將關聯的名稱加入`std`命名空間。 包含此標頭中宣告的宣告 C 標準程式庫標頭中使用外部連結的名稱可確保`std`命名空間。

## <a name="syntax"></a>語法

```cpp
#include <cerrno>
```
