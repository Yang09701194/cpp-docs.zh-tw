# <a name="chartraitsltchargt-struct"></a>char_traits&lt;char&gt; 結構
結構，其為類型 `char` 之元素的樣板結構 **char_traits\<CharType>** 的特製化。  
  
## <a name="syntax"></a>語法  
  
```
template <>  
struct char_traits<char>;
```  
  
## <a name="remarks"></a>備註  
 特製化可讓結構利用操作此 `char` 類型物件的程式庫函式。
