# char_traits&lt;char&gt; 結構
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

為範本結構 **char\_traits\<CharType\>** 特製化的 `char`型別元素的結構。  
  
## 語法  
  
```  
template<> struct char_traits<char>;  
```  
  
## 備註  
 特製化允許結構利用操作 `char`型別物件的程式庫函式。  
  
## 範例  
 為包含型別 `char`的項目範例參閱樣板類別 [char\_traits 類別](../standard-library/char-traits-struct.md)\> \<**CharType**的 typedef 和成員函式。  
  
## 需求  
 **標頭：**\<string\>  
  
 **命名空間:** std
