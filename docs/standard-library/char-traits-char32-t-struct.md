# char_traits&lt;char32_t&gt; 結構
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

為範本結構 **char\_traits\<CharType\>** 特製化的 `char32_t`型別元素的結構。  
  
## 語法  
  
```  
template<> struct char_traits<char32_t>;  
```  
  
## 備註  
 特製化允許結構利用操作 `char32_t`型別物件的程式庫函式。  
  
## 需求  
 **標頭：**\<string\>  
  
 **命名空間:** std
