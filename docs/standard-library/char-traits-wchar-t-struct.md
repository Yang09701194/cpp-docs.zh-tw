# <a name="chartraitsltwchartgt-struct"></a>char_traits&lt;wchar_t&gt; 結構
類別，其為類型 `wchar_t` 之元素的樣板結構 **char_traits\<CharType>** 的特製化。  
  
## <a name="syntax"></a>語法  
  
```
template <>  
struct char_traits<wchar_t>;
```  
  
## <a name="remarks"></a>備註  
 特製化可讓結構利用操作此 `wchar_t` 類型物件的程式庫函式。  
  
## <a name="requirements"></a>需求  
 **標頭：**\<string>  
  
 **命名空間：** std
