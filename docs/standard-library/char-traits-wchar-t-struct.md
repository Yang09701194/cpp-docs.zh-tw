# <a name="chartraitsltwchartgt-struct"></a>char_traits&lt;wchar_t&gt; 結構

類別是特製化的樣板結構**char_traits\<CharType >** 類型之項目的**wchar_t**。

## <a name="syntax"></a>語法

```cpp
template <>
struct char_traits<wchar_t>;
```

## <a name="remarks"></a>備註

特製化可讓結構利用操作此類型物件的程式庫函式**wchar_t**。

## <a name="requirements"></a>需求

**標頭：**\<string>

**命名空間：** std
