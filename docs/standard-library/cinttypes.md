# <a name="ltcinttypesgt"></a>&lt;cinttypes&gt;
包含標準 C 程式庫標頭 \<inttypes.h>。 包含此標頭也會包含 \<cstdint>。  
  
## <a name="syntax"></a>語法  
  
```  
#include <cinttypes>  
  
```  
  
## <a name="remarks"></a>備註  
 包含此標頭可保證，透過使用 Standard C 程式庫標頭中的外部連結所宣告的名稱會在 `std` 命名空間中宣告。
