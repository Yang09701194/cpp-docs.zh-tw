# <a name="ltciso646gt"></a>&lt;ciso646&gt;

包含標準 C 程式庫標頭 \<iso646.h>，並將相關名稱新增至 `std` 命名空間。

> [!NOTE]
> 使用這個標頭中不需要C++。

## <a name="requirements"></a>需求

**標頭：** \<ciso646 >

**命名空間：** std

## <a name="remarks"></a>備註

包含此標頭可保證，透過使用 Standard C 程式庫標頭中的外部連結所宣告的名稱會在 `std` 命名空間中宣告。
