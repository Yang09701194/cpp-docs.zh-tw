# &lt;cmath&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

包括 Standard C 程式庫標頭 \<math.h\>，並將相關聯的名稱加入至 `std` 命名空間。  
  
## 語法  
  
```  
  
#include <cmath>  
  
```  
  
## 備註  
 包含此標頭可保證，透過使用 Standard C 程式庫標頭中的外部連結所宣告的名稱會在 `std` 命名空間中宣告。
