# <a name="container-classconstiterator"></a>容器類別::const_iterator
> [!NOTE]
>  本主題位於 Visual C++ 文件內，可做為 C++ 標準程式庫中所用容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。  
  
 描述的物件可作為受控制序列的常數迭代器。  
  
## <a name="syntax"></a>語法  
  
```  
 
typedef T6 const_iterator;  
```  
  
## <a name="remarks"></a>備註  
 在此將其描述為未指定之 **T6** 類型的同義字。
