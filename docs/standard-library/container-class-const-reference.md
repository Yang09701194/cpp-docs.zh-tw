# <a name="container-classconst_reference"></a>容器類別::const_reference

> [!NOTE]
> 本主題位於 Microsoft C++檔中，做為C++標準程式庫中所使用之容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。

描述的物件可作為受控制序列之項目的常數參考。

## <a name="syntax"></a>語法

```cpp
typedef T3 const_reference;
```

## <a name="remarks"></a>備註

在此將其描述為未指定類型 `T3` （通常是 `Alloc::const_reference`）的同義字。
