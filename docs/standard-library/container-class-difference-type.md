# <a name="container-classdifferencetype"></a>容器類別::difference_type

> [!NOTE]
> 本主題位於 Microsoft C++檔中, 做為C++標準程式庫中所使用之容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。

描述的物件可代表受控制序列中任兩個元素位址之間的差距。

## <a name="syntax"></a>語法

```

typedef T1 difference_type;
```

## <a name="remarks"></a>備註

在此將其描述為未指定類型`T1`的同義字 (通常`Alloc::difference_type`為)。
