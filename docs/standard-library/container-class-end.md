# <a name="container-classend"></a>容器類別::end

> [!NOTE]
> 本主題位於 Microsoft C++檔中，做為C++標準程式庫中所使用之容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。

傳回指向序列結尾之外的迭代器。

## <a name="syntax"></a>語法

```cpp
const_iterator end() const;

iterator end();
```
