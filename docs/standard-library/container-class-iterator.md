# <a name="container-classiterator"></a>容器類別::iterator

> [!NOTE]
> 本主題位於 Visual C++ 文件內，可做為 C++ 標準程式庫中所用容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。

描述的物件可作為受控制序列的迭代器。

## <a name="syntax"></a>語法

```

typedef T5 iterator;
```

## <a name="remarks"></a>備註

其描述為未指定類型的同義字`T5`。 型別的物件`iterator`型別的物件可以轉型[const_iterator](../standard-library/container-class-const-iterator.md)。
