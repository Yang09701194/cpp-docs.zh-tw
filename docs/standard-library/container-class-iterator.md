# <a name="container-classiterator"></a>容器類別::iterator
> [!NOTE]
>  本主題為 Visual C++ 文件中 C++ 標準程式庫所用容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。  
  
 描述的物件可作為受控制序列的迭代器。  
  
## <a name="syntax"></a>語法  
  
```  
 
typedef T5 iterator;  
```  
  
## <a name="remarks"></a>備註  
 在此將其描述為未指定之 **T5** 類型的同義字。 **iterator** 類型的物件可以轉型為 [const_iterator](../standard-library/container-class-const-iterator.md) 類型的物件。
