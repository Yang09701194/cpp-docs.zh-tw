# 容器類別::iterator
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  本主題將 Visual C\+\+ 文件做為用於 Standard C\+\+ 程式庫的容器的非執行的範例。  如需詳細資訊，請參閱 [STL 容器](../standard-library/stl-containers.md)。  
  
 描述可以當做 Iterator Base 受控制序列的物件。  
  
## 語法  
  
```  
  
typedef T5 iterator;  
  
```  
  
## 備註  
 它會說明此為未指定型別的 **T5**同義字。  型別 **iterator** 物件可以轉換成型別的 [const\_iterator](../standard-library/container-class-const-iterator.md)物件。
