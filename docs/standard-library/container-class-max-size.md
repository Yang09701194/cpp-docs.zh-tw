# <a name="container-classmaxsize"></a>容器類別::max_size
> [!NOTE]
>  本主題為 Visual C++ 文件中 C++ 標準程式庫所用容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。  
  
 傳回物件可以控制之最長序列的長度 (不論受控制序列的長度為何，且為定時的狀態)。  
  
## <a name="syntax"></a>語法  
  
```  
 
size_type max_size() const;
```
