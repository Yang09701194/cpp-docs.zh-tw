# <a name="container-classreverseiterator"></a>容器類別::reverse_iterator

> [!NOTE]
> 本主題位於 Microsoft C++檔中, 做為C++標準程式庫中所使用之容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。

描述的物件可作為受控制序列的反向迭代器。

## <a name="syntax"></a>語法

```

typedef T7 reverse_iterator;
```

## <a name="remarks"></a>備註

在此將其描述為未`T7`指定類型的同義字 (通常`reverse_iterator` **\<** 是[iterator](../standard-library/container-class-iterator.md) **>** )。
