# 容器類別::reverse_iterator
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  本主題將 Visual C\+\+ 文件做為用於 Standard C\+\+ 程式庫的容器的非執行的範例。  如需詳細資訊，請參閱 [STL 容器](../standard-library/stl-containers.md)。  
  
 描述能以反向 Iterator Base 受控制序列的物件。  
  
## 語法  
  
```  
  
typedef T7 reverse_iterator;  
  
```  
  
## 備註  
 它會說明此為未指定型別的 **T7** \(通常是 `reverse_iterator` **\<**[iterator](../standard-library/container-class-iterator.md)**\>**\) 同義字。
