# <a name="container-classsizetype"></a>容器類別::size_type

> [!NOTE]
> 本主題位於 Microsoft C++檔中, 做為C++標準程式庫中所使用之容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。

描述的物件可代表任何受控制序列的長度。

## <a name="syntax"></a>語法

```

typedef T0 size_type;
```

## <a name="remarks"></a>備註

在此將其描述為未指定類型`T0`的同義字 (通常`Alloc::size_type`為)。
