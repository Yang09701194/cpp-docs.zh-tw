# <a name="container-classsize"></a>容器類別::size

> [!NOTE]
> 本主題為 Visual C++ 文件中 C++ 標準程式庫所用容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。

傳回受控制序列的長度 (不論受控制序列的長度為何，且為定時的狀態)。

## <a name="syntax"></a>語法

```

size_type size() const;
```
