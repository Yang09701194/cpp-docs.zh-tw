# <a name="container-classvaluetype"></a>容器類別::value_type
> [!NOTE]
>  本主題位於 Visual C++ 文件內，可做為 C++ 標準程式庫中所用容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。  
  
 作為 **Ty** 範本參數的同義字。  
  
## <a name="syntax"></a>語法  
  
```  
 
typedef T4 value_type;  
```  
  
## <a name="remarks"></a>備註  
 在此將其描述為未指定之 **T4** (通常為 **Alloc::value_type**) 類型的同義字。
