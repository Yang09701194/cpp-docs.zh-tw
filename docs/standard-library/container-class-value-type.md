# <a name="container-classvalue_type"></a>容器類別::value_type

> [!NOTE]
> 本主題位於 Microsoft C++檔中，做為C++標準程式庫中所使用之容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。

作為 *Ty* 樣板參數的同義字。

## <a name="syntax"></a>語法

```cpp
typedef T4 value_type;
```

## <a name="remarks"></a>備註

在此將其描述為未指定類型 `T4` （通常是 `Alloc::value_type`）的同義字。
