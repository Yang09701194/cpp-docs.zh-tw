# <a name="ltcstringgt"></a>&lt;cstring&gt;

包含標準 C 程式庫標頭 \<string.h>，並將關聯名稱加入 `std` 命名空間。

## <a name="syntax"></a>語法

```cpp
#include <cstring>

```

## <a name="remarks"></a>備註

包含此標頭可保證，透過使用 Standard C 程式庫標頭中的外部連結所宣告的名稱會在 `std` 命名空間中宣告。
