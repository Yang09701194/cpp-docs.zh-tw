# <a name="ltctgmathgt"></a>&lt;ctgmath&gt;

實際上，包括 C++ 標準程式庫標頭 \<ccomplex> 和 \<cmath>，提供相當於 \<tgmath.h> 的泛型算術巨集。

## <a name="syntax"></a>語法

```cpp
#include <ctgmath>
```

## <a name="remarks"></a>備註

標準 C 程式庫標頭 \<tgmath.h> 的功能是由 \<ccomplex> 和 \<cmath> 的多載提供。

包含此標頭可保證，透過使用 Standard C 程式庫標頭中的外部連結所宣告的名稱會在 `std` 命名空間中宣告。
