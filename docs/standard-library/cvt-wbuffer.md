# <a name="ltcvtwbuffergt"></a>&lt;cvt/wbuffer&gt;

舊版 Visual Studio 中的 `<cvt/wstring>` 標頭已在 stdext::cvt namespace 中定義範本類別 [wbuffer_convert 類別](../standard-library/wbuffer-convert-class.md)。 為了回溯相容性，會維護此標頭。 新的程式碼應該使用 `std` 命名空間中 [\<locale>](../standard-library/locale.md) 所定義的類別版本。

## <a name="syntax"></a>語法

```cpp
#include <cvt/wbuffer>
```
