# &lt;cvt/wbuffer&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

舊版 Visual Studio 中的標頭 `<cvt/wstring>` 已在 stdext::cvt namespace 定義樣板類別 [wbuffer\_convert 類別](../standard-library/wbuffer-convert-class.md)。  為了回溯相容性，會維護此標頭。  新的程式碼應該使用 `std` 命名空間中 [\<locale\>](../standard-library/locale.md) 所定義的類別版本  
  
## 語法  
  
```  
  
#include <cvt/wbuffer>  
  
```
