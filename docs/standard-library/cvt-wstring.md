# <a name="ltcvtwstringgt"></a>&lt;cvt/wstring&gt;

舊版 Visual Studio 中的標頭 `<cvt/wstring>` 定義了 stdext：： cvt 命名空間中的類別範本[Wstring_convert 類別](../standard-library/wstring-convert-class.md)。 為了回溯相容性，會維護此標頭。 新的程式碼應該使用 `std` 命名空間中 [\<locale>](../standard-library/locale.md) 所定義的類別版本。

## <a name="syntax"></a>語法

```cpp
#include <cvt/wstring>
```
