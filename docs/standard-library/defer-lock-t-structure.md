# <a name="deferlockt-structure"></a>defer_lock_t 結構

表示定義 [defer_lock](../standard-library/mutex-functions.md#defer_lock) 物件的類型，其可用來選取 [unique_lock](../standard-library/unique-lock-class.md) 的其中一個多載建構函式。

## <a name="syntax"></a>語法

```cpp
struct defer_lock_t;
```

## <a name="requirements"></a>需求

**標頭：** \<mutex >

**命名空間：** std
