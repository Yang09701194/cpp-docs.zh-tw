## <a name="swap"></a>  swap

交換兩個 deque 的項目。

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>參數

*left*<br/>
`deque` 類型的物件。

*right*<br/>
`deque` 類型的物件。

### <a name="example"></a>範例

請參閱 [deque::swap](../standard-library/deque-class.md#swap) 範例。
