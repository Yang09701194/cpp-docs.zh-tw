# <a name="ltdequegt-functions"></a>&lt;deque&gt; 函式

## <a name="swap"></a> 交換

交換兩個 deque 的項目。

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>參數

*左邊*\
`deque` 類型的物件。

*權限*\
`deque` 類型的物件。

### <a name="example"></a>範例

請參閱 [deque::swap](../standard-library/deque-class.md#swap) 範例。
