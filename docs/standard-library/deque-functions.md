## <a name="swap"></a>  swap

交換兩個 deque 的項目。

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>參數

`left` 型別的物件`deque`。

`right` 型別的物件`deque`。
