##  <a name="swap"></a>  swap  
 交換兩個 deque 的項目。  
  
```
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```  
  
### <a name="parameters"></a>參數  
 `left`  
 `deque` 類型的物件。  
  
 `right`  
 `deque` 類型的物件。
