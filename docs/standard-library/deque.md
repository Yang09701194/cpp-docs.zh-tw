# &lt;deque&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

定義容器範本類別 deque，以及數個支援的範本。  
  
## 語法  
  
```cpp  
  
#include <deque>  
  
```  
  
### 運算子  
  
|||  
|-|-|  
|[operator\!\=](../Topic/operator!=%20\(%3Cdeque%3E\).md)|測試運算子左邊的 deque 物件是否不等於右邊的 deque 物件。|  
|[運算子 \<](../Topic/operator%3C%20\(%3Cdeque%3E\).md)|測試運算子左邊的 deque 物件是否小於右邊的 deque 物件。|  
|[運算子 \<\=](../Topic/operator%3C=%20\(%3Cdeque%3E\).md)|測試運算子左邊的 deque 物件是否小於或等於右邊的 deque 物件。|  
|[operator\=\=](../Topic/operator==%20\(%3Cdeque%3E\).md)|測試運算子左邊的 deque 物件是否等於右邊的 deque 物件。|  
|[運算子 \>](../Topic/operator%3E%20\(%3Cdeque%3E\).md)|測試運算子左邊的 deque 物件是否大於右邊的 deque 物件。|  
|[運算子 \>\=](../Topic/operator%3E=%20\(%3Cdeque%3E\).md)|測試運算子左邊的 deque 物件是否大於或等於右邊的 deque 物件。|  
  
### 函式  
  
|||  
|-|-|  
|[swap](../Topic/swap%20\(%3Cdeque%3E\).md)|交換兩個 deque 的項目。|  
  
### 類別  
  
|||  
|-|-|  
|[deque 類別](../standard-library/deque-class.md)|序列容器的範本類別，以線性排列方式排列指定類型的項目，並且像向量一樣允許快速隨機存取任何項目，可有效率地在容器背面插入和刪除。|
