# <a name="ltforward_listgt"></a>&lt;forward_list&gt;

定義容器類別範本 forward_list 和數個支援的範本。

## <a name="requirements"></a>需求

**標頭：** \<forward_list>

**命名空間:** std

> [!NOTE]
> @No__t_0forward_list > 程式庫也會使用 `#include <initializer_list>` 語句。
