# <a name="ltinitializerlistgt"></a>&lt;initializer_list&gt;

定義容器範本類別 `initializer_list` 及數個支援範本。

## <a name="syntax"></a>語法

```cpp
#include <initializer_list>
```

### <a name="classes"></a>類別

|類別|描述|
|-|-|
|[initializer_list](../standard-library/initializer-list-class.md)|提供對項目的陣列之存取，其中每個成員都有指定的類型。|
