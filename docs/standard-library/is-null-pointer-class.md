# <a name="isnullpointer-class"></a>is_null_pointer 類別

測試類型是否為 std::nullptr_t。

## <a name="syntax"></a>語法

```cpp
template <class T>
struct is_null_pointer;
```

### <a name="parameters"></a>參數

*T*<br/>
要查詢的類型。

## <a name="remarks"></a>備註

如果型別述詞的執行個體保留 true 的型別*T*是`std::nullptr_t`，否則為 false。

## <a name="requirements"></a>需求

**標頭：**\<type_traits>

**命名空間：** std
