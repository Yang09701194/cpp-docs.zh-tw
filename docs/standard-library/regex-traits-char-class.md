# <a name="regextraitsltchargt-class"></a>regex_traits&lt;char&gt; 類別

regex_traits for char 的特製化。

## <a name="syntax"></a>語法

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>備註

類別是範本類別明確特製化[regex_traits](../standard-library/regex-traits-class.md)類型的項目**char** （使其能夠運用操作此類型物件的程式庫函式）。

## <a name="requirements"></a>需求

**標頭︰**\<regex>

**命名空間：** std
