# <a name="regextraitsltchargt-class"></a>regex_traits&lt;char&gt; 類別

針對 char 的特製化。  `regex_traits`

## <a name="syntax"></a>語法

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>備註

類別是類型為**char**之元素的樣板類別[RegEx_traits](../standard-library/regex-traits-class.md)的明確特製化 (讓它可以利用可操作此類型物件的程式庫函式)。

## <a name="requirements"></a>需求

**標頭︰** \<regex>

**命名空間：** std
