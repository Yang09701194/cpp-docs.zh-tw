# <a name="regex_traitsltchargt-class"></a>regex_traits&lt;char&gt; 類別

針對**char**`regex_traits` 的特製化。

## <a name="syntax"></a>語法

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>備註

類別是類別樣板[RegEx_traits](../standard-library/regex-traits-class.md)的明確特製化，適用于類型為**char**的元素（因此，它可以利用可操作此類型物件的程式庫函式）。

## <a name="requirements"></a>需求

**標頭︰** \<regex>

**命名空間:** std
