# regex_traits&lt;wchar_t&gt; 類別
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

regex\_traits for wchar\_t 的特製化。  
  
## 語法  
  
```  
template <>  
    class regex_traits<wchar_t>  
```  
  
## 備註  
 此類別是針對類型 `wchar_t` 的項目為範本類別 [regex\_traits 類別](../standard-library/regex-traits-class.md) 所做的明確特製化 \(使其能夠運用操作此物件類型的程式庫函式\)。  
  
## 需求  
 **標頭：**\<regex\>  
  
 **命名空間：**std
