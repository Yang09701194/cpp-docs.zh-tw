# <a name="ltsample-containergt"></a>&lt;sample container&gt;

> [!NOTE]
> 本主題位於 Visual C++ 文件內，可做為 C++ 標準程式庫中所用容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。

顯示 C++ 標準程式庫中的容器標頭結構。

## <a name="syntax"></a>語法

```

#include <sample container> // Nonfunctional header
```
