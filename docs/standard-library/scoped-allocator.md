# &lt;scoped_allocator&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

定義容器樣板類別 scoped\_allocator。  
  
## 語法  
  
```cpp  
#include <scoped_allocator>  
```  
  
### 運算子  
  
|||  
|-|-|  
|[operator\!\=](../Topic/operator!=%20Operator%20\(%3Cscoped_allocator%3E\).md)|測試，如果在運算子左方的 scoped\_allocator 物件不等於右邊的清單物件。|  
|[operator\=\=](../Topic/operator==%20Operator%20\(%3Cscoped_allocator%3E\).md)|測試，如果在運算子左方的 scoped\_allocator 物件等於右邊的清單物件。|

|-|-|  
|[scoped\_allocator\_adaptor 類別](../standard-library/scoped-allocator-adaptor-class.md)|封裝一或多個配置器巢的樣板類別。|
