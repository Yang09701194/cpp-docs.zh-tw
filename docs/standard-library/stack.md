# <a name="ltstackgt"></a>&lt;stack&gt;

定義樣板類別堆疊和兩個支援的樣板。

## <a name="requirements"></a>需求

**標頭︰** \<stack>

**命名空間：** std

> [!NOTE]
> \<堆疊 > 程式庫也會使用`#include <initializer_list>`陳述式。
