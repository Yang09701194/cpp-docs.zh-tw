##  <a name="a-namegenericerrnoa--genericerrno"></a><a name="generic_errno"></a>  generic_errno  
 類型，代表針對 `<errno.h>` 中 Posix 所定義的所有錯誤碼巨集提供符號名稱的列舉。  
  
```
typedef errc generic_error;
```  
  
### <a name="remarks"></a>備註  
 此類型與 [errc](../standard-library/system-error-enums.md#errc_enumeration) 同義。
