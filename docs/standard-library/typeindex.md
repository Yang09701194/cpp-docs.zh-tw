# &lt;typeindex&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

包含標準標頭 typeindex \<\> 定義類別和功能支援類別 [type\_info](../cpp/type-info-class.md)物件索引。  
  
## 語法  
  
```cpp  
#include <typeindex>  
```  
  
## 備註  
 [hash 結構](../standard-library/hash-structure.md) 定義套用至對應型別 [type\_index](../standard-library/type-index-class.md) 的值設定為索引值的 `hash function` 。  
  
 `type_index` 類別會封裝指標對協助的 `type_info` 物件在索引。
