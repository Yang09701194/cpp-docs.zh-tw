# <a name="lttypeinfogt"></a>&lt;typeinfo&gt;
包含標準標頭 `<typeinfo>` 以定義與類型識別運算子 [typeid 運算子](../cpp/typeid-operator.md)相關聯的多種類型，該運算子會同時產生靜態和動態類型的資訊。  
  
## <a name="syntax"></a>語法  
  
```  
#include <typeinfo>  
  
```
