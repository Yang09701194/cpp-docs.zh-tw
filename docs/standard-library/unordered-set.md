# &lt;unordered_set&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

定義容器樣板類別 [unordered\_multiset](../standard-library/unordered-multiset-class.md) 和 [unordered\_set](../standard-library/unordered-set-class.md) 及其支援的樣板。  
  
## 語法  
  
```  
#include <unordered_set>  
```  
  
### 類別  
  
|||  
|-|-|  
|[unordered\_multiset 類別](../standard-library/unordered-multiset-class.md)|儲存金鑰雜湊資料表。|  
|[unordered\_set 類別](../standard-library/unordered-set-class.md)|儲存金鑰雜湊資料表。|  
  
### 函式  
  
|||  
|-|-|  
|[operator\!\= \(unordered\_multiset\)](../Topic/operator!=%20\(unordered_multiset\).md)|測試，如果在運算子左方的 unordered\_multiset 物件不等於右邊的 unordered\_multiset 物件。|  
|[operator\=\= \(unordered\_multiset\)](../Topic/operator==%20\(unordered_multiset\).md)|測試，如果在運算子左方的 unordered\_multiset 物件等於右邊的 unordered\_multiset 物件。|  
|[swap 函式 \(unordered\_multiset\)](../Topic/swap%20Function%20\(unordered_multiset\).md)|交換兩個多重集。|  
|[operator\!\= \(unordered\_set\)](../Topic/operator!=%20\(unordered_set\).md)|測試，如果在運算子左方的 unordered\_set 物件不等於右邊的 unordered\_set 物件。|  
|[operator\=\= \(unordered\_set\)](../Topic/operator==%20\(unordered_set\).md)|測試，如果在運算子左方的 unordered\_set 物件等於右邊的 unordered\_set 物件。|  
|[swap 函式 \(unordered\_set\)](../Topic/swap%20Function%20\(unordered_set\).md)|交換兩個集合。|
