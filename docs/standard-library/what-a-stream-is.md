# 什麼是資料流
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

與 C， C\+\+ 沒有內建輸入\/輸出功能。  所有 C\+\+ 編譯器，然而，來與系統，物件導向 I\/O 套件中所提供，稱為 iostream 類別。  資料流是 iostream 類別的核心概念。  您可以將資料流物件做為的來源和目的端的位元組智慧檔案。  自訂的插入和擷取運算子取決於資料流的特性是其類別和。  
  
 透過裝置驅動程式，磁碟作業系統處理鍵盤、螢幕、印表機和通訊埠當做擴充檔。  從類別與這些擴充之檔案互動。  內建類別支援讀取和寫入與語法的記憶體與該磁碟 I\/O 的，可讓您輕鬆地衍生自 Stream 類別。
