# 支援 ANSI
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

雖然整個 MFC 架構已經逐漸向僅支援 Unicode 字元集方向演進，但多數 MFC 類別及方法都還支援 ANSI 字元集。  由於 [!INCLUDE[windowsver](../build/reference/includes/windowsver_md.md)] 和 Windows 通用控制項 6.1 版中的功能持續加強，若干 ANSI 類別及方法的支援都已經被取代。 如需詳細資訊，請參閱[已被取代的 ANSI 應用程式開發介面](../mfc/deprecated-ansi-apis.md)與[Unicode 的支援](../text/support-for-unicode.md)。
