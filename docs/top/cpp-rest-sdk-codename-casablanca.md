# C++ REST SDK (Codename &quot;Casablanca&quot;)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

您可以使用 C\+\+ REST SDK \(Codename "Casablanca"\) 更輕鬆地撰寫現代化的非同步的 C\+\+ 程式碼，以連接雲端架構服務。  
  
 卡薩布蘭卡 \(Casablanca\) 專案不再與 Visual Studio 一起提供。  文件和來源裝載於 CodePlex 上，不定期更新。  如需這個專案的詳細資訊，請參閱 [C\+\+ REST SDK \(Codename "Casablanca"\)](http://casablanca.codeplex.com/)。
