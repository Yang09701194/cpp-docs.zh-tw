### 線上和離線文件  
 您可以檢視 MSDN Library 和 Windows 開發人員中心的線上內容。 此內容經常更新。  
  
 您也可以下載離線說明檢視器中的內容，在本機電腦中檢視。 離線文件按照線上叢書的相關內容整理排列，這也會定期更新。 您可以下載可供下載的有興趣書籍。 如需詳細資訊，請參閱 [Microsoft 說明檢視器](../Topic/Microsoft%20Help%20Viewer.md)。
