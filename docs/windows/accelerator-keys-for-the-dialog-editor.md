# <a name="accelerator-keys-for-the-dialog-editor"></a>對話方塊編輯器的快速鍵
以下是預設值對話方塊編輯器命令的快速鍵。 若要變更快速鍵，請按一下**選項**上**工具**功能表，然後選擇 **鍵盤**下**環境**資料夾。 如需詳細資訊，請參閱[識別及自訂鍵盤快速鍵](/visualstudio/ide/identifying-and-customizing-keyboard-shortcuts-in-visual-studio)。  
  
> [!NOTE]
>  根據您目前使用的設定或版本，您所看到的對話方塊可用選項，以及功能表命令的名稱和位置，可能會與 [說明] 中描述的有所不同。 若要變更設定，請從 [ **工具** ] 功能表中選取 [ **匯入和匯出設定** ]。 如需詳細資訊，請參閱 [在 Visual Studio 中自訂開發設定](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3)  
  
|命令|按鍵|描述|  
|-------------|----------|-----------------|  
|Format.AlignBottoms|按 CTRL + SHIFT + 向下鍵|對齊主控項選取的控制項下緣|  
|Format.AlignCenters|SHIFT + F9|對齊主控項選取的控制項的垂直置中|  
|Format.AlignLefts|CTRL + SHIFT + 向的左鍵|對齊主控項選取的控制項的左的緣|  
|Format.AlignMiddles|F9|對齊主控項選取的控制項的水平置中|  
|Format.AlignRights|CTRL + SHIFT + 向右箭號|對齊主控項的選取控制項的右邊緣|  
|Format.AlignTops|按 CTRL + SHIFT + 向上鍵|將選取的控制項，以主控項的上邊緣對齊|  
|Format.ButtonBottom|CTRL + B|將選取的按鈕，沿著對話方塊底端中央|  
|Format.ButtonRight|CTRL + R|在對話方塊的右上角中將選取的按鈕|  
|Format.CenterHorizontal|CTRL + SHIFT + F9|將控制項在對話方塊內的水平置|  
|Format.CenterVertical|CTRL + F9|在對話方塊內，以垂直方式將控制項置|  
|Format.CheckMnemonics|CTRL + M|助憶鍵的檢查唯一性|  
|Format.SizeToContent|SHIFT + F7|調整大小以配合標題文字已選取的控制項|  
|Format.SpaceAcross|ALT + 向左箭號|均等選取的控制項的水平|  
|Format.SpaceDown|ALT + 向下鍵|均等選取的控制項的垂直|  
|Format.TabOrder|CTRL + D|設定控制項的對話方塊內的順序|  
|Format.TestDialog|CTRL + T|執行測試的外觀和行為對話方塊|  
|Format.ToggleGuides|CTRL + G|循環之間沒有格線、 指導方針，格線編輯對話方塊|  
  
 如需將資源加入至 managed 專案的詳細資訊，請參閱[桌面應用程式中的資源](/dotnet/framework/resources/index)中 *.NET Framework 開發人員手冊 》。* 如需手動將資源檔加入 managed 專案、 存取資源、 顯示靜態資源及指派資源字串給屬性的資訊，請參閱[建立桌面應用程式的資源檔](/dotnet/framework/resources/creating-resource-files-for-desktop-apps)。 全球化和當地語系化的受管理應用程式的資源上的資訊，請參閱[全球化和當地語系化的.NET Framework 應用程式](/dotnet/standard/globalization-localization/index)。
