# <a name="activationfactoryrelease-method"></a>ActivationFactory::Release 方法
遞減參考計數的目前**ActivationFactory**物件。  
  
## <a name="syntax"></a>語法  
  
```cpp  
STDMETHOD_(  
   ULONG,  
   Release  
)();  
```  
  
## <a name="return-value"></a>傳回值  
 若成功，則為 S_OK，否則會是 HRESULT 指出失敗。  
  
## <a name="requirements"></a>需求  
 **標頭：** module.h  
  
 **命名空間：** Microsoft::WRL
