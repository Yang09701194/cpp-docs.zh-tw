# 在對話方塊加入控制項
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### 在對話方塊加入控制項  
  
1.  確定對話方塊索引標籤式視窗是編輯器框架中的目前文件。 如果對話方塊不是目前文件，則不會在 \[工具箱\] 中看到 \[對話方塊編輯器\] 索引標籤。  
  
2.  在 [&#91;工具箱&#93; 視窗](../Topic/Toolbox.md)的 [&#91;對話方塊編輯器&#93; 索引標籤](../mfc/dialog-editor-tab-toolbox.md)上，選取您要的控制項，然後：  
  
    -   在您要放置控制項的位置按一下對話方塊。 控制項會在您按的位置出現。 如需相關資訊，請參閱[加入多個控制項](../mfc/adding-multiple-controls.md)。  
  
         \-或\-  
  
    -   將控制項從 \[工具箱\] 視窗拖放到您的對話方塊位置。 如需詳細資訊，請參閱[加入控制項時調整它的大小](../mfc/sizing-a-control-while-you-add-it.md)。  
  
         \-或\-  
  
    -   按兩下 \[工具箱\] 視窗中的控制項 \(出現在您的對話方塊上\)，然後將控制項重新定位到您要的位置。  
  
 如需 \[工具箱\] 視窗上可用控制項類型的相關資訊，請參閱[工具箱視窗、對話方塊編輯器索引標籤](../mfc/dialog-editor-tab-toolbox.md)。  
  
 如需將資源加入 Managed 專案的相關資訊，請參閱《.NET Framework 開發人員手冊》中的[應用程式中的資源](../Topic/Resources%20in%20Desktop%20Apps.md)。如需手動將資源檔加入 Managed 專案、存取資源、顯示靜態資源及指派資源字串給屬性的相關資訊，請參閱[逐步解說：將 Windows Form 當地語系化](http://msdn.microsoft.com/zh-tw/9a96220d-a19b-4de0-9f48-01e5d82679e5)和[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)。  
  
 需求  
  
 Win32
