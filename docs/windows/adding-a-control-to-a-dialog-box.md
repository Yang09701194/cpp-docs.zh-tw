# <a name="adding-a-control-to-a-dialog-box"></a>在對話方塊加入控制項
### <a name="to-add-a-control-to-a-dialog-box"></a>在對話方塊加入控制項  
  
1.  確定對話方塊索引標籤式視窗是編輯器框架中的目前文件。 如果對話方塊不是目前文件，則不會在 [工具箱]  中看到 [對話方塊編輯器] 索引標籤。  
  
2.  在 [[工具箱] 視窗](../windows/dialog-editor-tab-toolbox.md) 的 [[對話方塊編輯器] 索引標籤](/visualstudio/ide/reference/toolbox)上，選取您要的控制項，然後：  
  
    -   在您要放置控制項的位置按一下對話方塊。 控制項會在您按的位置出現。 如需相關資訊，請參閱 [加入多個控制項](../windows/adding-multiple-controls.md)。  
  
         \-或-  
  
    -   將控制項從 [工具箱] 視窗拖放到您的對話方塊位置。 如需詳細資訊，請參閱 [加入控制項時調整它的大小](../windows/sizing-a-control-while-you-add-it.md)。  
  
         \-或-  
  
    -   按兩下 [工具箱] 視窗中的控制項 (出現在您的對話方塊上)，然後將控制項重新定位到您要的位置。  
  
 如需 [工具箱] 視窗上可用控制項類型的相關資訊，請參閱 [工具箱視窗、對話方塊編輯器索引標籤](../windows/dialog-editor-tab-toolbox.md)。  
  
 如需將資源加入至 managed 專案的詳細資訊，請參閱[桌面應用程式中的資源](/dotnet/framework/resources/index)中 *.NET Framework 開發人員手冊 》。* 如需手動將資源檔加入 managed 專案、 存取資源、 顯示靜態資源及指派資源字串給屬性的資訊，請參閱[建立桌面應用程式的資源檔](/dotnet/framework/resources/creating-resource-files-for-desktop-apps)。 全球化和當地語系化的受管理應用程式的資源上的資訊，請參閱[全球化和當地語系化的.NET Framework 應用程式](/dotnet/standard/globalization-localization/index)。  
  
 需求  
  
 Win32
