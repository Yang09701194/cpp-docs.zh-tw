# <a name="array-attributes"></a>陣列屬性
下列屬性套用至陣列或**SAFEARRAY**s。  
  
|屬性|描述|  
|---------------|-----------------|  
|[library_block](../windows/library-block.md)|將.idl 檔案的程式庫區塊內的建構。|  
|[satype](../windows/satype.md)|指定的 SAFEARRAY 結構的資料類型。|  
|[string](../windows/string-cpp.md)|表示一維`char`， `wchar_t`，**位元組**（或對等） 陣列或這類陣列的指標必須視為字串。|
