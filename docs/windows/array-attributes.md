# <a name="array-attributes"></a>陣列屬性
下列屬性套用至陣列或`SAFEARRAY`s。  
  
|屬性|描述|  
|---------------|-----------------|  
|[library_block](../windows/library-block.md)|會建構的.idl 檔案的程式庫區塊內。|  
|[satype](../windows/satype.md)|指定的資料型別`SAFEARRAY`結構。|  
|[string](../windows/string-cpp.md)|表示一維**char**， **wchar_t**， `byte` （或同等權限） 做為字串，則必須處理陣列或這類陣列的指標。|
