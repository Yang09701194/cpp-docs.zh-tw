# Array Attributes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下列屬性可套用至陣列或**安全陣列時發生**s。  
  
|屬性|描述|  
|--------|--------|  
|[library\_block](../windows/library-block.md)|將.idl 檔案的媒體櫃區塊內的建構函式。|  
|[satype](../windows/satype.md)|指定資料型別安全陣列時發生結構。|  
|[string](../windows/string-cpp.md)|表示一維`char`， `wchar_t`， **位元組** \(或對等鍵\) 陣列或指標以這類陣列必須被視為字串。|
