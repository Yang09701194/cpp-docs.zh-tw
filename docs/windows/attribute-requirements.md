# Attribute Requirements
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C \+ \+ 屬性所列的需求描述的專案類型、 編譯器設定，以及其他資訊所需的屬性才能運作。  如下所述的資訊類別。  
  
> [!NOTE]
>  不支援使用屬性，也會使用屬性的類別衍生的類別。  
  
## Header  
 這個欄位會列出之前可以使用的屬性必須包含的標頭檔。  
  
## 專案  
 這個欄位將告訴您可以使用屬性的專案類型。  
  
## 編譯器  
 這個欄位會提供必須要有使用這個屬性的編譯器選項。
