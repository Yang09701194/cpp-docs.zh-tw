# <a name="attribute-requirements"></a>屬性需求
針對 c + + 屬性列出的需求描述專案類型、 編譯器設定和所需的屬性其他資訊來運作。 如下所述的資訊類別。
  
> [!NOTE]
> 不支援使用衍生自的類別，也會使用屬性的類別上的屬性。
  
## <a name="header"></a>頁首
 此欄位會列出可用屬性之前必須包含的標頭檔。
  
## <a name="project"></a>專案
 此欄位描述的專案類型，可以在其中使用的屬性。
  
## <a name="compiler"></a>編譯器
 此欄位提供的編譯器選項，必須要有使用這個屬性。
