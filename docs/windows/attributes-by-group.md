# Attributes by Group
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C\+\+ 屬性會組織成下列功能群組。  
  
|屬性|描述|  
|--------|--------|  
|[COM 屬性](../windows/com-attributes.md)|插入程式碼支援 COM 開發和 .NET Framework Common Language Runtime 開發許多範圍。|  
|[IDL 屬性](../windows/idl-attributes.md)|啟用 .idl 檔的修改從原始程式碼檔案，而不需使用精靈，而且沒有該檔案結構和語法的知識。|  
|[OLE DB 消費者屬性](../windows/ole-db-consumer-attributes.md)|inject 程式碼根據 [OLE DB 消費者樣板](../data/oledb/ole-db-consumer-templates-reference.md)—建立執行工作 \(例如開啟資料表，來執行命令和存取資料的工作的 OLE DB 消費者。|  
|[編譯器屬性](../windows/compiler-attributes.md)|提供由 Visual C\+\+ 編譯器。|  
  
 如需如何定義您的 .NET Framework 應用程式的自訂屬性的詳細資訊，請參閱 [User\-Defined Attributes](../windows/user-defined-attributes-cpp-component-extensions.md)。
