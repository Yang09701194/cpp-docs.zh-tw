# <a name="attributes-by-usage"></a>依使用方式分類的屬性

本主題列出根據它們所套用的 c + + 語言項目屬性。

如果屬性在之前的 Visual c + + 項目不在屬性的範圍內，屬性區塊會被視為註解。

|屬性|描述|
|---------------|-----------------|
|[模組屬性](../windows/module-attributes.md)|適用於[模組](../windows/module-cpp.md)屬性。|
|[介面屬性](../windows/interface-attributes.md)|適用於[__interface](../cpp/interface.md) c + + 關鍵字。|
|[類別屬性](../windows/class-attributes.md)|適用於 c + + 關鍵字。|
|[方法屬性](../windows/method-attributes.md)|套用至類別、 coclass 或介面中方法。|
|[參數屬性](../windows/parameter-attributes.md)|適用於類別或介面中方法的參數。|
|[資料成員屬性](../windows/data-member-attributes.md)|套用至類別、 coclass 或介面中的資料成員。|
|[Typedef、Enum、Union 和 Struct 屬性](../windows/typedef-enum-union-and-struct-attributes.md)|適用於 c + + 關鍵字。|
|[陣列屬性](../windows/array-attributes.md)|套用至陣列或`SAFEARRAY`s。|
|[獨立屬性](../windows/stand-alone-attributes.md)|操作起來更像是程式碼行，但不是能進行 c + + 關鍵字。 獨立屬性陳述式需要行尾的分號。|
|[自訂屬性](../windows/custom-attributes-cpp.md)|可讓使用者擴充中繼資料。|
