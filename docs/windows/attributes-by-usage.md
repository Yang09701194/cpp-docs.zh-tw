# Attributes by Usage
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

本主題列出屬性，根據它們所套用的 C\+\+ 語言項目。  
  
 如果屬性之前不在屬性的範圍是 Visual C\+\+ 項目，屬性區塊會被視為註解。  
  
|屬性|描述|  
|--------|--------|  
|[模組屬性](../windows/module-attributes.md)|適用於[模組](../windows/module-cpp.md)屬性。|  
|[介面屬性](../windows/interface-attributes.md)|適用於 [\_\_interface](../cpp/interface.md) C\+\+ 關鍵字。|  
|[類別屬性](../windows/class-attributes.md)|適用於 C\+\+ 關鍵字。|  
|[方法屬性](../windows/method-attributes.md)|適用於類別、 coclass 或介面中的方法。|  
|[參數屬性](../windows/parameter-attributes.md)|會套用至類別或介面中方法的參數。|  
|[資料成員屬性](../windows/data-member-attributes.md)|適用於類別、 coclass 或介面中的資料成員。|  
|[Typedef、 列舉、 等位，以及結構屬性](../windows/typedef-enum-union-and-struct-attributes.md)|適用於 C\+\+ 關鍵字。|  
|[陣列的屬性](../windows/array-attributes.md)|會套用至陣列或**安全陣列時發生**s。|  
|[獨立屬性](../windows/stand-alone-attributes.md)|操作更接近程式碼行，但不是能進行 C\+\+ 關鍵字。  獨立屬性陳述式需要分號結尾的行。|  
|[自訂屬性](../windows/custom-attributes-cpp.md)|允許使用者擴充中繼資料。|
