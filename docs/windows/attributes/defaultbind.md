# <a name="defaultbind"></a>defaultbind

表示最能代表物件的單一、 可繫結屬性。

## <a name="syntax"></a>語法

```cpp
[defaultbind]
```

## <a name="remarks"></a>備註

**Defaultbind** c + + 屬性具有相同的功能[defaultbind](/windows/desktop/Midl/defaultbind) MIDL 屬性。

## <a name="example"></a>範例

範例，請參閱[可繫結](bindable.md)如需如何使用的範例**defaultbind**。
