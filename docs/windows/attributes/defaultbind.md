# <a name="defaultbind"></a>defaultbind

表示最能代表物件的單一可系結屬性。

## <a name="syntax"></a>語法

```cpp
[defaultbind]
```

## <a name="remarks"></a>備註

**Defaultbind** C++屬性具有與[defaultbind](/windows/win32/Midl/defaultbind) MIDL 屬性相同的功能。

## <a name="example"></a>範例

如需如何使用**defaultbind**的範例，請參閱可系[結的範例](bindable.md)。
