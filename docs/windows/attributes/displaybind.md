# <a name="displaybind"></a>displaybind

表示應該向使用者顯示為可系結的屬性。

## <a name="syntax"></a>語法

```cpp
[displaybind]
```

## <a name="remarks"></a>備註

**Displaybind** C++屬性具有與[displaybind](/windows/win32/Midl/displaybind) MIDL 屬性相同的功能。

## <a name="example"></a>範例

如需如何使用**displaybind**的範例，請參閱可系[結的範例](bindable.md)。
