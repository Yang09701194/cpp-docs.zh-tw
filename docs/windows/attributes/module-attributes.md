# <a name="module-attributes"></a>模組屬性
下列屬性只能套用至[module](module-cpp.md)屬性。

|屬性|描述|
|---------------|-----------------|
|[helpstringdll](helpstringdll.md)|指定要用來執行檔字串查閱（當地語系化）之 DLL 的名稱。|
