# <a name="propput"></a>propput

指定屬性設定函式。

## <a name="syntax"></a>語法

```cpp

```

## <a name="remarks"></a>備註

**Propput** c + + 屬性具有相同的功能[propput](/windows/desktop/Midl/propput) MIDL 屬性。
