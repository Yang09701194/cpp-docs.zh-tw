# <a name="requestedit"></a>requestedit

表示屬性支援`OnRequestEdit`通知。

## <a name="syntax"></a>語法

```cpp
[requestedit]
```

## <a name="remarks"></a>備註

**Requestedit** c + + 屬性具有相同的功能[requestedit](/windows/desktop/Midl/requestedit) MIDL 屬性。

## <a name="example"></a>範例

範例，請參閱[可繫結](bindable.md)範例使用**requestedit**。
