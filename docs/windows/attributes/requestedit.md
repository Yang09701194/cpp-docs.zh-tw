# <a name="requestedit"></a>requestedit

表示屬性支援 `OnRequestEdit` 通知。

## <a name="syntax"></a>語法

```cpp
[requestedit]
```

## <a name="remarks"></a>備註

**Requestedit** C++屬性具有與[requestedit](/windows/win32/Midl/requestedit) MIDL 屬性相同的功能。

如需**requestedit**的範例用法，請參閱可系[結的範例](bindable.md)。
