# <a name="unique-c"></a>unique (C++)

指定唯一的指標。

## <a name="syntax"></a>語法

```cpp
[unique]
```

## <a name="remarks"></a>備註

**Unique** C++屬性的功能與[唯一](/windows/win32/Midl/unique)的 MIDL 屬性相同。
