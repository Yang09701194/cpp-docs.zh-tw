# <a name="unique-c"></a>unique (C++)

指定唯一的指標。

## <a name="syntax"></a>語法

```cpp
[unique]
```

## <a name="remarks"></a>備註

**唯一**C++屬性具有相同的功能[唯一](/windows/desktop/Midl/unique)MIDL 屬性。
