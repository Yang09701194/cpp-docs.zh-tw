# <a name="boolstructmember-data-member"></a>BoolStruct::Member 資料成員

支援 WRL 結構，而且不是直接從您的程式碼使用。

## <a name="syntax"></a>語法

```cpp
int Member;
```

## <a name="remarks"></a>備註

指定[ComPtr](../windows/comptr-class.md) ，或不是，管理介面的物件存留期。

## <a name="requirements"></a>需求

**標頭：** internal.h

**命名空間：** Microsoft::WRL::Details
