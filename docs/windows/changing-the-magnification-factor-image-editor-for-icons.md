# Changing the Magnification Factor (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

依預設，影像編輯器會在左窗格內以實際大小顯示檢視，並在右窗格內以實際大小的 6 倍顯示檢視。  縮放比例倍數 \(顯示於工作區底端的狀態列\) 是影像的實際大小與顯示大小之間的比率。  預設因數為 6，範圍為 1 到 10。  
  
### 若要縮放比例倍數  
  
1.  選取要變更其縮放比例倍數的 \[影像編輯器\] 窗格。  
  
2.  在[影像編輯器工具列](../mfc/toolbar-image-editor-for-icons.md)上，按一下[縮放比例工具](../mfc/toolbar-image-editor-for-icons.md)右邊的箭頭，然後從子功能表中選取縮放比例倍數：\[1X\]、\[2X\]、\[6X\] 或 \[8X\]。  
  
    > [!NOTE]
    >  若要選取**放大**工具中列出的縮放比例倍數以外的因數，請使用快速鍵。
