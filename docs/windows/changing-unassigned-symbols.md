# Changing Unassigned Symbols
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在 [&#91;資源符號&#93; 對話方塊](../windows/resource-symbols-dialog-box.md)中，您可以編輯或刪除尚未指派給資源或物件的現有符號。  
  
### 變更未指派的符號  
  
1.  在 \[**名稱**\] 中，選取未指派的符號，然後按一下 \[**變更**\]。  
  
2.  在 \[**變更符號**\] 對話方塊提供的方塊中編輯符號的名稱或值。  
  
    > [!NOTE]
    >  若要變更*已*指派給資源或物件的符號，您必須使用資源編輯器或 \[屬性\] 視窗。  如需詳細資訊，請參閱[變更符號或符號名稱](../windows/changing-a-symbol-or-symbol-name-id.md)。  
  
 如需將資源加入至 Managed 專案的詳細資訊，請參閱《.NET Framework 開發人員手冊》中的[應用程式中的資源](../Topic/Resources%20in%20Desktop%20Apps.md)。  
  
 需求  
  
 Win32
