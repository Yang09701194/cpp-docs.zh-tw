# <a name="classfactoryaddref-method"></a>ClassFactory::AddRef 方法
目前的參考計數會遞增**ClassFactory**物件。  
  
## <a name="syntax"></a>語法  
  
```cpp  
STDMETHOD_(  
   ULONG,  
   AddRef  
)();  
```  
  
## <a name="return-value"></a>傳回值  
 若成功，則為 S_OK，否則會是 HRESULT 指出失敗。  
  
## <a name="requirements"></a>需求  
 **標頭：** module.h  
  
 **命名空間：** Microsoft::WRL
