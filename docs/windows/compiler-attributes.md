# <a name="compiler-attributes"></a>編譯器屬性
編譯器屬性會提供各種功能。  
  
|屬性|描述|  
|---------------|-----------------|  
|[emitidl](../windows/emitidl.md)|判斷是否會處理並放在產生的.idl 檔中所有後續的 IDL 屬性。|  
|[event_receiver](../windows/event-receiver.md)|建立事件接收器。|  
|[event_source](../windows/event-source.md)|建立事件來源。|  
|[export](../windows/export.md)|會導致資料結構，以便放入.idl 檔案。|  
|[實作](../windows/implements-cpp.md)|指定強制為成員的 IDL coclass 的分派介面。|  
|[importidl](../windows/importidl.md)|將指定的.idl 檔案插入至產生的.idl 檔案。|  
|[importlib](../windows/importlib.md)|讓已編譯成其他類型程式庫的類型可供正在建立的類型程式庫使用。|  
|[includelib](../windows/includelib-cpp.md)|會導致的.idl 或.h 檔案包含在產生的.idl 檔案。|  
|[library_block](../windows/library-block.md)|將.idl 檔案的程式庫區塊內的建構。|  
|[no_injected_text](../windows/no-injected-text.md)|防止編譯器插入程式碼，因為屬性使用。|  
|[satype](../windows/satype.md)|指定的資料型別**SAFEARRAY**。|  
|[version](../windows/version-cpp.md)|識別多個版本之間的介面或類別的特定版本。|
