# <a name="comptrdetach-method"></a>ComPtr::Detach 方法
解除這**ComPtr**從它所代表之介面的物件。  
  
## <a name="syntax"></a>語法  
  
```  
T* Detach();  
```  
  
## <a name="return-value"></a>傳回值  
 這由介面的指標**ComPtr**物件。  
  
## <a name="requirements"></a>需求  
 **標頭：** client.h  
  
 **命名空間：** Microsoft::WRL
