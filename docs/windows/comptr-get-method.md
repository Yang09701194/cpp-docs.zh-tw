# <a name="comptrget-method"></a>ComPtr::Get 方法

## <a name="syntax"></a>語法  
  
```  
T* Get() const;  
```  
  
## <a name="return-value"></a>傳回值  
 與這個 ComPtr 相關聯的介面指標。

 **命名空間：** Microsoft::WRL
