# <a name="comptrinternalrelease-method"></a>ComPtr::InternalRelease 方法
在與這個 ComPtr 相關聯的介面上，執行 COM 釋放作業。  
  
## <a name="syntax"></a>語法  
  
```  
void InternalRelease();  
```  
  
## <a name="remarks"></a>備註  
 這個方法會受到保護。  
  
## <a name="requirements"></a>需求  
 **標頭：** client.h  
  
 **命名空間：** Microsoft::WRL
