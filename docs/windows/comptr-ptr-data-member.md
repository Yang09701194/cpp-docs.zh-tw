# <a name="comptrptr-data-member"></a>ComPtr::ptr_ 資料成員
包含與這個 ComPtr 相關聯並受其管理之介面的指標。  
  
## <a name="syntax"></a>語法  
  
```  
InterfaceType *ptr_;  
```  
  
## <a name="remarks"></a>備註  
 `ptr_` 是內部、 受保護的資料成員。  
  
## <a name="requirements"></a>需求  
 **標頭：** client.h  
  
 **命名空間：** Microsoft::WRL
