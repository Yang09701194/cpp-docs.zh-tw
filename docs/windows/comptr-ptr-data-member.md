# <a name="comptrptr-data-member"></a>ComPtr::ptr_ 資料成員

包含相關聯，並管理由這個介面的指標**ComPtr**。

## <a name="syntax"></a>語法

```cpp
InterfaceType *ptr_;
```

## <a name="remarks"></a>備註

**ptr_** 是內部、 受保護的資料成員。

## <a name="requirements"></a>需求

**標頭：** client.h

**命名空間：** Microsoft::WRL
