# ComPtr::ptr_ 資料成員
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

含有指向關聯，並由這個 ComPtr 管理的介面。  
  
## 語法  
  
```  
InterfaceType *ptr_;  
```  
  
## 備註  
 `ptr_` 是一個內部，受保護的資料成員。  
  
## 需求  
 **標題:** client.h  
  
 **命名空間：**Microsoft::WRL
