# <a name="comptrswap-method"></a>ComPtr::Swap 方法
將受目前 ComPtr 管理的介面，與受指定 ComPtr 管理的介面互相交換。  
  
## <a name="syntax"></a>語法  
  
```  
void Swap(  
   _Inout_ ComPtr&& r  
);  
  
void Swap(  
   _Inout_ ComPtr& r  
);  
```  
  
#### <a name="parameters"></a>參數  
 `r`  
 ComPtr。  
  
## <a name="requirements"></a>需求  
 **標頭：** client.h  
  
 **命名空間：** Microsoft::WRL
