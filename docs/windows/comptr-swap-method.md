# <a name="comptrswap-method"></a>ComPtr::Swap 方法

交換目前所管理之介面**ComPtr**與管理指定之介面**ComPtr**。

## <a name="syntax"></a>語法

```cpp
void Swap(
   _Inout_ ComPtr&& r
);

void Swap(
   _Inout_ ComPtr& r
);
```

### <a name="parameters"></a>參數

*r*  
A **ComPtr**。

## <a name="requirements"></a>需求

**標頭：** client.h

**命名空間：** Microsoft::WRL
