# <a name="comptrcomptr-destructor"></a>ComPtr::~ComPtr 解構函式
取消初始化 ComPtr 的執行個體。  
  
## <a name="syntax"></a>語法  
  
```  
WRL_NOTHROW ~ComPtr();  
```  
  
## <a name="requirements"></a>需求  
 **標頭：** client.h  
  
 **命名空間：** Microsoft::WRL
