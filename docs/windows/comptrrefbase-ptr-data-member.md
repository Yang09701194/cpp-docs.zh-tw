# ComPtrRefBase::ptr_ 資料成員
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支援 WRL 基礎結構，而且不是為了要直接從您的程式碼中使用而設計。  
  
## 語法  
  
```  
T* ptr_;  
```  
  
## 備註  
 對目前樣板參數所指定型別的指標。  
  
## 備註  
 `ptr_` 是受保護的資料成員。  
  
## 需求  
 **標題:** client.h  
  
 **命名空間：** Microsoft::WRL::Details
