# Controls in Dialog Boxes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

您可以使用[工具箱視窗](../Topic/Toolbox.md)中的[對話方塊編輯器索引標籤](../mfc/dialog-editor-tab-toolbox.md)將控制項加入對話方塊，讓您選取您要的控制項並且將之拖曳到對話方塊。  根據預設，工具箱視窗會設為自動隱藏。  當對話方塊編輯器開啟時，它會以您的解決方案的左邊界上的索引標籤出現。  然而，您可以藉著按一下視窗右上角中的 \[自動隱藏\] 按鈕來固定工具箱視窗。  如需如何控制這個視窗的行為的詳細資訊，請參閱[視窗管理](../Topic/Customizing%20window%20layouts%20in%20Visual%20Studio.md)。  
  
 將控制項加入至對話方塊、重新調整現有控制項的位置，或將控制項從一個對話方塊移往另一個對話方塊的最快方法是使用拖放方法。  直到它放入對話方塊前，控制項的位置以虛線表示。  當您以拖放方法將控制項加入對話方塊時，控制項會使用適合於此控制項型別的標準列高。  
  
 當您將控制項加入對話方塊或重新調整位置時，它的最後位置是由輔助線或邊界決定，或者您是否已啟動配置格線。  
  
 一旦已將控制項加入至對話方塊，您可以變更屬性，例如，[屬性視窗](../Topic/Properties%20Window.md)中的標題。  您可以一次選取多個控制項並且變更它們的屬性。
