# <a name="creating-a-console-application"></a>建立主控台應用程式
建立 Win32 主控台應用程式的最簡單方式是使用 Win32 應用程式精靈。  
  
### <a name="to-create-a-win32-console-application"></a>建立 Win32 主控台應用程式  
  
1.  請依照下列說明主題中的指示[使用 Visual c + + 應用程式精靈建立專案](../ide/creating-desktop-projects-by-using-application-wizards.md)。  
  
2.  在**新專案**對話方塊中，選取**Win32 主控台專案**開啟精靈的 [範本] 窗格中。  
  
3.  定義您[應用程式設定](../windows/application-settings-win-32-project-wizard.md)使用[Win32 應用程式精靈](../windows/win32-application-wizard.md)。  
  
    > [!NOTE]
    >  若要保留精靈的預設值，請略過此步驟。  
  
4.  按一下**完成**在關閉精靈，且您新建立的專案開啟**方案總管 中**。
