# 建立主控台應用程式
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

建立 Win32 主控台應用程式最容易的方式是使用 Win32 應用程式精靈。  
  
### 若要建立 W32 主控台應用程式 \(Console Application\)  
  
1.  請依照說明主題[使用 Visual C\+\+ 應用程式精靈建立專案](../ide/creating-desktop-projects-by-using-application-wizards.md)中的指示操作。  
  
2.  在 \[**新增專案**\] 對話方塊中，選取 \[範本\] 窗格中的 \[**Win32 主控台專案**\] 以開啟精靈。  
  
3.  使用 [Win32 應用程式精靈](../windows/win32-application-wizard.md)來定義[應用程式設定](../windows/application-settings-win-32-project-wizard.md)。  
  
    > [!NOTE]
    >  若要保留精靈的預設值，請略過此步驟。  
  
4.  按一下 \[**完成**\] 以關閉精靈，而新建立的專案會在 \[**方案總管**\] 中開啟。
