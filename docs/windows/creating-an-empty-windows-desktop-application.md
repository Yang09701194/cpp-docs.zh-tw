# <a name="creating-an-empty-windows-desktop-application"></a>建立空的 Windows 桌面應用程式
### <a name="to-create-an-empty-windows-desktop-application"></a>建立空的 Windows 桌面應用程式  
  
1.  從 [ **檔案** ] 功能表選取 [ **新增]、[專案**]。  
  
2.  在 [ **新增專案** ] 對話方塊的左窗格中，按一下 [ **Win32** ]，並在中間窗格中選取 [ **Win32 主控台應用程式**]。  
  
3.  輸入新的專案名稱、專案目錄的路徑，然後按一下 [確定] 。  
  
4.  在 [ [Win32 應用程式精靈](../windows/win32-application-wizard.md)] 中，按一下 [應用程式設定]  頁面。 選取您的原始程式碼檔所要建立的 **應用程式類型** ，然後選取 [其他選項]  下的 [空專案] 核取方塊。  
  
5.  按一下 [確定 **Deploying Office Solutions**]。  
  
 此專案會出現在 **方案總管** 中，並會顯示三個目錄，分別包含原始程式檔案、標頭檔案與資源檔案。  
  
 接下來您可以 [將檔案加入空白的 Visual C++ 專案中](../windows/adding-files-to-an-empty-win32-applications.md)。
