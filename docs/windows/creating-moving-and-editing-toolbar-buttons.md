# <a name="creating-moving-and-editing-toolbar-buttons"></a>建立、移動和編輯工具列按鈕
您可以輕鬆地建立、 移動、 複製和編輯工具列按鈕。  
  
 根據預設，新的或空白的按鈕會顯示工具列的右邊。 您可以移動此按鈕，然後再編輯它。 當您建立新的按鈕時，就會顯示另一個的空白按鈕右邊的 [編輯] 按鈕。 當您儲存工具列時，將不會儲存空白的按鈕。  
  
-   [建立新的工具列按鈕](creating-a-new-toolbar-button.md)  
  
-   [移動工具列按鈕](moving-a-toolbar-button.md)  
  
-   [從工具列資源複製按鈕](../windows/copying-buttons-from-a-toolbar.md)  
  
-   [刪除工具列按鈕](deleting-a-toolbar-button.md)  
  
-   [在工具列資源的按鈕之間插入空格](../windows/inserting-a-space-between-buttons-on-a-toolbar.md)  
  
-   [工具列按鈕之間的空格的特寫](../windows/removing-space-between-buttons-on-a-toolbar.md)
