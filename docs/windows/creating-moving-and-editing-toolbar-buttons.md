# Creating, Moving, and Editing Toolbar Buttons
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

您可以輕鬆地建立、移動、複製與編輯工具列按鈕。  
  
 在預設狀態下，新的或空白的按鈕將顯示在工具列右端。  在進行編輯前可先移動此按鈕。  建立新的按鈕時，另一個空白的按鈕會出現在編輯過的按鈕右側。  儲存工具列時，並不會儲存該空白按鈕。  
  
-   [建立新的工具列按鈕](../mfc/creating-a-new-toolbar-button.md)  
  
-   [移動工具列按鈕](../mfc/moving-a-toolbar-button.md)  
  
-   [從工具列資源複製按鈕](../mfc/copying-buttons-from-a-toolbar.md)  
  
-   [刪除工具列按鈕](../mfc/deleting-a-toolbar-button.md)  
  
-   [在工具列資源的按鈕之間插入空白](../mfc/inserting-a-space-between-buttons-on-a-toolbar.md)  
  
-   [去除工具列按鈕之間的空白](../mfc/removing-space-between-buttons-on-a-toolbar.md)  
  
-   [編輯工具列的屬性頁](../mfc/changing-the-properties-of-a-toolbar-button.md)  
  
 如需將資源加入至 Managed 專案的詳細資訊，請參閱《.NET Framework 開發人員手冊》中的[應用程式中的資源](../Topic/Resources%20in%20Desktop%20Apps.md)。 如需手動將資源加入至 Managed 專案、存取資源、顯示靜態資源和指定屬性的資源字串等詳細資訊，請參閱[逐步解說：將 Windows Form 當地語系化](http://msdn.microsoft.com/zh-tw/9a96220d-a19b-4de0-9f48-01e5d82679e5)和[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)。
