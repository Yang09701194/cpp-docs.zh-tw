# <a name="creatormapfactorycache-data-member"></a>CreatorMap::factoryCache 資料成員
支援 WRL 結構，而且不是直接從您的程式碼使用。  
  
## <a name="syntax"></a>語法  
  
```  
FactoryCache* factoryCache;  
```  
  
## <a name="remarks"></a>備註  
 儲存的處理站快取指標**CreatorMap**。  
  
## <a name="requirements"></a>需求  
 **標頭：** module.h  
  
 **命名空間：** Microsoft::WRL::Details
