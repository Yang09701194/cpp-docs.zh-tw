# <a name="creatormapservername-data-member"></a>CreatorMap::serverName 資料成員
支援 WRL 基礎結構，並不是直接從您的程式碼使用。  
  
## <a name="syntax"></a>語法  
  
```  
const wchar_t* serverName;  
```  
  
## <a name="remarks"></a>備註  
 儲存 CreatorMap 的伺服器名稱。  
  
## <a name="requirements"></a>需求  
 **標頭：** module.h  
  
 **命名空間：** Microsoft::WRL::Details
