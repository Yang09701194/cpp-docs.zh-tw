# CreatorMap::serverName 資料成員
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支援 WRL 基礎結構，而且不是為了要直接從您的程式碼中使用而設計。  
  
## 語法  
  
```  
const wchar_t* serverName;  
```  
  
## 備註  
 儲存 CreatorMap 的伺服器名稱。  
  
## 需求  
 **標題:** module.h  
  
 **命名空間：** Microsoft::WRL::Details
