# CriticalSection::IsValid 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

指出目前關鍵區段是否有效。  
  
## 語法  
  
```  
bool IsValid() const;  
```  
  
## 傳回值  
 根據預設，一定會傳回 `true`。  
  
## 需求  
 **標題:** corewrappers.h  
  
 **命名空間：**Microsoft::WRL::Wrappers
