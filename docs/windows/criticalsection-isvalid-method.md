# <a name="criticalsectionisvalid-method"></a>CriticalSection::IsValid 方法
指出目前的重要區段是否有效。  
  
## <a name="syntax"></a>語法  
  
```cpp  
bool IsValid() const;  
```  
  
## <a name="return-value"></a>傳回值  
 根據預設，一律會傳回 **，則為 true**。  
  
## <a name="requirements"></a>需求  
 **標頭：** corewrappers.h  
  
 **命名空間：** Microsoft::WRL::Wrappers
