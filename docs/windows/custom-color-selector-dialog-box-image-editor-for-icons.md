# 自訂色彩選取器對話方塊 (圖示影像編輯器)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

這個對話方塊可讓您自訂用於影像的色彩。  如需詳細資訊，請參閱[自訂或變更色彩](../windows/customizing-or-changing-colors-image-editor-for-icons.md)。  
  
 **漸層色彩顯示**  
 變更所選取色彩的值。  將十字型游標置於您要變更的色彩上。  然後將滑桿上移或下移來變更色彩的明暗度或 RGB 值。  
  
 **亮度列**  
 設定漸層色彩顯示方塊內選取之色彩的明暗度。  按一下並向上 \(亮度升高\) 或向下 \(亮度降低\) 拖曳列上的白色箭頭。  請注意，\[色彩\] 方塊會顯示您所選取的色彩以及您設定之明暗度的影像。  
  
 **色彩**  
 列出您正在定義之色彩的色調 \(色彩輪轉值\)。  值範圍從 0 到 240，其中 0 是紅色，60 是黃色，120 是綠色，180 是青色，200 是洋紅色，240 是藍色。  
  
 **色調**  
 列出您正在定義之色彩的色調 \(色彩輪轉值\)。  值範圍從 0 到 240，其中 0 是紅色，60 是黃色，120 是綠色，180 是青色，200 是洋紅色，240 是藍色。  
  
 **彩度**  
 指定您正在定義之色彩的飽和度值。  飽和度是在指定色調中的色彩量。  值範圍從 0 到 240。  
  
 **亮度**  
 列出您正在定義之色彩的明暗度 \(亮度\)。  值範圍從 0 到 240。  
  
 **紅色**  
 指定您正在定義之色彩的紅色值。  值範圍從 0 到 255。  
  
 **綠色**  
 指定您正在定義之色彩的綠色值。  值範圍從 0 到 255。  
  
 **藍色**  
 指定您正在定義之色彩的藍色值。  值範圍從 0 到 255。
