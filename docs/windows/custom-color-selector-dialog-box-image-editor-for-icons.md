# <a name="custom-color-selector-dialog-box-image-editor-for-icons"></a>自訂色彩選取器對話方塊 (圖示影像編輯器)
此對話方塊可讓您自訂您的映像您使用的色彩。 如需詳細資訊，請參閱[自訂或變更色彩](../windows/customizing-or-changing-colors-image-editor-for-icons.md)。  
  
 **漸層色彩顯示**  
 變更選取的色彩值。 您想要變更位置十字形狀的色彩。 滑桿上移或下移來變更的明暗度或色彩的 RGB 值。  
  
 **亮度列**  
 設定您在漸層色彩顯示方塊中選取之色彩的亮度。 按一下並拖曳白色箭號，為更高亮度列向上或向下的鍵小於。 請注意，[色彩] 方塊會顯示所選取的色彩和您所設定的明暗度的影響。  
  
 **色彩**  
 列出您正在定義之色彩的色調 （色彩滾輪值）。 值範圍從 0 到 240，其中 0 是紅色、 60 是黃色，120 為綠色、 180 是青色、 200 是洋紅、 240 是藍色。  
  
 **色調**  
 列出您正在定義之色彩的色調 （色彩滾輪值）。 值範圍從 0 到 240，其中 0 是紅色、 60 是黃色，120 為綠色、 180 是青色、 200 是洋紅、 240 是藍色。  
  
 **星期六**  
 指定正在定義之色彩的濃度值。 飽和度是數量中指定的色調色彩。 值範圍從 0 到 240。  
  
 **[亮度]**  
 列出您正在定義之色彩的亮度 （亮度）。 值範圍從 0 到 240。  
  
 **紅色**  
 指定正在定義之色彩的紅色值。 值範圍從 0 到 255 之間。  
  
 **綠色**  
 指定正在定義之色彩的綠色的值。 值範圍從 0 到 255 之間。  
  
 **藍色**  
 指定正在定義之色彩的藍色值。 值範圍從 0 到 255 之間。
