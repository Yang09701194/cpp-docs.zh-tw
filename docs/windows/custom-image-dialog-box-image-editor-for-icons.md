# <a name="custom-image-dialog-box-image-editor-for-icons"></a>自訂影像對話方塊 (圖示影像編輯器)
可讓您建立新的映像的自訂大小與色彩數目。
 **寬度**  
 提供空間讓您自訂影像的寬度輸入像素為單位 （1 – 512，限制為 2048年）。  
  
 **高度**  
 提供空間讓您輸入高度，單位為像素 （1 – 512，限制為 2048年） 自訂映像。  
  
 **色彩**  
 提供空間讓您選擇的自訂映像的色彩數目： 2、 16 或 256。  
  
## <a name="requirements"></a>需求  
 無
