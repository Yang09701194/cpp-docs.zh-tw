# 自訂影像對話方塊 (圖示影像編輯器)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

可讓您以自訂的大小和色彩數建立新的影像。  如需 Windows 所使用之圖示大小的詳細資訊，請參閱 Windows SDK 文件中的[圖示](_win32_Icons_cpp)。  
  
 **Width**  
 提供可讓您輸入自訂影像寬度的空間，以像素為單位 \(1 – 512，限制為 2048\)。  
  
 **Height**  
 提供可讓您輸入自訂影像高度的空間，以像素為單位 \(1 – 512，限制為 2048\)。  
  
 **Colors**  
 提供可讓您輸入自訂影像色彩數 \(2、16 或 256\) 的空間。  
  
## 需求  
 None
