# Customizing or Changing Colors (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

影像編輯器的[調色盤](../windows/colors-window-image-editor-for-icons.md)一開始會顯示 16 個標準色彩。  除了顯示的色彩以外，您還可以自行建立自訂色彩。  然後，您可以[儲存及載入自訂的調色盤](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md)。  
  
### 變更調色盤上的色彩  
  
1.  從 \[**影像**\] 功能表中，選擇 \[**調整色彩**\]。  
  
2.  在 \[[自訂色彩選取器](../windows/custom-color-selector-dialog-box-image-editor-for-icons.md)\] 對話方塊中，在適當的文字方塊中輸入 RGB 或 HSL 值，或在 \[**漸層色彩顯示**\] 方塊中選擇色彩，以定義色彩。  
  
3.  移動 \[**光度**\] 列上的滑桿，以設定光度。  
  
4.  許多自訂色彩會被遞色。  如果您想要最接近遞色色彩的純色，請按兩下 \[**色彩**\] 方塊。  
  
     如果您稍後決定要遞色色彩，請移動 \[**亮度**\] 列上的滑桿，或是再次移動 \[**漸層色彩顯示**\] 方塊中的交叉線，以還原遞色。  
  
5.  按一下 \[**確定**\] 以加入新色彩。
