# defaultbind
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

表示可最佳表示物件的單一、 可繫結屬性。  
  
## 語法  
  
```  
  
[defaultbind]  
  
```  
  
## 備註  
 **Defaultbind** C\+\+ 屬性具有相同的功能，為 [defaultbind](http://msdn.microsoft.com/library/windows/desktop/aa366790) MIDL 屬性。  
  
## 範例  
 請參閱範例的[可繫結](../windows/bindable.md) 如需如何使用範例  **defaultbind**。

|**必要的屬性**|None|  
|**無效的屬性**|None|
