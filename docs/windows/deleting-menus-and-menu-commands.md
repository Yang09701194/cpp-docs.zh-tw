# Deleting Menus and Menu Commands
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### 刪除功能表或功能表命令  
  
1.  以滑鼠右鍵按一下功能表名稱或命令。  
  
2.  從捷徑功能表選擇 \[**刪除**\]。  
  
    > [!NOTE]
    >  同樣地，您可以使用捷徑功能表來執行其他動作，例如複製、剪下、貼上、插入新項目、插入分隔字元、編輯 ID、以快顯方式檢視、檢查助憶鍵等。  
  
 如需將資源加入至 Managed 專案的詳細資訊，請參閱《.NET Framework 開發人員手冊》[](../Topic/Resources%20in%20Desktop%20Apps.md)中的*應用程式中的資源*。
