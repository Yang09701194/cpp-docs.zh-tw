# <a name="disabling-guides"></a>停用輔助線
若要停用貼齊輔助線效果，您可以搭配滑鼠使用特殊按鍵。 使用 ALT 鍵停用選取本指南的貼齊的效果。 移動的指南與 SHIFT 鍵可防止貼齊的控制項移動輔助線。  
  
### <a name="to-disable-the-snapping-effect-of-the-guides"></a>若要停用貼齊輔助線效果  
  
1.  將控制項拖曳時按住**ALT**索引鍵。  
  
### <a name="to-move-guides-without-moving-the-snapped-controls"></a>輔助線移動而不移動貼齊的控制項  
  
1.  按住拖曳指南**SHIFT**索引鍵。  
  
### <a name="to-turn-off-the-guides"></a>若要關閉指南  
  
1.  從**格式**功能表上，選擇**輔助線設定**。  
  
2.  在[輔助線設定對話方塊](../windows/guide-settings-dialog-box.md)下**版面配置輔助線**，選取**無**。  
  
    > [!NOTE]
    >  您也可以按兩下尺規列來存取**輔助線設定** 對話方塊。  
  
 \-或-  
  
-   在**格式**功能表上，按一下 **切換輔助線**。
