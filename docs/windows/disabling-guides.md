# Disabling Guides
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

您可以將特殊的按鍵與滑鼠一起使用來停用輔助線的貼齊效果。  使用 ALT 鍵停用選取輔助線的貼齊效果。  以 SHIFT 鍵移動輔助線可以避免貼齊控制項受移動輔助線的影響。  
  
### 若要停用輔助線的貼齊效果  
  
1.  按住 **ALT** 鍵的同時，拖曳控制項。  
  
### 若要移動輔助線但不要移動貼齊的控制項  
  
1.  按住 **SHIFT** 鍵的同時，拖曳輔助線。  
  
### 若要關閉輔助線  
  
1.  從 \[格式\] 功能表選擇 \[輔助線設定\]。  
  
2.  在[輔助線設定對話方塊](../mfc/guide-settings-dialog-box.md)的 \[配置輔助線\] 下選取 \[無\]。  
  
    > [!NOTE]
    >  您也可以按兩下尺規列來存取 \[輔助線設定\] 對話方塊。  
  
 \-或\-  
  
-   在 \[格式\] 功能表，按一下 \[切換輔助線\]。
