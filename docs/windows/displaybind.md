# displaybind
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

表示應該顯示給使用者，可繫結之屬性。  
  
## 語法  
  
```  
  
[displaybind]  
  
```  
  
## 備註  
 **Displaybind** C\+\+ 屬性具有相同的功能，為 [displaybind](http://msdn.microsoft.com/library/windows/desktop/aa366804) MIDL 屬性。  
  
## 範例  
 請參閱範例的[可繫結](../windows/bindable.md) 如需如何使用範例  **displaybind**。
