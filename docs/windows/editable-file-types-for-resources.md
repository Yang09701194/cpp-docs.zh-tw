# <a name="editable-file-types-for-resources-c"></a>可編輯的檔案類型的資源 （c + +）

您可以開啟下列類型的檔案，並編輯所包含的資源。

|檔案名稱|描述|
|---------------|-----------------|
|.rc|資源指令碼檔。|
|.rct|資源範本檔。|
|.res|資源檔。|
|.resx|Managed 資源檔。|
|.exe|可執行檔。|
|.dll|動態連結程式庫檔案。|
|.bmp、.ico、.dib 和 .cur|點陣圖、圖示、工具列和游標檔案。|

## <a name="requirements"></a>需求

Win32
