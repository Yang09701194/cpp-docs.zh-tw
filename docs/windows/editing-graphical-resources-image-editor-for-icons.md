# Editing Graphical Resources (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

圖形資源是您為應用程式定義的影像。  您可徒手繪圖或使用形狀繪圖。  您可以選取影像的一部分編輯、翻轉或調整大小，也可以從影像的選定部分建立自訂筆刷，然後以該筆刷繪圖。  您可定義影像屬性、以不同格式儲存影像，以及將影像從某種格式轉換成另一種格式。  
  
 除了建立新的圖形資源之外，您還可以[匯入現有影像](../windows/how-to-import-and-export-resources.md)以進行編輯，然後將它們加入您的專案。  您也可以開啟並編輯不屬於專案一部分的影像，以進行[獨立式影像編輯](../mfc/editing-an-image-outside-of-a-project-image-editor-for-icons.md)。
