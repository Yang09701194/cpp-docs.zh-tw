# <a name="editing-in-an-accelerator-table"></a>在快速鍵對應表中編輯
### <a name="to-edit-in-an-accelerator-table"></a>在快速鍵對應表中編輯  
  
1.  它在圖示上按兩下開啟快速鍵對應表[資源檢視](../windows/resource-view-window.md)。  
  
    > [!NOTE]
    >  如果您的專案尚未包含 .rc 檔，請參閱 [建立新的資源指令碼檔](../windows/how-to-create-a-resource-script-file.md)。  
  
2.  在資料表中選取一個項目，然後按一下即可啟動就地編輯。  
  
3.  從下拉式方塊中選取，或輸入位置進行變更。  
  
    -   如[識別碼](id-property.md)，從選取清單或輸入以編輯。  
  
    -   如[修飾詞](../windows/accelerator-modifier-property.md)、 從清單中選取。  
  
    -   如[金鑰](../windows/accelerator-key-property.md)，從選取清單或輸入以編輯。  
  
    -   如[類型](../windows/accelerator-type-property.md)，從清單中選取 ASCII 或 VIRTKEY。  
  
 如需將資源加入至 managed 專案的詳細資訊，請參閱[桌面應用程式中的資源](/dotnet/framework/resources/index)中*.NET Framework 開發人員手冊 》。*
