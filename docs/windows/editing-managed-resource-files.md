# Editing Managed Resource Files
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Managed 資源檔 \(.resx\) 屬於 XML 檔案。  當您從 \[**加入新項目**\] 對話方塊將 Managed 資源檔加入專案時，\[**Managed 資源編輯器**\] 便會根據預設開啟。  
  
 如需詳細資訊，請參閱 [.Resx 檔案格式的資源](http://msdn.microsoft.com/zh-tw/0c476133-87e4-47e8-b0ef-4b88f4ef3dc5)。
