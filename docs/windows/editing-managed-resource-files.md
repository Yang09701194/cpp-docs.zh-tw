# <a name="editing-managed-resource-files"></a>編輯 Managed 資源檔
Managed 的資源檔 (.resx) 是 XML 檔案。 當您將受管理的資源檔加入您的專案從**加入新項目** 對話方塊中， **Managed 資源編輯器**預設會開啟。  
  
 如需詳細資訊，請參閱[中的資源。Resx 檔案格式](http://msdn.microsoft.com/0c476133-87e4-47e8-b0ef-4b88f4ef3dc5)。
