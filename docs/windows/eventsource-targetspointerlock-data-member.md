# <a name="eventsourcetargetspointerlock-data-member"></a>EventSource::targetsPointerLock_ 資料成員
同步處理時存取內部資料成員，即使事件處理常式這**EventSource**正在加入、 移除或叫用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
Wrappers::SRWLock targetsPointerLock_;  
```  
  
## <a name="requirements"></a>需求  
 **標頭：** event.h  
  
 **命名空間：** Microsoft::WRL
