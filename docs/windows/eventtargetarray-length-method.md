# <a name="eventtargetarraylength-method"></a>EventTargetArray::Length 方法
支援 WRL 基礎結構，並不是直接從您的程式碼使用。  
  
## <a name="syntax"></a>語法  
  
```  
size_t Length();  
```  
  
## <a name="return-value"></a>傳回值  
 目前的事件處理常式內部陣列中的元素數目。  
  
## <a name="remarks"></a>備註  
 取得目前的元素數目的事件處理常式內部的陣列。  
  
## <a name="requirements"></a>需求  
 **標頭：** event.h  
  
 **命名空間：** Microsoft::WRL::Details
