# EventTargetArray::~EventTargetArray 解構函式
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支援 WRL 基礎結構，而且不是為了要直接從您的程式碼中使用而設計。  
  
## 語法  
  
```  
~EventTargetArray();  
```  
  
## 備註  
 去初始化目前的 `EventTargetArray` 類別。  
  
## 需求  
 **標題:** event.h  
  
 **命名空間：** Microsoft::WRL::Details
