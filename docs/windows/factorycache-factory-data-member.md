# <a name="factorycachefactory-data-member"></a>FactoryCache::factory 資料成員
支援的 Windows 執行階段 c + + 樣板程式庫基礎結構，並不是直接從您的程式碼使用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
IUnknown* factory;  
```  
  
## <a name="remarks"></a>備註  
 指向 Windows 執行階段或 COM class factory。  
  
## <a name="requirements"></a>需求  
 **標頭：** module.h  
  
 **命名空間：** Microsoft::WRL::Details
