# Finding an Entry in an Accelerator Table
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### 在開啟的快速鍵對應表中尋找項目  
  
1.  在 \[[資源檢視](../windows/resource-view-window.md)\] 中按兩下快速鍵對應表的圖示，加以開啟。  
  
    > [!NOTE]
    >  如果您的專案尚未包含 .rc 檔，請參閱[建立新的資源指令碼檔](../windows/how-to-create-a-resource-script-file.md)。  
  
2.  按一下欄位標題，依字母順序排序欄位的內容。  例如，按一下 **ID**，依字母順序顯示快速鍵對應表中的所有識別碼。  
  
 然後就可以掃描清單並找到項目。  
  
 如需將資源加入至 Managed 專案的詳細資訊，請參閱《.NET Framework 開發人員手冊》中的[應用程式中的資源](../Topic/Resources%20in%20Desktop%20Apps.md)。  
  
 **需求**  
  
 Win32
