# <a name="finding-binary-data"></a>尋找二進位資料
您可以搜尋 ASCII 字串或十六進位位元組。 例如，若要尋找"Hello"，您可以針對在字串中搜尋"Hello"或"48 65 6C 6c 6F 」 （十六進位對等項目）。  
  
### <a name="to-find-binary-data"></a>若要尋找二進位資料  
  
1.  從**編輯**功能表上，按一下 [尋找](/visualstudio/ide/reference/find-command)。  
  
2.  在**尋找**方塊中，從下拉式清單中選取先前的搜尋字串或輸入您想要尋找的資料。  
  
3.  選取任何**尋找**選項。  
  
4.  按一下**找下一個**。
