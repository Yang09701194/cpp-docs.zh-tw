# <a name="flipping-an-image-image-editor-for-icons"></a>翻轉影像 (圖示影像編輯器)

您可以翻轉或旋轉的影像：

- 建立原始的鏡像映像。

- 面朝下開啟映像。

- 旋轉 90 度到右的映像，一次。

### <a name="to-flip-the-image-horizontally-mirror-image"></a>要水平翻轉影像 （鏡像映像）

1. 從**映像**功能表上，選擇**水平翻轉**。

### <a name="to-flip-the-image-vertically-turn-upside-down"></a>若要垂直翻轉影像 （要上下顛倒）

1. 從**映像**功能表上，選擇**垂直翻轉**。

### <a name="to-rotate-the-image-90-degrees"></a>若要將影像旋轉 90 度

1. 從**映像**功能表上，選擇**旋轉 90 度**。

   > [!NOTE]
   > 您也可以使用[快速鍵 （捷徑）](../windows/accelerator-keys-image-editor-for-icons.md)這些命令或從捷徑功能表 （按一下影像編輯器中的映像之外） 中存取命令。
