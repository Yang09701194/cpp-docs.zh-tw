# <a name="grid-settings-dialog-box-image-editor-for-icons"></a>格線設定對話方塊 (圖示影像編輯器)
可讓您指定格線設定您的映像，並顯示格線透過編輯映像。 格線可用於編輯映像，但不是會儲存為影像本身的一部分。  
  
 **像素格線**  
 有選取時，顯示的方格周圍每個像素影像編輯器中。 方格會顯示只在 4 × 和較高的解析度。  
  
 **並排顯示方格**  
 選取時，顯示的方格周圍的像素區塊在影像編輯器中，方格間距值所指定。  
  
 **寬度**  
 指定每個圖格區塊的寬度。 繪製點陣圖包含定期排列的多個映像時，這非常有用。  
  
 **高度**  
 指定的每個圖格區塊的高度。 繪製點陣圖包含定期排列的多個映像時，這非常有用。  
  
## <a name="requirements"></a>需求  
 無
