# <a name="guide-settings-dialog-box"></a>輔助線設定對話方塊
## <a name="layout-guides"></a>版面配置輔助線  
 會顯示配置輔助線設定。  
  
 **無**  
  
 隱藏配置工具。  
  
 **尺規和輔助線**  
  
 啟用時，將尺規加入版面配置工具;指南可置於尺規。 預設輔助線是會被拖曳的邊界。 按一下尺規放置輔助線。 控制 「 對齊 」 輔助線移動控制項上方或旁邊時。 它們附加到它，一旦也與指南移動控制項。 當控制項附加至每一邊的指南，並移動輔助線時，控制項將會調整大小。  
  
 **格線**  
  
 建立版面配置方格。 新控制項至格線，會自動對齊。  
  
## <a name="grid-spacing"></a>格線間距  
 對話方塊單位 (Dlu) 中顯示的格線間距的設定。  
  
 **寬度： Dlu**  
  
 Dlu 中設定的版面配置方格的寬度。 水平 DLU 是由四個分割對話方塊字型的平均寬度。  
  
 **高度： Dlu**  
  
 Dlu 中設定版面配置方格的高度。 垂直 DLU 是平均除以八個對話方塊字型的高度。
