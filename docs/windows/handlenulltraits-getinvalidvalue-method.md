# HANDLENullTraits::GetInvalidValue 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## 語法  
  
```  
inline static Type GetInvalidValue();  
```

 永遠傳回 `nullptr`。
