# <a name="handletattach-method"></a>HandleT::Attach 方法

將指定的控制代碼與目前**HandleT**物件。

## <a name="syntax"></a>語法

```cpp
void Attach(
   typename HandleTraits::Type h
);
```

#### <a name="parameters"></a>參數

*h*  
控制代碼。

## <a name="requirements"></a>需求

**標頭：** corewrappers.h

**命名空間：** Microsoft::WRL::Wrappers
