# HandleT::IsValid 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

表示目前 HandleT 物件是否表示控制代碼。  
  
## 語法  
  
```  
bool IsValid() const;  
```  
  
## 傳回值  
 `true` ，如果 HandleT 表示控制代碼，否則， `false`。  
  
## 需求  
 **標題:** corewrappers.h  
  
 **命名空間：**Microsoft::WRL::Wrappers
