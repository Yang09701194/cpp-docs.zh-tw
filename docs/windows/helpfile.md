# <a name="helpfile"></a>helpfile
設定型別程式庫的說明檔的名稱。  
  
## <a name="syntax"></a>語法  
  
```  
  
      [ helpfile(  
   "filename"  
) ]  
```  
  
#### <a name="parameters"></a>參數  
 *filename*  
 包含 [說明] 主題的檔案名稱。  
  
## <a name="remarks"></a>備註  
 **Helpfile** c + + 屬性具有相同的功能[helpfile](http://msdn.microsoft.com/library/windows/desktop/aa366853) MIDL 屬性。
