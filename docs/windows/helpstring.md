# <a name="helpstring"></a>helpstring
指定用來描述所套用之項目的字元字串。  
  
## <a name="syntax"></a>語法  
  
```  
[ helpstring(  
   "string"  
) ]  
```  
  
### <a name="parameters"></a>參數  
 *string*  
 [說明] 字串文字。  
  
## <a name="remarks"></a>備註  
 **Helpstring** c + + 屬性具有相同的功能[helpstring](http://msdn.microsoft.com/library/windows/desktop/aa366856) MIDL 屬性。
