# <a name="helpstring"></a>helpstring
指定的字元字串，用來描述它所套用的項目。  
  
## <a name="syntax"></a>語法  
  
```  
  
      [ helpstring(  
   "string"  
) ]  
```  
  
#### <a name="parameters"></a>參數  
 `string`  
 [說明] 字串的文字。
