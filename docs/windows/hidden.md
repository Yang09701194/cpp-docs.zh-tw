# <a name="hidden"></a>隱藏
表示項目存在，但不是會顯示在使用者導向的瀏覽器中。  
  
## <a name="syntax"></a>語法  
  
```  
  
[hidden]  
  
```  
  
## <a name="remarks"></a>備註  
 **隱藏**c + + 屬性具有相同的功能[隱藏](http://msdn.microsoft.com/library/windows/desktop/aa366861)MIDL 屬性。  
  
## <a name="example"></a>範例  
 請參閱範例的[可繫結](../windows/bindable.md)如需如何使用的範例**隱藏**。
