# hidden
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

表示該項目存在，但不是會顯示在使用者導向的瀏覽器中。  
  
## 語法  
  
```  
  
[hidden]  
  
```  
  
## 備註  
 **隱藏** C\+\+ 屬性具有相同的功能，為 [隱藏](http://msdn.microsoft.com/library/windows/desktop/aa366861) MIDL 屬性。  
  
## 範例  
 請參閱範例的[可繫結](../windows/bindable.md) 如需如何使用範例 **隱藏**。  
  
## 需求  
  
### 屬性內容  
  
|||  
|-|-|  
|**適用於**|`interface` **類別**， `struct`，方法、 屬性|  
|**可重複**|否|  
|**必要的屬性**|**coclass** \(用於**類別**或`struct`\)|  
|**無效的屬性**|None|  
  
 如需詳細資訊，請參閱[屬性內容](../windows/attribute-contexts.md)。
