# <a name="hstringattach-method"></a>HString::Attach 方法
將指定**HString**物件與目前**HString**物件。  
  
## <a name="syntax"></a>語法  
  
```cpp  
void Attach(  
       HSTRING hstr  
       ) throw()  
```  
  
### <a name="parameters"></a>參數  
 *hstr*  
 將現有**HString**物件。  
  
## <a name="requirements"></a>需求  
 **標頭：** corewrappers.h  
  
 **命名空間：** Microsoft::WRL::Wrappers
