# HString::Detach 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

解除其基礎值的指定 HString 物件。  
  
## 語法  
  
```  
HSTRING Detach() throw()  
```  
  
## 傳回值  
 起始中斷連結作業之前的基礎 HString 值。  
  
## 需求  
 **標題:** corewrappers.h  
  
 **命名空間：**Microsoft::WRL::Wrappers
