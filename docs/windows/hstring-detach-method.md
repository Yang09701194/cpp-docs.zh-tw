# <a name="hstringdetach-method"></a>HString::Detach 方法
取消指定的關聯**HString**與其基礎值的物件。  
  
## <a name="syntax"></a>語法  
  
```  
HSTRING Detach() throw()  
```  
  
## <a name="return-value"></a>傳回值  
 基礎**HString**卸離作業啟動之前的值。  
  
## <a name="requirements"></a>需求  
 **標頭：** corewrappers.h  
  
 **命名空間：** Microsoft::WRL::Wrappers
