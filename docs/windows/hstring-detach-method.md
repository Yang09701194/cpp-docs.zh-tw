# <a name="hstringdetach-method"></a>HString::Detach 方法
解除指定的 HString 物件從其基礎值的關聯。  
  
## <a name="syntax"></a>語法  
  
```  
HSTRING Detach() throw()  
```  
  
## <a name="return-value"></a>傳回值  
 卸離作業啟動之前的基礎 HString 值。  
  
## <a name="requirements"></a>需求  
 **標頭：** corewrappers.h  
  
 **命名空間：** Microsoft::WRL::Wrappers
