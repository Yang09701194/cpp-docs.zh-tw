# <a name="hstringget-method"></a>HString::Get 方法

## <a name="syntax"></a>語法  
  
```  
HSTRING Get() const throw()  
```  
  
## <a name="return-value"></a>傳回值  
 基礎 HSTRING 控制代碼值

 **命名空間：** Microsoft::WRL::Wrappers
