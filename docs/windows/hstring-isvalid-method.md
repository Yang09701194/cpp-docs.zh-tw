# <a name="hstringisvalid-method"></a>HString::IsValid 方法

指出是否目前**HString**物件是否為空的。

## <a name="syntax"></a>語法

```cpp
bool IsValid() const throw()  
```

### <a name="parameters"></a>參數

**真**如果目前**HString**物件不是空的否則**false**。

## <a name="requirements"></a>需求

**標頭：** corewrappers.h

**命名空間：** Microsoft::WRL::Wrappers
