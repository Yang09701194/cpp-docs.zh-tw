# HString::IsValid 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

指出目前的 HString 物件是否是空的。  
  
## 語法  
  
```  
bool IsValid() const throw()  
```  
  
#### 參數  
 `true` ，如果目前 HString 物件不是空的，否則， `false`。  
  
## 需求  
 **標題:** corewrappers.h  
  
 **命名空間：**Microsoft::WRL::Wrappers
