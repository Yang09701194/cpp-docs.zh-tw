# <a name="hstringrelease-method"></a>HString::Release 方法

## <a name="syntax"></a>語法  
  
```cpp  
void Release() throw()  
```  
  
## <a name="requirements"></a>需求

 **命名空間：** Microsoft::WRL::Wrappers
