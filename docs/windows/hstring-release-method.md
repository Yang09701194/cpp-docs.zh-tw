# <a name="hstringrelease-method"></a>HString::Release 方法
會刪除基礎字串值，並初始化目前**HString**為空值的物件。  
  
## <a name="syntax"></a>語法  
  
```  
void Release() throw()  
```  
  
## <a name="requirements"></a>需求  
 **標頭：** corewrappers.h  
  
 **命名空間：** Microsoft::WRL::Wrappers
