# <a name="id-property"></a>ID 屬性

ID 屬性要參考程式碼中的每個快速鍵對應表項目。 這是當使用者按下快速鍵或按鍵組合時，程式將收到的命令值。 若要讓功能表項目相同加速器，讓它們的 Id 相同 （只要快速鍵對應表的識別碼是功能表資源的識別碼相同）。

## <a name="requirements"></a>需求

Win32
