# id
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

指定`dispid` \(屬性或方法，在介面或分配介面\) 的成員函式的參數。  
  
## 語法  
  
```  
  
      [ id(  
   dispid  
) ]  
```  
  
#### 參數  
 `dispid`  
 介面方法分派識別碼。  
  
## 備註  
 **Id** C\+\+ 屬性具有相同的功能，為 [id](http://msdn.microsoft.com/library/windows/desktop/aa367040) MIDL 屬性。  
  
## 範例  
 請參閱範例的[可繫結](../windows/bindable.md) 如需如何使用範例  **id**。  
  
## 需求  
  
### 屬性內容  
  
|||  
|-|-|  
|**適用於**|介面方法|  
|**可重複**|否|  
|**必要的屬性**|None|  
|**無效的屬性**|None|  
  
 如需詳細資訊，請參閱[屬性內容](../windows/attribute-contexts.md)。
