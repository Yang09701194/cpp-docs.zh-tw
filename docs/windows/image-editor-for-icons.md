# <a name="image-editor-for-icons-c"></a>圖示影像編輯器 (C++)

當您選取**方案總管**中的影像檔案 (例如 .ico、.bmp、.png) 時, 影像會在 [**影像編輯器**] 中開啟, 就像在程式**代碼編輯器**中開啟程式碼檔案一樣。 當 [**影像編輯器**] 索引標籤處於作用中狀態時, 您會看到工具列具有許多用來建立和編輯影像的工具。 除了點陣圖、圖示和游標之外, 您還可以使用 [**影像**] 功能表上的命令和 [**影像編輯器**] 工具列上的 [工具], 來編輯 GIF 或 JPEG 格式的影像。

圖形資源是您為應用程式定義的影像。 您可以使用圖形繪製手繪或繪製。 您可以選取影像的某些部分進行編輯、翻轉或調整大小, 也可以從影像的選取部分建立自訂筆刷, 並使用該筆刷繪製。 您可以定義影像屬性、以不同的格式儲存影像, 並將影像從一種格式轉換成另一種格式。

> [!NOTE]
> 使用**影像編輯器**, 您可以查看32位映射, 但無法進行編輯。

您也可以使用**影像編輯器**和[二進位編輯器](binary-editor.md)來處理 managed 專案中的資源檔。 您想要編輯的任何 Managed 資源皆必須為連結的資源。 Visual Studio 資源編輯器不支援編輯內嵌資源。

除了建立新的圖形資源之外, 您還可以匯[入現有的影像](../windows/how-to-copy-resources.md#import-and-export-resources)進行編輯, 然後將它們新增至您的專案。 您也可以開啟和編輯不屬於專案一部分的影像, 以進行[獨立的影像編輯](../windows/editing-an-image-outside-of-a-project-image-editor-for-icons.md)。

如需**影像編輯器**的相關資訊, 請參閱如何[建立圖示或其他影像](../windows/creating-an-icon-or-other-image-image-editor-for-icons.md)、[編輯影像](../windows/selecting-an-area-of-an-image-image-editor-for-icons.md)、[使用繪圖工具](../windows/using-a-drawing-tool-image-editor-for-icons.md)、[色彩](../windows/working-with-color-image-editor-for-icons.md)和[快速鍵](../windows/accelerator-keys-image-editor-for-icons.md)。

> [!NOTE]
> 免費下載**Visual Studio 映射庫**, 其中包含許多可在應用程式中使用的動畫、點陣圖和圖示。 如需有關如何下載程式庫的詳細資訊, 請參閱[Visual Studio 映射庫](/visualstudio/designers/the-visual-studio-image-library)。

## <a name="image-menu"></a>影像功能表

[**影像**] 功能表, 只有在**影像編輯器**作用中、具有編輯影像的命令、管理調色板, 以及設定 [**影像編輯器**] 視窗選項時, 才會出現。 此外, 使用圖示和游標時, 也可以使用裝置映射的命令。

|命令|描述|
|---|---|
|**色彩反色**|反轉您的色彩。|
|**水平翻轉**|水平翻轉影像或選取範圍。|
|**垂直翻轉**|垂直翻轉影像或選取範圍。|
|**旋轉90度**|旋轉影像或選取範圍 90 度。|
|**顯示色彩視窗**|開啟 [**色彩**] 視窗, 您可以在其中選擇要用於影像的色彩。|
|**使用選取專案做為筆刷**|可讓您從影像的一部分建立自訂筆刷。<br/><br/>您的選取專案會成為自訂筆刷, 可將選取範圍中的色彩分散到影像中。 選取的複本會沿著拖曳路徑向左。 拖曳的速度愈慢, 就會產生更多的複本。|
|**複製和大綱選取範圍**|建立一份目前選取範圍的複本，並建立此複本的外框。<br/><br/>如果背景色彩包含在目前的選取範圍內, 則會在您選取 [透明] 時予以排除。
|**調整色彩**|開啟 [**自訂色彩選取器**], 可讓您自訂用於影像的色彩。|
|**載入調色板**|開啟 [**載入色彩**選擇區] 對話方塊, 可讓您載入先前儲存至 pal 檔案的調色板色彩。|
|**儲存調色板**|將調色板色彩儲存至 pal 檔案。|
|**繪製不透明**|選取此選項時, 會使目前的選取範圍不透明。<br/><br/>清除時, 會使目前的選取範圍變成透明。|
|**工具列編輯器**|開啟 [[新增工具列資源] 對話方塊](../windows/new-toolbar-resource-dialog-box.md)。|
|**方格設定**|開啟 [**方格設定**] 對話方塊, 您可以在其中指定影像的格線。|
|**新增映射類型**|開啟 [[新增\<裝置 > 映射類型] 對話方塊](../windows/new-device-image-type-dialog-box-image-editor-for-icons.md)。<br/><br/>單一圖示資源可以包含幾個不同大小的影像, 而 windows 可以使用適當的圖示大小, 視其顯示方式而定。 新的裝置類型不會修改圖示的大小, 而是在圖示中建立新的影像。 僅適用于圖示和游標。|
|**目前的圖示/游標影像類型**|開啟子功能表, 其中列出前9個可用的游標或圖示影像。 子功能表上的最後一個命令 [**更多**] 會開啟 [[開啟\<裝置 > 影像] 對話方塊](../windows/open-device-image-dialog-box-image-editor-for-icons.md)。|
|**刪除映射類型**|刪除選取的裝置映射。|
|**工具**|啟動子功能表, 其中包含 [**影像編輯器**] 工具列所提供的所有工具。|

[**方格設定**] 對話方塊可讓您指定影像的方格設定, 並在編輯過的影像上顯示格線。 這幾行對於編輯影像很有用, 但不會儲存為影像本身的一部分。

|屬性|描述|
|---|---|
|**圖元方格**|若有選取時, 會在**影像編輯器**中的每個圖元周圍顯示方格。<br/><br/>方格只會以4×和更高的解析度顯示。|
|**磚方格**|選取時, 會在 [**影像編輯器**] 中以格線間距值所指定的圖元區塊來顯示格線。|
|[寬度]|指定每個磚區塊的寬度。<br/><br/>當繪製的點陣圖包含以固定間隔排列的多個影像時, 這個屬性會很有用。|
|[高度]|指定每個磚區塊的高度。<br/><br/>當繪製的點陣圖包含以固定間隔排列的多個影像時, 這個屬性會很有用。|

## <a name="toolbar"></a>工具列

[**影像編輯器**] 工具列包含用於繪製、繪製、輸入文字、清除和操作視圖的工具。 它也包含選項選取器, 可讓您選取使用每項工具的選項。 例如, 您可以選擇各種筆刷寬度、縮放比例和線條樣式。

[**影像編輯器**] 工具列上的所有可用工具也可從 [功能表**影像** > **工具**] 取得。 若要使用 [**影像編輯器**] 工具列和**選項**選取器, 請選取您想要的工具或選項。

![影像編輯器工具列](../mfc/media/vcimageeditortoolbar.gif "vcImageEditorToolbar")<br/>
**影像編輯器**工具列

> [!TIP]
> 當您將游標移到工具列按鈕上方時, 會出現工具提示。 這些秘訣可協助您識別每個按鈕的功能。

因為許多繪圖工具都可以從[鍵盤](../windows/accelerator-keys-image-editor-for-icons.md)取得, 所以隱藏**影像編輯器**工具列有時會很有用。

- 若要顯示或隱藏 [**影像編輯器**] 工具列, 請移至 [功能表**視圖** > ]**工具列**, 然後選擇 [**影像編輯器**]。

> [!NOTE]
> 當目前專案或方案中的影像檔案未在**影像編輯器**中開啟時, 此工具列中的元素將會顯示為無法使用。

### <a name="text-tool"></a>文字工具

使用 [**文字工具**] 對話方塊, 將文字加入至游標、點陣圖或圖示資源。

若要存取此對話方塊, 請開啟**影像編輯器**並移至 [功能表] [**影像** > **工具**], 然後選取 [**文字工具**] 命令。

> [!TIP]
> 您可以用滑鼠右鍵按一下 [**文字工具**] 對話方塊來存取預設的快捷方式功能表, 其中包含標準 Windows 命令的清單。

開啟 [**文字工具字型**] 對話方塊, 即可變更游標字型的字型、樣式或大小。 變更會套用至**文字**區域中顯示的文字。

若要存取這個對話方塊, 請選取 [**文字工具**] 對話方塊中的 [**字型**] 按鈕。 可用的屬性如下:

|屬性|描述|
|---|---|
|**字型**|列出可用的字型。|
|[字型樣式]|列出指定字型的可用樣式。|
|**Size**|列出指定字型的可用點大小。|
|**範例**|顯示文字如何使用指定的字型設定來顯示的範例。|
|**指令碼**|列出指定字型的可用語言腳本。<br/><br/>當您選取不同的語言腳本時, 就可以使用該語言的字元集來建立多語系檔。|

#### <a name="to-change-the-font-of-text-on-an-image"></a>變更影像上的文字字型

以下範例示範如何將文字加入 Windows 應用程式中的圖示, 以及操作文字的字型。

1. 建立C++ Windows Forms 應用程式。 如需詳細資訊，請參閱[如何：建立 Windows Forms 應用](/previous-versions/visualstudio/visual-studio-2008/s69bf10x(v%3dvs.90))程式。 根據預設, 會將*應用程式 .ico*檔案加入您的專案中。

1. 在**方案總管**中, 按兩下 [file]*應用程式。* **影像編輯器**隨即開啟。

1. 移至 [功能表**影像** > **工具**], 然後選取 [**文字] 工具**。

1. 在 [**文字工具**] 對話方塊中, *C++* 輸入空的文字區域。 此文字會出現在**影像編輯器**中, 位於*應用程式 .ico*左上角的可調整大小方塊中。

1. 在**影像編輯器**中, 將可調整大小的方塊拖曳到*app.config*的中央, 以改善文字的可讀性。

1. 在 [**文字工具**] 對話方塊中, 選取 [**字型**] 按鈕。

1. 在 [**文字工具字型**] 對話方塊中:

   - 從 [字型] 清單方塊中所列的可用**字型**清單中選取 [ **Times New Roman** ]。

   - 從 [**字型樣式**] 清單方塊中所列的可用字型樣式清單中, 選取 [**粗體**]。

   - 從 [**大小**] 清單方塊中所列的可用點大小清單中選取 [ **10** ]。

   - 選擇 [確定]。 [**文字工具字型**] 對話方塊將會關閉, 而新的字型設定會套用至您的文字。

1. 選擇 [**文字工具**] 對話方塊上的 [**關閉**]。 文字周圍的可調整大小方塊會從**影像編輯器**中消失。

文字區域會顯示顯示為資源一部分的文字。 此區域一開始是空的。

> [!NOTE]
> 如果已設定**透明背景**, 則只有文字會放入影像中。 如果設定了不**透明背景**, 則會在文字後方放置以背景色彩填滿的周框。

## <a name="window-panes"></a>視窗窗格

[**影像編輯器**] 視窗會顯示影像的兩個視圖, 而分割列會分隔兩個窗格。 您可隨意地拖曳分隔列，以變更窗格的相對大小。 現用窗格會顯示選取框線。

其中一個 view 是實際大小, 另一個則是以預設放大因數6放大。 這兩個窗格中的 views 會自動更新, 而您在某個窗格中所做的任何變更都會立即顯示在另一個窗格中。 這兩個窗格可讓您輕鬆地使用影像的放大視圖, 您可以在其中區分個別圖元, 同時觀察您的工作對影像實際大小視圖的影響。

左窗格會使用所需的空間 (最多一半的**影像**視窗) 來顯示影像的預設1:1 縮放比例。 右窗格會顯示預設的6:1 縮放星號調整影像。 您可以使用 [**影像編輯器**] 工具列上的 [**放大**] 工具, 或使用快速鍵, 來變更每個窗格中的縮放比例。

您可以放大 [**影像編輯器**] 視窗的較小窗格, 並使用這兩個窗格來顯示大型影像的不同區域。 在窗格中選取以選擇它。

您可以變更窗格的相對大小, 方法是將指標放在分割列上, 然後將分割列移到右邊或左方。 如果您只想要在一個窗格上工作, 分割列可以移動到任一側。

如果 [**影像編輯器**] 窗格是以4個或更大的因數放大, 您可以顯示圖元格線, 以分隔影像中的個別圖元。

### <a name="to-change-the-magnification-factor"></a>變更放大因數

根據預設,**影像編輯器**會在左窗格中顯示實際大小的視圖, 而右窗格中的視圖會在 [實際大小] 的6倍。 [放大因數] (出現在工作區底部的狀態列中) 是影像實際大小與顯示大小之間的比例。 預設因數為 6, 範圍是從1到10。

1. 選取您想要變更其縮放比例的 [**影像編輯器**] 窗格。

1. 在 [**影像編輯器**] 工具列上, 選取 [**放大**] 工具右邊的箭號, 然後從子功能表選取 [縮放因數]:**1x**、 **2x**、 **6X**或**8x**。

   > [!NOTE]
   > 若要選取 [**放大**] 工具中所列以外的縮放因數, 請使用快速鍵。

### <a name="to-display-or-hide-the-pixel-grid"></a>若要顯示或隱藏圖元方格

針對縮放因數為4或更大的所有**影像編輯器**窗格, 您可以顯示格線, 以分隔影像中的個別圖元。

1. 移至 [功能表] [**影像** > **方格設定**]。

1. 選取 [**圖元方格**] 核取方塊以顯示方格, 或清除方塊以隱藏方格。
