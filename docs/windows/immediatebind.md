# <a name="immediatebind"></a>immediatebind

表示資料庫將會立即被告知資料繫結物件的屬性的所有變更。

## <a name="syntax"></a>語法

```cpp
[immediatebind]
```

## <a name="remarks"></a>備註

**Immediatebind** c + + 屬性具有相同的功能[immediatebind](/windows/desktop/Midl/immediatebind) MIDL 屬性。

## <a name="example"></a>範例

請參閱[可繫結](../windows/bindable.md)如需如何使用的範例**immediatebind**。
