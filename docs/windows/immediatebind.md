# <a name="immediatebind"></a>immediatebind
指示資料庫將會立即被告知資料繫結物件的屬性的所有變更。  
  
## <a name="syntax"></a>語法  
  
```  
  
[immediatebind]  
  
```  
  
## <a name="remarks"></a>備註  
 **Immediatebind** c + + 屬性具有相同的功能[immediatebind](http://msdn.microsoft.com/library/windows/desktop/aa367045) MIDL 屬性。
