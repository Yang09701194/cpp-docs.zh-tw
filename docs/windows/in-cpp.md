# in (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

表示參數從呼叫的程序傳遞至被呼叫的程序。  
  
## 語法  
  
```  
  
[in]  
  
```  
  
## 備註  
 **在** C\+\+ 屬性具有相同的功能，為 [在](http://msdn.microsoft.com/library/windows/desktop/aa367051) MIDL 屬性。  
  
## 範例  
 請參閱[可繫結](../windows/bindable.md) 如需如何使用範例 **在**。  
  
## 需求  
  
### 屬性內容  
  
|||  
|-|-|  
|**適用於**|介面參數，介面方法|  
|**可重複**|否|  
|**必要的屬性**|None|  
|**無效的屬性**|**retval**|  
  
 如需有關屬性內容的詳細資訊，請參閱[屬性內容](../windows/attribute-contexts.md)。
