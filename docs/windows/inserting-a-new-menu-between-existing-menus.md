# Inserting a New Menu Between Existing Menus
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### 在現有的功能表間插入新的功能表  
  
1.  選取現有的功能表名稱並按 **INSERT** 鍵。  \[**新項目**\] 方塊會插入在選取的項目之前。  
  
     \-或\-  
  
2.  在功能表列上按一下滑鼠右鍵，然後從快顯功能表中選擇 \[**插入新項目**\]。  
  
 如需將資源加入至 Managed 專案的詳細資訊，請參閱《.NET Framework 開發人員手冊》中的[應用程式中的資源](../Topic/Resources%20in%20Desktop%20Apps.md)。

 Win32
