# <a name="inserting-a-space-between-buttons-on-a-toolbar"></a>在工具列的按鈕之間插入空白
一般情況下，若要插入按鈕之間的空白，只是將它們拖曳搶奪工具列上。 若要移除的空間，請將它們拖曳朝向彼此。  
  
### <a name="to-insert-a-space-before-a-button-that-is-not-followed-by-a-space"></a>若要後面沒有空格的按鈕之前插入空格  
  
1.  將按鈕向右或向下直到它重疊下一步按鈕的相關地球的另一端。  
  
### <a name="to-insert-a-space-before-a-button-which-is-followed-by-a-space-and-to-retain-that-trailing-space"></a>前面插入空格後接空格的按鈕，並保留該結尾的空白字元  
  
1.  將按鈕拖曳右側或底端邊緣剛接觸 [下一步] 按鈕，或只與重疊之前。  
  
### <a name="to-insert-a-space-before-a-button-that-is-followed-by-a-space-and-close-up-that-following-space"></a>前面插入空格後接空格的按鈕，並關閉的空白  
  
1.  將按鈕向右或向下直到它重疊下一步按鈕的相關地球的另一端。
