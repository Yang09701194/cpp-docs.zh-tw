# Inverting the Colors in a Selection (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

影像編輯器可方便您反轉所選影像部分中的色彩，以便指出影像如何以反轉的色彩顯示。  
  
### 反轉目前選取範圍中的色彩  
  
1.  在 \[**影像**\] 功能表上，按一下 \[**反轉色彩**\]。
