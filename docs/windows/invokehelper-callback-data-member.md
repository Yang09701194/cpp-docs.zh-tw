# <a name="invokehelpercallback-data-member"></a>InvokeHelper::callback_ 資料成員
支援 WRL 基礎結構，並不是直接從您的程式碼使用。  
  
## <a name="syntax"></a>語法  
  
```  
TCallback callback_;  
```  
  
## <a name="remarks"></a>備註  
 表示事件發生時要呼叫的事件處理常式。  
  
 `TCallback`樣板參數指定事件處理常式的類型。  
  
## <a name="requirements"></a>需求  
 **標頭：** event.h  
  
 **命名空間：** Microsoft::WRL::Details
