# 載入調色盤色彩對話方塊 (圖示影像編輯器)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使用這個對話方塊，可以載入特殊的調色盤以便在專案中使用。  如需詳細資訊，請參閱[儲存及載入不同的調色盤](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md)。  
  
 **查詢**  
 指定要尋找檔案或資料夾的位置。  按一下箭頭來選取另一個位置，或是按一下工具列上的資料夾圖示移至上一層。  
  
 **檔案名稱**  
 提供空間，供您輸入您要開啟的檔案名稱。  若要快速尋找先前已開啟的檔案，請按一下下拉式清單 \(Drop\-Down List\) 中的檔名 \(如果有的話\)。  
  
 如果您正在搜尋檔案，可使用星號 \(\*\) 做為萬用字元。  例如，您可以輸入 \*.\* 來查看所有檔案的清單。  您也可以輸入檔案的完整路徑，例如 C:\\My Documents\\MyColorPalette.pal 或 \\\\NetworkServer\\MyFolder\\MyColorPalette.pal。  
  
 **檔案類型**  
 列出要顯示的檔案類型。  Palette \(\*.pal\) 是調色盤的預設檔案類型。
