# MakeAllocator::MakeAllocator 建構函式
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支援 WRL 基礎結構，而且不是為了要直接從您的程式碼中使用而設計。  
  
## 語法  
  
```  
MakeAllocator();  
```  
  
## 備註  
 初始化 MakeAllocator 類別的新執行個體。  
  
## 需求  
 **標題:** implements.h  
  
 **命名空間：** Microsoft::WRL::Details
