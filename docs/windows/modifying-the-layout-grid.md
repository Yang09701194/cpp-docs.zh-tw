# Modifying the Layout Grid
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在對話方塊中放置或排列控制項時，您可以使用配置格線來使控制項的定位更準確。  啟動格線時，控制項出現以「貼齊」格線的虛線，就好像被磁鐵吸住一樣。  您可以開啟和關閉這項「貼齊格線」功能並且變更配置格線儲存格的大小。  
  
### 若要開啟和關閉配置格線  
  
1.  從 \[格式\] 功能表選擇 \[輔助線設定\]。  
  
2.  在[輔助線設定對話方塊](../mfc/guide-settings-dialog-box.md)中，選取或清除 \[格線\] 按鈕。  
  
     您仍然可以使用[對話方塊編輯器工具列](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md)上的 \[切換格線\] 按鈕來控制個別對話方塊編輯器視窗中的格線。  
  
### 若要變更配置格線的大小  
  
1.  從 \[格式\] 功能表選擇 \[輔助線設定\]。  
  
2.  在[輔助線設定對話方塊](../mfc/guide-settings-dialog-box.md)中，以 DLU 為格線中的儲存格的高度和寬度單位輸入高度和寬度。  最小的高度或寬度是 4 DLU。  如需 DLU 的詳細資訊，請參閱[對話方塊上控制項的排列方式](../mfc/arrangement-of-controls-on-dialog-boxes.md)。  
  
 如需將資源加入至 Managed 專案的詳細資訊，請參閱《.NET Framework 開發人員手冊》中的[應用程式中的資源](../Topic/Resources%20in%20Desktop%20Apps.md)。 如需手動將資源加入至 Managed 專案、存取資源、顯示靜態資源和指定屬性的資源字串等詳細資訊，請參閱[逐步解說：將 Windows Form 當地語系化](http://msdn.microsoft.com/zh-tw/9a96220d-a19b-4de0-9f48-01e5d82679e5)和[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)。
