# <a name="moduledecrementobjectcount-method"></a>Module::DecrementObjectCount 方法
此模組所追蹤的物件數目的遞減。  
  
## <a name="syntax"></a>語法  
  
```cpp  
virtual long DecrementObjectCount();  
```  
  
## <a name="return-value"></a>傳回值  
 遞減作業之前計數。  
  
## <a name="requirements"></a>需求  
 **標頭：** module.h  
  
 **命名空間：** Microsoft::WRL
