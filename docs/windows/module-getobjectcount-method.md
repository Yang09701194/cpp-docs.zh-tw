# <a name="modulegetobjectcount-method"></a>Module::GetObjectCount 方法
擷取這個模組所管理的物件數目。  
  
## <a name="syntax"></a>語法  
  
```  
virtual long GetObjectCount() const;  
```  
  
## <a name="return-value"></a>傳回值  
 目前受此模組的物件的數目。  
  
## <a name="requirements"></a>需求  
 **標頭：** module.h  
  
 **命名空間：** Microsoft::WRL
