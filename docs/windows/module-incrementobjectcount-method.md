# Module::IncrementObjectCount 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

遞增模組所追蹤的物件的數目。  
  
## <a name="syntax"></a>語法  
  
```  
virtual long IncrementObjectCount();  
```  
  
## <a name="return-value"></a>傳回值  
 前遞增作業數。  
  
## <a name="requirements"></a>需求  
 **標頭︰** module.h  
  
 **命名空間：** Microsoft::WRL
