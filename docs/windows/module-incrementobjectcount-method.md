# <a name="moduleincrementobjectcount-method"></a>Module::IncrementObjectCount 方法
遞增模組所追蹤的物件數目。  
  
## <a name="syntax"></a>語法  
  
```cpp  
virtual long IncrementObjectCount();  
```  
  
## <a name="return-value"></a>傳回值  
 之前的遞增作業計數。  
  
## <a name="requirements"></a>需求  
 **標頭：** module.h  
  
 **命名空間：** Microsoft::WRL
