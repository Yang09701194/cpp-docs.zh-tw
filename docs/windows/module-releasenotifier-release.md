# <a name="modulereleasenotifierrelease"></a>Module::ReleaseNotifier::Release
刪除目前**releasenotifier**物件如果物件以參數的建構 **，則為 true**。  
  
## <a name="syntax"></a>語法  
  
```cpp  
void Release() throw();  
```  
  
## <a name="requirements"></a>需求  
 **標頭：** module.h  
  
 **命名空間：** Microsoft::WRL
