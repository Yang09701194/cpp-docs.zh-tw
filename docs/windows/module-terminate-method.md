# <a name="moduleterminate-method"></a>Module::Terminate 方法
會導致所有關閉的模組來具現化的處理站。  
  
## <a name="syntax"></a>語法  
  
```cpp  
void Terminate();  
```  
  
## <a name="remarks"></a>備註  
 釋放快取中的處理站。  
  
## <a name="requirements"></a>需求  
 **標頭：** module.h  
  
 **命名空間：** Microsoft::WRL
