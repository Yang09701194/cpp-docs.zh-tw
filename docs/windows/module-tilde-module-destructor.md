# <a name="modulemodule-destructor"></a>Module::~Module 解構函式

取消初始化目前的執行個體**模組**類別。

## <a name="syntax"></a>語法

```cpp
virtual ~Module();
```

## <a name="requirements"></a>需求

**標頭：** module.h

**命名空間：** Microsoft::WRL
