# <a name="modulebasemodulebase-constructor"></a>ModuleBase::ModuleBase 建構函式

支援 WRL 結構，而且不是直接從您的程式碼使用。

## <a name="syntax"></a>語法

```cpp
ModuleBase();
```

## <a name="remarks"></a>備註

初始化 `Module` 類別的執行個體。

**標頭：** implements.h

**命名空間：** Microsoft::WRL::Details
