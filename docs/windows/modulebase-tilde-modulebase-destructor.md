# <a name="modulebasemodulebase-destructor"></a>ModuleBase::~ModuleBase 解構函式
支援 WRL 結構，而且不是直接從您的程式碼使用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
virtual ~ModuleBase();  
```  
  
## <a name="remarks"></a>備註  
 取消初始化目前的執行個體**ModuleBase**類別。  
  
## <a name="requirements"></a>需求  
 **標頭：** implements.h  
  
 **命名空間：** Microsoft::WRL::Details
