# <a name="modulebasemodulebase-destructor"></a>ModuleBase::~ModuleBase 解構函式
支援 WRL 基礎結構，並不是直接從您的程式碼使用。  
  
## <a name="syntax"></a>語法  
  
```  
virtual ~ModuleBase();  
```  
  
## <a name="remarks"></a>備註  
 取消初始化 ModuleBase 類別的目前執行個體。  
  
## <a name="requirements"></a>需求  
 **標頭：** implements.h  
  
 **命名空間：** Microsoft::WRL::Details
