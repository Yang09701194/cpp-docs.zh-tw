# 新增自訂資源對話方塊
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

讓您可以建立新的自訂資源。  
  
 **資源類型**  
 提供讓您輸入自訂資源類型名稱的文字方塊。  當您結束 \[**新增自訂資源**\] 對話方塊時，Visual C\+\+ 會自動將名稱變成大寫。  
  
## 備註  
 不要指定 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 保留的資源名稱或 [!INCLUDE[TLA2#tla_id](../windows/includes/tla2sharptla_id_md.md)]。  保留的名稱為 DESIGNINFO、HWB 和 TEXTINCLUDE，而保留的 [!INCLUDE[TLA2#tla_id](../windows/includes/tla2sharptla_id_md.md)] 則為 255。
