# <a name="nil-structure"></a>Nil 結構
支援 WRL 基礎結構，並不是直接從您的程式碼使用。  
  
## <a name="syntax"></a>語法  
  
```  
struct Nil;  
```  
  
## <a name="remarks"></a>備註  
 用來指出未指定，選用的範本參數。  
  
 Nil 是空的結構。  
  
## <a name="inheritance-hierarchy"></a>繼承階層  
 `Nil`  
  
## <a name="requirements"></a>需求  
 **標頭：** implements.h  
  
 **命名空間：** Microsoft::WRL::Details
