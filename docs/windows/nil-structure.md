# Nil 結構
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支援 WRL 基礎結構，而且不是為了要直接從您的程式碼中使用而設計。  
  
## 語法  
  
```  
struct Nil;  
```  
  
## 備註  
 用來表示未指定，選擇性樣板參數。  
  
 Nil是空白結構。  
  
## 繼承階層架構  
 `Nil`  
  
## 需求  
 **標題:** implements.h  
  
 **命名空間：** Microsoft::WRL::Details
