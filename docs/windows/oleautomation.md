# <a name="oleautomation"></a>oleautomation

表示與 Automation 相容介面。

## <a name="syntax"></a>語法

```cpp
[oleautomation]
```

## <a name="remarks"></a>備註

**Oleautomation** c + + 屬性具有相同的功能[oleautomation](/windows/desktop/Midl/oleautomation) MIDL 屬性。

## <a name="example"></a>範例

請參閱範例[defaultvalue](../windows/defaultvalue.md)並[nonextensible](../windows/nonextensible.md)的範例使用**oleautomation**。
