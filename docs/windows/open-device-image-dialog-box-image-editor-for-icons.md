# 開啟 &lt;裝置&gt; 影像對話方塊 (圖示影像編輯器)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

這個對話方塊可以用來開啟裝置影像。  它會列出目前的資源中既存的裝置影像 \(為目前資源之一部分的影像\)。  
  
 **目前的影像**  
 列出資源內包含的影像。  選取您要開啟的影像類型。  
  
## 需求  
 None
