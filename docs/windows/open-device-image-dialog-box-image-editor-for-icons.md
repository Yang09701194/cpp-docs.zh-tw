# <a name="open-ltdevicegt-image-dialog-box-c-image-editor-for-icons"></a>開啟&lt;裝置&gt;影像對話方塊 （c + +） （圖示影像編輯器）

使用此對話方塊中，以開啟 c + + 專案中的 裝置映像。 它會列出目前的資源 （屬於目前的資源的映像） 的現有裝置映像。

### <a name="current-images"></a>目前的映像

列出資源中包含的映像。 選取您想要開啟的影像類型。

## <a name="requirements"></a>需求

無
