# Opening the Resource Editor for a Given Symbol
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

當您瀏覽 \[[資源符號](../windows/resource-symbols-dialog-box.md)\] 對話方塊中的符號時，您可能會想要深入了解特定符號的使用方式。  \[**檢視使用**\] 按鈕可讓您快速取得這項資訊。  
  
### 移至正在使用符號的資源編輯器  
  
1.  在 \[**資源符號**\] 對話方塊的 \[**名稱**\] 方塊中選取符號。  
  
2.  在 \[**使用依據**\] 方塊中，選取您想要使用的資源類型。  
  
3.  按一下 \[**檢視使用**\] 按鈕。  
  
     資源會出現在適當的編輯器視窗中。  
  
 如需將資源加入至 Managed 專案的詳細資訊，請參閱《.NET Framework 開發人員手冊》中的[應用程式中的資源](../Topic/Resources%20in%20Desktop%20Apps.md)。  
  
 需求  
  
 Win32
