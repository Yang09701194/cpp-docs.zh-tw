# <a name="propget"></a>propget
指定屬性存取子函式。  
  
## <a name="syntax"></a>語法  
  
```  
[propget]  
```  
  
## <a name="remarks"></a>備註  
 **Propget** c + + 屬性具有相同的功能[propget](http://msdn.microsoft.com/library/windows/desktop/aa367145) MIDL 屬性。
