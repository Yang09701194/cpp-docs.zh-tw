# <a name="propget"></a>propget

指定屬性存取子函式。

## <a name="syntax"></a>語法

```cpp
[propget]
```

## <a name="remarks"></a>備註

**Propget** c + + 屬性具有相同的功能[propget](/windows/desktop/Midl/propget) MIDL 屬性。
