# propget
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

指定的屬性存取子函式。  
  
## 語法  
  
```  
  
[propget]  
  
```  
  
## 備註  
 **Propget 而言** C\+\+ 屬性具有相同的功能，為  [propget 而言](http://msdn.microsoft.com/library/windows/desktop/aa367145) MIDL 屬性。  
  
## 範例  
 請參閱範例的[可繫結](../windows/bindable.md) 的範例使用  **propget 而言**。  
  
## 需求  
  
### 屬性內容  
  
|||  
|-|-|  
|**適用於**|方法|  
|**可重複**|否|  
|**必要的屬性**|None|  
|**無效的屬性**|**propput**，  **propputref**|  
  
 如需有關屬性內容的詳細資訊，請參閱[屬性內容](../windows/attribute-contexts.md)。
