# <a name="propput"></a>propput

## <a name="syntax"></a>語法  
  
```cpp  
[propput]  
```  
  
## <a name="remarks"></a>備註  
 **Propput** c + + 屬性具有相同的功能[propput](http://msdn.microsoft.com/library/windows/desktop/aa367146) MIDL 屬性。
