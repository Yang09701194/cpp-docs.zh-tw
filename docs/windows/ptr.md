# <a name="ptr"></a>ptr

指定指標做為完整的指標。

## <a name="syntax"></a>語法

```cpp
[ptr]
```

## <a name="remarks"></a>備註

**Ptr** c + + 屬性具有相同的功能[ptr](/windows/desktop/Midl/ptr) MIDL 屬性。
