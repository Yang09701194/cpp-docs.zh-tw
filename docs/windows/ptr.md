# <a name="ptr"></a>ptr
將指標指定為完整的指標。  
  
## <a name="syntax"></a>語法  
  
```  
  
[ptr]  
  
```  
  
## <a name="remarks"></a>備註  
 **Ptr** c + + 屬性具有相同的功能[ptr](http://msdn.microsoft.com/library/windows/desktop/aa367149) MIDL 屬性。
