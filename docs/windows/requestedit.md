# requestedit
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

指示屬性支援 **OnRequestEdit** 通知。  
  
## 語法  
  
```  
  
[requestedit]  
  
```  
  
## 備註  
 **Requestedit** C\+\+ 屬性具有相同的功能，為 [requestedit](http://msdn.microsoft.com/library/windows/desktop/aa367155) MIDL 屬性。  
  
## 範例  
 請參閱範例的[可繫結](../windows/bindable.md) 的範例使用  **requestedit**。
