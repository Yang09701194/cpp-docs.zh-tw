# <a name="resizing-an-entire-image-image-editor-for-icons"></a>調整整個影像大小 (圖示影像編輯器)
### <a name="to-resize-an-entire-image-using-the-properties-window"></a>調整整個影像使用 [屬性] 視窗的大小  
  
1.  開啟您想要變更其屬性的影像。  
  
2.  在**寬度**和**高度**中[屬性 視窗](/visualstudio/ide/reference/properties-window)，輸入您想要的維度。  
  
     如果您要增加影像的大小，影像編輯器擴充到右方時，向下拖曳，映像，或兩者，並使用目前的背景色彩填滿新的區域。 不會自動縮放影像。  
  
     如果您要縮小影像的大小，影像編輯器會裁剪影像右側或底端邊緣，或兩者。  
  
    > [!NOTE]
    >  您可以使用 Width 和 Height 屬性只整個調整影像大小，不若要調整大小部分選取。
