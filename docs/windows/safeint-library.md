# SafeInt 程式庫
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

SafeInt 函式庫可以在應用程式執行數學運算時，協助防止整數溢位和其他可能遭利用的錯誤。  
  
## 在本節中  
  
|區段|說明|  
|--------|--------|  
|[SafeInt 類別](../windows/safeint-class.md)|這個類別可防止整數溢位。|  
|[SafeInt 函式](../windows/safeint-functions.md)|可以使用，而不需要建立 `SafeInt` 物件的函式。|  
|[SafeIntException 類別](../windows/safeintexception-class.md)|例外狀況類別與 `SafeInt` 類別相關的類別。|
