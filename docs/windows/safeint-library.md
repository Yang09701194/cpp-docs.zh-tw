# <a name="safeint-library"></a>SafeInt 程式庫

**SafeInt**程式庫可協助防止可能會造成應用程式執行數學運算時的整數溢位。 此程式庫的最新版本是位於[ https://github.com/dcleblanc/SafeInt ](https://github.com/dcleblanc/SafeInt)。
