# <a name="selecting-foreground-or-background-colors-image-editor-for-icons"></a>選取前景或背景色彩 (圖示影像編輯器)
除了**橡皮擦**，工具上**影像編輯器**工具列繪製以目前前景或背景色彩您分別按滑鼠左鍵或右鍵時。  
  
### <a name="to-select-a-foreground-color"></a>選取前景色彩  
  
1.  左邊的滑鼠按鈕時，按一下您想的要在[色彩調色盤](../windows/colors-window-image-editor-for-icons.md)。  
  
### <a name="to-select-a-background-color"></a>選取背景色彩  
  
1.  以滑鼠右鍵，按一下 您想要的色彩**色彩**調色盤。
