# Selecting Foreground or Background Colors (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

當您分別按下滑鼠左鍵或右鍵時，\[**影像編輯器**\] 工具列上的工具會以目前前景或背景色彩繪製 \(\[**橡皮擦**\] 除外\)。  
  
### 選取前景色彩  
  
1.  使用滑鼠左鍵，按一下[色彩調色盤](../windows/colors-window-image-editor-for-icons.md)上您想要的色彩。  
  
### 選取背景色彩  
  
1.  使用滑鼠右鍵，按一下**色彩調色盤**上您想要的色彩。
