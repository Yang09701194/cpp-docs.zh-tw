# <a name="selecting-multiple-controls"></a>選取多個控制項
### <a name="to-select-multiple-controls"></a>若要選取多個控制項  
  
1.  在[工具箱視窗](/visualstudio/ide/reference/toolbox)，選取**指標**工具。  
  
2.  拖曳指標以畫出您想要在對話方塊中選取的控制項周圍的選取方塊。  
  
     當您放開滑鼠按鈕時，所有控制內、 交集與選取方塊已選取。  
  
     \-或-  
  
-   按住 SHIFT 鍵，然後按一下您想要在選取項目中包含的控制項。  
  
     \-或-  
  
-   按住 Ctrl 鍵，然後按一下您想要在選取項目中包含的控制項。  
  
#### <a name="to-remove-a-control-from-a-group-of-selected-controls-or-to-add-a-control-to-a-group-of-selected-controls"></a>若要移除選取的控制項群組的控制項，或將控制項加入選取的控制項群組  
  
1.  與控制項選取的群組，按住 SHIFT 鍵並按一下您想要移除或加入到現有的選取範圍的控制項。  
  
    > [!NOTE]
    >  按住 CTRL 鍵並按一下選取範圍中的控制項將可控制主控該選取項目中。 如需詳細資訊，請參閱[指定主控](../windows/specifying-the-dominant-control.md)。  
  
 如需將資源加入至 managed 專案的詳細資訊，請參閱[桌面應用程式中的資源](/dotnet/framework/resources/index)中 *.NET Framework 開發人員手冊 》。* 如需手動將資源檔加入 managed 專案、 存取資源、 顯示靜態資源及指派資源字串給屬性的資訊，請參閱[建立桌面應用程式的資源檔](/dotnet/framework/resources/creating-resource-files-for-desktop-apps)。 全球化和當地語系化的受管理應用程式的資源上的資訊，請參閱[全球化和當地語系化的.NET Framework 應用程式](/dotnet/standard/globalization-localization/index)。
