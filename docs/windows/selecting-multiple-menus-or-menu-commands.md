# <a name="selecting-multiple-menus-or-menu-commands"></a>選取多個功能表或功能表命令
您可以選取多個功能表名稱或功能表命令，來執行大量作業，例如刪除或變更屬性。  
  
### <a name="to-select-multiple-menu-commands"></a>選取多個功能表命令  
  
1.  按住 CTRL 鍵，按一下功能表或您想要的子功能表命令。
