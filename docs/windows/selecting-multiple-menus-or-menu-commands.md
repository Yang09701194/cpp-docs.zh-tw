# Selecting Multiple Menus or Menu Commands
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

您可以選取多個功能表名稱或功能表命令，來執行大量作業，例如刪除或變更屬性。  
  
### 選取多個功能表命令  
  
1.  按住 CTRL 鍵，按一下功能表或您想要的子功能表命令。  
  
 如需將資源加入 Managed 專案的相關資訊，請參閱《.NET Framework 開發人員手冊》中的[應用程式中的資源](../Topic/Resources%20in%20Desktop%20Apps.md)。  
  
 **需求**  
  
 Win32
