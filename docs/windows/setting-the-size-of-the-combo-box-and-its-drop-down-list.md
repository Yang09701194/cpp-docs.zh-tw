# <a name="setting-the-size-of-the-combo-box-and-its-drop-down-list"></a>設定下拉式方塊和下拉式清單的大小
當您將它加入至對話方塊中，您可以調整大小的下拉式方塊。 您也可以指定下拉式清單方塊的大小。  
  
### <a name="to-size-a-combo-box"></a>若要調整大小的下拉式方塊  
  
1.  在對話方塊中，選取下拉式方塊控制項。  
  
     一開始只左邊和右邊的調整大小控點會有作用中。  
  
2.  您可以使用調整大小控點來設定下拉式方塊的寬度。  
  
 您也可以設定下拉式方塊的下拉部分的垂直大小。  
  
#### <a name="to-set-the-size-of-the-combo-box-drop-down-list"></a>若要設定大小的下拉式方塊的下拉式清單  
  
1.  按一下右邊的下拉式方塊的下拉箭號按鈕。  
  
     ![在 MFC 專案中下拉式方塊的箭號](../mfc/media/vccomboboxarrow.gif "vcComboBoxArrow")  
  
     外框的控制項變更，以擴充的下拉式清單區域顯示下拉式方塊的大小。  
  
2.  您可以使用較低的調整大小控點，變更下拉式選單區域的初始大小。  
  
     ![組合 &#45; MFC 專案中的方塊大小](../mfc/media/vccomboboxsizing.gif "vcComboBoxSizing")  
  
3.  按一下下拉箭號，即可關閉下拉式方塊的下拉式選單部份。
