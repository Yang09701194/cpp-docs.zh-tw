# Setting the Size of the Combo Box and Its Drop-Down List
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

當您將下拉式方塊加入對話方塊時，您可以調整下拉式方塊的大小。  您也可以指定下拉式清單 \(Drop\-Down List\) 方塊的大小。  
  
### 若要調整下拉式方塊的大小  
  
1.  在對話方塊中選取下拉式方塊控制項。  
  
     一開始，只有右邊和左邊的縮放控點是作用中的。  
  
2.  使用縮放控點來設定下拉式方塊的寬度。  
  
 您也可以設定下拉式方塊的下拉式部分的垂直大小。  
  
#### 若要設定下拉式方塊下拉式清單的大小  
  
1.  按一下下拉式方塊右邊的下拉式方向鍵按鈕。  
  
     ![MFC 專案中下拉式方塊上的箭頭](../mfc/media/vccomboboxarrow.png "vcComboBoxArrow")  
  
     控制項的外框變更以顯示具下拉式清單 \(Drop\-Down List\) 區域延伸的下拉式方塊的大小。  
  
2.  使用較低的縮放控點來變更下拉式清單區域的初始大小。  
  
     ![在 MFC 專案中調整下拉式方塊大小](../mfc/media/vccomboboxsizing.png "vcComboBoxSizing")  
  
3.  再按一下下拉式方向鍵關閉下拉式方塊的下拉式清單部分。
