# Swap 函式 (Windows 執行階段 C++ 樣板程式庫)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支援 WRL 基礎結構，而且不是為了要直接從您的程式碼中使用而設計。  
  
## 語法  
  
```  
WRL_NOTHROW inline void Swap(  
   _Inout_ T& left,  
   _Inout_ T& right  
);  
```  
  
#### 參數  
 `left`  
 第一個引數。  
  
 `right`  
 第二個引數。  
  
## 傳回值  
  
## 備註  
 交換兩個指定的引數值。  
  
## 需求  
 **標題:** internal.h  
  
 **命名空間：** Microsoft::WRL::Details
