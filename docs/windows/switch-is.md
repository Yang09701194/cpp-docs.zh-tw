# <a name="switchis"></a>switch_is

指定的運算式或做為選取的等位成員的聯集判別的識別項。

## <a name="syntax"></a>語法

```cpp
[switch_is]
```

## <a name="remarks"></a>備註

**Switch_is** c + + 屬性具有相同的功能[switch_is](/windows/desktop/Midl/switch-is) MIDL 屬性。
