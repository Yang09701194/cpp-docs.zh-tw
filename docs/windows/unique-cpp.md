# <a name="unique-c"></a>unique (C++)
指定唯一的指標。  
  
## <a name="syntax"></a>語法  
  
```  
[unique]  
```  
  
## <a name="remarks"></a>備註  
 **唯一**c + + 屬性具有相同的功能[唯一](http://msdn.microsoft.com/library/windows/desktop/aa367294)MIDL 屬性。
