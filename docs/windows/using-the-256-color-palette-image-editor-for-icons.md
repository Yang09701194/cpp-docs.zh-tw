# <a name="using-the-256-color-palette-image-editor-for-icons"></a>使用 256 色調色盤 (圖示影像編輯器)
若要選取繪製從 256 色調色盤，您需要從色彩調色盤中選取的色彩[色彩視窗](../windows/colors-window-image-editor-for-icons.md)。  
  
### <a name="to-choose-a-color-from-the-256-color-palette-for-large-icons"></a>若要從 256 色調色盤大型圖示選擇色彩  
  
1.  選取大型圖示或游標，或建立新的大型圖示或游標。  
  
2.  從 256 色顯示在 選擇色彩**色彩**調色盤中的**色彩**視窗。  
  
     選取的色彩將會變成目前的色彩調色盤中**色彩**視窗。  
  
    > [!NOTE]
    >  使用 256 色影像的初始調色盤符合 CreateHalftonePalette Windows API 所傳回的調色盤。 適用於 Windows shell 的所有圖示應該都使用這個調色盤來防止重繪閃動調色盤實現期間。
