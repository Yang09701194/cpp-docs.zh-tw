# VerifyInterfaceHelper::Verify 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支援 WRL 基礎結構，而且不是為了要直接從您的程式碼中使用而設計。  
  
## 語法  
  
```  
static void Verify();  
```  
  
## 備註  
 驗證目前樣板參數的介面是否符合特定需求。  
  
## 需求  
 **標題:** implements.h  
  
 **命名空間：** Microsoft::WRL::Details
