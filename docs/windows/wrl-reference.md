# WRL 參考
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

本節包含 [!INCLUDE[cppwrl](../windows/includes/cppwrl_md.md)] \([!INCLUDE[cppwrl_short](../windows/includes/cppwrl_short_md.md)]\) 參考資訊。  
  
> [!NOTE]
>  [!INCLUDE[cppwrl_short](../windows/includes/cppwrl_short_md.md)] 定義支援 [!INCLUDE[cppwrl_short](../windows/includes/cppwrl_short_md.md)] 基礎結構並不適合直接從程式碼使用的功能。  這類功能已在此文件中指出。  
  
## 在本節中  
 [Microsoft::WRL 命名空間](../windows/microsoft-wrl-namespace.md)  
 定義組成 [!INCLUDE[cppwrl_short](../windows/includes/cppwrl_short_md.md)]的基本型別。  
  
 [Microsoft::WRL::Wrappers 命名空間](../windows/microsoft-wrl-wrappers-namespace.md)  
 定義 Resource Acquisition Is Initialization \(RAII\) 簡化物件、字串和控制代碼的存留期管理的包裝函式型別。  
  
 [Microsoft::WRL::Wrappers::HandleTraits Namespace](../windows/microsoft-wrl-wrappers-handletraits-namespace.md)  
 描述一般基礎控制代碼的資源類型的特性。  
  
 [Windows::Foundation 命名空間](../windows/windows-foundation-namespace.md)  
 啟用 [!INCLUDE[wrt](../atl/reference/includes/wrt_md.md)] 基本功能，例如物件和 Factory 建立。
