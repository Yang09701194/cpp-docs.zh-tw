# <a name="nil-structure"></a>Nil 結構

支援 WRL 結構，而且不是直接從您的程式碼使用。

## <a name="syntax"></a>語法

```cpp
struct Nil;
```

## <a name="remarks"></a>備註

用來指出未指定，選用的範本參數。

**Nil**是空的結構。

`Nil`
